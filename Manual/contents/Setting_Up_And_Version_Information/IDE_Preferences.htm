IDE環境設定

プリファレンスウィンドウは GameMaker の重要な部分です。IDE の設定やカスタマイズ、特定のプラットフォームの SDK などの設定を行うことができます。このセクションでは、IDE の設定ウィンドウの特定のエリアをカバーします。個々のターゲットプラットフォームについては、ターゲットプラットフォームの設定のセクションを参照してください。このウィンドウで設定を変更した後、適用 ボタンをクリックして、設定を有効にする必要があることに注意してください。

ここでオプションを変更してGameMakerを自分好みにカスタマイズした後、設定した環境設定をバックアップする場合は、Windowsのこのフォルダに移動する必要があります。

%AppData%/GameMaker/<your_account_name>/

Macの場合はこのフォルダー

~/.config/GameMaker

そして、次の2つのファイルを安全な場所にコピーしてください。

  • local_settings.json
  • cloud_settings.json

ただし、変更した環境設定をリセットしたい場合は、ウィンドウ上部の「デフォルトに戻す」ボタンをクリックする必要があります。これは、現在編集しているセクションのデフォルトを復元するだけであることに注意してください。

IDE左上のFile メニューから、GameMakerPreferencesを開くことができます。

環境設定ウィンドウを開くと、次のようなさまざまなオプションがあります（このリストには、プラットフォーム固有の環境設定はすべて含まれていません。）

  • 一般
  • スタート画面
  • 画像編集ソフト
  • スプライトエディタ
  • オブジェクトエディタ
  • フォントエディタ
  • ルームエディター
  • スプラインエディタ
  • タイルセットエディター
  • タイムライン
  • エクステンションエディタ
  • テキストエディター
  • 言語設定
  • フェザーの設定
  • マーケットプレイス
  • プラグイン
  • ランタイムフィード
  • キーの再定義
  • アニメーション・カーブ・エディター
  • アセットブラウザー
  • シークエンス・エディター
