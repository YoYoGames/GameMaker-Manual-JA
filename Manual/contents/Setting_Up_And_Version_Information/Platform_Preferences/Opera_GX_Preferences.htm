Opera GXの環境設定

Opera GXの環境設定には、以下のオプションがあります。

  • Emscripten SDKの場所です。Opera GX 用の YYC ビルドに必要な Emscripten SDK の場所です。SDK のセットアップについては、「Opera GX 用のセットアップ」をお読みください。
  • Default WebServer Port: Opera GX Webserver のデフォルトポートを変更することができます。デフォルト値は51264です。
  • デフォルトのWebServer アドレス。ここでは、Opera GXのWebserver のデフォルトのWebserver アドレスを変更することができます。デフォルト値は127.0.0.1です。
  • Default WebServer Allowed IPs:ここで、Webserver 、任意の許可されたIPを追加することができます。これらは、ターゲットマネージャを介して追加されたすべてのデバイスのデフォルトとして設定されます。
