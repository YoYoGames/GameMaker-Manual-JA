HTML5の環境設定

HTML5の環境設定には、以下のオプションがあります。

  • デフォルトのパッケージングタイプです。最終的なHTML5実行形式パッケージを構築する際、Package as loose files（フォルダ内に必要なファイルを全て生成する）かPackage as Zip（ゲームと追加ファイルを含む圧縮ファイル ZIP を生成）のどちらかを選択することができます。デフォルトでは、コンパイル時にこの2つのオプションのどちらを選ぶか選択できるようになっていますが、常にどちらかを使うように設定することができます。このオプションがShow Dialogueに設定されている場合、コンパイル時にポップアップする実際のダイアログウィンドウで、Remember Packaging Optionを選択すると、この設定はあなたが選択したオプションに設定されることに注意してください。
  • Default WebServer Port: HTML5 Webserver のデフォルトポートを変更することができます。デフォルト値は51264です。
  • Default WebServer Address:ここでは、HTML5 の Webserver のデフォルトの Webserver アドレスを変更することができます。デフォルトは127.0.0.1です。
  • Default WebServer Allowed IPs:ここで、Webserver 、任意の許可されたIPを追加することができます。これらは、ターゲットマネージャを介して追加されたすべてのデバイスのデフォルトとして設定されます。

重要特定のターゲットデバイス（ブラウザなど）を使用する場合、これらの設定ではなく、その特定のデバイスに設定されたIPとポートの値が使用されます。デバイスの一覧とその詳細については、「デバイスマネージャー」を参照してください。
