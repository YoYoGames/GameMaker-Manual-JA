Windows UWPの環境設定

Windows UWP 環境設定には、以下のオプションがあります。

  • Visual Studioの場所。このオプションは、ビルドPCにインストールされているMicrosoftVisual Studio にGameMakerを指定するためのものです。
  • UWPツールの場所を指定します。このオプションは、ビルドPC上のUWPツールのインストール先（path ）をGameMakerに指定するためのものです。
  • パッケージのデフォルトのアーキテクチャ。デフォルトでは、UWP実行ファイルをコンパイルする際に、x86、x64 、ARMの いずれかのアーキテクチャを選択するためのダイアログウィンドウが表示されます。このオプションを設定することで のいずれかを選択すると、実行ファイルをコンパイルする際にデフォルトで選択されるようになります。
  • ストアパッケージ作成時の自動対応：コンパイル時に表示されるダイアログウィンドウで、ストアパッケージを作成するかどうかを選択できるようにしました。このオプションはYes またはNoの いずれかに設定することができ、デフォルトで使用されます。 実行ファイルをコンパイルするときは、必ずこのボタンをクリックします。
