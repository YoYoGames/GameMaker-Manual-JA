Windows環境設定

Windows 環境設定には、以下のオプションがあります。

  • Visual Studioの場所。このオプションは、ビルドPCにインストールされているMicrosoftVisual Studio にGameMakerを指定するためのものです。
  • デフォルトのパッケージングタイプです。最終的なWindows 実行可能パッケージを構築する際、インストーラー（NSIS を使用）または圧縮された ZIP ファイルのどちらかを作成することができます。 その中にゲームと追加ファイルが含まれます。デフォルトでは、コンパイル時にこの2つのオプションを選択するダイアログが表示されますが、このプリファレンスを設定して、常にどちらかを使用するようにすることができます。このオプションが Show Dialog, in actual dialog window that pops up on compile, enable "Remember Packaging Option" will set this preference to the option you select.
