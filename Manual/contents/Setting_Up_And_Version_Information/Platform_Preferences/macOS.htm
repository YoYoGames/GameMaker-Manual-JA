macOS環境設定

macOS 環境設定には、以下のオプションがあります。

  • デフォルトのチーム識別子。ここでは、Appleから割り当てられたデフォルトのチームIDを追加することができます。このチーム ID は、アプリをビルドするためにゲームファイルをXcode に送信するときに使用され、Xcode が必要な署名証明書を生成することを許可します。この設定は、macOS でビルドされたすべてのゲームにデフォルトで適用されますが、プロジェクトごとに GeneralmacOS Game Options から上書きすることができることに注意してください。
  • Automatic packaging choice: ここでは、macOS ターゲットの実行ファイルを作成する際のデフォルトの環境設定を行うことができます。Show Dialog" オプションは、実行ファイルを作成する前に必ずダイアログを表示し、DMG と Zip のどちらかを選択するよう求めますが、このオプションを Preferences でどちらかに設定すると、このダイアログをスキップすることができます。なお、ダイアログで「パッケージのオプションを記憶」を有効にすると、このオプションも選択されたパッケージの種類に設定されます。
