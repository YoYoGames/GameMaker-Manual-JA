HTML5

HTML5のデバイスマネージャーは、実行時にターゲットとする異なるブラウザーをデバイスリストに追加するためのものです。まず始めに をクリックします。 Detect Browsers"ボタンをクリックすると、お使いのマシンにインストールされている各ブラウザのデバイスが作成されます。お使いのブラウザがリストにない場合は、[ブラウザの検出]ボタンをクリックすることもできます。 Add New Browser」ボタンを押し、デバイスに名前を付け、GameMakerにブラウザの実行ファイル用のpath を指定する。

追加の利用可能なフィールドを設定することで、ブラウザがプロジェクトを実行する方法を制御するのに役立ち、以下のオプションが利用可能です。

  • ブラウザのコマンドライン引数。ここでは、ブラウザpath に追加する任意のコマンドライン引数を追加することができます。これらのリストは、主な一般的なブラウザの次のリンクから見つけることができます。
    • クローム(Chrome)
    • ファイアフォックス
    • インターネットエクスプローラー
  • WebServer アドレス。Webのアドレス サーバをターゲットにします（デフォルトは127.0.0.1）。これはAPIの仕様によって変更する必要があります。例えば、Facebookではこれを http://localhost (そして、そのアプリコンソールでホワイトリストに登録されます)。
  • WebServer Port: 設定します。 webserver のポート番号（デフォルトは51264）。
  • Allowed WebServer IPs / CIDR IP Ranges:必要に応じて、許可された Webserver 、ホスト名、および CIDR IP 範囲を設定します。

Ubuntu (Linux)

Ubuntu ( ) デバイスをマネージャに追加するには、まず「LinuxAdd New Device 」というボタンをクリックする必要があります。 デバイスが追加され、接続を有効にするために必要ないくつかの空白のフィールドが作成されます。

Windows UWP

Windows 10 UWP ターゲットのデバイスを追加する前に、デバイスでDeveloper Modeを有効にしてDevice Portalをセットアップする必要があります。これを行わないと、GameMakerが通信できなくなる。 デバイスと連携し、UWPアプリを適切に作成することができます。

Windows 検索で「Developer Mode」と入力し、表示される「設定」項目をクリックすることで、簡単に開発者モードを有効にすることができます。すると、以下のような画面が表示されます。

デベロッパーモードのオプションを選択する必要があります（この際、Windows がダウンロードされる場合があります）。 をインストールし、その下のオプション「Enable Device Portal」を選択します。デバイスポータルは、GameMakerが Windows 10 PCと通信するためのもので、有効にすると、次のことが要求されます。 ユーザー名とパスワードを入力してください。

ユーザー名とパスワードは、ポータルに固有のものでなければならず、Microsoftアカウントで使用されているものであってはなりません。 またはその他のWindows の認証情報です。この情報を入力したら、「適用」 ボタンをクリックし、設定マネージャーを閉じて、GameMakerに戻ります。

デバイスポータルの詳細については、こちらのマイクロソフトの記事を参照してください。デスクトップ用デバイスポータル。

これで、GameMakerの デバイスマネージャーを開き、使用するデバイス（複数可）を設定することができます。

Windows 10 UWP 準拠のデバイスをマネージャーに追加するには、まず「Add New Device」と書かれたボタンをクリックする必要があります。 デバイスが追加され、接続を有効にするために必要ないくつかの空白のフィールドが作成されます。

まず、ターゲットリストで識別できるようにデバイス名を 付け、次にホストを 指定します。 これは、ゲームをホストするネットワーク上のWindows 10 デバイスの IP アドレスで、ポート番号を設定するために " :50443" を末尾に付ける必要があります。その後、Device Portalで設定したUser Nameと Passwordを 追加します（前述のとおり）。PINは空白のままでもかまいません。 は、UWPWindows 10 Desktop では必要ありません。

異なるUWP出力（デスクトップ、XboxOne）をカバーする、より具体的な設定の詳細については、YYGヘルプデスクをご覧ください。UWPのセットアップを参照してください。

すべてのデバイスウィンドウの下部には、「デバイスの消去」というオプションがあり、これをクリックすると、接続されているすべてのデバイスがリストから消去されます。

デバイスを設定したら、GameMaker IDE の上部にあるターゲットオプションから、どれか1つのデバイスに接続すれば、ゲームのテスト/ビルドがそのデバイスにプロジェクトをプッシュします。
