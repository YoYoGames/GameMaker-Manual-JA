スプラインエディタの環境設定

スプラインエディタの環境設定は、次のように使用します。 この設定は、IDEの他の部分（Physics Shape など）にも影響しますが、主にPath Editorのスムースpaths の見え方を定義するために使用します。 エディター)。以下のオプションが存在します。

  • Insertpath ポイント精度 (px)。これは、path 、すなわち、マウスがこの範囲内になければ、新しい点を追加するのではなく、点を挿入するための精度値（ピクセル単位）を設定します。デフォルトは20pxです。
  • アンカーポイントの幅：アンカーポイントの幅をピクセル単位で設定します。デフォルトは8pxです。
  • アンカーポイントの高さ。アンカーポイントの高さをピクセル単位で設定します。デフォルトは8pxです。
  • パスカラー：編集時にpath を表示するために使用する色です。デフォルトは (ARGB) #FFFFFFFF です。
  • Highlightedpath segment color: 選択されたpath セグメントをハイライトするために使用される色です。デフォルトは(ARGB)#FFFFFFFF。
  • Invalidpath color: 有効でないpath を強調するために使用される色。デフォルトは(ARGB)#FF0000FF。
  • アンカーポイントの色：path アンカーポイントの描画に使用される色。デフォルトは (ARGB) #FFC0C0C0 です。
  • 無効なアンカーポイントの色：path アンカーポイントを描画するために使用する色です。デフォルトは(ARGB)#FF800000。
  • Selected anchor point color: ハイライト（選択）されたpath アンカーポイントの描画に使用される色です。デフォルトは(ARGB)の#FF00A4FF。
  • 選択された無効なアンカーポイントの色：ハイライトされているが無効なアンカーポイントを描画するために使用される色です。デフォルトは(ARGB)の#FFA400FFです。
  • マウスオーバー時のアンカーポイント色：アクティブになっているアンカーポイントの描画に使用される色です。デフォルトは(ARGB)#FFFFFFFFです。
  • マウスオーバー無効なアンカーポイント色：有効だが無効なアンカーポイントを描画するために使用する色。デフォルトは(ARGB)#FF0000FF。
  • Mask overlay color: マスクが有効な場合（上記参照）、マスクのオーバーレイカラーを設定します。デフォルトは(ARGB)#FF404040です。
