ルームエディターの環境設定

ルームエディター環境設定] は、ルームエディターの外観と動作を定義するために使用します。一般的な設定は以下の通りです。

  • 鎖のフェードアウト開始距離（px）。チェーンリンクがフェードアウトする際の親ウィンドウからの距離を設定します。デフォルトは10pxです。
  • 鎖のフェードアウト停止距離（px）。チェーンリンクがフェードアウトを終了する親ウィンドウからの距離を設定します。デフォルトは100pxです。
  • チェーン・ミニマム・アルファチェーンアルファのフェードアウトのフォールオフポイントの後の最終的なアルファ値を設定する。デフォルトは0.3。
  • レイヤーの深さのデフォルトの間隔。room エディタで複数のレイヤーを作成すると、レイヤーリスト内での位置に基づいて、自動的に深さが割り当てられます。この深さはここで指定された値だけ増えます。デフォルト値は 100 で、各レイヤーは前のレイヤー + 100 の深さになります。
  • Defaultroom width: 新しいroom のデフォルトとして選択される初期幅を設定します。デフォルトは1024pxです。
  • デフォルトのroom の高さ。新しいroom のデフォルトとして選択される初期高さを設定します。デフォルトは768pxです。
  • デフォルトのroom レイヤーを追加します。新しいroom リソースを作成すると、GameMakerは背景レイヤーとインスタンスレイヤーの2つの「基本」レイヤーを生成します。この挙動をswitch でオフにすることで、新しいrooms にあらかじめ用意されたレイヤーを持たせないことができます。これはデフォルトでオンになっています。
  • room ビューを表示します。room 内でビューがアクティブになっている場合、room 内のビューの境界線がアウトラインで表示されます。この領域の可視性は、エディタ自体の専用ボタンを使用して切り替えることができます。しかし、このオプションをオフにすることで、room を開いたときにビューの境界を非表示にすることができます（ボタンを使用して再び表示することは可能です）。デフォルト値はオンで、room を開いたときにビューの境界線が見えるようになります。
  • インスペクタの選択色：インスペクタの複数選択リストで選択されている項目を強調するために使用される色です。
  • その他のカラーオプション。これらは、room エディタインターフェイス全体で使用されているさまざまな色に関連しており、いずれかのスウォッチをダブルクリックすると、その設定の色を編集することができます。
  • Master-volume setting（マスターボリューム設定）。Room Editorでプレビュー再生するときのマスターボリュームの設定で、0〜1の値を指定します。

以下のサブカテゴリーから、さらにオプションを編集することができます。

確認ダイアログ

Confirm Dialogs 環境設定は、特定のアクションを実行したときにroom エディタに表示されるメッセージを変更するために使用します。変更できるメッセージは以下の通りです。

  • レイヤーの深さを更新した場合の自動応答: レイヤーの深さを変更すると、他のすべてのレイヤーの深さも自動的に更新されることを警告するメッセージが表示されます。このオプションでは、メッセージを表示するか、「OK」をクリックした場合と同じようにレイヤーを更新するか、「いいえ」をクリックした場合と同じように何も更新しないかを選択することができます。
  • レイヤーがロックされている警告への自動対応 room でレイヤーを削除しようとしたときに、たまたまそれがロックされていた場合、デフォルトではメッセージが表示されます。しかし、このオプションを「OK」に設定すると、メッセージは表示されず、ただレイヤーを削除することができなくなります。
  • アイテムロック警告への自動対応 ロックされているアイテムを削除しようとした場合、デフォルトではメッセージが表示されます。しかし、このオプションを「OK」に設定すると、メッセージは表示されず、アイテムの削除ができなくなります。
  • アイテム所有者のレイヤーがロックされている警告への自動対応レイヤー上のアイテムを削除しようとしたときに、レイヤーがロックされていた場合（アイテムがロックされていなくても）、デフォルトでは削除できないことを知らせるメッセージが表示されます。しかし、このオプションを「OK」に設定すると、メッセージは表示されず、アイテムの削除ができなくなります。
  • シーケンス／スプライトを不適当なレイヤーにドラッグすると、新しいasset レイヤーを作成する。sprite レイヤーでないレイヤーにsequence （またはasset ）を追加しようとすると、デフォルトではそれができないことを知らせるメッセージが表示され、ドラッグされるアイテムのために新しいasset レイヤーを作成するかどうかを尋ねられます。しかし、このオプションを「OK」に設定すると、メッセージは表示されず、自動的にsprites (およびsequences) の新しいレイヤーが作成されます。
  • 親room のアイテム削除に対する自動応答。ここでは、親room の一部であり、すべての子rooms に存在するアイテムを削除したときにどうするかを選択できます。デフォルトでは警告メッセージが表示され、どうするか決められますが、「すべて削除」を選択するとメッセージが表示されず、親room とその子すべてからアイテムが削除されます。また、「すべて保持」を選択するとメッセージが表示されず、削除が中止されます。
  • 親room のレイヤーを削除したときの自動応答。ここでは、親room の一部であるレイヤーを削除した場合の動作を選択できます（したがって、すべての子rooms に存在します）。デフォルトでは警告メッセージが表示され、どうするか決められますが、「すべて削除」に設定するとメッセージが表示されず、親room とその子すべてからレイヤーが削除されますし、「すべて保持」を選択するとメッセージが表示されず、削除が中止されます。
  • 無効なasset ドラッグエラーへの自動対応。asset のタイプに指定されていないレイヤーにasset を追加しようとすると（インスタンス レイヤーにpath を追加するなど）、デフォルトではそれができないことを通知するメッセージが表示されます。しかし、このオプションを「Ok」に設定すると、メッセージは表示されず、asset を追加できないだけです。このオプションは、インスタンス、sprites またはsequences には適用されないことに注意してください。
  • objects を不適当なレイヤーにドラッグした場合、新しいインスタンス レイヤーを作成する。インスタンス レイヤーでないレイヤーにobject インスタンスを追加しようとすると、デフォルトでは、それができないことを知らせるメッセージが表示され、ドラッグされるアイテムのために新しいインスタンス レイヤーを作成するかどうか尋ねられます。しかし、このオプションを "Ok" に設定すると、メッセージは表示されず、自動的にインスタンスのための新しいレイヤーが作成されます。
  • 無効なレイヤー選択ドラッグエラーへの自動応答。レイヤーエディターで複数のレイヤーを選択している（またはレイヤーを選択していない）ときに、room エディター上にリソースをドラッグしたときの動作を選択できます。デフォルトでは警告メッセージが表示されますが、このオプションを「OK」に設定すると、メッセージが抑制され、リソースを追加せずに続行できます。
  • ロックされた、または不可視のレイヤードラッグエラーへの自動応答。ここでは、ロックされたレイヤーや不可視のフラグが立っているレイヤーにリソースをドラッグしたときの動作を選択できます。デフォルトでは警告メッセージが表示されますが、このオプションを「OK」に設定すると、メッセージが抑制され、リソースを追加せずに続行することができます。

グリッドの設定

グリッド設定］環境設定は、グリッドオーバーレイの動作方法を変更するために使用し、以下のオプションが利用できます。

  • Grid horizontal spacing (px)（グリッドの水平方向の間隔）。room グリッドのガードセルの水平方向の間隔をピクセル単位で設定します。デフォルト値は32pxです。
  • グリッドの垂直方向スペーシング（px）。room グリッドのガードセルの垂直方向の間隔をピクセル単位で設定します。デフォルト値は32pxです。
  • グリッドを表示する。このオプションは、最初にroom を開いたときに、room グリッドのオン/オフを切り替えます。デフォルトではオンになっていますが、オフにした場合でも、room エディタ自体の適切なボタンを使って再び有効にすることができます。
  • グリッドにスナップ room このオプションでは、assets を最初に開いたときに、room がroom グリッドにスナップするかどうかをデフォルトで設定することができます。デフォルトではオンになっていますが、オフにした場合でも、room エディタ自体の適切なボタンを使って再び有効にすることができます。
  • グリッド線の色：このオプションでは、room のグリッド線の色（アルファ成分を含む）を設定することができます。デフォルトは(ARGB) #30FFFFFF です。


スマートガイド

このセクションでは、Room エディタの「スマートガイドの環境設定」について詳しく説明します。利用可能なオプションは次のとおりです。

  • スマートガイドを有効にする。Room Editorでスマートガイドの使用を有効/無効にするために使用します。デフォルトは有効です。
  • ルーラーを有効にする。このオプションは、Room Editorのサイドルーラーを有効/無効にします。デフォルトは有効です。
  • ルーラーをエッジにスナップ(Snap Rulers To Edges)。Room このオプションを有効にすると、ルーラからassets Editor にドラッグされたガイドがroom に配置された任意のものにスナップされます。 デフォルトは有効です。
  • スナップ距離(Snap Distance)。Roomこの設定は、ガイドがasset の側面に「スナップ」するまでの距離をasset Editor に指示します。 デフォルト値は 6px です。
