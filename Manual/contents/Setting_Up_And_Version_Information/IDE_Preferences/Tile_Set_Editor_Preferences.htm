タイルセットエディタの環境設定

Tile Set EditorのPreferencesには、Auto Tilingのオプションがあります。

Autotile background color: このオプションは、Autotile set を作成するセクションのTiles エディター内の背景色を設定します。デフォルトの色は、(rgba) #808080FFです。

Tile Set Editor 環境設定には、Confirm Dialogs のサブセクションもあります。

ここで利用可能なオプションは、セルを変更したときに表示されるメッセージを制御します。 tile set のサイズを変更すると、room タイルマップ レイヤーのレンダリング方法に影響します（レイヤーがクロップされます）。デフォルトでは、操作を続けるかキャンセルするかのプロンプトが表示されますが、このオプションで「OK」を選択すると、このメッセージを抑制することができます。 この場合、tile set タイルマップレイヤーへの影響に関係なく、room セルの変更が実行されます。デフォルト値は "Show Message "です。
