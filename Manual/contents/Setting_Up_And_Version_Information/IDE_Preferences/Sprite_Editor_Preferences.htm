スプライトエディタの環境設定

スプライトエディターの環境設定は、スプライトエディターの外観や機能を定義するために使用されます。以下のオプションがあります。

  • sprite のデフォルトの原点です。このオプションでは、新規に作成するsprite リソースのデフォルトの原点位置を設定することができます。デフォルトは「Top Left」です。
  • Defaultsprite collision mask kind:このオプションでは、新しいsprites で使用するコリジョンマスクのデフォルトの種類を設定することができます。選択肢は、矩形、回転矩形、楕円、菱形、または精密/フレームごとの精密です。デフォルトは矩形です。
  • sprite デフォルトのアニメーションモード。このオプションは、sprites のデフォルトアニメーションモードをsprite エディタで設定することを許可します。Frames per second" と "Frames per game frame" のどちらかを選択できます。 デフォルトは"Frames per second"です。
  • Defaultsprite animation speed: このオプションでは、新しいsprite asset を作成する際のデフォルトのアニメーション速度を設定することができます。この値はアニメーションモードと連動していることに注意してください。 は、1ゲームフレームあたり30フレームと同じではないので、好みのモードを参照してこの値を正しく設定するようにしてください。デフォルトは30です。
  • Defaultsprite width: このオプションでは、新規に作成されるすべてのsprites のデフォルトの幅を設定することができます。デフォルト値は64です。
  • sprite のデフォルトの高さ。このオプションは、すべての新しいsprites のデフォルトの高さを設定することができます。デフォルト値は64です。
  • リサイズとスケールをsprite の原点に適用します。このオプションをチェックすると、sprite がサイズ変更または拡大縮小されたときにsprite の原点が変更され、新しい原点はsprite の新しい値と一致するように移動/拡大縮小されます。 これは、sprite エディターでロックされていないカスタムsprite オリジンにのみ適用されます。原点がロックされている場合、またはsprite でデフォルトの位置（例："Center" や "Top Left" など）を使用している場合、このオプションが適用されます。 はグレーアウトしており、変更することはできません。デフォルトは有効です。

また、Sprite Editorには以下のサブカテゴリが存在します。

確認ダイアログ

ここで選択できるオプションは、アクションの確認が必要なときに表示されるメッセージを制御します。これらの設定を変更することで、メッセージを常に表示したり、特定のアクション（YesまたはNoのいずれか）でメッセージを解除したりすることができます。

  • ファイルインポート時の自動対応。新しい画像をインポートする際、すでに画像が存在する場合、デフォルトではインポートを続行して既存の画像を上書きするか、操作をキャンセルするかを尋ねるプロンプトが表示されます。このオプションを使うと、このメッセージを表示しないようにできます。 常に「いいえ」を選択してインポートをキャンセルするか、「はい」を選択して現在のsprite を上書きしてください。デフォルトは "メッセージを表示 "です。
  • 画像読み込み失敗時の自動対応新しい画像をインポートする際、インポートに失敗すると、デフォルトでは失敗したことを知らせる警告メッセージが表示されます。このオプションで「OK」を選択すると、このメッセージが表示されないようにできます。 初期値は「メッセージを表示」です。
