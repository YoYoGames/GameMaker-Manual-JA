スタート画面の環境設定

スタート画面の環境設定では、スタートページの動作を変更することができます。主なオプションは次のとおりです。

  • ツアーを表示する。IDE の起動時にイントロダクション・ツアーを表示するかどうかを選択できます。これは、最初にツアーをスキップしたときに自動的に無効になります。
  • セットアップ・ウィザードを表示する。IDE の起動時にセットアップウィザードを表示するかどうかを選択できます。セットアップ・ウィザードは、基本的に新しいプロジェクトを作成するために必要なステップを案内してくれます。
  • 最近のプロジェクト一覧のスタイル。Start PageのRecent Projectsリストの表示を「List」と「Grid」の間で変更できます。
