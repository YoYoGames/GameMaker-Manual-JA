画像編集ソフトの環境設定

イメージエディタ環境設定] は、イメージエディタウィンドウの外観を定義するために使用します。以下は一般的なオプションです。

  • Expandsprite controls overlay by default:これをチェックすると、編集のためにsprite を開くたびに、sprite コントロールとフレーム情報がイメージエディタに表示されます（これは、折りたたみボタン ただし、後でsprite を編集する際に再び開きます）。このチェックを外すと、sprite のコントロールとフレーム情報が非表示になります。 sprite デフォルトでは、このチェックはオフになっています。
  • デフォルトのLoop モード。イメージエディターのloop ボタンは、2つの異なるモードで機能するように設定できます：1回再生（最後のフレームに到達するとアニメーションが停止します）、loop 最初から再生（アニメーションが移動します また、ピンポン方式では、画像の上流から下流に向かい、再び上流から下流に向かうアニメーションが表示されます。デフォルトは "loop "です。
  • 選択範囲の輪郭を細く描画します。チェックを入れると、選択範囲の輪郭は画像の縮尺に関係なく絶対的な幅1pxになり、チェックを外すと相対的な幅1pxになります。 で、画像に合わせて拡大縮小します。デフォルトでは、このオプションはチェックされています。
  • 選択範囲の輪郭の原色: 選択範囲の輪郭は、2つの色を交互に並べた破線になります。このオプションはそのうちの最初の色 (アルファを含む) を設定するもので、デフォルト値はARGB#FFFFFD0 です。
  • 選択部分のアウトラインの2色：選択部分のアウトラインは、2つの色を交互に繰り返す破線です。このオプションはそのうちの2番目の色 (アルファを含む) を設定するもので、初期値は ARGB #0000D0 です。
  • Editor background color:sprite フレーム編集中に画像が拡大・縮小されたり、ウィンドウ内の描画全体に収まらなくなることがあります。デフォルト値はARGB #282828FF。

画像編集環境設定には、以下のサブカテゴリもあります。

背景

ここでは、sprite の画像を編集する際の背景のオプションを設定することができます。デフォルトでは、グレーの正方形のグリッドですが、以下の設定により変更することができます。

  • キャンバスのスタイルです。ここでは、背景の「スタイル」を設定します。 ブロックパターン" または " 単色」です。シングルカラーは、背景を好みの色に設定するだけです（以下で説明するように、カラー1の設定を使用します）が、ブロックパターンはデフォルトの設定です（残りのダイアログを使用して編集することもできます のオプションがあります)。デフォルトは " ブロックパターン」。
  • キャンバスブロックサイズ。を選択した場合 背景のブロックパターンここでは、個々のブロックの大きさを設定することができます。デフォルトは16pxで、最小値2px、最大値256pxまで許容されます。
  • キャンバスの原色 背景に「単色」を選択した場合、この色が使用されます。色見本をクリックすると、カラーエディターが開き、変更することができます。を使用する場合 ブロックパターンオプションでは、代替ブロックの最初の色を設定します。デフォルトの色は、( argb) #ffc3c3c3です。
  • キャンバス地の2次色（ブロックパターン風）。もし、あなたが 背景にブロックパターンを選択すると、色1（上）と交互に使用される色はこの色になります。色見本をクリックするとカラーエディターが開き、変更することができます。デフォルトの色は(ARGB)#FF8080です。

確認ダイアログ

ここでは、現在のsprite でさまざまなアクションを実行したときの確認ダイアログの動作を選択することができます。これらは

  • フレームを削除する際の自動応答デフォルトでは、フレームを削除する際に、本当に削除してよいかどうかを確認するメッセージが表示されます。このオプションを「OK」にすることで、このメッセージが表示されなくなり、フレーム削除が可能になります。 それ以上のメッセージは表示されません。デフォルトは「メッセージを表示」です。
  • ストリップのインポート時に画像を入れ替えた場合の自動対応。イメージドロップダウンメニューからストリップのインポートを選択すると、sprite リソースにすでにあるイメージを上書きすることになるという警告メッセージが表示されます。 をクリックし、続行するかどうかを選択します。このメッセージを表示するか、IDE が単に「はい」または「いいえ」を選択するかは、ここで設定することができます。デフォルトは「メッセージを表示する」です。
  • ツール変更時の変更をコミットする際の自動対応。フレームを編集した後に描画ツールを変更すると、デフォルトで変更をコミットするかどうかを確認するメッセージが表示されます。このオプションを "OK "に設定することで このメッセージを削除することで、ツールを変更したときにこれ以上メッセージを表示することなくフレームエディションをコミットすることができます。No」を選択した場合、ツールを変更したときにフレーム編集は静かにキャンセルされます。デフォルトは "Show メッセージ」を表示します。
  • ロックされたレイヤーの警告に自動的に対応するようにしました。ロックされたレイヤーを編集しようとすると、デフォルトでは編集できないことを知らせるメッセージが表示されます。このオプションを "OK" に設定すると、メッセージは表示されません。 で、エディションは無言で失敗します。
  • グループ選択警告への自動対応レイヤーエディターで複数のレイヤーを選択し、描画または編集しようとすると、デフォルトでそれが不可能であることを通知するメッセージが表示されます。OK "を選択すると をクリックすると、このメッセージが表示されなくなり、1つのレイヤーのみが選択されるまで、無言で画像をまとめて編集することができなくなります。デフォルトは「メッセージを表示」です。



グリッドの設定

グリッド 設定環境は、画像グリッドの表示方法を設定するもので、以下のオプションが用意されています。

  • Grid horizontal spacing (px)。この値は、画像グリッドの水平線の間隔を設定します。デフォルト値は1pxです。
  • Grid vertical spacing (px)。この値は、画像グリッドの垂直線の間隔を設定します。デフォルト値は1pxである。
  • グリッドを表示する。このオプションをオンにすると、イメージエディタはイメージグリッドを表示した状態で開かれます。デフォルトはオフです。
  • グリッドにスナップ(Snap To Grid)これをチェックすると、イメージエディターは特定のツールを自動的にイメージグリッドにスナップします。デフォルトはオフです。
  • グリッド線の色：ここでは、画像グリッドの色（アルファ値を含む）を設定できます。デフォルトは( argb) #ff000000 です。

ツール

ここでは、さまざまなオプションを設定することができます。 は、いくつかの描画ツールの操作方法と表示方法について説明します。

を使用している場合 円弧ツール で、ここで をクリックすると、ノードプロパティの描画に使用されるさまざまな色を設定できます。

  • アンカーポイントカラー：スプライン/円弧上の異なるアンカーポイントのカラーを設定します。デフォルトは( argb) #ffff0000 です。
  • 制御点の色：アンカーポイント上の制御点「ハンドル」の各色を設定します。デフォルトは(ARGB)#FFFF0000です。
  • 接線色：制御点とアンカーを結ぶ線の色を設定します。デフォルトは (ARGB) #FF00FFFF です。
  • 選択点カラー：スプライン/円弧上の選択されたアンカーポイントに使用されるカラーを設定します。デフォルトは (ARGB) #FF00FF00 です。
  • ポイントサイズ。使用するさまざまなアンカーポイントやコントロールポイントの大きさを設定します。デフォルトは10pxです。
