シークエンス・プリファレンス

Sequences Preferencesは、Sequences Editorとその要素の機能に関する特定のプロパティを定義するために使用されます。主なオプションは次のとおりです。

  • 新しく作成されたシーケンスにキャンバスフレームを表示する。有効にすると、すべての新しいSequence のsequences Editor のキャンバスにキャンバスフレームスマートガイドが表示されます。このオプションは、エディタ自体のキャンバスプロパティからも必要に応じて有効/無効を設定することができます。デフォルト値は有効です。
  • 新規作成されたシーケンスのキャンバスフレーム幅。ここでは、すべての新しいsequences のデフォルトのフレーム幅を設定することができます。デフォルト値は1366pxです。
  • 新規作成されたシーケンスのキャンバス枠の高さ。ここでは、すべての新しいsequences のデフォルトのフレームの高さを設定することができます。デフォルト値は 768px です。
  • キャンバス上でクリックすると一時停止する。このオプションをオンにすると、Sequence Editor キャンバスのどこかをクリックすると、sequence の再生が一時停止されます。これはデフォルトで有効になっています。
  • Overlap selection delay（オーバーラップ選択遅延）：ミリ秒単位で指定します。Sequence Editor のキャンバス上でアイテムが重なっている場合に、Item Toolbox がポップアップするまでの時間を制御します。デフォルト値は1000msです。
  • 翻訳path ポイント挿入マージン。Sequence Editor キャンバスで、asset を追加し、位置パラメータトラックを作成すると、path が移動するときに取るasset を示す点線が描かれます。この線の近くにマウスを置くと、新しいパラメータトラックのポイントを追加することができます。このオプションは、ポイントマーカーが表示されるまでにマウスが移動線から必要な距離を設定します。デフォルト値は6pxです。
  • デフォルトのパラメータトラックを自動的に作成する。このオプションを有効にすると、sequence のトラックパネルで作成された新しいトラックに、位置、スケール、回転、原点のパラメータトラックが自動的に追加されます。 デフォルトは無効です。
  • 新しく作成されたアセットを自動的に引き伸ばす。このオプションは、新しく作成されたすべてのasset キーをsequence の長さに合わせて伸張します。 デフォルトは無効です。
  • DopesheetにおけるAnim Curve Trackの高さ（Height of Anim Curve Track in Dopesheet）。このオプションは、keyframe ビューで、カーブを使用するパラメータトラックがドープシート内で何本のトラックラインを占めるかを設定するために使用できます。 デフォルトは2ですが、1または3のいずれかを選択することもできます。
  • 新規 Text Track のデフォルトテキスト。string Sequence で新しいテキストトラックを作成したときに使用されるテキストです。
  • インスペクタの選択色：インスペクタのマルチセレクトリストで選択されたトラックを強調するために使用される色です。

確認ダイアログ

Confirm Dialogs 環境設定は、特定のアクションを実行したときにsequence エディタに表示されるメッセージを変更するために使用します。変更できるメッセージは以下の通りです。

  • 編集したカーブを埋め込む際の自動選択編集したanimation curve を埋め込もうとすると、デフォルトで動作確認のメッセージが表示されます。しかし、このオプションを "Yes "に設定すると、メッセージが表示されなくなり、毎回埋め込むことができます。No "に設定すると、毎回埋め込むことができなくなります。
  • カーブからキーへの変換を自動選択 animation curve をkeyframes に変換しようとすると、デフォルトでは続行するかどうかを尋ねるメッセー ジが表示されます。しかし、このオプションを「Ok」に設定すると、メッセージは表示されず、カーブの変換ができなくなります。

スマートガイド

ここでは、Sequences Editor で作業する際のスマートガイドの動作を選択できます。選択できる項目は次のとおりです。

  • スマートガイドを有効にする。ここでは、Sequences Editor キャンバスのスマートガイドを有効または無効にすることができます。デフォルトでは有効になっています。
  • ルーラーを有効にするここでは、Sequences Editor キャンバスのスマートガイドを有効または無効にすることができます。デフォルトでは有効になっています。
  • ルーラーをエッジにスナップこのオプションは、ルーラーをassets キャンバスのsequences のエッジに「スナップ」させることを可能にします（逆も同様）。デフォルトは有効です。
  • スナップ距離。アイテムがルーラーに（またはルーラーがアイテムに）「スナップ」するために必要な距離（ピクセル単位）です。デフォルト値は6pxです。
