一般的な環境設定

一般設定は、GameMaker IDE の一般的な動作を定義するために使用されます。最も一般的なものはメインページにリストアップされていますが、他の多くのものは複数のサブカテゴリに分かれており、このページのさらに下で説明されています。

  • IDE言語。このオプションでは、IDE の言語を提供されているオプションのリストから選択することができます。デフォルトは英語です。
  • IDEフォント。このオプションで、font 全体で使用されるIDE を変更することができます。 デフォルトはOpen Sans です。
  • GameMaker Editorの最大フレームレート。GameMaker IDE が更新する最大フレームレートを設定します。モニターの更新速度に合わせてフレームレートを設定できます。デフォルトは60です。
  • IDEスキン。ここでは、IDE が使用できる様々なスキンを選択することができます。GameMakerにはデフォルトでライトとダークの2つのスキンが用意されており、初期設定ではダークのスキンが使用されます。
  • デフォルトのオーディオデバイス。GameMaker IDE で使用するデフォルトのオーディオデバイスを選択することができます。Refresh Button をクリックすると、リストを更新することができます。
  • 終了確認への自動対応GameMakerを終了する際に、確認メッセージを表示するかどうかを設定します。デフォルトでは「メッセージを表示する」になっていますが、「OK」をクリックしたときのようにメッセージをスキップするように設定することも可能です。
  • システムFont 変更ダイアログへの自動応答。このオプションでは、システムfont が追加、削除、編集されたときに確認メッセージを表示するかどうかを設定することができます。デフォルトでは「メッセージを表示する」に設定されていますが、「OK」をクリックした場合と同様にメッセージをスキップするように設定することもできます。
  • 通知を有効にする。これをチェックすると、IDE が新しいruntimes などの通知を表示するようになります。デフォルトではオンになっています。
  • 通知タイムアウト（ms）。これは、通知が表示される時間の長さ（ミリ秒）を設定します。デフォルトは6000です。
  • 終了時にログアウト IDE ：このオプションがチェックされていると、GameMakerが終了したときに自動的にユーザーセッションがログアウトされます。これは主に教育関係のユーザーを対象としており、チェックした場合、プログラムが終了するたびにユーザーがログアウトされ、他のユーザーがセッションシートを使用できるようになることを意味します。デフォルトではオフになっています。
  • 起動時にウェルカムスクリーンを表示するこれはデフォルトでオンになっており、プログラム起動時に簡単なスプラッシュスクリーンを表示するようGameMakerに指示するだけです。このチェックを外すと、スプラッシュスクリーンが表示されなくなります。
  • 空のタブをショートカットで閉じられるようにする：開いているタブはキーボードショートカット / +W で閉じることができますが、このオプションを無効にするとこのショートカットが機能しなくなります。このオプションは、デフォルトで有効になっています。
  • IDEにファイルをドラッグすると、編集用に新しいリソースを開くことができます。sprites 特定のリソース（Workspace やサウンドなど）をGameMakerのメインIDE にドラッグすると、Asset Browser で新しいリソースとして検出・追加されます。デフォルトでは、このとき対応するリソースエディタも編集用に開かれますが、このオプションをオフにするとエディタが開かれなくなり、手動で開く必要があります。
  • 作成時に新しいリソースを編集用に開くこれをチェックすると、Asset Browser で新しいリソースを作成するたびに、対応するリソースエディタが自動的に開かれます。これはデフォルトでチェックされており、チェックを外すと、新しく作成されたすべてのリソースを手動で開く必要があります。
  • 変更されたファイルを自動的に再読み込みする。これをチェックすると、IDE が自動的にファイルをスキャンしてチェックし、GameMaker以外で変更されたものがあれば再読み込みします。
  • IDE のトランジションアニメーションを無効にする。GameMaker IDE には、ウィンドウを開くなどのためにアニメーションによるトランジションが多数用意されています。しかし、これらは万人受けするものではなく、古いマシンや低パワーのマシンでは、これらを無効にするとパフォーマンスが向上する場合があります。ここで無効にすることができ、デフォルトでオンになっています。
  • フィードバックアンケートに参加する。このオプションがチェックされていると、GameMakerにフィードバック調査への参加を希望していることを示します。つまり、作業中に時々、作業中のエリアに関するメッセージが表示され、評価やフィードバックを求めることができます。このオプションはデフォルトでオンになっています。
  • DPI オーバーライドを有効にする。このオプションを使用すると、デフォルトのDPI設定を独自のもので上書きすることができます。まずこのオプションを有効にし、Raw DPi Override値（デフォルト値は96）またはネイティブDPiのパーセンテージ（デフォルト値は100%）を設定する必要があります。変更すると、IDE を再起動するよう促されるので、現在開いているプロジェクトで行われている作業を保存する必要があるかもしれません。DPi のパーセンテージは最小 50%、最大 500% で、情報アイコンをマウスオーバーすると、使用しているディスプレイの推奨最大設定値が表示されます。

    DPiの推奨値を超えると、DPiをリセットするための通知が表示されます。

上記のオプションは、「一般」 環境設定から利用できる主なオプションですが、以下に示すサブカテゴリーには、さらに多くのオプションが用意されています。

  • 背景
  • コンパイル
  • 確認ダイアログ
  • デバッギング
  • ファイルウォッチャー
  • ゴーゴー
  • ヘルプ
  • 入力
  • 出力ウィンドウ
  • パス
  • パワーセーブ
  • 最近のWindows
  • リソース
  • ワークスペース
