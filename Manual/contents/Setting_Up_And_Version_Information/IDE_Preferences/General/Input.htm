入力環境設定

入力 環境設定は、IDE におけるマウスとキーボード入力の特定の側面を制御します。 利用可能なオプションは次のとおりです。

  • ダブルクリックの時間が遅い(ms)。IDE の一部の領域では、リソースの名前を変更するときなど、正しく動作するために遅いダブルクリックが必要です（速いダブルクリックでリソースエディタを開き、遅いダブルクリックで名前を変更することができます）。このオプションでは検出速度を設定でき、デフォルト値は500msです。
  • スクロールロック時間(ms)。このオプションは、GameMakerがオーバーラップしているスクロールウィンドウの間で待つべき時間を扱います。基本的には、ウィンドウ "A" でスクロールしているときに、カーソルがウィンドウ "B" の別のスクロール可能な領域に重なることがあります。この環境設定は、"A" のスクロールを停止してから "B" をスクロールするまでの待ち時間を定義し、誤って別のパネルをスクロールし始めることがないようにするものです。デフォルト値は 500ms です。
  • パンニング時にカーソルを画面に回り込ませる。有効にすると、パンするときにカーソルを画面の境界線に巻き付けますworkspaces 。デフォルトは有効です。
  • スクロール速度：異なるworkspaces 内のスクロール速度を制御します。デフォルト値は1です。
  • デスクトップの最大化を解除するためのタブドラッグ距離(px)。workspace タブをIDE 内のデフォルト位置からドラッグしたとき、その時点でアクティブなworkspace が 1 つしかなければGameMakerウィンドウは最大化されず、マウスで移動します (workspace が複数ある場合はIDE の新しいインスタンスが作成されてworkspace が保持されます)。この設定は、workspace がタブを認識するまでにドラッグしなければならない距離を制御します。デフォルト値は 50px です。
  • ウィンドウの内容を操作するための最小ズームレベル (%)。このオプションは、マウスでworkspace ウィンドウを操作できなくなるズーム・カットオフ・ポイントをコントロールします（例：ノードの追加、ウィンドウのドラッグ、など）。デフォルト値は40です。
  • マウスホイールズームにはCtrlキーが必要です。これをチェックすると、すべてのエディタ（およびメインworkspace ）で、 / + マウスホイール によるズームが有効になり、チェックを外すと、マウスホイール MMB Icon のみによるズームが有効になります。
  • Ctrl + MMB で、マニュアルではなく、継承されたイベントを開くことができます。これをチェックすると、object の中で / + マウスの中ボタン を押すと、マニュアルではなくobjects の継承イベントを開きます（継承イベントがある場合）。
  • テキスト選択にバイアスをかけ、行飛びを防止します。この機能をオンにすると、任意のテキストウィンドウで1行を選択する際に、考えられた行の高さがパッドされ、垂直方向の不正確さが発生しにくくなります。デフォルトで有効になっています。
  • すべてのデバイスでラップトップモードを有効にする。これをチェックすると、IDE にラップトップモードボタンが表示され、GameMakerを起動しているデバイスがラップトップかどうかに関わらず、必要に応じてswitch のオン/オフが可能になります。デフォルトではチェックされていません。
  • Laptopモードを強制する。この設定は、Laptop Modeを使用可能かどうかを変更するもので、デフォルトで有効になっています。これを無効にすると、ラップトップでIDE を起動したときにラップトップモードが自動的に有効にならないようにすることができます。
