リソース環境設定

この設定には、プロジェクトで特定のリソースを どのように描画するかのオプションが含まれています。

  • 新規プロジェクトの初期デフォルトの描画色です。font を使用したり、色を指定せずにGameMaker の関数やGML Visual アクションを使用して図形を描画した場合、ここで設定された色が使用されます。デフォルトの色 は白 (ARGB: $FFFFFFFF ) です。
  • 新規プロジェクトのウィンドウの初期デフォルト色。このオプションは、HTML5 ターゲットの HTML ページの背景色を設定するために使用され、また、すべてのターゲット (HTML5 を含む) でrooms の外側に表示される色も設定されます。デフォルトの色 は黒（ARGB: $000000FF）です。
