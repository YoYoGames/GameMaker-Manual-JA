アセットブラウザの環境設定

アセットブラウザの環境設定は、アセットブラウザとその要素の機能に関する特定のプロパティを定義するために使用されます。主なオプションは以下の通りです。

  • クイックアクセスを表示する。このオプションを有効にすると、Asset ブラウザの上部に「Quick Access」セクションが表示されます。このオプションは、デフォルトで有効になっています。
  • 新規プロジェクトにデフォルトフォルダを作成します。このオプションを有効にすると、すべての新規プロジェクトはデフォルトのasset フォルダがすでに作成された状態で開始されます。無効にすると、すべてのプロジェクトで必要なデフォルトのroom と共にroom フォルダのみが作成されます。このオプションはデフォルトで有効になっています。
  • 警告を出さずに同時に開くことができる最大数のassets 。 / + で複数のassets を選択し、 を右クリックしてすべて開くを選択すると、一度に複数の を開くことができます。このオプションで指定された数以上を選択した場合、警告が表示されます。一度に開くassets が多すぎると、ローエンドのマシンでパフォーマンスの問題が発生する可能性があるからです。デフォルト値は10です。
  • 最近編集した項目の最大値。この設定は、assets ブラウザの「最近」セクションに表示されるAsset の数を制御します。デフォルト値は10です。
  • カラーパレットサイズ：ここでは、asset ブラウザのパレットで利用できる色の数を、asset グループで設定することができます。デフォルト値は16で、最小値は16、最大値は100です。
  • クイックアクセスでasset の色を適用します。このオプションをオンにすると、ブラウザのクイックアクセス・セクションのasset の色に、任意のasset タイプに設定された色が使用されます。このオプションをオフにすると、asset のカラーは使用されません。このオプションはデフォルトで有効になっています。
  • グループカラーは子カラーを上書きする。チェックすると、グループ内のすべてのassets の色にトップレベルのグループカラーが使用され、個々のasset に設定されたカラーは無視されます。デフォルトでは無効になっています。
  • asset ダイアログスタイルを作成します。Asset ブラウザの「Add Asset」ボタンをクリックすると、プロジェクトに追加可能なさまざまなassets を表示するウィンドウが表示されます。このウィンドウは、リストビューまたはグリッドビューで表示することができ、この設定は、デフォルトで表示される方を制御します。デフォルトはグリッドです。
  • デフォルトのソート順です。ブラウザでassets 、アルファベット順、AからZ、またはZからAへのソート順を設定することができます。デフォルトはAからZです。
  • デフォルトでフォルダーを先にソートします。これを有効にすると、asset ブラウザは、まずフォルダをソートし、次に「緩い」assets をソートします。一方、これを無効にすると、すべてのassets とフォルダが一緒にソートされます。この設定はデフォルトで有効になっています。
  • Sprite ダブルクリックで画像エディタが開く: デフォルトではasset の上で をダブルクリックすると、そのasset が編集用に開かれます。しかしsprites を扱う場合、初期設定の後、新しいasset を作成すると、sprite のプロパティではなく、画像を編集するためにそれを開くことがよくあるのです。このオプションを選択すると、sprite asset 自体の代わりに、sprite asset の画像エディタが開かれます。 を右クリックして「編集」を選択すれば、sprite のエディタを開くことができます。このオプションはデフォルトではオフになっています。
  • オブジェクトをダブルクリックすると、object の全イベントが開きます。デフォルトでは、任意のasset の をダブルクリックすると、そのasset が編集用に開かれます。しかし、objects を扱う場合、初期設定の後、新しいasset を作成すると、object のプロパティではなく、イベントのみを編集するために開くことがよくあります。このオプションを選択すると、さまざまなobject イベントが新しいworkspace で開かれ、各イベントは新しいworkspace の中で独自のタブが与えられます。 を右クリックして「編集」を選択すると、引き続きobject エディタを開くことができます。このオプションはデフォルトではオフになっています。
  • ルームオーダーアングループは、空のグループを削除します。有効にすると、グループ フォルダーに属しているすべてのrooms をグループ解除すると、グループ フォルダーも削除されます。無効の場合、同じ操作でrooms のグループ化を解除しますが、空のグループ フォルダはasset ブラウザに残ります。これはデフォルトで有効になっています。
  • デフォルトのズーム（%）。新しいプロジェクトを開始したときのAsset Browser のデフォルトのズーム率を設定します。デフォルト値は100%です。
  • 最小ズーム（%）。 Asset Browser で許可される最小ズーム値を設定します。デフォルトは50%です。
  • 最大ズーム（%）。 Asset Browser で許可される最大ズーム値を設定します。デフォルトは250%です。
  • ツリーラインを表示するここでは、assets ブラウザでAsset の横に表示されるツリーラインを有効/無効にすることができます。デフォルトでは、ツリーラインを表示するようにチェックされています。
  • フィルター変更時にツリーを展開する。このオプションは、Asset Browser でフィルタオプションを変更したときに、折りたたまれているツリーノードを展開するよう強制します。このオプションを無効にすると、この現象は起こりません。デフォルトは有効です。
