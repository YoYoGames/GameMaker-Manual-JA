インストールガイド

GameMakerは、ここに記載されている最低スペックのmacOS およびWindows デスクトップコンピュータでの使用を想定しています。

GameMakerのインストーラーは、YoYo Gamesウェブサイトの「YoYo Account」ページからダウンロードすることができます。インストール後、初めてGameMakerを起動すると、製品を使用する前にアップデートを要求される場合があります。アップデートのインストール後、製品を使用する前にYoYo Games アカウントのライセンス情報（詳細はこちら）を入力する必要がありますので、GameMaker の初回使用時にはインターネット接続環境が必要です。初回ログイン後はオフラインで使用できますが、定期的にログインアカウントを確認するために接続が必要となります。

なお、お持ちのライセンスの種類によっては、特定のプラットフォーム用にGameMakerを設定し使用するために、追加のコンポーネントをインストールしたり、追加の手順を踏んだりする必要がある場合があります。各プラットフォームで必要なものの詳細については、YoYo Games Helpdeskのこちらの記事をご覧ください。GameMaker必須SDK.

安定版とベータ版

YoYo Games Web ページからダウンロードしたバージョンは、現在のGameMaker の安定 版であるべきですが、最終的な安定ビルドに入る前に新機能の紹介やテストに使われる「ベータ」バージョンも利用できる場合があります。バージョンの詳細は以下の通りです。

  • Stableです。このバージョンは、一般的な開発に使用されるもので、あらゆる商用製品での使用を推奨します。ベータ版で利用可能な一部の機能が欠けている可能性がありますが、これは完全にテストされていないか、追加の作業が必要なためです。
  • ベータ版ベータ版GameMakerは、安定版に追加される前の新機能を試したい方、または問題の修正が必要な方のためのものです。ただし、プロジェクトの障害となっている重要なバグの修正が含まれていない限り、最終的なゲーム制作にベータ版を使用することはお勧めできません。GameMakerのBetaビルドは、YoYo アカウントの「ダウンロード」ページから入手でき、Stableビルドと並行してインストール、実行することが可能です。Steamを通して GameMakerを起動している場合、Steam を使用して製品のベータ版ビルドにアクセスすることはできませんが、デスクトップ版をダウンロードして使用することはできますのでご注意ください。
