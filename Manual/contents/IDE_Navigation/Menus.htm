メニュー

GameMaker IDE の上部は、メニューバーで占められています。ここには、いくつかのドロップダウンメニューがあります。 IDE 、タスクを実行したり、ゲーム制作に役立つツールのある追加ウィンドウを開いたりすることができます。ここでは、利用可能なデフォルトのメニューをリストアップしていますが、asset エディタの中には、コンテキストに応じて追加のメニューが追加されるものがあることに注意してください。これらの追加メニューは メニューについては、作成したエディタに関連するページで説明しています。

デフォルトで用意されているメニューは以下の通りです。

  • ファイルメニュー
  • 編集メニュー
  • ビルドメニュー
  • Windows メニュー
  • ツールメニュー
  • Marketplace メニュー
  • レイアウトメニュー
  • ヘルプメニュー
