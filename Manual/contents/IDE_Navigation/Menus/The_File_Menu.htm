ファイルメニュー

file menu では、プロジェクトの変更、保存、インポートのほか、GameMaker IDE の環境設定を行うことができます。利用可能なオプションは以下の通りです。

  • 新規プロジェクト - GameMakerでゼロから新しいプロジェクトを作成するためのものです。Save "ダイアログが表示され、新しいプロジェクトを保存する場所を選択するよう促されます。プロジェクト名と保存場所を決めると、新しい空のワークスペースに移動し、作業を開始します。プロジェクトファイルは、 *.yyp ファイルextension で保存されます。
  • Open Project- ファイルエクスプローラを開いて、以前に作成した *.yyz プロジェクトファイルを選択し、読み込むことができます。プロジェクトが2.3.0より低いバージョンで作成された場合、インポート時に次のような警告メッセージが表示されますのでご注意ください。 続行を選択すると、ファイルエクスプローラが開き、プロジェクトに最新の名前と保存場所を与えることができるので、元のプロジェクトが上書きされることはありません。プロジェクトが新しい2.3.0+フォーマットに変換されると、IDE で開き、変換に成功した旨のメッセージが表示されます。変換処理によって生成された互換性レポートを確認し、プロジェクトを実行またはコンパイルする前に修正する必要があるものがあるかどうかを確認するために進みます。2.3.0 以前のバージョンで作成されたプロジェクトを開こうとして何度もエラーが発生する場合は、変換しようとしている元の *.yyp ファイルを提示してYoYo Games に連絡してください（ヘルプ メニューの「バグを報告する」オプションを使用します）。
  • プロジェクトのインポート- *.yyz アーカイブとしてエクスポートされたプロジェクトをインポートすることができます（下記「プロジェクトのエクスポート」参照）。Studio 1.4で作成したプロジェクトの場合、このオプションを選択すると、その製品ラインの *.gmx または *.gmz ファイルのいずれかをインポートできます。プロジェクトを開くときと同様に、 *.yyz が 2.3.0 アップデート以前のGameMakerで作成されたものであれば、そのアップデートでプロジェクトファイルのフォーマットが変更されたため、変換する必要があります。

注意：以前のGameMaker プロジェクトをGameMakerにインポートするには、1.4 バージョンで作成されている必要があり、その場合でもインポート時に直接動作しない場合があります。GameMakerとGameMaker Studio 1.4では、非常に多くのことが異なっています。GameMakerとGameMaker: Studio 1.4では多くの点が異なるため、古いプロジェクトを動作させるには若干の調整が必要になる場合があります。GameMaker Knowledge BaseのPorting A GM:S 1.4 Game ToGameMakerの記事と、マニュアルのObsolete Functions andCompatibility Scriptsのセクションを参照してください。

重要! font リソースを含む *.yyz プロジェクトをインポートする場合、fonts を削除/再追加するか、font エディターでfont を変更し、再度設定することにより、「再生成」する必要がある場合があります。これは、エクスポート時に既存のfont の情報が *.yyz で保持され、後でインポートするときや別のマシンでインポートするときに、プロジェクトに必要なfonts がすでにインストールされていない（または同じ名前の別のfont がある）可能性があるためです。これは、インポート後のfont のレンダリングに問題がある場合のみ行う必要があります。

  • 最近使ったプロジェクト- このオプションを使って、最近使ったプロジェクトを表示できます。これを選択すると、スタートページが再び開き（閉じている場合）、そこにあるリストから最近使用したプロジェクトを選択することができます。
  • Save Project- プロジェクトを作成したときに定義した場所（または開いた場所）にゲームを保存します。
  • Save Project As- 上記と同じですが、プロジェクトの場所と新しい名前を指定します。
  • プロジェクトのエクスポート - プロジェクトをファイルまたはテンプレートとしてエクスポートすることができます。
    • YYZ- これは、単一の *.yyz ファイルとしてプロジェクトをエクスポートし、コピーを人に送信したり、バックアップを保存するために非常に便利です。
    • テンプレートとして- これは、あなたが後でスタートページの他のプロジェクトのためのベースとして使用することができ、テンプレートとしてプロジェクトをエクスポートすることができます。このオプションを選択すると、次のプロンプトが表示されます。
      まず、[テンプレート情報の編集]をクリックしてテンプレート情報が正しいことを確認し、2回目にこのプロンプトを開いたときに[エクスポート]を選択してテンプレートとして保存する必要があります。その後、スタート画面の「テンプレート」メニューの下に表示されます。
  • 環境設定 - これを選択すると、環境設定ウィンドウが表示され、GameMakerの外観や使用感を設定することができます。
  • New IDE- これを選択すると、IDE の新しいインスタンスが開かれ、使用することができます（スタートページに開かれます）。
  • マイアカウント- このオプションを選択すると、デフォルトのウェブブラウザが開き、YoYo アカウントページが表示されます。
  • ログアウト- これを選択すると、GameMakerからログアウトします。後で製品を使用するためには、同じライセンス（または他のライセンス）で再度ログインする必要があります。

重要! GameMakerのFreeライセンスを使用していて、他のライセンスを購入した場合、新しいライセンスが正しく適用されているか確認するために、IDE からログアウトし、GameMakerを再起動して再度ログインする必要があるかもしれません（自動的に行われるはずですが）。

  • Exit - GameMakerをデスクトップに終了させます。
