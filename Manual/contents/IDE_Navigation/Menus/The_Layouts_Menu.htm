レイアウトメニュー

このメニューは、IDE の異なるレイアウトの保存と読み込みに関連するものです。IDE を自由に設定し、特定の要素（Asset Browser など）を設定することができます。 やOutputWindows など、デフォルトとは異なる配置になっています。また、このメニューのオプションで、現在のプロジェクトのレイアウトを *.xml ファイルとして保存し、どのプロジェクトにもロードして同じレイアウトに設定することができます。このメニューのオプションは以下の通りです。

  • Load Layout- ファイルエクスプローラを開き、以前に保存したレイアウト *.xml ファイルを選択してロードすることができます。
  • Save Layout- ファイルエクスプローラを開き、現在のIDE レイアウトに名前を付けて *.xml ファイルとして保存することができます。
  • Reset Layout- 現在のIDE のレイアウトをデフォルト設定にリセットします。
