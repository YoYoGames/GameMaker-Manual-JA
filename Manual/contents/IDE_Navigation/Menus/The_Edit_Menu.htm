編集メニュー

edit menu では、GameMakerインターフェイスの一般的なエディションツールを使用することができます。利用できるオプションは以下の通り。

  • Undo - 操作を元に戻す。
  • Redo - 前のアクションをやり直す。
  • 検索と置換 - 指定した入力に対して、コード全体（room 作成コード、scripts 、イベントを含む）を検索することができる検索ウィンドウが表示されます。

Edit menu から「検索と置換」を選択すると、次のようなウィンドウが表示されます。

これを使って、ゲーム内のすべてのスクリプトassets を検索することができます。room コード、イベントコード、scripts 、shaders を含みます。検索するとき 上部のボックスに新しい検索語を入力するか、右側の矢印アイコンをクリックすると、以前に検索した語のリストが表示されます。また、最初の検索語を置き換えるための置換語を指定することもできます。 この場合も、右側の矢印ボタンから、以前に使用した置き換えstring を使用するオプションがあります。

検索は一般的なものから、検索語入力の下にあるオプションを使って絞り込むことができます。

  • 大文字と小文字を区別して検索することもできます。この場合、検索対象の単語はstring の大文字と小文字に完全に一致するものだけが含まれます。たとえば、このオプションを指定せずに "My_Var" と検索すると "my_var" と表示された結果が得られます。 「mY_VAR"、"My_Var "のいずれかですが、このオプションをオンにすると、"My_Var "だけが検索結果に含まれるようになります。
  • この場合、検索語を含むが完全一致しない検索結果は無視されます。 をチェックすると、関数 "game_end()" や変数 "game_state" のようなものが表示されますが、これをチェックして検索すると "game" そのものの結果のみが表示されます。
  • コメントを無視した検索を行うかどうかを選択することができます。これをチェックすると、検索語を含むコード内のコメントは単に無視され、出力結果には表示されません。
  • asset ブラウザーのさまざまなasset の種類に基づいて、検索結果にフィルタを 適用することができます。デフォルトでは「すべて」に設定されていますが、これをクリックすると以下のメニューが表示され、チェックを入れたり外したりすることができます。 assets を検索対象にします。

検索語（オプションで置換語）を指定し、検索オプションを設定したら、下部にあるさまざまなボタンを使って を選択し、検索方法と実行するアクションを選択します。これらのボタンは、次のとおりです。

  • すべて検索すべての検索語を検索し、出力ウィンドウに表示します。
  • 前検索：検索語の前のインスタンスを検索します。これにより、asset の参照を含むworkspace が開かれ (まだ開かれていない場合)、その参照にパンすることができます。
  • 次を探す検索語の次のインスタンスを検索します。これにより、asset の参照を含むworkspace が開かれ (まだ開かれていない場合)、その参照にパンすることになります。
  • 全置換：このオプションを設定すると、指定した用語を検索し、さらに、assets 、指定した新しい置換用語に置き換えます。
  • 置換します。現在のエディタで現在選択されている用語がハイライトされている箇所を、選択した置き換え用語で置き換えます。
  • 検索を停止する。検索が進行中の場合は、検索を停止します。

このウィンドウが開いているときにEscape キーを押すと、選択したフィールドがクリアされますが、そのフィールドがすでに空の場合は、ウィンドウが閉じられます。

検索と置換」ウィンドウを使用したすべての結果は、IDE の下部にある「検索出力ウィンドウ」に表示されます。それぞれの結果は、 ダブルクリックすると、現在のasset にその用語を含むworkspace を開き、そこに移動して編集できるようになります。
