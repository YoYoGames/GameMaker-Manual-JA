Windows メニュー

ドッキングされた一般的なIDE のウィンドウを開くためのメニューです。ドックは再配置可能で、その内容は独立したウィンドウとして削除でき、また完全に閉じることもできるので、このウィンドウを使えば、何かで必要になったときに再び開くことができます。 ここから開くことができるウィンドウは、アセットブラウザウィンドウと、すべての異なる出力ウィンドウ（出力、検索、Source Control, Breakpoints とコンパイル/シンタックスエラー)。

次に、インスペクターウィンドウを開くことができます。このウィンドウは、現在選択されているIDE 要素のプロパティを検査するために使用され、これらのプロパティは必要に応じて編集することができます。

paths インスペクター] オプションの後に [最近のウィンドウ] リストがあり、[環境変数]ウィンドウを開くオプションもあります。 環境設定で使用されている変数が指し示すものです。これらの変数は編集できませんが（名前もパスも設定も）、選択してコピー＆ペーストすることで を他の場所に移動します。

最後に、「チュートリアルウィンドウを開く」オプションがあります。これは、チュートリアル・プロジェクトに関連するウィンドウを再び開くものです。 ただし、そのプロジェクトがチュートリアルの一部として作成されたものである場合のみです（そうでない場合は、このオプションは使用できません）。独自のチュートリアルプロジェクトの作成については、マーケットプレイスのセクションを参照してください（すべてのチュートリアルは、そこから作成、ホスト、ダウンロードされるため）。
