キーボードショートカット

GameMaker には膨大な数のキーボードショートカットが用意されており、そのほとんどはIDE で関連するメニュー項目の横にマークされていますが、すべてのショートカットが用意されているわけでも、最初からユーザーにとって明らかであるわけでもありません。ここでは、簡単なリファレンスガイドとして、すべてのキーボードショートカットの完全なリストを見つけることができます。一般的に、ショートカットは次の 3 つのカテゴリに分類されます。

  • グローバル- グローバルショートカットとは、GameMaker IDE のどこにいても使えるショートカットのことで、特定のウィンドウやガジェットにフォーカスが当たっていて、それが上書きされない限りは使えるショートカットです。
  • ウィンドウ- ウィンドウのショートカットは、ガジェットのショートカットに上書きされない限り、1つのウィンドウまたはタブに固有のショートカットです。
  • ガジェット- ガジェットのショートカットは非常に特殊で、1つのガジェット（例えばカラーピッカー）を使用するときにのみ機能します。

以下は、ショートカットの完全なリストで、エディターごとにセクションが分かれています。
