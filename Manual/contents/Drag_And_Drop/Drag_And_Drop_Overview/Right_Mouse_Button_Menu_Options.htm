このメニューには、以下のオプションがあります。

  • カット／ コピー／ ペースト。workspace で選択されているアクションをカット、コピー、ペーストすることができます。 なお、アクションの外側で + を使用してドラッグしたり、個々のアクションで / + を使用することができます。 を選択すると、複数を選択して、そのすべてにコマンドが適用されます。
  • コメント/ コメントの削除。このオプションは、GML Visual コードブロックにコメントを追加したり削除したりすることを可能にします。コメント］を選択すると、選択したGML Visual ブロックの右側にテキスト・エディターが開き、そこで以下を行うことができます。 何かを変更する際の注意事項、チュートリアルに役立つコメント、あるいはチームメイトへの指示など、必要なテキストを追加します。
    コメントを追加したら、それをクリックしてドラッグすることで、垂直方向や水平方向に配置できます。 親ノードの右側のみ -GML Visualノードを折りたたむと、コメントも折りたたまれ、その内容の概要が表示されます（この場合、位置の変更はできません）。コメントが不要になった場合、そのコメントをすべて削除することができます。 を使用するか、メニューの［コメント削除 ］を使用します。ノードをコピーすると、コメントも一緒にコピーされます。また、ノードをGML Visual からGML に変換すると、コメントは保持されます。 を作成し、下の図のように、作成されたGML に追加します。
    上記のように、コメントは1つのノードにしか適用できませんが、複数のノードに適用できるように拡張することができます。 この場合、後続のノードと重ならないように、コメントの位置を変更する必要があるかもしれません。
    一度コメントを追加したら、「コメント」オプションを選択することができます。 または、 をダブルクリックすると、編集が可能になります。
  • 無効化／ 有効化。これらのコマンドは、選択したアクションを無効化または有効化するために使用されます。アクションを無効にすると、workspace に表示されたままですが、ゲームをコンパイルするときに実行されなくなります。このように このように、テスト用とdebugging 用にアクションを追加/削除することができます。下の画像は、アクションを無効にしたGML Visual コードを示しています。
