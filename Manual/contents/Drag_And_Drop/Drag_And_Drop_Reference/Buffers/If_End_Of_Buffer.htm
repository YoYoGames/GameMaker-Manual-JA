If バッファーの終わり

このアクションは、読み取り/書き込み位置がbuffer の終端に達したかどうかを確認するために使用できます。アクションCreate Bufferが返す一意のbuffer ID 値を指定すると、アクションは、読み取り/書き込み位置がbuffer のすべての利用可能データの終端に達した場合は true 、達していない場合は false を返します。flag "not "を指定すると、アクションが読み取り/書き込み位置がbuffer の末尾にないかをチェックします。
