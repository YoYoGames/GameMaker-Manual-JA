コピーバッファー

このアクションは、あるbuffer に保存されているすべてのデータを別のものにコピーするために使用することができます。2つのbuffers を使用し、一方から他方へコピーする場合、両方のbuffers が事前にCreate Bufferアクションを使用して作成されている必要があります。 で、from/to 引数として返された一意のbuffer ID を指定します。

注：同じbuffer にコピーすることはできません。

アクションのシンタックス。

論拠となる。

論証 商品説明
Buffer からコピーするbuffer のインデックス（変数に格納）。
Copy To にコピーするbuffer のインデックス（変数に格納）。

例

上記のアクションブロックのコードは、新しいbuffer を作成し、その ID を一時変数に格納します。 次に、指定されたグローバル変数のインデックスを持つbuffer が存在するかどうかを確認し、存在する場合は一時的なbuffer にコピーして読み取り/書き込み位置を先頭にリセットし、存在しない場合は作成します。
