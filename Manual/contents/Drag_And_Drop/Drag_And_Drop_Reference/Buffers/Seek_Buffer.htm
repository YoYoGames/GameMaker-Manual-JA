シークバッファー

このアクションは、buffer 内の位置に移動 (シーク) するために使用します。アクションCreate Bufferが返す一意のbuffer ID 値と、移動先のオフセット値 (バイト) を指定します。オフセット値は buffer したがって、この関数を呼び出すと、読み取り/書き込み位置がbuffer の先頭に移動し、そこから指定されたバイト数だけ移動して、新しい読み取り/書き込み位置がそこに配置されます。ただし，フラグ を相対値として指定すると、現在の位置から前方（オフセットが負の値の場合は後方）へ読み書きの位置を移動させることができます。
