バッファの削除

このアクションは、buffer をメモリから削除し、その作成に使用したリソースを解放し、現在含まれているかもしれないデータを削除するために使用することができます。アクション「Create Buffer」が返す一意のID値（buffer ）を指定します。 削除する複数のbuffers を選択するには、プラスアイコン アクションの横にある 別のbuffer ID を指定します。

注：メモリリークを防ぐため、動的に作成されたリソースは、不要になった時点で必ずメモリから削除することが重要です。
