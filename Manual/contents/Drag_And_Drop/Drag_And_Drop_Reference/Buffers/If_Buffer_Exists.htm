If ExistsBuffer

このアクションは、指定されたbuffer ID が有効で、buffer が実際にメモリ内に存在するかどうかを確認するために使用することができます。buffer IDが有効で、かつ存在する場合、このアクションは true を返し、そうでない場合は false を返します。flag "not "を指定することができます。 を使用すると、buffer が存在しないかどうかを確認するアクションになります。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。
