ポップスタック

このアクションを使用すると、スタックデータ構造の先頭から値をポップアップすることができます。スタックインデックスを保持する変数（Create Stackアクションによって返される）を指定し、さらにそのインデックスを保持すべきターゲット変数を指定します。 を返します（一時的なローカル変数としてフラグを立てることができます）。

アクションのシンタックス。

論拠となる。

論証 商品説明
Stack 使用するスタックのインデックス(変数に格納)
Target 戻り値を格納するターゲット変数

例

上記のアクションブロックのコードは、与えられたスタックが空かどうかをチェックし、空でなければ、ポップス を取り出し、一時変数に格納します。スタックが空の場合、10 個のランダムな値がスタックにプッシュされます。
