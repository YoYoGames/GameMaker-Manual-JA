リスト作成

このアクションは新しいリストデータ構造を作成し、インデックス値を返すので、後で他のデータ構造アクションでリストにアクセスすることができます。リストのインデックスは、指定されたターゲット変数に返されます。 Assign VariableまたはDeclare Tempを使用して作成するか、flag "Temp" チェックボックスを使用して、一時的なローカル変数に名前を付けて作成することができます。 を使用して、script またはイベントの終了まで値を保存します。新しく作成されたリストデータ構造は、「空」、つまりリスト項目を含んでいないとみなされます。プラスアイコンをクリックすると、さらにDSリストを作成することができます。 アクションの横にある別の変数を選択し、リストIDを格納します。

重要!データ構造を作成すると、ターゲット・プラットフォームのメモリを消費します。 メモリリークにより、ゲームのパフォーマンスが低下したり、クラッシュしたりする可能性があります。これは、データ構造のインデックスを格納するために一時的なローカル変数を使用している場合に特に関連します。 だからといって、データ構造が削除されるわけではありません。データ構造はまだ存在しますが、それを参照する方法がないため、インスタンス変数を使用して後で構造体を解放するか、または イベントや関数のインデックスが一時変数に格納されている場合。
