クリアデータ構造

このアクションは、指定されたデータ構造からすべての値をクリアします。まず、クリアするデータ構造の種類を選択し、次にデータ構造のインデックスを指定します（インデックスは を使用します。）クリアされたデータ構造は、グリッドを除いて「空」（つまりエントリーがない）とみなされます。グリッドの場合、このアクションは単にすべてのグリッドセルを0にクリアします。

注：これはデータ構造をメモリから解放するのではなく、あらゆる情報を消去するのみです。データ構造に割り当てられているメモリを解放するには、Free Data Structureアクションを使用する必要があります。
