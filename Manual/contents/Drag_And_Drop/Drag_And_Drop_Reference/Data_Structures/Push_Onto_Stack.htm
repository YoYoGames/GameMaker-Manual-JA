プッシュオンザスタック

このアクションを使うと、スタックデータ構造上に値をプッシュすることができます。スタックインデックスを保持する変数（Create Stackアクションによって返される）を指定し、その上にプッシュする値を指定します。スタックデータ 構造体では、値は常にスタックの「先頭」に追加され、最後や途中には追加されません (この機能が必要な場合は、リストデータ構造体を使用します)。

アクションのシンタックス。

論拠となる。

論証 商品説明
Stack 使用するスタックのインデックス(変数に格納)
Value スタックにプッシュする値

例

上記のアクションブロックのコードでは、インスタンス変数と新しいスタックデータ構造を作成しています。 スタックのインデックスが新しい変数に格納され、次にloop が実行され、10 個のインスタンスが作成され、それらの固有の ID 値がスタックにプッシュされます。
