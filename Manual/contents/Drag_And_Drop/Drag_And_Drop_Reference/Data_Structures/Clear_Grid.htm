クリアグリッド

このアクションでは、グリッドデータ構造を単一の値にクリアすることができます（グリッド内の各セルは指定された値に設定されます）。グリッドのインデックスを保持する変数 (アクションCreate Grid が返す) を指定します。 を指定し、次にグリッドをクリアする値を指定します。
