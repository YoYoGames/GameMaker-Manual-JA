リストに追加

このアクションは、リストに新しい値（任意のデータ型）を追加するために使用することができ、この値は最後に追加されます。リストのインデックスを格納する変数(Create Listアクションによって返される)と値 を格納する。

アクションのシンタックス。

論拠となる。

論証 商品説明
List に追加するリストのインデックス（変数に格納）。
Value リストに追加する値

例

上記のアクションブロックのコードは、グローバルスコープ変数を作成し、その後に 新しいリストデータ構造体を作成し、そのインデックス値をグローバル変数に代入しています。その後、object "obj_Enemy_Parent" のすべてのインスタンスが、その固有のインスタンス ID 値をリストに追加するように、スコープが変更されます。
