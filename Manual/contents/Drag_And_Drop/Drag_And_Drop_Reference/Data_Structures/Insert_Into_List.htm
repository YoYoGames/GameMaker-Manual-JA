リストへの挿入

このアクションを使うと、与えられたリスト内の項目のインデックス位置を復元することができます。リストのインデックスを格納する変数 (リストを作成するアクションによって返されたもの) とインデックス位置を見つけたい値を指定します。 と、返された項目のインデックス位置を格納するターゲット変数 (script またはイベントの終了まで使用する一時的なローカル変数としてフラグを立てることができます) を指定します。リストに値を挿入するとき、すべてのインデックス つまり、4つの項目からなるリスト（インデックス位置0〜3）のインデックス位置2に新しい項目を置くと、前の項目2が項目3に、前の項目3が項目4になります。 探している値がリスト内に存在しない場合は、返すべきリストのインデックス位置が存在しないので、戻り値は-1になることに注意してください。

List に挿入するリストのインデックス（変数に格納）。
Index 値を挿入するリスト内のインデックス位置
Value の位置に格納される値。

例

上記のアクションブロックのコードでは、リピート（loop ）を使ってリストのサイズを取得し、次に リスト内のランダムな位置次にランダムな値を生成し、この値を先に定義したランダムな位置のリストに挿入します。
