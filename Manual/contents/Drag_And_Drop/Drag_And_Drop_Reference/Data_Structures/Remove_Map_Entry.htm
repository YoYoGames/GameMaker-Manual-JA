マップエントリーを削除する

このアクションを使うと、マップデータ構造からキー（とそれに対応する値）を削除することができます。マップのインデックスを保持する変数（Create Mapアクションによって返される）を与え、次に "キー "を与える（これは はマップ内の値を表す識別子です）、このアクションはそのキーをマップから削除します。このアクションは何も返さないので、指定されたキーの値が必要な場合は、削除する前にマップの値を取得するアクションを使用する必要があることに注意してください。

アクションのシンタックス。

論拠となる。

論証 商品説明
Map から値を削除するマップのインデックス（変数に格納）。
Key 使用するキー（real またはstring ）。

例

上記のアクションブロックは、マップデータから特定のキーの値を取得します。 構造体から削除し、その値が 0 以下であることを確認します。
