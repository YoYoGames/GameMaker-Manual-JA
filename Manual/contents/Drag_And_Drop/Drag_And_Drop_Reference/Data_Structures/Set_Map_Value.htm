マップ値の設定

このアクションでは、キーと値のペアをマップデータ構造に追加することができます。マップのインデックスを保持する変数（Create Mapアクションによって返される）を与え、次に "キー"（これは識別子である）を与える。 と、そのキーに関連づけられる「値」である。どちらのキーも整数値（0, 200, 36712, など）か、strings （"Name", "Shield" など）であるが、値には任意の 有効なデータ型である。

アクションのシンタックス。

Map 設定するマップのインデックス（変数に格納）。
Key 設定するキー（integer またはstring ）。
Value 与えられたキーとともに格納される値
