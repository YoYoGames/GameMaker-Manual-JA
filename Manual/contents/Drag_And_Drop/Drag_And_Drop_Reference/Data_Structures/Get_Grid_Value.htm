グリッド値取得

このアクションでは、グリッドデータ構造内の特定のセルに値を設定することができます。グリッドのインデックスを保持する変数（Create Grid アクションによって返される）を指定し、x と y の位置（column と行)を設定し、最後にそのセルの値を設定する。また、返されたセルの値を格納するターゲット変数（一時的なローカル変数としてフラグを立てることができる）を設定する必要があり、その値は有効な任意のデータ型にすることができる。注意 セルの x, y 位置はグリッドの範囲内で整数でなければならない。

アクションのシンタックス。

論拠となる。

論証 商品説明
Grid 取得するグリッドのインデックス(変数に格納)
x 取得するX軸方向のセル位置（列）。
y 取得するY軸方向のセル位置（行）。
Target 戻り値を格納するために使用するターゲット変数
