グリッドの設定値

このアクションでは、グリッドデータ構造内の特定のセルに値を設定することができます。グリッドのインデックスを保持する変数（Create Grid アクションによって返される）を指定し、x と y の位置（column と行)を設定し、最後にそのセルの値を設定する。値は任意の有効なデータ型でよいが、x と y の位置はグリッドの範囲内で整数でなければならない。

アクションのシンタックス。

論拠となる。

論証 商品説明
Grid 設定するグリッドのインデックス(変数に格納)
x 設定するx軸方向のセル位置（列）。
y 設定するy軸方向のセル位置（行）。
Value グリッドセルに設定する値
