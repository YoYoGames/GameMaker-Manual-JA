一覧から削除する

このアクションは、与えられたリストからインデックス項目を削除するために使用することができます。リストのインデックスを格納する変数 (アクションCreate List が返すもの) と、削除するリスト内のインデックス位置 (ここで インデックス位置は 1 から (リスト長 -1) の間である。このアクションは何も返さないので、インデックス位置の値が必要な場合は、インデックスを削除する前にGet List Item Atを使用する必要があります。

アクションのシンタックス。

論拠となる。

論証 商品説明
List から削除するリストのインデックス(変数に格納)。
Index 削除するリスト内のインデックス

例

上記のアクションブロックのコードは、インスタンス位置での衝突をチェックし、もし そのインスタンスに固有の ID 値が一時変数に格納され、それがリストデータ構造内に存在するかどうかが確認されます。もし存在すれば、そのアイテムはリストから削除され、インスタンスは破棄されます。
