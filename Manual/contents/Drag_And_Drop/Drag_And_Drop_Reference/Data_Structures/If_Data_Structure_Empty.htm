データ構造体が空の場合

このアクションを使うと、与えられたデータ構造が空かどうかをチェックすることができます。まず、チェックするデータ構造の種類を選び（リスト、マップ、スタックのいずれか、グリッドは空にもできないので不可）、次に データ構造のインデックスを保持する変数（Create List、Create Map、Create Stackの各アクションで返されます）。このアクションは、以下の場合に true を返します。 の場合、データ構造は空であり、空でない場合は false となる。なお、flag "not" をアクションに入れると、データ構造が空でないかをチェックして、データを保持しているかどうかを確認することができる。
