マップバリューを取得する

このアクションを使うと、マップデータ構造内の指定されたキーに関連する値を取得することができます。マップインデックスを保持する変数（Create Mapアクションによって返される）を与え、"key "を与える。 はマップ内の値の識別子である）、さらに返された値を格納するターゲット変数（これは一時的なローカル変数としてフラグを立てることができる）を指定します。

注：マップに存在しないキーを指定した場合、このアクションは undefined を返します。これは、If Undefinedアクションを使用して確認することができます。

アクションのシンタックス。

論拠となる。

論証 商品説明
Map 値を取得するマップのインデックス（変数に格納）。
Key 取得するためのキー（本物またはstring ）。
Target 返されたマップの値を格納するターゲット変数

例

上記のアクションブロックのコードは、インスタンス位置での衝突をチェックし、衝突した場合 が見つかったら、マップを検索してキー "damage" から値を取得し、この値を使って呼び出し元のインスタンス変数 "hp" を変更します。
