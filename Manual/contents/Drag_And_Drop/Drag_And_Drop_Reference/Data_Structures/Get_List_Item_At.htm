リスト項目を取得

このアクションを使うと、与えられたリスト内の項目のインデックス位置を回復することができます。リストのインデックスを格納する変数 (リストを作成するアクションによって返されたもの) と、リスト内のインデックス位置を指定して と、返された項目の値を格納するターゲット変数（script またはイベントが終了するまで使用する一時的なローカル変数としてフラグを立てることができます）を取得します。

注意: 指定したリストサイズ以外の位置を指定した場合 (たとえば、10個の値からなるリストの11番目の位置)、このアクションは undefined を返します。これを確認するには、If Undefinedアクションを使用します。

アクションのシンタックス。

論拠となる。

論証 商品説明
List から削除するリストのインデックス(変数に格納)。
Index 項目を取得するためのリスト内のインデックス位置
Target 戻り値を格納するターゲット変数
