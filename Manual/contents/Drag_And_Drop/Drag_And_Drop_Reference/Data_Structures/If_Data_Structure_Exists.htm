データ構造が存在する場合

このアクションを使用すると、変数がデータ構造のインデックスを作成しているかどうかを確認することができます。まず、チェックするデータ構造の種類（リスト、マップ、グリッド、スタック）を選び、次にそのデータ構造を保持する変数 インデックス（Create List、Create Map、Create Grid、Create Stackの各アクションで返されます）。このアクションは、データ構造が存在する場合は true を、存在しない場合は false を返します。変数がデータ構造のインデックスを保持していないかどうかを確認するために、アクションでflag "not "を指定できることに注意してください。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。
