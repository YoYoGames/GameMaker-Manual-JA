リスト項目のインデックスを取得する

このアクションを使うと、与えられたリスト内の項目のインデックス位置を復元することができます。リストのインデックスを格納する変数 (リストを作成するアクションによって返されたもの) とインデックス位置を見つけたい値を指定します。 と、返された項目のインデックス位置を格納するターゲット変数 (script またはイベントが終了するまで使用する一時的なローカル変数としてフラグを立てることができます) を指定します。もし、探している値が がリスト内に存在しない場合、返すべきリストのインデックス位置は存在しないので、返り値は-1である。

アクションのシンタックス。

論拠となる。

論証 商品説明
List から削除するリストのインデックス(変数に格納)。
Value リストをチェックするための値
Target 返されたインデックスを格納するターゲット変数

例

上記のアクションブロックのコードは、インスタンス位置での衝突をチェックします。 もし見つかったら、そのインスタンスに固有の ID 値を一時変数に格納し、それがリストのデータ構造内に存在するかどうかをチェックします。もし存在すれば、そのアイテムはリストから削除され、インスタンスは破棄されます。
