ファイルの削除

以前に作成されたファイルを削除することができます。削除するファイルのファイル名（string ）を指定します。ゲームバンドルに含まれるファイルは削除できませんのでご注意ください。 Close IniまたはSave Bufferアクションを使用します（詳しくは「ファイルシステム」のセクションを参照してください）。

アクションのシンタックス。

論拠となる。

論証 商品説明
Filename 削除するファイルの名前（string ）。

例

上記のアクションブロックのコードは、ファイル "checkpoint.sav" が存在するかどうかを確認し、存在する場合は、次のようになります。 次に、"checkpoint_OLD.sav" が存在するかどうかを確認します。もしそのファイルが存在すれば、削除され、元の "checkpoint.sav" ファイルは "checkpoint_OLD.sav" に名前が変更されます。最後にバッファ が "checkpoint.sav" として保存されます (本質的には、保存するたびに保存されたbuffer ファイルをバックアップしているので、常に「現在の」保存ファイルと「古い」保存ファイルが存在することになります)。
