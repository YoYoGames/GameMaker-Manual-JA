ロードバッファー

このアクションでは、Save Bufferアクションで保存したbuffer を読み込むことができます。このアクションは、新しいbuffer を作成し、そこにデータファイルをロードし、buffer を識別するためのユニークな ID 値を返します。 変数に格納されます。ロードするファイルの名前（string ）と、返されたbuffer ID を格納する変数を指定します。flag を一時的なローカル変数として使用する場合は、この関数が作成するbuffer を必ず事前に削除する必要があることに注意してください。 script さもないと、その変数への参照を失い、その変数が破棄されたときにメモリリークが発生します。

アクションのシンタックス。

論拠となる。

論証 商品説明
Filename 開くファイルの名前（string ）。
Target ロード時に作成されるbuffer のユニークな ID 値を保持するためのターゲット変数

例

上記のアクションブロックのコードは、buffer データを格納する変数を作成し、次に buffer 保存ファイルが以前に作成されている。ファイルが存在すれば、作成した変数buffer に読み込まれますが、存在しなければ、作成、書き込みを行い、次回の読み込みに備えてセーブアウトされます。
