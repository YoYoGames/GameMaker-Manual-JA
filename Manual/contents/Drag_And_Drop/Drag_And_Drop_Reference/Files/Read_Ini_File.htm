Iniファイルの読み込み

Iniファイルからデータを読み込むアクションです。事前にアクションOpen Ini File を使って Ini ファイルを読み書きのために開いておく必要があり、次にこのアクションで、読み込むデータのタイプ (実数または "" で囲まれたstring ) と、データを取得するセクションヘッダとそのセクション内のキーを指定します。また，デフォルト値として実数(string)を指定することもできます． は、アクションで設定された「タイプ」と一致する。このデフォルトのアクションは、以下のいずれかが発生した場合にアクションから返されます。

  • 開いたIniファイルが存在しない
  • 参照するセクションヘッダーが存在しない
  • データを取得しようとしているキーが存在しない

上記のいずれの状況下でも、Iniファイルからの値ではなく、デフォルト値が返されます。どのような値が返されても、指定したターゲット変数に格納されます。 を一時的なローカル変数として使用します。

重要！「Iniファイルを開く」アクションで以前に開いていないIniファイルからデータを読み込もうとすると、深刻なエラーが発生し、ゲームがクラッシュします。

アクションのシンタックス。

論拠となる。

論証 商品説明
Type 読み込むデータの種類。 string (引用符 "" で囲む)、または real number
Section 読み込むIniファイルのセクション（string ）。
Key 読み込むセクション内のキー（string として )
Default ファイル、セクション、キーのいずれかが存在しない場合に返されるデフォルト値
Target アクションからの戻り値を保持するためのターゲット変数
