Iniファイルへの書き込み

Iniファイルに数値またはstring を書き込むアクションです。事前にアクションOpen Ini Fileを使用してIniファイルを読み書きのために開いておく必要があります。 実数または""で囲まれたstring ）、セクションヘッダ、セクション内のキー、キーの値などを書き込むことができます。Iniファイルの構造を視覚化するために、次の画像を参照してください。

ご覧の通り、Iniファイルはセクションに分割され、各セクションは独自のヘッダー名を持ちます。 その後、各セクションは様々なキーと値のペアに分割されます（Mapデータ構造のようなものです）。Iniファイルは、テキスト処理ソフト（メモ帳など）を使って自分で作ることもできます。 Windows としてインクルードし、ゲーム内で読み書きができるようにします。

重要!Iniファイルを開く］アクションを使用して、以前に開いていないIniファイルからデータを読み込もうとすると、深刻なエラーが発生し、ゲームがクラッシュします。

重要!これはメモリにデータを書き込むだけで、最終的にディスク上のファイルにデータを書き込むには、 Close Ini Fileを呼び出す必要があります。

論拠となる。

論証 商品説明
Type 書き込むデータの種類。 string (引用符 "" で囲む)、または real number
Section 書き込むIniファイルのセクション（string ）。
Key 書き込むセクション内のキー（string ）。
Value 書き込む値。"type "引数（上記）に基づき、string または実数で指定する。
