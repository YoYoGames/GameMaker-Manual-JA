Iniファイルを閉じる

このアクションは、現在開いている Ini ファイルを閉じます。このアクションは、開いている Ini ファイルの読み書きが終わった時点で呼び出す必要があります（Ini ファイルは、アクションOpen Ini File を使って開かれます）。Write To Ini Fileアクションを使用した後にこのアクションを使用しない場合、このアクションが呼ばれるまでファイル情報がメモリに保持されるため、ディスクには何も書き込まれませんが、その時点でメモリから情報が取り出されます。 を作成し、ディスク上のファイルに書き込む。また、以前に別のファイル（または同じファイル）を閉じていない状態でIniファイルを開こうとすると、エラーが発生することに注意してください。
