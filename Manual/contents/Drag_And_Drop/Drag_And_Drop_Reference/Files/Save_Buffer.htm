バッファの保存

このアクションでは、buffer からのデータを含むファイルを保存することができます。一意のbuffer ID (Create Buffer でbuffer を作成したときに返されたもの) を与え、ファイルの名前 (string, extension を選択すると、buffer がこのファイルに書き出され、Load Buffer を使って再び読み込むことができます。

論証 商品説明
Buffer buffer 保存する一意のbuffer ID 値。
Filename 保存するファイルの名前（string ）。

例

上記のアクションブロックのコードは、buffer データを格納する変数を作成し、次に buffer 保存ファイルが以前に作成されている。ファイルが存在すれば、作成した変数buffer に読み込まれますが、存在しなければ、作成、書き込みを行い、次回の読み込みに備えてセーブアウトされます。
