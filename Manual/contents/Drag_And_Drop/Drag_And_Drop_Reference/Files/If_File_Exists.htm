ファイルが存在する場合

このアクションを使うと、指定された名前のファイルが存在するかどうかを確認することができます。チェックするファイルを選択すると（extension を含む）、そのファイルが存在すれば true を、存在しなければ false をこの関数が返します。ただし この場合、ファイルがIncluded Fileとしてゲームバンドルに存在する場合、またはゲームの実行中にファイルが保存された場合、 true 。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。

例

上記のアクションブロックのコードは、buffer のデータを格納する変数を作成し、その後に buffer 保存ファイルが以前に作成されている。ファイルが存在すれば、作成した変数buffer に読み込まれますが、存在しなければ、作成、書き込みを行い、次回の読み込みに備えてセーブアウトされます。
