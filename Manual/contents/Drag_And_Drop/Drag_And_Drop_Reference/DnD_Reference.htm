GMLビジュアルリファレンス

このセクションでは、GML Visualを使ってゲームをプログラムする際に利用可能なさ まざまなアクションをリストアップしています。object GML Visual Toolbox で利用可能なアクションライブラリは以下の通りです。各ライブラリ のセクションでは、個々のアクションについて、使用例とともに説明しています。

  • 共通アクション
  • インスタンスアクション
  • シーケンスアクション
  • マウスとキーボードの操作
  • ゲームパッドアクション
  • ムーブメントアクション
  • コリジョンアクション
  • ドローイングアクション
  • タイルアクション
  • オーディオアクション
  • ループアクション
  • スイッチング動作
  • データ構造アクション
  • バッファーアクション
  • ファイル操作
  • データ型アクション
  • ランダムアクション
  • カメラアクション
  • ルームアクション
  • パスアクション
  • タイムラインアクション
  • ゲームアクション
  • インスタンス変数アクション
  • その他のアクション
  • パーティクルアクション
  • タイムソースアクション
  • ロールバック・マルチプレイヤー・アクション

上記のアクションライブラリとは別に、「お気に入り」ライブラリもあります。これは、ツールボックスの特別なセクションで、アクションをドラッグして、最もよく使うものをすべて手元に置いておくことができます。
