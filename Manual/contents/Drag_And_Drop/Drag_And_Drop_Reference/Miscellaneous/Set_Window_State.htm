ウィンドウの状態を設定する

このアクションで、ゲームウィンドウの状態をフルスクリーンまたはウィンドウズに設定することができます。ここでは3つのオプションがあります。

  • トグル- このオプションを選択すると、アクションを呼び出すたびに、ウィンドウがフルスクリーンとウィンドウの間でトグルします。
  • Fullscreen- ウィンドウ表示状態からフルスクリーンモードにします。
  • Windowed- フルスクリーンの場合、ウィンドウの状態をウィンドウに戻します。

フルスクリーン状態でのゲームの見え方は、ターゲットプラットフォームのゲームオプション - グラフィックス設定に依存することに注意してください。 アスペクト比や解像度が異なると、引き伸ばされたり（ゲームオプションがフルスケールの場合）、黒いバーが追加されたり（ゲームオプションがアスペクト比を維持する設定の場合）することがあります。一般に、何も設定しない場合 異なる解像度に対応するために、コンパイルするターゲットプラットフォームには、アスペクト比を維持するを使用する必要があります。
