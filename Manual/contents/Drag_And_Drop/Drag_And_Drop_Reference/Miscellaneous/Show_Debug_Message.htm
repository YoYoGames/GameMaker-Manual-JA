デバッグメッセージの表示

このアクションを使用すると、デバッグメッセージをCompiler Outputに送信することができます。このメッセージには、string または実数を指定できますが、両方を使用する場合は、数値を文字列に変換する必要があります（Number To String アクションを参照してください）。このアクションを使用して複数のメッセージを出力するには、 アイコンをクリックしてアクションを展開し、さらに出力フィールドを追加します。
