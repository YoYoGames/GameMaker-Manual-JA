パスに沿った位置の設定

このアクションはpath に沿ってインスタンスの位置を設定します。 値は 0 - 1 で正規化されているので、例えば 0.5 に設定すると、インスタンスはpath のちょうど真ん中に移動します。 このアクションが効果を持つためには、Start Following Pathアクションを呼び出してインスタンスにpath を割り当てておく必要があることに注意してください。
