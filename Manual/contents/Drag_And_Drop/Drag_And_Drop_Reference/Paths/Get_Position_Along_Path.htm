経路上の位置の取得

このアクションはpath に沿ったインスタンスの位置を取得します。 返される値は 0 - 1 で正規化されるので、例えば 0.5 という結果が出た場合、インスタンスはpath のちょうど真ん中にあることになります。 このアクションが 0 以外の値を返すには、Start Following Pathアクションを呼んでインスタンスにpath を割り当てている必要がありますので注意してください。 返される値は与えられたターゲット変数に格納されますが、flag でその変数を一時ローカル変数にして値を割り当てることが可能です。
