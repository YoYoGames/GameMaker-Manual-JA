パーティクルスプライトの設定

このアクションは、パーティクルが描画時に使用できるsprite を設定するために使用できます。sprite を設定するタイプのユニークな ID 値を提供します（アクションCreate Particle Type でタイプを作成したときに返されます）。 と、使用するリソース（sprite ）を指定します。sprite にサブ画像がある場合、パーティクルは Image Editor で設定した速度でsprite のフレームをアニメーションします。また、パーティクルに色の値を設定すると（Set Particle Colourアクションを使用）、この色は使用するsprite にブレンドされます。インスタンスの画像ブレンドを設定するとインスタンスのsprite に色がブレンドされるのと同じです（ブレンドしない場合は白を使用します）。

注：このアクションを使用する場合は、Set Particle Shapeアクションは使用しないでください。sprite またはシェイプしか持つことができませんが、両方は持つことができません。
