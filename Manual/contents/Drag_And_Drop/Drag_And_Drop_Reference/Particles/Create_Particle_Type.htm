また、パーティクルに使用するブレンドモードを設定することができます。デフォルトでは "normal "ブレンドモードで描画され、パーティクルの下に描かれたものによって色が変更されることはありませんが、 "Blend" オプションを選択すると つまり、パーティクルの下に描かれたものに基づいて色を変えて描画されます。加算ブレンドは、非常に明るく「光る」エフェクトを提供します。 爆発、レーザー、火などに最適で、雨、煙、破片などのエフェクトには通常のブレンディングが適しています。下の画像は、加算ブレンドをオフ（左）とオン（右）で同じ効果を示しています。
