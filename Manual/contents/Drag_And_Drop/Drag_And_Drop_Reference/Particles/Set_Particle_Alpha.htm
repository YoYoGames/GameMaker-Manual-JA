パーティクルアルファの設定

このアクションは、パーティクルを作成したときのアルファと、パーティクルのライフタイム中のアルファを設定します。まず、パーティクル タイプの ID 値を指定し（Create Particle Typeアクションを使用してパーティクルを作成したときに返された値と同じである必要があります）、次に使用するアルファ値を設定できます。

使用するアルファ値を3つ選ぶことができ、パーティクルの寿命が尽きるまで、それらの間でブレンドされます。例えば、パーティクルのアルファ値が0.5、1、0、寿命が60フレームの場合、作成時（フレーム0）には アルファ0.5で、アルファ1に向かってフェードアウトし（フレーム30）、その後フェードアウトしてアルファ0になり（フレーム60）見えなくなる。
