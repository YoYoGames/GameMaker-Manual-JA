Do効果

このアクションは、ゲームroom で描画されているレイヤーの上または下に、ビルトインのパーティクルエフェクトを作成します。ドロップダウン・リストから作成するエフェクトの種類を選び（下図）、位置を設定します（相対位置でも可）。 インスタンスの位置）、エフェクトのサイズ（小、中、大）を選択します。また、使用する色（アルファ値は無視されます）を設定し、エフェクトをすべてのレイヤーの下に作成するかどうかを決定します。 room room 雨や雪の効果を使用する場合、その効果は画面全体に表示されるように設計されているため、位置とサイズの値は考慮されないことに注意してください。

アクションのシンタックス。

論拠となる。

論証 商品説明
Type 作成するパーティクルエフェクトの種類（上図参照）
Where パーティクルをroom レイヤーの上に作成するか下に作成するか
Size 効果の大きさ(小、中、大)
X 効果を出すためのroom 内の x 位置（ピクセル単位）。
Y 効果を出すためのroom 内の y 位置（ピクセル単位）。
Colour エフェクトの色（アルファ値は無視されます。）
