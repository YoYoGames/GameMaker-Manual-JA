パーティクルを放出する

エミッターからパーティクルを作成するためのアクションです。エミッタが属するパーティクルシステムID（Create Particle Systemアクションが返す）を与え、次に実際のエミッタID（CreateParticle Systemアクションが返す）を与えます。 最後に作成するパーティクルの種類を指定します（Create Part icle Typeアクションによる）。次に、"バースト "を選択します。 エミッターからパーティクルを "ストリーム "させるかどうか、また、作成するパーティクルの数を指定します。

パーティクルをストリームさせる場合、このアクションをステップイベントやドローイベントのようなゲームフレームイベントに配置する必要はありません。 このアクションを再度呼び出して停止させるまで（カウント値を0にするまで）、1フレームとします。パーティクルをバーストさせる場合は、任意のイベントでこのアクションを呼び出すことができ、パーティクルは一度に作成されます。

アクションのシンタックス。

論拠となる。

論証 商品説明

Event パーティクルのイベントタイプ。"Burst "または "Stream"
Count バーストまたはストリームするパーティクルの数
