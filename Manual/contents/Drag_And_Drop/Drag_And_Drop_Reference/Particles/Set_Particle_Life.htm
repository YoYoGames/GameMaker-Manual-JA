パーティクルライフを設定する

このアクションは、指定されたパーティクルタイプの寿命を設定します。まず、パーティクル タイプの ID 値（Create Particle Typeアクションを使用してパーティクルを作成したときに返された値）を指定し、その後、パーティクル タイプの寿命を設定します。 で、パーティクルの寿命の最小値と最大値をゲームフレーム単位で設定できます。例えば、ゲームのフレーム数が60の場合、ここで寿命の値を両方とも60に設定すると、パーティクルは画面上で1秒持続してから が消えます。異なる2つの値を使用した場合、生成される各パーティクルの寿命は、指定された最小値と最大値の範囲内でランダムに設定されます。すべての値は正の整数である必要があることに注意してください。

重要!非常に長い寿命を設定し、何百もの粒子を放出すると、性能の低いマシンやモバイルデバイスで問題が発生する可能性があります。パーティクルの寿命を設定する際は、視覚的な効果を最大化しつつ、パフォーマンスを最小化するように注意してください。 コストがかかるので、寿命の長いパーティクルは控えめに使用するようにしてください。
