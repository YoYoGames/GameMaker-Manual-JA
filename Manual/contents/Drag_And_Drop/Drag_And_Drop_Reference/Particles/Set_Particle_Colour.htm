パーティクルカラーを設定する

このアクションは、sprite の作成時とそのライフタイムで使用するカラーを設定します。まずパーティクル タイプの ID 値を指定し（Create Particle Typeアクションを使用してパーティクルを作成したときに返された値と同じである必要があります）、次に使用する色を設定します。

使用する色を3つ選ぶことができ、パーティクルの寿命が尽きるまで、sprite がそれらの色をブレンドします。例えば、パーティクルの色が赤、緑、黄で、寿命が60フレームだとすると、作成時（0フレーム目）には その後、寿命の半分（30フレーム目）で緑色になり、寿命の終わり（60フレーム目）で黄色になります。パーティクル形状ではなく、カスタムsprite を使用する場合、白以外の色はブレンドされます。 を使用した場合、sprite 。

注：色のアルファ成分は使用されず、カラーピッカーからのRGB成分のみが使用されます。アルファ値を設定するには、Set Particle Alphaアクションを使用します。
