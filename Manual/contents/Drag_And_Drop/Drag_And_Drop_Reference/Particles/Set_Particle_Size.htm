設定された粒子径

このアクションは、粒子径を設定するために使用します。粒子径は、静的または時間と共に増加/減少するように設定できます。サイズを設定するタイプのユニークなID値（Create Particle Typeアクションでタイプを作成したときに返されます）を与えます。 そして、初期サイズの範囲を指定し、次に増加量、最後に "wiggle "を指定します。パーティクルのサイズはスカラー値で、1はベースとなるパーティクル画像と同じサイズ、0.5はその半分のサイズになります。 の場合、2倍となります。最小値と最大値が同じでない場合、作成されるパーティクルの初期サイズはその間のランダムな値になります。

パーティクルサイズを大きくするオプションでは、パーティクルが生きている間、各ゲームフレームごとに基本スカラー値に追加される値を入力することができます。つまり、この値を例えば0.1に設定すると、各ゲームフレームでスケール値に0.1が加算されます。 なお、ここで負の値を使うと、パーティクル画像もスケールダウンします。なお、スケール減少の最小値は-0.01、増加の最小値は0.01です。

最後に "wiggle "オプションを設定することができます。この値は0から指定した値の間でランダムに生成され、パーティクルの寿命の各ステップごとにサイズに加算または減算されるため、描画される画像は変動します。 の大きさです。明らかに小さい値より大きい値の方が顕著で、この値はマイナスにもでき、最大範囲は-20から20の間である。
