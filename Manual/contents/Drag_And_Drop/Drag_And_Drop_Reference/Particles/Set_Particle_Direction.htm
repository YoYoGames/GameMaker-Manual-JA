パーティクルの方向を設定する

このアクションは、パーティクルの方向を設定するために使用され、静的または時間と共に増加/減少するように設定できます。サイズを設定するタイプのユニークなID値を与えます（Create Particle Typeアクションでタイプを作成したときに返されます）。 を指定し、最初に開始方向を指定し、次に増加量を指定し、最後に "wiggle "を指定します。パーティクルの生成方向は、0が右、90が上、180が左、270が下となるように計算されるので、最小値として を-45、最大を45に設定すると、例えば右方向にしか動かないパーティクルを放出します。

パーティクルの方向を増加させるオプションは、パーティクルが生きている間、各ゲームフレームで基本方向値に追加される値を入力することができます。例えばこの値を0.1に設定すると、各ゲームで0.1ずつ方向が追加されます。 フレームになります。正の値は反時計回りに、負の値は時計回りにパーティクルを螺旋状に動かすことができます。なお、方向の減少の最小値は-0.01、増加の最小値は0.01です。

最後に "wiggle "オプションを設定することができます。この値は0から指定した値の間でランダムに生成され、パーティクルの寿命の各ステップごとに方向に加算または減算されるため、描画された画像は変動することになります の方向を指定します。明らかに小さい値より大きい値の方が顕著で、この値はマイナスにもでき、最大範囲は-20から20です。

Type の方向を設定するパーティクルタイプのユニークなID値です。
Min Dir パーティクルの種類に応じた最小方向（-360～+360、デフォルト0）
Max Dir パーティクルの種類に応じた最大方向（-360～+360、デフォルト360）
Increase ゲームフレームごとにパーティクルの種類方向に追加される値（負の値も可能、最小値は+/-0.01）
Wiggle 粒子方向に1ステップごとにランダムに加算・減算する最大値（±20）。
