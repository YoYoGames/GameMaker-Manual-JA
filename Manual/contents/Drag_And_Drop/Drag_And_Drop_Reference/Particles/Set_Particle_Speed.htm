粒子速度の設定

このアクションは、パーティクルの移動速度を設定するために使用します。サイズを設定するタイプのユニークなID値（Create Particle Typeアクションでタイプを作成したときに返される）を与え、次に初速の範囲、増加量、最後に "wiggle "を指定します。パーティクルの生成速度はピクセル/ゲームフレームで計算されるので、最小値を2、最大値を5に設定すると、2～5ピクセル/ゲームフレームの間で移動するパーティクルが生成されます。

パーティクルの速度を上げるオプションでは、パーティクルが生きている間、各ゲームフレームごとに基本速度値に追加される値を入力することができます。例えばこの値を0.1に設定すると、各ゲームフレームで0.1ずつ速度が追加されます。負の値も入力するとパーティクルの速度が遅くなりますが、速度が0以下になることはありません。

最後に "wiggle "オプションを設定することができます。この値は 0 から指定した値の間でランダムに生成され、粒子の寿命の各ステップごとに速度に加算または減算されるため、描画される画像の速度が変動することになります。明らかに大きな値は小さな値よりも顕著で、この値はマイナスにもなり、最大範囲は-20から20の間です。

アクションのシンタックス。

論拠となる。

論証 商品説明
Type サイズを設定するパーティクルタイプのユニークなID値
Min Speed パーティクルの種類に応じた最小速度（デフォルト0.1）
Max Speed パーティクルの種類に応じた最大速度（デフォルト1）
Increase ゲームフレームごとにパーティクルタイプの速度に追加する値（負の値も可能、最小値は+/-0.01）
Wiggle 1ステップごとにランダムに粒子速度を加減する最大値（±20）。

例

上記のアクションブロックのコードでは、新しいパーティクル タイプを作成し、そのユニークな ID 値をグローバル変数に代入しています。その後、パーティクル タイプのすべてのプロパティを設定します。
