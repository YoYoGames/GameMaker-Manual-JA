セット ヘッドSequence

このアクションを使うと、sequence の指定されたroom 要素の再生ヘッド位置を、指定されたフレームに設定することができます。指定するのは、sequence 要素 ID（アクションCreate Sequence が返すもの）および 新しい再生位置（フレーム単位）。プレイヘッドに関する情報は、「ドープシートの使用」を参照してください。

位置は フレーム であり、総フレーム数より大きい（または 0 より小さい）値を設定すると、実際の最終プレイヘッド位置は、選択されているsequence 再生の種類によって異なることに注意してください。 を、以下のルールに沿って設定します。

  • ループしない。再生ヘッドは、loop の終わり（指定された値が負の場合は、始まり）にクランプします。
  • ループする。再生ヘッドは、sequence の始まりに余分なフレームを加えて折り返します。例えば、30 フレームのsequence で再生ヘッドを 45 フレームに設定すると、実際の再生ヘッド位置は 15 になります。もし を負の値で指定すると、sequence の再生ヘッドが逆方向にラップされます。
  • ピンポン・ルーピング再生ヘッドが最後まで進み、全フレーム長を超えるフレーム数分、再生方向を反転させて戻る。sequence が30フレームで、プレイヘッドを設定した場合。 を40に設定すると、最終的な再生ヘッド位置は20になり、方向は逆になります（0に下がる）。 sequence のピンポン再生に負の値を与えてはいけないことに注意してください。 の結果が得られます。

このようにヘッド位置を設定しても再生は停止せず、sequence 、一時停止しない限り新しいプレイヘッド位置から再生が継続されるだけです。
