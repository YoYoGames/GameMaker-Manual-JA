Get PositionSequence

このアクションでは、指定されたsequence 要素の現在の位置をroom から取得できます。sequence 要素の ID（アクションCreate Sequence が返すもの）を指定すると、このアクションは のXとYの位置がターゲット変数に格納される。ターゲット変数は一時的な（ローカル）変数としてフラグを立てることができ、アクションのために作成され、そのアクションがあるイベントの後続のすべてのアクションで使用できることに注意してください。

アクションのシンタックス。

論拠となる。

論証 商品説明
Variable 位置を取得するsequence 要素の ID。
Target X x 位置を保持するターゲット変数。
Target Y y 位置を保持するターゲット変数。
