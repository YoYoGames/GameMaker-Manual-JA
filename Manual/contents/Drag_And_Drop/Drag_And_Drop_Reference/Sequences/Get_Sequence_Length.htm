Get LengthSequence

このアクションでは、sequence の指定されたroom 要素の長さ (フレーム数) を取得できます。 (アクションCreate Sequence が返す)sequence 要素 ID を指定すると、このアクションが返します。 その長さをターゲット変数に格納する。これは、Sequence が実行されるフレーム数となる。

ターゲット変数には一時的（ローカル）なフラグを付けることができ、アクションのために作成され、そのアクションが属するイベントの後続のすべてのアクションで使用できることに注意してください。

アクションのシンタックス。

論拠となる。

論証 商品説明
Variable 長さを取得するsequence 要素の ID。
Target 長さを保持するためのターゲット変数。

例

上記のアクションブロックは、変数 "seq_id" に格納されているsequence の要素 ID の長さを取得する。そしてその値を使って、同じsequence 要素の先頭位置を、その終わりの 10 フレーム前に設定する。
