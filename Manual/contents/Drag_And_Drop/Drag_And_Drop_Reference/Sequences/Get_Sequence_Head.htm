Get HeadSequence

このアクションでは、sequence の指定されたroom エレメントの現在のプレイヘッド位置を取得できます。sequence エレメント ID (アクションCreate Sequence が返すもの) を指定します。 で、このアクションはシーケンスの再生ヘッドが乗っている現在のフレームを target 変数に返します。プレイヘッドについては、「ドープシートを使う」を参照してください。

ターゲット変数には一時的（ローカル）なフラグを付けることができ、アクションのために作成され、そのアクションが属するイベントの後続のすべてのアクションで使用できることに注意してください。
