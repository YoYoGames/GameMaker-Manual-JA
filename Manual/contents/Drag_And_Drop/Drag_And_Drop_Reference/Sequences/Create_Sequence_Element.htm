シーケンスの作成

このアクションを使用すると、sequence ブラウザから任意のsequence のAsset 要素を作成することができます。sequence の名前と、作成されるroom 内の x/y 位置、作成されるレイヤー（このレイヤーはアセットレイヤーである必要があります）を指定します。相対"flags にチェックを入れると、アクションコードブロックを実行しているインスタンスに対して、sequence を相対的に配置することができます。この方法でゲーム内にsequence を作成すると、一意の要素 ID値が与えられ、その後のアクションでsequence を識別できます。したがって、このアクションでは、この ID 値を保持する「ターゲット」変数を定義できます（以下の例に示すとおり）。targetは一時的な（ローカル）変数としてフラグを立てることができ、アクションのために作成され、アクションがあるイベントの後続のすべてのアクションで使用することができることに注意してください。
