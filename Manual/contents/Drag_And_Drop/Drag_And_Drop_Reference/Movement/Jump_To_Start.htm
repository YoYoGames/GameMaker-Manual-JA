Jump To Start

このアクションは、インスタンスを現在の位置から最初に作成された位置に移動させるために使用されます。インスタンスが作成された元の位置は、 xstart と ystart という2つのインスタンス変数に格納されています。これらの変数は他のアクションで使用することができ、また記録された開始位置を変更するために設定することができます。

アクションのシンタックス。

例
