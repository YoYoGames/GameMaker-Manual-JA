セットポイント方向

このアクションは、インスタンスの現在位置と与えられた位置で形成されるベクトルから方向を設定するために使用されます。x と y の位置を指定して、インスタンスの位置からの相対的なフラグを立てることができます。 インスタンスからその点までの方向がインスタンスに使用される。

アクションのシンタックス。

論拠となる。

論証 商品説明
x 方向ベクトルのx成分。
y 方向ベクトルの y 成分。

例

上記のアクションブロックのコードでは、オブジェクトのインスタンスがあるかどうかをチェックします。 "obj_Player" がroom に存在する場合、そのインスタンスの方向が "obj_Player" の x/y 位置を指すように設定します。
