設定方向 ランダム

このアクションは、インスタンスの移動方向を8つの基点から選ばれた1つまたは複数のランダムな方向に設定するために使用されます。アクションの中にある8つの方向矢印を1つ以上クリックすると、その中からランダムに1つを選びます。 をインスタンスの方向として使用します。どの矢印も選択しない場合、アクションはインスタンスの現在の方向に影響を与えません。

アクションのシンタックス。

例

上記のアクションブロックのコードは、 true または false のどちらかを選択し、その結果を一時的な（ローカル）変数に格納します。次にこの変数がチェックされ、 true であればランダムな方向の1セットが選択され、 false であれば別のセットが選択される。
