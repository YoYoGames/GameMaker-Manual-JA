設定速度

このアクションは、インスタンスの移動速度を設定するために使用されます。速度は、インスタンスが1つのゲームフレーム（ステップ）内で移動しなければならないピクセル数として計算されます。デフォルトでは、スピードの値は方向ベクトルに追加されます。 speedを2に、方向を45°に設定すると、インスタンスはゲームフレームごとに2ピクセルずつ上下に移動します。しかし、 direction フィールドをクリックして、Horizontalか Verticalのいずれかの速度を選択することもできます。

どちらかを選択すると、水平または垂直方向のベクトルに対してのみ速度が設定されます。注 flag また、負の数を指定すると、現在の速度とは逆方向に移動します。 方向値です。
