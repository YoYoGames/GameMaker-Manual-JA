セットフリクション

このアクションは、インスタンスの「摩擦」を設定するために使用されます。摩擦とは簡単に言うと、ゲームフレームごとに速度ベクトルから特定の量が差し引かれることを意味し、差し引く量を指定することができます。通常、以下のような小さな値が最適です。 0.2ですが、実際の量はプロジェクトの設定によります。相対的なflag を設定した場合、与えられた量は設定する代わりに現在の摩擦の値に追加されます。

アクションのシンタックス。

論拠となる。

論証 商品説明
friction 適用する摩擦（正の値のみ）

例

上記のアクションブロックのコードでは、動きに関連する様々なインスタンスプロパティを設定しています。 重力方向は下（270°）に、重力は0.05に、摩擦は0.2に設定されます。
