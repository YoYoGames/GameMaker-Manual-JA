セット方向固定

移動方向を固定するためのアクションです。設定可能な方向は、アクションブロック内のボタンで示され、 どれかを クリックするとその方向が設定されます。なお、GameMakerの方向は は度数で表され、右が0度、上が90度、左が180度、下が270度となっています。
