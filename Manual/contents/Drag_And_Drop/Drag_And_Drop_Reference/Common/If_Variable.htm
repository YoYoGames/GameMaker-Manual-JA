If Variable

このアクションは、特定の式を使用して、与えられた変数の状態を与えられた値に対してチェックするために使用されます。if" 文は常に true または false のいずれかを返し、チェックするための式は以下のいずれかになります。 を以下に示します。

  • Equals to- 変数と値の両方が正確に等しい。
  • 未満- 変数が値より小さいこと
  • Greater than- 変数が値より大きい。
  • Less than or Equal to- 変数が値より小さいか等しい。
  • 大なり等しい- 変数が値より大きいか等しい。

引数「Not」をflag 、上記は否定式となり、例えば「equals to」は「notequals to」となる。また、variableやvalueの引数は数値に限定されないことに注意してください。 で、string 、ポインター、リソースID、その他関数が返したり使ったりできるものなら何でも構いません（詳しくはデータ型のセクションを参照してください）が、2つの 例えば、変数がstring を保持しており、それが数値と等しいかどうかをチェックしようとすると、シンタックスエラーが発生します。

重要な注意：浮動小数点数の精度の問題により、2つの値が正確に等しいかどうかを確認すると、一方が正確に1である可能性がある一方で、 false を返す場合があります。 もう一方は1.00000000000001である可能性があります。これは、チェックする前にDecimal to Integerアクションを使用するか、"greater than" または "less than" 式を使用することにより回避できます。
