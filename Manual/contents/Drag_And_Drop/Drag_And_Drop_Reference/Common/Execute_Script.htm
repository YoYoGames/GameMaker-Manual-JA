スクリプトの実行

このアクションは、script 関数またはユーザー定義関数を呼び出すために使用できます。一般的に、このアクションはレガシーサポートにのみ使用され、現在のプロジェクトや新しいプロジェクトではあまり使用されるべきではありません。このアクションを使うには アクションを呼び出すには、呼び出すscript asset 、または呼び出すユーザー定義関数の変数名を指定し、それが必要とする引数、つまりスクリプトに入力したい値（strings 、整数、リソースインデックス、など）を指定します。 または関数が使用できるようにします。必要に応じて引数を追加するには、プラスアイコン アクションの横にある

scripts とユーザー定義関数の中には、後でアクションブロックリストで使用するために値を返すものもあるので、この戻り値を格納するターゲット変数を指定することができます。この変数は、以前に作成されたものであっても、新しい一時的なものであっても構いません。 ローカルなものです（「Temp」チェックボックスをオンにした場合）。targetとargumentの欄は、必要なければ空白にしておくことができます。

一般に、ユーザー定義関数を呼び出すには、Function Callアクションを使用した方が良いことに注意してください。このアクションを使用したり、コード内でscript asset を直接呼び出す必要は一般的にはないはずです。
