新構造

このアクションは、 コンストラクタ関数としてフラグが立てられた関数を使用してstructs を作成する場合にのみ使用されます (詳細については、「新しい関数の宣言」を参照してください)。このアクションは、以前に定義した関数のメソッドと、いくつかの引数を受け取ります。必要であれば、左側のアイコン を使って、さらに引数を追加することができます。これらの引数は、関数が必要とする入力に対応する必要があり、作成されるstruct に入力するために使用されます。struct はターゲット変数に返されます。ターゲット変数は、一時的なローカル変数としてフラグを立てることもできますし、立てないこともできます。

Argument0 ... Argument3 script または関数に渡すさまざまな引数（値）（未使用の引数は空白にしておくことができます。）
Target 戻り値の対象となる変数名（空白可）
