未定義の場合

このアクションは、変数や式が "undefined "の値を返すかどうかをチェックするために使用します。undefinedの場合は、このアクションは true を返し、そうでない場合は、 false を返します。 また、undefined の値は、プロジェクトのクラッシュや重大なエラーを引き起こす可能性があるので、これを使用してそれをチェックし、発生する前にキャッチすることができます。もし、flag "Not "引数を指定した場合は、値が は "undefined"ではない。

これらのアクションは、"if" の評価値が true であった場合に実行されます。 他の場所にドロップされたアクションは、"if "ブロックの後に実行されます。なお、undefined のチェックを同じアクションに「重ねる」ためには、プラスアイコンの の横にある を指定し、別の変数を指定してチェックします。これにより、1回のチェックで与えられたすべての変数がundefined であるかどうかをチェックし、そうである場合にのみ追加されたアクションコードが実行されます。
