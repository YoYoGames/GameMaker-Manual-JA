グローバル変数の設定

このアクションは、グローバル変数に値を割り当てる（その過程で値を作成する）か、既存のグローバル変数の値を変更するために使用されます。グローバル変数とは、特定のインスタンスに「属する」変数ではなく、全体の そのため、どのインスタンス、script またはコードからもアクセスおよび変更することができます。グローバル変数に関するより詳細な情報は、こちらを参照してください。

グローバル変数の名前を与え、次に値を与えます。必要であれば、元の値から相対的に値を追加することができます。新しい変数名（ゲーム全体で一度も代入されたことのない変数）を指定すると は新しいグローバル変数を作成し、指定した値を代入します。一方、既存のグローバル変数の名前を指定した場合は、単に値を新しいものに変更します。Relativeをチェックすると、その値が既存の また、以前に作成した式や変数を値として使用することもできます。

変数にはstring やポインタ、リソースIDなど、関数が返したり使ったりできるものなら何でも使えるからです。string の場合、相対チェックをすると、その変数がそれまで持っていた値と連結されます。したがって、変数が「Hello」であった場合、相対チェックで「World」を代入すると、最終的な変数の値は「Hello World」となります。注意しなければならないのは ただし、変数の値が両方とも同じデータ型であることを確認してください。例えば、整数とstring を足そうとすると、コンパイラーエラーが発生します。

また、同じアクションに変数を追加するには、プラスアイコン アクションの横にある別のグローバル変数 という名前と値を設定します。変数に関するより詳細な情報は、「変数と変数のスコープ」のセクションを参照してください。

Relative チェックすると、前の値との相対値が設定されます。
