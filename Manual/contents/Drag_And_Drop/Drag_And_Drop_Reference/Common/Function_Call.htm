機能呼び出し

このアクションは、任意のGML関数や ユーザー定義関数を呼び出すために使用します。呼び出す関数を指定し、その関数に追加で つまり、関数に入力したい値（strings 、整数、リソースインデックスなど）を入力し、関数で使用できるようにします。引数を追加するには、プラスアイコン の横にある また、引数の数は使用する関数の数と一致させる必要があります（例：関数が3つの引数を必要とする場合、3つの引数を与える必要があります）。関数によっては、後でアクションブロック この変数は、以前に作成されたものでも、新しい一時的なローカル変数でもかまいません (チェックボックス "Temp" をオンにした場合)。ターゲット と引数のフィールドは、必要なければ空白にしておくことができます。

アクションのシンタックス。

論拠となる。

Argument0 ... ArgumentX 関数に渡されるさまざまな引数（値）。
Target 戻り値の対象となる変数名（空白可）。
