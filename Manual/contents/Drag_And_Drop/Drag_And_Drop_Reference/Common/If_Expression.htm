If Expression

このアクションは、式が true または false のいずれかであることを確認するために使用します。ここで確認する論理式は、それがどちらかの値として評価される限り、どのような論理式でも構いません。例えば、以下の式のいずれかを使用することができます。

x < (room_width / 2)
a == b
++val >= 100

引数「Not」をflag にすると、例えば "==" が "not=" になるなど、上記が否定された式になります。また、式は数値に限らず、strings 、ポインター、リソースIDなど、関数が返したり使ったりできるものをチェックできます（詳細はデータ型のセクションを参照）。ただし、エラーを出さずに2つの異なるデータ型を比較することはできません。たとえば、式の中でstring と整数を使用しようとすると、エラーが発生することに注意しましょう。式の詳細については、ここを参照してください。

重要な注意：浮動小数点数の精度の問題により、2つの値が正確に等しいかどうかを確認すると、一方が正確に1である可能性がある一方で、 false を返す場合があります。 もう一方は1.00000000000001である可能性があります。これは、チェックする前にDecimal to Integerアクションを使用するか、"greater than (>)" または "less than (<)" 式を使用することによって避けることができます。
