グローバル変数の取得

このアクションは、グローバル変数の値を取得し、別のインスタンスまたは一時的な（ローカル）変数に割り当てるために使用されます。グローバル変数の値を代入する変数には、以前に作成されたものを使用することができます。 をチェックし、新しい変数名を指定すると、返された値を保持するための新しい一時的な（ローカル）変数が作成されます。

同じアクションの中に、さらに変数を追加するには、プラスアイコン アクションの横で、別のグローバル 変数名と一時的なローカル変数名です。変数に関するより詳細な情報は、「変数と変数のスコープ」のセクションを参照してください。

Target 返り値を格納するターゲット変数。"Temp "がチェックされていない場合は、以前に作成されているはずである（下記参照）。
Temp 戻り値を格納する一時的な（ローカル）変数を作成する（デフォルトはオフ）
