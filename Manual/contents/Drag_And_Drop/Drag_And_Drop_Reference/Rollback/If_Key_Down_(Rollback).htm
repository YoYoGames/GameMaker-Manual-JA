If キーダウン（ロールバック）

このアクションは、与えられた入力が現在のプレーヤーのために保持されているかどうかをチェックします。

右側には、与えられた入力が見つかったときだけ実行されるアクションを付けることができます。

アクションのシンタックス。

論拠となる。

論証 商品説明
Input チェックする入力の名前
Not 入力が保持されていないかどうかを確認するために有効にする

例

上記の動作は、入力に応じてプレーヤーを左右に移動させます。
