入力の取得（ロールバック）

このアクションは、プレーヤの入力値を含むstruct を返します。オプションの引数には、入力値が返されるプレーヤーのIDを指定します。

マルチプレイでの入力チェックには、If Key Down (Rollback),If Key Pressed (Rollback),If Key Up (Rollback) を使用することをお勧めします。

Define Player (Rollback)で作成したプレイヤーインスタンスで使用する場合、プレイヤーIDを指定する必要がないため、player_idを指定すればよい。

それ以外のインスタンス、または手動で作成したプレイヤーインスタンスで使用する場合は、プレイヤーIDを指定する必要があります。

どのような入力が返され、どのように変更できるかは、「入力の定義」を参照してください。
