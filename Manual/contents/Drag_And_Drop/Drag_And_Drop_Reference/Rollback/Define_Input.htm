入力の定義（ロールバック）

この関数は、すべてのプレーヤーのカスタム入力を定義するために使用されます。InputとValueのペアを受け取り、Inputは入力の名前、Valueはそれに割り当てられた入力定数です。

カスタム入力を割り当てる方法については、「カスタムコントロール」を参照してください。

使用する場合、この関数はjoin/createアクションの前に実行する必要があります。

アクションのシンタックス。

論拠となる。

論証 商品説明
Input 定義する入力の名前
Value 割り当てる入力定数（「カスタムコントロール」参照）
