マニュアルスタート（ロールバック）を使用する

デフォルトでは、マルチプレイヤーゲームは、すべてのプレーヤーが参加した時点で自動的に開始されます。

このアクションを呼び出すと、自動起動の動作が無効になり、ゲームは手動で起動するのを待ちます。

使用する場合、このアクションはjoin/createアクションの前に実行する必要があります。
