ロールバックゲームを開始

このアクションは、すべてのプレイヤーが参加する前に、現在のマルチプレイヤーゲームを強制的に開始させます。まず、Create Rollback Gameを使用してマルチプレイヤーゲームを作成する必要があります。

通常、システムはすべてのプレイヤーが参加するのを待ってから、自動的にゲームを開始します。このアクションは、すべてのプレーヤーが参加する前に、ゲームを早く開始するために使用することができます。

Use Manual Startを呼び出すと、すべてのプレイヤーが参加した後でも、Rollbackにこの呼び出しを待たせることができます。

このアクションを実行すると、Rollback Startイベントが発生します。
