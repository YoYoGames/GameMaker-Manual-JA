ロールバック・マルチプレイヤー・アクション

ロールバックアクションは、GameMakerのロールバックシステムを使ってマルチプレイヤーゲームを作る際に使用します。

紹介はこちら。ロールバック・マルチプレイヤー

ロールバックシステムで使用される変数のリストについては、こちらを参照してください。ロールバック変数

ロールバックマルチプレイでは、以下のアクションが用意されています。

  • クリエイトゲームズ
  • ゲームに参加する
  • ゲーム開始
  • 放置ゲーム
  • 入力の定義
  • 入力フレーム遅延の定義
  • モック入力の定義
  • プレイヤーの定義
  • イベント表示
  • 入手方法
  • Get Input
  • If キーダウン（ロールバック）
  • キーが押された場合（ロールバック）
  • イフ・キー・アップ（ロールバック）
  • ロールバックイベントが発生した場合
  • シンクオンフレーム
  • ランダム入力の使用
  • マニュアルスタート使用
  • メッセージの送信
