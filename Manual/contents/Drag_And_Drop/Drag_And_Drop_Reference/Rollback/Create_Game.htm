ロールバックゲームの作成

このアクションは、新しいマルチプレイヤーゲームをホストします。ゲームを開始するために必要な接続人数を入力し、オプションで同期テストを無効にすることができます。

Sync Testを無効にして呼び出すと、指定された人数のプレイヤーが参加するまで待機してからゲームを開始します。ゲームが始まるまで、マネージドインスタンスを作成または変更しないようにしてください。

ヒント Start Gameを実行することで、すべてのプレイヤーが参加する前にゲームを開始させることができます（オプション）。

現在、1つのゲームに参加できるのは4人までです。

警告このアクションでデフォルトのリージョンを設定することは、プレーヤーがメニューからリージョンを選択できるようにしている場合を除き、推奨されません。

Count of players ゲームを開始するために接続しなければならないプレイヤーの数です。Define Playerを使用した場合、自動的に作成されるインスタンスの数です。
Sync test Sync Testを有効にするかどうかを指定し、デフォルトで有効になっています。無効にすると、オンライン機能が有効になります。
Region 仮想room を作成するリージョンを指定します。リージョンの取得方法については rollback_create_game().
