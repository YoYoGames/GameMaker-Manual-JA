モック入力の定義（ロールバック）

このアクションはSync Test中に使用され、指定されたプレーヤーに一時的な「モック」入力を割り当てることができます。これは、プレーヤーに定義されたコントロールとSync Test からのランダム入力をオーバーライドします。

アクションのシンタックス。

論拠となる。

論証 商品説明
Player ID このモック入力セットを取得するプレイヤー番号
Input 定義する入力の名前
Value 割り当てる入力定数（「カスタムコントロール」参照）

例

上記のアクションは、プレイヤー1のモック入力fire,left,right,up,downを定義しています。
