ロールバックゲームに参加する

このアクションは、ゲームに参加しようとするものです。参加するには、バーチャルroom の ID を含む URL にアクセスする必要があります。

room が参加した場合、このアクションに付属するすべてのアクションが実行されます。

自分がすでに参加しているゲームに新しいプレイヤーが参加すると、「ロールバックイベント」イベントが発生します。詳しくは、「ロールバックイベント」を参照してください。
