メッセージの送信（ロールバック）

このアクションは、他のプレイヤーにstring メッセージを送信するために使用されます。

送信するstring 、最大128バイトのサイズを指定します。

第2引数を指定しない場合、送信者を含む全プレイヤーに送信される。

第2引数には、メッセージを受け取るプレイヤーであるプレイヤーIDを指定することができます。

使い方は？

このアクションは「ローカル」に使用されることを意図しているため、ロールバック入力ではなく、ローカル入力に起因することになります。

最下部の使用例を参照してください。Rollback Eventイベントで受信したメッセージをリッスンします。

アクションのシンタックス。

論拠となる。

論証 商品説明
Message 送るべきメッセージとして、string 。
To OPTIONALメッセージを送信するプレイヤーのIDです。この引数を指定しないか、または-1を指定すると、すべてのプレーヤーにメッセージが送信されます。

例

ローカルでエンターキーが押されると、その値が全プレイヤーに送信されます。 keyboard_stringの値が全プレイヤーに送信されます。その後、そのstring はクリアされ、新しいメッセージを入力することができます。

これにより、（送信者を含む）すべてのクライアントがメッセージを受信すると、ロールバックイベントがトリガーされます。
