オーディオが再生されている場合

このアクションは、ゲーム内で任意のサウンドリソースのインスタンスが現在再生されているかどうかをチェックするために使用します。asset Explorerからチェックするサウンドリソースを指定すると、この関数は true if it is playing and falseを返します。 ただし、"not"flag をチェックすると、これは逆になり、サウンドが再生されていない場合は true を、そうでない場合は false を返します。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。

これらのアクションは、"if" が true と評価された場合に実行されます。 は、"if "ブロックの後に実行される。
