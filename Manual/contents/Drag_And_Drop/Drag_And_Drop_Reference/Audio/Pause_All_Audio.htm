すべての音声を一時停止する

このアクションは、現在再生中のすべてのサウンドを一時停止するために使用できますが、アクションを呼び出した後に再生されたサウンドは一時停止されません。これらのサウンドは、Resume All Audioアクションを使えば再び再生が始まります。

アクションのシンタックス。

例

上記のアクションブロックのコードは、グローバル変数をチェックし、それが true と評価されれば、すべてのサウンドを一時停止し、そうでなければ、すべて再開します。
