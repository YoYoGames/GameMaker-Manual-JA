マスターボリュームの設定

サウンドシステム全体の音量を設定するアクションです。設定できる音量は0（無音）から1（フル音量）の間で、スケールは直線的で、0.5の値は半分の音量となります。このアクションは、すべての room で現在再生されているすべてのサウンドのインスタンスに影響し、今後再生されるすべてのサウンドのインスタンスに影響します。また、このアクションは個々のサウンドのボリューム設定に依存しないので、あるサウンドを0.5ボリュームで再生するように設定している場合 で、マスターボリュームを0.5に設定すると、その音は0.25の音量で再生されます。

アクションのシンタックス。

論拠となる。

論証 商品説明
Volume すべてのサウンドの新しいマスターボリューム（0〜1、デフォルト1）

例

上記のアクションブロックのコードは、グローバル変数が true かどうかをチェックします。 false となります。 true であれば、サウンドシステムのマスターボリュームが取得され、一時的なローカル変数に格納されます。そして、その値が0.5でないかどうかをチェックし、0.5でなければ、サウンドシステムのマスターボリューム が0.5に設定される。グローバル変数の評価値が false であれば、マスターボリュームは 1 に設定される。
