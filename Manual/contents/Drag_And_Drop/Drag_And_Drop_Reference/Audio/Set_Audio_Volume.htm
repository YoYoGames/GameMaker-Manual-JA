オーディオの音量を設定する

このアクションでは、指定されたサウンドの音量を設定することができます。asset Explorerからサウンドを選択し、音量値を設定します。ボリュームは0（無音）から1（フルボリューム）の間で設定でき、スケールはリニアで、0.5なら半分のボリュームになります。この操作は、room で現在再生されているサウンドのすべてのインスタンスに影響し、今後再生されるサウンドのすべてのインスタンスに影響することに注意してください。

Time "フィールドは、音量の変更が有効になるまでの時間を制御します。値が0の場合は即座に変化し、値が大きい場合は指定されたミリ秒の間に徐々に音量が変化します。

アクションのシンタックス。

論拠となる。

論証 商品説明
Sound の音量を設定するサウンドリソースです。
Volume サウンドの新しい音量（0〜1、デフォルト1）
Time 音量をフェードさせる時間（ミリ秒単位）。

例

上記のアクションブロックのコードは、グローバル変数が true か false かをチェックする。 true であれば、指定されたサウンドの音量を取得し、一時的なローカル変数に格納する。そして、その値が0.5でないかどうかをチェックし、0.5でなければ、1000ミリ秒（1秒）後に音のボリュームが0.5になるように設定される。グローバル変数の評価値が false であれば、サウンドの音量は1に設定されます。ここではtimeの値が0になっているので、音量の変化はすぐに起こります。
