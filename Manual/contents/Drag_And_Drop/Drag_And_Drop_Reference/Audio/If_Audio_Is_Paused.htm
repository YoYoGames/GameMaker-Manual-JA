音声が一時停止した場合

このアクションは、任意のサウンドリソースのインスタンスがゲーム内で現在一時停止しているかどうかをチェックするために使用します。asset Explorerからチェックするサウンドリソースを指定すると、この関数は true if it is paused and falseを返します。 ただし、"not"flag をチェックすると、これは逆になり、サウンドが一時停止していない場合は true を、そうでない場合は false を返します。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。

これらのアクションは、"if" の評価値が true であった場合に実行されます。 他の場所でドロップされたアクションは、"if "ブロックの後で実行されます。
