オーディオボリュームを取得する

このアクションは、指定されたサウンドの音量を返します。asset Explorerからサウンドを選び、ボリューム値を返すターゲット変数を指定します（flag 、一時的なローカル変数とすることができます）。返される値は、0から 1 で、0 は無音、1 はフルボリュームである。

アクションのシンタックス。

論拠となる。

論証 商品説明
Sound の音量を取得するための音源です。
Target 返されたボリュームを格納するターゲット変数

例

上記のアクションブロックのコードでは、グローバル変数が true か false かをチェックしています。 true であれば、与えられたサウンドの音量を取得し、一時的なローカル変数に格納する。そして、その値が0.5でないかどうかをチェックし、0.5でなければ、サウンドの音量を0.5に設定する。もし グローバル変数が false と評価された場合、そのサウンドの音量は 1 に設定される。
