オーディオを再生する

このアクションは、指定されたサウンドを再生するように設定します。Asset Explorerからサウンドを選択し、サウンドをループさせるかどうかを "Loop"flag で設定します（ループしたサウンドは停止するまで何度でも再生されます）。

さらに、ゲイン、オフセット（再生を開始するトラックの位置）、再生するピッチを設定することができます。

再生されるサウンドのゲインとピッチの最終的な値は、サウンドに設定されたゲインとピッチの値の積になりますasset.オフセット値は、"Offset "フィールドに入力された値、またはフィールドを空にした場合は再生中のサウンドアセットのオフセット値です（例2参照）。

アクションのシンタックス。

論拠となる。

論証 商品説明
Sound サウンドasset 再生する
Loop loop を鳴らすかどうか（デフォルトはオフ）
Target 戻り値を格納するターゲット変数
Gain ゲインマルチプライヤーの値（デフォルトは1）。ここで設定されたゲインマルチプライヤーは、サウンドアセットのゲインマルチプライヤーと掛け合わされる
Offset 開始点を設定する時間（秒単位）（デフォルト0）。与えられた音の長さより長い値は無視されます。
このフィールドを空白にすると、サウンドアセットのオフセット（トラック位置）が使用されます。
Pitch ピッチマルチプライヤー（初期値1）。ここで設定されたゲインマルチプライヤーは、サウンドアセットのゲインマルチプライヤーと掛け合わされる

例1．

上記のアクションブロックのコードは、マウスボタンが押されたかどうかをチェックし、押されたことが検出されると、グローバル変数が true でないかをチェックします。 true でない場合は、サウンドが再生（ループ）され、グローバル変数が true に設定されます。そうでない場合は、サウンドが停止され、グローバル変数が false に設定されます。
サウンドはデフォルト設定で再生されます。ゲインおよびピッチ乗算器は1、オフセットは0です（サウンドのデフォルトオフセットasset ）。

例2．

上記のアクションブロックでは、まず、サウンドasset "snd_MusicBoss "のトラックポジションを20秒に設定します。次に、インスタンス変数 "skip_intro" に true が設定されているかどうかをチェックします。もし設定されていれば、先に設定したトラック位置の値である20秒から "snd_MusicBoss" の再生を始めます（これは "Offset" フィールドが空のままなので、アセットオフセットを使用しているためです）。最初の20秒の音声はスキップされます。
インスタンス変数 "skip_intro "を false に設定すると、"snd_MusicBoss "の再生も始まりますが、"Offset "が0に設定されているため、最初から再生が始まります。サウンドアセットに先ほど定義したトラック位置の20秒は無視されます。
