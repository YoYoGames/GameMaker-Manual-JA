キーダウンの場合

このアクションを使うと、キーボードをポーリングして、現在キーが押されているかどうかを確認することができます。このアクションは、キーボードのキーが押されているときは毎ステップ true を返し、そうでないときは false を返します。ただし、 not 修飾子をクリックすると、キーが押されていないかどうかを確認できます。つまり、キーを押していないときは true を返し、押しているときは false を返します。もし、1回だけキーが押されたことを確認したいのであれば、If Key Pressedアクションを使用してください。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。

例

上記のアクションブロックは、1ステップごとにキーボードの状態をポーリングし、キーがある場合は キーが押されているかどうかをチェックする。初回押下時のチェックが true であった場合、音を鳴らし、キーが押されている間、sprite のブレンドを赤に変更します。キーが押されていない場合、チェックが行われます。 キーを離すと、ブレンドカラーが白にリセットされます。
