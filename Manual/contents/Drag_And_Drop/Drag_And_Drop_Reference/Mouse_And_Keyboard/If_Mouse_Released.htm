マウスが発売された場合

このアクションは、マウスボタンがいつ離されたかをチェックする。マウスボタンが離された1つのゲームステップ（フレーム）で true 、それ以外では false を返します。 not モディファイアを有効にすると、このアクションは逆に、マウスボタンが離されていないかどうかをチェックし、離されているゲームステップ（フレーム）では false を返し、それ以外では true を返します。

なお、「if」ブロックにアクションを追加するには、下の画像のように、アクションの横にドロップする必要があります。

これらのアクションは、"if "の評価値が true である場合に実行され、他の場所にドロップされたアクションは、"if "ブロックの後に実行されます。

例

上記のアクションブロックのコードは、1ステップごとにマウスボタンの状態をポーリングし、マウスが押されている場合は、マウスが最初に押されたかをチェックします。チェックが true であれば、最初の押下時にインスタンス速度が設定され、その後、ボタンが押されている間、sprite ブレンディングが赤に変更されます。マウスが押されていない場合は、マウスを離したときにチェックが行われ、ブレンドの色が白にリセットされ、速度が0に設定されます。
