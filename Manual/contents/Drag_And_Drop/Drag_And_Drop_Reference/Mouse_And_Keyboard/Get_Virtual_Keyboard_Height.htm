仮想キーボードの高さを取得する

このアクションは、ディスプレイのサイズに基づき、仮想キーボードの現在の高さをピクセル単位で返します。キーボードが表示されていない場合は、0が返されます。

アクションのシンタックス。

例

上記のアクションブロックのコードは、OSの仮想キーボードが表示されているかどうかをチェックし、表示されていればキーボードの高さ（画面ピクセル単位）を取得し、その上にキーボード入力によるテキストを描画しています。
