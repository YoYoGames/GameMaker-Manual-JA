仮想キーボードを隠す

このアクションは、ゲームを実行しているデバイスの仮想キーボードを隠すために使用することができます。このアクションを呼び出すと、システム非同期イベントが発生し、 async_load  DS マップに以下のキー/値ペアが入力されます。

  • "event_type" - 仮想キーボード操作によってトリガーされた場合、これは "仮想キーボードの状態" になります。
  • "screen_height" - 仮想キーボードの高さ (ピクセル単位)です。キーボードが見えない場合は、0 になります。
  • "keyboard_status" - キーボードの現在のステータスで、以下の文字列のうちの1つとして返されます。
    • "隠蔽"
    • "非表示"
    • "表示"
    • "可視"

アクションのシンタックス。

例

上記のアクションブロックのコードは、マウスダウン操作をチェックし、マウスダウン操作があった場合 OSの仮想キーボードが表示されているかどうかをチェックします。表示されていない場合は、表示するように設定し（テンキーのタイプ）、すでに表示されている場合は非表示にします。
