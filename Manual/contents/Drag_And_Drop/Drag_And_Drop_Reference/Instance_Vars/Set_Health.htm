セットヘルス

このアクションでは、健康状態のインスタンス変数を作成または設定することができます。変数healthに設定する値、または変数healthに追加する値を指定します（relativeチェックボックスをチェックすると、その値は現在の "health "の値)。もし、Draw Instance Healthbarアクションを使用したい場合、この値は0から100の間でなければならないことに注意してください。また、以下の点に注意してください。 通常のインスタンス変数の場合、"health "の変数名は内部に保存されるため、スコア値を取得するためにはGet Healthアクションを使用する必要があります。

Health 健康状態を設定する量（relativeにチェックを入れた場合は健康状態に上乗せする量）
