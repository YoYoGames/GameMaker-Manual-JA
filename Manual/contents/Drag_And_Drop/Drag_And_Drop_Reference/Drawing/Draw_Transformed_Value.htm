変換後の値を描画する

このアクションでは、string キャプションおよび/または変数や値を画面に描画することができます。caption 引数は、値の前に追加したいstring で、必要なら空白にしておくことができます。また、値は、変数、固定値、または string - この場合、引用符で囲む必要があり、必要に応じて空白にすることもできます。次に、string が描画される位置を指定します。これは、絶対位置でも、インスタンスが描画する位置からの相対位置でもかまいません。 テキストを作成します。さらに、テキストのスケールを設定し、水平軸または垂直軸に引き伸ばすことができます。 右）。スケーリングは指定された倍率で行われ、0.5倍でテキストの幅が半分に、2倍で2倍になり、1がスケーリングなしの初期値です。テキストがどのように描画されるかは、以下の点に依存します。 font 、使用されている色、および整列。これらのことは、下の例に示すように、描画の前に変更することができます。

注意：このアクションは、各種描画イベントでのみ使用され、他の場所で使用しても何も描画されません。
