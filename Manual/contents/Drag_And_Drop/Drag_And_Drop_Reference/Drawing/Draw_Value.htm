ドローバリュー

このアクションでは、string キャプションおよび/または変数や値をスクリーンに描画することができます。caption 引数は、値の前に追加したいstring で、必要であれば空白のままでも構いませんが、テキストを含めたい場合には を引用符で囲みます。また、値は変数、固定値、または別のstring （この場合も引用符で囲む必要があります）でもかまいませんが、空白にすることはできません。次に、string が描画される位置を指定します（絶対位置でも可）。 またはテキストを描画しているインスタンスからの相対位置で指定します。テキストがどのように描画されるかは、使用するfont や色、配置に依存することに注意してください。これらはすべて、以下の例に示すように、描画の前に変更することができます。

注意：このアクションは、各種描画イベントでのみ使用され、他の場所で使用しても何も描画されません。

アクションのシンタックス。

x 描画する x 位置
