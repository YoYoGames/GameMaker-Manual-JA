ドローフォント取得

このアクションを使用すると、現在描画に割り当てられているfont の ID 値を取得することができます。返される値は、font が割り当てられていない場合は -1、またはfont ブラウザからAsset 定数を表す値 >= 0 のいずれかになります。戻り値は、以下のように保存されます。 は、指定したターゲット変数で、以前に作成されたものでも、新しい一時的なものでもよい（「Temp」チェックボックスをチェックした場合）。
