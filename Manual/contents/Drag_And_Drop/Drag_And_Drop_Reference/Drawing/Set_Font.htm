フォントを設定する

このアクションでは、その後のすべてのテキスト描画アクションで使用するfont を設定することができます。ダイアログをクリックすると、次のウィンドウが表示されますので、使用するfont を選択してください。

これは、その後のすべてのドローテキストの呼び出しに使用するfont を設定することに注意してください。 ゲーム開始時に一度だけ呼び出すと、描画イベントに限らず、すべての描画テキストアクションで、設定されたfont が使用されます。ただし、ゲーム中に複数のfonts が必要な場合は、描画の前にこのアクションを呼び出す必要があります。 のテキストは、必要なfont 。
