テキストアライメントを設定する

このアクションは、その後のすべてのDraw Textアクションのfont アライメントを設定します。水平方向の配置を左、中央、右に、垂直方向の配置を上、中、下に設定することができ、テキストはx/y の位置は、下の画像のように、描画アクションの位置です。

テキストが左側に水平に配置される
テキストは水平に中央に配置されます。
テキストが右側に水平に配置される
テキストは上部に縦書きで配置されます
テキストが中央に縦に並ぶ
テキストが下に縦に並ぶ

これは、その後のすべてのドローテキストの呼び出しに使用するアライメントを設定するので、例えば、ゲームの開始時に一度だけ呼び出すことができます - Drawイベントだけでなく、任意のイベント - すべてのドローテキストアクションは、設定されたアライメントを使用します。 ただし、複数のドローテキストアクションを異なる方法で配置する必要がある場合は、異なる配置にしたいドローテキストアクションの前にこのアクションを呼び出す必要があります。
