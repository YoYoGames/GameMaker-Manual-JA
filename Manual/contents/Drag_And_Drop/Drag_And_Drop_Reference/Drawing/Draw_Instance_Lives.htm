ドローインスタンスライブ

このアクションは、インスタンスが現在持っている各生命を表すために、sprite を描画します。描画する数は、ローカルなインスタンス変数 lives の現在の値から取得されます。 インスタンスを必要とするアクションを使用した場合。ライフに描画するsprite とスタックの順番（Row（水平、左から右）かColumn（垂直、上から下））、位置を指定します。 位置はroom 内の絶対位置か、描画を行うインスタンスの位置からの相対位置で、画像間の間隔はsprite の幅または高さに基づきます。 これは単に静止画像を描画することに注意してください。 - 与えられたsprite の最初の単一画像 (フレーム 0) は無視され、それ以降のフレームは、インスタンス変数 ( image_xscale や image_blend など) の変更によってインスタンスに追加されたトランスフォームも同様に無視されます。 キャプションテキストはライブ値の前に描画され、それがテキストである場合は "" でラップする必要があり、両方とも現在の描画font 、色、整列を使用してフォーマットされます。

注意：このアクションは、各種描画イベントでのみ使用され、他の場所で使用しても何も描画されません。
