Draw TransformedSprite

この関数は、Draw Spriteと同様にsprite を描画しますが、描画するsprite のスケール、ブレンディング、回転、フレームを変更するオプションを追加します。これらの値を変更しても また、関数内のすべての引数に直接値を指定する代わりに、利用可能な任意のスプライトプロパティ変数を使用することができます。下図は 値の違いによるスプライトの描画への影響

注意事項 カラー ブレンディングは、WebGL が有効な場合にのみ HTML5 ターゲットに推奨されます。ただし、WebGL が有効でない場合でも、ブレンディング カラーを設定すれば、通常どおりsprite をブレンドします。ただし、この方法によるすべてのブレンディングは、複製を作成します。 sprite これはcache に保存され、必要なときに使用されます。これは最適とは程遠く、複数の色変更を使用する場合は、WebGL を有効にしない限り、ゲームのパフォーマンスが低下します。 font cache のサイズを制限する必要がある場合は、関数 sprite_set_cache_size().

注意：このアクションは、各種描画イベントでのみ使用され、他の場所で使用しても何も描画されません。
