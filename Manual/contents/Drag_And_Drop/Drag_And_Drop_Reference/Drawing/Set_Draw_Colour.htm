描画色を設定する

このアクションでは、任意の図形（矩形/楕円/線）や、その後に描画されるテキストに使用する色を設定することができます。このアクションは、すべての描画アクションの前に呼び出す必要はなく、ゲーム開始時に一度だけ設定できることに注意してください。 例えば、すべてのテキストを黄色にする場合、ゲームの最初のroom の Create Event でこのアクションを呼び出すと、それ以降すべてのテキスト（とシェイプ）が黄色に描画されます。ただし、複数の色を使い分ける必要がある場合は、以下のようにします。 の場合、異なる色を必要とする各テキストやシェイプの前にこれを呼び出す必要があります。

このアクションの下部には、「アルファを使用する」のチェックボックスがあります。これにより、色のアルファ成分が描画アルファの設定にも使用されるようになります（描画アルファの設定の使用とほぼ同じです）。もし を無効にすることができます（デフォルトでは有効になっています）。
