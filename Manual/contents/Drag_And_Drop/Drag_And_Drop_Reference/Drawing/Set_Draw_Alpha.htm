ドローアルファの設定

このアクションでは、任意の図形（矩形/楕円/線）またはその後に描画されるテキストに使用するアルファを設定できます。このアクションはすべての描画アクションの前に呼び出す必要はなく、ゲーム開始時に一度だけ設定することができることに注意してください。 Alpha は、どの程度「透明」なものが描画されるかを制御する設定で、0 から 1 のスケールに基づいており、0 は完全に透明、1 は完全に不透明になります。
