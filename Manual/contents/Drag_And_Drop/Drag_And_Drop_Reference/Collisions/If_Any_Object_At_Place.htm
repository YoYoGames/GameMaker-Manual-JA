If Any At PlaceObject

このアクションは、呼び出したインスタンスのコリジョン マスクに基づく位置に、任意の object のインスタンスが配置されているかどうかを確認するために使用されます。位置は絶対位置でもインスタンスからの相対位置でもかまいません。 で、このアクションは衝突があれば true を、なければ false を返します。また、"Not"flag をチェックすると、指定した位置に衝突がないかどうかを確認できます。アクションは、衝突がない場合は true を、そうでない場合は false を返します。衝突は、呼び出し元のインスタンスのコリジョン マスクが、その位置にある任意のインスタンスのコリジョン マスクと重なるかどうかに基づいて計算されます（あたかもその位置に「配置」されているように）。

重要!衝突は、有効なコリジョン マスクを持つインスタンスにのみ登録されます。つまり、sprite が sprite_index に、またはsprite が mask_index に割り当てられているインスタンスです。 コリジョンマスクがない場合、そのインスタンスが何を描いているかに関係なく、コリジョンは検出されません。

なお、「if 」ブロックにアクションを追加する場合は、下の画像のように、アクションの横にドロップする必要があります。
