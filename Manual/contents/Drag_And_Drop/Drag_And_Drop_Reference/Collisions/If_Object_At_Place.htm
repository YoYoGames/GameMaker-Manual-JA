このアクションは、呼び出したインスタンスのコリジョンマスクに基づいて、現在のインスタンスがobject のある点に置かれている状態で、与えられたroom の1つ以上のインスタンスとの衝突があるかどうかをチェックし確認するために使用されます。このアクションは、衝突が検出された場合は true に評価され、そうでない場合は false に評価されます。チェックするobject と、チェックを実行するためにインスタンスを「配置」する位置（room の絶対位置でもインスタンスからの相対位置でも可）を指定します。 - また、Not flag をチェックして、指定した位置に衝突がないことを確認することもできます。この場合、アクションは衝突が見つからなかった場合のみ true と評価され、それ以外は false と評価されます。
