ゲームパッドボタンのしきい値を設定する

このアクションは、ゲームパッドのアナログボタンの「しきい値」を設定するために使用することができます。設定するゲームパッドのインデックスを指定し、0〜1の値を設定し、アナログボタンの入力量が指定した値より小さい場合、ゲームパッドのボタン これは、ゲームパッドに接続されたすべてのアナログボタンに影響するグローバルな 設定であることに注意してください。この値は、指定されたゲームパッドの全てのダウン、押下、リリースチェックで使用されますが、Get Gamepad Triggerアクションでは無視されます。
