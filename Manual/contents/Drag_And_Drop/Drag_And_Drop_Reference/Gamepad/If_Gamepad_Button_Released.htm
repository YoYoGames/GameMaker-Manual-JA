ゲームパッドボタンを離した場合

このアクションは、指定されたゲームパッド・インデックスのゲームパッド・ボタンがいつリリースされたかをチェックするものである。ボタンが離された1回のゲームティックで true を返し、それ以外の時は false を返します。 not モディファイアを有効にした場合。 この動作は逆で、ボタンが解放されていないかどうかをチェックし、解放されているゲームティックでは false を返し、それ以外では true を返します。

チェック対象のボタンがアナログボタンの場合、Set Gamepad Button Thresholdアクションで設定した閾値を下回るまで、リリースは検出されません。
