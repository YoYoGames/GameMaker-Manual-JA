ゲームパッドボタンが押された場合

このアクションを使うと、ゲームパッドにポーリングを行い、指定されたボタンが現在押されているかどうかを確認することができます。 not このアクションは、ボタンが押されているときは毎ステップ true を返し、そうでないときは false を返します。 つまり、ボタンが押されていないときは true を返し、押されているときは false を返します。もし、ボタンが1つ押されただけであれば、If Gamepad Button Pressedというアクションを使います。

チェック対象のボタンがアナログボタンの場合、Set Gamepad Button Thresholdアクションで設定した閾値を超えるまで、ボタンが押されているとはみなされません。
