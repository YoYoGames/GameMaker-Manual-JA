ゲームパッド軸のデッドゾーンを設定する

このアクションは、ゲームパッドのスティック軸の「デッドゾーン」を設定するために使用することができます。設定するゲームパッドのインデックスを指定し、0〜1の値を設定し、入力量が指定した値より小さい場合、ゲームパッドのスティック軸を不感帯と見なします。 これはグローバルな設定であり、接続されたスティックのすべての軸に影響することに注意してください。

ゲームパッドの軸を取得するアクションは、常に-1〜0、0〜1の値を返します。 例えば、デッドゾーンを0.2に設定すると、スティックを右に押したときに つまり、生の軸の値が0.2のとき、戻り値は0になり、生の値が0.5のとき、戻り値は0.375になり、生の値が0.9のとき、戻り値は0～1の値になります。 の値は0.875となる。
