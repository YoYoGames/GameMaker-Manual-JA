ゲームパッドボタンが押された場合

このアクションを使うと、ゲームパッドのボタンが現在押されているかどうかをポーリングすることができます。このアクションは、ゲームパッドのボタンが押されている（そして保持されている）場合は、毎ステップ true を返し、そうでない場合は false を返します。 not つまり、ボタンが押されていないときは true を返し、ボタンが押されているときは false を返します。もし、ボタンが1回押されたかどうかだけ確認したいのであれば ゲームパッドのボタンが押されたらアクションを使用します。

チェック対象のボタンがアナログボタンの場合、Set Gamepad Button Thresholdアクションで設定した閾値を超えるまで、押下は検出されません。
