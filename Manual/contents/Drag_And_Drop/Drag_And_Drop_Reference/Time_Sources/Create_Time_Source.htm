タイムソースの作成

このアクションは、新しいTime Sourceを作成し、Target変数に格納します。概要については、「Time Sources」を参照してください。

に基づいています。 time_source_create()関数に基づいています。引数の詳細については、そのページをお読みください。

タイムソースの作成後にタイムソースを有効にするには、Start Time Sourceを使用する必要があります。

タイムソースが不要になった場合は、「タイムソースの破棄」を使ってタイムソースを破棄する必要があります。

アクションのシンタックス。

論拠となる。

論証 商品説明
Target 新しいタイムソースが格納される変数
Parent 親となるタイムソース：ビルトインのタイムソースまたはカスタムのタイムソース
Period タイムソースの期間長、有効期限までの時間
Units 周期を表す単位（秒またはフレーム）
Callback を指定します。 methodまたはタイムソースの有効期限が切れたときに呼び出すスクリプト関数
Arguments OPTIONALコールバック関数に渡す引数を含む配列。
Repetitions OPTIONAL タイムソースを合計で何回実行するか。
Expiry Type OPTIONAL タイムソースの有効期限タイプ

例

このコードブロックは、次のような処理を行います。

  • Temp」オプションを有効にして、新しい関数を作成します。これは、ローカル・メソッド変数を作成します。
    • このメソッドは引数を1つ取り、msgを出力ログに表示します。
  • そして、1秒の周期を持つ新しいTime Sourceを作成する。
    • コールバックには変数callback_methodを、関数には引数1個の配列を指定している。
    • このTime Sourceは、Repetitionsに-1が指定されているように、無限に繰り返すように設定されている。
  • その後、タイムソースを起動します。

このタイムソースは、1秒ごとにコールバックを実行し、「A second has passed！」と出力ログに出力する。
