BPMから秒数へ

このアクションは、beats-per-minuteの値を取り、各ビートの長さを秒単位で返します。

これは、Time Sourceを作成するときに、Time SourceにBPM値を使用するために使用できます。このようなTime Sourceでは、単位に 秒を使用することが重要です。

アクションのシンタックス。

論拠となる。

論証 商品説明
Value ビート・パー・ミニッツの値
Target 返された秒/拍の値を格納するための変数
Temp 一時的な変数として作成するかどうか
