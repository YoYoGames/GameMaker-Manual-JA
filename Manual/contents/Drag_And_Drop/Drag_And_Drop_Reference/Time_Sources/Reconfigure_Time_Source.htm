タイムソースの再設定

このアクションでは、「タイムソース」フィールドで指定された既存のタイムソースの設定を変更することができます。

ここで提供される設定は、親を除いて、タイムソースの作成と同じです。

タイムソースがリセットされ、無効になるので、再度起動する必要があります。

アクションのシンタックス。

論拠となる。

論証 商品説明
Time Source は タイムソースIDを再設定する
Period タイムソースの期間長、有効期限までの時間
Units 周期を表す単位（秒またはフレーム）
Callback は methodは、タイムソースの有効期限が切れたときに呼び出す
Arguments OPTIONALコールバック関数に渡す引数を含む配列。
Repetitions OPTIONAL タイムソースを合計で何回実行するか。
Expiry Type OPTIONAL タイムソースの有効期限タイプ
