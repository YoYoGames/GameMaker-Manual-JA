タイムソース (GML Visual )

このページでは、タイムソースで利用できるGML ビジュアルアクションを一覧で紹介します。タイムソースの概要については、こちらをご覧ください。

GML Visual のタイムソースには、以下のアクションが用意されています。

  • タイムソースの作成
  • デストロイタイムソース
  • 開始時間 ソース
  • ストップタイムソース
  • ポーズタイム・ソース
  • レジュームタイムソース
  • タイムソースの再設定
  • リセット時間ソース
  • ゲットチルドレン
  • ゲットペアレント
  • 取得期間
  • レップス取得完了
  • 残りレップ数を取得
  • ゲットステート
  • 残り時間取得
  • 単位取得
  • タイムソースが存在する場合
  • If Time Source 親
  • If Time Source State
  • If Time Source Units

時間軸の変換には、以下のアクションが用意されています。

  • 秒からBPM
  • BPMから秒数へ

以下の定数は、Time Sourceアクションで使用されます。

  • 内蔵のタイムソース
  • タイムソース単位
  • タイムソースの有効期限タイプ
  • タイムソースステート
