数値から文字列へ

この操作は、実数をstring に変換します。数値が整数値でなく、小数点以下がある場合は、小数点以下は丸められますので、例えば、数値 123.45678 は、string "123.46" を返します。

アクションのシンタックス。

論拠となる。

論証 商品説明
Value 変換する実数
Target 戻り値を格納するターゲット変数
