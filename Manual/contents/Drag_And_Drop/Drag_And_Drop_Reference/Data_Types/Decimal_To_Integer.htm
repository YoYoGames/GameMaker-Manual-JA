10進数から整数へ

このアクションは、10進数を受け取って整数に変換します。変換する値を指定し、それを整数値にするために行う丸めの種類を選択します。選択できる丸めの種類は以下のとおりです。

  • ラウンド10進数を受け取り、最も近い整数に切り上げまたは切り下げを行います。指定された数値がちょうど半分の整数である場合（たとえば，1.5，17.5，-2.5 など），数値は 例えば、2.5は2に、3.5は4に丸められます。この種の丸めはバンカーズ丸めと呼ばれ、多数の反復計算や浮動小数点演算を使用する場合、統計的に最も近い値を与えます。 従来の「0.5を超えたら切り上げ、それ以外は切り捨て」よりも良い丸め方ができます。
  • Ceil: 10進数の値を、最も近い整数に切り上げます。round（上記）とは異なり、これは値の小数部分を考慮しないので、5.0000001のような数値はやはり6に切り上げられます。これは Random アクションのGet Random Numberを使ってランダムな小数の値を生成し、それを丸めるためにこのアクションを呼び出した場合、予期しない結果が得られる可能性があることに注意してください。ランダム アクションは0を返す可能性があり、それに対して「Ceil」を使用すると1ではなく0を返すので、0から9までのランダムな小数を取得してこの関数を使用すると、1から10までの整数を取得できると期待するかもしれませんが、実際には、1から10までの整数を取得できる可能性があります。 0から10まで。この可能性は低いですが、このアクションを使用する際には考慮する必要があります。
  • 階です。10進数の値を、最も近い整数値に切り捨てます。round（上記）とは異なり、これは値の小数部分を考慮しないので、5.9999999のような数値は、やはり床に落とされます。 5.

丸めの種類を選択したら、丸める数値を与え（これは変数でもよい）、新しい整数値を返すターゲット変数を与えなければなりません（これは一時的なローカル変数としてフラグを立てることができます）。
