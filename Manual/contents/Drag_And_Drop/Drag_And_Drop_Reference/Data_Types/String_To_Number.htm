文字列から数値へ

このアクションは、string を数値に変換することができます。変更するstring を指定し、実数を返すターゲット変数を指定します（この変数は一時的なローカル変数としてフラグを立てることができます）。もし，string が何らかの文字を持っている場合は ただし、string が数字で始まっている場合、それらは変換され、string の残りの部分は除外されます。注意すべきは，この関数に小数を渡す場合です。 アクションを実行した場合、返されるstring は小数点以下2桁に丸められるだけです。以下は、入力値と返される値の表です。

文字列 戻り値
"123" 123
"123 Hello World" 123
"Hello World" 0（デフォルトでは，string が変換できない場合，0が返されます。）
"Hello World 123" 0
"123.45678" 123.46

アクションのシンタックス。

論拠となる。

論証 商品説明
Value 変換するstring
Target 戻り値を格納するターゲット変数

例

上記のアクションブロックのコードは、グローバル変数を初期化した後、ファイルをロードして buffer に保存されている。strings から2つのbuffer を読み出して一時的なローカル変数に格納し、そのうちの1つを実数に変換して別の一時的なローカル変数に格納する。これらはグローバル変数に代入され buffer は削除される。
