コールペアレントイベント

このアクションブロックは、実際のインスタンスでアクションを実行し続ける前に、そのインスタンスが子である親object のイベントを実行します。親であるobject をインスタンスに指定すると、そのobject のイベントはすべて「継承」されます。 を子として持つ。しかし、子object にイベントを追加すると、親イベントをオーバーライドします。このアクションブロックを使用すると、子object 、親イベントとインスタンス自体のイベントの両方を実行するように強制することができます。親の詳細については、「オブジェクト・エディタ」のセクションを参照してください。
