インスタンスの回転を設定する

このアクションブロックは、インスタンスの image_angle (回転) を設定します。角度は度数で表され、右が0度、上が90度、左が180度、下が270度となっています。この変数を0に設定すると、インスタンスはリセットされ、次のようになります。 は、sprite エディタで定義されたとおりに、割り当てられたすべてのsprite を描画します。このアクションを変更するには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite を描画する）、描画される必要があることに注意してください。 を使用して、自己を描画するアクションです。
