インスタンススケールの設定

このアクションブロックは、インスタンスの image_xscale と image_yscale の値を設定します。ここでの入力値は、インスタンスに割り当てられたsprite に適用されるモディファイアで、1 (デフォルト値) はスケールなしを表します。 の拡大縮小（1:1）、小さい値では縮小（例えば0.5ではsprite の幅の半分）、大きい値では拡大縮小となります。負の値を使用すると、sprite を反転して拡大縮小することができますが、使用する値が正確に -1 である場合（その場合は sprite は、原点を中心に反転またはミラーリングされるだけで、スケーリングは行われません)。水平または垂直方向のスケールを現在の値から相対的に設定することができ、その場合、現在のスケール値から新しい値を加算または減算することになります。

これらの値を変更すると、オブジェクトエディタで別の mask_index (collision mask) を指定しない限り、インスタンスが衝突を検出する方法も変更されることに注意してください。以下の点に注意してください。 この変数の変更が見えるようにするには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite を描画する）、Draw Selfアクションを使って描画される必要があります。
