インスタンス数の取得

この操作で、object 内で現在アクティブになっている任意のroom のインスタンスの数を調べることができます。 リソースリストから任意のobject を選択するか、特別なキーワードを使用します。 all を使用します。 アクティブなすべてのインスタンスの数を取得します。また、その値を返すターゲット変数も必要で、それを使ってさらにアクションを起こすことができる。flag ターゲット変数を "Temp "とすると、新しいテンポラリ変数が作成されます。 (ローカル)変数に返り値を格納する。
