Object インスタンスの作成

このアクションを使うと、object ブラウザから任意のAsset のインスタンスを作成することができます。object の名前と、作成するroom 内の x/y 位置、インスタンスを作成するレイヤーの名前（必ず Instance Layer)を選択します。relative"flags にチェックを入れると、アクションコードブロックを実行しているインスタンスから相対的にインスタンスを配置することができます。ゲーム内でインスタンスを作成すると、そのインスタンスに一意のID値が与えられます。 そのため、このアクションでは、このID値を保持する「ターゲット」変数を定義することができます（以下の例に示すとおりです）。ターゲットは一時的な（ローカル）変数としてフラグを立てることができることに注意してください。 このアクションのために作成され、その後のすべてのアクションで使用することができます。

アクションのシンタックス。

論拠となる。

論証 商品説明
Object インスタンスを作成するobject の名前。
x インスタンスを生成するためのroom における x 位置 (呼び出し元のインスタンスからの相対位置でも可)。
y インスタンスを生成するためのroom における y 位置 (呼び出し元のインスタンスからの相対位置でも可)。
Layer インスタンスを作成するレイヤーの名前（string ）です。
Target 戻り値の対象となる変数。

例

上記のアクションブロックのコードは、新しいインスタンスを作成し、その一意のID値を 一時的な変数です。この変数は、次にApplies To... アクションで、作成されたインスタンスのいくつかのプロパティを設定するために使用されます。
