インスタンス・アルファの設定

このアクションブロックは、インスタンスの image_alpha の値を設定します。イメージのアルファ値は、インスタンスに割り当てられたsprite を描画するための「透明度」の値で、0 と 1 の間の値を持ち、0 は完全に透明、1 は は完全に不透明になります。sprite に透明または半透明のピクセルがある場合、その透明度は量によってスケールされることに注意してください。したがって、sprite に 50% の透明ピクセルがあり、インスタンスのアルファを 0.5 に設定すると、最終的にアルファ の場合、透明なピクセルは25%になります。この値は相対的な量として設定することができます。

この変数の変化が見えるようにするには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite ）、Draw Self アクションを使って描画される必要があることに注意してください。
