スプライトを設定する

このアクションブロックは、インスタンスの sprite_index をsprite Browser から別のAsset に設定します。変更するsprite を選択し、表示する画像 (アニメーションフレーム) を指定します。もし、sprite の後にアニメーションをさせたくない場合は、次のことに注意してください。 フレームを設定する場合は、Set Animation Speedアクションを使用して0に設定する必要があります。また、sprite のアニメーションを継続させたい場合は、 image_index の組み込み変数変数を使用するように値を設定します。 の代わりに

sprite を変更すると、インスタンスのコリジョンマスクも変更されます。ただし、Object Editor で別の mask_index も指定した場合は除きます。このアクションの変更には注意が必要です。 を表示させるには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite ）、Draw Selfアクションを使って描画される必要があります。
