インスタンスカラーを設定する

このアクションブロックは、インスタンスsprite との「ブレンド」用に image_blend の色を設定します。デフォルト値は 16 進コード $FFFFFFFF (ARGBフォーマットで白を表す) で、これはsprite をブレンドなし、完全なアルファを使って描画します。その他の値（ c_red や c_aqua のような内部カラー定数を含む）を指定すると、指定した色と元のsprite がブレンドされ、また image_alpha ただし、アルファを使用するチェックボックスがチェックされている場合に限ります。これは、Set Instance Alphaアクションを使用して以前に設定されたアルファ値を上書きします。

このアクションの変更が見えるようにするには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite ）、Draw Selfアクションを使って描画される必要があることに注意してください。また ブレンドされたsprite はそれぞれ別々にキャッシュされるため、ブレンドされたsprites が多数あるとパフォーマンスに悪影響が出る可能性があります。
