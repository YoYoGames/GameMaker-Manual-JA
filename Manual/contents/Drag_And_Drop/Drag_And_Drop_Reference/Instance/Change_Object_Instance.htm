変更 インスタンスObject

このアクションを使うと、あるインスタンスを別のインスタンスに変更することができます。インスタンスを別のインスタンスに変更するには、object を指定する必要があります。このアクションはインスタンスを変更し、Destroy イベントと Clean Up イベントを実行します。 を実行し、新しいインスタンスの作成イベントを実行します。方向や速度などを指定しない場合、新しいインスタンスはこれらの値を「継承」することに注意してください。たとえば、最初のインスタンスが image_angle (回転) を持つ場合 は180、移動速度は1ゲームティックあたり2px、 image_speed （アニメーション速度）は0.5で、新しいインスタンスのCreateイベントで明示的に設定しない限り、この値が維持されます。

インスタンスを変更すると、次のゲームティックまでそのインスタンスを使ったアクションができなくなることに注意してください。基本的に、あなたは インスタンスを変更しても、現在のステップが終了するまで実際には使用できないため、インスタンスが含む変数に直接アクセスすることはできません（たとえば、 inst_changed.x を呼び出すなど）。
