アラームカウントダウンを設定する

このアクションブロックは、指定されたアラームをカウントダウンするように設定し、アラームがゼロになると関連するアラームイベントがトリガーされます。アラーム番号を0から11の間で指定します。 次にアラームを設定する値を指定します。使用される値は、アラームイベントがトリガーされるまでに経過させたい「ゲームのステップ」数で、現在のアラーム値に対して相対的に設定することができます（現在のアラーム値に加算または減算されます）。 の値）。この値は整数値である必要があり、-1に設定するとアラームが停止することに注意してください。アラームがカウントダウンする場合、0になるとイベントが実行されますが、その後アラームが終了したことを示すために-1までカウントダウンされることに注意してください。アラームを アラームを0に設定しても、次のステップで-1までカウントダウンされるため、イベントは実行されませんので、アラームをすぐに実行するには1に設定する必要があります。アクションの横にあるプラスアイコン（ ）をクリックすると、同じアクションに別のアラームを設定することができることに注意してください。
