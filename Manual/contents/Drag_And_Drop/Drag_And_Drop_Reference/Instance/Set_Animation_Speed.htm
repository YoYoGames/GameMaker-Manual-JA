アニメーションの速度の設定

このアクションブロックは、インスタンスに割り当てられたsprite の image_speed (アニメーション速度) を設定します。スプライトエディターで理想の秒間フレーム数を設定し、この値 で修正することができます。デフォルトのアニメーション速度は1に設定されています。これは、sprite に定義した秒間フレーム数を維持しようとすることを意味しますが、1より大きい値を使用して速くアニメーションさせることができます（フレームはスキップされることがあります）。 または10進数でゆっくり（フレームは様々なステップで表示されます）、さらに負の値でアニメーションの速度を逆にすることもできます。

例

上記のアクションブロックのコードでは、新しいsprite を設定し、その他いくつかの プロパティで、sprite がどのように表示されるか、定義された速度の半分でアニメーションするように設定することも含まれます。
