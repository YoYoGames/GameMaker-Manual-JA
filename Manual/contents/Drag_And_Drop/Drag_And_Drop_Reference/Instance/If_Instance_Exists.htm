インスタンスが存在する場合

このアクションは、特定のobject のインスタンスがゲーム内に存在するかどうかroom 、または特定のインスタンス自体が存在するかどうかを確認するために使用することができます。object エクスプローラーからasset を渡し、room にインスタンスが存在すれば、このアクションは true を返し、そうでなければ false を返します。ここで、特定のインスタンスのユニークな ID (たとえば、CreateObject Instance というアクションで返されるもの) を変数として使うこともできますし、インスタンス ID を使うこともできます。 の値は、インスタンスを追加したときにroom エディタから返されたものです。

引数 "Not "をflag にすると、アクションはインスタンスが存在しないかどうかをチェックし、1つも存在しない場合は true を、1つでも存在する場合は false を返します。
