Destroy インスタンスObject

インスタンスを「破壊」したいときはいつでもこのアクションを呼び出し、room からインスタンスを削除します。インスタンスを破壊するときは、破壊を呼び出すアクションの直後にそのDestroy イベントが呼び出されることに注意する必要があります。また、Destroy Event はアクションが呼ばれた瞬間に実行されますが、インスタンスはすぐにゲームから削除されるわけではなく、そのまま継続されます。 を実行し、現在のイベントが終了して初めてインスタンスがゲームから削除され、その時点でクリーンアップイベントが呼び出されます。
