セーブゲーム

このアクションは、与えられたファイルへのゲームの基本的な保存を行います。ファイル名（string とextension を含む）を指定すると、現在のゲーム状態から限定的にセーブを実行するアクションです。重要なのは、このアクションが 現在のゲーム状態の「スナップショット」を取るように設計されており、動的リソース（データ構造やパーティクルなど）を保存しないため、ユニバーサルセーブシステムというよりもチェックポイントシステムとして使用するためのものです。そのため、このアクションはエラーやメモリリークを起こさないよう、非常に慎重に使用する必要があります。ゲームデータを保存するためのより包括的なアプローチについては、ファイルアクションを参照してください。
