ロードゲーム

このアクションは、Save Gameアクションで保存されたゲームをロードします。ロードするファイルのファイル名（string 、extension を含む）を指定すると、ゲームがロードされ、実行が継続されます。 を保存したところからスタートします。これは万能のセーブ/ロードシステムとして設計されているわけではないので、以前のプロジェクトで保存したゲームをロードしようとするとエラーが発生することがあります（特にデータストラクチャや パーティクルなどを使用した場合）。また、ファイルは現在のイベントまたはscript が終了するまでロードされないので、ロードアクションが呼び出された後のアクションはまだ実行されていることに注意してください。セーブされたゲームデータのロードに関するより包括的なアプローチについては、ファイルアクションを参照してください。
