ゲームを再起動する

この操作で、ゲームを再開することができます。これは、ゲームを初めて起動するのと同じで、Other - Game Start EventとOther - Game End Event が発生します。 ただし、グローバル変数は、明示的にコード化されていない限り、再初期化されません。つまり、ゲームの最初のroom で定義して設定しなければ、以前の値を保持することになります。 データ構造とパーティクルは解放されないので、Restart Gameアクションが呼ばれる前か、Game End Eventで実行する必要があります。また、このアクションが呼ばれた瞬間にゲームが再開されるのではなく、アクションが配置されたscript やイベントの終了時に再開されることに注意してください。このアクションの後に配置されたアクションがあったとしても、ブロックの終了時にそれらは呼び出されてゲームが再開されます。
