退出ゲーム

このアクションでゲームを終了させ、Other - Game End イベントをトリガーすることができます。これはアクションが呼ばれた瞬間ではなく、現在のアクションの終了時に発生することに注意してください。 イベントまたはscript で、このアクションが呼び出された後のアクションは、イベント/スクリプトの終了に達するまで実行されます。

なお、このアクションには以下の制約があります。

  • iOSのターゲットでは、そのマーケットプレイスの条件に反するため、エラーが報告されます。
  • HTML5およびWindows UWPのターゲットで呼び出されても、効果はありません。
  • Windows（Steam ）、Android、Ubuntu（Linux ）、Macで動作します。
