セル内の インデックスを取得Tile

このアクションを使用すると、指定したレイヤーのtile インデックスを取得できます。レイヤーは与えられたレイヤー名（string 、Room エディターでレイヤー名を付けるのに使用）から指定し、次にタイル のインデックスを取得します。最後に、返されたtile のインデックスを格納するターゲット変数を指定します。0 はtile がなく、正の整数がtile のインデックスです。ターゲット変数には、一時（ローカル）変数として使用できるフラグを付けることができることに注意してください。 イベント終了まで

アクションのシンタックス。

論拠となる。

論証 商品説明
Layer tile を取得するレイヤー
Column セルの列(左から右へ)
Row セルの行（上から下へ）。
Target 戻り値の対象変数

例

上記のアクションブロックコードは、loop を作成して room 与えられたtile レイヤーの一番上の行で、その列のセルのインデックスが 0 になっているかどうかをチェックします。そうでない場合、そのセルのtile インデックスが 0 に設定され、ループの前に一時（ローカル）変数がインクリメントされます。 は別のチェックを行う。
