Set Data TransformTile

このアクションでは、tile データにある 3 つのtile トランスフォームのうち 1 つを変更することができます。tile データを保持する変数をtile に指定し、変更する変換を選択します。適用」チェックボックスをクリックすると、その変換が適用されます。 チェックを外すと削除されます。tile データは、tile のインデックスと、適用されたトランスフォーム、および使用したカスタムtile マスクを反映した値で、このアクションではこれらの値のうち 3 つを変更することが可能です。 このアクションでは、次の3つの値を変更できます：反転、ミラー、回転。通常、Gettile Data In CellまたはGetTile Data At Pixel を使用してTile データを最初に取得し、次に必要な このアクションは値を設定するだけで、取得した値をtile に適用するわけではないので、 tile Data In Cell またはSetTile Data At Pixel を使用してTile マップのtile を再度設定します。 tile データの詳細については、「タイルマップ機能」のGML の項を参照してください。
