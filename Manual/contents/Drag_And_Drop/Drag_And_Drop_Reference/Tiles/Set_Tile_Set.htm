セットタイルセット

このアクションでは、Asset ブラウザで作成したものとは異なるタイルセットを使用するようにレイヤーを設定することができます。レイヤーの名前（string 、Room エディターで定義されたもの）を与え、次に tile set リソースを使用すると、そのレイヤーのtiles にあるすべてのroom が、新しいtile set で描画されます。

アクションのシンタックス。

論拠となる。

論証 商品説明
Layer 新しいtile set を設定するレイヤーです。
Tileset レイヤーに使用するtile set 。

例

上記のアクションブロックは、レイヤー "Floor_Tiles "上のtiles が、"Floor_Tiles "を使用しているかどうかをチェックするものです。 tile set "tl_PalaceRuins" を使用するように設定されます。
