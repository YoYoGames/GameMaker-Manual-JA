タイルマップ変数の取得

このアクションでは、レイヤーのタイルマップ要素について、指定された数の変数のうちいずれかを取得することができます。tile エディターでroom レイヤーを作成すると、このレイヤーはタイルマップ要素を保持し、次にタイルセットからtiles を入力します。このtile マップ要素には、オフセット位置、使用されているタイルセット、幅、高さなど、特定の値が関連付けられています。取得できる値の完全なリストは次のとおりです。

  • X座標。 tile map 要素が配置されたレイヤーの x 軸に沿った位置です。
  • Y座標。 tile map 要素が配置されたレイヤーの y 軸に沿った位置です。
  • 列。 tile map 要素の幅（tile セルの列数として定義される）。
  • 行数です。tile map 要素の高さ (tile セルの行数として定義される)。
  • タイル幅： tile のセル1枚の幅。
  • タイルの高さ。 tile のセル1枚の高さ。
  • タイルセットリソース（Tile Set Resource）。 tile map elementに描画用に割り当てられたtile-set resource。
  • 現在のフレーム。アニメーションのために描画されている現在のフレームtiles 。
  • Mask： tile のマップエレメントのマスクデータ。
  • Global Mask: tile の全マップエレメントに対応するグローバルマスクデータ。

このアクションを使用する場合、string マップ要素データを取得するレイヤー名（Room 、tile エディタで定義）を指定し、次に取得したいデータの種類（上のリストに示すとおり）を指定します。返された値は、次の場所に保存されます。 ターゲット変数は、以前に作成されたものでも、（"Temp "チェックボックスをチェックした場合）新しい一時的なものでもよい。さらに別の値を取得するには、プラス・アイコンである 横 アクションを実行し、別の変数を選択し、返された値を格納する別の変数を指定します。
