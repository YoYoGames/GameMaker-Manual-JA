Pixelで データを取得するTile

このアクションを使うと、ある位置からタイルマップレイヤー上のtile セルのtile データを取得することができます。tile データは、tile のインデックスと、フリップ、ミラー、ローテートの値、およびカスタムtile マスクが反映された値です。 を使用しました。このアクションでは、まずstring エディタで定義されたレイヤーの名前 (room) を使ってターゲットとするレイヤーを指定し、次にroom でタイルデータを取得する x 軸と y 軸を指定します。返されるtile データ値は、以下のものと一致します。 その時点のタイルマップセルが何であれ。例えば、タイルセットtiles が 16x16 の場合、タイルマップ層は 16x16 ピクセルのセルでtiles を保持します。したがって、このアクションを使ってroom の位置 (8, 8) をターゲットにすると、次のようになります。 タイルマップのセル(0, 0)のタイルデータ。

注：選択したレイヤーは、Tile エディターでroom Map Layer として定義されている必要があります。そうでない場合はエラーが発生することがあります。

返された値は、ターゲット変数に格納されます。ターゲット変数は、以前に作成されたものでも、（「Temp」チェックボックスをオンにした場合）新しい一時的なものでもかまいません。tile データについての詳細は、GML タイルマップ関数のセクションを参照してください。
