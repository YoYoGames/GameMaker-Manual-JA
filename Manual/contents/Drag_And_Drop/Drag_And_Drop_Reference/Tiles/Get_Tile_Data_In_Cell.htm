セル内の データを取得するTile

タイルマップレイヤー上のtile セルのtile データを取得することができるアクションです。tile データは、tile のインデックスを反映した値で、フリップ、ミラー、ローテートの値も含まれています。 および使用したカスタムtile マスク。このアクションでは、まず、string エディターで定義されたレイヤーの名前（room ）を使用してターゲットとするレイヤーを指定し、次に、タイルデータを取得するtile マップレイヤーの列と行を指定します。 のためのものです。セル "とは、データを取得したいtile を保持するtile マップグリッド上の領域です。例えば、tiles が 16x16 で、room が 1024x768 の場合、タイルマップは 64 列と 48 行で構成されることになります。

注：選択したレイヤーは、Tile エディターでroom Map Layer として定義されている必要があります。そうでない場合はエラーが発生することがあります。

返された値は、ターゲット変数に格納されます。ターゲット変数は、以前に作成されたものでも、（「Temp」チェックボックスをオンにした場合）新しい一時的なものでもかまいません。tile データについての詳細は、GML タイルマップ関数のセクションを参照してください。
