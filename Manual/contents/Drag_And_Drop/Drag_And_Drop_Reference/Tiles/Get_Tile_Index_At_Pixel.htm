Tile Pixelでインデックスを取得

このアクションは、指定されたレイヤー上のtile インデックスを取得するために使用します。レイヤーは、与えられたレイヤー名（string 、Room エディターでレイヤー名を付けるのに使用）から指定し、次にインデックスを取得するtile を見つけるための x と y 位置を指定します。タイル は、指定された位置がroom に含まれるものとなり、インデックスがターゲット変数に返されます。0 はtile がなく、正の整数がtile のインデックスとなります。 また、ターゲット変数には、イベントの終了まで使用できる一時的な（ローカル）変数としてのフラグを付けることができる。

アクションのシンタックス。

論証 商品説明
Layer tile を設定するレイヤー
x チェックするxの位置
y チェックするY位置
Target tile のインデックスを格納するターゲット変数
