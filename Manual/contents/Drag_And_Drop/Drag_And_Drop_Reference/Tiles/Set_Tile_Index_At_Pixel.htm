Tile インデックスを画素に設定する

このアクションでは、指定されたレイヤーのtile を新しいインデックスに設定することができます。レイヤーは、与えられたレイヤー名（string 、Room エディターでレイヤー名をつけるのに使われる）から指定し、次に、設定するtile を得るための x と y 位置を指定します。このxと y 位置は、呼び出したインスタンスの位置からの相対的なものとしてフラグを立てることができ、設定されるtile は、与えられた位置がroom の中に入るものとなり、使用されるtile set は、部屋内のレイヤーに割り当てられたものとなります。 エディタを使用します。最後に、tile に設定するtile インデックスを指定します（tiles は左から右、上から下へのインデックスで、0 は左上です）。なお、tile のインデックスを0に設定すれば、いつでもtile をクリアすることができます。
