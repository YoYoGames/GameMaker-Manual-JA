Tile データを画素に設定する

タイル地図レイヤーの特定の位置にあるtile セルにtile データを設定することができます。tile データは、tile のインデックスを反映した値です。 flip、mirror、rotateの各値と、使用したカスタムtile マスクがあります。このアクションでは、まずstring エディタで定義されたレイヤーの名前 (room) を使ってターゲットとするレイヤーを指定し、次にルーム内の x 軸と y 軸を指定します。 に、tile のデータを設定します。tile のデータは、指定した位置のtile マップの "セル "に設定されることに注意してください。例えば、tile set tiles が 16x16 の場合、tile マップレイヤーは 16x16 ピクセルのセルにtiles を保持します、したがって このアクションでroom の位置 (8, 8) をターゲットにすると、tile マップのセル (0, 0) にあるtile を変更することになります。

注：選択したレイヤーは、Tile エディターでroom Map Layer として定義されている必要があります。そうでない場合はエラーが発生することがあります。

アクションの最後の引数は、tile データそのものです。通常、アクションGettile Data At Pixelを使ってtile マップセルのTile データを取得し、アクションSetTile Data Transformを使って操作してから、このアクションで再度設定することになります。tile データについての詳細は、GML タイルマップ関数のセクションを参照してください。
