プロジェクトイメージジェネレーター

プロジェクトイメージジェネレーターは、"Tools "メニューからアクセスすることができます。 GameMaker IDE の上部にあります。このツールは、さまざまなターゲットプラットフォームで必要とされるスプラッシュスクリーン、プロモ画像、アイコンを、1つの画像セットから素早く作成するために設計されています。ベースとなる画像 とベースアイコンが表示され、Generate ボタンをクリックすると、各プラットフォームのGame Optionsが新しい画像で更新されます。

正しく動作させるためには、高解像度のスプラッシュ画面と高解像度のアイコン画像を提供する必要があります。推奨されるサイズは以下の通りです。

  • スプラッシュスクリーン：最低1920x1080px、ポートレートの場合は1080x1920pxで、透明度のない24bit *.png としてオーサリングされたもの。
  • アイコン：1024x1024px以上（これは必要な最大アイコンのサイズです）、24bitでオーサリングされ、透過性を持つことができます *.png

これらの画像を提供すると、プレビューウィンドウに表示され、異なるプラットフォームに適用する際に使用する「フィット」の種類を設定することができます。以下の中から選択することができます。

  • アスペクト比を維持- 画像をアスペクト比を維持したまま拡大縮小し、1:1でない場合は背景色を追加して横を塗りつぶします。
  • Stretch To Fit- 画像を引き伸ばして、画像/アイコンのフルサイズにフィットさせます。
  • Crop To Fit- ベースとなる画像を取得し、水平または垂直軸のどちらかに沿って最適になるようにし、残りをクロップします。例えば、640x480の画像を480x800のスプラッシュスクリーンとして使用しなければならない場合。 の場合、画像は縦軸（480→800）に合わせて拡大され、残りは切り取られます。同様に、1080x1920の画像を800x800に収めなければならない場合、画像は横軸に合うように縮小されます。 を軸に、残りを切り取る。

また、アイコンと画像の両方に背景色を設定することができます。この色は、アスペクト比で拡大縮小されたときに、画像の「パッド」に使用されます。次の画像は、すべての設定が完了したときの様子です。

あとは、Generateを クリックするだけで、GameMakerがゲームオプションの画像と アイコンの 設定に、各プラットフォームの画像を使用します。

注：上記の記述の例外は、Android Adaptive Iconsです。これらは、その構造上、このツールを使用して生成されることはありません。

重要!こ のツールは、あなたが必要とするすべての画像を生成しますが、それはそれが完璧であることを意味するものではありません!各プラットフォームの異なるサイズや解像度に合うよう、最善を尽くしますが、時にはそれが不可能な場合もあります。 というのも、「1つのサイズですべて解決する」ということはあり得ないからです。したがって、このツールを実行した後、各プラットフォームのゲームオプションですべての画像を修正し、正確にどのように見えるかわからないものを調整することをお勧めします。 を手動で行ってください。
