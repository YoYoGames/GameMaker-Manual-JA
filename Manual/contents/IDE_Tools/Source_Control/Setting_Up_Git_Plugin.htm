Git プラグインを設定する

Git プラグインを使用する前に、環境設定のプラグイン -Git (Source Control)セクションでGit 実行ファイルを使用するように設定する必要があります。

Git 実行ファイルpath がGameMaker で見つかった場合は自動入力されるはずですが、フィールドが空の場合は、 ボタンをクリックして、Git インストールの cmd フォルダに含まれる git.exe ファイル（通常は C:\Program Files\Git\ の中）を手動で選択することができ ます。

その他のGit オプションについては、「Plugin Preferences」ページを参照してください。

プロジェクトにSource Control を使用する前に、Game Options で Source Control を有効にする必要があります。
