リポジトリオプション

Git 実行ファイルを設定し、Source Control をプロジェクトで有効にした後、GameMaker IDE をリポジトリにリンクする必要がありますが、これには2つの方法があります。

  • プロジェクトリポジトリの作成
  • 既存のリポジトリをクローンする

最初のケースであるプロジェクトリポジトリの作成では、プロジェクトがある場所に直接Git リポジトリが作成されます。Git の用語では、これは "非ベアリポジトリ" で、この場合Push/Pull は意味を持ちませんが、プロジェクトの変更をコミットして追跡し、必要なら変更とリビジョンを元に戻すことができます。この設定を行うには、メインメニューのSource Control>Create Project Repositoryをクリックするだけです。

便利な機能として、IDE から外部リポジトリのクローンを 作成することもできます。ここでも、接続するリポジトリのユーザー名とパスワードを ID 環境設定で設定し、Source Control>Clone Repository をクリックするだけです。URL を入力し（SSH ではなく HTTPS を使用します）、どこに移動したいかを指定します。

クローンが完了すると、自動的にファイルブラウザが起動し、必要に応じてプロジェクトを開くことができます。
