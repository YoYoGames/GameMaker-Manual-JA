コードスニペット

コード編集の際に非常に便利なのが、コードスニペットの利用です。script エディタを使用しているときに、 F4 を押すと、コードスニペットのポップアップが表示され、よく使われるコードメソッドを一覧から選択することができます。

このウィンドウから、使用するスニペットをマウスで選択するか、右側に表示されている関連するホットキーを押すことができます。この操作により、スニペットがコードに追加され、編集できるようになります。

また、必要であれば、独自のコードスニペットを定義することもできます。これを行う前に、以下のディレクトリのいずれかに "snippets.txt" というファイルを作成する必要があります。

  • Windowsです。 %ProgramData%\GameMaker\User\
  • macOSです。 /Users/Shared/GameMaker/User/

このフォルダはGameMakerのアップデート中も変更されないので 、編集したファイルはそのまま残ります（ただし、インストールディレクトリのベーススニペットファイルの編集は元に戻されます）。

Userディレクトリにファイルが作成されたら、以下のルールに従って、任意のテキストエディタで編集することができます。

  • 各スニペットは独立した行である必要があります（空白行は不可）。
  • 各スニペットは、使用するホットキーの後に "-" を続け、スニペット名（メニューに表示されるもの）の後にコロン ":" を続ける必要があります、例．

I - インスタンス作成。

コロンの後に、コードのスニペットを追加します。

I - Instance Create:instance_create_layer(x, y, |layer|,object);

また、追加するコードは、特定の書式に従わなければなりません。

  • すべてのコードは1行に
  • 改行には "$" を使用します。
    • 注：複数の改行を行う場合、各「$ 」記号の間にスペースを入れる必要があります。例えば、3つの改行を追加する場合は、$ $と記述します。
  • |" 記号の間にハイライトしたいテキストを入れます。スニペットが追加されると、コードエディターが自動的にこのテキストを選択して編集します。

上記のルールに従ってどのように設定されるかは、すでにベースとなるスニペットファイルに入っているサンプルをご覧ください。ベースとなるファイルは、GameMakerのインストールディレクトリにあります。

  • Windowsの場合。 GameMaker\୧⃛(๑⃙⃘◡̈๑⃙⃘)
  • macOS: /Applications/GameMaker.app/Contents/MonoBundle/TextEditor/snippets.txt
