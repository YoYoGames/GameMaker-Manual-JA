アニメーション・カーブ・エディター

animation curve は、値が時間とともにどのように変化するかを表す1つ以上の曲線を含む で、曲線内の異なる点間を移動するためにasset 線形補間、スムーズ補間、ベジエ補間を使用することができます。縦軸に設定する値は、任意の2つの値の間（デフォルトでは-1〜1）で、横軸に沿った期間は常に 0〜1で正規化されているため、以下のことが容易に行えます。 ゲームコードで乗数を使用して異なる時間範囲をターゲットにしたり、シーケンスエディタでトラックに合わせて引き伸ばしたりすることができます。

アニメーションカーブは「チャンネル」で構成され、各チャンネルにカーブ設定を持たせることができるため、例えば、空間位置をx/y位置の2チャンネルで表現したり、カラーグラデーションを4チャンネルで表現することができます。 Animation curve assets は、Sequences で作業するときに使用でき、コードでアクセスすることもできるため、ゲームを作成するときの強力なツールになります。

animation curve asset を初めて作成すると、エディタウィンドウが開き、以下のセクションが表示されます。

カーブ名

ここで、animation curve に固有の名前（英数字と「_」アンダースコア記号のみ）を付けることができます。この名前は、カーブとそのすべてのチャンネルを参照するために使用されます。 コードまたはsequence でアクセスする場合。

カーブライブラリ

ボタンを押すと アニメーション・カーブ・ライブラリウィンドウを表示します。 アニメーションカーブのカーブタイプを変更したり、選択したチャンネルにプリセットを適用したりすることができます。

アニメーション・カーブ・ライブラリ」の項を参照してください。 カーブの種類とプリセットについて ボタンからウィンドウを開くと、デフォルトで「上書き」適用モードが選択されていることに注意してください。

チャンネル一覧

チャンネルリストでは、animation curve asset を構成するさまざまなチャンネルを追加し、名前を付けることができます。animation curve を最初に作成したときのデフォルトでは、1つのチャンネルが追加されています。 をクリックすると、カーブ名の左にある のボタンが表示されます。各チャンネルにはユニークな名前を付ける必要があります（英数字とアンダースコア記号「_」のみ使用）。 また、左側の色見本をダブルクリック（ ）すると、カラーピッカーが開き、新しい色を選択することができるので、チャンネルの色を設定することができます。 チャンネルの名前を変更するには、ゆっくりダブルクリック して名前を変更します。また、マウスの右ボタン を使ってメニューを開き、名前の変更、色の変更、チャンネルの削除を選択することも可能です。

チャンネルは、名前の横の矢印をクリックすることで拡張でき、チャンネルカーブの各ポイントを表示します。これらのポイントは、現在の値をクリックし、新しい値を入力することで、手動で編集することができます。

目」アイコン をクリックするとチャンネルの表示/非表示を切り替えられ、「ビン」アイコン をクリックするとチャンネルを削除することができます。

チャンネル範囲

チャンネル範囲ボタンは、チャンネルエディタの視覚的な垂直範囲を設定するために使用します。デフォルトでは -1 から 1 に設定されていますが、このボタンをクリックすると次のようなウィンドウが表示されます。

これらの値を変更することで、チャンネルエディタセクションでのデータの表現方法が変わり、以下のことが可能になります。 のみならず、チャンネルカーブに沿ってより高い／低い値を設定することができます。これは純粋に視覚的なものであり、チャンネルの値をクランプするものではないことに注意してください。

チャンネルエディター

チャンネルエディターは、animation curve で使用されている様々なチャンネルを視覚的に表示します。 デフォルトでは、すべてのチャンネルがここに表示されますが、チャンネルリストから現在選択されているチャンネルのみを編集することができ、そのチャンネルはハイライト表示され ます。 をエディタで表示します。チャンネルカーブに沿ったポイントの位置は、 をクリックし、エディタ内でドラッグすることで変更することができます。 最初と最後の点は、垂直軸に沿ってのみ変更可能で、水平方向の値は常にそれぞれ0と1になります。

チャンネルカーブに直接ポイントを追加するには、メインチャンネルラインにマウスを近づけ、カーソルが "ポイントの追加" に変わった時に をクリックします。 その後、 をクリックしてドラッグするか、チャンネルリストの拡張チャンネルオプションで値を変更することにより、これらのポイントを編集することができます。

をクリックし、エディタ内でドラッグして編集したいポイントをすべて選択するか、 / + を使ってポイントを個別に選択範囲に追加すると、複数のポイントを一度に編集することができます。マウスボタンを離した後、選択範囲内の任意のポイントを再度クリックすることで、複数のポイントを一度に編集することができます。 をクリックしてドラッグすると、まとめて移動できます。

チャンネルエディタで任意の場所を右クリックして、アニメーションカーブライブラリウィンドウを開くことができます（選択されたカーブタイプがベジェの場合のみ）。この方法でプリセットを適用すると、チャンネル全体に影響します。チャンネルの一部だけを変更するには、 個々のラインを右クリックしてプリセットを適用するか、 をクリックしてドラッグして複数のラインを選択し、 選択部分を右クリックして選択したラインにプリセットを適用することができます。

チャンネルエディタからアニメーションカーブライブラリウィンドウを開くと、デフォルトで「間」適用モードが選択されていることに注意してください。
