Timeline 編集部

多くのゲームでは、ある事柄を正確なタイミングで発生させたいものです。アラームイベントを使用することで、これを実現することができますが、物事が複雑になりすぎると、これ以上うまくいきません。そのため、タイムラインアセットが 用意されています。timeline では、ゲーム時間内の特定の瞬間に起こるべきアクションを指定します。object のさまざまなイベントやコードで利用できるすべてのアクションを使用して、これを定義することができます。タイムラインを作成したら、それをobject に割り当てます。そのobject のインスタンスは、room 内に配置または作成されると、指定された時間の瞬間にアクションを実行します。

例えば、ガードを作りたいとして、このガードは左へ20タイムステップ、次に上へ10タイムステップ、次に右へ20タイムステップ、次に下へ10タイムステップ移動して、最後に止まるとします。これを実現するには、timeline 、まず左への動きを設定する。20秒後に上方向の動き、30秒後に右方向の動き、50秒後に下方向の動き、そして60秒後に停止します。これができたら、このtimeline を警備員object に割り当てると、警備員のインスタンスがroom に配置されたときに計画通りに動作します。時間軸にアラームを設定して、一定時間後に繰り返されるようにすれば、警備員が地域を巡回することも可能です。また、タイムラインを使って、よりグローバルにゲームをコントロールすることもできます。例えば、目に見えないコントローラobject を作り、特定の瞬間に敵を作り出すタイムラインを作り、それをコントローラobject に割り当て、インスタンスをroom に配置します。これは、宇宙船ゲームなどで、敵を時間差でスポーンさせる簡単な方法です。

新しいモーメントを追加する際に、GML Visualと GML Codeのどちらかを選択するよう求められる場合があります。詳しくはGameMaker Languageをご覧ください。

時間軸を作成するには、アセットブラウザ上で を右クリックし、「タイムライン作成」を選択します。すると、このページの上部にある写真のようなウィンドウが開き、以下のセクションが表示されます。

名称

ここでは、timeline のリソースに名前を付けます。この名前は、コードを通してtimeline を参照するために使用されるため、英数字またはアンダーバー「_」のみを含む必要があり、数字で始めることはできません。この名前は、timeline Visual アクションやコード関数でGML を識別するために使用できる、一意のtimeline ID にもなります。


モーメント

ここでは、モーメント 値を設定することができます。モーメント値は、timeline が始まってからのゲームステップ数として計算されます。つまり、0と60の位置にモーメントを持つtimeline がある場合、モーメント0はtimeline が開始 されたときに発生し、モーメント60はその60ステップ後に発生します。room の速度も 60 に設定されている場合、モーメント 60 はtimeline が開始された 1 秒後に発生します。モーメント値をダブルクリック するか、右ボタン メニューから「変更」（下のボタンメニューの項を参照）を選択して変更することができます。新しいモーメントを追加するには、エディタの下にあるAdd ボタンをクリックするだけです。新しいモーメントを追加すると、コードエディタ（またはアクションエディタ）が開き、そのモーメントで呼び出したい関数またはGML Visual を追加することができます。


モーメント名

timeline が何をしているのかを理解しやすくするために、モーメントに名前をつけて識別することが許されています。これは単にstring であり、ゲームコードや他のどこにも使われません。純粋にtimeline の動作を整理し、理解するためのものだからです。モーメントに名前と説明をつけるには、コードブロックの最初の行で /// @description の形式を使う必要があります。

/// @description Hello World

というと、こんな感じになります。

timeline エディターでは、 + を使って、最初のクリック と 2 番目のクリック の間のすべてのモーメントを選択するか、 / + を使って、単独のモーメントをクリックして選択できます。Timeline モーメントは、選択した要素の 1 つで、右クリック メニューを使って編集することができま す。

  • 追加します。このオプションで、新しいモーメントをtimeline に追加することができます。 最後のモーメントの直後に追加されるので、timeline の最後のモーメントが 120 であれば、次に追加するのは 121 になります。モーメントの位置を変更するには、エディターでモーメントの値をダブルクリック（ ）するか、右クリック（ ）で「Change」を選びます。
  • Edit：現在のtimeline モーメントを編集用に開きます。
  • モーメントを変更する。選択したtimeline のモーメントの値を変更します。これは、エディタウィンドウのtimeline の値をゆっくりダブルクリック（ ）するのと同じです。
  • 説明文を変更する。選択されたtimeline 瞬間の説明的な名前を変更します。これは、エディタウィンドウのtimeline の説明文の上で をゆっくりダブルクリックするのと同じです。
  • 複製します。これは、選択したすべてのtimeline の瞬間を複製し、現在のtimeline の末尾に連続して追加します。
  • マージする。timeline から2つ以上のモーメントを選択した場合、それらをマージすることを選択することができます。これにより、最初に選択したモーメントの位置に、すべてのコードとアクションが同時に含まれた1つのモーメント（timeline ）が作成されます。
  • 削除します。これは、選択したモーメント（または複数のモーメント）をtimeline から削除します。
  • カット／ コピー／ ペースト。これらのオプションにより、1つまたは複数のtimeline の瞬間をカット、コピー、ペーストすることができます。ここでも標準的なキーボードショートカットが使用できます。 / + "X", / + "C", and / + "V".
  • すべて選択このオプションで、timeline のすべてのモーメントを選択できます。 選択したモーメントは、削除、結合、複製、カット、コピーすることができます。
  • GML Visual / Codeに変換します。タイムラインモーメントはプロジェクト作成時に選択した方法、つまりコードを使うかGML Visual アクションを使うかによって作成されます。このオプションを使えば、プロジェクトの種類に関係なく、一方から他方へ変換することができます。
