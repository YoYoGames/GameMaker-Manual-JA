アニメーションのカーブを利用する

ドープシートの重要な特徴は、sequence 内のパラメータトラックにアニメーションカーブを割り当てて、パラメータカーブの代わりに作成できることです。 のパラメータkeyframes 。パラメータトラックに使用できるカーブには、2つのタイプがあります。

  • エンベデッドカーブ。sequence に埋め込まれているカーブで、使用するパラメータトラックに固有のものです。
  • 共有カーブ。Animation Curve Editorを使用して、sequence とは別に作成されたカーブです。

まず、アニメーションカーブ（Animation Curve）asset の使い方を説明します。これは、新しいカーブを作成し、アニメーションカーブエディタでチャンネルを設定し、カーブasset をdope sheet にドラッグして、パラメータに適用する簡単な方法です。 のトラックを使用します。


また、マウスの右ボタン メニューのオプション「共有カーブの割り当て（インポート）」 を使って、animation curve アセットを追加することができます。 を選択したパラメータトラックに追加します。これにより、Asset エクスプローラが開き、表示されたカーブの中から必要なものを選択することができます。

animation curve アセットには、適用するトラックに対して正しい数のチャンネルが必要です。 十分なチャンネル数を持たない は、 ブラウザに インジケータとともに表示されます。animation curve animation curves asset

asset 以下は、各カーブで必要となるパラメータトラックとチャンネルのリストです（チャンネルは、各パラメータタイプに示された順序で追加する必要があります）。

  • ポジション：X、Yの2チャンネル。
  • 回転させる。1チャンネル
  • カラーマルチプライス。アルファ、レッド、グリーン、ブルーの4チャンネルで、範囲は0〜255であること。
  • 画像インデックス。1チャンネル
  • イメージスピード：1ch.
  • オリジン：X、Yの2チャンネル。
  • スケール：XスケールとYスケールの2チャンネルで、値は乗数なので、基本値は1（スケールなし）で、1より大きいと画像が大きくなり、1より小さいと小さくなることに注意してください。負の値は、反転または 画像をミラーリングし、スケールを適用します。
  • Pitch: 1チャンネル。最低でも0でなければならず、1がサウンドのデフォルトピッチとなる。
  • 音量：1ch。最小値0、最大値1であること 。

パラメータトラックのチャンネル数が足りないanimation curve をインポートした場合、足りないチャンネルをアニメーションカーブに追加するかどうかを選択するプロンプトが表示されます。

また、外部エディタ（animation curve ）で必要なチャンネルを削除すると、共有カーブが無効になることがあります。このような場合、共有カーブはdope sheet で無効と表示されますので、「チャンネルを追加」をクリックして、必要なチャンネルを作成します。 を削除すると、カーブは再び有効になります。

デフォルトでは、animation curve は、添付されているasset キーの全長に沿って適用されますが、keyframes と同様に、曲線上の をクリックしてドラッグすると、timeline の位置を変更できます。また、 キーを押しながらドラッグすると、曲線の長さを変更することが可能です。 (これは、埋め込まれたカーブにも適用されます）。

パラメータカーブをトラックに追加するもう一つの方法は、それを埋め込むことです。これは、マウスの右ボタン を選択するか、Toggle Curve Mode ボタンをクリックして、トラックをanimation curve としてdope sheet の全領域に表示させ、Convert To Curveボタンをクリックすることができます。

パラメータkeyframe トラックをカーブに変換したら、dope sheet で編集することができます。 アニメーションのカーブアセットを編集するのと同じように、ポイントの追加や削除、位置の変更、カーブタイプの変更（線形、スムーズ、ベジエ）、プリセットの適用、チャンネル名や色の変更などが可能です。 に関連付けられます。

重要：この方法でパラメータトラックに適用されたanimation curve assets を編集することもできますが、編集した内容は、アニメーションカーブを使用しているすべてのトラックに適用されます。 カーブのコピーではなく、asset 。あるトラックに適用されたanimation curve asset をそのトラックだけで編集するには、まずパラメータトラック上でマウスの右メ ニュー を使い、Embed Curve を選択する必要があります。これにより、animation curve asset が複製され、そのトラックだけの独立したカーブとしてパラメー タトラックに埋め込まれます。逆に、埋め込まれたアニメーションカーブを変換することもできます。 マウスの右ボタン メニューの「Export To Shared Curve」を使って、「Animation Curve」asset に変換します。これにより、埋め込まれた カーブを新しいasset 、通常通り編集したり、必要に応じて他のパラメータトラックに適用することができます。

チャンネル一覧

チャンネルリストでは、パラメータカーブを構成するさまざまなチャンネルを編集することができます。各チャンネルにはユニークな名前（英数字とアンダースコア記号「_」のみ使用）を付ける必要があります。 また、左側の色見本をダブルクリック（ ）すると、カラーピッカーが開き、新しい色を選択することができるので、チャンネルの色を設定することができます。 チャンネルの名前を変更するには、ゆっくりダブルクリック して名前を変更します。また、マウスの右ボタン を使ってメニューを開き、名前の変更、色の変更、チャンネルのコピーを選択することも可能です。チャンネルは名前の横の矢印をクリックすることで拡張でき、チャンネルカーブの各ポイントが表示されます。 を入力し、新しい値を入力します。また、dope sheet の「目」のアイコン をクリックすると、チャンネルの表示/非表示を切り替えることができます。

変換制御

ここでは、keyframes をパラメータカーブに変換することができます（このページで詳しく説明します）が、パラメータカーブをkeyframes に変換するボタンもあります。ただし、この場合、いくつかの注意点があることに注意してください。キーフレーム そのため、いくつかのカーブ値は最も近いフレームに丸める必要があり、その結果、パラメータトラックの動作が変化します。また、パラメータカーブがcatmull-romやbezier補間を使用している場合、これは失わ れます。 keyframes の間の補間は、線形補間に設定されます。

カーブメニュー

このメニューには、エディタ内の他の場所で使用できるオプションが含まれていますが、すべて一箇所にまとめられています。オプションは以下の通りです。

  • 範囲を設定します。パラメータカーブの垂直方向の範囲を設定します（下記「チャンネルレンジ」参照）。
  • カーブに変換する。keyframes を使って選択したパラメータトラックを、パラメータカーブに変換します。
  • カーブを埋め込む。パラメータトラックで使用されている共有カーブasset をトラックに埋め込む。
  • Assigned Shared Curve (Import)。パラメータトラックの共有カーブとして使用するアニメーションカーブasset をアサインします。
  • キーに変換する。パラメータカーブを使って選択したパラメータトラックをkeyframes に変換します。
  • Shared Curveにエクスポートします。埋め込みカーブを共有カーブに変換するasset.

チャンネル範囲

チャンネル範囲ボタンは、チャンネルエディタの視覚的な垂直範囲を設定するために使用します。デフォルトでは -1 から 1 に設定されていますが、このボタンをクリックすると次のようなウィンドウが表示されます。

これらの値を変更すると、チャンネルエディタセクションでのデータの表現方法が変わり、チャンネルカーブに沿って-1〜1よりも高い値や低い値を設定することができます。これは純粋に視覚的なものであり、チャンネルの値をクランプするものではありません。 を設定します。

チャンネルエディター

チャンネルエディターは、パラメータカーブで使用される様々なチャンネルを視覚的に表示します。デフォルトでは、すべてのチャンネルがここに表示されますが、編集できるのはチャンネルリストから現在選択されているチャンネルのみで、ハイライト表示されます。 を表示します。チャンネルカーブに沿ったポイントの位置は、 をクリックし、エディタ内でドラッグすることで変更することができます。 最初と最後の点は、垂直軸に沿ってのみ変更可能で、水平方向の値は常にそれぞれ0と1になります。

チャンネルカーブに直接ポイントを追加するには、メインチャンネルラインにマウスを近づけ、カーソルが "ポイントの追加" に変わった時に をクリックします。 その後、 をクリックしてドラッグするか、チャンネルリストの拡張チャンネルオプションで値を変更することにより、これらのポイントを編集することができます。

をクリックし、エディタ内でドラッグして編集したいポイントをすべて選択するか、 / + を使ってポイントを個別に選択範囲に追加すれば、複数のポイントを一度に編集することができます。マウスボタンを離した後、選択範囲内のいずれかのポイントを再度クリックしてドラッグすると、すべてのポイントをまとめて移動することができます。

チャンネルエディタで任意の場所を右クリックして、アニメーションカーブライブラリウィンドウを開くことができます（選択されたカーブタイプがベジェの場合のみ）。この方法でプリセットを適用すると、チャンネル全体に影響します。チャンネルの一部だけを変更するには、 個々のラインを右クリックしてプリセットを適用するか、 をクリックしてドラッグして複数のラインを選択し、 選択部分を右クリックして選択したラインにプリセットを適用することができます。

チャンネルエディタからアニメーションカーブライブラリウィンドウを開くと、デフォルトで「間」適用モードが選択されていることに注意してください。

パラメータカーブの編集が終わったら、Toggle Curve Mode ボタンをクリックすると、異なるトラックの通常のdope sheet ビューに戻ることができます。
