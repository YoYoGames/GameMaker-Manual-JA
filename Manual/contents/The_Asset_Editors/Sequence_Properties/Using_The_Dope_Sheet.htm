Dope Sheetは 、sequence に追加されたさまざまなトラックのデータが時系列で表示され、各トラックの概要だけでなく、トラックの位置やパラメータの変更タイミングなどを編集する手段でもあります...

アセットトラック

アセットトラックはdope sheet 上で「キー」として表現されます。これはasset の位置とsequence でアクティブな時間の長さを表す色付きのバーです。 上述のように、これらは必要に応じて移動し、長くしたり短くしたりすることができます。asset トラックのキーがasset の長さよりも長い場合（例えば、サウンドが5フレームで、そのキーを15フレームに伸ばす場合）、そのキーの長さ全体が繰り返されます。Asset トラックとキーは、標準のキーボードショートカットでコピー＆ペーストでき、さらに右マウスボタンの メニューオプション（このページの下部で説明）で変更することができます。

asset トラックキーを削除しても、トラック自体は削除されず、追加されたトラックパラ メータも削除されず、単にasset キーがdope sheet から削除されます。このトラックのキーがもうない場合、asset ブラウザからAsset をdope sheet にドラッグして、新しいキーとして追加できます。ただし、元のasset と異なるasset を追加することはできません（例：sprite のキーを削除すると、そのキーポジションにサウンドasset を追加できません）。

object にあるSequence のトラックは、Sequence にroom の要素が作成されると同時に、そのインスタンスが作成されることに注意してください。そのasset のキーがトラック上のどこから始まるかは関係ありません。Sequence コントローラは、トラックのasset キーの位置と時間に応じて、インスタンスの可視性を非表示と表示に切り替えるだけで、インスタンスの作成と破棄を繰り返すことは ありません。その結果、インスタンスは、sequence 要素が作成されたときに Create イベントを実行し、そのアセットキーが開始されたときには実行されません。

パラメーター・トラック

パラメータトラックは、dope sheet 、所属するメイントラック（asset ）の下に色付きのバーで表示されます。この色のついたバー自体は編集することができず、純粋にパラメータキーが使用されているトラックを視覚的に表現しています。 編集するのはパラメータ・トラック上の個々のポイントです（後述の「パラメータ・キー」をご参照ください）。

プレイヘッド

プレイヘッドとは、現在timeline をプレビューしている時間的な位置のことです。この位置は、assets をシーケンスキャンバスにドロップして、追加した位置を設定するときに使用されます。 パラメータを追加したり、キーを分割したりと、さまざまなことができます。sequence エディタキャンバスで "play" を押すと、プレイヘッドがsequence の中を移動します。また、 をクリックしてドラッグすると、sequence の中の特定のフレームに移動したり、複数のフレームを "scrub" して、すばやく表示することができます。 を再生することなく、プレビューすることができます。


パラメーターキー

パラメータキーとは、パラメータトラックに表示される、パラメータに変化があった箇所の「点」のことです。これらのポイントは必要に応じて移動したり、1フレーム以上続くように拡張することができます（このページでさらに詳しく説明します）。 トラックパネルから、またはキャンバス上のasset を変更することで値を編集できます（スケール、位置、回転のパラメータの場合）。なお、すべてのパラメータトラック キーは、そのパラメータが属するアセットキートラック 上のポイントとしても反映されますが、これらのポイントは編集することができず、トラックパネルで異なるパラメータトラックを折りたたんだときの情報補助として存在するものです。


パラメーターカーブ

dope sheet では、パラメータトラックでkeyframes の代わりにパレメータカーブを 使用して、パラメータ値を変更することもできます。これらの曲線は、sequence asset の一部として埋め込むことができます。 または、パラメータトラックに適用される外部アニメーションカーブ assets を使用することができます。の項を参照してください。 アニメーションカーブの使用をご覧ください。

RMBメニュー

アセットトラックの dope sheet 内でマウスの右ボタン（ ）をクリックすると、以下のメニューが表示されます（クリックしたトラックの部分にキーがあるかないかで、表示されないオプションがあることに注意してください）。

  • カット、コピー、ペーストのキー（複数可）。これらのオプションにより、カット／コピー／ペーストを行うことができます。asset 。これらのオプションを使ってdope sheet にペーストされたキーは、プレイヘッドの位置にペーストされることに注意してください。
  • Add Key In Place（キーを所定の位置に追加する）。これは、新しいasset キーを、プレイヘッドの位置ではなく 、マウスがクリックされた場所に最も近いフレーム位置のasset トラックに追加します。
  • Split Key In Place（キーを分割する）。使用すると、マウス下のasset キーを2つのキーに分割します。分割は、再生ヘッド位置ではなく、マウスがクリックされた場所に最も近いフレームで作成されます。
  • キーを所定の位置に貼り付けます。このオプションを使用すると、コピーまたはカットされたキーは、再生ヘッド位置ではなく 、マウスの右ボタンをクリックした位置に最も近いフレーム位置にペーストされます。
  • Delete Key(s):現在選択されているキーを削除します。
  • 開始位置にコピーする。このオプションは、選択したキー（複数可）をsequence の一番最初にコピーします（ クリップ領域の開始点ではありません）。
  • コピー・トゥ・エンド。このオプションは、選択したキー（複数可）をsequence の一番最後にコピーします（ クリップ領域の最後ではありません ）。
  • StretchAsset Key:これを選択すると、選択したasset キーがsequence の終わり（クリップ領域の終わりではありません）まで、または同じトラック上の他のasset キーの始まりと重なるまで引き伸ばされます。
  • Asset キーを削除します。これを使用すると、選択されているasset キーが削除されます。その後、削除されたasset トラックにasset ブラウザからドラッグして、新しいdope sheet をトラックに追加することができます。この操作により、新しいasset キーが作成されます。このキーを編集することができ、asset に関連するパラメータトラックは維持されます。

パラメータトラックの dope sheet 内でマウスの右ボタン をクリックすると、以下のメニューオプションが表示されます（クリックしたトラックの部分にキーフレームがあるかどうか、トラックがkeyframes とパラメータカーブのどちらを使っているかによって、表示されないオプションがあることに注意してください）。

  • カット、コピー、ペーストキー（複数可）。これらのオプションにより、パラメータキーをカット/コピー/ペーストすることができます。これらのオプションを使ってdope sheet にペーストされたキーは、プレイヘッドの位置にペーストされることに注意してください。
  • Add Key In Place（キーを所定の位置に追加する）。これは、パラメータトラックに新しいパラメータキーを、再生ヘッド位置ではなく 、マウスがクリックされた場所に最も近いフレーム位置に追加するものです。
  • Split Key In Place（キーを分割する）。使用すると、マウス下のパラメータキーが2つに分割されます（keyframes ）。分割は、再生ヘッド位置ではなく、マウスをクリックした位置に最も近いフレームで作成されます。
  • キーを所定の位置に貼り付けます。このオプションを使用すると、コピーまたはカットされたキーフレームは、再生ヘッド位置ではなく、マウスの右ボタンをクリックした位置に貼り付けられます。
  • Delete Key(s)。これを選択すると、現在選択されているkeyframe またはkeyframes が削除されます。
  • 埋め込み型アニメーションカーブに変換する。このオプションは、選択されたパラメータトラック（複数可）のすべてのkeyframes を埋め込みアニメーションカーブに変換します（詳しくは「アニメーションカーブの使用」のセクションを参照してください）。また、アニメーションカーブ（asset ）を使用しているパラメー タカーブで使用すると、カーブ（asset ）が埋め込みパラメータカーブとしてトラックにコピーされます。
  • 外部アニムカーブに設定する。このオプションでは、パラメータトラックにアニメーションカーブ（asset ）を割り当てることができます（詳しくは「アニメーションカーブの使用」のセクションを参照してください）。
  • キーに変換する。このオプションで、animation curve を個々のkeyframes に変換することができます。もし曲線がcatmull-rom補間を使用していた場合、これは失われ、代わりにkeyframes は曲線上のポイント間の線形補間を使用することに注意してください。また、animation curve から生成されたキーはフレームに収まらなければならないことに注意してください。これは、フレーム間にある点を持つ曲線は、キーに変換されるときに最も近い整数フレーム値に配置され、曲線の挙動が変化する可能性があることを意味します。
  • 埋め込みアニメーションカーブをエクスポートします。このオプションを使用すると、埋め込まれたanimation curve をアニメーション カーブ アセットに変換し、Asset Browser に表示し、他の場所で使用することができます。
  • エディターを開く。これはdope sheet カーブエディタを開いて、パラメータトラックに適用された組み込みまたは外部animation curves を編集するためのものです。これは、dope sheet のToggle Curve Mode ボタンを使用するのと同じです。
  • 開始位置にコピーする。このオプションは、選択されたキーフレームをsequence の一番最初にコピーします（ クリップ領域の最初ではありません ）。
  • コピー・トゥ・エンド。このオプションは、選択されたキーフレームをsequence の一番最後にコピーします（ クリップ領域の最後ではありません ）。
  • パラメータキーの伸縮。これを選択すると、選択したasset キーがsequence の終わり（クリップ領域の終わりではありません）ま で、またはトラック上の次のkeyframe までストレッチされます。ストレッチされたパラメータキーは、ストレッチされたフレーム数だけその値を維持することを意味します。パラメータカーブで使用すると、カーブはsequence の終わりまで、またはトラック上の他のカーブに出会うまで、ストレッチされます。
  • パラメータキーの削除。選択されているパラメータキーフレームが削除されます。
