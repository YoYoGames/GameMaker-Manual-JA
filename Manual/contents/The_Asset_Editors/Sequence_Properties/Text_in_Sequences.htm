テキスト・イン・シークエンス

テキストトラックは、エディターを通してSequence に追加することができます。これは、Canvas Toolboxから Add a Text Trackボタンを選択し、キャンバス内でクリック（オプションでドラッグ）してテキストトラックを作成することで行われます。

このページでは、以下の項目について説明します。

  1. テキストの作成
  2. テキストの修正
  3. フォント資産
  4. パラメータ
  5. インスペクター

テキストの作成

テキストトラックを作成するには、2つの方法があります。ツールボックスから Textツールを選択した後、次のようにします。

  • キャンバス内で1回クリックします。これは「アンラップ」されたテキストトラックを作成します。つまり、テキストは好きなだけ長くすることができ、1行に収まります。ラッピングは、後でインスペクタから簡単に有効にできます。
  • キャンバス内をクリック＆ドラッグします。これは「ラップ」テキストトラックを作成します。つまり、テキストはフレーム内にのみ表示されます。このフレームのサイズは、テキストトラックの作成中にドラッグして設定します。このサイズは、後でパラメータトラックやインスペクタを使って変更することもできます。 ラップされたテキストトラックを使用すると、テキストの位置合わせが非常に簡単になり、アニメート中に一貫した結果を得ることができるため、使用することをお勧めします。

テキストの修正

新しいテキストトラックを作成すると、GameMaker はインスペクタを開き、自動的にテキストフィールドを選択し、テキストを書き始めることができます。

トラックのテキストをいつでも編集するには、そのトラックを選択し、インスペクタに移動してテキストフィールドを再度修正するだけです。

リサイズビヘイビア

キャンバスのテキストトラックのサイズ変更は、テキストトラックがラップされているか、アンラップされているかによって動作が異なります。

ラップされていない場合、テキスト トラックは、sprites やobjects をスケールする方法と同様にスケールされます。ただし、ラップされたテキスト トラックのサイズ変更は、そのスケール値を変更するのではなく、フレーム サイズを変更します。
ラップされたテキスト トラックを拡大縮小するには、代わりにGizmosを使用するか、Scale パラメータ トラックの値を手動で編集してください。

フォント資産

テキストトラックはFontアセットに紐付いています。新しいテキストトラックを作成すると、プロジェクト内でfont asset が検索され、最後に追加されたものが使用されます。プロジェクトにfonts がない場合、自動的に新しいfont asset が作成されます。

font を削除すると、そのfont を使用しているすべてのテキストトラックも削除されるため、誤ってfont で使用されているassets Sequences を削除しないように注意してください。

パラメータ

sprite トラックに用意されている位置、回転、スケール、オリジン、カラーマルチプライなどの通常のパラメータをすべて変更し、アニメートすることができます。これらのパラメータを使って、テキストを移動したり回転させたり、大きくしたり小さくしたり、色やアルファを変更することができます。

また、テキスト固有のパラメータも用意されており、これもアニメーション化することができる。

  • 文字間隔。各文字の間隔（ピクセル）です。
  • 行間。テキストの各行の間隔です（単位：ピクセル）。
  • パラグラフ・スケーリング（Paragraph Scaling）。テキストの各段落間のスペースです（単位：ピクセル）。新しい段落は、 を押して手動で改行を追加したときに開始されます。
  • フレームサイズ。これはテキストの枠のサイズで、インスペクタで「折り返し」が有効になっている場合のみ使用されます（詳しくは以下のセクションを参照してください）。

インスペクター

インスペクタからしか変更できないプロパティがあります。このようなプロパティは、パラメータトラックではなく、トラック全体に適用される単純なオプションであるため、アニメーション化することはできません。

テキストトラックを選択すると、インスペクタに以下のプロパティが表示されます。

注：インスペクタは、テキストトラックを作成または選択したときに自動的に開くはずですが、インスペクタが閉じている場合は、上部にあるWindowsメニューから手動で開くことができます。

パラメータトラックの一部（色、文字間隔、行間など）はここに含まれ、これらを変更すると、 Automatically record changesが有効な場合は再生ヘッドにkeyframe が作成され、選択したkeyframe が変更されます。その他のオプションは「静的」（アニメートできないため）であり、以下の通りです。

  • 上部には、選択されているfont asset が表示されます。このオプションを使って、既存のfont asset を選択するか、または新しいものを作成して、選択されたテキストトラックに割り当てることができます。
  • テキストです。テキストトラックに表示されるテキストです。ここに複数行入力できますが、空白行は表示されません。
  • 整列。ここでは、フレーム内でのテキストの水平方向（左、中央、右、両端揃え）および垂直方向（下、上、中央）の配置を変更できます。これは、「折り返し」オプションが有効な場合にのみ適用されます。
  • 折り返し。定義されたフレーム内でテキストを自動的に折り返すかどうかを制御します。これが有効な場合、テキストはフレームの端に達すると自動的に次の行に移動し、テキストがフレームの高さより長い場合は常に切断されます。この機能を無効にすると、すべてのテキストは1行で表示され、フレームはキャンバス内に表示されなくなります。
