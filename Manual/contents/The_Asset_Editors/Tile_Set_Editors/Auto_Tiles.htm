オートタイル

オートタイル機能は、トップダウンまたはサイドオンビューのプロジェクトにかかわらず、レベル構築をはるかに容易にするために使用できる、非常に強力なツールです。基本的には、オートtile ライブラリを作成します。 そのライブラリからtile 、周囲のtiles と「接続」され、シームレスな壁やプラットフォームが作成されます。

ただし、オートtile 機能を使用する前に、tile set sprite を正しく設定し、適切なオートtile の種類を選択することが非常に重要です。利用可能なタイプは、47tile autotiles 、または 16tile autotiles 、（一般的に）16tiles はトップダウンや風景に使用され（よりきれいな遷移が得られるため）、47 はプラットフォーマー/サイドオンビューや壁が複数の接続を必要とするトップダウンに使用されますが、これは は、決して厳密なルールではなく、最終的なプロジェクトをどのように見せたいかによります。
