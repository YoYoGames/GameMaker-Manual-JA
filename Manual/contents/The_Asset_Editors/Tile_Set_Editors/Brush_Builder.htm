Tile Set ブラシビルダー

デフォルトでは、tiles エディタでタイルマップレイヤーに room を「ペイント」するとき、単一のtile を選択し、それを使ってペイントします。しかし、tile sets は、ほとんどの場合、セクションを持つように設計されています。 があり、それらをさまざまな方法で組み合わせて、エリアやアイテム全体を形成します。例えば、RPGのtile set には景観の特徴tiles があり、タイルの数に応じて、木や建物など、より大きなまたは小さな特徴を作るために接続することができます。 を使用します。さて、このように複数のフィーチャーをroom レイヤーに配置すると、tile を変更するために何度も往復しなければならず、ワークフロー上よろしくない。これを解決するために、GameMakerではtile Brush を 作成することができます。 のブラシビルダーボタンをクリックすると、Tile Set エディターが表示されます。

ブラシビルダーでは、左側にオリジナルのtile set 、右側に白紙の「キャンバス」が表示されます。この状態で 左からtile を選び、右側にペイントして、カスタム「ブラシ」を作成し、room エディタで使用することができます。マウスの左ボタン をクリックしたままtile set をドラッグすると、ブラシキャンバスにペイントする複数のtiles を選択できます。また、 / + を押し続けると、特定の個々のtiles を追加でき、 + を押し続けると、特定のtiles を削除できます。

ツールボックスは、Tile Set エディタでのさまざまな作業に使用するツールを選択する場所です。 オートタイルライブラリに定義されているものがある場合。各ツールの概要は以下の通りです。（なお、tile エディタでRoom レイヤーを選択すると、このツールボックスが部屋の上部に表示されます。 workspace その他の描画ツールとともに)。

ペンシル これは鉛筆ツールです。これは選択されたtile 、マウスの左ボタン 、ブラシウィンドウで描画し、マウスの右ボタン 、消去することができますを使用します。
イレーザー 消しゴムツールでは、マウスの左ボタン を使って、ブラシウィンドウの任意のtile を消去することができます。基本的に、この操作はtile のインデックスを 0 に設定するだけです。 予約された「空」tile 。
セレクション これは選択ツールで、ブラシウィンドウの中で作業する領域を定義するのに使用します。マウスの左ボタン をクリックし、マウスをドラッグすることで、選択範囲を指定できます。 を押して長方形の領域を作成するか、 / + を押して複数の選択範囲を追加し、 + を押して選択範囲の一部をクリアすることができます。選択範囲全体を消去するには、 を押します。ブラシ・ウィンドウのある領域を選択すると、その他のツール (ペンシル、反転、回転など) は選択された領域でのみ使用できます。ウィンドウ内でtiles を選択し、コピー ( /+ "C") 、カット ( / + "X") 、ペーストすることも可能です。 ( / + " V「switch 、ツールを鉛筆に変更し、それらを使って「描く」ことができるようになります。
オートタイル このツールをクリックすると、オートタイリングのペイントスタイルが有効になります。これを有効にすると、Autotile Libraries タブから任意のtile を選択し、Brushes ウィンドウにペイントすると、GameMakerが自動的にそれを変化させます。 tiles を正しく設定している限り、周囲の Tile と一致させることができます。tile からオートタイルライブラリに含まれないtile set を選択すると、描画ツールは標準的なものにリセットされます。 鉛筆ツール
フリップ マウスの左ボタン （またはキーボードショートカット "X" ）で反転ツールをクリックすると、現在表示されているtile またはtiles を反転させることができます。 描画ツールはそのままで(カスタムブラシを選択している場合は、ブラシ全体が反転します)、横 軸に沿った描画が可能です。描画対象としてtile を選択していないが、tiles のグループを選択している場合。 を選択すると、フリップツールは選択されたtiles をフリップします。
ミラー マウスの左ボタン を押しながらミラーツールをクリックすると（あるいはキーボードショートカット " Y" を使うと）、現在のタイルがミラーリングされます。 描画ツールはそのままで、縦 軸に沿った描画を行います（カスタムブラシを選択して描画している場合は、ブラシ全体がミラーリングされます）。描画対象としてtile を選択していないが、tiles のグループを選択している場合。 を選択すると、ミラーツールは選択されたtiles をミラーリングします。
回転 マウスの左ボタン を押しながら回転ツールをクリックすると（あるいはキーボードショートカット " Z" を使うと）、現在回転しているタイルが回転します。 描画ツールはそのままに、描画対象として選択されているブラシを時計回りに90°回転させます（カスタムブラシを描画対象として選択している場合は、ブラシ全体が回転します）。描画対象としてtile が選択されておらず、tiles のグループが選択されている場合、描画ツールは変更されません。 ブラシウィンドウの回転ツールは、選択されたtiles を回転させます。

ツールの下には、2種類のセクションがあり、任意の オートタイルまたは現在のtile set の画像を使用して作成されたアニメーションタイルです。 sprite に使われる1つのtile set は、その中にたくさんのシングルセル画像を持つことができます。これらをアニメーションやオートタイルエディタで組み合わせてカスタムブラシを作成すると、これらのセクションに表示され、次のように使用することができます。 通常の静的なtiles でブラシを作成します（アニメーションtile は、ライブラリから選択したか、ベースtile set から選択したかにかかわらず、アニメーションすることに注意してください）。
