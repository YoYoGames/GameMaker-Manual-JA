このイベントは、Skeletal Animation Functions と共に使用するためにのみ設計されています。このイベントがトリガーされるのは、オブジェクト sprite object が割り当てられたsprite のデフォルトの描画を使用している場合、またはsprite が draw_self() によって描画されている場合です。現在のところ、それ以外の方法で描画されたスケルトン・アニメーションsprites ではトリガーされません。 の方法ではトリガーしません。このイベントは、sprite アニメーションを作成するために使用されるエディタで定義された、アニメーションの特定のイベントに割り当てられたデータをキャプチャして送信するために設計されています。

アニメーションイベントが発生すると、event_data map という特別なDS Mapが作成されます（非同期イベントの async_load map とほぼ同じです）。 このマップは以下のキーと値のペアを持ち、イベントに関する情報を取得するためにパースすることができます。

  • "name": イベントの名前 (アニメーションプログラムで定義されたもの)
  • "track": アニメーションが再生されているトラックのインデックス (デフォルトは 0)
  • "integer": イベントに関連する整数値 (アニメーションプログラムで定義されたもの。デフォルトは0)
  • "float": イベントに関連するフロート値 (アニメーションプログラムで定義されたもの。デフォルトは0.0)
  • "string": イベントに関連付けられたstring の値 (アニメーションプログラムで定義されたもの。デフォルトは空のstring "")
