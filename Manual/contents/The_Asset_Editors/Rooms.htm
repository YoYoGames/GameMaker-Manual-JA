Room 編集部

room エディターは、ゲームルームを作成する場所です。すべてのゲームは、少なくとも1つのroom 、実行する必要があります。room 、インスタンス、sprites 、tiles 、paths 、および背景を配置することができます。これらの異なるassets はそれぞれ独自のレイヤーに 配置することができ、レイヤーエディターで好きなように並べることができます。room エディタは複雑なため、まず最も重要な機能の概要を説明し、その後、以下のセクションのリンクからより詳細な詳細をご覧いただけます。

room リソースを作成したら、Asset ブラウザで を右クリックしてコンテキストメニューを開き、子rooms の作成（詳細は継承のページを参照）、編集用にroom を開く、新しいリソースグループを追加してrooms を整理、room の名前を変更、または削除することが可能です。room の順序や継承を変更するには、ルームマネージャーを使用する必要があります。ルームマネージャーはAsset ブラウザーの右上にあるメニューから開くことができます。

room エディタはそれ自体がワークスペースなので、タブの をクリックして、メインウィンドウからそれ自身の新しいウィンドウにドラッグすることができます。また、タブを の上部にドラッグしてマウスボタンを離すと、メインウィンドウに戻すことができます。 IDE

room エディターのユーザーインターフェイスは、操作が簡単で、さまざまな個別のセクションに分かれています。ドッキングされているエディタのこれらの部分 -ルームプロパティと レイヤーエディタ、および異なるレイヤープロパティセクション - は、単にそれらをworkspace にドラッグすることによってドックから削除することもできますし、それらをworkspace の側面または底面にドラッグすることによって再びドックに戻すことができます。

以下は、Room Editorの各セクションの概要です。

定規・ガイド

room キャンバスの縁に沿ったルーラは、その中に置かれたものの位置を示し、キャンバスの中心でありroom の原点である (0, 0) からマークされています。ルーラをクリックしてドラッグすると、水平または垂直ガイドをassets に引き込むことができ、このガイドを使って、使用中の異なるasset を正確に配置することができます。assets 内でroom を位置決めしている間、スマートガイドがassets 間の距離、およびroom 境界または中心点からの距離を一時的に表示します。

assets がガイドに「スナップ」する距離やその他のプロパティは、Room Editor Preferences で設定することができます。

レイヤーエディタ

room エディタは、room 内のレイヤーに物を配置します。各レイヤーは、個別の「深さ」にあり、レイヤーウィンドウのリストの一番下に表示されているものは、一番上に表示されているものの下に描かれます。

重要!レイヤーの深さは、最小で-16000から最大で16000まであります。この範囲外のレイヤーに配置されたものは描画されませんが、すべてのイベントは通常通り実行されます。

レイヤーは、作成したいレイヤーの種類に応じたボタンをクリックすることで作成されます。

インスタンス このレイヤーは、room で使用されるインスタンスを配置するためのものです。
タイルマップ タイルセットから tiles 、タイルマップとして配置するためのレイヤです。
パス このレイヤーは、room でパスを表示したり追加したりするためのものです。
資産 assets このレイヤーは、スプライトや シーケンスなど、room に追加されるビジュアル用のレイヤーです。
フィルター/エフェクト このタイプのレイヤーは、それ以下のすべてのレイヤーに適用されるフィルター/エフェクトを定義します。
背景 このタイプのレイヤーは背景を定義し、基本的に単色またはsprite の画像で、room 全体に敷き詰められていることがあります。
UI This creates a global UI layer that is shared among rooms.

また、フォルダボタン（ ）を使ってレイヤーフォルダを作成し、選択したレイヤーをグループ化することができます。また、削除ボタン（ ）を使って選択したレイヤーを削除することも可能です。これらのレイヤーの種類については、それぞれここで詳しく説明します。

レイヤーエディタでは、レイヤーの順序と可視性に影響する継承を切り替えることができることに注意してください。可視性は、各レイヤーの横にある目のアイコン をクリックすることで設定でき、一番上にあるアイコンをクリックすると、すべてのレイヤーを一度に有効/無効にできます。また、南京錠のアイコン をクリックすると、レイヤー（またはすべてのレイヤー）を変更できないように「ロック」することも可能です。不可視に設定されたレイヤーは、ゲーム実行時に表示されないことに注意してください。

レイヤープロパティ

room レイヤープロパティウィンドウは、レイヤーエディタで現在選択されているレイヤーによって変化します。

各ウィンドウには、現在選択されているレイヤーの種類に関連するさまざまなプロパティとリストが表示され、レイヤーの表示方法と実際のレイヤー上の内容に関する基本的な詳細を編集することができるようになります。詳しくはこちらをご覧ください。

ルームプロパティ

room エディタのこのセクションでは、room のすべての基本プロパティを定義します。たとえば、サイズ、物理学 を使用するかどうか、必要なカメラやビューポートなどです。このセクションについては、こちらで詳しく説明しています。

レイヤーツールボックス

レイヤーの種類によっては、レイヤーツールボックスの IDE の上部に追加ツールが表示されます（例：tile レイヤー、path レイヤーなど）。正確なツールは、現在編集中のレイヤーの種類によって変わるため、各レイヤーの説明ページで解説しています（こちら）。

ルームキャンバス

room エディタウィンドウの中央は、実際の編集が行われるエリアになっています。ここでインスタンスやassets を配置したり、tiles を描画したり、paths を配置したりします。マウスのホイールまたは上部にあるroom コントロールを使用してズームインおよびズームアウトすることができます。マウスの中ボタン を押しながらマウスを移動するか、移動中に + を使用すると、周囲をパンすることができます。

asset レイヤーやインスタンスレイヤーを作成する場合、asset ブラウザーからドラッグして、配置したい場所にドロップするだけで、Asset やインスタンスを配置することができます。また、asset ブラウザからAsset またはインスタンスを選択し、 を押し続けると、リソースを「その場」でプレビューできます。さらに、マウスの左ボタン をクリックすると、ボタンを押したままマウスをドラッグしてインスタンスを追加し、room レイヤに「ペイント」することも可能です。デフォルトでは、この方法でsprites に追加されたインスタンスとroom は、定義されたroom グリッド設定にスナップされますが、 / を押し続けると、インスタンスがマウスに正確に追従してスナップしなくなります（ / を解放すると再びスナップできるようになります）。

paths では、path エディタ内から新しいpath レイヤーとRoom リソースを作成できます。または、新しいpath リソースを作成し、インスタンスの場合と同様にメインエディタウィンドウにドラッグし、path とその接続もエディタウィンドウで編集できます。tile sets については、tile set エディタから「ペイント」することができます。

/ + 、同じレイヤーから複数のassets を選択して移動または削除できます。また、room エディタのほとんどのリソースでは、外側の端または角に沿って をクリックし、マウスをドラッグすると、（単一またはグループとして）それらを回転および拡大縮小することができ ます。asset (tiles, インスタンス,sprites) の上で をダブルクリックすると、エディタがその場所に移動し、選択したasset のプロパティウィンドウを開くことができます。
