レイヤーの種類とプロパティ

ルームエディタで作成するroom に入るものは、すべてレイヤーに配置されます。レイヤーはレイヤーエディターから追加、削除、ソートすることができ、様々な種類があり、それぞれにオプションと作業方法があります。各レイヤーの種類は数に制限されず、複数のtile レイヤー、path レイヤー、インスタンス レイヤーなどを持つことができます。レイヤーの種類ごとにプロパティウィンドウがあり、レイヤーの種類によって異なる設定をすることができます。また、レイヤーの可視性を切り替えることもできます。不可視のレイヤーはruntime では処理されません （ただし、適切なレイヤー関数を使用していつでも再び可視化することができます）。

上の画像はレイヤエディタで、上部に現在のレイヤが一覧表示され、下部にレイヤを作成するためのさまざまなボタンが表示されます。switch このエディタから、名前の上でゆっくりダブルクリックすることで、任意のレイヤーの名前を変更できます（名前には、文字と数字とアンダーバー「_」文字以外は使えないことに注意してください）。また、目のアイコン をクリックすると、その可視性をオンまたはオフにすることができます。room が親からレイヤーを継承している場合room 、下部にあるボタンで継承を切り替えることができますが、これはすべてのレイヤーに影響することに注意してください（継承の詳細については、こちらをご覧ください）。また、 または / を使用し、複数のレイヤーを左クリック して、複製や削除のために選択することもできます（これらのオプションは、マウスの右クリック メニューからも利用できます）。

レイヤーを整理整頓するために、フォルダアイコン をクリックして、レイヤーフォルダ内にレイヤーのグループを作成することができます。また、削除ボタン（ ）をクリックすることで、選択された1つまたは複数のレイヤーを削除することができます。すべてのレイヤーは深度 値を持ち、その内容がroom でレンダリングされる際に、そのレイヤーが描画順のどこに位置するかを定義します。レイヤーは深度の高い ものから低いものへと描画されるので、深度 -100 のレイヤーは深度 200 のレイヤーの上に 描画されることになります。

レイヤーの深さは、最小で-16000から最大で16000まであります。この範囲外のレイヤーに配置されたものは描画されませんが、そのレイヤーにあるものについては、すべてのイベントが通常通り実行されます。

任意のレイヤー上で右クリック（ ）すると、レイヤーメニューが表示されます。

ここでは、選択したレイヤーのプロパティウィンドウを開いたり、レイヤーを削除したり、レイヤー名を変更したり、サブレイヤーを追加したりすることができます。サブレイヤーを追加する場合、新しいレイヤーはその下に作成され、右側のタブに表示されます。サブレイヤーのプロパティは、親レイヤーからinherit を選択できます。また、親レイヤーからの可視性をinherit にするかどうかも設定できます。

また、 + でレイヤー間の選択、 / + でレイヤーを 1 枚ずつ選択し、複数のレイヤーを移動することができます。レイヤーをレイヤーフォルダの上に置くと、移動してドロップしたフォルダのサブレイヤーとして設定されます。また、ロックボタン を使って、間違って編集されないようにレイヤーをロックすることもできます。

背景

ボタンをクリックすると、新しい背景レイヤーを追加することができます。これにより、新しいレイヤーが追加され、背景のプロパティウィンドウが表示されます。

新しい背景レイヤーを作成すると、デフォルトでsprite が設定されていないため、GameMaker は代わりにベタ塗りをします（プレビューボックスをクリックしてカラーピッカーを開き、使用する色を変更することができます）。単純な色を使用する場合、他のレイヤー設定はゲーム内では使用されません。

代わりにsprite を選択すると、その画像を水平または垂直にtile することができ、画像をroom 全体に広げるかどうかを選択し、画像を好きなようにオフセットすることができます。さらに、背景をある方向にスクロールさせたい場合は、水平および垂直方向の速度も設定できます（1ゲームステップごとに、ここで設定したピクセル数だけ指定した軸に沿って移動します）。また、xおよびyオフセット値を設定して、背景画像の位置を変更することもできます。また、選択したsprite リソースに複数のサブ画像がある場合、背景のアニメーション速度を設定できます。ただし、デフォルトではSprite Editor で使用する画像に設定した値にロックされています。最後に、レイヤーの深さを明示的に設定できますが、これもデフォルトではロックされており、この値を変更するには、レイヤーエディタでレイヤーの順序をドラッグして並べ替えるだけです。

なお、このウィンドウでも、一番上のボタンをクリックすることで、背景の継承を切り替えることができます。

インスタンス

ボタンをクリックすると、新しいインスタンスレイヤーを追加することができます。これにより、新しいレイヤーが追加され、インスタンスプロパティウィンドウが開かれます。

このウィンドウには、現在レイヤー上にあるすべてのインスタンスのリストが表示されます。アセットブラウザを開き、そこからobjects をroom にドラッグすることで、レイヤにさらにインスタンスを追加できます。また、Asset ブラウザからインスタンスを選択し、 を押し続けて、リソースを「その場」でプレビューできます。さらに、左マウスボタン をクリックすると、ボタンを押したままマウスを移動してインスタンスを追加し、room に「描く」ことさえ可能です。

レイヤーリストは、 をクリックし、リスト内のインスタンスを必要な位置にドラッグすることで順序付けることができます。ただし、これは純粋にroom エディタ内での整理のためのものであり、特定の作成順序を設定したい場合は、ルーム プロパティウィンドウ、またはルーム コンテキスト メニューから開くことができる作成順序ウィンドウを使用してください。

をダブルクリックすると、新しいウィンドウが開き、そのインスタンスを変更することができます（room エディタ内またはプロパティリスト内）。

デフォルトでは、インスタンスは "inst_38B0BFF" のような名前になりますが、この名前は、英数字とアンダーバー "_" (およびスペース) のみを使用する限り、任意の名前に変更することが可能です。しかし、ここで指定した名前（あなたが付けた名前でも、IDE が付けたデフォルトの名前でも）は定数とみなされるので、そのインスタンスで一意 でなければならないことに注意しましょう。これは、ゲーム内のコードでインスタンスを識別するために使用できることを意味しますが、そのインスタンスと同じroom にいる場合に限ります。

インスタンス名の下には、インスタンスが親であるinherit からroom へ移動するかどうかを設定するボタンがあり、インスタンスとしたいobject を変更したり、オブジェクトエディタを開いたりすることができます（そこで何かを変更したい場合）。Object Variableウィンドウには、変数のチェックと編集を行うボタンがあり、このインスタンスの初期特性を変更したり、ブレンドする色（デフォルトでは白）を設定したり、回転や反転させたりすることができます。また、インスタンスをどちらか、または両方の軸に沿ってスケールし、room 内の位置を設定し、sprite が使用するイメージ速度と初期フレームを設定することができます。

最後に、object の作成イベントコードの後に直接実行され、このインスタンスに固有の作成コードを与えることができます（例えば、すべてのインスタンスが持つ変数をobject で設定し、作成コードでそれを修正して、1つのインスタンスに固有の値にすることができます）。これをクリックすると、インスタンスにスクリプトウィンドウが連鎖し、関数を追加したり変数を変更したりできます。このコードは、インスタンスが作成され、インスタンス作成イベントが実行された後、直接実行されます。インスタンスの継承とは別に 、作成コードの継承のオン・オフを切り替えることができることに注意してください。

レイヤー上の各インスタンスは、ゲーム作成時に自動的にエクスポートのフラグが立てられることは、特筆すべきことです。しかし、特に継承を扱う場合、特定のインスタンスを最終的なゲームのroom に追加したくない 場合があります（または、すべてのroom にデバッグインスタンスがあり、最終的なゲームから単に除外したい場合もあります）。このような場合は、リスト内のインスタンスの左側にある「Export」チェックボックスをオフにすることで、実行ファイルの一部としてエクスポートされるのを防ぐことができます。ただし、このインスタンスを参照するコードがあると、ゲームが正しく実行されないので、この機能は注意して使用してください。

タイルマップ

ボタンをクリックすることで、新しいタイルマップレイヤーを追加することができます。これにより、新しいレイヤーが追加され、タイルマップレイヤープロパティウィンドウと タイル編集 ウィンドウが開かれます。

タイルマップレイヤープロパティでは、レイヤーの継承を切り替えたり、使用しているtile set を変更したり、レイヤーのグリッドをroom の (0, 0) 位置に正確に合わせない場合は x または y オフセットを設定できます。 また、レイヤーがレンダリングする深さを設定でき（これには独自の継承ボタンがあります）、この深さも「固定」できるので、作成後の編集を防止することができます。

レイヤープロパティウィンドウが開くと、使用中のtile set を表示する「タイル編集」ウィンドウも開きます（誤ってこのウィンドウを閉じてしまった場合は、IDE の上部にある「ルーム」→「 タイル編集」メニューから再び開くことができます）。任意のtile の上で をクリックして現在のブラシとして 選択し、room エディタでroom キャンバスの上で をクリックしてレイヤーに描画します。また、複数のtiles をまとめて描画したい場合は、マウスの左クリック でグループとして選択し、ドラッグしてタイルセットから選択してから描画することも可能です。まとめて描きたいtiles の範囲がブロック内にない場合は、 / を押したまま、マウスの左ボタン で個々のtiles を選択して、この方法でカスタムブラシを作成できます ( + を使用して、カスタムブラシからtiles を削除することもできます)。この方法で作成したカスタム ブラシは一時的なもので、1 回限りの使用を想定しています。ただし、ブラシタブの編集モードを使用すると、tile set アセットに保存される恒久的なブラシを作成できます。

Tile 編集ウィンドウでは、マウスホイールまたは上部のボタンでtile を拡大・縮小でき、 マウスの中ボタンまたは + を使ってパンすることができることに注意してください。グリッドオプションもあり、オン/オフを切り替えてウィンドウ内のtiles を区切ることができます。グリッドの色も設定できますが、tiles はグリッドが表示されているかどうかにかかわらず常にスナップされます。

Tile 編集ウィンドウでは、ライブラリタブからブラシ、アニメーションタイル、オートタイルを選択することもできます。これらについては、タイルセットエディタのセクションで説明します。

room エディタの上部で、tile レイヤーを選択すると、タイルに特化したツール群が表示されます。

ペンシル これは鉛筆ツールです。これは、選択されたtile を使って、マウスの左ボタン でRoom Editor で描画し、マウスの右ボタン で消去することができます。tile のインデックス値は、Room Editor のRoom Status Bar の下部に表示されます。また、複数のtiles を選択してペイントしている場合（例：ブラシ）には、使用したすべてのtile インデックスのリストが表示されます。

ライン ラインツールを選択した場合、 をクリックしてからドラッグして放すと、クリックした位置と放した位置の間にtiles の線が作成されます。選択ツールでtile の地図レイヤーの領域を選択している場合、線は選択範囲内にのみ追加されます。線は、単一のブラシ（tile ）、さまざまなブラシの合成（tiles ）、またはライブラリにあるアニメーション（tile ）を使用して描くことができます。

フリップ マウスの左ボタン で反転ツールをクリックすると (またはキーボードショートカット "X" を使用すると)、描画ツールを変更せずに、現在描画対象として選択しているtile を水平 軸に沿って反転させます (カスタムブラシを選択している場合は、ブラシ全体が反転されます)。描画対象として選択されているtile がなく、レイヤー上で選択されているtiles のグループがある場合、反転ツールは選択されているtiles を反転させます。
ミラー マウスの左ボタン でミラーツールをクリックすると (またはキーボードショートカット " Y" を使用すると)、描画ツールを変更せずに、現在描画用に選択されているtile を垂直 軸に沿ってミラーリングします (描画用にカスタムブラシを選択している場合、ブラシ全体がミラーリングされます)。描画対象として選択されているtile がなく、レイヤー上で選択されているtiles のグループがある場合、ミラーツールは選択されているtiles をミラーリングします。

選択ツール を使って、tiles エディタキャンバスでroom のグループを選択し、カットまたはコピーすると、それらが一時的に現在のペイント用ブラシとなることに注意してください。

パス

フィルター/エフェクト

ボタンをクリックすると、新しいフィルタ／エフェクトのレイヤーを追加することができます。これにより、新しいレイヤーが追加され、asset レイヤープロパティウィンドウが開かれます。

フィルタ／エフェクト（または FX）レイヤーは、いくつかのレイヤーにビジュアルフィルタやエフェクトを適用するために使用されます。FX レイヤーを追加し、 「エフェクトタイプ」ドロップダウン メニューからエフェクトを選択すると、 このビジュアルに示されるように、 選択した FXレイヤーより下にある全てのレイヤーにそのエフェクトが適用されます。

つまり、FXレイヤーの下にあるすべての「影響を受ける」レイヤーに、選択したフィルタ（ここでは「脱色」）が適用されることになります。FXレイヤーの上にあるレイヤーは、影響を受けずに残ります。

インスペクタを使用して、シングルレイヤーFXを作成することができます。

フィルター／エフェクトのレイヤーについてより詳しく知りたい方は、「フィルターとエフェクト」をお読みください。

インスペクター

インスペクターウィンドウを開いて、room レイヤーのプロパティを検査することができ、そのプロパティを編集したり、さらにフィルター/エフェクトを適用することができます。

インスペクタを使って非FXレイヤーにフィルタ/エフェクトを適用する場合、常に「シングルレイヤー」エフェクトとして適用されます。

詳しくは、「The Inspector」をお読みください。
