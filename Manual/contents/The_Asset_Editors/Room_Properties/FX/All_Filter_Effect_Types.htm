FXの種類とパラメータ

このページでは、GameMakerに用意されているフィルター／エフェクトの種類と、各FXについて以下の情報を掲載しています。

  • 実行時にFXを作成するために fx_create() に渡すことができる内部名(例："_filter_tintfilter")。
  • 商品説明
  • 画像例
  • レイヤープロパティで編集可能なIDEパラメータ
  • ランタイムパラメーターは、runtime 、フィルターやエフェクト機能を使ってFXを修正するために使用することができます。
    • Runtime パラメーターはそのタイプについても言及しているので、どのパラメーターにどのようなタイプのデータを適用すればよいかがわかります。
