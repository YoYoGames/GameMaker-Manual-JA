画像編集ソフト

イメージエディターでは 、オリジナルのスプライトを作成したり、インポートしたスプライトを編集したりすることができます。他の画像ツールを使ったことがある人なら、すぐにわかると思いますが、GameMakerImage Editorを使う際には、いくつか注意する点があります。

  • 複数のフレームを作成すると、そのフレームに描画しながらアニメーションさせることができます!つまり、フレームコントロールバーの再生ボタン を押してから、ブラシを選択してエディタウィンドウで描画を開始すると、フレームがエディタ内でアニメーションし、あなたが描いたものがその瞬間に対応するフレームに追加されるのがわかるでしょう。気に入ったものができるまで、さまざまなフレームレートを試してみることができます。この仕組みのコツをつかめば、ダイナミックなsprites やエフェクトをすぐに作成できるようになるでしょう。
  • メインウィンドウ（workspace ）からイメージエディタタタブ を独立したウィンドウに移動することができます。これは、マルチモニタを使用している場合などに便利です。
  • コピー 機能を使って画像の選択範囲をコピーすると、コピーされた選択範囲はブラシリストに追加されます。つまり、画像の一部を選択し、それを他のペイントツールで使って、線や多角形を描いたり、自由にペイントしたりできるのです。ブラシは拡大縮小したり、色とブレンドすることもできます。貼り付け 機能は、エディタ内部で使用するためのものではありません。コピーしたものはすべてブラシリストに追加され、「貼り付け」用に選択できるようになるからです。貼り付けは、外部プログラムから何かをキャプチャして、イメージ エディタに貼り付けるために使用します。
  • 便利なレイヤーエディタにより、各sprite に複数のレイヤーを設定することができます。これらのレイヤーは順番に並べたり、フォルダーに入れたりできるので、より複雑な画像を簡単に作成することができます。
  • キーボードの"1" と"2" キーで、エディター内のサブ画像（フレーム）を素早く切り替えることができます。

これらは、イメージエディタが提供する、より高度な機能の一部に過ぎません。全ての機能を知りたい方は、以下の項目でエディタの各機能の詳細をご覧ください。

フレームビュー

この部分には、アニメートされる順番にフレームが表示されます。 timelineフレームビューの左上には、switch オニオンスキニング があります。これは、任意のsprite アニメーションの 1 つのフレームにおいて、現在編集中のイメージフレームの下に半透明のレイヤーとして、前後のフレームを表示することを選択できること を意味します。この素晴らしい点は、各フレームが前のフレームよりわずかに明るく描かれるため、次のフレームを描きながら、フレーム間でアニメーションがどのように進行しているかを見ることができることです。

をクリックしてドラッグすることで、表示する前のフレーム数と後のフレーム数を設定することができます。

このページで説明するViewconextメニューから、オニオンスキンの特定のプロパティを設定することができます。

Onion Skin ボタンの下にあるLoop Controlボタン は、sprite loops （最後のフレームに到達するとフレーム 1 に戻る）かピンポン（最後のフレームに到達するとフレームを逆行する）かを設定するために使用されます。下部の再生ボタン は、アニメーションのプレビューを開始/停止するためのものです。

選択されたフレームには、上部に Delete ボタン（ ）があり、これをクリックすると、そのフレームがsprite から削除されます。また、複数のフレームをクリックすると、 / + が一緒に選択され、その場合は Delete ボタンをクリックすると選択したフレームが全て削除されます。最後に、標準的なキーボードショートカットのカット、コピー 、ペーストを使って 、選択したフレームを編集することができます。 / + "X", / + "C", and / + "V".

フレーム情報

このバーには「フレーム情報」が表示されます。ここでは、左側の入力ボックスでアニメーションのフレームを再生する速度を設定することができます。フレームスピードは、「Frames per second」または「Frames per game frame」のいずれかを選択できます。利用可能なフレームの総数は、選択されている現在のフレームとともに右側に表示されます。このバーは、左側の アイコンをクリックすることで折りたたむ（隠す）ことができます。イメージエディターの環境設定から、常に開いた状態で起動するか、常に閉じた状態で起動するかを設定できます。

ここでアニメーションの速度に使用した値は、ゲーム内でレイヤー上のsprite sprite として、またはインスタンスに割り当てられたasset として、sprite が使用される際のアニメーションの方法も設定することに注意してください。ただし、適切な関数と 変数を使って runtime で変更することができます。

キャンバスコントロール

キャンバスコントロールは、メインの描画キャンバスとそのコンテンツの表示方法に関する様々な側面を扱います。オプションは以下の通りです。

  • Toggle Canvas Grid: キャンバスグリッドのON/OFFを切り替えます。これはGameMakerがメインキャンバスを分割するために描くグリッドで、デフォルトでは1pxのサイズに設定されています。しかし、Grid Menu アイコン をクリックすると、グリッドオプションが表示されます。
    これらのオプションでは、グリッドの色とアルファ、および X 軸と Y 軸に沿ったグリッドのセル値を設定できます。また、グリッドのスナップの有効・無効を設定することもできます（デフォルトでは無効）。
  • Canvas Zoom Controls: これらのボタンは現在のキャンバスのズームレベルを制御します。ズームイン、ズームアウトが可能で、 ボタンをクリックすると、キャンバスが編集中の画像と1:1になるようにリセットされます。また、Window Fit ボタン をクリックすると、画像キャンバス全体が現在のエディタworkspace に収まるようになります（収まるように適宜ズームイン／アウトされます）。 / とマウスホイール を使ってもズームイン、ズームアウトできます。 / + を押すと、room キャンバスが画像と 1:1 になるように設定されます。
  • キャンバスの分割:これをクリックすると、キャンバスを水平方向に分割するか、垂直方向に分割して1つのキャンバスとするかを切り替えます。キャンバスの分割表示では、それぞれのキャンバスにキャンバスコントロールがあり、例えば一方を拡大し、もう一方を縮小することができ、どちらかのキャンバスを編集すれば、その変更がもう一方のキャンバスに反映されます。キャンバス表示の切り替えを変更するには、「分割オプション」 ボタンをクリックして、使用したい方を選択するか、「キャンバスを分割」ボタンをクリックして、使用したい方に切り替えるだけでよいのです。

ドローキャンバス

画像の主な編集ウィンドウです。ここでは、右側のツールのいずれかを使用して、現在選択されているレイヤーにペイントを行うことができます。マウスの左ボタン（ ）または右ボタン（ ）でペイントし、カラーピッカーでそれぞれの色を指定します。また、エディタの右上にあるブラシを使用することもできます。また、 / とマウスホイール を使って、ズームインまたはズームアウトできます。 / + を押すと、キャンバスが画像と 1:1 になります（または、上記のキャンバスコントロールを使うこともできます）。また、マウスの中ボタン または + を使って、キャンバスの周りをパンすることができます。

ブラシ

ここでは、ペイントに使用するブラシを 選択することができます。デフォルトでは、さまざまなサイズの四角や丸のブラシが用意されていますが、画像レイヤー上でペイントし、新しいブラシに使用したい部分を選択して / + "C" でコピーすれば、簡単に独自のブラシを作成できます。コピーされた部分はブラシに追加され、他のブラシと同じようにペイントすることができ、これには線を引くなどのさまざまなペイントツールを使用できます。

なお、新しいブラシのデフォルトのブレンドカラーは白ですが、カラーピッカーで色を変更し、そのブラシを使ってペイントすると、この新しい色でブレンド されることになります。リストからブラシを削除したい場合は、そのブラシをクリックし、 キーを押すだけで、削除されます。

注：ブラシはセッション間で保持されません。プロジェクトを終了すると、作成したブラシはすべて削除されます。

カラーピッカー

このセクションの上部には、選択可能ないくつかの定義済み色見本があり、どちらかのボタンでクリックすることで、マウスの左 と右 にそれぞれ割り当てられます（色見本の横にある矢印アイコンをクリックすると、各ボタン間で色を入れ替えることもできます）。選択された色は、下のマウスの画像の横に色見本として表示され、これをクリックするとカラーピッカーウィンドウが開きます。

このウィンドウでは、利用可能なスライダーやガジェットを使って色を定義したり、ARGB値やHSV値の値を直接入力したりすることができます。

CTRL/CMD を押しながら をクリックすると、カラーピッカーツールが表示され、sprite のデフォルトパレットを編集できます。この場合、カラーピッカーで「OK」をクリックすると、パレットに新しい色が設定され、編集中のsprite asset と共に保存されます。

ツール

以下に、GameMakerでデフォルトで利用できるツールの一覧を示します。すべてのツールは、マウスの左 と右 のボタンで使用でき、そのほとんどにキーボードショートカットがあります（重要なものはツールとともにここに記載されていますが、全リストについてはショートカットのページを参照してください）。ほとんどのツールには、下図のように画像エディタの上部に表示される追加オプションもあります。

ペンシル これは鉛筆ツールです。これは、選択されたブラシを使って、マウスボタンに選択された色で描きます。画像エディタのトップバーからサイズを設定したり、スムージング（補間）の有効/無効を設定したりできます。また、 + "Z" を押しながら、 をクリックし、マウスをドラッグしてブラシの比率を変更したり、右上の定義済みブラシを選択することで、任意のブラシのサイズやスケールを変更することができますのでご注意ください。
イレーザー 消しゴムツールです。選択したブラシを使って、マウスのどちらかのボタンをクリックすると、その部分を消すことができます。これを選択すると、エディタのトップバーに、ブラシの拡大縮小やスムージング（線形補間）の有効・無効を設定するためのツールが追加されます。
塗装・塗りつぶし これは、ペイントバケツ（または塗りつぶし）ツールです。これは、エディタのトップバーに表示されるスライダを使って変更できる選択されたアルファ許容度設定に基づいて、画像の領域を塗りつぶします。
カラーリムーブ これは色の除去ツールです。これを選択すると、画像内の任意のピクセルで または をクリックすると、同じ色のすべてのピクセルが削除されます (技術的には、アルファ値 0 の黒に置き換えられます)。この値が低いほど、クリックした色と正確に一致しなければならず、値が高いほど、クリックした色に類似した色も含まれるようになります。選択ツールで選択した場合は、選択範囲内のピクセルのみをクリックすると、選択範囲内のピクセルだけが削除されます。
カラーリプレイス これは色の置き換えツールです。これを選択すると、画像内の任意のピクセルで または をクリックすると、同じ色のすべてのピクセルが、使用したマウスボタンに割り当てられた色に置き換えられます。上のバーには許容範囲スライダーがあり、ピクセルの色の許容範囲を設定できます。この値が低いほど、クリックした色と正確に一致しなければならず、値が高いほど、クリックした色と類似した色も含まれるように拡大します。選択ツールで選択した場合は、選択範囲内のピクセルのみをクリックすると、色の変更は選択範囲内のピクセルにのみ適用されます。
ライン このツールは、2点間に線を描きます。どちらかのボタンで一度クリックして最初の点を作成し、別の場所でもう一度クリックすると別の点が作成され、それらが結合されます。線は、選択したブラシを使って輪郭を描きます。エディタのトップバーを使ってブラシのプロパティの一部を編集できます（サイズ、線形補間など...)。
長方形 このツールは、塗りつぶしまたはアウトライン化された長方形を描画します。アイコンの左上をクリックするとアウトライン化され、右下をクリックすると塗りつぶされます。両方が必要な場合は、イメージエディタのトップバーからそのオプションを選択し（または + でツールボックスアイコンのどちらかを選択し、もう一方を選択します）、使用するブラシのサイズとスムージングを設定することも可能です。選択したブラシで矩形が描かれます。なお、描画中に を押しながらクリックすると、マウス位置の中心から描画されます。
楕円 このツールは、塗りつぶされた楕円と輪郭のある楕円のどちらかを描きます。アイコンの左上をクリックすると輪郭が描かれ、右下をクリックすると塗りつぶされます。両方が必要な場合は、イメージエディタのトップバーからそのオプションを選択し（または + を使ってツールボックスアイコン上で一方を選択し、他方を選択します）、使用するブラシのサイズとスムージングを設定することが可能です。楕円は、選択したブラシを使って輪郭を描きます。描画中に を押しながらクリックすると、マウス位置の中心から描画されることに注意してください。
ポリゴン このツールは、塗りつぶしまたはアウトライン化されたポリゴンを描きます。アイコンの左上をクリックするとアウトライン化され、右下をクリックするとフィルドになります。両方が必要な場合は、イメージエディタのトップバーからそのオプションを選択し（または + でツールボックスアイコン上で一方を選択し、他方を選択します）、使用するブラシのサイズとスムージングを設定できます。 または をクリックして、選択したブラシを使用してポリゴンにポイントを追加します。結果に満足したら、別のツールを選択して変更をコミットし、画像を生成する必要があります。ポリゴンは、選択したブラシを使って描かれます。ポリゴンの定義をキャンセルするには、いつでも を押すことができます。
スプライン スプラインツールです。選択されたブラシを利用してスプラインを描きます。マウスのどちらかのボタンをクリックしたままポイントを設定し、マウスをドラッグしてスプラインを生成し、マウスを放してポイントやカーブを追加することを繰り返します。スプラインは選択されたブラシで描かれ、エディタのトップバーでサイズやスムージングを設定することができます。トップバーには「Iterations 」の設定もあり、出来上がったスプラインの滑らかさを定義します（低い値はより荒く、高い値はより滑らかになります。）
テキスト
テキスト入力ツールです。新しいテキストレイヤーを作成し、そこに必要なものを入力することができます。イメージエディターのトップバーから、使用するfont と書式を設定できます。
点眼器 スポイトツールは、現在編集中の画像から色を抽出するツールです。

回転 回転ツールです。選択ツールで領域を選んでいるときはこれをクリックするとその領域を 0° から 360° の範囲で任意に回転させられます。画像に選択範囲がないときは、このツールで現在のブラシを回転させます。回転させるには、マウスの左ボタン を押しながら左右にドラッグするか、トップバーで直接値を設定します。変更を受け入れるには、 を押すか、他のツールを選択します。変更を受け入れると、ソース素材は変更されず、選択した画像または設定した量だけ回転したブラシで使用するための新しいブラシリソースが作成されることに注意してください。また、キーボードショートカット "Z" は、選択範囲やブラシを90°回転させるのに使用します。このオプションは新しいブラシを作成せず、現在のブラシの描画方法にのみ影響することに注意してください。

レイヤーエディタ

イメージエディタでは、画像のさまざまな部分にレイヤーを 作成でき、これらのレイヤーは、さまざまなブラシやツールを使って操作したり、描画したりすることができます。新しいレイヤーを作成するには、 アイコンをクリックするだけです。また、削除するには、 アイコンを再度クリックします。また、 をクリックしてレイヤーを上下にドラッグすると、レイヤーを並べることができます。さらに、 ボタンを使ってフォルダを作成し、その中にレイヤーをドラッグすると、レイヤーを整理することができます。レイヤーの をダブルクリック（または を使用し、レイヤーのプロパティを編集を選択）すると、レイヤーのプロパティウィンドウにアクセスすることができます。

ここでは、レイヤーの名前、ブレンドモード（下のレイヤーと「混合」する方法）を設定し、通常、加算、減算 、乗算の 中から選択することができます。最後に、スライダーでレイヤーの不透明度（透過度）を設定できますが、これは個々のピクセルの実際のアルファ値と累積されることに注意してください。例えば、50%の不透明度を持つピクセルがあり、レイヤーの不透明度を50%に設定すると、最後のピクセルは25%の不透明度で描かれます。

これらのメニューについては、以下の項目で説明します。

画像

画像] メニューには、次のオプションがあります（現在のエディタのフォーカスにより、利用できるものとできないものがあります）。

  • フレーム／選択範囲を カット - エディタのフォーカスによって、イメージからフレームまたは選択範囲をカットします。選択範囲をカットする場合、これはブラシに追加されます。
  • フレーム／選択範囲を コピー - エディタのフォーカスによって、イメージからフレームまたは選択範囲をコピーします。選択範囲をコピーする場合、これはブラシにそれを追加します。
  • フレーム/選択範囲の 貼り付け - フレームはクリップボードから、選択範囲はクリップボードから画像に貼り付けます。これはIDE の現在のフォーカスによります。画像に貼り付ける場合、このオプションは外部プログラムからのクリップボードデータにのみ使用するように設計されていることに留意してください。イメージエディターで画像からクリップボードにカットしたりコピーしたりすると、 エディターの右上にあるブラシにこれが追加されるので、 画像に貼り付けようとするのではなく、 カット/コピーした選択範囲でブラシを選択して描画する必要があるのです。また、クリップボードで使用されるフォーマットのため、多くのプログラムでクリップボードからの透過画像をGameMakerのイメージエディタにペーストする際に問題が発生することに注意してください。
  • すべて選択 - 画像全体を選択します。
  • 選択範囲を解除する - 画像から選択範囲を解除します。
  • 選択範囲の反転- 現在の選択範囲を反転させます。
  • Add Frame-sprite に新しいフレームを追加します。
  • フレーム挿入- 現在表示されているフレームの後に新しいフレームを挿入します。
  • Delete Selected Fames- 選択したフレームをsprite から削除します（複数のフレームを選択するには / + and/or + を使用します）。
  • 画像の読み込み- フレームとして使用する画像をディスクから読み込みます。エクスプローラウィンドウでインポートするファイルを個別に選択することで、一度に複数のフレームをロードすることができることに注意してください。ファイルを選択すると、次のようなインポートウィンドウが表示されます。
    ここでは、インポートした画像を現在のsprite にどのように追加するかを選択でき、描画キャンバス全体のサイズ変更、クロップ、パディング、また新しい画像を配置する「アンカー」位置の設定が可能です。
  • フレームに変換- このオプションは、単一のイメージをいくつかの個別のフレームに変換します。まず、下図のようなアニメーションのすべてのパーツを含む単一のフレームを作成（または単一のストリップ画像をインポート）する必要があります。
    これは単一の画像であり、個々のフレームに変換したいので、このオプションを使用することができ、次のウィンドウが表示されます。
    ここでは、フレーム数、フレームの幅と高さ、およびオフセット値を設定して、フレームを作成するために単一の画像を分割する方法を設定することができます。
  • ストリップ画像のインポート- このオプションは、ファイルエクスプローラを開き、フレームを作成するために使用する単一のストリップ画像ファイルを探すように要求します。画像を選択すると、インポートストリップウィンドウが開きますが、これは上記のフレームに変換オプションと全く同じです。
  • Export to PNG- このオプションを選択すると、sprite が1つの PNG イメージとしてエクスポートされます。sprite に複数の画像がある場合、すべての画像を順番に並べた「帯」sprite が作成され、保存ファイル名には自動的に "_stripX" が追加されます。"X" はエクスポートされるフレームの数です。例えば、5フレームsprite は、"sprite_name_strip5.png"としてエクスポートされます。
  • Resize All Frames- このオプションを選択すると、次のウィンドウが開き、スプライトを構成するすべてのアニメーションフレームのサイズを変更する方法を設定することができます。
    イメージを拡大縮小 するか、キャンバスをリサイズするか、どちらかを選ぶことができます。画像の拡大縮小を選択した場合、各フレームの新しい幅や高さ、同じアスペクト比を使用するかどうか、またピクセル単位かパーセント単位かを選択するだけです。スケーリングするとき、線形補間をオンまたはオフにすることができます。オンにすると、画像をスケーリングするときにピクセルがぼやけたり、「スミア」が発生します（一般に、高解像度アートではオンに、ピクセルアートではオフにすることをお勧めします）。

    キャンバスのサイズを変更するオプションは、画像内のピクセルを変更するのではなく、画像が描かれているキャンバスのサイズを大きくしたり小さくしたりします。矢印のアイコンを使ってサイズ変更の「アンカー」を設定し、新しいサイズ（ピクセルまたはパーセント）を設定し、さらに元のフレームのアスペクト比を維持するかどうかを設定できます。フレームを元のsprite の画像より小さくすると、ピクセルが切り取られることに注意してください。
  • Crop All Frames To Selection- このオプションは、メインの描画キャンバスで画像の領域を選択しているときのみ利用可能です。このオプションを選択すると、編集中の画像（およびアニメーションの他のすべてのフレーム）は、選択範囲の境界線にクロップされます。
  • Auto Trim All Frames- このオプションは、sprite のエッジからすべてのゼロアルファピクセルをトリミングします。これはsprite 内のすべてのイメージフレームを考慮するので、各サイドは、最も遠い非ゼロアルファピクセルがあるフレームのエッジにクロップされます。
  • Reverse Frames- これを選択すると、sprite のフレーム順が単純に逆になります。
  • Mirror - このオプションは、sprite を構成するすべてのフレームを垂直軸に沿ってミラーリングすることを可能にします。
  • Flip - このオプションは、sprite を構成するすべてのフレームを水平軸に沿って反転させることができます。
  • すべてのフレームを回転（時計回り90°）- このオプションは、アニメーションのすべてのフレームを時計回りに90°回転させます。
  • すべてのフレームを回転（反時計回りに90°）- このオプションは、アニメーションのすべてのフレームを反時計回りに90°回転させます。

表示

表示］メニューには、次のオプションがあります。

  • 1:1- 画像を解像度と1:1になるようにスケーリングし（つまり実寸）、画像編集ウィンドウの中央に配置します。
  • 画面に合わせる- 画像を画像編集ウィンドウいっぱいに拡大縮小します。
  • グリッドを表示- ピクセルグリッドをトグルします。これはキャンバスコントロールにあるグリッドボタンと同じです（このページでさらに詳しく説明します）。
  • オニオンスキン設定- オニオンスキンの設定を変更します。以下のウィンドウが表示され、オニオンスキンボタンで表示されるフレームや、スキンされたフレームの不透明度や色などを設定することができます。

効果

エフェクトメニューは、独自のエフェクトプラグインを追加するためのもので、デフォルトで以下のエフェクトが用意されています。

  • ブラー - イメージにブラー効果を適用します。強度の設定、使用するぼかしの種類（ボックスまたはガウス）、水平軸または垂直軸に沿ったぼかしのみか、または両方に沿ったぼかしを設定することができます。また、現在のレイヤーのみ、表示されているレイヤー、またはすべてのレイヤーに適用するように選択することができます。
  • グレースケール - 指定された量だけイメージを脱色します。この処理は、現在のレイヤーのみ、表示されているレイヤー、またはすべてのレイヤーに適用することができます。
