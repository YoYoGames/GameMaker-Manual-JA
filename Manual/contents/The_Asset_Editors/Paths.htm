Path 編集部

ゲーム作成中に、インスタンスの1つがレベルを通してpath を追跡する必要がある場合があります。この場合、room 内に位置の配列を作成し、インスタンスがその間を移動することでコード化できますが、セットアップが面倒で、インスタンスの変更、テスト、適応が困難です。このためGameMakerでは、path のリソースと専用のエディタ（path ）を用意しています。

基本的な考え方はとてもシンプルで、path をpath エディタで描画して定義し、object の作成イベントにアクション（またはコード）を配置すると、インスタンスがその特定のpath に従うように指示します。また、path に従う速度を設定したり、path 内のroom の位置や向きに関連する他の多くのアクションを設定できます。 最初にpath リソースを作成すると、path エディタウィンドウが次のオプションで開きます。

パスプロパティ

GameMakerのすべてのリソースと同様に、新しいpath にもユニークな名前を付けてください。この名前は、作成中のゲーム全体でこのリソースを識別するために使用されます。 コードやアクションで認識しやすいように（例： path_Enemy_1 や PowerUp_path ）。

path という名前をつけたら、右側のメインエディタでポイントを配置して定義を始めます。各ポイントは、位置と速度とともに、ここのリストに追加されます。path ポイントのスピード値は、パーセンテージで定義されます。 インスタンスをpath に沿って移動させる速度のことです（デフォルトは100%）。つまり、path に沿って速度 2（1ステップにつき2ピクセル）でインスタンスを開始し、path ポイントの 1 つが速度値 50% の場合、インスタンスは速度を下げます。 に近づくと、速度が1（path の50%）になります。同様に、速度を150%に設定すると、インスタンスはポイントに到達するにつれて加速します。このように、コード無しでダイナミックな動きを作ることができます。 を使用します。

特筆すべきは、リスト内の定義されたポイントのいずれかを高速でダブルクリック（ ）することで、エディタをパンすることができることです。 または、値の1つをゆっくりダブルクリック（ ）して手動で編集します（x、y、または速度）。

また、path 全体の接続の種類と、path を閉じるかどうかを指定できます (path を閉じると、path 上の最後の点と最初の点が接続されます)。接続の種類は、直線、またはスムース です。 直線接続では、path の各点を直線的に接続し、各点で角度を変更します。一方、滑らかな接続では、どの点も直接通過せず、むしろ以下の点を取る、より柔らかいpath を作成します。 スムーズな補間で美しいカーブを描く。また、曲線の精度（デフォルトは4）も指定でき、1〜8の間で設定可能です。 が最も滑らかです。下の画像は、これらのオプションがパスに与える影響の一例です。


パス編集

path 編集ウィンドウの任意の場所をクリックすると、ポイントが追加されます。エディタに配置されたポイントは、以下の方法で移動できます。 をクリックしてドラッグするか、左側に表示される X と Y の位置と速度の値を変更します（前出の「パスのプロパティ」を参照）。また / を使い、複数のポイントを左クリック でまとめて選択すると、このポイント群をドラッグして好きなように配置することができます。 を押すと、選択したポイントを削除することもできます。

エディタウィンドウで を右クリックすると、以下のような追加メニューが用意されています。

このメニューの各項目は、次のように動作します。

  • Delete Points- 選択されたポイントを削除します。
  • Pan To Path- エディタ内のビューをpath の中心にパンします。
  • Reverse-path ポイントの順序を逆にします。これは移動するのではなく、処理される順番を変更し、開始点が終了点に、またはその逆が行われます。
  • Flip-path を水平軸に沿って反転させます。
  • Mirror-path を垂直軸に沿って反転させます。

ツールボックス

上部にあるツールボックスのボタンは、エディタウィンドウでの描画をコントロールするためのものです。背景のグリッドのオン・オフを切り替えたり、path ポイントの追加をこのグリッドにスナップさせるかどうかを設定したりできます。

スナップの値は、グリッドセルの大きさを示します。 この値は任意に変更することができます。残りのツールは、エディタエリアのズームイン、ズームアウト、パン専用で、 ボタンをクリックすると、ディスプレイと 1:1 になるようにリセットできます。path を描画したい場所からパンしすぎた場合は、マウスの右ボタンで メニューを表示し、「Pan to Path」をクリックして作業していた場所までスクロールして戻れます。


パスポイント

パスとは、単純に線で結ばれた点の集まりのことです。エディタでは、マウスの左ボタン を使って、これらの点を配置します。room エディタ内のどこにでもポイントを追加でき、それらはポイントリストに連続して追加され、path を作成します。 接続線や他のノードの近くにポイントを配置すると、新しいポイントはpath に追加されるのではなく、挿入されます。 この検出距離は環境設定で設定できます）。 / を押しながら、すでに配置されているポイント上で をクリックして選択すると、path のポイントに対して行った移動などの操作は、すべてのポイントに適用されます。 をグループとして選択します。ポイントまたは選択したポイントを削除するには、 を右クリックして「Delete ポイント "をクリックするか、 キーを押してください。

なお、以下のショートカットを使用すると、選択したpath ポイントのマウスの右ボタンメニューでは利用できない、いくつかの追加操作を行うことができます。

  • T +:path のポイントをトランスレート（移動）します。
  • R +:path のポイントを回転させます。
  • S +:path のポイントをスケーリングします。
