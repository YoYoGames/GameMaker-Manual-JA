Marketplace 資産の作成

IDE Marketplace Menuから Create Packageオプションを選択すると、次のワークスペースが表示されます: こちらまず、asset パッケージを公開するパブリッシャー ID を選択し (パブリッシャーダッシュボードで設定済み)、表示名と バージョン番号を指定する必要があります。パッケージ ID は表示名とパブリッシャーアカウントに基づいて作成されますが、逆 URL 形式を維持する限り、他のものに編集することができます。次に、作成するasset パッケージの種類を選択する必要があります。

  • アセット：マーケットプレイスで販売を希望するassets (sprites,scripts,objects, etc...) のコレクションです。
  • デモです。コンセプトやゲームのアイデアを実証するためのデモです。
  • チュートリアルです。アップロードされるasset はチュートリアルで、GameMaker IDE (詳しくはこちらのページをご覧ください) の中で実行するように設計されています。

右側のセクションで、プロジェクトに含まれるassets の中からAsset パッケージに追加したいものを選択します。assets をすべて追加することも、一部だけ追加することも、間違えて追加してしまったものを削除することも可能です。アップロードするasset は、あらかじめGameMakerで作成しておき、現在のプロジェクトにアップロードに必要なものがすべて含まれているものとします。

覚えておいてください。 asset パッケージを作成または使用するには、事前にユーザープロファイル証明書を作成しておく必要があります。これは、Marketplace Preferencesから行うことができます。これについては、こちらで説明しています。

assets の選択に問題がなく、パッケージをMarketplace にアップロードする場合は、下部にある「OK」ボタンをクリックするだけで、asset パッケージが作成され、Marketplace にアップロードされます。アップロードが完了すると、新しいworkspace が開き、編集用のasset ページがブラウザで表示されます。設定方法の詳細については、以下のページをご覧ください。

  • 資産上場

Marketplace Asset を更新する必要がある場合、元のパッケージを作成したときとほぼ同じ操作を行いますが、利用可能なパッケージのドロップダウン・リストから更新するパッケージを指定する必要があります。

更新の際、パッケージ ID を変更することはできません。また、表示されるVersion Numberは現在の バージョン番号であり、適宜インクリメントしないと更新に失敗します (同じバージョン番号のアセットパッケージを2つ持つことはできないため)。また、asset の変更履歴のテキストを追加できる欄があります。これはasset と一緒にアップロードされ、Marketplace ページに表示されます。

重要！ GameMakerバージョン 2.3+ で作成されたAssets は、バージョン 2.3 以上を使用しているユーザーにのみ提供されます。以前のバージョンからasset をアップデートする場合、そのasset はアップデートしていないユーザーも利用可能で、アップデートした 2.3+ バージョンと共存します。

Webページからのアップロード

assets ではなく、実際のMarketplace ウェブページからIDE をアップロードできます。これは、IDE からのアップロードが何らかの理由で失敗した場合に必要なことです。まず、パッケージを作成し、上記のようにIDE からアップロードを試みます。失敗した場合は、出力ウィンドウでログをチェックし、アップロードを試みる前に作成したパッケージがどこに保存されたかを示す行を確認します。例えば、以下のようになります。

Windowsの場合。

Created package file:
C:\Users\WINDOWSUSERNAME\AppData\Roaming\GameMaker\GM_USERNAME\Marketplace\Packages\com.publishername.assetname.yymp

macOSの場合。

Created package file:
/Users/MACUSERNAME/.config/GameMaker/GM_USERNAME/Marketplace/Packages/com.publishername.assetname.yymp

そのファイルの場所に移動して、 *.YYMP ファイルを安全な場所にコピーする必要があります。これは、Marketplace にアップロードするファイルになるからです。

この時点で、ウェブブラウザでMarketplace を開き、asset のasset のリストに移動し、下の画像に示すように、バージョンセクションに移動し、アップロードバージョンのボタンをクリックする必要があります。

このページでは、アップロードするバージョンに関する情報を入力したり、「Choose FIle」ボタンをクリックして作成した *.YYMP ファイルを選択することで、ファイルを提供することができます。

ファイルを追加すると、バージョンがアップロードされ、「進行状況」バーが表示されます。アップロードが完了し、詳細が確認できたら、「保存」ボタンをクリックして、asset をリストに保存することができます。なお、asset は、公開されるまでに数分間、処理段階にあります。
