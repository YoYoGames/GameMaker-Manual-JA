シーケンス

このガイドで最後に説明するasset は、Room Editor でどのように組み立てるかを説明する前に、Sequencesアセットを説明します。sequence は、基本的に他のassets を一箇所に集めて、一緒にアニメートしたものです。最終的なアニメーション(sequence)は、インスタンスと同様に、ゲーム内のrooms に追加することができます。

シーケンスの作成は シーケンスエディターアセットブラウザで新しいsequence asset を作成するたびに開かれます。

基本的なsequence を作成するには、サウンド、sprite またはインスタンスをシーケンスキャンバスに追加し、sequence のスタート地点に配置します。 アセットトラックをトラックパネル に、またa を追加します。 キーフレームをドープシートで表示し、asset が表示される時間を表示します。

sequence の時間はフレーム（インスタンスのステップと同じ、つまり1フレームは1ゲームステップと同じ）であり、任意のフレームで何が起こるかは、シーケンス プレイヘッドdope sheetプレイヘッドの位置は、キーフレームを 追加するためにも使用されます。パラメータキーは パラメータトラックを指定した場合、asset 。

そこで、asset の位置を決め、sequence のフレーム 0 でそのスケールと色、その他のプロパティを設定します。次に、asset のパラメータキーを記録し、次のasset の状態（位置、色など）を設定するフレームに再生ヘッドを移動させます。このフレームで、再び異なるパラメータを変更して、そのキーを記録し、このプロセスをsequence の長さ分、必要なだけ繰り返します。sequence の再生を押すと、asset がアニメーション化し、sequence が異なるパラメータの状態を補間して、最終的にこのような結果になります。

sequences がどのように機能するかについてはここでは詳しく説明しませんが、カットシーン、メニュー、キャラクターの状態など、あらゆることを行うための非常に強力なツールであることは十分承知しています。
