タイルセットの作成

タイルセットは sprites から作成されますが、GameMakerの扱い方がasset ともsprites とも少し異なるため、別のobjects タイプとして分類されます。つまり、tile set は、tiles の集合体として作成されます。 sprite tile の一部としてGameMaker room に追加されるグラフィカルなリソースです。 タイルマップ.これらのtile マップは、次に描画されます。 CPU/GPUの処理をほとんど必要としないため、レベル内の静的なアイテムのデザインに最適です。

tiles を高速に処理するために、いくつかの制限があります。最も基本的なものは、長方形か正方形しか使えないということです（これ以外のものが必要な場合は、通常のスプライト で、room エディタでアセットレイヤーを使用します。これについては、後のルームのセクションで説明します）。その他の制限としては、90度回転、ミラーリング、反転が可能なこと、そして 2の累乗（2、4、8、16など）のフレームでしかアニメートできない。
