GameMakerで作るすべてのゲームは、実行するために少なくとも1つのroom （ただし、もっとたくさん持つことができます）。room は、ゲームを構成するobjects のインスタンスと、tiles やその他のグラフィックリソースを置くスペースに過ぎないのです。 アセットブラウザで room を初めて作成すると、新規の ルームエディタworkspace で、そのプロパティを編集することができます。 のようなものです。

デフォルトでは、room ツールが左側に配置され、3 つの主要なセクションがあります。

  • レイヤーエディター。room のすべてはレイヤー上に配置され、選択できるさまざまなレイヤーの種類があります（これについては後で説明します）。レイヤーは深さによって 並べられ、この深さがレイヤーの順番を決めます。 レイヤーの内容は、ゲーム実行時に画面にレンダリングされます。深度の順序は最高から最低までで、深度が低いほど「カメラ」に近く、深度が高いほど遠くなります。例えば、レイヤーの深度が 深度-300のレイヤーは深度-100のレイヤーの上にレンダリングされ、深度1000のレイヤーはこれより小さい深度のすべてのものの下にレンダリングされます。デフォルトでは、GameMakerはレイヤーの深さを位置によって並べ替えます。 のレイヤーエディターで、マウスを使ってレイヤーをドラッグして並べ替えると、GameMakerがそれに合わせて深さを並べ替えます。
  • レイヤープロパティ。room に追加した各レイヤーには独自のプロパティがあり、それらのプロパティはレイヤーの種類によって変化します。このウィンドウでは、これらのプロパティを変更し、レイヤーの表示方法を編集することができます。 レンダリングされます。
  • ルームプロパティ：Rooms にも様々なプロパティがあり、ここで設定することができます。room のサイズ、アクティブにするカメラビューポート、その他いくつかの項目をここで設定することができます。

レイヤーには様々な種類があることを説明しましたが、その種類とゲーム制作時の使い方を説明します。

背景レイヤー 背景レイヤーは、単色 または単一の 画像で塗りつぶすことができるレイヤーです。これは、room 内で移動および配置することができ、複数の背景レイヤーを持つことができます。一般的にこれは、その名が示すように、使用されます。 room の他のすべてのレイヤーのために一定の背景を生成します。デフォルトでは、新しいroom は常に背景レイヤーを含みますが、必要ない場合は削除することができます。
インスタンス層 インスタンスレイヤーは、ゲームに必要なobjects のインスタンスをすべて配置する場所です。インスタンスをレイヤーに追加するには、 をクリックするだけで、object が表示されます。 asset ブラウザを起動し、room エディタworkspace にドラッグします。マウスボタンを離すと、そのobject のインスタンスがroom に追加されます。 デフォルトでは、新しいroom は常にインスタンスレイヤーを含みますが、それを削除することもできます。 が必要です。
タイル地図レイヤー タイルセットを作成した後、tiles をroom に追加する必要があります。これは、タイルマップレイヤーを作成することによって行われます。tile マップレイヤーは、作成したtiles tile set からassets を追加できるレイヤーで、tile set のセルサイズのグリッドを使用するように自動的にセットアップされます。各グリッドセルには1つのtile しか追加できないので、同じスペースを占有するために複数のtiles が必要な場合は、複数のtile マップレイヤーを使用する必要があります。
パスレイヤ path レイヤーは、ゲーム実行時に実際にレンダリングされない唯一のレイヤーです。このレイヤーは、実際のpaths スペース内でroom を追加または編集できるため、Path アセットの作成または編集を支援する「便利」なレイヤーです。
アセットレイヤー 素敵なグラフィック効果が欲しいけれども、インスタンスを使用するオーバーヘッドが必要でない場合や、インスタンスが自分自身を描く以外のことをする必要がない場合があります。tile マップを使うこともできますが、tile マップの使用はグリッドに限定され は、1つのグリッドセルにつき1枚の画像であるため、少々制約があります。そんなときは、asset レイヤーを使ってください。asset sprite レイヤーは、単にroom リソースを取り込んで、Sprites に追加したときに設定したパラメータを使用して描画します。 sequences は、objects のインスタンスをインスタンス層に追加するのと同じ方法で、このレイヤーに追加できます。つまり、sprite ブラウザでasset の をクリックし、次に room エディタworkspace にドラッグしてください。
Filter/Effect Layer This layer applies a filter or effect to all the layers below it. Note that a single-layer effect can be applied to any layer via The Inspector.
UI Layer This creates a global UI layer that is shared among rooms.

room エディタにインスタンスを追加するには、まず左側のレイヤーエディタからインスタンスレイヤーを選択する必要があります（存在しない場合は作成する必要があります）。その後、Asset Browserで、 をクリックし、そこから好きなobject をドラッグして、好きな位置のroom にドロップするだけで、そのobject のインスタンスが1つ、ゲームに追加されます。

room に複数のインスタンスを配置したい場合は、リソースツリーからobject を選択し、 を長押しすると、リソースをプレビューすることができます。 さらにマウスの左ボタン をクリックすると、room のレイヤーに「ペイント」することができます。

assets はroom エディタに追加される際、room の原点を基準とした X 軸と Y 軸の位置に基づいて位置が決められることに注意してください。room 原点は (0, 0) の位置で、GameMakerでは左上隅になります。 で、右に行くほど+X、下に行くほど+Yとなります、例．

上記の例では、インスタンスはroom の中央の位置 (X:512, Y:384) に配置されています。の外に出ることも可能である。 room の境界を超え、room の軸の長さよりも大きい位置、あるいはroom の原点よりも小さい位置を持つことになります。この場合、X/Y 座標は負の値になります。
