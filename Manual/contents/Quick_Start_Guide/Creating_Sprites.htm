ご覧のように、ウィンドウの左上にはsprite の名前を 入力する欄があります。 すべてのsprites （および他のすべてのassets ）には、その名前を入力する必要があります。 という名前をつけて、ユーザー（とGameMaker）が簡単に識別できるようにする必要があります。 変数を保持するID値 リソース、この場合はsprite を「指し示す」ものです。特定のsprite がasset なのかsprite なのか、それとも別のものなのかを一目で識別できるように、各object に説明的な名前を付けるのがベストです。 のように、リソースに " spr" という文字をプレフィックスまたはサフィックスとして付けることで、 " spr_Ball" のような表現になります。リソース名の使用は、文字、数字、アンダースコアのみに制限されています。 記号 "_" で始まる必要があります。また、すべてのassets の名前は、数字ではなく文字で始まる必要があります。

sprite エディタの他の機能については、このセクションの最後で説明しますが、まず、sprite の描き方を説明する必要があります。 これは 画像エディタ.画像 エディタは、ゲームのグラフィックを作成するための非常に強力なツールで、Edit Image ボタンをクリックすることで開きます。

ここで使用できるツールについては、マニュアルの「画像編集」のセクションを参照してください。

  • 左上のフレームビューには、sprite のフレームが表示され、虫眼鏡アイコンを使ってサイズを変更することができます。
  • 中央には描画用のキャンバスがあり、現在選択されているフレームが表示されています。マウスの中ボタン で移動し、マウスホイール で拡大・縮小することが可能です。
  • 右上には基本的なブラシがあり、ここで描画用のブラシを選択することができます。
  • 右の真ん中には、カラーピッカーがあります。ここでは、マウスの左ボタンと右ボタンに使う色を選択できます。
  • カラーピッカーのすぐ下にはツールが あります。ブラシ、フラッドフィル、マスキング、シェイプツールなど、さまざまな描画ツールです。いくつかのツールは斜めに分割されており、輪郭を描いたり塗りつぶしたりすることができます。 の形状は、どちらをクリックして選択するかによって異なります。
  • 他の多くの描画ツールと同様、レイヤーを作成してその上に描画することで、位置や色などを試すことができます。 の下にあるもの。

最初に知っておくべきことは、sprite の原点を 設定する方法です。原点は、sprite の中でroom を「固定」するために使われる点で、この場合、中心に設定します。

そこで、原点設定用のフロムダウンメニューから 、「middle center」を選択します。

プレビュー画像の青い原点マーカーがsprite の中心に移動するのがわかります。なお、原点は をクリックすることで、好きな場所に配置できます。 また、上部のx/yフィールドに値を追加することで、原点位置を手動で入力することもできます。

もうひとつ重要なのは、sprite のコリジョンマスクを設定する方法です。コリジョンマスクとは、ゲーム内で他のインスタンス（object ）と衝突があったかどうかをGameMaker が確認するための領域です。 デフォルトでは矩形に設定されています（解決するのも一番早い）。の検出モードを設定できることに注意してください。 バウンディングボックスを自動にすることができます。 (デフォルト値)または手動です。マニュアルを選択した場合、バウンディングボックスの値を左、右、上、下と設定することができるようになります。

矩形以外のコリジョンマスクの場合、解決に時間がかかりますが、小規模でシンプルなゲームを作る場合は、通常問題にはなりません。なお、他のソフトで作ったsprites をインポートすることもできます。 sprite また、画像ファイルをGameMaker IDE にドラッグして、sprites としてAsset Browser に追加することもできます。また、sprites BrowserからAsset をドラッグして直接追加することもできます。この場合、IDE がハイライト表示されます。 sprite に追加できる場所です。
