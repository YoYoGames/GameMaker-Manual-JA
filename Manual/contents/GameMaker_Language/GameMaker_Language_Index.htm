GML Code

GameMaker Language (単にGML とも呼ばれる) は、独自の GameMaker スクリプティング言語です。この言語は、ユーザーが直感的で柔軟な方法でゲームを作成できるように構成されており、他の主要なプログラミング言語のすべての力を提供します。また、GMLビジュアルベースにもなっており、必要に応じて併用することができます。
