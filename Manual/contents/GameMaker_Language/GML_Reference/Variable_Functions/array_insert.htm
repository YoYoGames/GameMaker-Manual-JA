array_insert

この関数を使うと、配列の任意の位置に値（複数可）を挿入することができます。この関数では、配列を格納する変数と、挿入する配列のインデックス（位置）、そして少なくとも1つの値を指定する必要があります。

指定したインデックスが配列の長さの範囲外である場合、値は指定したインデックスに追加され、配列の最後の値と挿入された値の間の空のスロットは、デフォルト値0に設定されることに注意してください。

構文です。

array_insert(variable, index, value, [value], [value], [etc...]);

論証 タイプ 商品説明
variable アレイ The variable that holds the array.
index リアル The index (position) in the array to insert the value(s) 
value バリアブル The value to insert
[value], [value], [etc...] バリアブル OPTIONAL 配列に挿入する追加の値

リターンです。

N/A

例

array_insert(score_array, score_rank, current_score);

上記のコードは、変数 score_rank で定義されたインデックスに、現在のスコアをスコア配列に挿入します。
