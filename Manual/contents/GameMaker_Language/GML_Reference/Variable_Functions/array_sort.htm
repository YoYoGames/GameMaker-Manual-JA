array_sort

この関数を使用すると、配列を昇順あるいは降順で並べ替えたり、 カスタム関数を使用して並べ替えの順序を決めたりすることができます。この関数では、並べ替えの対象となる配列を指定したうえで、 以下のいずれかの関数を使用します。

  • 定数 true または false は、昇順 (true) または降順 (false) のソート順を示す。
  • ソート順を定義するカスタム関数

この関数は、指定された元の配列の内容を変更します。ソートされたコピーを作成するには、 array_copy() を使ってまず配列を複製し、それをこの関数に渡します。

カスタム関数の使用

ソート用のカスタム関数を使用する場合、2つの引数を取る必要があり、それぞれ現在の要素と次の要素の値を受け取ります。この関数は、ソート順を変更するために、以下の値のいずれかを返す必要があります（ここで、elm1 は現在の要素、 elm2 は次の要素です）。

  • 0: elm1 と elm2 が等しい場合
  • <= -1 (負の整数): が先になる場合 elm1 elm2
  • >= 1 (正の整数): の後に行く場合 elm1 elm2

注：カスタム関数が返す値は整数でなければなりません。1 未満の浮動小数点値は 0 として読み込まれます。 sign()または round()を使用する必要があります。

配列がstrings のセットを含んでいる場合、デフォルトの昇順/降順ソートタイプを使用すると、strings は英語のアルファベットに基づきアルファベット順にソートされます。他のすべてのデータ型は、その数値に基づいてソートされます。その正確な値は、データ型自体に依存します (たとえば、buffers の配列は、そのメモリアドレスの数値に基づいてソートされます)。

構文です。

array_sort(variable, sorttype_or_function);

論証 タイプ 商品説明
variable アレイ The variable that holds the array.
sorttype_or_function  ブーリアンまたは スクリプト関数/メソッド The sort type (true は昇順、 は降順) またはソートに使用する関数のリファレンス。 false

リターンです。

N/A

例

var _a = [10, 9, 8, 7, 6, 5];

array_sort(_a, function(elm1, elm2)
{
    return elm1 - elm2;
});

上記のコードでは、独自のソート関数を使用して、配列を昇順にソートしています。このソート関数は、配列の 2 つの要素 elm1 と elm2 を受け取り、引き算を行います。 elm1 が elm2 よりも大きい場合、減算の結果は正の数となり、 elm1 は elm2 よりも後になるように押され、昇順となります。
