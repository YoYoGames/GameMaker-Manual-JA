array_resize

この関数を使うと、既存の配列の次元を新しいサイズにリサイズすることができます。サイズを変更する配列とその新しいインデックスを指定すると、この関数はその配列のサイズを変更します。この関数は、配列のサイズを小さくするために設計されていることに注意しましょう。この関数を使用して配列のサイズを変更した場合は、 新しいインデックスはデフォルトの 0 に設定されます。

この関数は多次元配列にも使用でき、配列のインデックスを指定する際に、以下のパターンでどの次元のサイズを変更したいかを指定することが必要である。

// Resize the first dimension of the array
array_resize(my_array, 10);

// Resize the second dimension of the array (only for the first slot)
array_resize(my_array[0], 10);

// Resize the third dimension of the array (only for the first slots)
array_resize(my_array[0][0], 10);

// ...and so on.

このコードでは、2次元目の最初のスロットだけがサイズ変更されます。これは、任意の配列次元のスロットは相互に接続されておらず、異なるサイズを持つ可能性があるからです。

array_resize(my_array[0], 10);
array_resize(my_array[1], 22);

上記のコードでは、配列の2次元目の長さが、1スロット目では10ですが、2スロット目では22になっています。

構文です。

array_resize(array_index, new_size);

論証タイプ 商品説明
array_index The index of the array to resize.
new_size The new size for the array (an integer, starting from 0).

リターンです。

例

if array_length(a) > 10
{
    array_resize(a, 10);
}

上記のコードでは、配列の長さをチェックし、インデックスが10個以上ある場合は、サイズを変更します。
