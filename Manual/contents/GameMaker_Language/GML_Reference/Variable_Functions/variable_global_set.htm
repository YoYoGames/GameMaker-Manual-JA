variable_global_set

この関数を使うと、指定したグローバル変数の値を設定することができます。値を設定するグローバル変数の名前を文字列として与え（下記のコード例を参照）、次に設定する値を与えます（有効なデータ型であれば何でもかまいません）。グローバル変数がゲーム内に存在しない場合は、作成され、値が割り当てられます。

構文です。

variable_global_set(name, val);

論証 タイプ 商品説明
name 文字列 The name of the global variable to set (as a string)
val バリアブル The value to set the global variable to

リターンです。

N/A

例

if !variable_global_exists("enemy_num")
{
    variable_global_set("enemy_num", 0);
}

上記のコードでは、グローバル変数が存在するかどうかを確認し、存在しない場合は作成して0に設定します。
