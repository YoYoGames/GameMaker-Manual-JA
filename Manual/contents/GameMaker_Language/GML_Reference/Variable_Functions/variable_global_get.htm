variable_global_get

この関数を使うと、指定した名前のグローバル変数から値を取得することができます。グローバル変数の名前を文字列で指定すると（以下のコード例を参照）、この関数はそのグローバル変数が持つ値を返しますが、その変数が存在しない場合は undefined を返します。

重要!取得するグローバル変数が存在しない場合、この関数はキーワード undefined を返し、ゲームが機能しなくなるようなエラーが発生する可能性があります。 variable_global_existsを最初に指定します。

構文です。

variable_global_get(name);

論証タイプ 商品説明
name The name of the global variable to get (as a string)

リターンです。

Variable (any data type) or undefined (if the named variable does not exist)

例

if variable_global_exists("enemy_num")
{
    show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

上記のコードは、グローバル変数が存在するかどうかを確認し、存在する場合はコンソールに出力します。
