is_bool

この関数は、与えられた変数が boolean (true ior false) であるか否かを返します。GameMakerの変数がブール値を保持しているかどうかを確認したい場合があり、そのようなときにこの関数を使用します。

構文です。

is_bool(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if is_bool(val)
{
    global.Sound = val;
}
else
{
    global.Sound = true;
}

上記のコードでは、変数valが実数かどうかをチェックし、実数であればそれを使ってグローバル変数を設定します。
