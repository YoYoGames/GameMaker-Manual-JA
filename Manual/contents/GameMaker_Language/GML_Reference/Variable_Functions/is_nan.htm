is_nan

この関数は、与えられた変数が NaN (数字ではない) かどうかを返す。もしそうなら true を、そうでないなら false を返す。

構文です。

is_nan(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if is_nan(global.value)
{
    show_debug_message("Value is not a number");
}

上記のコードでは、グローバル変数「value」が数値かどうかをチェックし、数値でない場合はデバッグメッセージを表示します。
