is_string

この関数は、与えられた変数がstring の数値であるかどうかを返します。GameMakerの変数が実数ではなく、string を保持しているかどうかを確認したい場合があり、そのようなときにこの関数を使用します。この関数はstring を返すのではなく、 true や false を返すので、例えば n が "fish" の場合は true を返しますが、n が 200 の場合は false を返します。

構文です。

is_string(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if is_string(val)
{
    name = "Player: " + val;
}

上記のコードは、変数 "val "にstring が含まれているかどうかをチェックし、含まれていればそれを別のstring に追加し、それを変数 "name "に代入しています。
