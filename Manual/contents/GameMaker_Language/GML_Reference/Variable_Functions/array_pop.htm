array_pop

説明： 指定された配列の最後の要素を削除し、その値を返す。配列が空の場合は，undefined を返す。

構文です。

array_pop(array);

論証タイプ 商品説明
array The array to remove the last value from.

リターンです。

Variable or undefined

例

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

上記のコードは、与えられた配列から最後の値を取り出し、いくつかのテキストとともに画面に描画します。また、取り出されたエントリは score 配列から取り除かれます。
