real

この関数は、与えられたstring を実数に変換するために使用することができます。この関数を使うとき，string の中の数字，マイナス記号，小数点，指数部分が考慮される。 を投げる。string に他の文字が含まれていることが分かっている、あるいは疑わしい場合は、次のようにします。 string_digits()を使用して数字以外の文字をすべて削除してから、この関数を使って結果の文字列を を実数に変換します。

 

構文です。

real(string)

論証 タイプ 商品説明
string String The string to be converted to a real value.

 

リターンです。

Real

 

例

var t_str = string_digits(input_str);
age = real(t_str);

上記のコードでは、入力string から数字以外の文字を取り除き、変数 "age" に実数値をセットしてstring を返します。
