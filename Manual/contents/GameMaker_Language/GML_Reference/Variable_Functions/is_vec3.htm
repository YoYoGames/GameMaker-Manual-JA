is_vec3

GameMakerで変数がどのようなデータ型を持っているかを確認したい場合があります。そのような時にこの関数を使います。値が3値ベクトルかどうかで、 true または false を返す。

構文です。

is_vec3(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if !is_vec3(val)
{
    show_debug_message("Not a 3 value vector!");
}

上記のコードは、変数 "val "にvec3が含まれているかどうかをチェックし、含まれていない場合はデバッグコンソールにメッセージを表示します。
