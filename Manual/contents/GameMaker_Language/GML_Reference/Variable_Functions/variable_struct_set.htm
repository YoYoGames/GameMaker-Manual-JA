変数構造セット

この関数を使うと、struct の指定された変数の値を設定することができます。struct の参照と、値を設定する変数の名前を文字列で指定し（以下のコード例を参照）、最後に設定する値（任意の有効なデータ型が可能）を指定します。もし変数がstruct にまだ存在しない場合は、変数が作成され、値が割り当てられます。

構文です。

variable_struct_set(struct, name, val);

論証 タイプ 商品説明
struct 構造 The struct reference to set
name 文字列 The name of the variable to set (as a string)
val バリアブル The value to set the variable to

リターンです。

N/A

例

if !variable_struct_exists(mystruct, "shields")
{
    variable_struct_set(mystruct, "shields", 0);
}

上記のコードは、与えられた変数が与えられたstruct に存在するかどうかを確認し、存在しない場合は、その変数を作成して 0 に設定します。
