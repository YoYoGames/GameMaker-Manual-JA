変数_グローバル_存在

この関数を使うと、グローバルスコープの変数が存在するかどうかをチェックすることができます。チェックしたいグローバル変数の名前を文字列で指定します（以下のコード例を参照）。この関数は、指定された名前のグローバル変数が存在する場合は true を、そうでない場合は false を返します。

構文です。

variable_global_exists(name);

論証 タイプ 商品説明
name 文字列 The name of the global variable to check for (as a string)

リターンです。

Boolean

例

if !variable_global_exists("enemy_num")
{
    global.enemy_num = instance_number(obj_Enemey_Parent);
}

上記のコードでは、「enemy_num」というグローバル変数が存在するかどうかを確認し、存在しない場合は作成して値を設定します。
