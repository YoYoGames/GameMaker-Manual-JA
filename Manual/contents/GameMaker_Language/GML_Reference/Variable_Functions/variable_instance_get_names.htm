variable_instance_get_names

インスタンスのインスタンス変数名や、ゲームのグローバル変数名を格納した配列を取得することができます。インスタンス ID の値を渡すと、配列の各エントリは、そのインスタンスに作成されたインスタンススコープ変数に対応する変数名（string ）になります。一方、キーワード global を渡すと、配列の各エントリは、グローバルスコープ変数に対応する変数名のstring になります。

構文です。

variable_instance_get_names(instance_id/global);

論証 タイプ 商品説明
instance_id/global インスタンスIDまたは global The unique ID value of the instance to check or the keyword global

リターンです。

Array (each entry is a string)

例

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++)
{
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
}

上記のコードは、コードブロックを実行しているインスタンスのすべてのインスタンススコープ変数の配列を取得し、デバッグ出力にこれらの変数とその値を表示します。
