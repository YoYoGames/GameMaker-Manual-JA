可変インスタンス名カウント

この関数は、あるインスタンスに定義されている変数の総数を求めることができます。インスタンス ID を指定すると、変数の数を表す整数値、あるいは (指定した ID のインスタンスが存在しない場合は) -1 を返します。

構文です。

variable_instance_names_count(instance_id);

論証 タイプ 商品説明
instance_id インスタンスID The unique ID value of the instance to check.

リターンです。

Real

例

var _num = variable_instance_names_count(myinst);
show_debug_message("Instance Variables = " + string(_num));

上記のコードは、与えられたインスタンスの変数数を取得し、その値でコンソール出力にデバッグメッセージを表示します。
