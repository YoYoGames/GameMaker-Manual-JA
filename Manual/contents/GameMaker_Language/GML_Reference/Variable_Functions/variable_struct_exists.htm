変数構造体が存在する

この関数を使うと、与えられたstruct の中に変数が存在するかどうかをチェックすることができます。使用するstruct の参照と、チェックする変数名を文字列で指定します (以下のコード例を参照してください)。この関数は、与えられた名前の変数がstruct に存在する場合は true を、そうでない場合は false を返します。

構文です。

variable_struct_exists(struct, name);

論証 タイプ 商品説明
struct 構造 The struct reference to check
name 文字列 The name of the struct variable to check for (as a string)

リターンです。

Boolean

例

if !variable_struct_exists(mystruct, "shields")
{
    mystruct.shields = 0;
}

上記のコードは、与えられたstruct に "shields" という変数が存在するかどうかを確認し、存在しない場合は作成されて 0 に初期化されます。
