bool

この関数は、与えられた値をブーリアンデータ型に変換しようとするもので、値が 0.5 以上の場合は true として、そうでない場合は false として返されます。

 

構文です。

bool(n);

論証 タイプ 商品説明
n Real The value to convert.

 

リターンです。

Boolean

 

例

if !is_bool(val)
{
    val = bool(val);
}

上記のコードでは、変数 "val "がブール値であるかどうかをチェックし、ブール値でない場合はブール値に変換しています。
