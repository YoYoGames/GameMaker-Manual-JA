is_real

この関数は、与えられた変数が実数（シングル、ダブル、整数）であるか否かを返します。ある変数が実数を保持しているかどうかを調べたい場合がありますが、そのようなときにこの関数を使用します。この関数は実数を返すのではなく、 true または false を返すので、例えば値が "fish" の場合は false を返し、値が 200 の場合は true を返します。

注：この関数は、いかなる場合にもfalse を返します。 enumの値は、int64s として格納されているためです。そのデータ型を確認するための正しい関数は、is_int64() です。

構文です。

is_real(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if is_real(val)
{
    score += val;
}

上記のコードは、変数valが実数かどうかをチェックし、実数であればそれをスコアに加算している。
