is_undefined

この関数は、与えられた変数が定義されているかどうかを返します。GameMakerで変数がどのようなデータ型を持っているかを確認したいときに、この関数を使うことがあります。値が定義されているかどうかによって、 true または false を返します。

注：この機能は、変数の存在を確認するために使用することはできません。使用方法 variable_instance_exists()または variable_global_exists()代わりに

構文です。

is_undefined(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

var val = ds_map_find_value(map, 13);
if (is_undefined(val))
{
    show_debug_message("Map entry does not exist!");
}

上記のコードでは、変数 "val "がundefined かどうかをチェックし、 の場合はデバッグメッセージを表示します。
