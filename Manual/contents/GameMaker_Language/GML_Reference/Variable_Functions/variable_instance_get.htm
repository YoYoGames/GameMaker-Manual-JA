variable_instance_get

この関数を使うと、指定された名前の変数から値を取得することができます。一意のインスタンス ID 値 (これはroom エディタのインスタンスプロパティから見つけることができます。また、この関数を呼び出すと返されます) を指定します。 instance_create_layer()) と、値を取得する変数の名前を文字列で指定します (以下のコード例を参照してください)。この関数は、その変数が持つ値を返しますが、変数が存在しない場合は、 undefined 。

重要!取得する変数が存在しない場合、この関数はキーワード undefined を返し、ゲームの機能を停止させるエラーが発生する可能性があります。 variable_instance_existsを最初に指定します。

構文です。

variable_instance_get(instance_id, name);

論証 タイプ 商品説明
instance_id インスタンスID The unique ID value of the instance to use
name 文字列 The name of the variable to get (as a string)

リターンです。

Variable (any data type) or undefined (if the named variable does not exist)

例

if variable_instance_exists(id, "shields")
{
    var ss = variable_instance_get(id, "shields");
}
else
{
    var ss = -1;
}

上記のコードでは、変数が存在するかどうかを確認し、存在する場合はその値が取得され、ローカル変数に格納されます。存在しない場合は、ローカル変数に-1がセットされます。
