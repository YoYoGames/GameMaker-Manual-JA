is_infinity

この関数は、与えられた変数が infinity (無限大)であるかどうかを返します。もしそうなら true を、そうでないなら false を返します。

構文です。

is_infinity(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if is_infinity(global.value)
{
    show_debug_message("Value is infinite!");
}

上記のコードでは、グローバル変数「value」が無限大かどうかをチェックし、無限大の場合はデバッグメッセージを表示します。
