variable_struct_get

この関数を使うと，struct 内の指定された名前の変数から値を取得することができます。struct の参照と，値を取得する変数の名前を文字列で指定します（以下のコード例を参照してください）。この関数は、その変数が持つ値を返しますが、その変数が存在しない場合は、 undefined 。

重要!取得する変数が存在しない場合、この関数はキーワード undefined を返し、ゲームの機能を停止させるエラーが発生する可能性があります。 variable_struct_exists()を最初に指定します。

構文です。

variable_struct_get(struct, name);

論証タイプ 商品説明
struct The struct reference to use
name The name of the variable to get (as a string)

リターンです。

Variable (any data type) or undefined (if the named variable does not exist)

例

if variable_struct_exists(mystruct, "shields")
{
    var ss = variable_struct_get(mystruct, "shields");
}
else
{
    var ss = -1;
}

上記のコードでは、与えられたstruct に変数が存在するかどうかを確認し、存在する場合はその値が取得され、ローカル変数に格納されます。存在しない場合は、ローカル変数に-1がセットされます。
