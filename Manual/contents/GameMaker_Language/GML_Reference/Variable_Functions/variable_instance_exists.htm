variable_instance_exists

この関数を使うと、インスタンススコープの変数が存在するかどうかをチェックすることができます。一意のインスタンス ID 値 (room エディタのインスタンスプロパティで確認できます。また、この関数を呼び出すと返されます) と、チェックする変数名を文字列で指定します。 instance_create_layer()を呼び出したときに返されます）、およびチェックする変数名を文字列として指定します（以下のコード例を参照してください）。この関数は、指定された名前の変数がインスタンスに存在する場合は true を、そうでない場合は false を返します。

構文です。

variable_instance_exists(instance_id, name);

論証 タイプ 商品説明
instance_id インスタンスID The unique ID value of the instance to check
name 文字列 The name of the variable to check for

リターンです。

Boolean

例

if !variable_instance_exists(id, "shields")
{
    shields = 0;
}

上記のコードでは、コードを実行しているインスタンスに「shields」という変数が存在するかどうかを確認し、存在しない場合は作成して0に初期化します。
