この関数は、与えられた値をポインターのデータ型に変換しようとします。値は real 、 string 、 int64 、 int32 、 ptr のいずれかでなければなりません。それ以外は、エラーメッセージとともにゲームがクラッシュします。

構文です。

ptr(n);

論証 タイプ 商品説明
n Real,String, またはPointer The value to convert.

リターンです。

Pointer

例

if !is_ptr(val)
{
    val = ptr(application_surface);
}

上記のコードでは、変数 "val "にポインタが含まれているかどうかをチェックし、含まれていない場合はポインタを代入しています。
