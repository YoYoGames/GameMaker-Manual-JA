変数構造体削除

この関数を使うと、struct から変数を削除することができます。 変数を削除するstruct ID と、削除する変数のstring 名を指定します。

構文です。

variable_struct_remove(struct, name);

論証 タイプ 商品説明
struct 構造 The struct reference to remove the variable from
name 文字列 The name of the variable to remove (as a string)

リターンです。

N/A

例

if variable_struct_exists(mystruct, "shields")
{
    variable_struct_remove(mystruct, "shields");
}

上記のコードは、与えられた変数が与えられたstruct に存在するかどうかを確認し、存在する場合はそれを削除します。
