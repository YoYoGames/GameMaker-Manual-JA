array_height_2dDEPRECATED

警告!この関数は、配列が1次元または2次元に限定されなくなったため、非推奨（ array_length() ）となっています。したがって、この関数はレガシープロジェクトのサポートのためにのみ提供されています。すべての新しいプロジェクトは、多次元配列の作成とアクセスに現在の方法を使用する必要があります（詳しくはここを参照してください）。

この関数を使うと、2次元配列の1次元目の高さ（エントリ数）を得ることができます。チェックする配列を指定すると、その配列が何個の初期エントリを含んでいるかが出力されます。配列の 2 次元のエントリ数を得るには，関数 array_length_2d.

構文です。

array_height_2d(array_index);

論証 タイプ 商品説明
array_index アレイ The index of the array to check.

リターンです。

Real

例

for (var i = 0; i < array_height_2d(a); ++i;)
{
    for (var j = 0; j < array_length_2d(a, i); ++j;)
    {
        a[i, j] = -1;
    }
}

上記のコードは、loop 、2D配列を通過し、各エントリーを-1に設定します。
