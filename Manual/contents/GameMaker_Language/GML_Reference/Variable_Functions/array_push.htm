array_push

この関数を使うと、配列の長さを知らなくても、配列の末尾に値（複数可）をプッシュすることができます。この関数では、配列を保持する変数と、プッシュする値を少なくとも1つ指定する必要があります。

構文です。

array_push(variable, value, [value], [value], [etc...]);

論証 タイプ 商品説明
variable アレイ The variable that holds the array.
value バリアブル The value to push onto the end of the array
[value], [value], [etc...] バリアブル OPTIONAL 配列に追加される値

リターンです。

N/A

例

array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr);

上記のコードは、4つの値を与えられた配列の末尾にプッシュします。
