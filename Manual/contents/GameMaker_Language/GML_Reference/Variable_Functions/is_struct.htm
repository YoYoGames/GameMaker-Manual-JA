is_struct

この関数は，与えられた値がstruct であるかどうかを調べます。 もしそうであれば true を返し，そうでなければfalse を返します。

なお，メソッド変数も true を返し，オブジェクトのインスタンスは false を返します．

構文です。

is_struct(val);

論証 タイプ 商品説明
val バリアブル The value to check.

リターンです。

Boolean

例

if is_struct(a)
{
    delete(a);
}

上記のコードでは、変数がstruct であるかどうかをチェックし、関数が true を返した場合、struct を削除しています。
