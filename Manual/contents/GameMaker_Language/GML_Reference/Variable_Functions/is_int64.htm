is_int64

この関数は、与えられた変数が64ビット整数であるかどうかを返します。GameMakerで変数がどのようなデータ型を持っているか確認したいとき、この関数を使うことがある。値がint64かどうかで、 true または false を返す。

構文です。

is_int64(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if !is_int64(val)
{
    show_debug_message("Not a 64bit integer!");
}

上記のコードは、変数 "val "にint64が含まれているかどうかをチェックし、含まれていない場合はデバッグコンソールにメッセージを表示します。
