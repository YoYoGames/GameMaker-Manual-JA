array_create

この関数を使うと、指定されたサイズの1次元配列を作成することができます。この関数に作成する配列の長さを指定すると、その配列の「ハンドル」が返され、変数に代入することができます。この方法で作成された配列は、初期化値を指定しない限り各項目が 0 で初期化されます。配列を初期化するための値を指定した場合は、 新しい配列のすべてのインデックスを 0 ではなくその値に設定します。

構文です。

array_create(size, [value]);

論証 タイプ 商品説明
size リアル The size of the array to create.
value バリアブル OPTIONAL すべての配列のインデックスを初期化するために使用する値です。

リターンです。

Array

例

instance_array = array_create(100, noone);

上記のコードでは、100個の項目からなる新しい配列を作成し、それぞれの項目にキーワード noone を設定し、変数「instance_array」に代入しています。
