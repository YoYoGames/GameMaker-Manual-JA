method_get_index

この関数を使うと、メソッドが定義されたscript のスクリプトインデックスを取得できます。メソッドがscript で定義されていない場合、この関数は -1 を返し、そうでない場合はscript のインデックス値を返します。

構文です。

method_get_index(method);

論証 タイプ 商品説明
method 方法 The method variable to check

リターンです。

Script Asset or Real

例

var _index = method_get_index(light_setup);
if _index != -1
{
    show_debug_message(script_get_name(_index));
}

上記のコードは、メソッド「light_setup」のscript のインデックスをチェックし、それが -1 でない場合はscript の名前をコンソールに出力します。
