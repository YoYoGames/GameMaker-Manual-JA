array_equals

この関数を使うと、2つの配列が等しいかどうか（同等か同じかどうか）を調べることができます。チェックしたい 2 つの配列を指定すると、この関数は、それらが等しい（同等または同じ）場合は true を、等しくない場合は false を返します。この関数は、2つの配列が同じ値を保持しているかどうかを調べるのではなく、2つの配列が同じ初期配列を参照しているかどうかを調べるだけです。たとえば

var a = [1,2,3,4];
var b = [1,2,3,4];

if (a == b)
{
    show_debug_message( "This will never fire, as a and b do not reference the SAME array" );
}

if (array_equals(a, b))
{
    show_debug_message( "This will fire as both arrays contain similar values" );
}

なお、定数の中には、自分自身と等しくないものもあり、この関数は失敗することがあります。以下はその例である。

if (array_equals([NaN], [NaN]))
{
    show_debug_message( "This will never fire as NaN cannot be equal to itself" );
}

詳しくは、「イコライザーテーブル」をご覧ください。

構文です。

array_equals(var1, var2);

論証 タイプ 商品説明
var1 アレイ The first array.
var2 アレイ The second array.

リターンです。

Boolean

例

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

上記のコードは、2つの配列が同等の値を保持しているかどうかを確認し、保持していない場合は、配列「inventory_array」の内容全体を配列「item_array」にコピーします。
