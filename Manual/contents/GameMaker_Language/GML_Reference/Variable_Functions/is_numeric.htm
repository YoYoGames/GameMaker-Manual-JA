is_numeric

この関数は、与えられた変数が数値（real, int32, int64, boolean）であるか否かを返します。GameMakerの変数が数値を持っているかどうかを調べたいとき、この関数を使うことがある。この関数は、与えられた入力が数値であれば true を、そうでなければ false を返す。

構文です。

is_numeric(n);

論証 タイプ 商品説明
n バリアブル The input to check.

リターンです。

Boolean

例

if is_numeric(val)
{
    current_val += val;
}

上記のコードでは、変数valが数値であるかどうかをチェックし、数値であれば与えられた変数に追加しています。
