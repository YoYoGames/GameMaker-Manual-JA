array_set

この関数を使うと、配列のインデックスの値を値に設定することができます。この関数では，配列を格納する変数と，設定するインデックス，設定する値を指定します。この関数は多次元配列にも使用できます。配列のインデックスを指定する際に、どの次元の値を設定するかを次のパターンで指定します。

// 1D array
array_set(my_array, 0, 100);
// 2D array
array_set(my_array[0], 0, 100);
// 3D array
array_set(my_array[0][0], 0, 100);
// etc...

構文です。

array_set(variable, index, value);

論証 タイプ 商品説明
variable アレイ The variable that holds the array.
index リアル The index of the array element to set the value for.
value バリアブル The value to set.

リターンです。

N/A

例

for (var i = 0; i < 10; ++i;)
{
    array_set(score_array, i, i*100));
}

上記のコードは、与えられた配列の最初の 10 項目に値を設定します。
