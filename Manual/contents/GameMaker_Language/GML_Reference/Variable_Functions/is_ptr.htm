is_ptr

この関数は、与えられた変数がポインタであるかどうかを返します。GameMakerで変数がどのようなデータ型を持っているかを調べたいとき、この関数を使うことがある。値がポインターかどうかによって、 true または false を返す。

構文です。

is_ptr(n);

論証 タイプ 商品説明
n バリアブル The argument to check.

リターンです。

Boolean

例

if !is_ptr(val)
{
    show_debug_message("Not a valid texture!");
}

上記のコードは、変数 "val "にポインタが含まれているかどうかをチェックし、含まれていない場合はデバッグコンソールにメッセージを表示します。
