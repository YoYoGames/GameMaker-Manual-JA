変数関数

このページの関数は、GameMaker Language をゲームで使用する際に、あらゆる種類の変数と変数タイプを扱うために設計されています。変数の詳細については、変数と変数スコープのページを参照してください。

以下の変数機能があります。

  • variable_instance_exists
  • variable_instance_get_names
  • variable_instance_names_count
  • variable_instance_get
  • variable_instance_set
  • variable_global_exists
  • variable_global_get
  • variable_global_set
  • struct_exists
  • struct_get
  • struct_set
  • struct_remove
  • struct_get_names
  • struct_names_count
  • instanceof

 

また、配列の使用にはいくつかの関数が関連します。これらは、コードに柔軟性を持たせるために設計されており、ゲームを作る際に、より機能的でダイナミックな配列を作成することが可能になります。 これらの関数は以下の通りです。

  • array_create
  • array_copy
  • array_equals
  • array_get
  • array_set
  • array_push
  • array_pop
  • array_insert
  • array_delete
  • array_sort
  • array_length
  • array_resize
  • array_length_1d(非推奨)
  • array_length_2d(非推奨)
  • array_height_2d(非推奨)

 

次の関数は、すべてメソッド変数の使用に関連するものです。

  • method
  • method_get_self
  • method_get_index

 

最後に、変数の型を決定するための以下の関数も用意されています（データ型についてはこちらを、型テーブルについてはこちらをご覧ください）。

  • is_string
  • is_real
  • is_numeric
  • is_bool
  • is_array
  • is_struct
  • is_method
  • is_ptr
  • is_int32
  • is_int64
  • is_vec3
  • is_vec4
  • is_matrix
  • is_undefined
  • is_nan
  • is_infinity
  • typeof
  • bool
  • real
  • ptr
  • int64
