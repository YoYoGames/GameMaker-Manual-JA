変数インスタンスセット

この関数を使うと、インスタンスに指定した変数の値を設定することができます。一意のインスタンス ID 値 (これはroom エディタのインスタンスプロパティから見つけることができます。また、この関数を呼び出すと返されます) を指定します。 instance_create_layer()) 、値を設定する変数名を文字列で指定し (以下のコード例を参照)、最後に設定する値 (有効なデータ型であれば何でもよい) を指定します。変数がまだインスタンスに存在しない場合は、変数が作成され、値が割り当てられます。

構文です。

variable_instance_set(instance_id, name, val);

論証 タイプ 商品説明
instance_id インスタンスID The unique ID value of the instance to use
name 文字列 The name of the variable to set (as a string)
val バリアブル The value to set the variable to

リターンです。

N/A

例

if !variable_instance_exists(id, "shields")
{
    variable_instance_set(id, "shields", 0);
}

上記のコードでは、呼び出したインスタンスにインスタンス変数が存在するかどうかを確認し、存在しない場合は作成して0をセットします。
