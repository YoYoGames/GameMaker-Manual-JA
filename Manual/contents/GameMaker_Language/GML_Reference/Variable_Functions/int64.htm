イントネーション64

この関数は、与えられた値を符号付き64ビット整数に変換しようとします。値は、real、string 、 int64 、 int32 、 ptr のいずれかでなければなりません。それ以外の場合は、エラーメッセージを表示してゲームをクラッシュさせる原因となります。変数がint64を保持しているかどうかを調べるには、関数 is_int64().

構文です。

int64(val);

論証 タイプ 商品説明
val リアル, 文字列または ポインタ The value to convert.

リターンです。

int64 (signed 64-bit integer)

例

steam_handle = int64(global.fileReadString);

上記のコードでは、グローバル変数に保持されている値を64bitの整数に変換しています。
