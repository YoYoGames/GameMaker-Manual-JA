array_get

この関数を使うと、配列のインデックスから値を取得することができます。この関数では、配列を格納する変数と、配列のインデックスから値を取得するためのインデックスを指定する必要があります。配列のインデックスが範囲外である場合、ゲームはエラーでクラッシュすることに注意してください。この関数は多次元配列にも使用できます。配列のインデックスを指定する際に、どの次元の値を取得したいのかをこのパターンに従って指定すればよいのです。

// 1D array
array_get(my_array, 0);
// 2D array
array_get(my_array[0], 0);
// 3D array
array_get(my_array[0][0], 0);
// etc...

構文です。

array_get(variable, index);

論証 タイプ 商品説明
variable アレイ The variable that holds the array.
index リアル The index of the array element to get the value from.

リターンです。

Variable (Any valid data type that an array can hold)

例

for (var i = 0; i < 10; ++i;)
{
    show_debug_message(array_get(my_array, i));
}

上記のコードは、与えられた項目のうち、最初の10項目をコンソールに出力します。
