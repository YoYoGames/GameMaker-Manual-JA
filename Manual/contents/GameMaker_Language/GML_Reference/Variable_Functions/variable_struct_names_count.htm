変数名カウント

この関数を使うと，あるstruct で定義されている変数の総数を調べることができます。調べるためにstruct の ID を与えると，この関数は出会った変数の数を整数値で返します（与えられた ID のstruct が存在しない場合）。 -1.

構文です。

variable_struct_names_count(struct_id);

論証タイプ 商品説明
struct_id The unique ID value of the struct to check.

リターンです。

例

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

上記のコードは、指定されたstruct の変数の数を取得し、この値でコンソール出力にデバッグメッセージを表示します。
