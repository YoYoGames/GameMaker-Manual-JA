method_get_self

この関数を使用すると、インスタンス ID またはstruct のリファレンスを取得することができます。 selfのコンテキストは、メソッドが呼び出されたときに使用されます。変数がメソッドでない場合、この関数は undefined を返します。この関数は定数 pointer_null を返すこともあり、その場合は呼び出し時に現在の自己が使用されていることに注意してください。

構文です。

method_get_self(method);

論証タイプ 商品説明
method The method variable to check.

リターンです。

Instance ID, Struct, undefined, or pointer_null

例

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

上記のコードは、与えられたメソッド変数のセルフコンテキストを取得し、コンソールに出力しています。
