is_array

この関数は、変数が配列を保持しているかどうか（ true ）を確認するために使用することができます（その場合、 false ）を返します。

構文です。

is_array(n);

論証 タイプ 商品説明
n バリアブル The variable to check.

リターンです。

Boolean

例

if is_array(a)
{
    a = -1;
}

上記のコードでは，変数が配列であるかどうかをチェックし，関数が true を返した場合，変数に -1 をセットして配列を削除しています．
