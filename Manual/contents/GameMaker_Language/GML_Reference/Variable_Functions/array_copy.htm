array_copy

この関数を使うと、配列の全部または一部を、任意の位置の別の配列にコピーすることができます。コピー元配列とコピー先配列（両方とも以前に作成されている必要があります）、コピー元配列の位置、コピー先配列の位置を指定する必要があります。最後に、コピーする配列の長さ (またはコピーする部分の長さ) を指定する必要があります。コピーするデータがコピー先配列の長さを超える場合は、データを受け入れるために配列が拡張されます。

この関数は多次元配列にも使用でき、配列のインデックスを指定する際にどの次元をコピーするかを指定すれば、このパターンに従って使用することができる。

// Copy to the first dimension of an array
// from the second dimension of an array
array_copy(item_array, 0, inventory_array[0], 0, len);

// Copy to the third dimension of an array
// from the first dimension of an array
array_copy(item_array[0][0], 0, inventory_array, 0, len);

// etc...

構文です。

array_copy(dest, dest_index, src, src_index, length);

論証 タイプ 商品説明
dest アレイ The ID of the array to copy to.
dest_index リアル The index within the array to copy to.
src アレイ The ID of the array to copy from.
src_index リアル The index with the array to start copying from.
length リアル The length (number of array indices) to copy.

例

if !array_equals(inventory_array, item_array)
{
    var len = array_length(inventory_array);
    array_copy(item_array, 0, inventory_array, 0, len);
}

上記のコードは、2つの配列が同等の値を保持しているかどうかをチェックし、保持していない場合は、配列「inventory_array」の内容全体を配列「item_array」にコピーします。
