文字列_upper

この機能を使うと、string が大文字だけを含むように強制することができます。

注意この機能は、A～Zの26文字のアルファベットのみを検出します。

構文です。

string_upper(string);

論証タイプ 商品説明
string The string to convert to uppercase.

リターンです。

例

str1 = "Hello World";
str2 = string_upper(str1);

上記のコードでは、変数 str2 に "HELLO WORLD "が設定されます。
