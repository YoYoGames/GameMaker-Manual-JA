文字列幅

この関数は、入力string の幅（ピクセル）を、テキストが持つかもしれない改行を考慮して返します。これは、文字列を構成する文字の幅の合計から、テキスト要素間の距離を計算するのに非常に便利です。 で描画されるように draw_text()を使用し、現在定義されているfont を使用します。

構文です。

string_width(string);

論証タイプ 商品説明
string The string to measure the width of.

リターンです。

例

var ww = string_width(str_Name + " ");
draw_text(32, 32, str_Name));
draw_text(32 + ww, 32, "has won the game!");

上記のコードは、与えられたstring の幅を取得し、返されたstring の幅を区切りとして、2行のテキストを描画します。
