文字列_ord_at

この関数は、string の中の特定の位置にある特定の文字コードを返すために使用します。インデックス は、最初の文字が 1 から始まります。文字が見つからなかったり，string が index に与えられた値より短い場合は，-1 が返される。

構文です。

string_ord_at(str, index);

論証タイプ 商品説明
str The string to check.
index The position to get the character code from.

リターンです。

例

str = "Hello World";
char_code = string_ord_at(str, 7);

これは、string "str "の7文字目（"H "を1文字目と数える）の文字コードを取得し、変数 "char_code "に格納するものです。
