string_lettersdigits

この関数は、与えられたstring から文字と数字を除いたコピーを返します。つまり、例えばログイン名やパスワードから不要な文字（"#"や"？"など）を取り除くために使用できます。

注意この機能は、A～Zの26文字のアルファベット（大文字または小文字）と、0～9の数字のみを返します。

構文です。

string_lettersdigits(string);

論証タイプ 商品説明
string The string to convert to letters and digits.

リターンです。

例

if (string_length(password) > string_length(string_lettersdigits(password)))
{
    draw_text(32,32, "Invalid Password! Only Letters and numbers please!");
}

上記のコードは、string の長さを、同じstring の長さ（ただし、すべての記号を削除したもの）と比較してチェックします。もし両者が同じでなければ、画面にメッセージが表示される。
