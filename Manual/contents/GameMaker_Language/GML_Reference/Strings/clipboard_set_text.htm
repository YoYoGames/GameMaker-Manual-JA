クリップボード_セット_テキスト

この関数は、定義されたstring を保持するようにクリップボードを設定します。空のstring ""を設定すると、クリップボードのテキストを効果的に消去することができます。

構文です。

clipboard_set_text(string);

論証 タイプ 商品説明
string 文字列 The text (as a string) to set the clipboard to hold.

リターンです。

N/A

例

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上記のコードでは、クリップボードにテキストがあるかどうかをチェックし、もしテキストがあれば、string 、変数 "str "に読み込まれます。最後にクリップボードを空のstring にセットしてクリアします。
