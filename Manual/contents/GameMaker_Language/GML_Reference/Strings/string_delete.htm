string_delete

この関数は、与えられたstring の一部を削除するために使用される。入力string と、そのstring 内で文字を削除する開始位置 (インデックスは 1 から始まります)、そして削除する文字数を指定します。この関数は，その部分を除いた新しいstring を返します．

構文です。

string_delete(str, index, count);

論証タイプ 商品説明
str The string to copy and delete from.
index The position of the first character to remove (from 1).
count The number of characters to remove.

リターンです。

String

例

str1 = "Helloo World";
str2 = string_delete(str1, 5, 1);

これは str2 を "Hello World" に設定します。"Hello" から余分な "o" (位置 5 に存在) を削除しているためです。
