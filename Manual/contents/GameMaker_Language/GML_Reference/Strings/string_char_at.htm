文字列_char_at

この関数はstring 内の特定の位置にある特定の文字を返すために使用します。文字が見つからないか、string が与えられたインデックス値より短い場合は、空のstring ""が返されます。しかし、与えられたインデックスが 0 以下の場合は、string の最初の文字が返されます。

構文です。

string_char_at(str, index);

論証タイプ 商品説明
str The string to check.
index The position to get the character from.

リターンです。

例

str1 = "Hello World";
str2 = string_char_at(str1, 7);

これにより、 str2 は、string str1 の7文字目（"H "を1文字目として数える）、ここでは "W "に設定されます。
