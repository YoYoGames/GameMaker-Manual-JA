ansi_char

この関数は、生の BYTE 値が設定された文字を含むstring を返します。これは表示されませんし、されるべきでもありませんが、エンコーディングに使用するためにディスクに正しく保存されます。

構文です。

ansi_char(val);

論証タイプ 商品説明
val The raw byte value.

リターンです。

(Single character)

例

var str1 = ansi_char($EF);
var str2 = ansi_char($BB);
var str3 = ansi_char($BF);
file_text_write_string(global.saveFile, str1 + str2 + str3);

上記のコードは、生のバイトデータからstring を作成し、（以前に開いた）ファイルに書き込んでいます。
