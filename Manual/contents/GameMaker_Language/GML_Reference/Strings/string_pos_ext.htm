文字列_pos_ext

この関数は、string 内の部分文字列のインスタンスの文字位置を、開始位置として与えられた位置からstring を通して前方に検索して返します。この関数は、検索対象のstring が見つからなかった場合は 0 を返し、見つかった場合は検索対象のstring の最初の文字の位置を返します。レガシーサポートでは、strings は 1 からインデックスされるので、1 はstring の最初の位置であり、期待される 0 ではないことに留意してください。

構文です。

string_pos_ext(substr, str, start_pos);

論証タイプ 商品説明
substr The substring to look for in the string.
str The string.
start_pos The starting position to search from.

リターンです。

例

if (string_pos_ext(",", text, 20) != 0)
{
    string_insert(name, text, string_pos_ext(",", text, 20));
}

上記のコードは、変数「text」に格納されているstring 、20文字目以降のカンマを検索し、見つかった場合はその位置に「name」という部分文字列を挿入しています。
