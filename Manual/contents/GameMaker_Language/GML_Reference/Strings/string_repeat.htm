string_repeat

この関数は、単に同じstring をそれ自身の上に与えられた回数だけ繰り返して返します。

構文です。

string_repeat(str, count);

論証タイプ 商品説明
str The string to repeat.
count The number of times to repeat the string.

リターンです。

例

str1 = "Hello World";
str2 = string_repeat(str1, 3);

上記のコードでは、 str2 、"Hello WorldHello WorldHello World "が設定されます。
