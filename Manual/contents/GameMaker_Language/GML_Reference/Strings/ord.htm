オード

この関数は、1 文字の入力string を受け取り、その文字に対応する Unicode (UTF8) 値を返します。ただし keyboard_check*関数では，入力string は1文字だけで，0から9までの数字か，AからZまでの大文字のローマ字だけです。

構文です。

ord(string);

論証 タイプ 商品説明
string 文字列 The string with which to find the Unicode value.

リターンです。

Real

例

if keyboard_check(ord("W"))
{
    y -= 4;
}

これは、キーWを押したままだと、呼び出し元のインスタンスが4ピクセル上に移動します。
