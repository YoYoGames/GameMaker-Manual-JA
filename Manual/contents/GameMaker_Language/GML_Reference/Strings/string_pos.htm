文字列_pos

この関数は，string 内の部分文字列のインスタンスの文字位置を，string の先頭から検索して返します（末尾から検索する場合は，この関数を使用します）。 string_last_pos()).この関数は，全く見つからなければ0を返し，見つかった場合はその部分文字列の最初の文字の位置を返します．レガシーサポートのために、strings は 1 からインデックスが付けられるので、1 はstring の最初の位置であり、期待されるような 0 ではないことに留意してください。この用途の1つは、不快と思われる単語をフィルタリングしたり、別のstring にテキストを挿入する正しい場所を探したりすることです。

構文です。

string_pos(substr, str);

論証タイプ 商品説明
substr The substring to look for in the string.
str The string.

リターンです。

例

if (string_pos(",", text) != 0)
{
    string_insert(name, text, string_pos(",", text));
}

上記のコードは、変数「text」に格納されているstring 、カンマを検索し、見つかった場合はその位置に「name」という部分文字列を挿入しています。
