文字列バイト長

この関数は、string のバイト数を返します。UTF8 で保持されているため、string の文字数とは一致しません。

構文です。

string_byte_length(string);

論証 タイプ 商品説明
string 文字列 The string to measure the number of bytes of.

リターンです。

Real

例

bytesize = string_byte_length("Hello World");

この場合、bytesizeには "Hello World "のバイト数が設定される。
