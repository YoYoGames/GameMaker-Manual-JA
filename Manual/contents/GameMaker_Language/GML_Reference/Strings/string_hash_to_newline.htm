文字列_hash_to_newline

この関数は、"#"記号が改行されたstring を返します。つまり、string のような書式になります。

"Hello#World"

として画面にプリントされることになる。

Hello
World

重要!この機能は、GameMakerと旧バージョンのソフトウェアとのインポート互換性のために提供されているものですので、旧機能を再現する必要がある場合を除き、この機能を使用することはできません。代わりに エスケープ文字を使用して改行する必要があります。インポートの互換性についての詳細は、こちらをご覧ください。

構文です。

string_hash_to_newline(string);

論証タイプ 商品説明
string The string to convert over multiple lines

リターンです。

例

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上記のコードは、ハッシュ記号のついたstring を、2行に分割したstring に変換して、画面に表示しています。
