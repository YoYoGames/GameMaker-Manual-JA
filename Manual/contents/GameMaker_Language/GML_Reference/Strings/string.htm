string

この関数を使うと、任意の実数をstring に変換することができます。実数が整数の場合は、小数点以下ゼロで保存され、それ以外の場合は、小数点以下2桁で保存されます。それ以上の小数点以下の桁数が必要な場合は，関数 string_format().また、配列、データ構造、構造体を格納する変数にこの関数を使用すると、これらの変数の内容がstring に変換され、コンソールに出力したり、ファイルに保存してdebugging にしたりできることに注意してください。

構文です。

string(val);

論証 タイプ 商品説明
val リアル The real number to be turned into a string.

リターンです。

String

例

draw_text(100, 100, "Score: " + string(score) + " / Health: " + string(health));

上記のコードでは、 string 関数を使って実数とstrings の両方を一緒に描画しています（draw はstring か実数のどちらかのみを受け付けますが、両方を受け付けることはできません）。
