文字列_最後尾の位置

この関数は，string 内の部分文字列のインスタンスの文字位置を，string の末尾から先頭まで検索して返します（したがって string_pos().この関数は、検索対象string が見つからなかった場合は 0 を返し、見つかった場合は検索対象string の最初の文字の位置を返します。レガシーサポートのため、strings は 1 からインデックスされるので、1 はstring の最初の位置であり、期待されるような 0 ではないことに留意してください。

構文です。

string_last_pos(substr, str);

論証タイプ 商品説明
substr The substring to look for in the string.
str The string.

リターンです。

例

if (string_last_pos(",", text) != 0)
{
    string_insert(name, text, string_last_pos(",", text));
}

string上のコードは、変数 "text "に格納されているstring の末尾からカンマを検索しています。カンマが見つかれば、その位置に "name "という部分文字列を挿入しているのです。
