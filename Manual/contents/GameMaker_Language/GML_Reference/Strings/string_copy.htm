文字列コピー

この機能を使うと、string の中から何文字か選んで、別の文字にコピーすることが簡単にできます。string の最初の文字のインデックスは常に 1 であり、期待されるような 0 ではありません。したがって、（例えば）string の最初の 5 文字をコピーするには、 string_copy(str, 1, 5) となります。以下に、その例を示します。

構文です。

string_copy(str, index, count);

論証タイプ 商品説明
str The string to copy from.
index The position of the first character in the string to copy from (numbered from 1)
count The number of characters, starting from the position of the first, to copy

リターンです。

例

name = keyboard_string;
if (string_length(name) > 15)
{
    keyboard_string = string_copy(name, 1, 15);
}

上記のコードでは、プレーヤーがキーボードからstring を入力し、それが変数 name に格納されます。 name stringこの変数は、最大長15文字を超えていないかどうかチェックされ、超えている場合は、keyboard_stringが15文字のコピーで置き換えられます。 これは、効果的にプレーヤーの入力を15文字に制限しています。
