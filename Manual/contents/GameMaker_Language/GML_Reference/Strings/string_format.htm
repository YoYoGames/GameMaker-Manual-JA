文字列フォーマット

実数を独自の書式でstring に変換します。string に保存される「位」の数と、小数点以下の桁数を選択することができます。どちらも非常に便利で、ゲームによってはスコアを設定された桁数で表示することを好みますし、小数点以下の桁数のコントロールは string()は提供できません。指定した桁数が表示する値より大きい場合、および/または、指定した桁数に小数点以下の桁数を加えた数が合計桁数に満たない場合は、値の前にスペースを追加して差を埋めます（以下の例を参照してください）。指定された値が合計値および含めるべき小数点以下の桁数より小さい場合、小数点以下の桁数にゼロが追加されます。デフォルトの書式は、左側に余分なスペースがなく、右側に小数点以下2桁のみ、例えば "265.73 "のようなものです。

構文です。

string_format(val, tot, dec);

論証 タイプ 商品説明
val リアル The real number to be turned into a string.
tot リアル The total number of places of the main number to be shown. Zeroes or spaces will be inserted to match this.
dec リアル The number of decimal places to be included.

リターンです。

String

例

str1 = string_format(1234, 8, 0);
str2 = string_format(pi, 1, 10);
str3 = string_format(pi, 5, 5);

これにより、 str1 に「1234」、 str2 に「3.1415926535」、 str3 に「3.14159」が設定されます。
