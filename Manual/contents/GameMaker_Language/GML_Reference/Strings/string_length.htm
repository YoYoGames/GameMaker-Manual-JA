文字列の長さ

この関数は、指定されたstring を構成する文字数を返します。この関数は、カスタムテキスト入力の文字数を制限する場合などに便利です（例：プレイヤーの名前を10文字に制限する場合など）。とは異なることに注意してください。 string_width()とは異なり、string の文字数を測定するものであり、画面に描画される幅をピクセル単位で測定するものではありません。

構文です。

string_length(string);

論証 タイプ 商品説明
string 文字列 The string to measure the number of characters of.

リターンです。

Real

例

if (string_length(name) > 10)
{
    name = string_copy(name, 1, 10);
}

これは、nameのstring が10文字以上かどうかを調べ、10文字以上であれば最初の10文字だけをコピーして使用します。
