clipboard_get_text

この関数は、クリップボードに含まれるテキストをstring で返します。テキストが格納されていない場合は、空のstring ""を返します。

構文です。

clipboard_get_text();

リターンです。

String

例

if clipboard_has_text()
{
    str = clipboard_get_text();
    clipboard_set_text("");
}

上記のコードでは、クリップボードにテキストがあるかどうかをチェックし、もしテキストがあれば、string 、変数 "str "に読み込まれます。最後にクリップボードを空のstring にセットしてクリアします。
