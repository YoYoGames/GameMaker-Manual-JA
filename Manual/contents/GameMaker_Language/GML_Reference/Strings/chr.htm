クリス

この関数は、表示するための入力 Unicode コードに関連する文字を含むstring を返します。この文字は、現在の描画fonts 文字セットコードページに依存し、font が設定されていない場合は、そのデフォルトコードページが使用されます。 マシン。

構文です。

chr(val);

論証タイプ 商品説明
val The Unicode code value to get the string from

リターンです。

例

mystring = chr(53) + chr(48);

この場合、chr(53)が「5」、chr(48)が「0」なので、 mystring に「50」(整数ではなく、string)がセットされます。
