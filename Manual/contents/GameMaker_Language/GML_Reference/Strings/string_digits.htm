string_digits

この関数を使うと、与えられたstring を解析して、そこから任意の数値を得ることができます。例えば、「私は81歳です」というテキストがあったとします。この関数を使うと、"81 "という数字がstring 。

構文です。

string_digits(string);

論証タイプ 商品説明
string The string to get the digits from.

リターンです。

例

var t_str = string_digits(input_str);
age = real(t_str);

string 上記のコードは、入力string から数字以外のすべての文字を取り除き、変数 age に戻り値の実数値を格納します（したがって、入力string が、たとえば「私は18です」だった場合、このコードは 関数は "18 "を返す）。
