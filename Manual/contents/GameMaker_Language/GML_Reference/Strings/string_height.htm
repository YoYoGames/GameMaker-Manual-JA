string_height

この関数は、入力string の高さ（ピクセル）を、テキストが持つ行間や改行を考慮に入れて返します。この関数は、テキスト要素間の距離を、文字列の中で最も背の高いものから計算するのに非常に便利です。 で描かれるようなstring を構成している。 draw_text()を使用し、現在定義されているfont を使用します。

構文です。

string_height(string);

論証タイプ 商品説明
string The string to measure the height of.

リターンです。

例

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

上記のコードは、与えられたstring の高さを取得し、返されたstring の高さをセパレータとして、2行のテキストを描画します。
