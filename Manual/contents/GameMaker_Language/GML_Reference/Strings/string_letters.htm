string_letters

この機能を使うと、文字として分類されないすべての文字を削除することができます。これは、テキスト入力時に不要な文字を入力されるのを防ぐのに便利です（「#」を入力すると強制的に改行されるなど）。

注意この機能は、A～Zの26文字の英字（大文字または小文字）のみを検出して返します。

構文です。

string_letters(string);

論証タイプ 商品説明
string The string to convert to letters.

リターンです。

例

username = string_letters(username);

上記のコードでは、「ユーザー名」変数に、元のstring の文字だけをセットします（元のstring が "MacSw33ny" であった場合、返されるstring は "MacSwny" となります）。
