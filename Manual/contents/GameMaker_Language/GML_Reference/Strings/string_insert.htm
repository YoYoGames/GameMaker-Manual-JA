string_insert

この機能を使うと、2つのstring からなる新しいstrings を作成することができます。例えば、ユーザー名を定義済みのテキストに追加して、ゲームのプレイヤーを感じさせたい場合に便利です。 よりアクションに参加することができます。挿入する位置を計算するとき、strings は 1 からインデックスされるので、string の最初の文字は 1 であり、期待されるような 0 ではないことに留意してください。

構文です。

string_insert(substr, str, index);

論証タイプ 商品説明
substr The substring to be inserted.
str The string to be copied.
index The position in characters of the string to insert the substring.

リターンです。

例

str2 = string_insert(username, "Hello, , how are you?", 8);

これは、"username "変数にあるstring を与えられたフレーズに挿入し、結果としてstring は次のようになります： "Hello, NAME, how are you?".
