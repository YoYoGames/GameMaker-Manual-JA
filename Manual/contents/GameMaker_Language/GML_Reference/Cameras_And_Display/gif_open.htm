ギフオープン

この関数を使用すると、データを追加できる空のGIF形式の画像を作成できます。この関数を呼び出す際には，GIF の幅と高さを指定する必要があります（単位はピクセルで，使用可能なメモリを除いてサイズに上限はありません）．また、オプションの引数で GIF の「透明色」を設定することができます。これは RGB 値 (アルファ成分なし) で、GIF に何かを追加する前にこの色にクリアされます。クリアカラーの引数を指定しなかった場合、デフォルトの黒が使用されます。

この関数を使用する場合、必ず gif_save()を実行して GIF の作成を終了してから、別のファイルを開いて記録します (従って、 gif_open() 毎に、それに付随する gif_save() が必要です)。

構文です。

gif_open(width, height, [clear_colour]);

論証 タイプ 商品説明
width リアル The width of the gif to create
height リアル The height of the gif to create
[clear_colour] カラー OPTIONAL GIFをクリアする色(RGB)

リターンです。

GIF ID

例

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

上記のコードでは、アプリケーションの表面から30フレームを取り込んだGIF画像ファイルを作成し、保存します。
