window_set_position

この機能により、ゲームウィンドウをディスプレイ内（macOS, Linux(Ubuntu),Windows)またはブラウザ内（HTML5）の特定の位置に設定することができます。

注：HTML5ゲームでカスタム index.html を使用し、それがキャンバスを固定位置に設定する場合、この関数はウィンドウ位置に影響を与えません。

構文です。

window_set_position(x, y);

論証タイプ 商品説明
x The x coordinate of where to position the window.
y The y coordinate of where to position the window.

リターンです。

例

window_set_position(0, 0);

上記のコードは、ゲームウィンドウをブラウザまたはディスプレイの左上隅に配置します（使用するターゲットモジュールによって異なります）。
