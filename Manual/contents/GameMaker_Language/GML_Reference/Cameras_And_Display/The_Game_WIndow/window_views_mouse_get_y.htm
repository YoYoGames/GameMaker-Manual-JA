window_views_mouse_get_y

この関数は、すべてのアクティブビューに対するマウスのy座標を返し、同じ値を返します。 mouse_y.

注：通常のマウス機能については、「マウス入力」の項を参照してください。

構文です。

window_views_mouse_get_y();

リターンです。

例

myy = window_views_mouse_get_y();

この場合、myyは、すべてのアクティブなビューに対するウィンドウ内のマウスのy座標に設定されます。
