ウィンドウの設定_フルスクリーン

この機能により、ゲームウィンドウをフルスクリーン（true ）にするかしないか（false ）設定することができます。macOSをターゲットとする場合、ゲームオプションの「フルスクリーンで開始」のチェックを外し、「プレイヤーがゲームウィンドウをリサイズできるようにする」のチェックを入れていないと、この関数は失敗しますので、ご注意ください。また、この機能は、「clickable」コールバックとして追加されない限り、HTML5上で動作しないことに注意してください（詳しくはこちらをご覧ください）。

構文です。

window_set_fullscreen(full);

論証タイプ 商品説明
full Whether to set the screen to fullscreen (true) or not (false).

リターンです。

例

if mouse_check_button_pressed(mb_left)
{
    if window_get_fullscreen()
    {
        window_set_fullscreen(false);
    }
    else
    {
        window_set_fullscreen(true);
    }
}

上記のコードは、マウスボタンが押されたことを確認し、ウィンドウがまだであればフルスクリーンに、すでにあればウィンドウドに設定します。
