display_set_sleep_margin

この関数は、ゲームのレンダリングタイミングに使用するスリープマージン値を設定するために使用でき、ミリ秒の値を要求します。デフォルト値は以下の通りです。

プラットフォーム ミリ秒
ウィンドウズ 10
マクロス 10
ウブントゥ 10
HTML5 10
アンドロイド 4
iOS 4
Windows UWP 10
Xbox 10
PS4 10
ニンテンドースイッチ 10

なお、タイミング方式をvsyncタイミングに設定しても、スリープマージンはゲームの描画に何らかの影響を与えるため、この値を設定する場合は注意が必要です。表示タイミングの詳細については display_set_timing_method().

注：スリープマージンに加えて、Windows でスレッドスケジューラの解像度を調整することで、パフォーマンスと消費電力をさらに制御できます。runtime 。

構文です。

display_set_sleep_margin(milliseconds);

論証 タイプ 商品説明
milliseconds リアル The number of milliseconds for the sleep margin

リターンです。

N/A

例

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

上記のコードでは、タイミングメソッドをチェックし、 tm_sleep に設定されていなければそれに設定し、スリープマージンを20に設定します。
