display_set_gui_maximise

この関数は、GUIレイヤーを最大化し、デフォルトのアプリケーションサーフェス位置と寸法ではなく、スクリーン寸法に関連してスケールとオフセットを設定するために使用することができます。デフォルトでは、GUIレイヤーはアプリケーションサーフェスの解像度と1：1で、サーフェスの（0,0）位置にも描画されます。しかし、これは常に望むものではないので、この関数を使用して、ディスプレイまたはゲームウィンドウの絶対位置（0,0）に対して相対的に描画されるように設定することができます。この関数に与える引数は、GUIレイヤーに与えたい効果に依存します。引数なしでこの関数を呼び出すと、GUIレイヤーはスクリーンまたはゲーム・ウィンドウの(0,0)の位置に描画され、幅と高さは領域全体にフィットするようにスケーリングされるように設定されます。

display_set_gui_maximise();

ただし、GUIレイヤーのスケーリングファクターを設定すると、幅と高さがその分拡大されます。GUIレイヤーは常にディスプレイやゲームウィンドウ、またはアプリケーションサーフェイスのサイズに合わせて作られるため、この値を1以外に設定すると、基本的に幅と高さに沿って画素数がスケールされることを覚えておいてください。ディスプレイが1024x768で、スケーリングファクターを0.5に設定すると、GUIレイヤーはディスプレイの半分のサイズになり、事実上ピクセルサイズは2倍になります。

display_set_gui_maximise(0.5, 0.5);

このように値を設定すると、描画位置もディスプレイやゲームウィンドウの(0,0)に設定されるので、オフセット値を与えて(0,0)の位置を「移動」させることができるようになります。

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

最後に、この関数を使用してGUIレイヤーをリセットするには、スケーリング係数を-1に設定します。これにより、GUIレイヤーは再び1:1のスケールを持つようになり、描画位置はディスプレイやウィンドウではなく、アプリケーションサーフェスの(0,0)の位置に設定されます。

display_set_gui_maximise(-1, -1);

構文です。

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

論証 タイプ 商品説明
xscale リアル OPTIONAL 水平方向のスケーリングファクター（デフォルトに戻すには-1を使用）。
yscale リアル OPTIONAL 垂直方向のスケーリングファクター（デフォルトに戻すには-1を使用）。
xoffset リアル OPTIONAL 描画のためのXオフセット位置。
yoffset リアル OPTIONAL 描画のためのyオフセット位置。

例

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

上記のコードは、draw GUIイベントを与えられた幅と高さに固定し、その比率ですべてのコンポーネントをディスプレイに合うようにスケーリングします。
