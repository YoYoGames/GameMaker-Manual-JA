display_get_timing_method

この関数は、ゲームのレンダリングに使用されるタイミングメソッドを取得するために使用できます。メソッドには、以下に示す定数のいずれかを指定します。

各タイミング方法の詳細は、以下をご覧ください。 display_set_timing_method().

構文です。

display_get_timing_method();

リターンです。

Display Timing Method Constant

表示タイミング方式 一定
定数 商品説明
tm_sleep スリープマージン値は、主なタイミング方式
tm_countvsyncs Vsyncタイミングが主なタイミング方法（すべてのサポート対象プラットフォームでデフォルト）

例

if display_get_timing_method() != tm_sleep
{
    display_set_timing_method(tm_sleep);
    if display_get_sleep_margin() != 20
    {
        display_set_sleep_margin(20);
    }
}

上記のコードでは、タイミングメソッドをチェックし、 tm_sleep に設定されていなければそれに設定し、スリープマージンを20に設定します。
