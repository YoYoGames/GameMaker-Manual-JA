gif_save_buffer

この関数を使うと、GIFアニメーションを保存することができます。GIF のインデックス（この関数が返すもの）を指定します。 gif_open()この関数は、GIF データを 1 バイトアラインの growbuffer で返します。なお、最終的なGIFデータはUniversal 884 Paletteでパレタイズされています（詳しくはこちらをご覧ください）。

構文です。

gif_save_buffer(gif_index);

論証タイプ 商品説明
gif_index The ID of gif to save

リターンです。

Buffer ID

例

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        global.capture_buff = gif_save_buffer(gif_image);
        count = 0;
        save_gif = false;
    }
    count++;
}

上記のコードでは、アプリケーションの表面から撮影した30フレームのGIF画像ファイルを作成し、buffer.
