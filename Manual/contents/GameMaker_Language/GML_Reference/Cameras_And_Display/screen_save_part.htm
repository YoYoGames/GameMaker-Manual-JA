スクリーンセーブパート

この機能を使うと、現在ゲーム画面に表示されている部分のスクリーンショットを保存することができます。選択座標はディスプレイ（フルスクリーンでない場合はウィンドウ）の絶対サイズに基づいているので、（0,0）は常に画面（またはウィンドウ）の左上隅になります。下の画像は、この機能がどのように動作するかを示しています。

このファイルは、ゲームの作業ディレクトリに保存されます。この関数は、Draw GUI End Eventで呼ばれるように設計されており、すべてのプラットフォームで一貫した結果が得られるはずであることに注意することが重要です。他のイベントでこの関数を使用すると、期待通りに動作しない可能性があり、異なるターゲットやデバイスでも異なる結果をもたらすかもしれません。

注意事項 この機能は、HTML5ターゲットでは動作しません。

構文です。

screen_save_part(fname,x,y,w,h);

論証タイプ 商品説明
fname The filename for the screenshot to save
x The x coordinate of the top left corner of the area of the screen to capture
y The y coordinate of the top left corner of the area of the screen to capture
w The width of the area of the screen to capture
h The height of the area of the screen to capture

リターンです。

例

if keyboard_check_pressed(ord("S"))
{
    screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)
}

上記のコードでは、ユーザーが「S」キーを押したときに、ゲーム画面の100ピクセル四方の部分のスクリーンショットを撮影します。このスクリーンショットは特定のフォルダに保存され、変数を使って番号が振られ、それがインクリメントされます。
