display_set_ui_visibility

この関数は、Android のみでシステム UI の表示／非表示を切り替えるために使用できます。この関数では、1つまたは複数のシステムフラグを整数値で指定する必要があります。複数のシステムフラグを使用する場合、flag 、以下の例に示すように、ビット単位の "or "を使用してこれらをマージする必要があります。Android システムflags の一覧は、こちらでご覧いただけます。

構文です。

display_set_ui_visibility(flags);

論証 タイプ フラグ
flags リアル The Android SYSTEM flags to use (as an integer value).

リターンです。

N/A

例

var flags = 1024|4096;
display_set_ui_visibility(flags);

上記のコードでは、Android システムflags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" と "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" を使用してディスプレイの表示状態を設定します。
