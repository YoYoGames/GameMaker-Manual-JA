display_reset

この機能は、ゲーム起動時の表示設定に戻すだけでなく、現在使用しているフルスクリーンアンチエイリアスのレベルや垂直同期を使用するかどうかを変更することが可能です。使用可能なアンチエイリアスのレベルは0、2、4、8で、起動時のデフォルト値は0に設定され、Vシンクのデフォルト設定はfalse（オフ）になっています。Vシンクをオンにすると、よりスムーズなゲーム体験が得られますが、より多くの処理能力を必要とするため、その影響は使用前に慎重に検討する必要があります。

ターゲットデバイスはすべて同じではないので、たとえば8倍や4倍のアンチエイリアスをサポートしないものもあります。そのため、ゲームを実行しているデバイスが表示できるさまざまなレベルのAAを取得するための読み取り専用変数が用意されています。

display_aa

この変数は、異なるレベルのビットの設定に基づいた整数値を返す。つまり、2xAAのみの場合は2、2xと4xの場合は6を返します。8倍と4倍の場合は12を返します。3つのレベルすべて(2,4,8)の場合は、14を返します。

構文です。

display_reset(aa, vsync);

論証 タイプ 商品説明
aa リアル The level of anti-aliasing filtering (0, 2, 4 or 8).
vsync ブーリアン Toggle vertical synchronisation to on (true) or off (false).

リターンです。

Real

例

if (display_aa > 12)
{
    display_reset(8, true);
}

上記のコードでは、サポートされている場合はアンチエイリアシングレベルを8に設定し、switch v-syncをオンに設定します。
