ギフセーブ

この関数を使うと、GIFアニメーションを保存することができます。GIF のインデックス（この関数が返すもの）を指定します。 gif_open()) と共に保存するファイル名を指定します。なお、GameMakerは .gif ファイルextension を自動的に付加しないので、保存したファイルが GIF であることを確認したい場合は、ファイル名string の一部としてこれを含める必要があります。作成されたGIFはUniversal 884 Paletteを使ってパレタイズされます（詳しくはこちらをご覧ください）。

この関数が成功し、GIF が正しく保存された場合は 0 を、そうでない場合は -1 を返すことに注意してください。

構文です。

gif_save(gif_index, fname);

論証タイプ 商品説明
gif_index The ID of gif to save
fname The filename to use for the gif

リターンです。

例

if save_gif == true
{
    if (count == 0)
    {
        gif_image = gif_open(room_width, room_height);
    }
    else if (count < 30)
    {
        gif_add_surface(gif_image, application_surface, 6/100);
    }
    else
    {
        gif_save(gif_image, "GameCapture.gif");
        count = 0;
        save_gif = false;
    }
    count++;
}

上記のコードでは、アプリケーションの表面から30フレームを取り込んだGIF画像ファイルを作成し、保存します。
