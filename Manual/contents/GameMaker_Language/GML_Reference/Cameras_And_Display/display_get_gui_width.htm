ディスプレイの幅

この関数を使用すると、GUI の描画イベントで使用される GUI の幅（ピクセル単位）を取得することができます。

構文です。

display_get_gui_width();

リターンです。

例

ads_move(display_get_gui_width() - ads_get_display_width(0), 0, 0);

上記のコードで、ディスプレイの右上に広告を表示するように設定します。
