screen_save

この機能を使うと、現在ゲームウィンドウに表示されているゲームのスクリーンショットを保存することができます。

このファイルは、ゲームの作業ディレクトリに保存されます。Windowsの場合は <drive>:/Users/YOURUSERNAME/AppData/Local/[Game Name]/ 、Macの場合は ~/Library/Application Support/[Game Name]/ に位置します。キャプチャされた画像は、ゲームのアプリケーションサーフェスの最終レンダリング、または（アプリケーションサーフェスが無効になっている場合）ディスプレイまたはウィンドウ全体の画像となります。

この関数はDraw GUI End Event で呼び出されるように設計されており、すべてのプラットフォームで一貫した結果が得られるはずであることに注意することが重要です。他のイベントでこの関数を使用すると、期待通りに動作しない可能性があり、異なるターゲットやデバイスでも異なる結果をもたらす可能性があります。

保存される内容をより詳細に制御する必要がある場合は、関数 screen_save_part().

注意 この機能は、HTML5ターゲットでは動作しません。

構文です。

screen_save(fname);

論証 タイプ 商品説明
fname 文字列 The filename for the screenshot to save

リターンです。

N/A

例

if keyboard_check_pressed(ord("S"))
{
    screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png")
}

上記のコードでは、ユーザーが「S」キーを押したときに、ゲームのスクリーンショットを撮影します。このスクリーンショットは特定のディレクトリに保存され、変数を使って番号が振られ、それがインクリメントされます。
