ディスプレイサイズ

この関数は、Draw GUI イベントで描画されるものにのみ影響し、そのイベントですべての GUI コンポーネントに特定の幅と高さを設定するために使用でき、どのような GUIを表示するために使用されるディスプレイまたはウィンドウのサイズは、それが正しくフィットするようにスケーリングされます。GUIがどこに配置されるか（つまり、(0,0)の位置）は、この関数を使用したかどうかによって異なります。 display_set_gui_maximise() を使用します。この機能を使用しない場合（デフォルト設定）、GUIレイヤーの（0,0）点はアプリケーションサーフェス（ゲームが描画される領域）と同じ位置に配置され、幅と高さは、アプリケーションサーフェスの幅と高さになります。 例えば、アプリケーションサーフェスが512pxのときに、この関数でGUIの幅を256pxに設定すると、その256ピクセルがストレッチされてフィットし、実質的にピクセルが2倍になるように設定されます。これは、以下のようなゲームに最適です。 グローバルゲーム設定でアスペクト比補正が設定されている場合。

display_set_gui_maximise() 関数を使用した場合、GUI レイヤーの (0,0) 位置は、ゲーム ウィンドウまたはディスプレイ（フルスクリーンまたはデバイスの場合）の左上隅がデフォルトとなり、この関数で設定されたスケーリング係数が使用されます。 で設定され、幅と高さはこちらで設定されます。この関数を幅と高さに-1を指定して呼び出すと、いつでもGUIレイヤーをデフォルトの設定に戻すことができます - この場合、アプリケーションサーフェスの位置が(0, 0)になり、幅と高さが-1になります。 と高さも表面と1:1にリセットされます。

構文です。

display_set_gui_size(width, height);

論証タイプ 商品説明
width The width of the GUI
height The height of the GUI

リターンです。

例

display_set_gui_size(768, 1024);

上記のコードは、draw GUIイベントを与えられた幅と高さに固定し、その比率でアプリケーションの表面またはディスプレイにフィットするようにすべてのコンポーネントをスケーリングします。
