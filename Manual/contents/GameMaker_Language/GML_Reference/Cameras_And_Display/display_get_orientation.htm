display_get_orientation

この関数は、GameMakerが持つ4つの定数のうちの1つを返し、ゲームを実行しているデバイスがランドスケープモードで保持されているかポートレートモードで保持されているかを知らせます（下の表を参照）。

なお、この機能をHTML5のターゲットモジュールで使用する場合、デバイスの向きを正しく検出できないことがあります。しかし、以下のスクリプトを使用することで、これを簡単に模倣することができます。

return (browser_width < browser_height);

このような関数は、縦長の場合は true を、横長の場合は false を返すことになる。

構文です。

display_get_orientation()

リターンです。

Display Orientation Constant

表示オリエンテーション定数
定数 商品説明
display_landscape 端末を横向きに持ち、左から右に長い辺があり、右側にメニューボタンがある状態。
display_landscape_flipped 上記と同様、メニューボタンが左側に表示されるようになっただけです。
display_portrait デバイスを縦に持ち、上から下に長い辺があり、メニューボタンが下にある状態です。
display_portrait_flipped 上記と同様、メニューボタンが上部に表示されるようになっただけです。

例

if display_get_orientation() == display_landscape
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

上記のコードでは、デバイスの向きをチェックし、返された値に応じてグローバル変数をセットします。
