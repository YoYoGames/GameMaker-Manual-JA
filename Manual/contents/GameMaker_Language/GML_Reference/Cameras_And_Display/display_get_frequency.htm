display_get_frequency

この関数は、ゲームがプレイされているディスプレイの周波数（またはリフレッシュレート）を返します。例えばモニターが60Hzなら60、144Hzなら144というように、実際の値を返します。

注）この機能は、HTML5には対応していません。

構文です。

display_get_frequency();

リターンです。

例

var _freq = display_get_frequency();

if (_freq >= 240)
{
    game_set_speed(240, gamespeed_fps);
}
else if (_freq >= 120)
{
    game_set_speed(120, gamespeed_fps);
}
else if (_freq >= 60)
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

上記のコードは、ディスプレイの周波数を取得し、いくつかの条件を実行して、ゲームを240、120、60、30fpsで動作するように設定します。つまり、ディスプレイが90Hzの場合は60fpsで、144Hzの場合は120fpsでゲームが動作する、ということです。もちろん、ディスプレイの周波数を直接 game_set_speed() に渡して、ゲームのフレームレートとして使用することもできます。
