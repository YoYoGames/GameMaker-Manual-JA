カメラ・ゲット・ビュー・マット

この関数は、ビューマトリックスを取得するために使用することができます。この関数は，行列の配列を返します．この配列は，他の行列関数で利用したり，別のカメラのビュー行列を設定したりするために利用することができます（ただし，この関数は camera_set_view_mat()).この関数は一般的に camera_create()しかし、room エディタで作成されたものにも使用できます。 camera_create_view()この場合，返される行列は，与えられたカメラがビューポートで初めて使用された後にのみ有効となります．

構文です。

camera_get_view_mat(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

Matrix Array

例

mat = camera_get_view_mat(camera_view[0]);

上記のコードでは、ビューポート[0]に割り当てられたカメラのビューマトリックスを取得しています。
