camera_set_default

この機能を使うと、デフォルトカメラに、以前にいずれかの方法で作成したカスタ ムカメラを使用するように設定することができます。 camera_create()関数を使用します。アクティブなビューポートがない状態でroom を作成した場合 やビューカメラを使用しても、GameMakerはゲーム内のアクションを表示するためにカメラを使用します。このカメラはデフォルトカメラと呼ばれ、他のカメラと同じように設定や操作（破壊も可能）ができます。

構文です。

camera_set_default(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

例

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

上記のコードでは、新しいカメラを作成し、それをデフォルトのカメラとして使用するように割り当てています。
