camera_get_begin_script

この関数は、指定されたカメラの beginscript として割り当てられているスクリプト関数のID を取得するために使用されます。script 関数が割り当てられていない場合、この関数は -1 を返します。

構文です。

camera_get_begin_script(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

Script Function or -1 if no function is assigned

例

var scr = camera_get_begin_script(camera_view[0]);
if (scr != cutScene)
{
    camera_set_begin_script(view_camera[0], cutscene);
}

上記のコードでは、ビューポート[0]に割り当てられたカメラの beginscript として割り当てられたscript 関数をチェックし、それが "cutScene" でない場合はその関数に設定されます。
