camera_get_proj_mat

この関数は、投影行列を取得するために使用することができます。この関数は，行列の配列を返します．この配列は，他の行列関数で利用したり，別のカメラの投影行列を設定したりするために利用されます（以下のように利用されます）． camera_set_proj_mat()).

構文です。

camera_get_proj_mat(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

Matrix Array

例

mat = camera_get_proj_mat(camera_view[0]);

上記のコードでは、ビューポート[0]に割り当てられたカメラの投影行列を取得しています。
