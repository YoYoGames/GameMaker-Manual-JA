view_enabled

この変数は、room 内に表示されているビューポートが有効かどうかを制御します。ビューポートを可視に設定し、このオプションを無効にすると、ビューポートを通して異なるカメラが描画されるのではなく、room 全体がウィンドウサイズにスケーリングされて画面に描画されるようになります。

構文です。

view_enabled

リターンです。

(true: enabled, false: disabled)

例

if !view_enabled
{
    view_visible[0] = true;
    view_enabled = true;
}

上記のコードは、room に対してビューが有効かどうかをチェックし、有効でない場合は、ビューポート[0]が表示されていることを確認した上で、room に対してビューを有効にしています。
