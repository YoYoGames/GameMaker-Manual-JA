view_get_xport

この関数は、与えられたビューポートの x 位置を取得するために使用することができます。

構文です。

view_get_xport(view_port)

論証タイプ 商品説明
view_port The view port to target (0 - 7)

リターンです。

Real

例

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
    view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

上記のコードは、ビューポート[0]のx位置をチェックし、必要な位置にない場合はその位置に設定されます。
