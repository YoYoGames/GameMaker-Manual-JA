camera_set_update_script

この関数は、カメラが可視かつアクティブなビューポートに割り当てられているゲームフレームごとに呼び出されるスクリプト関数を設定するために使用することができます。一意のカメラ ID 値（さまざまな camera_create 関数によって返される）と、呼び出される関数の ID を指定します。カメラに割り当てられた関数と、カメラビューの実際のレンダリングの順序は以下のとおりです。

  • すべての可視およびアクティブなビューポートのカメラには、その更新関数として
  • 次に、各ビューについて。
    • そのビューのカメラのbegin関数が 呼び出されます。
    • そのビューの描画イベントが実行される
    • カメラの終了関数が 呼び出される
  • 次のアクティブな可視ビューに移動して、繰り返します...

カメラにカスタムアップデート関数を適用すると、そのデフォルトのアップデート動作が上書きされます。たとえば、 camera_create_view() を呼び出したときに、カメラがobject (たとえばobj_player) に従うように設定した場合、またはRoom プロパティでそのように設定した場合、カスタムアップデートobject が設定されると、そのscript への追従が停止されます。

構文です。

camera_set_update_script(camera_id, script)

論証 タイプ 商品説明
camera_id カメラID The unique camera ID value returned when you created the camera.
script スクリプト機能 The script function to run each game frame

リターンです。

N/A

例

camera_set_update_script(view_camera[0], updateCamera);

上記のコードでは、ビューポート[0]に割り当てられたカメラの更新機能を設定しています。
