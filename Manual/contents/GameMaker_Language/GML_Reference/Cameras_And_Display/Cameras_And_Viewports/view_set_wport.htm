view_set_wport

この関数は、与えられたビューポートの幅を設定するために使用することができます。ビューポートのインデックス（0から7まで）と、使用する新しい幅を指定します。

構文です。

view_set_wport(view_port, w)

論証タイプ 商品説明
view_port The view port to target (0 - 7)
w The new width (in pixels)

リターンです。

例

if view_get_wport(0) != (display_get_width () / 2)
{
    view_set_wport(0, display_get_width() / 2);
}

上記のコードは、ビューポート[0]の幅をチェックし、それがディスプレイ幅の半分でなければ、その値に設定されます。
