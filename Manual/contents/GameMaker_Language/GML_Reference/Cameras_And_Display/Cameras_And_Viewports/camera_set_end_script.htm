カメラセット終了スクリプト

この関数は、カメラが可視かつアクティブなビューポートに割り当てられ、そのビューカメラのすべてがレンダリングされた後、各ゲームフレームの終わりに呼び出されるスクリプト関数を設定するために使用することができます。カメラIDの値を指定します。 camera_create()関数)と呼び出す関数のIDを指定します。カメラにアタッチされた関数と実際のカメラビューのレンダリングの順序は以下の通りです。

  • すべての可視およびアクティブなビューポートのカメラには、その更新関数として
  • 次に、各ビューについて。
    • そのビューのカメラのbegin関数が 呼び出されます。
    • そのビューの描画イベントが実行される
    • カメラの終了関数が 呼び出される
  • 次のアクティブな可視ビューに移動して、繰り返します...

構文です。

camera_set_end_script(camera_id, script)

論証 タイプ 商品説明
camera_id カメラID The unique camera ID value returned when you created the camera.
script スクリプト機能 The script to run each game frame

リターンです。

N/A

例

camera_set_end_script(view_camera[0], endCamera);

上記のコードでは、ビューポート[0]に割り当てられたカメラのエンドscript を設定します。
