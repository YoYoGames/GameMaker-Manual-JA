ビューイングビジブル

この変数は、特定のビューポートが現在表示されているかどうかを調べるために使用することができます。また、この変数の値を true (visible) または false (invisible) に設定することで、効果的にビューを "on" または "off" にすることができます。ビューポートをvisibleに設定しても、ビューポートが有効でない場合（組み込み変数 view_enabledまたはRoom Editor で有効にした場合、画面には描画されません。

構文です。

view_visible[0 ... 7];

リターンです。

(true: enabled, false: disabled)

例

if !view_visible[0]
{
    view_visible[0] = true;
}

上記のコードでは、view[0]が可視かどうかをチェックし、可視でない場合は、可視になるように設定しています。
