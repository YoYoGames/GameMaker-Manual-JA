ビューカメラ

この配列は、指定されたビューポートに割り当てられた固有のカメラ ID を保持し、新しいカメラを設定したり、現在のカメラを読み込んで取得したりすることができます。room では、最大 8 つのビューポートをアクティブにできます（配列の値は 0 から 7）。 camera_create()または camera_create_view()).Room Editorでビューポートにカメラを設定した場合、この変数にはそのポート用のカメラIDが格納されます。

構文です。

view_camera[0 ... 7];

リターンです。

Camera ID or -1 (if no camera is assigned)

例

if view_camera[0] == -1
{
    view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

上記のコードでは、ビューポート[0]にカメラが接続されているかどうかを確認し、接続されていない場合はカメラを作成し、接続されているカメラに設定します。
