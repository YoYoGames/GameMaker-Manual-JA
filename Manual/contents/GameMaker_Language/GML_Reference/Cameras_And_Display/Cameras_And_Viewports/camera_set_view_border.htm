カメラのビュー枠を設定する

この関数は、room 内のビューカメラのフォローボーダーを設定するために使用します。 一意のカメラ ID 値（異なる camera_create 関数が返すもの）を与え、x ボーダーサイズと y ボーダーサイズ（水平方向）を与えます。 と垂直方向)。これらの値は、ビューカメラにフォローするインスタンスターゲットが割り当てられている場合にのみ使用されます（Room Editor または関数 camera_set_view_target()) に関連しており は、インスタンスがビューの境界線からどのくらい離れていれば、ビューはその位置を更新して追従するかということです。

構文です。

camera_set_view_border(camera_id, x_border, y_border)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.
x_border The border (in pixels) for the view camera along the horizontal axis
y_border The border (in pixels) for the view camera along the vertical axis

リターンです。

例

camera_set_view_border(view_camera[0], 64, 64);

上記のコードでは、ビューポート[0]に割り当てられたカメラに対してビューカメラの境界線を設定します。
