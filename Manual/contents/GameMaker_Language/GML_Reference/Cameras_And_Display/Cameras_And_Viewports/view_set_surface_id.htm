view_set_surface_id

この変数を使うと、サーフェスに描画するビューポートの内容を設定することができます。サーフェスを扱う場合、画面の可視領域全体をサーフェスに取り込むことが必要になることが多いので、この関数を使ってビューポートに割り当てることになります。これにより、選択したビューポートに表示されているものは全て割り当てられたサーフェスに描画されるようになり、そのビューポートの内容は表示されなくなるので、どちらかが必要ということになります。

  • 新しいビューを有効にし、そのビューにのみサーフェスを描画します。 view_currentどのビューで描画されているかを確認するため）
  • これらのイベントはビューに依存しないので、インスタンスの Draw GUI または Post Draw イベントでサーフェイスを描画します。

この関数を使用する場合、ビューポートインデックス（0から7まで）とサーフェスインデックス（0から7まで）を指定します。 application_surfaceまたは、この関数が返すユニークなインデックス値 surface_create()または、以前にサーフェスが割り当てられていて、それを取り除きたい場合は、値 -1 を指定します。 ビューポートをサーフェスに設定するための詳しい例は、変数 view_surface_id.

構文です。

view_set_surface_id(view_port, surf)

論証 タイプ 商品説明
view_port リアル The view port to target (0 - 7)
surf サーフェスID The surface to apply to the view

リターンです。

Real

例

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

上記のコードは、ビューポート[0]にサーフェイスが割り当てられているかどうかを確認し、割り当てられていない場合は、割り当てます。
