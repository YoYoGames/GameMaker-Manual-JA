view_current

この読み取り専用変数は 描画イベントで、現在レンダリングされているビューポートを返します。draw イベントは各ビューポートごとに連続して呼び出されるため、さまざまなビューがある場合、戻り値は draw イベント中に変化します。そのため、例えばゲーム内でビューポート[0]とビューポート[1]を使っている場合room 、ALLインスタンスのdrawイベントがポートごとに2回実行され、この変数で現在どのビューが描画されているかを確認することができます。一般的には、複数のビューポートがroom に同時に表示されているときに、1つのポートにのみ特定の詳細をレンダリングするために使用されます。以下のコード例を参照してください。

構文です。

view_current

リターンです。

Real (view port index from 0 to 7)

例

if view_current == 0
{
    var _xx = camera_get_view_x(view_camera[0]);
    var _yy = camera_get_view_y(view_camera[0]);
    draw_text(_xx + 32, _yy + 32, "Player 1");
}
else
{
    var _xx = camera_get_view_x(view_camera[1]);
    var _yy = camera_get_view_y(view_camera[1]);
    draw_text(_xx + 32, _yy + 32, "Player 2");
}

上記のコードでは、現在どのビューが描画されているかを確認し、その戻り値に基づいて各ビューに異なるテキストを描画しています。
