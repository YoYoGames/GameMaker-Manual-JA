camera_get_default

この関数は、デフォルトカメラ（ゲーム内でカメラビューやポートがアクティブでないときにGameMaker が使用するカメラroom ）のユニークなカメラ ID 値を取得するために使用することができます。

構文です。

camera_get_default()

リターンです。

Camera ID

例

var def = camera_get_default();
view_camera[0] = def;

上記のコードでは、デフォルトカメラのカメラIDを取得し、それを使用するようにport[0]のビューカメラを設定します。
