カメラ破壊

この関数で、任意のカメラを破壊することができます。この関数を呼び出す際には、一意のカメラID値を与えます。この値は camera_create_* 関数または view_cameraRoom Editor を使ってビューポートとビューを設定した場合は、配列が表示されます。また、メモリリークを防ぐため、ゲームに不要になったらコードで作成したカメラは必ず破棄してください。また、Room エディタでデフォルトカメラを割り当てている場合は破棄しても構いませんが、ビューポートに新しいカメラを割り当てる（または無効にする）必要があり、そうしないとおかしな結果を招くことになります。

構文です。

camera_destroy(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.

リターンです。

N/A

例

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

上記のコードでは、現在ビューポート[0]に割り当てられているカメラを破棄し、新しいカメラを作成してそのIDをポートに割り当てています。
