カメラアプリ

この関数は、与えられたカメラ設定を、現在レンダリング中のカメラビューに即座に適用します。特定の camera_set_* 関数を使用して任意のビューカメラを設定できますが、新しい設定は次のゲームフレームまでレンダリングに使用されませんが、この関数を使用すればすぐに変更することができます。複数のビューを使用する場合、この関数は描画イベントで使用する必要があり、変数 view_currentを使用すると、現在レンダリング中のカメラビューにのみ、指定したカメラを選択的に適用することができます。また、この関数をデフォルトカメラ（ビューカメラがアクティブでないときにGameMakerが使用する内部カメラ）に使用した場合、変更は次のゲームフレームで内部設定により上書きされます（room ）。

構文です。

camera_apply(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.

リターンです。

例

if view_current == 0
{
    camera_apply(cutscene_cam);
}

上記のコード（描画イベント内）は、現在どのカメラビューがレンダリングされているかを確認し、それがカメラビュー[0]であれば、変数「cutscene_cam」で参照されるカメラの設定が適用されます。
