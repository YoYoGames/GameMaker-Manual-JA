view_get_visible

この関数は、与えられたビューポートの可視性をチェックするために使用することができます。この関数は、表示されている場合は true を、表示されていない場合は false を返します。

構文です。

view_get_visible(view_port)

論証タイプ 商品説明
view_port The view port to target (0 - 7)

リターンです。

Boolean

例

if !view_get_visible(0)
{
    view_set_visible(0, true);
}

上記のコードでは、ビューポート[0]が可視であるかどうかを確認し、可視でない場合は可視に設定されます。
