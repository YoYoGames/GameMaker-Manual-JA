カメラの境界線を取得します。

この関数は、指定されたカメラの次のオブジェクト/インスタンスの X 軸方向のボーダー値（水平ボーダー）を取得するために使用することができます。戻り値はピクセル単位となります。

構文です。

camera_get_view_border_x(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

例

var xb = camera_get_view_border_x(view_camera[0]);
var yb = camera_get_view_border_y(view_camera[0]);
if xb != 200 || yb != 200
{
    camera_set_view_border(view_camera[0], 200, 200);
}

上記のコードは、view port[0]に割り当てられたカメラのxborderとyborderの値を取得し、それが与えられた値と一致するかどうかをチェックします。一致しない場合、ビューカメラは指定された値に設定されます。
