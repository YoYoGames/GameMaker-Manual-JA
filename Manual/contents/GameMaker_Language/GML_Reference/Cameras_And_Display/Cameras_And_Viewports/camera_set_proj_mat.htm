カメラの設定_proj_mat

この関数は、指定されたカメラの投影行列を設定します。この関数は，指定されたカメラの投影行列を設定します． camera_create()関数)と，適用する投影行列を指定します。投影行列の作成方法については，セクション「行列関数」，特に matrix_build_projection_perspective() と matrix_build_projection_ortho().

この機能を利用する際には、いくつかの注意点があります。

  • カメラが自動的にobject トラッキングを行う場合、つまり camera_create_view()で、object index / instance IDが-1でない場合、またはRoom Editorで定義したカメラを設定している場合、またはデフォルトカメラを設定している場合、この行列は次のゲームフレームで上書きされます。
  • この関数を使ってswitch を（デフォルトの正投影ではなく）透視投影にすると、タイル状のsprites の自動描画は機能しなくなり、手動で処理する必要があります。これには、水平または垂直タイリングを使用する背景レイヤー、および draw_sprite_tiled() と draw_sprite_tiled_ext() 関数が含まれ、透視投影が使用されている場合は、タイリングする代わりにsprite を一度だけ描画します。

構文です。

camera_set_proj_mat(camera_id, matrix)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

リターンです。

例

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上記のコードでは、新しいカメラを作成し、ビューポート[0]に割り当てています。このカメラには、ビューマトリックスとプロジェクションマトリックスが設定されます。
