カメラで撮影した画像を表示する

説明： この関数は、与えられたカメラのフォローターゲットを取得するために使用できる。戻り値はobject のインデックス、インスタンス ID、またはフォローターゲットが設定されていない場合は -1 となる。

構文です。

camera_get_view_target(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

Object Asset, Instance ID or -1

例

var target = camera_get_view_target(view_camera[0]);
if (target != obj_Player)
{
    camera_set_view_target(view_camera[0], obj_Player);
}

上記のコードでは、ビューポート[0]に割り当てられたカメラのフォローターゲットを取得し、これがobject のインデックス「obj_Player」と一致するかどうかをチェックします。一致しない場合、ビューカメラはそのobject のインスタンスをフォローするように設定されます。
