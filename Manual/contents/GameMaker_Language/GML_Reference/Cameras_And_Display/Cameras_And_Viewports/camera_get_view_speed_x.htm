camera_get_view_speed_x

この関数は、指定されたカメラの X 軸方向の移動速度（水平移動）を取得するために使用することができます。戻り値は、ゲームフレームあたりのピクセル数です。

構文です。

camera_get_view_speed_x(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

例

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
    camera_set_view_speed(view_camera[0], 5, 5);
}

上記のコードは、view port[0]に割り当てられたカメラのxspeedとyspeedを取得し、それが与えられた値と一致するかどうかをチェックします。一致しない場合、ビューカメラはその速度に設定されます。
