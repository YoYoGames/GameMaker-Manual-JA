ビューカメラの設定

この関数は、特定のビューポートにカメラを設定します。設定するビューポートを指定し（0から7まで）、一意のカメラID値を与えます（この値は camera_create()を使用した場合、または view_get_camera()). カメラ ID として -1 を指定すると、ビューポートからカメラを削除することになり、そのビューポートが有効かつ可視である場合、予期しない結果が得られる可能性があることに注意してください。

構文です。

view_set_camera(view_port, camera_id)

論証タイプ 商品説明
view_port The view port to target (0 - 7)
camera_id The unique camera ID value returned when you created the camera

リターンです。

例

var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
view_set_camera(0, cam);

上記のコードでは、新しいカメラを作成し、それをビューポート[0]に割り当てています。
