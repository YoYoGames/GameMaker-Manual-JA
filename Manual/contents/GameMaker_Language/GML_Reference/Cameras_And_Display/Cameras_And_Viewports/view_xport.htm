ビューエクスポート

この内蔵の配列を使って、指定したビューポートの x 位置を取得または設定することができます。ビューポートとは、画面上のビューを描画する領域のことで、一度に最大 8 つまでアクティブにできます (配列の値は 0 から 7 までで、合計 8 つになります)。 ポート)。さて、GameMakerのデフォルトでは、ゲームウィンドウ（または背景キャンバス）はroom と同じサイズですが、ビューポートやカメラを有効にすると、この動作は変わり、すべてのビューポートのバウンディングボックスの合計サイズが使用されます。つまり、2つの異なる位置に2つのビューポートがある場合、それらがカバーする領域の合計がゲームウィンドウの大きさを決定します。次の画像は、これを説明するためのものです。

ビューポートが1つしかない場合、ポートのxまたはyの値を設定することで、ビューポートを変更することができます。 には面白い効果がありますが、通常は実用的ではありません（下図参照）ので、xとyの位置は（0,0）のままにしておくのがよいでしょう。

構文です。

view_xport[0 .. 7];

リターンです。

例

view_xport[0] = 0;
view_yport[0] = 0;

上記のコードは、ビューポート[0]の位置をディスプレイの(0,0)の位置（左上隅）にリセットしています。
