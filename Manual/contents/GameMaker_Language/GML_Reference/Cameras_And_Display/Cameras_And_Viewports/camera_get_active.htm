camera_get_active

現在アクティブなカメラの固有カメラ ID 値を取得するために使用することができます。

構文です。

camera_get_active()

リターンです。

Camera ID

例

var active = camera_get_active();
if active != view_camera[0]
{
    view_camera[0] = active;
}

上記のコードでは、アクティブなカメラのカメラIDを取得し、それを使用するようにport[0]のビューカメラを設定しています。
