view_get_surface_id

この関数は、指定されたビューポートに割り当てられたサーフェイスの一意のID値を取得するために使用できます（サーフェイスが割り当てられていない場合は-1が返されます）。

構文です。

view_get_surface_id(view_port)

論証タイプ 商品説明
view_port The view port to target (0 - 7)

リターンです。

Surface ID

例

if view_get_surface_id(0) == -1
{
    view_set_surface_id(0, global.vSurf);
}

上記のコードは、ビューポート[0]にサーフェイスが割り当てられているかどうかを確認し、割り当てられていない場合は、割り当てます。
