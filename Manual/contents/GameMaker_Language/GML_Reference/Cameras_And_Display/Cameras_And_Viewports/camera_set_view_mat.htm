カメラの設定_ビューマット

この関数は、指定されたカメラのビューマトリクスを設定します。この関数は，指定されたカメラのビューマトリクスを設定します． camera_create()関数) と適用するビューマトリクスを指定します。ビューマトリックスの作成方法については、マトリックス関数のセクションで詳しく説明しています。 matrix_build_lookat().

カメラが自動的にobject トラッキングを行う場合、つまり、object インデックス/インスタンス ID が -1 でない camera_create_view() を使用して作成されている場合、またはRoom エディタで定義したカメラを設定している場合、またはデフォルトカメラを設定している場合、次のゲームフレームでこの行列が上書きされることに注意してください。

構文です。

camera_set_view_mat(camera_id, matrix)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.
matrix The unique matrix ID returned when you created the matrix.

リターンです。

例

view_camera[0] = camera_create();
var viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
var projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上記のコードでは、新しいカメラを作成し、ビューポート[0]に割り当てています。このカメラには、ビューマトリックスとプロジェクションマトリックスが設定されます。
