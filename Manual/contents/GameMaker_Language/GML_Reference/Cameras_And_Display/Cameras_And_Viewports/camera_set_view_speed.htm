camera_set_view_speed

この関数は、room 内のビューカメラの速度を更新するために使用できます。 一意のカメラ ID 値を指定します（別の camera_create()関数で指定し、x、y（水平方向 を設定したときに移動すべき速度である。速度は1ステップあたりのピクセル数で計算され、"-1 "に設定するとカメラは即座に動きますが、カメラがどのインスタンスにも従うように設定されていない場合は ここで設定した値は、目に見える効果はありません。

構文です。

camera_set_view_speed(camera_id, xspeed, yspeed)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera.
xspeed The speed (number of pixels per game frame) that the view should move on the horizontal (x) axis
yspeed The speed (number of pixels per game frame) that the view should move on the vertical (y) axis

リターンです。

例

camera_set_view_speed(view_camera[0], 5, 5);

上記のコードでは、ビューポート[0]に割り当てられたカメラのビューカメラスピードを設定します。
