camera_set_view_target

この関数は、room 内のビューカメラのフォローターゲットを設定するために使用することができます。 camera_create()関数) を使用し、次にカメラビューを設定したいターゲットインスタンスまたはobject ID を指定します。object ID を選択し、そのobject のインスタンスがroom の中に複数ある場合、GameMaker はどのインスタンスをフォローしたいのか知る術がないので、そのどれかになる可能性があることに注意。

構文です。

camera_set_view_target(camera_id, instance_id/object_id)

論証 タイプ 商品説明
camera_id カメラID The unique camera ID value returned when you created the camera
instance_id/object_id オブジェクトのアセットまたはインスタンスID Instance or object to have the camera target for following

リターンです。

N/A

例

camera_set_view_target(view_camera[0], id);

上記のコードでは、ビューポート[0]に割り当てられたカメラのビューカメラターゲットを、このコードを実行しているインスタンスのものに設定します。
