camera_get_view_angle

この関数は、指定されたカメラの角度を取得するために使用することができます。戻り値は 0 から 360 の間です。この関数は camera_create_view()またはroom エディタで追加されたものです。

構文です。

camera_get_view_angle(camera_id)

論証タイプ 商品説明
camera_id The unique camera ID value returned when you created the camera

リターンです。

例

var ang = camera_get_view_angle(view_camera[0]);
if (ang != 0)
{
    camera_set_view_angle(view_camera[0], 0);
}

上記のコードでは、ビューポート[0]に割り当てられたカメラの角度を取得し、それが与えられた値と一致するかどうかをチェックします。一致しない場合は、ビューカメラがその角度に設定されます。
