カメラセット_開始_スクリプト

この関数は、カメラが可視かつアクティブなビューポートに割り当てられている各ゲームフレームの最初に呼び出されるスクリプト関数を設定するために使用することができます。一意のカメラ ID 値（さまざまな camera_create() 関数によって返される）と、呼び出される関数の ID を指定します。scripts がカメラにアタッチされ、実際にカメラビューがレンダリングされる順番は以下の通りです。

  • すべての可視およびアクティブなビューポートのカメラには、その更新関数として
  • 次に、各ビューについて。
    • そのビューのカメラのbegin関数が 呼び出されます。
    • そのビューの描画イベントが実行される
    • カメラの終了関数が 呼び出される
  • 次のアクティブな可視ビューに移動して、繰り返します...

構文です。

camera_set_begin_script(camera_id, script)

論証 タイプ 商品説明
camera_id カメラID The unique camera ID value returned when you created the camera.
script スクリプト機能 The script function to run each game frame

リターンです。

N/A

例

camera_set_begin_script(view_camera[0], beginCamera);

上記のコードでは、ビューポート[0]に割り当てられたカメラのbeginscript 関数を設定しています。
