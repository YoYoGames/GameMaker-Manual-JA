camera_create_view

この関数は、新しいカメラを作成し、そのビューをroom に設定した後、今後の関数呼び出しでカメラを識別するための一意の ID 値を返します。カメラは、この関数で設定した位置、サイズ、その他の詳細の値を使用して作成され、これらはRoom エディタ（ここを参照）で設定した値と関連します。この関数は可変数の引数を取ることができますが、x/y 位置と width/height 引数のみが必須であることに注意してください。提供されない引数は、デフォルトの引数を使用します。

カメラは他のダイナミックリソースと同様に、作成時にメモリを占有することに注意してください。つまり、不必要にカメラを作成しないように注意し、不要なカメラは関数 camera_destroy().これは、カメラが global 変数に割り当てられた場合、room からroom まで持続することも意味します（インスタンスまたはローカル変数に割り当てられた場合も持続しますが、room を変更するとアクセスできなくなるので、メモリリークが発生します）。したがって、たとえば、新しいカメラを作成してグローバル変数に割り当て、各room の開始時に camera_set_* 関数を使用して、ビューポートに割り当てる前にプロパティを設定することができます。

構文です。

camera_create_view(room_x, room_y, width, height, [angle, object, x_speed, y_speed, x_border, y_border])

論証 タイプ 商品説明
room_x リアル The x position (within the room) of the left side of the area the camera view captures.
room_y リアル The y position (within the room) of the top of the area the camera view captures.
width リアル The width of the area the view captures.
height リアル The height of the area the view captures.
angle リアル OPTIONAL カメラ映像の角度を設定します（0°〜360°、デフォルトは0）。
object オブジェクトのアセットまたはインスタンスID OPTIONAL カメラビューを追従するためのオブジェクトID（またはインスタンスID）（デフォルトは-1、 はなし）。object
x_speed リアル OPTIONAL ビューが移動できる最大水平速度（"インスタント "フォローの場合は-1、デフォルトは-1）。
y_speed リアル OPTIONAL ビューが移動できる最大垂直速度（"インスタント "フォローの場合は-1、デフォルトは-1）。
x_border リアル OPTIONAL ビューエリアの端と、それが追従するように設定されているインスタンスとの間の最小の水平スペース（ビューが移動を開始する前、およびインスタンスを追従するように設定されている場合、デフォルトは0です）。
y_border リアル OPTIONAL ビューエリアのエッジと、それがフォローするように設定されているインスタンスとの間の最小垂直スペース（ビューが移動を開始する前と、それがインスタンスをフォローするように設定されている場合、デフォルトは0です）。

リターンです。

Camera ID

例

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

上記のコードは、viewport[0]に割り当てられた現在のカメラを削除し、新しいカメラを作成し、そのビュープロパティを定義して、それをviewport[0]に割り当てています。
