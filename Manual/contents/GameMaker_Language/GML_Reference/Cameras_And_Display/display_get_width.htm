display_get_width

この関数は、ディスプレイの幅をピクセル単位で返します。HTML5ターゲットでは、この値は物理的なディスプレイサイズではなく、ブラウザウィンドウの幅であることに注意してください。

構文です。

display_get_width();

リターンです。

例

mywidth = display_get_width();

この場合、変数「mywidth」にディスプレイの幅が設定される。
