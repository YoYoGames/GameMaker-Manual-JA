display_get_gui_height

この関数を使用すると、GUI の描画イベントで使用される GUI の高さ（ピクセル単位）を取得することができます。

構文です。

display_get_gui_height();

リターンです。

Real

例

ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0);

上記のコードで、ディスプレイの左下に広告を表示するように設定します。
