window_set_min_height

この関数は、ゲームのウィンドウの最小の高さを設定するために使用できます。ターゲットプラットフォームのゲームオプションでウィンドウサイズ変更オプションを有効にすると、プレイヤーはゲームウィンドウを好きなサイズに変更することができますが、この関数を使って を指定すると、最小高さを指定したサイズに制限することができます。デフォルトの動作（つまり最小値なし）に戻したい場合は、値を-1してください。

注：この機能は、Windows、macOS、Linuxのプラットフォームでのみ利用可能です。

構文です。

window_set_min_height(height);

論証タイプ 商品説明
height The minimum height in pixels for the game window.

リターンです。

例

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

上記のコードで、ゲームウィンドウの最小と最大の幅と高さが設定されます。
