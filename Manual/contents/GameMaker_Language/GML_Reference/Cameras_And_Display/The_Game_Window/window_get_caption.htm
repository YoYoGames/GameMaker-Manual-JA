window_get_caption

この関数は、ウィンドウのキャプション（ウィンドウの上部、アイコンの横に表示されるテキストです）を返します。デフォルトでは、現在表示されているroom のキャプションが表示されます。

構文です。

window_get_caption();

リターンです。

例

if window_get_caption() != ""
{
    window_set_caption("");
}

上記のコードは、ウィンドウのキャプションにテキストがあるかどうかをチェックし、テキストがある場合は、キャプションが表示されないように、空のstring を設定します。
