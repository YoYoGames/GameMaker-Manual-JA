ウィンドウズ_セット_キャプション

この機能により、現在表示されているroom のウィンドウキャプションを変更または設定することができます。このキャプションは、ゲームがフルスクリーンモードでない場合、ウィンドウの上部、ゲームアイコンの横に表示されます。

構文です。

window_set_caption(caption);

論証タイプ 商品説明
caption The new caption.

リターンです。

例

if window_get_caption() != ""
{
    window_set_caption("");
}

上記のコードは、ウィンドウのキャプションにテキストがあるかどうかをチェックし、テキストがある場合は、キャプションが表示されないように、空のstring を設定します。
