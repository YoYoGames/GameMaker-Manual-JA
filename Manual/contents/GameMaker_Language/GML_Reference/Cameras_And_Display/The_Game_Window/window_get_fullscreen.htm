window_get_fullscreen

この関数は、ゲームウィンドウがフルスクリーンモード(true)か、そうでないか(false)を返します。

構文です。

window_get_fullscreen();

リターンです。

例

if window_get_fullscreen()
{
    draw_text(32, 32, "Fullscreen is ON");
}
else
{
    draw_text(32, 32, "Fullscreen is OFF");
}

上記のコードは、ウィンドウがフルスクリーンモードであるかどうかを確認し、返された値に応じてテキストメッセージを描画します。
