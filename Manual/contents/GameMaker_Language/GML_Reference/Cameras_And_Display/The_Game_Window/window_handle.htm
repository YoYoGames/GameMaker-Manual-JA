ウィンドウハンドル

この関数で、内部のWindows id 値（HWND、ポインタ）を取得することができます。この関数は、DLLコードでextension APIを呼び出すために window handle が必要なWindows ライターにとってのみ有用です（返されたポインタをstring にキャストし、C++ではそれをHWNDにキャストするだけでよいのです）。以下の表は、サポートされるプラットフォームと、それらが返すものを示しています。

プラットフォーム リターンズ
ウィンドウズ ウィンドウズHWND
マクロス NSWindowクラス
Ubuntu (Linux) XWindowハンドル
HTML5 キャンバスID

構文です。

window_handle();

リターンです。

Pointer

例

win_id = windows_handle();
