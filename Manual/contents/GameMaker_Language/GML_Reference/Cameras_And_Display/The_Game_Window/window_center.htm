ウィンドウセンター

この機能により、ターゲットモジュールがWindows,Ubuntu (Linux) ,macOS の場合は、ゲームウィンドウをディスプレイの中央に、ターゲットモジュールが HTML5 の場合は、ブラウザの中央に表示させることができます。この機能は、他のデバイスには影響しません。

注意現在のステップでゲーム・ウィンドウのサイズを変更した場合（たとえば、フル・スクリーンからウィンドウズに切り替えたり、 window_set_size() ）、新しいウィンドウを中央に配置したい場合は、この関数を少なくとも1ステップ後に呼び出す必要があります（たとえば、 ）。 alarmそうでない場合は、正しく動作しません。

構文です。

window_center();

リターンです。

例

if !window_get_fullscreen()
{
    window_center();
}

上記のコードは、フルスクリーンモードでない場合、ゲームウィンドウをディスプレイ内の中央に配置します。
