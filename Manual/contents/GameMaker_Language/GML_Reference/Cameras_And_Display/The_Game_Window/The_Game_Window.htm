ゲームウィンドウ

作成したゲームはウィンドウ内で発生し（フルスクリーンの場合も）、このウィンドウには位置、サイズ、フルスクリーンかどうかなど、多くのプロパティがあります。これらの詳細は、通常、 ターゲットプラットフォームの ゲームオプションからの設定と組み合わせて有効になっているroom サイズとビューポートに基づいて自動的に設定されますが、このページに記載されている機能を使って、ゲーム中に変更することができます。

NOTEこれらの機能は、Windows、Ubuntu(Linux)、macOS 、HTML5ターゲットモジュール専用であり、その他のデバイスでは動作しない場合があります。

次の図は、いくつかの一般的なウィンドウの機能がどのように関連し、相互に作用しているかを示しています。

なお、上の画像はブラウザに描画されたゲーム・ウィンドウですが、他のターゲットについては、ブラウザをディスプレイに置き換えることができますので、 window_get_y() のような関数は、ディスプレイに対するゲーム・ウィンドウの上部の位置を返すことになります。
