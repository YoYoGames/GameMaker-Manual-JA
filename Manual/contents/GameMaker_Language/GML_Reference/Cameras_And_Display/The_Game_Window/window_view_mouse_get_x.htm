window_view_mouse_get_x

この関数は、選択されたビューに対するマウスの x 位置を返します。

注：通常のマウス機能については、「マウス入力」の項を参照してください。

構文です。

window_view_mouse_get_x( id );

論証 タイプ 商品説明
id リアル The id of the view to compare the mouse position to.

リターンです。

Real

例

if mouse_check_button_pressed(mb_left)
{
    var xx, yy;
    xx = window_view_mouse_get_x(0);
    yy = window_view_mouse_get_y(0);

    if xx > 0 && xx < 32 && yy > 0 && yy < 32
    {
        b_press[0] = true;
    }
}

上記のコードは、マウスボタンが押されているかどうかをチェックし、押されていれば、view[0]からの相対的なマウス位置を取得し、それを比較して、変数を true に設定すべきかどうかを判断しています。
