window_set_size

この機能により、ゲームウィンドウの大きさを変更することができます。デスクトップ・プラットフォームでは、通常、タイトルバーとボーダーを除いたウィンドウのクライアント領域のことを指します。

この機能を使ってゲームウィンドウのサイズを変更すると、ウィンドウの内容は背景のキャンバスに対して1：1の比率でフィットするように拡大縮小され、画像が鮮明に保たれます（ただし、ウィンドウがroom やビューサイズと異なる場合は、拡大縮小されます）。例えば、200 x 150 ピクセルのウィンドウがあり、プレーヤーが境界線をドラッグしてウィンドウを伸ばした場合、最終的なウィンドウサイズは 400 x 300 ピクセルになりますが、描画キャンバスは 200 x 150 のままで、ピクセル比率が 1:2 となるため、画像がぼやけた感じになります。しかし、この関数を使ってウィンドウサイズを変更すると、画像は新しいウィンドウサイズに合わせて拡大縮小されますが、1:1の比率が維持されます。

構文です。

window_set_size( w, h );

論証タイプ 商品説明
w The new width of the window.
h The new height of the window.

リターンです。

N/A

例

if mouse_check_button_pressed(mb_left)
{
    global.scale += 1;
    window_set_size(200 * global.scale, 150 * global.scale);
}

上記のコードは、マウスボタンが押されたときに、変数「global.scale」の値に基づいてウィンドウの大きさを変更するものです。
