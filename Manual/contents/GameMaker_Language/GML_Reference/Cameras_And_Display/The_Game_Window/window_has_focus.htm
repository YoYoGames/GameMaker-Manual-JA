window_has_focus

この関数を使うと、ウィンドウ（またはタブ）の状態をポーリングし、フォーカスが外れた場合は false を返し、そうでない場合は true を返すことができます。ほとんどの場合、単に os_is_paused()関数でテストできますが、非常に特殊なケース（例えば、Chromeアプリのゲームなど）では、この関数がトリガーされないことがあり、その場合は、代わりにこの関数を使用する必要があります。

注：この機能は、HTML5、Windows 、MacOSでのみ有効です。

構文です。

window_has_focus();

リターンです。

例

if !window_has_focus()
{
    PauseGame();
}

上記のコードでは、ゲームウィンドウにフォーカスがあるかどうかをチェックし、関数が false を返した場合、関数が呼び出されます。
