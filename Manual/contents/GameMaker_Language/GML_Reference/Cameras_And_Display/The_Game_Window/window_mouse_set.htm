ウィンドウマウスセット

この機能を使うと、ゲームウィンドウ内でのマウスの位置を変更または設定することができ、FPSゲームなどで便利です。この機能は、ゲームにフォーカスがあるときのみ動作し、alt + tabを使用するとマウスのロックが解除されます。

注意事項通常のマウス機能については、「マウス入力」の項を参照してください。

構文です。

window_mouse_set(x, y);

論証タイプ 商品説明
x The x coordinate in the window.
y The y coordinate in the window.

リターンです。

例

window_mouse_set(window_get_width() / 2, window_get_height() / 2);

上記のコードでは、マウスがゲームウィンドウの中央に配置されます。
