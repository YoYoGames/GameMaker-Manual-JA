uwp_license_trial_user

この関数は、現在トライアルライセンスが使用されているユーザーIDポインタを返します。また、ユーザーが現在アクティブでない場合、またはゲームが現在トライアルモードでない場合は、 pointer_null を返します。

構文です。

uwp_license_trial_user();

リターンです。

Xbox User ID

例

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user();
        uwp_show_help(_uid);
    }
}

上記のコードでは、ボタンが押されたかどうかを確認し、アプリがトライアルライセンスで実行されているかどうかをチェックします。もしそうなら、ライセンスのユーザーIDを取得し、そのユーザー用のヘルプファイルを開きます。
