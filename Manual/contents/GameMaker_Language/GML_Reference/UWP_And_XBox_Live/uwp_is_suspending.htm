uwp_is_suspending

この関数は、次のような方法で、1フレームの場合は true （それ以外の場合は false ）を返します。 os_is_paused().その後、ゲームはシステムによって終了させられる可能性がある前に、ゲーム状態を保存するための1秒の時間が与えられます。データの保存が完了したら、ゲームから uwp_suspend()を使用して、きれいにサスペンドします。ゲームは、システムによって再開された場合（完全に終了していない場合）、プレーヤーがゲームプレイの状況に直接戻されないように、おそらく一時停止状態にも入るべきです。

構文です。

uwp_is_suspending();

リターンです。

例

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

上記のコードでは、アプリがサスペンドに入るかどうかを確認し、サスペンドに入る場合はscript を呼び出してゲームデータを保存してからゲームをサスペンドしています。
