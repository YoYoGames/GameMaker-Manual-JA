ユーザーによって閉鎖された状態

この関数で、ゲームがプレイ中にユーザーによってシャットダウンされたかどうか（つまり、システムによってではなく、正しく終了させられたかどうか）を調べることができます。もしそうであれば、この関数は true を返し、そうでなければ、 false を返します。

構文です。

uwp_was_closed_by_user();

リターンです。

例

if uwp_was_closed_by_user()
{
    global.GameLoaded = false;
}

上記のコードでは、アプリが以前にユーザーによって終了されたかどうかを確認し、終了された場合は、グローバル変数を設定します。
