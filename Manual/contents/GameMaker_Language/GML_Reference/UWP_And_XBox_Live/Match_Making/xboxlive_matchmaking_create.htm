xboxlive_matchmaking_create（マッチメイキング・クリエイト

この関数は、XDPダッシュボードであらかじめ作成しておいたSession Template、matchmaking hopper、Secure Device Associationテンプレートを使って、マルチプレイヤー・セッションを作成するものです。を呼び出すことで、特定のゲーム・パッドについて取得できるユーザーIDを取得します。 xboxlive_user_for_pad()関数と可視性は、以下の定数のいずれかになります。

Xbox Match Visibility定数
定数 商品説明
xboxlive_match_visibility_open は、そのセッションに他の人が参加できることを指定します。
xboxlive_match_visibility_private は、セッションのメンバーでないユーザーには見えないようにすることを指定します。
xboxlive_match_visibility_usetemplate は、セッションテンプレートで使用されている値を使用することを指定します（XDPでテンプレートの値が設定されている場合、上書きすることができないので、通常はこれを使用すべきです）。

この関数は一意のリクエストID値を返し、それを使ってこの関数の正しいソーシャル非同期イベントを特定することができる。このイベントはセッションが作成されたときにトリガーされ、変数 async_load に以下のキーと値のペアを持つ DS Map が格納されます。

  • 「requestid" - 呼び出し関数によって返されたリクエストIDを含む。
  • "status" -string "session_created "を含み、どのような種類のイベントが発生したかを知らせます。
  • 「sessionid" - セッションIDが格納され、失敗した場合は-1が格納されます。
  • 「error" - 成功した場合は0，失敗した場合は-1が格納されます．
  • "correlationid" - この関数との相関をとるためのセッションハンドルです。 xboxlive_matchmaking_join_session()

構文です。

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

論証 タイプ 商品説明
user_id XboxユーザーID The user ID pointer to use
visibility Xbox Match Visibility定数 One of the constants listed above
template 文字列 The name of the session template
hopper 文字列 The name of the matchmaking hopper
sdatemplate 文字列 The name of the secure device association template
matchattributes リアル OPTIONAL マッチの属性

リターンです。

Async Request ID

例

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

上記は、与えられたゲームパッド上のユーザーのユーザーIDを取得し、そのユーザーのためのマッチメイキングセッションを初期化します。
