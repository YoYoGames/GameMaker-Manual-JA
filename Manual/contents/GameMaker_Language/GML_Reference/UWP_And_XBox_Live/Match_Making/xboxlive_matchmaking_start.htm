xboxlive_matchmaking_start

この関数は、ユーザーのマルチプレーヤーセッションを開始する前に呼び出す必要があります。この関数は、ユーザーIDポインターを受け取りますが、これは、特定のゲームパッドに対して xboxlive_user_for_pad() 関数を呼び出すと、マルチプレイヤー API が初期化されます。

構文です。

xboxlive_matchmaking_start(user_id);

論証タイプ 商品説明
user_id The user ID pointer to use

リターンです。

例

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

上記のコードは、指定されたパッドインデックスのユーザーIDを取得し、マルチプレイヤー機能を初期化するものです。
