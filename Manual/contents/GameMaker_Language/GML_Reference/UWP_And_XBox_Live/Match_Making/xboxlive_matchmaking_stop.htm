xboxlive_matchmaking_stop（マッチメイキングストップ

この関数は、指定されたユーザーのマッチメイキングセッションを終了するために使用することができます。

構文です。

xboxlive_matchmaking_stop(user_id);

論証タイプ 商品説明
user_id The user ID pointer to use

リターンです。

例

if !xboxlive_user_is_signed_in(user_id[0])
{
    xboxlive_matchmaking_stop(user_id[0]);
}

上記のコードは、指定されたユーザーのマッチメイキングセッションを終了させます。
