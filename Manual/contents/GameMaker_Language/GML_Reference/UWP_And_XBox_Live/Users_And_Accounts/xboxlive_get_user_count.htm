xboxlive_get_user_count

この関数を使用すると、現在 XBox システムにサインインしているユーザーの合計数を求めることができます。戻り値は 0 から N までの整数値で、N はサインインしているユーザーの数 -1 です。

構文です。

xboxlive_get_user_count();

リターンです。

例

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

上記は、loops 、サインインしているすべてのユーザーを調べ、そのユニークなIDポインターを配列に格納します。
