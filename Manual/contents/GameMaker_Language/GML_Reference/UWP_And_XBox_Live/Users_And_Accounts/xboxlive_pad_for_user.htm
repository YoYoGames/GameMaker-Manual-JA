xboxlive_pad_for_user

Xboxユーザーは、自分のユーザーIDに様々なパッドを割り当てることができ、それぞれが「スロット」を埋めることになります。スロットには0から番号が振られ、任意のパッドインデックスを保持することができますので、スロット0にはパッド3、スロット1にはパッド2を割り当てることができます。この関数は、任意のスロットに割り当てられたパッドを取得するために使用することができます。特に、ユーザーがゲームパッドを入力せずに一定時間放置すると、switch 自体がオフになるので、この関数を使用して任意のユーザー ID をチェックして、ゲームパッドがアクティブであることを確認し、そうでなければ（たとえば）ゲームを一時停止するなど、適切に対応することができます。パッドがアクティブであれば、そのインデックス値が返されますが、そうでなければ、この関数は-1を返します。

スロット0、1、2に3つのパッドが割り当てられていて、スロット0のパッドがアクティブでない場合、スロット1とtoはスロット0と1になります。

構文です。

xboxlive_pad_for_user(user_id, slot);

論証 タイプ 商品説明
user_id XboxユーザーID The User ID (a pointer) to check
slot リアル The index (an integer) of the gamepad slot to check

リターンです。

Xbox Pad ID

例

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
    for(var i = 0; i < gamepad_get_device_count(); i++;)
    {
        var u_id = xboxlive_user_for_pad(i);
        if (u_id == -1)
        {
            async_ID = xboxlive_pad_for_user(i, u_id);
        }
    }
}

上記のコードは、与えられたユーザーIDポインターのスロット0をチェックして、ゲームパッドが割り当てられているかどうかを確認し、割り当てられていない場合は、ゲームパッドを割り当てています。
