xboxlive_sponsor_for_userを使用する。

この関数は、指定されたユーザーのスポンサーの ID を取得します。

構文です。

xboxlive_sponsor_for_user(user_id);

論証 タイプ 商品説明
user_id XboxユーザーID The ID pointer of the user to check.

リターンです。

Xbox Sponsor ID

例

var _uid = xboxlive_user_for_pad(0);
sponsor = xboxlive_sponsor_for_user(_uid);

上記のコードでは、ゲームパッド[0]スロットに割り当てられたユーザーのユーザーIDポインタを取得し、そのIDを使用してユーザーのスポンサーIDを取得しています。
