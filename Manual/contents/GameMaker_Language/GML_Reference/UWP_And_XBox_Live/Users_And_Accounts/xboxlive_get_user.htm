xboxlive_get_user

この関数を使用すると、インデックスされたユーザのユーザIDポインタを取得することができます。そのユーザが存在しない場合、この関数は代わりに定数 pointer_null を返します。現在ログインしているユーザの数は、この関数で調べることができます。 xboxlive_get_user_count().

重要：この機能は、 xboxlive_get_user_count() で使用する必要があります。ゲームパッドでは使用しないでください。代わりに xboxlive_user_for_pad() を使用してください。

構文です。

xboxlive_get_user(index);

論証 タイプ 商品説明
index リアル The index (an integer) to get the User ID from.

リターンです。

Xbox User ID or pointer_null

例

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
}

上記は、loops 、サインインしているすべてのユーザーを調べ、そのユニークなIDポインターを配列に格納します。
