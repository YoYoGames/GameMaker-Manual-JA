xboxlive_gamedisplayname_for_user

この関数は、与えられたユーザ ID ポインタから、表示名 (string ) を取得します。ユーザーがローカルにいる場合は、返される値は単に表示名のstring となりますが、リモートユーザーの場合はstring がゲーマータグとなり、例えばボイスチャットでのミュート/アンミュートに使用することができることに注意してください。

構文です。

xboxlive_gamedisplayname_for_user(user_id);

論証タイプ 商品説明
user_id The ID pointer of the user to check.

リターンです。

例

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    user_name[i] = xboxlive_gamedisplayname_for_user(user_id[i]);
}

上記のコードでは、利用可能な各ユーザーアカウントのユーザーIDポインターを取得し、そのポインターを表示名とともに2つの配列に格納しています。
