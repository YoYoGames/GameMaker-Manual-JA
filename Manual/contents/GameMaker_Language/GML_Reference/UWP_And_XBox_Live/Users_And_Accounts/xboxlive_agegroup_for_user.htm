xboxlive_agegroup_for_user

この関数は、指定されたユーザーIDポインタに割り当てられた年齢範囲を示す4つの定数（以下に示す）のうちの1つを返す。

構文です。

xboxlive_agegroup_for_user(user_id);

論証 タイプ 商品説明
user_id XboxユーザーID The user ID (a pointer) to check

リターンです。

Xbox Age Group Constant

Xbox 年齢層別定数
定数 商品説明
xboxlive_agegroup_unknown ユーザーの年齢層は不明
xboxlive_agegroup_child ユーザーの年齢層は8歳から12歳（含）です。
xboxlive_agegroup_teen ユーザーの年齢層は13歳以上17歳未満（含、韓国は13歳以上19歳未満）。
xboxlive_agegroup_adult ユーザーの年齢層は18歳以上（韓国は20歳以上

例

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
    global.ContentControl = true;
}
else global.ContentControl = false;

上記のコードでは、アクティベートするユーザーの年齢層をチェックし、返された定数に応じてグローバル変数に true または false を設定しています。
