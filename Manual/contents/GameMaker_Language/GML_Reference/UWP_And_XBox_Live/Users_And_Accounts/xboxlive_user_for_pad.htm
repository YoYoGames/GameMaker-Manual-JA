xboxlive_user_for_pad

この関数は、与えられたゲームパッドのインデックス値に関連付けられたユーザIDポインタ、またはユーザが関連付けられていない場合は pointer_null を返します。この関数は、ゲームパッドインデックスに対してのみ使用されることに注意してください。 xboxlive_get_user_count().

構文です。

xboxlive_user_for_pad(pad_index);

論証タイプ 商品説明
pad_index The index (an integer) of the gamepad slot to check

リターンです。

Xbox User ID or pointer_null

例

for(var i = 0; i < 11; ++i;)
{
    var u_id = xboxlive_user_for_pad(i);
    if (u_id != pointer_null)
    {
        pad_name[i] = xboxlive_gamedisplayname_for_user(u_id);
    }
    else
    {
        pad_name[i] = "Press Play";
    }
}

上記のコードはloops 利用可能なすべてのパッドスロットに目を通し、それに関連するユーザーIDポインタがあるかどうかを確認します。もしあれば、ユーザーの表示名が配列に設定され、そうでなければ、配列には何らかのデフォルトテキストが設定されます。
