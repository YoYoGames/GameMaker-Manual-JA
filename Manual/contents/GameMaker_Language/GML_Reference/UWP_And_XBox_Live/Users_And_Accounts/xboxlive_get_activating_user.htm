xboxlive_get_activating_user

この関数を使用すると、コンソールからゲームを起動したユーザーのユーザーIDポインターを取得することができます。

注：このポインターをどのように使うかは、ゲーム内で何をしたいかによりますが、起動したユーザーがゲーム実行中にログアウトできるため、現在プレイしているユーザーIDに依存しないようにしてください。

構文です。

xboxlive_get_activating_user();

リターンです。

Xbox User ID

例

global.activUser = xboxlive_get_activating_user();

上記は、起動ユーザーIDポインターをグローバル変数に格納する
