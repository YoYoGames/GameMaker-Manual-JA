xboxlive_set_rich_presence (リッチプレゼンス)

この関数は、指定されたユーザーのリッチプレゼンスstring を設定します。リッチプレゼンスstring は、ユーザーがプレイしているゲーム名の後にハイフンで区切って、ユーザーのゲーム内アクティビティを表示します。このstring は、"Friends & Clubs" リストのプレイヤーのゲーマータグの下に表示され、プレイヤーの Xbox Live ユーザープロファイルにも表示されます。

この関数を使用する場合、ユーザーのユーザーIDポインターを提供する必要があります。そして、そのユーザーが現在ゲーム内でアクティブかどうか（ true/false を使用）をflag で指定します。 false を指定すると、リッチプレゼンスstring に "/afk" などの適切な文字が付加されます。次の引数は表示するリッチプレゼンスstring ID で、最後に (オプションで) service_config_id string を指定できます。 この引数はオプションで、指定しない場合は Xbox ゲームオプションで指定した SCID を使用します。

構文です。

xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]);

論証 タイプ 商品説明
user_id XboxユーザーID The ID pointer of the user to check.
is_user_active ブーリアン Flag the user as active or not.
rich_presence_string 文字列 The rich presence string ID to use (as defined in the Partner Center - max 50 characters)
service_config_id 文字列 OPTIONAL これは、ゲーム用のconfig_id です。string

リターンです。

N/A

例

var pad_uid = xboxlive_user_for_pad(0);
xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge");

上記のコードでは、ゲームパッド[0]スロットに割り当てられたユーザーのユーザーIDポインターを取得し、そのユーザーのリッチプレゼンスstring を設定しています。
