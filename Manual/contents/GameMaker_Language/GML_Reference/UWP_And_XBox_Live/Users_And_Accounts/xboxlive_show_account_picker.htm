xboxlive_show_account_picker（アカウントピッカー）。

この関数は、XBox アカウントピッカー画面を開き、ユーザーがサインインしていない場合はサインインできるようにします (サインインしている場合は、この関数は何も行いません)。この関数は、UWPビルドをPCで使用する場合にのみ適用され、Xboxでは適用されないことに注意してください。

構文です。

xboxlive_show_account_picker();

リターンです。

例

if !xboxlive_user_is_signed_in()
{
    if (!xboxlive_user_is_signing_in())
    {
        xboxlive_show_account_picker();
    }
}
else
{
    global.GamerTag = xboxlive_gamertag_for_user();
}

上記のコードでは、ユーザーが XBox Live にサインインしているかどうかを確認し、サインインしている場合はそのユーザーのゲーマータグを取得し、今後の使用のためにグローバル変数に格納します。サインインしていない場合は、サインイン中かどうかをチェックし、その場合は何も起こりません。サインインしていない場合は、アカウント ピッカーを開き、サインインを促します。
