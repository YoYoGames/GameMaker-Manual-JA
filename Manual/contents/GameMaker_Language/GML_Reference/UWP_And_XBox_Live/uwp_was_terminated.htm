uwp_was_terminated

この関数で、ゲームがシステムによって以前にシャットダウンされたかどうかを調べることができます（つまり、ゲームの初回実行時ではない）。もしそうであれば、この関数は true を返し、そうでなければ、 false を返します。例えば、ゲームがシステムによって終了されたかどうかを確認し、（ true を返した場合）プレイヤーがゲーム中にいた前のステージをロードするような使い方があります。XBox One をターゲットとする場合、これが検出された場合のゲームの要求動作は、認証ドキュメントの異なるバージョン間で異なるので、ゲームがどうすべきかを確認するために、最新バージョンをチェックする価値があります。

構文です。

uwp_was_terminated();

リターンです。

例

if xboxlivewas_terminated()
{
    buffer_load("Player_Save.sav");
    global.GameLoaded = true;
}

上記のコードでは、アプリが以前にシステムによって終了されたかどうかを確認し、終了された場合はbuffer をロードし、グローバル変数を設定します。
