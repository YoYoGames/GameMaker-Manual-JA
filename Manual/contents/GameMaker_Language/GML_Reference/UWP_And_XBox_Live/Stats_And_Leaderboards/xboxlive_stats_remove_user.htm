xboxlive_stats_remove_user（ユーザーを削除する

この関数は、統計マネージャからユーザーを削除（登録解除）し、統計データをライブserver にフラッシュするために使用できます。XBoxのドキュメントによると、XBox OSは定期的に統計データをフラッシュするので、ゲームは統計マネージャからユーザーを削除する必要はないとのことです。例えば、関数 xboxlive_get_user()この関数は、エラーが発生した場合やユーザーIDが無効な場合は-1、関数が正常に呼び出された場合はその他の値を返します。

注：ユーザーに設定されている統計が無効な場合（統計名にアルファベット以外の文字が含まれているなど）、ユーザーを削除するとエラーが返されることがあります。

この関数は、システム非同期イベントのトリガーとなるコールバックを生成します。このイベントは特別な DS マップ async_loadを解析して、次のキーを探します。

  • 「id" - 定数を保持します。 achievement_stat_event
  • "eventname" -string "LocalUserRemoved" を保持します。
  • "userid" - リクエストに関連付けられたユーザー ID。
  • "error" - 成功した場合は0、エラーが発生した場合は他の値を指定します。
  • "errormessage" - エラーメッセージがある場合は、string 。

なお、ユーザーを削除することなく、いつでも統計データをライブserver に流したい場合は、関数 xboxlive_stats_flush_user().

構文です。

xboxlive_stats_remove_user(user_id);

論証タイプ 商品説明
user_id The user ID (a pointer) to remove

リターンです。

例

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
}

上記のコードは、loop 保存されているユーザーアカウントIDを調べ、各ユーザーをstats managerから削除（登録解除）します。
