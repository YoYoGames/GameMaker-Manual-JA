xboxlive_achievement_load_leaderboard

この関数は、指定されたリーダーボードの情報について、server にリクエストを送信します。リーダーボードは、ゲームの開発者用ダッシュボードで以前に作成されている必要があり、与えられたIDは、関数の最初の引数として供給されなければならないものである。この関数は、コールバックのソーシャル非同期イベントをトリガーします。 async_loadマップに、関連するキーと値のペアを入力します。この DS Map のidキーは、正しいコールバックを識別するために使用されます（任意の非同期イベントに対して複数のトリガー関数が存在する場合があります）。また、定数 xboxlive_achievement_leaderboard_info と、各プレイヤーに対して他の多くのキー/値のペアが組まれます。マップの正確な内容は、以下のとおりです。

  • 「id" - この関数の場合、以下のようになります。 xboxlive_achievement_leaderboard_info
  • 「leaderboardid" - プロバイダー・ダッシュボードで定義された、リーダーボードの一意のIDです。
  • "numentries" - あなたが受け取ったリーダーボードへのエントリーの数です。
  • "PlayerN" - プレーヤーの名前。"N" は、リーダーボードのエントリーリスト内での位置に対応する整数値です。
  • "PlayeridN" - プレーヤーのユニークなユーザーID、"N".
  • 「RankN" - リーダーボード内でのプレイヤー "N "の順位。
  • "ScoreN" - プレイヤー "N "のスコア。

また、以下のフィルタ定数により、友人のスコアのみ、あるいは全プレイヤーのスコアを取得するようフィルタリングを要求することも可能です。

Xbox Liveのアチーブメントフィルターの定数
定数 商品説明
xboxlive_achievement_filter_friends_only  プレイヤーフレンドのスコアだけを取得する。
xboxlive_achievement_filter_all_players  全プレイヤーの全スコアを取得する。

構文です。

xboxlive_achievement_load_leaderboard(ident, minindex, maxindex, filter)

論証 タイプ 商品説明
ident 文字列 The unique ID of the leaderboard as shown on the developer dashboard.
minindex リアル The starting index value to get the leaderboard data from.
maxindex リアル The maximum index value to get the leaderboard data from.
filter Xbox Liveのアチーブメントフィルターの定数 Whether to filter the results to those that are on your friends list or to all players (see the Description above)

次のコードは、プレイヤーがゲームアカウントにログインした後に呼び出され、そのゲームのすべてのフレンドのリーダーボードの位置のリストを取得することができます。

xboxlive_achievement_load_leaderboard("CgkIs9_51u0PEAIQBw", 1, 100, xboxlive_achievement_filter_friends_only);

これは、与えられたリーダーボードの情報に対する要求を送り、以下のデータを含む特別な async_load DS マップで非同期コールバックを生成します。

var ident = ds_map_find_value(async_load, "id");
if (ident == xboxlive_achievement_leaderboard_info)
{
    var lbid = ds_map_find_value(async_load,"leaderboardid");
    if lbid == "CgkIs9_51u0PEAIQBw" || lbid == "leaderboard1id"
    {
        global.numentries = ds_map_find_value(async_load,"numentries");
        for(var i = 0; i < numentries; i++)
        {
            global.playername[i] = ds_map_find_value(async_load, "Player" + string(i));
            global.playerid[i] = ds_map_find_value(async_load, "Playerid" + string(i));
            global.playerrank[i] = ds_map_find_value(async_load, "Rank" + string(i));
            global.playerscore[i] = ds_map_find_value(async_load, "Score" + string(i));
        }
    }
}

上記のコードは、ソーシャル非同期イベントで返された DS Map をチェックし、その "id" がチェックされている定数と一致する場合、送信されるリーダーボードが元の関数呼び出しと一致するかどうかを確認した後、マップをループしてすべての異なる値をいくつかの配列に格納しています。
