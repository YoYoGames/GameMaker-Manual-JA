xboxlive_stats_get_stat

この関数は、指定されたユーザの stat マネージャから単一の stat 値を取得するために使用できます。によって返されるようなユーザIDを指定します。 xboxlive_get_user()そして、 xboxlive_stats_set_stat_* 関数の 1 つを使用して作成したときに定義された statstring です。戻り値は、string または実数 (チェックする stat に依存)、stat が存在しないかエラーが発生した場合はGML 定数 undefined のいずれかです。

構文です。

xboxlive_stats_get_stat(user_id, stat);

論証タイプ 商品説明
user_id The user ID pointer to get the stat names of
stat The statistic to set (a string)

リターンです。

Real, String or undefined

例

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

上記のコードでは、変数をチェックし、それが true であれば、「PercentDone」stat の値をチェックすることになります。この値が100より小さい場合、次に値が生成され、その値にstatが設定されます。
