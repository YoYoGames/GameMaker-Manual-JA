xboxlive_stats_get_leaderboard

この関数は、指定した統計値に対するグローバルリーダーボードの順位を取得するために使用できます。ユーザ ID (たとえば、この関数が返すもの)、統計情報 (「注目の統計情報」登録時に定義したもの)、そしていくつかの詳細を指定します。 xboxlive_get_user())、統計情報string (「注目の統計情報」として登録した際に定義したもの)、 そして取得したいリーダーボード情報の詳細を指定します。グローバルリーダーボードを取得できるのは、int または real 統計のみで、string の統計は取得できないことに注意してください。

重要!グローバルリーダーボードで使用される統計は、XDP/Windows Dev Centerで「Featured Stats」として登録されている必要があり、そうでない場合はエラーが返されます。ローカル(ソーシャル)リーダーボードが必要な場合は、次の関数を参照してください。 xboxlive_stats_get_social_leaderboard().

この関数は、ソーシャル非同期イベントのトリガーとなるコールバックを生成します。このイベントには、DS マップが組み込まれています。 async_load を持ち、以下のキーがパースされる必要があります。

  • 「id" - 定数を保持します。 achievement_stat_event
  • "event" -string "GetLeaderboardComplete" を保持します。
  • "userid" - リクエストに関連付けられたユーザー ID。
  • "error" -成功した場合は 0、エラーがあった場合は他の値。次の2つは、最もよく返されるエラーです。
    • 2145844844:要求されたリーダーボードが見つかりません (スタッツがフィーチャードスタッツとして登録されていません)
    • -2145844848:不正なリクエスト (スタットは有効なリーダーボード・タイプではありません。string であるべきです)
  • "errormessage" - エラーメッセージがある場合は、string 。
  • "display_name" - プロバイダーダッシュボードで定義された、リーダーボードの一意のIDです。
  • "numentries" - あなたが受け取ったリーダーボードへのエントリーの数です。

DSマップの残りの部分にも、次のような形式のリーダーボードデータが含まれます（「N」はリーダーボードデータ内の位置で、0から「numentries」までです）。

  • "PlayerN" - プレーヤーの名前。"N" は、リーダーボードのエントリーリスト内での位置に対応する整数値です。
  • "PlayeridN" - プレーヤーのユニークなユーザーID、"N".
  • 「RankN" - リーダーボード内でのプレイヤー "N "の順位。
  • "ScoreN" - プレイヤー "N "のスコア。

構文です。

xboxlive_stats_get_leaderboard(user_id, stat, num_entries, start_rank, start_at_user, ascending);

論証 タイプ 商品説明
user_id XboxユーザーID The user ID of the user to get the leaderboard for
stat 文字列 The stat (as a string) to create the global leaderboard from
num_entries リアル The number of entries from the global leaderboard to retrieve
start_rank リアル The rank in the leaderboard to start from (use 0 if the "start_at_user" argument is set to true)
start_at_user ブーリアン Set to true はユーザーIDのランクから開始、それ以外は (引数 "start_rank" が 0 以外の場合は に設定される) false false
ascending ブーリアン Set to true 昇順の場合、 降順の場合、 false

拡張された例。

以下は、この関数の使用方法の拡張例です。まず始めに、Room StartやCreate などのイベントでこの関数を呼び出します。

xboxlive_stats_get_leaderboard(user_id, "GlobalTime", 20, 1, false, true);

上記のコードは、ゲームのすべてのソーシャルリーダーボードポジションのリストを取得するために呼び出され、ソーシャル非同期イベントコールバックを生成し、次のように処理します。

if (async_load[? "id"] == achievement_stat_event)
{
    if (async_load[? "event"] == "GetLeaderboardComplete")
    {
        global.numentries = async_load[? "numentries"];
        for(var i = 0; i < numentries; i++;)
        {
            global.playername[i] = async_load[? "Player" + string(i)];
            global.playerid[i] = async_load[? "Playerid" + string(i)];
            global.playerrank[i] = async_load[? "Rank" + string(i)];
            global.playerscore[i] = async_load[? "Score" + string(i)];
        }
    }
}

上記のコードは、ソーシャル非同期イベントで返されたDSマップをチェックし、その「id」がチェック対象の定数と一致する場合、返されたリーダーボードデータによってイベントがトリガーされたかどうかをチェックしてからマップをループし、すべての異なる値をいくつかのグローバル配列に格納しています。
