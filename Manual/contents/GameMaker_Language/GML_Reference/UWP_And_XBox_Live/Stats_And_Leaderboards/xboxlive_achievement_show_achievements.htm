xboxlive_achievement_show_achievements

この関数は、選択されたプラットフォームの実績ページを開きます。この関数は非同期関数であることに注意してください。そのため、この機能を使用する際は、ゲームを一時停止するか、ゲームプレイに支障のない範囲でしか表示させないなど、注意が必要です。

構文です。

xboxlive_achievement_show_achievements()

リターンです。

例

if achievements_enabled
{
    global.Pause = true;
    instance_create_layer(0, 0, "Controllers", obj_Pause);
    xboxlive_achievement_show_achievements();
}

上記のコードでは、カスタム変数で実績が有効かどうかをチェックし、有効な場合はグローバル変数を設定し、コントローラの「pause」インスタンスを生成してから、画面に実績を表示する関数を呼び出しています。
