xboxlive_stats_get_stat_names

この関数は、指定したユーザについて、stat manager から定義されたすべての統計情報を取得するために使用できます。によって返されるユーザ ID を指定します。 xboxlive_get_user()そして、この関数はそのユーザーの統計情報を含むstrings の配列を返します。エラーが発生した場合、あるいはユーザが統計情報を持っていない場合も、配列は返されますが、要素数は 0 となります。

構文です。

xboxlive_stats_get_stat_names(user_id);

論証タイプ 商品説明
user_id The user ID pointer to get the stat names of

リターンです。

例

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i)
{
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

上記のコードは、与えられた変数に格納されているユーザーIDポインタに対して定義されたすべてのstatsを取得し、loops 、その配列を介して指定されたstatをstatマネージャから削除します。
