xboxlive_read_player_leaderboard

この関数は、指定されたユーザ名のリーダーボード情報を返します。リーダーボードは、ゲームのXDPダッシュボードで事前に作成されている必要があり、この関数を呼び出すと、ソーシャル非同期イベントのコールバックがトリガされ、その中に async_loadマップには、関連するキーと値のペアが入力されています。 achievement_leaderboard_info このDSマップの「id」キーは、正しいコールバックを識別するために使用されます（任意の非同期イベントには、複数のトリガー関数が存在する可能性があります）。マップの正確な内容は以下の通りです。

  • 「id" - この関数の場合、以下のようになります。 achievement_leaderboard_info
  • "リーダーボードID" - プロバイダ・ダッシュボードで定義された、リーダーボード用の一意のID。
  • "ヌーメンタルズ" - あなたが受け取ったリーダーボードへのエントリーの数です。
  • "プレイヤーN" - プレーヤーの名前。"N" は、リーダーボードエントリーリスト内での位置に対応する整数値です。
  • "プレイヤーIDN" - プレーヤーのユニークなユーザー ID、"N" です。
  • "ランクN" - リーダーボード内のプレイヤー "N "の順位。
  • "スコアN" - プレイヤー "N "のスコア。


この関数では、フィルタのプロパティを設定するために、以下の定数のいずれかを指定する必要があります。

定数 商品説明
xboxlive_achievement_filter_all_players すべてのスコアを取得する。
xboxlive_achievement_filter_friends_only 友達のスコアだけを昇順で取得します。
xboxlive_achievement_filter_favorites_only お気に入りのスコアだけを昇順で取得する
xboxlive_achievement_filter_friends_alt 友達のスコアだけを降順で取得する。
xboxlive_achievement_filter_favorites_alt お気に入りのスコアだけを降順で取得します。

構文です。

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

論証 タイプ 商品説明
leaderboard_name 文字列 The name of the leaderboard to read, as set up on XDP.
user_name XboxユーザーID The ID of the user to read from.
num_items リアル The number of items to retrieve.
friend_filter Xbox Liveのアチーブメントフィルターの定数 One of the filter constants (see the description, above).

リターンです。

Real

例

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

上記のコードはユーザーIDを取得し、それを使って指定されたリーダーボードの1位から10位までのすべてのプレイヤーデータを要求しています。
