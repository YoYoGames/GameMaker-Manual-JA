xboxlive_stats_setup

この関数は、他のXbox Live統計関数を使用する前に呼び出す必要があり、単にシステム上の必要なライブラリを初期化するだけです。引数 "user_id" は、この関数が返す生のユーザー ID です。 xboxlive_get_user()service_config" と "title_id" は、Xbox Live Dev Center に登録されているゲームのユニークな ID を指定します。

この機能は、Event-Based（2013）統計でのみ使用されます。

構文です。

xboxlive_stats_setup(user_id, service_config_id, title_id);

論証 タイプ 商品説明
user_id XboxユーザーID The ID pointer of the user to check.
service_config_id 文字列 This is the config_id string for the game.
title_id 文字列 The unique ID for your game on the Xbox Dev Center.

リターンです。

N/A

例

var user = xboxlive_get_user(0);
xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

上記のコードは、指定されたユーザーIDの統計システムを初期化します。
