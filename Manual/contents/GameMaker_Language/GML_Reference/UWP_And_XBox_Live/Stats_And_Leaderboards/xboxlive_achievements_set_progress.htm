xboxlive_achievements_set_progress

この関数は、アチーブメントの進捗状況を更新するために使用することができます。ユーザ ID は、例えばこの関数で返されたものです。 xboxlive_get_user()という文字列、そして達成度 (これはXDP/Windows Dev Centerで作成されたときに割り当てられた達成IDである)。最後に、進捗状況を0から100までの値で設定します。アチーブメントシステムは、現在の進捗状況より低い更新を拒否することに注意してください。 の値を設定します。

構文です。

xboxlive_achievements_set_progress(user_id, achievement, progress);

論証タイプ 商品説明
user_id The user ID of the user to set the achievement for
achievement The achievement to set (a string)
progress The progress value to set (from 0 to 100)

リターンです。

拡張された例。

以下に、この関数の使用方法の拡張例を示します。まず始めに、Room StartやCreate などのイベントでこの関数を呼び出します。

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

上記のコードは、ゲームのグローバルリーダーボードポジションのリストを取得するために呼び出され、ソーシャル非同期イベントのコールバックを生成し、次のように処理します。

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

上記のコードでは、いくつかのグローバル変数に基づいてパーセント値を作成し、それを使って「Game_Completed」達成の進捗を設定しています。
