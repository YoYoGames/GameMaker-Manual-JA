xboxlive_stats_set_stat_real

この関数は、与えられたユーザーIDのstatの値を設定するために使用することができます。によって返されるユーザ ID を指定します。 xboxlive_get_user()次に、設定する statstring (statstring がまだ存在しない場合、新しい stat が作成され、指定した値に設定されます)、および stat を設定する値 (実数) を指定します。stat名は英数字のみで、記号やスペースは使用できないことに注意してください。

スタッツ値を設定する際、以前の値はすべて上書きされるため、設定前に現在のスタッツ値と新しい値を比較して、スタッツ値を更新すべきかどうか（例えば、ハイスコアが実際に最高値であるかをチェックする）を判断するのはあなた次第です。

この関数は、エラーが発生した場合やユーザーIDが無効な場合は-1、関数が正常に呼び出された場合はその他の値を返します。

構文です。

xboxlive_stats_set_stat_real(user_id, stat, value);

論証タイプ 商品説明
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (a real)

リターンです。

例

if (game_over == true)
{
    if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
    {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
    }
}

上記のコードでは、変数をチェックし、それが true であれば、「PercentDone」stat の値をチェックすることになります。この値が100より小さい場合、次に値が生成され、その値にstatが設定されます。
