xboxlive_stats_set_stat_string

この関数は、指定されたユーザー ID の stat をstring に設定するために使用することができます。によって返されるユーザ ID を指定します。 xboxlive_get_user()そして、設定する statstring (statstring がまだ存在しない場合、新しい stat が作成され、指定された値に設定されます)、さらに stat を設定するstring を指定します。stat名は英数字のみで、記号やスペースは使用できませんが、設定するstring の値にはスペースや特殊文字が使用できることに注意してください。

statstring を設定する場合、以前のstring はすべて上書きされます。したがって、statstring を更新すべきかどうかは、設定する前に現在のstatstring と新しいものを比較することで判断してください。

この関数は、エラーが発生した場合やユーザーIDが無効な場合は-1、関数が正常に呼び出された場合はその他の値を返します。

構文です。

xboxlive_stats_set_stat_string(user_id, stat, value);

論証 タイプ 商品説明
user_id XboxユーザーID The user ID pointer to set the stat for
stat 文字列 The statistic to set (a string)
value 文字列 The value to set the stat to (a string)

リターンです。

Real

例

xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName);

上記のコードは、stat "LevelName "を与えられたグローバル変数に格納されているstring に設定します。
