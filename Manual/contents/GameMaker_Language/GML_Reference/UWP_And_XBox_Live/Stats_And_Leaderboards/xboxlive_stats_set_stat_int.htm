xboxlive_stats_set_stat_int

この関数は、与えられたユーザーIDのstatの値を設定するために使用することができます。によって返されるユーザIDを指定します。 xboxlive_get_user()を指定し、次にstatの文字列 (statstring がまだ存在しない場合は、新しい stat が作成され、指定した値に設定されます) および stat を設定する値 (整数) を指定します。stat名は英数字のみで、記号やスペースは使用できないことに注意してください。

スタッツ値を設定する際、以前の値はすべて上書きされます。したがって、スタッツ値を更新すべきかどうか（例えば、ハイスコアが実際に最高値であるかどうかをチェックする）、現在のスタッツ値と比較することによって判断する必要があります。 を設定する前に、新しいものを設定します。

この関数は、エラーが発生した場合やユーザーIDが無効な場合は-1、関数が正常に呼び出された場合はその他の値を返します。

構文です。

xboxlive_stats_set_stat_int(user_id, stat, value);

論証タイプ 商品説明
user_id The user ID pointer to set the stat for
stat The statistic to set (a string)
value The value to set the stat to (an integer)

リターンです。

例

var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed");
xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val);

上記のコードは、ステータス「GamesPlayed」の値を取得し、同じステータスをその値に1を足した値に設定しています。
