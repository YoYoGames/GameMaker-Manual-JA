xboxlive_stats_delete_stat

この関数は、与えられたユーザIDのstatマネージャからstatを削除するために使用することができます。によって返されるユーザIDを指定します。 xboxlive_get_user()そして、削除する statstring です。これは stat の値をクリアして stat マネージャから削除し、関数 xboxlive_stats_get_stat_names()または xboxlive_stats_get_stat().

この関数は、エラーが発生した場合やユーザーIDが無効な場合は-1、関数が正常に呼び出された場合はその他の値を返します。

構文です。

xboxlive_stats_delete_stat(user_id, stat);

論証タイプ 商品説明
user_id The user ID pointer to delete the stat of
stat The statistic to delete (a string)

リターンです。

例

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

上記のコードは、loops 接続しているユーザーを通して、各ユーザーの stat manager から指定された stat を削除しています。
