xboxlive_fire_event

この関数は、stat イベントを発生させるために使用できます。引数 "event_name "は、ゲームのXDPコンソールで定義された、発生させるイベントの名前です。また、以下の追加パラメータは、statに何を設定したかに依存します。この関数は、イベントが送信された (そしてserver で受信/処理される) 場合に 0 を返し、エラーがあった場合 (すなわち、プロジェクトに含まれるイベント マニフェスト ファイルに別の番号が書かれているため、イベントが設定されていない) には -1 を返します。

構文です。

xboxlive_fire_event(event_name, [stat_parameters...]);

論証 タイプ 商品説明
event_name 文字列 The name of the event to fire.
stat_parameters バリアブル OPTIONAL イベントに追加するパラメータ。

リターンです。

Real

例

var _user = xboxlive_get_user(0);
var _uid = xboxlive_user_id_for_user(_user);
result = xboxlive_fire_event("PlayerSessionStart", _uid, global.guid_str, 0, 42, 42);

上記のコードは、ユーザーに関するいくつかの情報を取得し、この（および追加の）データとともに stat イベントを発生させます。
