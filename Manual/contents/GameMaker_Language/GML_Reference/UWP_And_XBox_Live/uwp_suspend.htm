uwp_suspend

この関数は、アプリが停止要求の検出後、データの保存を終了したことをシステムに通知します。この関数を呼び出すことは、Xbox Oneサブミッションの必須条件であり、常に関数 uwp_is_suspending()以下の例のように

構文です。

uwp_suspend();

リターンです。

例

if uwp_is_suspending()
{
    scr_Save_Game_Data();
    uwp_suspend();
}

上記のコードでは、アプリがサスペンドに入るかどうかを確認し、サスペンドに入る場合はscript を呼び出してゲームデータを保存してからゲームをサスペンドしています。
