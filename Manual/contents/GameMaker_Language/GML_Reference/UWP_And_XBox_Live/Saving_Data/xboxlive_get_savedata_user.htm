xboxlive_get_savedata_user

この関数は、現在ファイル保存に関連するユーザ ID ポインタ（現在使用されているユーザ ID がない場合は定数 pointer_null ）を返します。参照 xboxlive_set_savedata_user()を参照してください。

構文です。

xboxlive_get_savedata_user();

リターンです。

Xbox User ID or pointer_null

例

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

上記のコードでは、あるユーザーが現在保存先として割り当てられているかどうかを確認し、割り当てられていない場合は割り当てます。
