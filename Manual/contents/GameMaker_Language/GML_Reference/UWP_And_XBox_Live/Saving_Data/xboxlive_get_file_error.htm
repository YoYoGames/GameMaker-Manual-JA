xboxlive_get_file_error

この関数は定数（下図）を返すので、ファイル関数が失敗した理由をデバッグし、ゲームコードで適切な対処をするのに役立ちます。ファイル関数のいずれかが失敗したとき、この関数を呼び出してエラーの内容を正確に知ることができます。

注：非同期保存・読み込み機能を使用する場合 async_loadマップには追加のキー "error" があり、これにも定数のひとつが含まれます (下図参照)。

構文です。

xboxlive_get_file_error();

リターンです。

Xbox File Error Constant

Xboxファイルエラー定数
定数 商品説明
xboxlive_fileerror_outoflocalstorage セーブ中にこの現象が発生した場合、利用可能なセーブゲーム容量をすべて使い切ってしまい、OSが提示した容量開放の申し出をユーザーが拒否したことを意味します。セーブせずにプレイを続けるかどうか、ユーザーに確認する必要があります。それでもセーブを希望する場合は、セーブ操作を再試行する。
xboxlive_fileerror_quotaexceeded この場合、空き容量がなくなるまでこれ以上データを保存できないことをユーザーに通知し、その方法を提示する必要があります。
xboxlive_fileerror_noerror ファイルの保存や読み込み機能は問題なく実行できました。

例

buffer_save(buff, "SaveGame.bin");
if (xboxlive_get_file_error() != xboxlive_fileerror_noerror)
{
    show_debug_message("Save failed. Error code = " + string(xboxlive_get_file_error()));
}

上記のコードでは、保存時にエラーが発生したかどうかを確認し、エラーが発生した場合はエラーコードをコンソールに出力しています。
