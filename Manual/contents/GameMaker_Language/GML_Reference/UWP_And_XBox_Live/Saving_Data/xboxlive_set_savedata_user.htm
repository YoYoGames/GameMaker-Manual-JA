xboxlive_set_savedata_user

この関数は、今後セーブゲーム領域で操作するファイル操作（すなわち、すべてのファイルの書き込み、およびバンドル領域にないファイルからの読み込み）を、指定されたユーザーIDポインタに関連付けることを指定する。これは、セーブデータを適切なユーザにリダイレクトするために何度でも呼び出すことができます。また、定数 pointer_null を使用して、一般的なマシン記憶領域にセーブすることも可能です。

構文です。

xboxlive_set_savedata_user(user_id);

論証 タイプ 商品説明
user_id Xbox ユーザー IDまたはpointer_null The user ID (a pointer) to set for saving, or pointer_null

リターンです。

N/A

例

if (xboxlive_get_savedata_user() != user_id[0])
{
    xboxlive_set_savedata_user(user_id[0]);
}

上記のコードでは、あるユーザーが現在保存先として割り当てられているかどうかを確認し、割り当てられていない場合は割り当てます。
