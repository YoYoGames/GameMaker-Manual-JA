uwp_is_constrained

この関数は、ゲームがリソース制限モードで動作している場合は true を、通常動作している場合は false を返します。リソース制約モードは、ゲームにフォーカスがないときにアクティブになることがあり、このモードではゲームのGPUリソースが減少しています。この場合、通常はゲームを一時停止する必要があります（ユーザーによる制御ができなくなるため）。また、このモードでスローダウンが観察される場合、特に負荷の高いエフェクトを無効化するのも良いアイデアかもしれません。とは異なることに注意してください。 uwp_is_suspending()関数では、この関数はゲームが制約モードにある限り（最初に発生したときだけでなく）継続的に true を返します。

構文です。

uwp_is_constrained();

リターンです。

例

if xboxliveis_constrained()
{
    if (!instance_exists(obj_PauseMenu))
    {
        instance_create(0,0,obj_PauseMenu);
    }
}

上記のコードでは、アプリが制約を受けているかどうかを確認し、制約を受けている場合はobject のインスタンスを作成しています（ゲームの一時停止に使用します）。
