uwp_license_trial_version

この関数は、ゲームがトライアルライセンスであるかどうかを確認するために使用することができます。試用版であれば、この関数は true を、そうでなければ false を返します。

構文です。

uwp_license_trial_version();

リターンです。

例

if uwp_license_trial_version()
{
    global.LevelCap = 10;
}

上記のコードでは、アプリがトライアルライセンスであるかどうかを確認し、トライアルライセンスであれば、グローバル変数を設定しています。
