uwp_show_help

この関数は、Xbox One ヘルプアプリを起動し、ゲームのヘルプファイル (XBox Developer Portal に事前にアップロードした HTML 形式のファイル) を表示するものです。この関数を呼び出す際には、ユーザーのロケール情報を表示するために、ユーザーIDを指定することに注意してください。ユーザー ID は、以下の関数で取得できます。 xboxlive_get_user(), xboxlive_get_activating_user()または uwp_license_trial_user().

構文です。

uwp_show_help(user_id);

論証タイプ 商品説明
user_id The User ID pointer to open the helpfile for.

リターンです。

例

if gamepad_button_check_pressed(0, gp_start)
{
    if uwp_license_trial_version()
    {
        var _uid = uwp_license_trial_user;
        uwp_show_help(_uid);
    }
}

上記のコードでは、ボタンが押されたかどうかを確認し、アプリがトライアルライセンスで実行されているかどうかをチェックします。もしそうなら、ライセンスのユーザーIDを取得し、そのユーザー用のヘルプファイルを開きます。
