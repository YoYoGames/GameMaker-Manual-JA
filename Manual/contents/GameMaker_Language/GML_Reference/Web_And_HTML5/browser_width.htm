ブラウザ幅

この変数は、ゲームが実行されているブラウザの幅（ピクセル単位）を保持する。ブラウザが存在しない場合は、ウィンドウサイズが返される。次の図は、この変数とWindow Functionsで利用可能な他の位置決め関数との関係を示している。

構文です。

browser_width

リターンです。

例

if (browser_height > window_get_height()) || (browser_width > window_get_width())
{
    var xx, yy;
    if (browser_width > window_get_width())
    {
        xx = (browser_width - window_get_width()) / 2;
    }
    else
    {
        xx = 0;
    }
    if (browser_height > window_get_height())
    {
        yy = (browser_height - window_get_height()) / 2;
    }
    else
    {
        yy = 0;
    }
    window_set_position(xx, yy);
}

上記のコードでは、ブラウザの幅と高さをチェックし、どちらかが現在のウィンドウの幅と高さより大きいかどうかを確認します。もしどちらかが大きければ、ゲーム・ウィンドウのブラウザ内での新しい位置を計算する。
