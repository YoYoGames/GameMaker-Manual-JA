クリッカブルセットスタイル

この関数は、指定されたボタンの CSS スタイルプロパティを、指定されたキーと値のペアで設定することができます。 DS Map.を使用して）ボタン要素を事前に作成しておく必要があります。 clickable_add()) と DS Map に格納されているインデックスを引数として与えます。

構文です。

clickable_set_style(index, map)

論証 タイプ 商品説明
index クリッカブルID The index of the clickable icon to style.
map DSマップID The index of the DS Map からスタイルを設定します。

リターンです。

N/A

例

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

上記のコードは、変数「button」に格納されたインデックスで、クリック可能なボタンのスタイルを変更します。
