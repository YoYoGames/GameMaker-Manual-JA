クリッカブルエキサイト

この関数は、指定されたインデックスでクリッカブル DOM アイコンが作成されているかどうかを返します。チェックに使われる値は以前に初期化されている必要があり、そうでない場合はエラーが発生してGameMakerが終了することに注意すること。

構文です。

clickable_exists(index);

論証 タイプ 商品説明
index クリッカブルID The index of the clickable to check.

リターンです。

Boolean

例

if !clickable_exists(home_but)
{
    home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

上記のコードでは、変数「home_but」がすでに存在するかどうかをチェックし、存在しない場合はゲームキャンバスが実行されているページの位置（32, 32）にクリック可能なDOMアイコンを作成します。アイコンはsprite ページから "spr_MS_Home" として参照されているtexture を使用し、クリックすると指定した URL の新しいウィンドウが、そのウィンドウ用に定義されたプロパティとともに開きます。
