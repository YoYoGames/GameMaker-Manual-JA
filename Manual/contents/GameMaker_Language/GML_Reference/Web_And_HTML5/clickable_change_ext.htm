クリッカブルチェンジエクスト

で作成したクリッカブルアイコンのsprite と位置を変更することができる機能です。 clickable_add().位置はキャンバスではなくウィンドウの (0,0) 位置を基準にしていることと、sprite はtexture ページから直接参照しなければならないことに留意してください（参照。 sprite_get_tpe()).この関数では、使用するsprite のスケール (デフォルトは 1、0.5 は半分、2 は 2 の倍数) と、スクリーン上の最終的なアイコンのアルファ値 (0 (完全に透明) から 1 (完全に不透明)) を変更することも可能です。

構文です。

clickable_change_ext(index, tpe, x, y, alpha, scale)

論証 タイプ 商品説明
index クリッカブルID The index of the clickable icon to change.
tpe テクスチャページエントリー The texture page entry for the sprite to be used
x リアル The new x position within the window.
y リアル The new y position within the window.
scale リアル The scale of the icon (default 1).
alpha リアル The image alpha of the icon (default 1).

リターンです。

N/A

例

switch (room)
{
    case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
    case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

上記のコードは、変数 "global.Help" にインデックスされたクリッカブルアイコンの画像インデックスと位置を、現在のroom に応じて変更し、アイコンのスケールとアルファも変更します。
