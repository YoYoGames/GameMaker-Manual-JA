クリッカブルデリート

で作成したクリック可能なアイコンを削除するには、この関数を使用する必要があります。 clickable_add()をゲーム・ウィンドウから削除する。

構文です。

clickable_delete(index);

論証 タイプ 商品説明
index クリッカブルID Index of the clickable icon to remove.

リターンです。

N/A

例

if clickable_exists(global.Help_Icon)
{
    clickable_delete(global.Help_Icon);
}

上記のコードは、グローバル変数 "Help_Icon "にインデックスされたクリック可能なアイコンが存在する場合、それをゲーム・ウィンドウから削除するものです。
