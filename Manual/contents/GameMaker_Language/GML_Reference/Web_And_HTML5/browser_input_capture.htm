browser_input_capture

この関数では、ブラウザウィンドウがすべての入力をキャプチャするか（ false に設定）、ゲームが入力をキャプチャするか（ true に設定）を設定することができます。なお、この関数はHTML5モジュールでのみ使用できます。

構文です。

browser_input_capture(enable);

論証 タイプ 商品説明
enable ブーリアン Enable or disable game input capture

リターンです。

N/A

例

browser_input_capture(true);

上記のコードでは、HTML5ターゲットプラットフォームで、すべてのブラウザ入力を取り込むようにゲームを設定しています。
