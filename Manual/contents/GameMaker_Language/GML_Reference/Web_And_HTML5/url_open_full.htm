url_open_full

指定されたURLを、選択されたターゲットデバイスのブラウザ、またはHTML5モジュールを使用している場合は、現在開いているブラウザで開きます。指定する「target」パラメータは、 open() メソッドを使用した場合の標準的なJavaScriptの「name」値と同じです（「_self」以外は、ブラウザがブロックしたり、ユーザーに許可するかどうか尋ねる結果になることがあるので注意してください）。「options」は、新しいウィンドウ/タブが表示すべきプロパティを制御するJavaScriptの「spec」パラメータと同じです（すべてのブラウザがすべての機能をサポートしていない可能性もあります）。

有効な対象は

ターゲット 商品説明
_blank リンク先の文書を新しいウィンドウまたはタブで開きます（ユーザーによってポップアップがブロックされている場合、これは機能しません。 clickable_*関数を使用してください）。
_自分 リンク先の文書を、クリックされたのと同じフレームで開きます（デフォルトはこれ）。
親 リンク先のドキュメントを親フレームで開きます。
_トップ リンク先のドキュメントをウィンドウの全面に表示します。

有効なオプションは以下の通りです。

パラメータ 商品説明
'height=[px]'です。 ウィンドウの高さを指定します。最小値は100です。
'幅=[px]' ウィンドウの幅を指定します。最小値は100です。
'left=[px]'とする。 ウィンドウの左位置。
'top=[px]'とする。 ウィンドウの先頭位置（IEのみ）。
'location=[boolean]'とする。 アドレスフィールドを表示するかどうか（デフォルトは1）。
'menubar=[boolean]'とする。 メニューバーを表示するかどうか（デフォルトは1）。
'resizable=[boolean]'です。 ウィンドウのサイズを変更可能かどうか（デフォルトは1）。
'scrollbars=[boolean]'とする。 スクロールバーを表示するかどうか（デフォルトは1）。
'status=[boolean]'とする。 ステータスバーを付けるかどうか（デフォルトは1）。
'titlebar=[boolean]'です。 タイトルバーを表示するかどうか。これは、呼び出し側のアプリケーションがHTMLアプリケーションまたは信頼できるダイアログボックスでない限り無視されます（デフォルトは1）。
'toolbar=[boolean]'です。 ブラウザのツールバーを表示するかどうか（デフォルトはyes）。

構文です。

url_open_full(url, target, options);

論証 タイプ 商品説明
url 文字列 The URL (website address) to link to.
target 文字列 This is the target area to open the URL in (see description).
options 文字列 Standard browser options (see description).

リターンです。

N/A

例

url_open_full("http://gamemaker.io", "_blank", "resizable=0, height=200, scrollbars=0");

この場合、GameMakerのホームページは、サイズ変更不可、高さ200ピクセル、スクロールバーなしの新しいウィンドウで開かれることになります。
