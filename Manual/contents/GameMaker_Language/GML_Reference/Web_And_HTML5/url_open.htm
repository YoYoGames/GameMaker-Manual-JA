url_open

指定されたURLを、選択されたターゲットデバイスのブラウザで、またはHTML5モジュールを使用している場合は、現在開いているブラウザで開きます。

構文です。

url_open(url);

論証 タイプ 商品説明
url 文字列 The URL (website address) to link to.

リターンです。

N/A

例

url_open("http://gamemaker.io");

そうすると、現在のウィンドウにGameMakerのホームページが開かれます。
