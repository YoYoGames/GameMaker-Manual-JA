クリッカブルアド

この関数を使うと、ゲームキャンバスの外側で、ブラウザウィンドウの中で「ボタン」として機能する、クリック可能なカスタムアイコンを作成することができます。この関数はID値を返します。このID値は変数に格納し、新しいクリッカブルを参照するすべてのボタン関数で使用する必要があります。この関数は、ボタンをクリックしたときにブラウザがポップアップウィンドウを作成するのを防ぐために非常に便利です。

texture まず、ブラウザのウィンドウの（0,0）に対する相対位置（左上隅）を設定し、ボタンアイコンに持たせたいsprite のページエントリーを指定します（参照。 sprite_get_tpe()).次に、開くURLか、script （GameMakerの Asset Browserから）を指定して、開いたり、実行したりすることができます。指定する "target "パラメーターは、 open() メソッドを使用したときの標準的なJavaScriptの "name "値と同じであり（'_self'以外はブラウザがブロックしたり、ユーザーに許可するかどうかを尋ねる結果になることがあるので注意）、"params" 引数は新しいウィンドウ/タブが表示すべきプロパティを制御するためのJavaScript "specs" パラメーターと同じだ（ブラウザによってはすべての機能がサポートしているとは限りません）。アイコンを使って内部を実行する場合script 、これら最後の2つの引数は空strings ""のままにしておくことができます。

有効な対象は

ターゲット 商品説明
_blank リンク先のドキュメントを新しいウィンドウまたはタブで開きます。
_自分 リンク先の文書を、クリックしたときと同じフレームで開きます（デフォルトはこれ）。
親 リンク先のドキュメントを親フレームで開きます。
_トップ リンク先のドキュメントをウィンドウの全面に表示します。

有効なパラメータは以下の通り。

パラメータ 商品説明
'height=[px]'です。 ウィンドウの高さで、最小値は100です。
'幅=[px]' ウィンドウの幅を指定します。最小値は100です。
'left=[px]'とする。 ウィンドウの左位置。
'top=[px]'とする。 ウィンドウの先頭位置（IEのみ）。
'location=[boolean]'とする。 アドレスフィールドを表示するかどうか（デフォルトは1）。
'menubar=[boolean]'とする。 メニューバーを表示するかどうか（デフォルトは1）。
'resizable=[boolean]'です。 ウィンドウのサイズを変更可能かどうか（デフォルトは1）。
'scrollbars=[boolean]'とする。 スクロールバーを表示するかどうか（デフォルトは1）。
'status=[boolean]'とする。 ステータスバーを付けるかどうか（デフォルトは1）。
'titlebar=[boolean]'とする。 タイトルバーを表示するかどうか。これは、呼び出し側のアプリケーションがHTMLアプリケーションまたは信頼できるダイアログボックスでない限り無視されます（デフォルトは1）。
'toolbar=[boolean]'です。 ブラウザのツールバーを表示するかどうか（デフォルトはyes）。


また、URLの引数がスクリプト関数の名前（つまり、"http://"で始まっていない）であり、script が gmcallback_* と呼ばれている場合、html5イベントに直接フィードできることも知っておくとよいでしょう。GameMakerのスクリプト関数名は、 gmcallback_ で始まっていれば難読化されない。

構文です。

clickable_add(x, y, tpe, url, target, params);

論証 タイプ 商品説明
x リアル The x position within the window.
y リアル The y position within the window.
tpe テクスチャページエントリー The texture page entry for the sprite to be used.
url 文字列 The URL (website address) to link to, or the script to be run.
target 文字列 This is the target area to open the URL in (see description).
params 文字列 The various parameters for the icon (see description).

リターンです。

Clickable ID

例

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

上記のコードでは、ゲームキャンバスが実行されているページの位置 (32, 32) に、クリック可能な DOM アイコンを作成しています。アイコンはsprite ページから "spr_MS_Home" として参照されているtexture を使用し、クリックすると指定した URL の新しいウィンドウが、そのウィンドウ用に定義されたプロパティとともに開きます。
