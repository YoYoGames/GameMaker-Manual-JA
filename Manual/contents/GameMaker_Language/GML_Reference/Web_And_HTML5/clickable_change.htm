クリッカブルチェンジ

で作成したクリッカブルアイコンのsprite と位置を変更することができる機能です。 clickable_add().なお、位置はキャンバスではなくウィンドウの（0,0）位置を基準にしていることと、sprite はtexture ページから直接参照する必要があることに注意してください（参照。 sprite_get_tpe()).

構文です。

clickable_change(index, tpe, x, y)

論証 タイプ 商品説明
index クリッカブルID The index of the clickable icon to change.
tpe テクスチャページエントリー The texture page entry for the sprite to be used.
x リアル The new x position within the window.
y リアル The new y position within the window.

リターンです。

N/A

例

switch (room)
{
    case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
    case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

上記のコードは、現在のroom に応じて、変数 "global.Help" にインデックスされたクリッカブルアイコンの画像インデックスと位置を変更します。
