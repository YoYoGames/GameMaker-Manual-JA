url_open_ext

指定されたURLを、選択されたターゲットデバイスのブラウザ、またはHTML5モジュールを使用している場合は、現在開いているブラウザで開きます。指定する「target」パラメータは、 open() メソッドを使用した場合の標準的なJavaScriptの「name」値と同じです。「_self」以外は、ブラウザがブロックしたり、ユーザーに許可するかどうか尋ねる結果になる可能性があるので、注意が必要です。

有効な対象は

ターゲット 商品説明
_blank リンク先の文書を新しいウィンドウまたはタブで開きます（ユーザーによってポップアップがブロックされている場合、これは機能しません。 clickable_*関数を使用してください）。
_自分 リンク先の文書を、クリックされたのと同じフレームで開きます（デフォルトはこれ）。
親 リンク先のドキュメントを親フレームで開きます。
_トップ リンク先のドキュメントをウィンドウの全面に表示します。

構文です。

url_open_ext(url, target);

論証 タイプ 商品説明
url 文字列 The URL (website address) to link to.
target 文字列 This is the target area to open the URL in.

リターンです。

N/A

例

url_open_ext("http://gamemaker.io", "_blank");

そうすると、GameMakerのホームページが新しいウィンドウで表示されます。
