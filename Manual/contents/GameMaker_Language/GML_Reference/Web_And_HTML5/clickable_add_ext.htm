クリッカブルアドエクスト

この関数を使うと、ゲームキャンバスの外側で、ブラウザウィンドウの中で「ボタン」として機能する、クリック可能なカスタムアイコンを作成することができます。この関数はID値を返します。このID値は変数に格納し、新しいクリッカブルを参照するすべてのボタン関数で使用する必要があります。この関数は、ボタンをクリックしたときにブラウザがポップアップウィンドウを作成するのを防ぐために非常に便利です。

この関数は clickable_add()関数では、ブラウザウィンドウの (0,0) を基準とした位置を設定し、ボタンアイコンに持たせたいtexture のページエントリをsprite に指定する必要があります (参照)。 sprite_get_tpe()) を指定し、開くべきURLか、script (GameMaker のAsset Browser から) を指定し、使用するターゲットとパラメータを設定する。ただし、この関数には2つのオプションがあり、アイコンのアルファ値を0（完全に透明）から1（完全に不透明）の間で設定できるほか、アイコンのスケール（使用するsprite に適用する乗数値）を設定することが可能です。つまり、値 1 はデフォルトのスケールで、(たとえば) 0.5 は半分のスケール、2 は 2 倍のスケールになります。

また、URLの引数がスクリプト関数の名前（つまり、"http://"で始まっていない）であり、script が gmcallback_* と呼ばれている場合、html5イベントに直接フィードできることも知っておくとよいでしょう。GameMakerのスクリプト関数名は、 gmcallback_ で始まっていれば難読化されない。

構文です。

clickable_add_ext(x, y, tpe, url, target, params, alpha, scale);

論証 タイプ 商品説明
x リアル The x position within the window.
y リアル The y position within the window.
tpe テクスチャページエントリー The texture page entry for the sprite to be used.
url 文字列 The URL (website address) to link to, or the script to be run.
target 文字列 This is the target area to open the URL in (see description).
params 文字列 The various parameters for the icon (see description).
alpha リアル The image alpha of the icon (default 1).
scale リアル The scale of the icon (default 1).

リターンです。

Clickable ID

例

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2);

上記のコードでは、ゲームキャンバスが実行されているページの位置 (32, 32) に、クリック可能な DOM アイコンを作成しています。アイコンはsprite ページから "spr_MS_Home" として参照されているtexture を使用し、クリックすると指定した URL の新しいウィンドウが、そのウィンドウ用に定義されたプロパティとともに開きます。ボタンはまた、元のsprite のサイズの2倍に拡大され、画像のアルファ値が 1 の半分となります。
