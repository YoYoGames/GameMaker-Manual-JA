アナリティクス・イベント

この関数は、HTML5ゲームオプションで設定した分析プロバイダーに、指定されたテキストを送信します。この関数は、使用しているプロバイダーの範囲外のものを追跡するためのカスタム分析を作成するために使用することができる。

構文です。

analytics_event(string);

論証タイプ 商品説明
string A string to send to the provider

リターンです。

例

if hs_new
{
    analytics_event("New hiscore of " + string (score));
}

上記のコードは、変数が true であるかどうかをチェックし、もしそうであれば、指定されたstring で特別な分析イベントをトリガーします。
