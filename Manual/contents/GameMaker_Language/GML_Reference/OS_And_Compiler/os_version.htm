os_version

この変数には、ゲームを実行しているOSのバージョン番号が入ります。例えば、Windows 10 で実行している場合、 os_version は 655360 となります。

構文です。

os_version

リターンです。

例

if (os_type == os_android) && (os_version > 10)
{
    global.GFX = 1;
}

上記のコードでは、OSの種類とバージョン番号をチェックし、両方が正しい場合は、グローバル変数に1が設定されます。
