外部通話

を使用して dll または dylib への外部関数呼び出しを作成した場合、その関数が使用されます。 external_define()この関数を使用して、その関数を呼び出すことができます。先に定義された関数の名前と，その関数が必要とする各引数（各引数は正しい型，実数またはstring でなければなりません）を与えれば，その関数は外部呼び出しの結果を返します．

構文です。

external_call(id, args[0...15]);

論証 タイプ 商品説明
id 外部機能 The name of the function that you want to call
args[0...10] 実数または文字列 The different arguments that you want to pass through to the external function

リターンです。

Variable (the type of value returned will depend on the defined function)

例

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

上記のコード例では、あらかじめ定義された外部関数を呼び出し、戻り値をローカル変数に格納しています。
