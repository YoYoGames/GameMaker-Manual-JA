environment_get_variable

この関数は、与えられた名前の環境変数の値（string ）（同じくstring ）を返します。macOS とUbuntu (Linux) で利用可能な環境変数は、ターミナルアプリに "env" と入力することで取得することができます。 Windows 環境変数の情報は、コマンドプロンプトを使用している場合は、"echo %PATH%"、PowerShell を使用している場合は、"ls env"と入力してください。macOS とUbuntu (Linux) の両方において、"HOME" と入力することに注意してください。 環境変数は、"~/"path を返し、これはmacOS の "/Users/<username>" とUbuntu (Linux) の "/home/<username>" に対応します。

注：この機能は、Windows 、macOS 、Ubuntu (Linux) のみです。

構文です。

environment_get_variable(name);

論証タイプ 商品説明
name The name (a string) of the environment variable to check.

リターンです。

例

e_str = environment_get_variable("APPDATA");

上記のコードは、環境変数 "%appdata%" の完全なpath を返します。通常、 "C:\Users\{username}\AppData\Roaming" です。
