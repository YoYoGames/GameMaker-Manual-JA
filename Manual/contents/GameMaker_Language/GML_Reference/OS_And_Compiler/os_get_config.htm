os_get_config

この関数は、ゲームで現在選択されている設定の名前（string ）を返します。コンフィギュレーションの詳細については、コンフィギュレーションのセクションを参照してください。

構文です。

os_get_config()

リターンです。

例

if os_get_config() = "Free_Version"
{
    global.Ads = true;
}
else global.Ads = false;

上記のコードは、どの設定が使用されているかを確認し、それが「Free_Version」と呼ばれるものであれば、ゲーム内で広告が有効になるようにします。
