os_powersave_enable

この機能により、デバイスの省電力機能をオンまたはオフにすることができます。これは、特定のゲーム（例えばチルト機能を使用するもの）では、OSがユーザー入力と解釈できるイベントが発生しないため、画面をシャットダウンしたりゲームを終了したりすることがあるため、重要な機能です。この機能を false に設定することで、省電力機能を無効にし、画面（およびゲーム）が常に機能するようにすることができます。

注意：iOS 、Android 対象に限定されます。

構文です。

os_powersave_enable(flag)

論証タイプ 商品説明
flag Set to true または 、パワーセーブ機能の有効・無効を設定します。 false

リターンです。

例

if (os_type == os_android) || (os_type == os_ios)
{
    os_powersave_enable(false);
}

上記のコードでは、OSの種類をチェックし、Android またはiOS のどちらかであれば、省電力機能を解除しています。
