パラメータ数

コマンドラインパラメーターは、プログラムの実行方法を変更するためにexeに追加できる追加コマンドのことです。この関数を使うと、現在のゲームのパラメータ数を知ることができます。 関数が返すインデックス（値0は特殊で、ゲーム実行ファイルのファイル名であり、path を含む）。この関数は、HTML5プラットフォーム上で、URLパラメータを取得するために動作することに注意する必要があります。

構文です。

parameter_count();

リターンです。

例

p_num = parameter_count();

上記のコードでは、ゲームに使用されたコマンドラインパラメータの数を変数「p_num」に格納します。
