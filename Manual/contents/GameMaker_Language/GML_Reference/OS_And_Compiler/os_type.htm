os_type

この読み取り専用の変数には、ゲームがどのオペレーティング・システム用に作成されたかを示すためにGameMakerが持つ様々な定数のうちの1つが格納されます。これは必ずしも実行中のデバイスのOSと同じではないことに注意してください。例えば、あなたのゲームはAmazon Fire OSで動作しているかもしれませんが、Android プラットフォーム向けに作られています（この場合、 os_type は os_android となります）。

以下の定数を返すことができる。

構文です。

os_type;

リターンです。

OS Type Constant

例

switch (os_type)
{
    case os_windows: global.Config = 0; break;
    case os_android: global.Config = 1; break;
    case os_linux: global.Config = 2; break;
    case os_macosx: global.Config = 3; break;
    case os_ios: global.Config = 4; break;
    case os_winphone: global.Config = 5; break;
}

上記のコードでは、ゲームを実行しているOSをチェックし、それに応じてグローバル変数を設定します。
