os_device

この読み取り専用変数は、以下に示す様々な定数値のうちの1つを保持し、現在どのデバイスでゲームを実行しているのかを知らせます。この変数は非推奨であることに注意してください。 os_get_info()これは、ゲームを実行しているデバイスに関するより正確な情報を返します。

構文です。

os_device

リターンです。

Device Type Constant

デバイスタイプ定数
定数 商品説明
device_ios_ipad iPad
device_ios_ipad_retina Retinaディスプレイサイズ2048×1536の新型iPad
device_ios_iphone6 iPhone6、ディスプレイサイズ1334×750
device_ios_iphone6plus ディスプレイ1920×1080でより大きくなったiPhone6
device_ios_iphone5 iPhone5、ディスプレイサイズ640×1136)
device_ios_iphone 旧型のiPhone/iPod Touch（画面サイズ480×320）または Android 携帯電話
device_ios_iphone_retina 960 x 640のRetinaディスプレイを搭載した新しいiPhone/iPod Touch
device_emulator デバイスの正体はエミュレータ（Windows Phone またはAndroid ）です。
device_tablet Androidタブレット
device_ios_unknown 不明またはiOSではない

例

if (os_browser = browser_not_a_browser)
{
    switch (os_device)
    {
        case device_ios_ipad: global.Config = 2; break;
        case device_ios_iphone: global.Config = 3; break;
        case device_ios_iphone_retina: global.Config = 4; break;
        case device_ios_unknown: global.Config = 5; break;
    }
}
else
{
    global.Config = 1;
}

上記のコードでは、ゲームがブラウザ上で動作しているかどうかをチェックし、その結果に応じてグローバル変数に値をセットしています。
