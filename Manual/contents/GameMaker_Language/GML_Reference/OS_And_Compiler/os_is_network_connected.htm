os_is_network_connected

この関数を使用すると、デバイスが現在インターネットに接続しているかどうかを確認できます。接続している場合は true を、接続していない場合は false を返します。OSによっては、値を返す前に接続を試みることがあります。

この関数には、呼び出されたときに接続を試みるかどうかを指定するオプションの引数があり、このオプションは現在のところNintendo Switch のターゲットに対してのみ有効です。このオプションを true に設定すると接続を試み、 false に設定すると、単に接続を確認するだけになります。接続を試みると、Switch OSが数秒間ハングアップする可能性があることに注意してください。

注 この関数は、接続を制御する内部デバイスAPIをチェックするため、ブルートゥース接続、Wi-Fi接続、またはインターネットアクセスを許可する通常のネットワーク接続がある場合、真を返す可能性があります。

構文です。

os_is_network_connected([attempt_connection]);

論証 タイプ 商品説明
attempt_connection ブーリアン OPTIONAL Nintendo Switchターゲットのみ。 に設定すると、呼び出されたときに OS レベルの接続を試みます。 true

リターンです。

Boolean

例

if os_is_network_connected()
{
    global.connected = true;
}

上記のコードでは、デバイスがインターネットに接続しているかどうかを確認し、接続している場合はグローバル変数を設定します。
