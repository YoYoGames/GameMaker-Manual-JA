スケジューラ解像度の取得

この関数は、Windows のスレッドスケジューラの解像度をミリ秒単位で取得するために使用されます。スケジューラの解像度がデフォルト値（Windows ）に設定されている場合、この関数は -1 を返します。

スレッドスケジューラに関する情報は、scheduler_resolution_set() のページをご覧ください。

注意：この機能はWindowsのみ です。

構文です。

scheduler_resolution_get();

リターンです。

(or -1 for default)

例

scheduler_resolution = scheduler_resolution_get();

この例では、スレッドスケジューラWindows の解像度を取得し、変数"scheduler_resolution" に格納する。
