os_get_region

この関数は、ゲームを実行している OS のISO3166-1規格で定められた 2 文字または 3 文字の Regional Code をstring に返します。情報がない場合は、単に空のstring ""を保持します。

注：返されるのは位置情報地域コードではなく、OSの地域言語コードです。

構文です。

os_get_region()

リターンです。

例

switch (os_get_language())
{
    case "zh":
        var region = os_get_region();
        if (region == "HK" || region == "MO" || region == "TW")
        {
            ini_open("chinese_traditional.ini");
        }
        else
        {
            ini_open("chinese_simplified.ini");
        }
    break;

    case "fr":
        ini_open("french.ini");
    break;

    case "it":
        ini_open("italian.ini");
    break;

    default:
        ini_open("english.ini");
    break;
}

上記のコードは、OSの言語をチェックし、それが中国語であれば、OSの地域をチェックし、返された値によって異なる *.ini ファイルを開く。
