os_browser

この読み取り専用変数は、GameMakerが現在どのブラウザでゲームを実行しているかを知るための様々な定数（以下に列挙）のうちの一つを保持している。

構文です。

os_browser

リターンです。

Browser Type Constant

ブラウザタイプ定数
定数 商品説明
browser_not_a_browser ブラウザでゲームをプレイしていない
browser_unknown 不明なブラウザ
browser_ie インターネットエクスプローラ
browser_ie_mobile モバイル端末のInternet Explorer
browser_firefox Mozilla Firefox
browser_chrome グーグルクローム
browser_safari サファリ
browser_safari_mobile モバイル端末のSafari
browser_opera オペラ
browser_tizen Tizenモバイルデバイスブラウザ
browser_windows_store Windowsアプリ

例

if (os_browser == browser_not_a_browser)
{
    global.Config = 0;
}
else
{
    global.Config = 1;
}

上記のコードでは、ゲームがブラウザ上で動作しているかどうかをチェックし、その結果に応じてグローバル変数に値をセットしています。
