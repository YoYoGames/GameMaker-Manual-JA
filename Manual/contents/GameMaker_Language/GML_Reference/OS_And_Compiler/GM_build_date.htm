GM_build_date

この定数は、実行中の実行ファイルがGameMakerによってビルドされた日付と時刻を保持する。この値はDate and Time関数を使ってパースすることができ、コンパイル時にシステムのUTC値から取得される。

構文です。

GM_build_date;

リターンです。

Datetime

例

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);

上記のコードでは、ゲームのバージョン番号とコンパイルされた日時が描画されます。
