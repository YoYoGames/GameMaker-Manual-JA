os_get_language

この関数は、ゲームを実行している OS のISO639規格で定められた 2 文字の Language Code をstring に返します。情報がない場合は、単に空のstring ""、または "英語" 言語を表す "en" が格納されます。言語によっては、地域コードもあるため、同じ国の異なる地域を区別するには、関数 os_get_region().

次の表は、ISO 639で定義されている主な2文字言語コードの例を示しています。

言語 コード
アラビア語 アール
唐 zh
デンマーク語 ダ
イングリッシュ エン
フレンチ フラン
ジャーマン デ
グリーク エル
イタリアン それ
やまと ジャ
ノルウェー語 ノー
ポーランド語 プラ
ポルトガル語 pt
ロシア語 ル
スパニッシュ ズ
スウェーデン語 ヴァイキング

注：返されるのは所在地の国コードではなく、OSの言語コードです。

構文です。

os_get_language()

リターンです。

例

switch (os_get_language())
{
    case "es": ini_open("spanish.ini"); break;
    case "fr": ini_open("french.ini"); break;
    case "it": ini_open("italian.ini"); break;
    default: ini_open("english.ini"); break;
}

上記のコードでは、OSの言語をチェックし、返された値に応じて異なる *.ini ファイルを開いています。
