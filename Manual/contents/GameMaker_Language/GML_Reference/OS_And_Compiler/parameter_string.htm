パラメータ文字列

コマンドラインパラメータとは、プログラムの実行方法を変更するためにexeに追加できる追加コマンドのことで、この関数を使用すると、選択したコマンドラインパラメータをstring 。 関数を使用すると、現在のゲームに関するパラメータの数を確認できます。 parameter_count()ここで、最初のパラメータはインデックス1、最後のパラメータは関数が返すインデックスです（値0は、ゲームの実行ファイルのファイル名であり、path を含むという点で特別です）。この関数は、HTML5プラットフォーム上で、URLパラメータを取得するために動作することに注意する必要があります。

構文です。

parameter_string(n);

リターンです。

例

var p_num;
p_num = parameter_count();
if p_num > 0
{
    var i;
    for (i = 0; i < p_num; i += 1)
    {
        p_string[i] = parameter_string(i + 1);
    }
}

上記のコードは、コマンドラインパラメータの数を取得し、1つ以上ある場合は、loop 、配列にstrings として格納します。
