外部フリー

この関数は、指定された名前の dll または dylib に関連するメモリを解放する。この関数は、そのファイルがゲーム内で不要になったとき、通常は（たとえば）Game End イベントで実行されるべきものです。

構文です。

external_free(id);

論証 タイプ 商品説明
id 外部機能 The name of the dll または を解放する必要があります。 dylib

リターンです。

N/A

例

external_free("MyDLL.dll");

上記のサンプルコードでは、指定されたDLLに関連するメモリを解放します。
