os_lock_orientation

この機能により、ターゲットプラットフォームでゲームオプションが有効なすべての方向性を再び許可するために「解放」するまで、デバイスを現在の方向に「ロック」することができます。ロックする前に、希望する方向であることを確認する必要があります。

構文です。

os_lock_orientation(flag)

論証タイプ 商品説明
flag Set to true または 、オリエンテーションロックを有効または無効にします。 false

リターンです。

例

if (os_type == os_android) || (os_type == os_ios)
{
    os_lock_orientation(true);
}

上記のコードはOSの種類をチェックし、Android またはiOS のどちらかであれば、オリエンテーションロックは true としてフラグが立てられます。
