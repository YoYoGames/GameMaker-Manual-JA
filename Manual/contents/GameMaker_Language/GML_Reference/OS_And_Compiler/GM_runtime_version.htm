GM_runtime_version（ランタイムバージョン

この定数は、プロジェクトのビルドに使用されているruntime として、環境設定で定義されたruntime バージョン番号を保持します。この値は、string として格納されます。

構文です。

GM_runtime_version;

リターンです。

String

例

draw_text(32, 32, date_time_string(GM_build_date));
draw_text(32, 64, "v" + GM_version);
draw_text(32, 96, "Runtime " + GM_runtime_version);

上記のコードでは、GMの日付、バージョン、runtime の定数を受け取り、画面に描画しています。
