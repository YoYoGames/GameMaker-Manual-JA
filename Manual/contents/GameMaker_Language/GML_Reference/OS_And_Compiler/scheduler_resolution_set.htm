スケジューラレゾリューションセット

この関数は、Windows のスレッドスケジューラの解像度を変更するために使用します。引数で与える分解能の値はミリ秒である必要がある。

デフォルトでは ゲームメーカーこれは、許容される最小の解像度であり、最高のパフォーマンスを発揮します。解像度を高い値に変更すると、パフォーマンスに悪影響が出る場合があります。 ただし、消費電力を抑えることができる場合があります。

引数に -1 を与えると、解像度をデフォルト値 (Windows で設定された値) に戻すことができます。スケジューラの解像度を変更すると、ゲームと同時に動作しているすべてのプロセスに影響があることに注意してください。

注意：この機能はWindowsのみ です。

構文です。

scheduler_resolution_set(milliseconds);

論証タイプ 商品説明
milliseconds The new resolution value (in milliseconds) or -1 for default

リターンです。

例

scheduler_resolution_set(2);

この例では、Windows のスレッドスケジューラの解像度を 2 ミリ秒に設定しています。
