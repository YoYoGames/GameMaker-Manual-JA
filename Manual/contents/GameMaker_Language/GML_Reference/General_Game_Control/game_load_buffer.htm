ゲームロードバッファ

この関数を使用すると、以前に保存されたゲーム状態を読み込むことができます。ゲームはあらかじめ作成された "grow"buffer (バッファを参照)からロードされ、buffer にはゲーム状態が game_save_buffer()関数を使用します。

構文です。

game_load_buffer(buffer);

論証タイプ 商品説明
buffer The buffer id to load from.

リターンです。

例

if keyboard_check_pressed(ord("L"))
{
    if global.Checkpoint game_load_buffer(save_buff);
}

上記のコードは、プレイヤーが「L」キーを押したときに、グローバル変数が true である場合に限り、変数「save_buff」にインデックスが付けられたbuffer から、以前に保存したゲーム状態をロードします。
