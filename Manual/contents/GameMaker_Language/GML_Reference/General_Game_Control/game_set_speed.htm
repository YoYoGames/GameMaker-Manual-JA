ゲームスピードの設定

この機能により、ゲームスピードを設定することができます。ゲームフレーム/秒（FPS）、ゲームフレーム/マイクロ秒（MPF）の2つの方法があり、次の2つの定数のいずれかを使用して設定します。

定数 商品説明
gamespeed_fps ゲームスピードをフレーム/秒単位で設定します。
gamespeed_microseconds ゲームスピードを1フレームあたりマイクロ秒単位で設定します。

つまり、例えばゲームスピードが30フレーム/秒の場合、1ゲームゲームフレームあたり33333マイクロ秒となり、この関数で表現すると、次のどちらかになります。

game_set_speed(30, gamespeed_fps);

または

game_set_speed(33333, gamespeed_microseconds);

構文です。

game_set_speed(speed, type);

論証 タイプ 商品説明
speed リアル The new game speed (as either FPS or MPF).
type ゲームスピード一定 The type of method used to set the game speed (see the constants above).

リターンです。

N/A

例

if os_browser == browser_not_a_browser
{
    game_set_speed(60, gamespeed_fps);
}
else
{
    game_set_speed(30, gamespeed_fps);
}

上記のコードでは、ゲームがブラウザで実行されているかどうかを確認し、それに応じてゲーム速度をFPS値として設定しています。
