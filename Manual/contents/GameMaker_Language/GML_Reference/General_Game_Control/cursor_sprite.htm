cursor_sprite

この変数を設定することで ゲームメーカーは、指定されたsprite をカーソルとして使用します (基本的に、1 ステップごとに現在のマウスの x/y 位置に設定します)。デフォルト値は-1で、カーソルにsprite を指定しませんが、ゲームsprite や外部リソースからインポートしたassets のインデックスを自由に割り当てることができます。アニメーションの速度やimage_indexを制御する方法はないので、sprite にサブ画像がある場合、これらはroom の速度と同じ速度で循環することに注意してください。カーソルsprite を削除するには、この変数を再度 -1 に設定します。

また、この変数はゲームウィンドウのカーソルを置き換えるものではなく、通常通り描画されることも知っておくとよいでしょう。これを避けるには、関数 window_set_cursor()を定数 cr_none に設定すると、標準カーソルは見えなくなります。

リターンです。

Sprite Asset

例

cursor_sprite = spr_CustomCursor;

上記のコードでは、変数「spr_CustomCursor」にインデックスされたsprite 、ゲーム用のカーソルsprite に設定されます。
