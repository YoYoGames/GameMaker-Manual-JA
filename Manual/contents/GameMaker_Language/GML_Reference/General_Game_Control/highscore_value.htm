ハイスコア・バリュー

この関数は、指定された位置のハイスコア・リストに格納されているスコア値を取得することができます。スコアが入力されていない場合、この関数は 0 を返します。

構文です。

highscore_value(place);

論証タイプ 商品説明
place The place on the table (1-10).

リターンです。

例

var i = 9;
repeat(10)
{
    scr[i] = highscore_value(i + 1);
    i -= 1;
}

上記のコードは、ハイスコアのリストをloop 、すべてのスコアを配列に格納します。
