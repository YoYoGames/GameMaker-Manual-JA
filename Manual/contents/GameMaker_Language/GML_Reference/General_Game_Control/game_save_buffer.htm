ゲームセーブバッファー

この機能は、 game_save() 機能の亜種です。その使用方法とこの機能に関する重要な情報が含まれていますので、まずそのページをお読みください。

この関数で、ゲームの現在の状態を、あらかじめ作成した "grow"buffer (バッファを参照) に保存することができます。 game_load_buffer()関数があります。この関数は、一度に1つのroom で使用するように設計されています（つまり、1つのbuffer でroom を保存し、別のbuffer からロードしようとするのはお勧めできません）。理想的には、チェックポイントまたはレベルの再起動にのみ使用する必要があります。

注意：この機能は非常に限定的で、初心者が素早くチェックポイントシステムを立ち上げられるように設計されています。上級者は、データ構造、サーフェス、追加したsprites などの動的リソースを一切保存しないため、ファイル機能を使って独自のシステムをコーディングする方がよいでしょう。

構文です。

game_save_buffer(buffer);

論証タイプ 商品説明
buffer The buffer id to save to.

リターンです。

例

if keyboard_check_pressed(ord("S"))
{
    global.Checkpoint = true;
    game_save_buffer(save_buff);
}

上記のコードは、グローバル変数に true を設定し、キー "S" が押されたときに、変数 "save_buff" にインデックスされたbuffer に現在のゲーム状態を保存するものです。
