game_idDEPRECATED

この読み取り専用変数は、作成したゲームの一意な識別子を返します。ユニークなファイル名が必要な場合や、ゲームだけを識別する何かが必要な場合に使用できます。これはゲームオプションで設定することができます。

構文です。

game_id

リターンです。

Real

例

ini_open("Score.ini");
ini_write_real("Scores", "0", score + game_id);
ini_close();

上記のコードでは、保存する前にgame_idをスコアに加えて、非常に基本的な暗号化を行なっています。これをゲームに読み込む際には、game_idを差し引いて正しい値を再び得ることになります。
