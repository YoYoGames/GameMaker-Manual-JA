ゲーム再開

この機能を使うと、ゲームを再開することができます。これは基本的にゲームを初めて起動するのと同じで、Game Start Eventと Game End Eventがトリガーされます。

この関数が呼ばれたとき、特定のものがリセットされないことに注意する必要があります。

  • グローバル変数は、そのように明示的にコーディングされない限り、再初期化されません。例えば、組み込みのグローバル変数 score は、ゲーム内ですでに変更されている場合、ゲーム再起動後にゼロから始まることはありません。
  • GPUの状態は変更されません（したがって、たとえば描画色やアルファを設定した場合、変更された値のままです）。
  • ゲームスピードは、ゲームコードで設定したとおりになります（変更した場合、この変更は永続的に適用されます）。
  • asset Browser のAsset は、ゲーム内で実行時に変更された場合（たとえば、sprite リソースの原点を変更したり、path リソースの位置を移動した場合）、リセットされることはありません。
  • buffers 、サーフェス、データ構造、インポートされたsprites などの動的リソースもクリーンアップまたは削除されません（ただし、それらへの参照は失われる可能性があるので、この関数を使うときは、動的リソースにグローバル参照を使用するか、この関数が呼ばれる前にそれらをクリーンアップしておくように注意してください）。

構文です。

game_restart();

リターンです。

N/A

例

if keyboard_check_pressed(ord("R")) game_restart();

これにより、プレイヤーが "R "キーを押したときにゲームが再開されるようになります。
