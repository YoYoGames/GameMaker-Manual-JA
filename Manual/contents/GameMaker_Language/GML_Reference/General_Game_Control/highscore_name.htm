ハイスコア・ネーム

この関数を使うと、ハイスコアリストの指定された位置に格納されている名前string を取得することができます。名前が入力されていない場合は、string "Unknown" が返されます。

構文です。

highscore_name(place);

論証タイプ 商品説明
place The place on the table (1-10).

リターンです。

例

var i = 9;
repeat(10)
{
    name[i] = highscore_name(i + 1);
    i -= 1;
}

上記のコードは、ハイスコアのリストをloop 、すべての名前を配列に格納します。
