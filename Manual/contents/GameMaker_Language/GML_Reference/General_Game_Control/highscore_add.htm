ハイスコア・アド

この関数を使用すると、内部グローバルハイスコアリストに名前とスコアを追加することができます。GameMakerは、保存された10位より大きい値のみを保存するので、スコアリストに入力するのに十分な高さかどうかを確認する必要はない。

構文です。

highscore_add(str, numb);

論証 タイプ 商品説明
str 文字列 The string (name) to attribute the score to.
numb リアル The number (score) to add.

リターンです。

N/A

例

if (lives == 0)
{
    highscore_add(global.Name, score);
    score = 0;
    room_goto(rm_MainMenu);
}

上記のコードでは、ライフをチェックして0になっていれば、現在のスコアとグローバル変数「Name」に保持されているstring をハイスコアリストに追加し、最後にスコアを0にして「rm_MainMenu」にインデックスされているroom にプレイヤーを送り込みます。
