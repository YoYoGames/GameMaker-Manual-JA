game_project_name

この読み取り専用変数は、ターゲットプラットフォーム用のゲームの表示名を、ターゲットプラットフォーム用の「保存しやすい」フォーマットで返します。表示名に、ファイル名として許可されていない文字が含まれている場合、それらは に自動的に置き換えられます。表示名はゲームオプションで設定することができます。なお、HTML5ではファイル名に関する制限がないため、このstring 。 が返すものと同じです。 game_display_name.

構文です。

game_project_name

リターンです。

例

var file = game_project_name + ".ini";
ini_open(file);
ini_write_real("Scores", "Highscore", score);
ini_close();

上記のコードでは、プロジェクト名を取得し、これを元にiniファイルを開き（または作成し）、書き込んでいます。
