ゲームエンド

この関数で、ゲームを終了させることができます（ゲーム終了イベントが発生します）。これは瞬時に起こるのではなく、現在のステップの終了時に起こるので、どのコードも この関数が呼ばれた後も、同じステップにあるものは実行されます。なお、この関数には以下の制約があります。

  • Android 端末では、 game_end() を呼び出すとアプリがバックグラウンドにプッシュされますが、アプリは終了しません。これはユーザーによって行われる必要があります。
  • iOS では、何もせず黙って失敗します。
  • すべてのコンソール（Xboxで実行する場合はUWPを含む）では、ゲームがクラッシュするか、せいぜいサイレントで失敗し、サブミッションフェイルになる可能性もあるので、この関数は使ってはいけません。
  • HTML5では、ゲームは終了しますが、ユーザーはウェブページ上の空白のドローキャンバスを見つめることになりますから、そのターゲットプラットフォームでの使用は推奨されません。
  • Windows (PC 上の UWP を含む)、Linux 、macOS では、この関数は単にゲームを終了してゲーム ウィンドウを閉じます (Game End Eventもトリガーされます)。

構文です。

game_end();

リターンです。

例

if keyboard_check_pressed(vk_escape) game_end();

これは、プレイヤーが「escape」キーを押すと、ゲームが終了してしまうというものです。
