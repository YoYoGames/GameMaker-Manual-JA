ゲームスピード

この関数は、ゲームスピードを1秒間に実行するゲームフレーム数、または1ゲームフレームあたりのマイクロ秒数として取得するために使用することができます。これは実際の動作速度であるFPS値ではないことに注意してください（そのためには fps_real変数）ではなく、ゲームが1秒間に維持しようとするゲームフレーム数（FPS）、またはゲームが維持しようとする各ゲームフレームの長さ（MPF）をマイクロ秒単位で表します。この関数を使用するときは、以下の定数のいずれかを指定する必要があり、それによって戻り値の型が決まります。

ゲームスピード一定
定数 商品説明
gamespeed_fps ゲームスピードをフレーム/秒単位で取得します。
gamespeed_microseconds ゲームスピードを1フレームあたりのマイクロ秒数で取得します。

つまり、例えばゲームオプションでゲームスピードを30に設定し、FPSタイプを使用した場合、関数は30を返しますが、MPFを使用した場合、関数は33333を返すことになります。

構文です。

game_get_speed(type);

論証 タイプ 商品説明
type ゲームスピード一定 The type of method used to get the game speed (see the constants above).

リターンです。

Real

例

if game_get_speed(gamespeed_fps) != 60
{
    game_set_speed(60, gamespeed_fps);
}

上記のコードでは、ゲームが60FPSのゲームスピードで動作しているかどうかを確認し、そうでない場合は60FPSに設定します。
