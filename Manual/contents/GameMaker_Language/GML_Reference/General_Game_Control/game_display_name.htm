ゲーム_表示名

この読み取り専用変数は、ゲームオプションで設定された、ターゲットプラットフォーム用のゲームの表示名を返します。

構文です。

game_display_name

リターンです。

例

var name = game_display_name;
var ver = string(GM_version);
draw_text(32, 32, name + ":" + ver);

上記のコードでは、ゲームの表示名とバージョン番号を取得して描画しています。
