game_save_id

この読み取り専用変数は、ゲームによってファイルを保存するために使用されているディレクトリの完全なpath IDを返します。このディレクトリは、プラットフォームによって、他のアプリケーションから見える場合と見えない場合があります。

HTML5ターゲットでは、空のstring を返します。

注この変数は純粋に情報のために存在し、ファイルpath を取得するためなどに使用しないでください。すべてのプラットフォームで正しく動作しません。

構文です。

game_save_id

リターンです。

String

例

save_dir = game_save_id;

これは、今後のためにファイルを保存するディレクトリを変数に格納するものです。
