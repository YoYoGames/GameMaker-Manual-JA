ロールバックマルチプレイヤー

GameMaker のロールバックマルチプレイヤーは、Opera GX ターゲットのネットワークマルチプレイヤーゲームを素早くセットアップ、作成することが可能です。

この機能は、GXサーバーを使用してプレイヤーの接続と入力の同期を行うため、開発者が新たにオンラインゲームを開始する際に、server の設定は必要ありません。

オンライン機能はOpera GXターゲットでしか使用できませんが、「Sync Test」を使用すると、ランダムまたはモック入力を他のプレイヤーに適用して、ローカルでマルチプレイヤーゲームをテストすることができます。Sync Test は、Opera GX のほか、Windows およびmacOS のターゲットで機能します。

はじめに

Create a Multiplayer Game（マルチプレイヤーゲームを作ろう）」を読んで、手順に従って最初のマルチプレイヤープロジェクトを作成してください。

最初のマルチプレイヤーゲームを作った後は、以下のページを読んでロールバックシステムについて理解を深め、何ができるかを知っておいてください。

  • ロールバック方式
  • 入力の定義
  • ロールバックコンストレイント
  • ロールバックイベント
  • ロールバック環境設定
