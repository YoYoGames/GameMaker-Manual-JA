rollback_use_player_prefs（ロールバックユーズプレイヤープレフ）。

プレイヤーの環境設定を有効にする機能で、全プレイヤーのデフォルトの環境設定を行うことができます。

デフォルトのプリファレンスは、どのようなタイプでも構いません。デフォルトの環境設定を指定しない場合、ゲームを開始する前にすべてのプレイヤーが環境設定を行う必要があります。

この関数を呼び出すと自動起動が無効になるため、手動でゲームを起動する必要があります。参照してください。マニュアルとオートスタートの比較

使用する場合、この関数はjoin/create関数の前に実行する必要があります。

必ずお読みください。プレイヤーの環境設定

構文です。

rollback_use_player_prefs([default]);

論証 タイプ 商品説明
default 任意の OPTIONAL すべてのプレーヤーに設定されるデフォルトの環境設定です。

リターンです。

N/A

例

rollback_use_player_prefs({)
character: 0,
色: c_white
});

上記のコードは、プレーヤーのプリファレンスを有効にし、デフォルトのプリファレンスをstruct に設定します。
