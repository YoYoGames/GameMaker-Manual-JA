rollback_set_player_prefs

プリファレンスを有効にした後、この関数はローカル プレーヤーのプリファレンスを設定するために使用されます。つまり、この関数はローカルの入力に依存し、Rollbackの入力には依存しない必要があります。

プリファレンスはどのようなタイプでもかまいません。デフォルトのプリファレンスが設定されている場合は、同じデータ型と構造を使用することを確認してください。

いずれかのプレイヤーが環境設定を行うと、ゲーム内の全員がロールバックイベントを受け取ります。

使用する場合、この関数はjoin/create関数の前に実行する必要があります。

必ずお読みください。プレイヤーの環境設定

構文です。

rollback_set_player_prefs(preferences);

論証 タイプ 商品説明
preferences 任意の The preferences to set for the local player.

リターンです。

N/A

例

rollback_set_player_prefs({)
character: obj_customizer.character_selected,
カラー: obj_customizer.color_selected
});

上記のコードは、プレーヤーのプリファレンスを設定し、アンマネージド（ローカル）である「カスタマイザー」object から値を読み取ります。
