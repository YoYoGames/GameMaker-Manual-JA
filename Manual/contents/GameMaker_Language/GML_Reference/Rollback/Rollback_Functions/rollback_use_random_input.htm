rollback_use_random_input

Sync Test を使ってゲームをプレイすると、基本的なテスト方法として、すべての「リモート」プレイヤーは、定義された入力に対してランダムな値を受け取ります。この機能を無効にするには、引数にfalseを指定してこの関数を呼び出し、再度有効にするにはtrue を指定します。

構文です。

rollback_use_random_input(enabled);

論証 タイプ 商品説明
enabled ブーリアン Specifies whether random input should be enabled (true)または無効(false)です。デフォルトでは、有効になっています。

リターンです。

N/A

例

rollback_use_random_input(false);

上記のコードでは、同期テスト中のランダム入力は無効となります。
