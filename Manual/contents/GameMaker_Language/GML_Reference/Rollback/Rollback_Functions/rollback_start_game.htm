ロールバック・スタート・ゲーム

この関数は、現在のマルチプレイヤーゲームを、すべてのプレーヤーが参加する前に強制的に開始させます。マルチプレイヤーゲームを作成するには、まず rollback_create_game().

通常、システムはすべてのプレイヤーが参加するのを待ってから、自動的にゲームを開始します。この機能を利用することで、すべてのプレイヤーが参加する前にゲームを開始することができます。

を呼び出すことができます。 rollback_use_manual_start()を呼び出すと、すべてのプレイヤーが参加した後でも、Rollbackはこの関数を待つようになります。

この関数を呼び出すと、ロールバック開始イベントが発生します。

構文です。

rollback_start_game();

リターンです。

N/A

例

// Step event
if (!rollback_game_running && keyboard_check_pressed(vk_enter))
{
    rollback_start_game();
}

ロールバックゲームが実行されていない状態で、Enterキーを押すとゲームが開始されます。

これにより、全プレイヤーが参加する前に、プレイヤーが手動でゲームを開始することができます。
