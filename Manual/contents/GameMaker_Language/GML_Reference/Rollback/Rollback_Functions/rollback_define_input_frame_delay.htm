rollback_define_input_frame_delay

この関数は、ローカルプレーヤー入力の固定フレーム遅延を定義する。

デフォルトでは、プレイヤー間の遅延に基づく適応型入力遅延が使用されます。これはほとんどすべてのゲームで機能しますが、特定のタイプのゲーム（例：格闘ゲーム）では、いつでも変更可能な適応型遅延ではなく、固定入力遅延を使用することを好む場合があります。

何をやっているのかよく分かっていない限り、これを変更することはお勧めしません。この関数は、マルチプレイヤーゲームの前と最中にいつでも呼び出すことができます。

遅延の値はフレーム数で指定する必要があります。1を指定すると、適応的な遅延に戻ります。設定可能な最大遅延は10フレームです。

構文です。

rollback_define_input_frame_delay(frames);

論証 タイプ 商品説明
frames リアル The number of frames to set as the fixed input delay, or -1 をクリックすると、デフォルトにリセットされます。

リターンです。

N/A

例

// 'Options' script
global.frame_delay = 5;

// obj_game: Create event
rollback_define_input_frame_delay(global.frame_delay);
var _joined = rollback_join_game();

if (!_joined)
{
    rollback_start_game(2, false);
}

フレーム遅延値をグローバル変数として定義し、ロールバックゲームが始まる前にrollback_define_input_frame_delay()を呼び出すときにそれを使用することができます。
