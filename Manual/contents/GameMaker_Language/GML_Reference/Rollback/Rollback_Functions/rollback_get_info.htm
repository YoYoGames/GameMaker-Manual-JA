rollback_get_info

この関数は、プレイヤーに関する情報を含むstruct を返します。これは、プレイヤー インスタンスを手動で管理する場合にのみ使用する必要があります。を使用している場合、各プレイヤーインスタンスは自動的にこの情報を変数として受け取ります。 rollback_define_player()を使用している場合、各プレイヤー・インスタンスは自動的にこの情報を変数として受け取ります。

この関数は、プレイヤーのIDを引数に取り、以下の変数を持つstruct 。

バリアブル タイプ 商品説明
player_avatar_url 文字列 プレイヤーのアバター画像を指すURL
player_avatar_sprite スプライトアセット プレイヤーのアバターが含まれるスプライト
player_name 文字列 プレーヤー名
player_type 文字列 GX.gamesにサインインしているかどうか。

ゲストユーザーの場合は「Guest」、サインインしている場合は「User」になります。
player_user_id 文字列 GX.gamesに登録されているプレイヤーのID
player_prefs 任意の プレーヤーに設定されているプリファレンス

構文です。

rollback_get_info(player_id);

論証 タイプ 商品説明
player_id リアル The ID of the player for which information will be returned.
