rollback_use_manual_start

デフォルトでは、マルチプレイヤーゲームは、すべてのプレーヤーが参加した時点で自動的に開始されます。

この関数を呼び出すと、自動起動の動作が無効になり、ゲームは手動で起動するのを待ちます。

使用する場合、この関数はjoin/create関数の前に実行する必要があります。

構文です。

rollback_use_manual_start();

リターンです。

N/A

例

rollback_use_manual_start();

if (!rollback_join_game())
{
    rollback_create_game(2);
}

上記のコードで手動起動を有効にし、マルチプレイヤーゲームに参加または新規作成します。
