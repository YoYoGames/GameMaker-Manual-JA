rollback_display_events

デフォルトでは、ゲームを実行すると、ゲームの左上隅にデバッグメッセージが表示され、マルチプレイヤー・イベントが発生したときに表示されます。この関数の引数にfalseを指定することでこれらのメッセージを無効にし、trueを指定することで再び有効にすることができます。

構文です。

rollback_display_events(enabled);

論証 タイプ 商品説明
enabled ブーリアン Specifies whether event display should be enabled (true)または無効(false)です。デフォルトでは、有効になっています。

リターンです。

N/A

例

rollback_display_events(false);

上記のコードでは、イベントの表示を無効にしています。
