rollback_get_input

この関数は、あるプレーヤーの入力値を含むstruct を返します。オプションの引数には、入力値が返されるプレーヤーのIDを指定します。

で生成されたプレイヤーインスタンスで使用する場合、プレイヤーIDを指定する必要はありません。 rollback_define_player()で作成されたプレイヤーインスタンスで使用する場合、プレイヤーIDを指定する必要がないため、引数なしでこの関数を呼び出すことができます。

それ以外のインスタンス、または手動で作成したプレイヤーインスタンスで使用する場合は、プレイヤーIDを指定する必要があります。

どのような入力が返され、どのように変更できるかは、「入力の定義」を参照してください。

構文です。

rollback_get_input(player_id);

論証 タイプ 商品説明
player_id リアル OPTIONAL 入力を返すプレーヤーのID。

リターンです。

Struct

例

var _input = rollback_get_input();

if (_input.left) x -= 2;
if (_input.right) x += 2;
if (_input.up) y -= 2;
if (_input.down) y += 2;

上記のコードでは、Rollback システムによって自動的に作成されたプレーヤー インスタンスで入力struct を取得します。返された値に基づいて、インスタンスを各方向に 2 ピクセルずつ移動します。

拡張例は、マルチプレイヤーゲームを作成するを参照してください。
