ロールバック・リーヴ・ゲーム

この関数は、現在のマルチプレイヤーセッションを終了し、プレーヤーが再び新しいゲームに参加または作成することを可能にします。

room のホストが退出すると、そのroom の所有権は他のプレイヤーに移るので、ゲームを続行することができる。すべてのプレーヤーが退出したときのみ、ゲームは終了する。

エラーを回避する

プレイヤーがマルチプレイヤーセッションを離れた後も、GameMaker はすべてのゲームコードを実行し続けますが、マネージドインスタンスを作成または変更できない「開始前」の状態に戻ります。

つまり、ゲーム終了時に手動でプレイヤーインスタンスを破棄し、他のゲームロジックは条件内でrollback_game_runningを使用して一時停止する必要があり、エラーを回避するために必要です。

ご覧ください。ロールバック開始前の状態

トリガーされたイベント

自分が参加しているゲームからプレイヤーが抜けると、ロールバックイベントが発生します。詳しくは、「ロールバックイベント」を参照してください。

構文です。

rollback_leave_game();

リターンです。

N/A

例

if (keyboard_check_pressed(vk_escape))
{
    rollback_leave_game();
}

上記のコードでは、Escapeキーが押されたときにプレイヤーがゲームから退出するようになっています。
