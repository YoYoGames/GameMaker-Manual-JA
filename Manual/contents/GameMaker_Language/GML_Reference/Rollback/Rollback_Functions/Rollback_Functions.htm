ロールバック機能

マルチプレイヤーゲームを開始したり、参加したり、退出したりするには、次のような機能があります。

  • ロールバックジョインゲーム
  • ロールバック・クリエイト・ゲーム
  • ロールバック・スタート・ゲーム
  • ロールバック・リーヴ・ゲーム

以下の関数は、ゲームの情報を定義したり取得したりするために使用されます。

  • セット
    • rollback_define_player（ロールバック・プレイヤー）。
    • rollback_define_input
    • rollback_define_mock_input
    • rollback_define_input_frame_delay
    • rollback_use_manual_start
    • rollback_use_player_prefs（ロールバックユーズプレイヤープレフ）。
    • rollback_set_player_prefs（ロールバック・セット・プレイヤー・プリフ）。
  • ゲット
    • rollback_get_input
    • rollback_get_info
    • rollback_get_player_prefs（ロールバック・ゲット・プレイヤー・プリフ）。

アクティブゲーム中に使用するために、以下の機能が用意されています。

  • ロールバックチャット
  • ロールバックシンクオンフレーム

以下の機能は、テスト、またはテストに関連する機能の有効化／無効化に使用されます。

  • rollback_use_random_input
  • rollback_display_events
