rollback_get_player_prefs（ロールバック・ゲット・プレイヤー・プリフ）。

この関数は、指定されたプレーヤーIDに設定されている環境設定を返します。

プレイヤーIDが指定されていない場合は、プレイヤーインスタンスの内部でこの関数を呼び出す必要があり、その場合、そのプレイヤーのプリファレンスが返されます。

また、インスタンス変数 player_prefs を使用することもできます。

この関数は、指定されたプレーヤーに対して同期された最後のプリファレンスを返すので、プリファレンスを設定しても、この関数が返すプリファレンスはすぐには更新されません。

必ずお読みください。プレイヤーの環境設定

構文です。

rollback_get_player_prefs([player_id]);

論証 タイプ 商品説明
player_id リアル OPTIONAL プリファレンスが返されるプレーヤーのID。

リターンです。

Any

例

var _my_prefs = rollback_get_player_prefs();

image_index = _my_prefs.character;

上記のコードでは、現在のプレイヤーに設定されているプリファレンスを取得し、インスタンス変数にプリファレンスを適用しています。
