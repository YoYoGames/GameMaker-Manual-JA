rollback_define_input

この関数は、すべてのプレーヤーのカスタム入力を定義するために使用されます。この関数はstruct を受け取り、すべての入力とそれに割り当てられた入力定数を含む必要があります。

カスタム入力を割り当てる方法については、「カスタムコントロール」を参照してください。

使用する場合、この関数はjoin/create関数の前に実行する必要があります。

構文です。

rollback_define_input(input_struct);

論証 タイプ 商品説明
input_struct 構造 A struct containing input names as variables, and their assigned inputs as values.

リターンです。

N/A

例

rollback_define_input({
    fire: mb_left,
    interact: vk_space,
    left: [ord("A"), vk_left],
    right: [ord("D"), vk_right]
});

上記のコードでは、すべてのプレーヤーに対してカスタム入力を定義しています。最後の2つの入力は、配列にリストすることで複数のコントロールを使用しています。

拡張例については、「入力の定義」を参照してください。
