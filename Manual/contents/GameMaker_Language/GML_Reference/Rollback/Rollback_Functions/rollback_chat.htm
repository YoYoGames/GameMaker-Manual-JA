rollback_chat

この機能は、他のプレイヤーにstring のメッセージを送信するために使用されます。

送信するstring 、最大128バイトのサイズを指定します。

第2引数を指定しない場合、送信者を含む全プレイヤーに送信される。

第2引数には、メッセージを受け取るプレイヤーであるプレイヤーIDを指定することができます。

使い方は？

この関数は「ローカル」での使用を想定しているため、ロールバック入力ではなく、ローカル入力の結果となる。

最下部の使用例を参照してください。Rollback Eventイベントで受信したメッセージをリッスンします。

構文です。

rollback_chat(message, [to]);

論証 タイプ 商品説明
message 文字列 The message to send, as a string.
to リアル OPTIONAL メッセージの送信先となるプレイヤーのIDを指定します。この引数を指定しないか、または-1を指定すると、すべてのプレーヤーにメッセージが送信されます。

リターンです。

N/A

例

if (keyboard_check_pressed(vk_enter))
{
    rollback_chat(keyboard_string);
    keyboard_string = "";
}

ローカルでエンターキーが押されると、その値が全プレイヤーに送信されます。 keyboard_stringの値がすべてのプレイヤーに送信されます。その後、そのstring はクリアされ、新しいメッセージを入力することができます。

これにより、（送信者を含む）すべてのクライアントがメッセージを受信すると、ロールバックイベントがトリガーされます。

ロールバックイベント

if (rollback_event_id == rollback_chat_message)
{
    // Get received information
    var _message = rollback_event_param.message;
    var _from = rollback_event_param.from;

    // Find sender's name
    var _name = rollback_get_info(_from).player_name;

    // Print to output log
    show_debug_message(_name + " says: " + _message);
}
