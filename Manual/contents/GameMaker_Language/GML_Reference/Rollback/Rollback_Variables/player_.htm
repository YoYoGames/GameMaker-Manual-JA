プレイヤー名

この変数には、現在のプレイヤーインスタンスの名前が格納されます。

によって作成されたプレイヤーインスタンスにのみ存在する変数です。 rollback_define_player()で作成されたプレイヤーインスタンスにのみ存在し、各プレイヤーごとに異なる。

ゲストプレイヤーはプレースホルダーの名前を返します。

構文です。

player_name;

リターンです。

String

例

// Draw event
draw_set_halign(fa_center);
draw_set_valign(fa_middle);

draw_text(x, y - 100, player_name);

draw_set_valign(fa_left);
draw_set_valign(fa_top);

これは、プレーヤーの名前をそのインスタンスの上に100pxで描画し、中央と中央で整列します。
