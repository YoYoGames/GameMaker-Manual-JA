プレイヤー_ユーザーID

この変数は、サーバへの直接のAPIコールで使用するために、現在のプレイヤーインスタンスのGX.games IDを格納します。

によって作成されたプレイヤーインスタンスにのみ存在する変数です。 rollback_define_player()で作成されたプレイヤーインスタンスにのみ存在し、各プレイヤーごとに異なる。

ゲストプレイヤーは、この変数に空のstring が代入されます。

構文です。

player_user_id;

リターンです。

String
