ロールバック_カレント_フレーム

マルチプレイヤーゲームが開始されてから経過したフレーム数を格納するグローバル変数です。

これは、「時間変数」で説明したように、 current_time およびその他の類似した変数の代わりに使用する必要があります。

構文です。

rollback_current_frame

リターンです。

Real

例

image_angle = dsin(rollback_current_frame) * 90;

上記のコードでは、 rollback_current_frame を使って dsin()を使って 1 と -1 の間で振動する値を計算し、それを 90 倍してインスタンスの角度に適用しています。
