rollback_game_running

このグローバル変数は、ロールバックゲームが現在実行中であるかどうかを示す。もしこれがtrueなら、ロールバックゲームが開始され、現在実行中であることを意味する。それ以外のときはfalseになる。

構文です。

rollback_game_running

リターンです。

Boolean

例

if (!rollback_game_running) return;

x += move_x;
y += move_y;

上記のコードは、プレーヤーの Step イベントで実行されます（object ）。ロールバックゲームが実行されていないときは、イベントを終了するので、ロールバックゲームが始まるまでプレーヤーを制御することはできません。
