プレイヤー_アバター_スプライト

この変数には、現在のプレイヤーインスタンスのsprite が格納されます。

によって作成されたプレイヤーインスタンスにのみ存在する変数です。 rollback_define_player()で作成されたプレイヤーインスタンスにのみ存在し、各プレイヤーごとに異なる。

ゲストプレーヤーは、プレースホルダーsprite を表示します。

構文です。

player_avatar_sprite;

リターンです。

Sprite Asset

例

// Draw event
draw_sprite_stretched(player_avatar_sprite, 0, x, y - 256, 64, 64);

このコードを実行しているプレイヤーインスタンスのアバターが、64x64の領域で、インスタンスの上方256pxに表示されます。
