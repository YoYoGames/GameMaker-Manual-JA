ロールバック確認済みフレーム

全プレーヤーの入力が同期して確認された最後のフレーム番号が格納されるグローバル変数です。

これをrollback_current_frame と比較することで、現在のフレームですべてのプレイヤーの入力が確認されているかどうかを確認することができます。

また、関数 rollback_sync_on_frame() 、現在のフレームが同期しているかどうかを確認することができます。

構文です。

rollback_confirmed_frame

リターンです。

Real

例

var _is_frame_confirmed = (rollback_current_frame == rollback_confirmed_frame);

if (_is_frame_confirmed)
{
    evaluate_game_over();
}

このコードは、現在のフレームの値が 確認されたフレームの値と等しいかどうかをチェックすることによって、現在のフレームがすべてのプレーヤーから確認された入力を持っているかどうかをチェックします。

それが真であれば、ゲームオーバーの条件が満たされているかどうかを確認する関数を実行し、ゲームを終了させることができます。

これは、予測に基づく状態でのゲーム終了を避けるために行われ、ゲームオーバー条件の確認を試みる前に、完全に同期されたフレームを待ちます。
