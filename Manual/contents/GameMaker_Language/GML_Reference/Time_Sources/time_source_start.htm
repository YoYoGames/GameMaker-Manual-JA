タイムソース開始

この関数は、与えられたタイムソースを起動し、その状態を time_source_state_activeに変更します。

指定されたタイムソースは、一度も開始されていない、一時停止、停止、または繰り返しのない期限切れの新しいタイムソースである可能性があります。

この関数は、与えられたタイムソースを「ソフトリセット」し、その有効期限と残り時間を設定された値にリセットする。

構文です。

time_source_start(id);

論証 タイプ 商品説明
id タイムソースID The Time Source を開始する。

リターンです。

N/A

例

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

この例では、300フレーム後にインスタンスが自己破壊するようにしたい。

このコードでは、まず、単純に instance_destroy().

次に、GameTime Sourceを継承したTime Sourceを作成します。その周期を300フレームに設定する。

最後にタイムソースを起動します。
