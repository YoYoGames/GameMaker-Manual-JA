タイムソースクリエイト

この関数は、新しいタイムソースを作成します。概要については、タイムソースを参照してください。

新しいタイムソースを作成しても、自動的に開始されるわけではありません。 time_source_start()を呼び出して起動させる必要があります。

新しいタイムソースは、repetitions引数が指定されない限り、一度だけ実行されます（以下を参照）。

タイムソースを破棄するには time_source_destroy()で破壊しなければならない。

論証

以下は、この関数の引数の内訳である。

保護者タイムソース

新しいタイムソースを制御する親タイムソースです。これはtime_source_global、time_source_game、またはすでに存在するカスタムのタイムソースである可能性があります。

参照。組み込みのタイムソース

期間

この期間は、次の引数で指定された単位に応じて、秒またはフレームで表現することができる。

フレームを単位とする場合、周期は整数でなければならない。整数以外の値は切り捨てられる（フロアード）。ただし、1より小さい値は1に切り上げられる。

単位

周期を表す単位で、time_source_units_secondsまたはtime_source_units_frames のいずれかを指定する。

ご覧ください。タイムソースユニット

time_bpm_to_seconds() を使って、1分あたりの拍数の値を使用することができます。

コールバックと引数

タイムソースの有効期限が切れたときに呼び出される "コールバック" を指定する必要があります。 methodを指定する必要があります。これは、タイムソースの有効期限が切れたときに呼び出されます。

オプションで、メソッドが呼ばれたときに渡されるべき引数を含む配列を指定することができます。

配列自体はメソッドに渡されず、配列の各要素が個別の引数として渡されます。

例えば、関数が引数(x, y,object) を必要とする場合、args配列は次のようになります。[30, 600, obj_player] のようになります。

繰り返し

オプションで、タイムソースを何回繰り返すかを指定することができます。1の値は1回だけ実行することを意味し、これがデフォルトの値です。

繰り返し回数の合計を指定することができ、-1で無限に繰り返すことができます。

例えば、これを3に設定し、タイムソースの期間を「4秒」にすると、タイムソースは合計12秒間アクティブになり、4秒ごとにコールバックメソッドを呼び出すことになります。

賞味期限切れタイプ

time_source_expire_nearestまたはtime_source_expire_after のいずれかを指定する。

をご覧ください。タイムソースの有効期限タイプ

構文です。

time_source_create(parent, period, units, callback, [args, repetitions, expiry_type]);

論証 タイプ 商品説明
parent タイムソースID The parent Time Source 新しいタイムソースを制御する
period リアル The period that the Time Source を実行し、与えられた単位で
units 時間源 単位 定数 The units that the given period is in
callback メソッドまたは スクリプト機能 The method or script function to call when the Time Source ぜついん
args アレイ OPTIONAL メソッドに渡す引数を含む配列。
repetitions リアル OPTIONAL タイムソースの繰り返し回数（無期限は-1
expiry_type タイムソース有効期限定数 OPTIONAL タイムソースの有効期限を直近のフレームで終了させるか、次のフレームで終了させるか。

リターンです。

Time Source ID

例1．

var _my_method = function()
{
    instance_destroy();
}

var _time_source = time_source_create(time_source_game, 300, time_source_units_frames, _my_method);

time_source_start(_time_source);

この例では、300フレーム後にインスタンスが自己破壊するようにしたい。

このコードでは、まず、単純に instance_destroy().

次に、GameTime Sourceを継承したTime Sourceを作成します。その周期を300フレームに設定する。

最後にタイムソースを起動します。

例2．

var _my_method = function()
{
    show_debug_message("A second has passed!");
}

global.time_per_second = time_source_create(time_source_game, 1, time_source_units_seconds, _my_method, [], -1, time_source_expire_after);

time_source_start(global.time_per_second);

この例では、1秒に1回期限切れになるグローバルなタイムソースを作成しています。このコードは script.

このコードでは、まず、「1秒経過しました！」というメッセージを出力ログに出力するメソッドを作成しています。

そして、GameTime Sourceを継承した新しいTime Sourceを作成します。その周期を1秒に設定する。

このメソッドはタイムソースに渡されるので、有効期限が切れるたびに呼び出すことができます。引数には空の配列が渡されます。

繰り返し回数が-1に設定されているため、タイムソースは停止することなく、永遠に繰り返される。有効期限は、有効期限が切れた後の最初のフレームでコールバックが実行されるように設定されている。

最後にタイムソースを起動します。
