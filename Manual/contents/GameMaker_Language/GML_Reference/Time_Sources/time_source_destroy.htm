time_source_destroy

この関数は、与えられたタイムソースを破壊する。

オプションで、すべての子タイムソースを削除するかどうかを指定することができます。

構文です。

time_source_destroy(id, [destroy_tree]);

論証 タイプ 商品説明
id タイムソースID The Time Source 破壊する
destroy_tree ブーリアン OPTIONAL 子供のタイムソースも破棄するかどうか、デフォルトではfalseです

リターンです。

N/A

例

// Create event
time_source = time_source_create(time_source_game, 1, time_source_units_seconds, my_method, [], -1);

// Clean Up event
time_source_destroy(time_source);

Createイベントコードは、1秒間に1回有効期限が切れて無期限に繰り返される新しいTime Sourceを作成します。

Clean Upイベントコードは、インスタンスが破壊された後に不要になるため、そのタイムソースを破棄します。
