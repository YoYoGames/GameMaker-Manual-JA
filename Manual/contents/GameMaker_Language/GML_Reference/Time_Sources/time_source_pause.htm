タイムソースポーズ

この関数は、指定されたタイムソースを一時停止します。この関数が呼ばれたとき、タイムソースはアクティブでなければなりません。

再開するには、電話 time_source_resume().

構文です。

time_source_pause(id);

論証 タイプ 商品説明
id タイムソースID The Time Source を一時停止する

リターンです。

N/A

例

if (keyboard_check_pressed(vk_space))
{
    var _state = time_source_get_state(time_source);

    if (_state == time_source_state_active)
    {
        time_source_pause(time_source);
    }
    else if (_state == time_source_state_paused)
    {
        time_source_resume(time_source);
    }
}

Spaceキーが押されると、Time Sourceの状態を取得します。

この状態がアクティブになると、Time Sourceを一時停止し、一時停止すると再開します。
