タイムソースストップ

この関数は、指定されたタイムソースを停止し、タイマーをリセットします。起動していない、または終了した状態のタイムソースは停止できません。

構文です。

time_source_stop(id);

論証 タイプ 商品説明
id タイムソースID The Time Source を停止する

リターンです。

N/A

例

if (time_source_get_state(time_source) != time_source_state_stopped)
{
    time_source_stop(time_source);
}

上記のコードは、タイムソースがまだ停止していないときに停止させます。
