タイムソースの有効期限タイプ

タイムソース有効期限定数
定数 商品説明 価値
time_source_expire_nearest タイムソースは、その有効期限に最も近いフレームで期限切れとなる 0
time_source_expire_after タイムソースの有効期限は、有効期限後最初のフレームで終了します。 1

秒を単位とするタイムソースは、必ずしもゲームフレームで期限切れになるとは限りません。ゲームフレームが処理された後、新しいフレームが処理を開始する前に期限切れとなる場合があります。

ただし、Time Sourceのコールバックはゲームフレームでしか実行できないため、Time Sourceのexpiryタイプによって、コールバックが実行されるタイミングを制御し、2つのフレームの途中で有効期限が切れた場合に対応します。

time_source_expire_nearestを使用すると、Time Sourceのコールバックは、その有効期限に最も近いフレームで実行されます（その前のフレーム、または後のフレームである可能性があります）。

time_source_expire_afterを使用すると、Time Sourceのコールバックは常にその有効期限後のフレームで実行されます。
