秒単位で表示する

この関数は、1分あたりの拍数を受け取り、各拍数の長さを秒単位で返します。

これは、Time Sourceを作成するときに、Time SourceにBPM値を使用するために使用できます。このようなTime Sourceでは、単位に 秒を使用することが重要です。

構文です。

time_bpm_to_seconds(bpm);

論証 タイプ 商品説明
bpm リアル The beats-per-minute value to convert into seconds-per-beat

リターンです。

Real

例

var _bpm = 90;
var _seconds = time_bpm_to_seconds(_bpm);

time_source = time_source_create(time_source_game, _seconds, time_source_units_seconds, function()
{
    show_debug_message("BEAT!");
}, [], -1);

このコードでは、90BPMという値を秒に変換し、それを使って無期限に動作するTime Sourceを作成します。

各ビートで、出力ログに「BEAT!
