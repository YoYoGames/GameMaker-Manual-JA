time_source_get_reps_completed

この関数は、指定されたタイムソースが最後にリセットされてから繰り返された回数を返します。これは、有効期限が切れてコールバック・メソッドを実行した回数になります。

構文です。

time_source_get_reps_completed(id);

id タイムソースID The Time Source の情報を得ることができます。
