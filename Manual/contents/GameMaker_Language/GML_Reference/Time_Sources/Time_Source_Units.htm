タイムソース単位

時間源 単位 定数
定数 商品説明 価値
time_source_units_seconds タイムソース期間に秒を使用する（フレーム非依存）。 0
time_source_units_frames タイムソース期間にフレームを使用する（フレーム依存） 1

これらの定数は time_source_create()と time_source_reconfigure()で使用され，タイムソースの周期に使用される単位を設定します。 time_source_get_units().

秒を使用する場合、タイムソースはゲームのフレームレートから独立して動作します。

フレームを使用する場合、Time Sourceはゲームのフレームレートに依存します。

を呼び出すことで、BPM（拍動毎分）を単位として使用することもできます。 time_bpm_to_seconds()を呼び出して BPM 値を秒単位に変換し、time_source_units_seconds を単位として使用することもできます。
