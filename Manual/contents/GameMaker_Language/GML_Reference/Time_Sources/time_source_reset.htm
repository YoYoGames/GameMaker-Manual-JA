time_source_reset

(1)タイムソースのリセット この関数は、与えられたタイムソースをリセットし、そのタイマと繰り返し情報をリ セットする。

この関数を呼び出した後、カウントダウンを行うには、再度タイムソースを 起動する必要があります。

構文です。

time_source_reset(id);

論証 タイプ 商品説明
id タイムソースID The Time Source リセットする

リターンです。

N/A

例

// Room Start event
time_source_reset(global.spawn_time_source);
time_source_start(global.spawn_time_source);

このコードは、新しいroom が始まるたびに与えられたTime Sourceをリセットし、Time Sourceを再スタートさせます。
