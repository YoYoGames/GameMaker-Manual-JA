タイムソース再コンフィギュレーション

この関数は、タイムソースのコアプロパティを変更するために使用され、まったく新しいものを作成する必要はありません。

既存のタイムソースを指定し、そのプロパティを設定します。 time_source_create()に設定されるプロパティを設定します。これらのプロパティの詳細については、そのページを参照してください。

指定したタイムソースはリセットされ無効化されるため、再度起動する必要があります。

構文です。

time_source_reconfigure(id, period, units, callback, [args, repetitions, expiry_type]);

論証 タイプ 商品説明
id タイムソースID The Time Source を再構成する。
period リアル The period that the Time Source を実行し、与えられた単位で
units 時間源 単位 定数 The units that the given period is in
callback 方法 The method to call when the Time Source ぜついん
args アレイ OPTIONAL メソッドに渡す引数を含む配列。
repetitions リアル OPTIONAL タイムソースの繰り返し回数（無期限は-1
expiry_type タイムソース有効期限定数 OPTIONAL タイムソースの有効期限を直近のフレームで終了させるか、次のフレームで終了させるか。

リターンです。

N/A

例

function change_spawn_delay(new_delay)
{
    time_source_reconfigure(obj_game.spawn_time_source, new_delay, time_source_units_frames, obj_game.spawn_method, [], -1);
    time_source_start(obj_game.spawn_time_source);
}

これは、ゲーム内の敵に使用されるスポーンディレイを変更する新しい関数を作成します。

obj_game.spawn_time_sourceというタイムソースを使用して敵をスポーンすると仮定すると、スポーン遅延が変更されるとそのタイムソースは更新される必要があります。

この機能はまさにその通り、新しい遅延時間でタイムソースを再設定し、再度起動します。
