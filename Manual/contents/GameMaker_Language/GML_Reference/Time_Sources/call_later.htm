コールレイター

この関数は、指定された時間の後、または一定の間隔の後にコールバック関数をトリガーします。これはタイムソースを作成することで行います（タイムソースを参照）。

コールバック関数をキャンセルするには、call_cancel を参照してください。

この関数は実際のタイムソースIDを返しますが、このIDはタイムソース関数で使用することはできません。ただし、これらはdebug_event("ResourceCounts")によって返されるタイムソースカウントに含まれ ます。

論証

この関数の引数の内訳は次のとおりです。

期間

この期間は、次の引数で指定された単位に応じて、秒またはフレームで表現することができる。

フレームを単位とする場合、周期は整数でなければならない。整数以外の値は切り捨てられる（フロアード）。ただし、1より小さい値は1に切り上げられる。

単位

周期を表す単位で、time_source_units_secondsまたはtime_source_units_frames のいずれかを指定する。

ご覧ください。タイムソースユニット

time_bpm_to_seconds() を使って、1分あたりの拍数の値を使用することができます。

コールバック

タイムソースの有効期限が切れたときに呼び出される "コールバック" を指定する必要があります。 methodを指定する必要があります。これは、タイムソースの有効期限が切れたときに呼び出されます。

ループ

タイムソースをloop にするかどうか。

構文です。

call_later(period, unit, callback, [loop=false]);

論証 タイプ 商品説明
period リアル The period that the Time Source を実行し、与えられた単位で
units 時間源 単位 定数 The units that the given period is in
callback メソッドまたは スクリプト機能 The method or script function to call after the specified time
loop ブーリアン OPTIONAL タイムソースをループさせるかどうか（デフォルトはfalse）

リターンです。

Time Source ID

例1．

var _callback = function()
{
    show_debug_message("Five seconds have passed!");
}
var _handle = call_later(5, time_source_units_seconds, _callback);

この例では、5秒経過後にデバッグメッセージを表示するコールバック関数を作成します。その後、基盤となるタイムソースは破棄されます。

例2．

var _create_enemy = function()
{
    var _x = random(room_width);
    var _y = random(room_height);
    instance_create_depth(_x, _y, depth, obj_enemy);
}

var _handle = call_later(10, time_source_units_seconds, _create_enemy, true);

この例では、room のランダムな位置に新しい敵を作成するコールバック関数を作成しています。 この関数は、call_later に渡すことで10秒ごとに実行されます。最後の引数には 真に設定されているので、この関数は10秒ごとに呼び出され続ける。

call_laterが返すハンドルでcall_cancelを使用し、タイムソースとコールバックへの呼び出しを停止します。
