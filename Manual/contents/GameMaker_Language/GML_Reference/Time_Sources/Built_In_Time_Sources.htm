内蔵のタイムソース

タイムソース定数
定数 商品説明 価値
time_source_global グローバルタイムソース 0
time_source_game ゲームタイムのソース 1

カスタムタイムソースの親として、 グローバルタイムソースまたは ゲームタイムソースを使用することができます。どちらのタイムソースもグローバルに利用可能です。

グローバルタイムソースはメインゲームの外部で動作します（loop ）。一方、ゲームタイムソースはゲームの一部として動作します（loop ）。

Global/Gameを選択しても、タイムソースがゲームのフレームレートの影響を受けるかどうかは、使用するユニットに依存します。

ご注文

グローバルタイムソースを継承するタイムソースは、ゲームタイムソースより 先に処理されます。

州

ゲームタイムソースは ステートを持ち、以下のことが可能です。 paused.GlobalTimeSourceには状態がなく、一時停止することはできません。

ゲームの状態に関係なく動作するタイマーにはGlobalTime Sourceを使用し、メインゲームに関連するタイマーにはGameTime Sourceを使用しますloop 。ゲームタイムソースを一時停止することで、ゲームに関連するすべてのタイマーを一時停止することができます。
