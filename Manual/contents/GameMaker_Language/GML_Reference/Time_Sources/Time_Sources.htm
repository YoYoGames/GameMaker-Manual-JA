タイムソース

タイムソース」とは、あなたが作成したカスタムタイマーのことです。指定された時間だけ動作し、終了時には期限切れとなります。

タイムソース（TS）には、有効期限切れ時にカスタムメソッド（「コールバック」）を呼び出す機能があります。

また、一定の回数だけ、あるいは永遠に繰り返すことができます。タイムソースの各繰り返しは「レップ」と呼ばれる。

タイムソースをどう使うか？

カスタムメソッドを呼び出すタイムソースを作成し、起動します。

  • 呼び出し time_source_create()/ タイムソースを作成するには、タイムソースの作成
    • グローバルタイムソース、ゲームタイムソース、またはカスタムタイムソースのいずれからinherits を指定します。
    • 実行する期間と、使用する単位（秒またはフレーム）を指定する。
    • カスタムメソッドまたはscript 関数を指定し、そのメソッドに渡す引数配列も指定します。
    • オプションで、繰り返し実行する回数を指定します。
      • デフォルトでは、1回だけ実行されます。
  • 呼び出し time_source_start()/Start Time Sourceは、そのタイムソースをすぐに開始するか、または後で開始する必要があるときに呼び出します。

こうすることで、タイムソースが実行され、指定した期間後に期限切れとなり、指定したメソッドが呼び出されます。

シンプルタイマー

タイムソースの設定には多くのコード行は必要ありませんが、一つの関数を使って素早くタイマーを設定し、管理することは便利です。

次の2つの関数は、タイムソースの作成と停止をより簡単に行うための略記法です。

  • コールレイター
  • call_cancel

注：タイムソース関数は、これらの関数で使用されるタイムソースIDでは使用できません。詳細はcall_laterを参照してください。
