gxc_stop_movie_recording

gxc_start_movie_recording() で開始した GXC 録画を停止する。また、引数で指定した URL に録画を投入するが、指定したstring の種類により動作が異なる。

  • ファイル名を指定すると、その名前でダウンロードフォルダに録画が保存されます
  • URLを指定すると、そのURLに録画がアップロードされます

URL方式は、GXCにアップロードする必要がある場合に使用し、ファイルは録画が成功したかどうかを確認するためにのみローカルに保存する必要があります。

構文です。

gxc_stop_movie_recording(filename);

論証 タイプ 商品説明
filename 文字列 The file name or URL where the recording is to be saved or uploaded

リターンです。

N/A

例

gxc_submit_challenge_score(_score, function(_status, _result) {
    var _uploadUrl = _result.data.gameplayVideoUploadUrl;
    gxc_stop_movie_recording(_uploadUrl);
});

上記のコードでは、GXC API Library を使用して現在アクティブな課題にスコアをアップロードし、そのコールバック メソッドでレスポンスを受信すると、動画のアップロード URL を取得して、そこに録画をアップロードしています。
