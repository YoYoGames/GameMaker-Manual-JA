gxc_start_movie_recording

この関数は、GXCで挑戦の記録を開始するために使用します。録画の幅、高さ、フレームレートを指定し、音声を録画するかどうかをブール値で指定します。

構文です。

gxc_start_movie_recording(width, height, fps, record_audio);

論証 タイプ 商品説明
width リアル The width of the recording
height リアル The height of the recording
fps リアル The frame rate (frames-per-second) of the recording
record_audio ブーリアン Whether audio should be recorded (true) か、そうでないか (false)

リターンです。

N/A

例

gxc_start_movie_recording(320, 240, 4, false);

これにより、解像度320x240、4フレーム/秒の動画撮影が開始され、音声は記録されません。
