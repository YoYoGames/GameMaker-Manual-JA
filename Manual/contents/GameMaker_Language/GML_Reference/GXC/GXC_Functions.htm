オペラGX、GXC

ここでは、Opera GXターゲットプラットフォームで提供される機能の概要を説明します。このページに記載されている機能を使用すると、画面を録画し、課題のスコアとともにGXCにアップロードできますが、これを実現するにはGXC APIライブラリを使用する必要があることに注意してください。GXC APIライブラリの実装と課題の処理方法については、YoYo Games HelpdeskのOpera GXセクションを参照してください。

録画した動画を課題付きでアップロードするために、以下の機能が用意されています。

  • gxc_start_movie_recording
  • gxc_stop_movie_recording
  • gxc_pause_movie_recording
  • gxc_resume_movie_recording
