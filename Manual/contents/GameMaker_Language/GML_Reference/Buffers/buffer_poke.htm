バッファローポーク

を使えば buffer_write()関数を使うと、与えられたbuffer の現在のシーク位置にデータを書き込むことができ、データを書き込んだり読み込んだりするたびに、この位置がバイト単位で進んでいきます。しかし、現在のシーク位置を変更せずに、あるデータを変更する必要がある場合があります。この関数には、buffer インデックスと、そのbuffer 内のbuffer 開始位置からのオフセット位置（バイト単位）、およびデータ型と書き込む値を指定するだけです。

構文です。

buffer_poke(buffer, offset, type, value);

論証 タイプ 商品説明
buffer バッファID The index of the buffer to use.
offset リアル The offset position (in bytes) within the buffer to write the given data to.
type バッファデータ型定数 The type of data that is to be written to the buffer (see the list of constants here).
value バリアブル The data to add to the buffer, in accordance with the type specified.

リターンです。

N/A

例

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

上記のコードは、画像ブレンドに使用した色の青成分の値を、変数 "buff "でインデックスされたbuffer 、buffer の3番目の位置、符号なし8ビット値として追加します。
