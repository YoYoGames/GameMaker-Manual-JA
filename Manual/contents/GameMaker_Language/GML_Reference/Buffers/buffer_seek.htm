バッファシーク

この関数はbuffer の中を移動し、開始、終了、またはデータの読み書きの際に最後に使用した位置からの相対的な位置を見つけるために使用します。たとえば、ベースが相対でオフセットが4であれば、buffer の位置は現在の位置から4バイト移動します。以下の点に注意してください。

  • buffer を通してシークバックするオフセットには、正の値だけでなく、負の値を使用することもできます。
  • buffer がラップタイプで、buffer の終端を過ぎてオフセットした場合、シーク位置もラップされます。
  • buffer が「wrap」タイプでない場合、オフセットによってシークがbuffer の制限外になる場合でも、シークはbuffer の先頭または末尾にクランプされます。

以下の定数は、シークトゥの「ベース」引数として受け入れられる。

バッファシーク定数
定数 商品説明
buffer_seek_start バッファの開始位置
buffer_seek_relative 現在の読み出し/書き込み位置からの相対位置
buffer_seek_end バッファの終端

構文です。

buffer_seek(buffer, base, offset);

論証 タイプ 商品説明
buffer バッファID The index of the buffer to use.
base リアル The base position to seek.
offset リアル The data offset value.

リターンです。

N/A

例

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

上記のコードは、変数 "buff "に格納されたidを持つbuffer の開始点を見つけ、そこに一連の符号付き16ビット整数値を書き込んでいます。
