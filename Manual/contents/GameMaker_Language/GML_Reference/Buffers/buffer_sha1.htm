バッファ_sha1

暗号技術において、SHA-1は米国国家安全保障局によって設計された暗号ハッシュ関数であり、広く使用されているいくつかのアプリケーションやプロトコルで採用されている、人気の高い Gitでは、ファイルの変更をチェックするために使用されています。この関数は、buffer から入力データを受け取り、160ビットのメッセージダイジェストをASCIIフォーマットで返します。このようにして、安全な鍵を生成することができます。 の情報を外部のserver （例）に送信する（または受信する）ことができます。この関数を使用してbuffers に適用する場合、使用するbuffer のbuffer id を指定し、次に開始位置を示すオフセット値（バイト単位）を指定し、次に ハッシュ化される領域のサイズ（これもバイト単位）。

注意：SHA-1は完全に安全ではなく、破られる可能性があります。詳しくはこのページをご覧ください。

構文です。

buffer_sha1(buffer, offset, size);

論証タイプ 商品説明
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

リターンです。

例

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

上記のコードは、変数 "buff" でインデックスされたbuffer に格納されている全データに対して sha1 ハッシュを作成し、返されたハッシュを変数 "check_string" に格納するものです。
