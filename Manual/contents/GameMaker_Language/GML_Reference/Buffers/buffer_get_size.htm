buffer_get_size

この関数を使うと、与えられたbuffer のサイズをバイト単位で得ることができます。

構文です。

buffer_get_size(buffer);

論証 タイプ 商品説明
buffer バッファID The index of the buffer to get the size of.

リターンです。

Real

例

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

上記のコードは、新しいbuffer を作成し、そのインデックスをローカル変数 "_temp" に格納します。この新しいbuffer のサイズは、変数 "player_data" にインデックスされたbuffer のサイズと同じになります。
