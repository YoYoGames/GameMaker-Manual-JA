buffer_delete

この機能を使うと、以前にこの機能を使って作成したbuffer を削除することができます。 buffer_create()をメモリから削除し、その作成に使用したリソースを解放し、現在含まれているかもしれないデータを削除します。

注：メモリリークを防ぐため、動的に作成されたリソースは、不要になった時点で必ずメモリから削除することが重要です。

重要!buffer を作成するとき、それを識別するために使用されるインデックス値は 0 から始まる整数値です。これらのインデックスはGameMaker で再利用されるので、破壊されたbuffer のインデックス値は、その後新しく作成されたものによって使用される可能性があります。 また、buffer を破棄した後、バッファインデックスを保持する変数に -1 を設定することをお勧めします。

構文です。

buffer_delete(buffer)

論証タイプ 商品説明
buffer The index of the buffer to delete.

リターンです。

例

buffer_delete(player_buffer);
player_buffer = -1;

上記のコードは、変数 "player_buffer "に格納されているidで以前作成されたbuffer を削除し、その変数に-1をセットします。
