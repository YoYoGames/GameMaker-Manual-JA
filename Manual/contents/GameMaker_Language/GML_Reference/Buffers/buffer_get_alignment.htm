バッファゲットアライメント

この関数を使用すると、指定されたbuffer ID のバイトアライメントを取得することができます。

構文です。

buffer_get_alignment(buffer);

論証タイプ 商品説明
buffer The index of the buffer to check.

リターンです。

例

alignment = buffer_get_alignment(buff);

上記のコードでは、変数 "buff "にインデックスされた値から、buffer のアライメントを取得し、変数に格納することになります。
