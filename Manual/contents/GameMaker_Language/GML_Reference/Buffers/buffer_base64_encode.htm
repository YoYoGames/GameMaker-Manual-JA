バッファベース64エンコード

この関数は、与えられたbuffer のデータを、base64 形式でエンコードされたstring に変換します。これは一般によく使われる符号化方式で，インターネット上でテキストとして保存または転送する必要があるあらゆるメディアでよく使われ，その結果 の出力は、人間の目には読めません。これを使うには、すでに作成されているbuffer 、オフセット値（エンコードを開始したいbuffer 内のポイント）、およびエンコードするbuffer メモリのサイズ（バイト単位）を指定する必要があります。

構文です。

buffer_base64_encode(buffer, offset, size);

論証タイプ 商品説明
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

リターンです。

例

var b_str = buffer_base64_encode(buff, 0, buffer_get_size(buff))

上記のコードでは、変数 "buff" をインデックスとしてbuffer に格納されているデータをフルエンコードし、返されたstring をローカル変数 "b_str" に格納します。
