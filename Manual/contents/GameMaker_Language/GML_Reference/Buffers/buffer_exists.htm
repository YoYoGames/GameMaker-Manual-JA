バッファの存在

この関数は、変数が有効なbuffer ID 値を保持しているかどうかを確認するために使用することができます。もし有効であれば、この関数は true を返し、そうでなければ、 false を返します。

構文です。

buffer_exists(buffer)

論証タイプ 商品説明
buffer The index of the buffer to check.

リターンです。

例

if buffer_exists(buff)
{
    buffer_delete(buff);
}

上記のコードでは、変数 "buff" がbuffer の ID を保持しているかどうかを確認し、保持している場合はbuffer を削除しています。
