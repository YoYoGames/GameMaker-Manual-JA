バッファリード

この関数は、以前に作成されたbuffer からデータを読み取るために使用することができます。戻り値は，読み込むデータの種類に依存し，それ自体は，以下の定数で定義される。

定数 商品説明
buffer_u8 符号なし8ビット整数。0から255までの正の値です。
buffer_s8 符号付き8ビット整数。128から127までの正または負の値（0は正として分類される）。
buffer_u16 符号なし、16ビットの整数。0〜65,535の正の値です。
buffer_s16 符号付き16ビット整数。32,768～32,767の正または負の値（0は正として分類される）です。
buffer_u32 符号なし32ビット整数。0から4,294,967,295までの正の値である。
buffer_s32 符号付き32ビット整数。2,147,483,648 から 2,147,483,647 までの正または負の値（0は正として分類される）です。
buffer_u64 64ビット符号なし整数。
buffer_f16 16bit float。65504の範囲内の正または負の値です。(現在はサポートされていません!)
buffer_f32 32bit float。正または負の値で，±16777216の範囲である。
buffer_f64 64bitのフロート。
buffer_bool ブーリアン値。1 または 0 (true または false) のどちらかのみです。
buffer_string 大きさを問わないstring 。
buffer_text string 任意の大きさで、最後のヌル終端文字を含まないもの。

この関数が成功すれば、指定された型の値を返しますが、失敗すればランナーエラーを引き起こします。

注読み込むデータに対して誤ったデータ型を使用すると、誤った値が返されます。

構文です。

buffer_read(buffer, type)

論証 タイプ 商品説明
buffer バッファID The index of the buffer to read from.
type バッファデータ型定数 The type of data that is to be read from the buffer (see the list of constants above).

リターンです。

Real, Boolean or String

例

var cmd = buffer_read(buff, buffer_s16);

上記のコードは、変数「buff」に格納されたidを持つbuffer から、符号付き16ビット値をローカル変数「cmd」に読み込んでいます。
