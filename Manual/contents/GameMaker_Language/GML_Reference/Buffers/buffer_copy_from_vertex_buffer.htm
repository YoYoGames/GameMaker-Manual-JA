バッファコピーバーテックスバッファ

この関数は、あるvertex vertex に保存されているbuffer データの一部（または全部）を、以前に作成した通常のbuffer にコピーするために使用することができます。この関数でvertex buffer から通常のbuffer にコピーする場合，両方のbuffers が以前に作成されている必要があります (この関数では vertex_create_buffer()と buffer_create()関数など)。vertex にコピーしたいbuffer データの範囲を指定します。開始点vertex は 0 から頂点の数 -1 までの任意の値で、そこからコピーする頂点の数を指定することができ ます。このとき、関数 vertex_get_number()をvertex buffer 、格納されている頂点の総数を取得する。最後に、buffer のデータをコピーするvertex のインデックスと、vertex のデータをコピーする先のbuffer の位置を定義するデータオフセットを指定します。

構文です。

buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset);

論証 タイプ 商品説明
vertex_buffer バーテックスBuffer ID The index of the vertex buffer to copy from.
start_vertex リアル The starting vertex.
num_vertices リアル The total number of vertices to use.
dest_buffer バッファID The index of the buffer to copy to.
dest_offset リアル The offset position to copy the data to (in bytes).

リターンです。

N/A

例

var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0);

上記のコードは、変数 "model_buffer "でインデックスされたvertex vertex に格納されているbuffer のデータをコピーし、変数 "player_buffer "でインデックスされたbuffer に貼り付けます。
