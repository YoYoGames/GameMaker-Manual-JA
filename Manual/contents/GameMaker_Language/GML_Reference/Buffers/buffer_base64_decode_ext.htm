バッファベース64デコードテキスト

この関数を使うと、base64 でエンコードされたstring (この関数を使って作成された) をデコードすることができます。 buffer_base64_encode()関数と異なり，buffer に変換することができます． buffer_base64_decode(), は、buffer を作成するのではなく、すでにbuffer を作成しておく必要があります ( buffer_create()) のIDを取得し、この関数で使用します。オフセット "は 与えられたbuffer をデコードするためのstring 内の位置(バイト単位)。

構文です。

buffer_base64_decode_ext(buffer, string, offset);

論証タイプ 商品説明
buffer The index of the buffer to decode the string into.
string The base64 encoded string to decode.
offset The data offset value.

リターンです。

例

buff = buffer_create(16384, buffer_grow, 2);
ini_open("Save.ini");
var str = ini_read_string("Save", "Slot1", "");
buffer_base64_decode_ext(buff, str, 0);
ini_close();

上記のコードでは、buffer を作成し、そのユニーク ID を変数 "buff" に格納します。次に、ini ファイルを開き、そこからstring をローカル変数 "str" に読み取ります。このstring は、新しく作成されたbuffer にデコードされた後、ローカル変数 "str "に格納されます。 再びiniファイルを閉じる。
