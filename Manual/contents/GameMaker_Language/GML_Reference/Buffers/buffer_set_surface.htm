バッファリング・セット・サーフェス

この関数を使うと、buffer から指定されたサーフェイスに情報を書き込むことができます。読み込まれるbuffer と書き込まれるサーフェイスの両方が以前に作成されている必要があり、読み込みを開始するbuffer へのオフセットを指定することができます。読み込みは常にbuffer の先頭とオフセット値から始まり、現在のシーク位置とオフセット値からは始まらないことに注意してください。

注）GameMakerのアップデート2.3.1以降、この関数が変更されました。以前のバージョンでは、定数 buffer_surface_copy を使用していましたが、現在は非推奨です。

構文です。

buffer_set_surface(buffer, surface, offset);

論証タイプ 商品説明
buffer The index of the buffer to use.
surface The index of the surface to use.
offset The data offset value.

リターンです。

例

buffer_set_surface(buff, application_surface, 0);

上記のコードは、変数 "buff "にインデックスされたbuffer に格納されているすべてのデータを、オフセットなしでアプリケーションサーフェスにコピーします。
