バッファ_md5

暗号技術において、MD5（Message-Digest algorithm 5）は、128ビットのハッシュ値を持つ暗号ハッシュ関数で、様々なセキュリティアプリケーションに広く採用されています。 また、データの完全性を確認するためにもよく使われている。bufferこの関数は、与えられたbuffer (またはその一部) の入力データを受け取り、そのデータに固有の 32 文字の 16 進法 MD5 ハッシュを返します。このようにして、あなたは を保存し、外部のserver に送信する (または受信する) 情報の完全性をチェックするために使用することができます (例)。この関数を使用してbuffers に適用する場合、buffer のbuffer id を指定する必要があります。 を使用し、次にどこから始めるかのオフセット値（バイト単位）、そしてハッシュ化する領域のサイズ（これもバイト単位）を指定します。

注：MD5は完全に安全ではなく、破られる可能性があります。詳しくはこのページをご覧ください。

構文です。

buffer_md5(buffer, offset, size);

論証タイプ 商品説明
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

リターンです。

例

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

上記のコードは、変数 "buff" でインデックスされたbuffer に格納された完全なデータに対して md5 ハッシュを作成し、返されたハッシュを変数 "check_string" に格納するものである。
