バッファサイズ

この関数は、与えられたBuffer Data Type定数（ここにリストされている）のサイズ（バイト単位）を返します。

構文です。

buffer_sizeof(type);

論証 タイプ 商品説明
type バッファデータ型定数 The type of data that is to be checked (see the list of constants here).

リターンです。

Real

例

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

上記のコードでは、まず符号なし8ビットデータ型に12を掛けて作成するbuffer のサイズを計算し（buffer に12個のデータを格納するので）、この値を使って固定のbuffer を正しいサイズに設定するようにしています。
