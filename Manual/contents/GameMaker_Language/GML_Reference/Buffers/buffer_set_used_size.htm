バッファの設定と使用サイズ

この関数は、主にextensions 内で使用するもので、与えられたbuffer の「使用済み」サイズ（書き込まれたバイト数）を設定することができます。

buffer からextension にデータを書き込むと、GameMaker はbuffer がextension のコードでどれだけ埋まったかわからず、そのデータを読み取ることができません。この関数はextension から呼び出され、何バイトのデータを書き込んだかをエンジンに伝えることができます。 がbuffer に書き込まれ、エンジンがそのデータを読むことができるようになりました。

構文です。

buffer_set_used_size(buffer, size);

論証タイプ 商品説明
buffer The index of the buffer to use.
size The number of bytes to set as the "used" size.

リターンです。

例

buffer_write(_bufferAddress, buffer_u8, 1);
buffer_write(_bufferAddress, buffer_u8, 2);
buffer_write(_bufferAddress, buffer_u16, 400);

buffer_set_used_size(_bufferAddress, 4);

上記の関数は、extension で呼び出され、あるデータをGameMaker buffer に（そのメモリアドレスを通して）書き込み、そのデータをエンジンが読めるように、書き込んだバイト数を設定することになります。
