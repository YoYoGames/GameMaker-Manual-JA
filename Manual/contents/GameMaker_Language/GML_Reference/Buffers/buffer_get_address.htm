バッファゲットアドレス

この関数で、アラインメントされた生のbuffer アドレスへのポインタを得ることができます。これは主にextensions で使用するためのもので、この値を渡すことでbuffer のデータにアクセスすることができます。ただし、いくつかの点で は、拡張機能からアクセスする場合、buffer で行うことはできません。

  • メモリーを解放することはできません。
  • メモリのリサイズはできません。
  • buffer のアドレスとbuffer のサイズを越えて書き込んではいけません。 buffer_get_size()関数を使用します) でないと、out of bounds エラーが発生します。

構文です。

buffer_get_address(buffer);

論証タイプ 商品説明
buffer The index of the buffer to use.

リターンです。

例

var b_address = buffer_get_address(buff_model);
var b_end_address = b_address + buffer_get_size(buff_model);

上記のコードでは、変数 buff_model でインデックスされたbuffer のメモリアドレスを取得し、 buffer_get_size() 関数を使用してbuffer の終端のメモリアドレスを取得し、両方の値をローカル変数に格納します。 を使用します。
