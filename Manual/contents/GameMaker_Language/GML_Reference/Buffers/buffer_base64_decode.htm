バッファベース64デコード

この関数を使うと、base64 でエンコードされたstring (この関数で作成された) をデコードすることができます。 buffer_base64_encode()bufferこの関数は，（1バイトにアラインされた "grow "バッファとして）buffer を生成し，buffer に対するユニークなインデックスを返します．

注：このように動的に作成されたリソースは、メモリリークを防ぐために、不要になったらメモリから削除することが重要です。作成したbuffer が終了したら、再度、以下を使用してリソースを解放してください。 buffer_delete().

構文です。

buffer_base64_decode(string);

論証タイプ 商品説明
string The base64 encoded string to decode

リターンです。

Buffer ID

例

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

上記のコードでは、iniファイルを開き、そこからデコード関数にstring を読み込みます。この関数はbuffer のインデックスを返します。このインデックスは変数 "buff" に格納され、以前にエンコードされて保存されたデータを含んでいます。その後、iniファイルは閉じられる。
