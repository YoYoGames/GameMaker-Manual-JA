buffer_copy

この機能は、あるbuffer に保存されているデータの一部（または全部）を別の にコピーするために使用することができます。2つのbuffers を使い、一方から他方へコピーする場合、両方のbuffers は、以前に buffer_create() 関数（例）を使用し、コピー元のbuffer からコピーするデータの開始点をbuffer の開始点に対して指定するデータオフセット（バイト単位）、およびデータをコピーする位置を定義する別のデータオフセットを指定することができます。 を宛先buffer に挿入します。

注：同じbuffer にコピーすることはできません。

構文です。

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);

論証タイプ 商品説明
src_buffer The index of the buffer to copy from.
src_offset The data offset to start copying from (in bytes).
size The size of the data to copy (in bytes).
dest_buffer The index of the buffer to copy to.
dest_offset The offset position to copy the data to (in bytes).

リターンです。

例

buffer_copy(buff1, 0, 2048, buff2, 2048);

上記のコードは、変数 "buff1 "にインデックスされたbuffer に格納されているデータをコピーし、変数 "buff2 "にインデックスされたbuffer に2048バイトだけオフセットしてペーストします。
