バッファゲットタイプ

この関数は、buffer の ID 値から参照されているbuffer の種類を取得するものです。戻り値は以下の定数のいずれかである。

定数 商品説明
buffer_fixed サイズが固定されているbuffer 。
buffer_grow データが追加されると動的に「成長」するbuffer
buffer_wrap buffer ここでデータは「ラップ」されます。追加されるデータがbuffer サイズの限界に達すると、上書きはbuffer の先頭に戻され、その時点からさらに書き込みが続けられる。
buffer_fast 特殊な "stripped"buffer で、読み書きが非常に高速になります。 buffer_u8 データ型でのみ使用可能で、1 バイトアライメントでなければならない。
buffer_vbuffer Avertex バッファ

構文です。

buffer_get_type(buffer);

論証タイプ 商品説明
buffer The index of the buffer to check.

リターンです。

Buffer Type Constant

例

type = buffer_get_type(buff);

上記のコードは、変数「buff」にインデックスされている値から、buffer の型を取得し、変数に格納します。
