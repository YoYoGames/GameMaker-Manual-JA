バッファ圧縮

この関数を使うと、buffer の一部（または全部）をzlib 圧縮で圧縮することができます。圧縮するbuffer の ID を指定します (これは buffer_create()) 、使用するbuffer 内のオフセット (バイト単位)、圧縮するbuffer データのサイズ (これもバイト単位) を指定します。この関数は，圧縮されたbuffer の新しいbuffer ID を返すか，何らかの理由で失敗した場合は 0 未満の値を返します．この関数は、元のbuffer を変更することはありません。

構文です。

buffer_compress(buffer, offset, size);

論証タイプ 商品説明
buffer The index of the buffer to compress.
offset The offset within the buffer to compress (in bytes).
size The size of the buffer area to compress (in bytes).

リターンです。

Buffer ID

例

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

上記のコードでは、buffer を作成し、string からのデータを入力します。 このbuffer は圧縮されて保存され、ソースと圧縮されたbuffers の両方が削除されます。
