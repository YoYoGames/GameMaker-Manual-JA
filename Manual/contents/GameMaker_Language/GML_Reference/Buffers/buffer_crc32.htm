buffer_crc32

この関数はbuffer から入力データを受け取り、crc32 チェックサムハッシュを返します。使用するbuffer のbuffer ID を指定し、どこから始めるかのオフセット値 (バイト単位) を指定し、ハッシュ化する領域のサイズ (これもバイト単位) を指定すると この関数は、その領域に対する32ビットの整数値を返す。

構文です。

buffer_crc32(buffer, offset, size);

論証タイプ 商品説明
buffer The index of the buffer to use.
offset The data offset value.
size The size of the buffer.

リターンです。

例

check_val = buffer_crc32(buff, 0, buffer_get_size(buff));

上記のコードは、変数 "buff" でインデックスされたbuffer に格納されているフルデータに対して crc32 チェックサムハッシュ値を作成し、返された整数値のハッシュ値を変数 "check_val" に格納するものである。
