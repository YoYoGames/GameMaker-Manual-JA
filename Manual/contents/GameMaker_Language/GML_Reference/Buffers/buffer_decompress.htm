バッファ解凍

この関数を使用すると、zlib圧縮を使用して以前に圧縮されたbuffer を解凍することができます。解凍するbuffer ID を与えると、この関数は解凍されたデータを含む新しいbuffer ID を返します。解凍に失敗した場合 (たとえば、圧縮されていないbuffer を指定した場合)、この関数は 0 未満の値を返します。

構文です。

buffer_decompress(buffer);

論証タイプ 商品説明
buffer The index of the buffer to decompress.

リターンです。

Buffer ID

例

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

上記のコードは、まず保存されたbuffer を読み込み、次にそれを解凍し、最後に解凍されたstring からbuffer のデータをグローバル変数に読み込むというものです。
