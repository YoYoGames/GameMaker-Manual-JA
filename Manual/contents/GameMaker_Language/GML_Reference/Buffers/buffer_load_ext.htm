バッファロードエクスト

この関数は、以前、buffer を使って保存したデータを読み込みます。 buffer_save()関数を既存のbuffer に追加します。ロードする前に作成したbuffer の ID、次にロードする保存したbuffer ファイル、最後にデータをロードするbuffer の先頭からのオフセット (バイト数) を指定します。

プラットフォーム別の注意事項については、 buffer_load() のページをお読みください。

構文です。

buffer_load_ext(buffer, filename, offset);

論証タイプ 商品説明
buffer The index of the buffer to load into.
filename The name of the file to load from.
offset The offset within the buffer to load to (in bytes).

リターンです。

例

var pos = buffer_seek(player_buffer, buffer_seek_end, 0);
buffer_load(player_buffer, "Data_Save.sav", pos);

上記のコードは、まず変数「player_buffer」にインデックスされたbuffer の終了位置を取得し、その位置に与えられたデータをロードします（この例は、「grow」または「wrap」buffer タイプでのみ動作することに注意してください）。
