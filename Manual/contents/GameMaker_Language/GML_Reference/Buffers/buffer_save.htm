バッファセーブ

この機能を使うと、buffer の内容をファイルに保存して、メモリに読み込むことができます。 buffer_load()関数を使用します。

注：HTML5では、この機能を使用すると、buffer の内容が base64 エンコードされたstrings として保存されます。

構文です。

buffer_save(buffer, filename);

論証タイプ 商品説明
buffer The index of the buffer to save.
filename The name of the file to save as.

リターンです。

例

buffer_save(buff, "Player_Save.sav");

変数 buff に格納された id を持つbuffer の現在の内容をファイルに保存する。
