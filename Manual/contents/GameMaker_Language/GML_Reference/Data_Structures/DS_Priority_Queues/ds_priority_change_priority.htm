ds_priority_change_priority（優先度変更）。

この関数は、与えられた値を受け取り、参照される優先順位内でその優先順位を変更します。queue 。指定された値は、すでに優先順位queue に存在する必要があります。

構文です。

ds_priority_change_priority(id, val, priority);

論証 タイプ 商品説明
id DSプライオリティID The id of the priority queue to change.
val バリアブル The value to change the priority of.
priority リアル The new priority of the value.

リターンです。

N/A

例

if (global.Game_Time < 1000)
{
    ds_priority_change(ai_priority, AI_Search, 1);
}

上記のコードでは、グローバル変数をチェックし、ある値以下であれば、優先度script に保持されているqueue 関数のインデックスの優先度を変更します。
