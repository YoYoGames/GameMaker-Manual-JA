ds_priority_add

この機能により、優先順位string に値（実数またはqueue ）を追加し、同時に優先順位の値を割り当てることができます。

構文です。

ds_priority_add(id, val, priority);

論証 タイプ 商品説明
id DSプライオリティID The id of the priority queue to add to.
val バリアブル The value to add to the priority queue.
priority リアル The priority of the value to add.

リターンです。

N/A

例

ds_priority_add(ai_priority, AI_Search, 5);

上記のコードでは、変数 "ai_priority "にインデックスされた優先度script にqueue 関数を追加し、優先度5を割り当てています。
