ds_priority_create

この関数は、新しい優先度queue のデータ構造を作成し、インデックス値を返す。この値は変数に格納され、優先順位に関連する以降のすべての関数呼び出しで使用される必要がありますqueue.

重要!データ構造を作成するとき、それを識別するためのインデックス値は0から始まる整数値です。 これは、異なるタイプのデータ構造でも同じインデックス値を持つことができることを意味します。 ds_exists()関数を使用してからアクセスします。また、インデックスは再利用されるため、破壊されたデータ構造のインデックス値が、その後に新しく作成されたデータ構造で使用される可能性があることに注意してください。

構文です。

ds_priority_create();

リターンです。

DS Priority ID

例

p_queue = ds_priority_create();

これにより、新しい優先順位queue が作成され、そのインデックス ID がインスタンス変数 "p_queue" に代入されます。
