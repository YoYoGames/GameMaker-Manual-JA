ds_priority_find_min

この関数では、優先順位が最も低いqueue に格納されている値を見つけることができます。同じ優先順位を持つ値が複数ある場合は、そのうちのどれかを任意の順序で返すことができます。ただし ds_priority_delete_min()この関数は、queue から値を削除することはありません。

構文です。

ds_priority_find_min(id);

論証タイプ 商品説明
id The id of the priority queue to use.

リターンです。

Real or String

例

if ai_move
{
    script_execute(ds_priority_find_min(ai_priority));
}

上記のコードは、インスタンス変数をチェックし、 true を返した場合、優先順位が最も低い値であるscript でインデックスされたqueue 関数を実行します。
