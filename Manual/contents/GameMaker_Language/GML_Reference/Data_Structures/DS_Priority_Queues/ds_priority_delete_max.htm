ds_priority_delete_max

この関数は、queue 、最も高い優先度を持つ値を返し、その値（と優先度）をデータ構造から削除する。複数の値が同じ優先度を持つ場合、そのうちのどれかを任意の順序で返すことができるが、同じ優先度を持つ他のすべての値はまだqueue に存在することになる。

構文です。

ds_priority_delete_max(id);

論証タイプ 商品説明
id The id of the priority queue to use.

リターンです。

(Data type stored in the priority)

例

if ai_move
{
    script_execute(ds_priority_delete_max(ai_priority));
}

上記のコードは、インスタンス変数をチェックし、 true を返した場合、最も高い優先度の値を持つscript でインデックスされたqueue を実行し、そのscript をqueue から削除しています。
