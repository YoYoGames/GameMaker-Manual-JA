ds_priority_clear

この関数を使用すると、指定された優先度queue のデータ構造からすべてのデータをクリアすることができます。これは，データ構造を破壊するものではありません（破壊するには ds_priority_destroy()) からすべてのデータを消去し、空の優先順位queue を返すだけです。

構文です。

ds_priority_clear(id);

論証タイプ 商品説明
id The id of the data structure to clear.

リターンです。

例

if (count = 15) && (!ds_priority_empty(command_queue))
{
    ds_priority_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

上記のコードは、ある変数が特定の値に達したかどうかをチェックし、もし達していれば、変数「command_queue」にインデックスされているDS優先度queue 、アラームを設定し、変数を0にリセットしています。
