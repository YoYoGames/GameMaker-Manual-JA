DS優先キュー

DS優先度queue は、DS優先度に似た特殊なデータ構造です。 DS queueただ、queue の値は、その値に割り当てられた優先順位（常に実数）により並べられる。この構造は、リーダーボードや情報リストの作成に非常に有効で、各項目の重要度を重み付けすることができる。

注：すべてのダイナミックリソースと同様に、データ構造はメモリを消費します。したがって、メモリリークによるゲームのスローダウンやクラッシュを防ぐために、不要になったら必ず破棄する必要があります。

注：データ構造が存在するかどうかを確認する必要がある場合は ds_exists()関数を使用します。

DS優先キューを扱う機能として、以下のものが存在する。

  • ds_priority_create
  • ds_priority_destroy
  • ds_priority_clear
  • ds_priority_empty
  • ds_priority_size
  • ds_priority_add
  • ds_priority_change_priority
  • ds_priority_delete_max
  • ds_priority_delete_min
  • ds_priority_delete_value
  • ds_priority_find_max
  • ds_priority_find_min
  • ds_priority_find_priority
  • ds_priority_copy
  • ds_priority_read
  • ds_priority_write
