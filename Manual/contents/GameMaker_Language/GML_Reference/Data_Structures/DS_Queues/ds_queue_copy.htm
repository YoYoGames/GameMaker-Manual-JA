ds_queue_copy

この機能は、あるqueue の内容を別の場所にコピーするために使用することができます。ただし、元のqueue から内容を削除したり、元のqueue を破壊したりするものではありません。この機能を使用する場合、コピー先のqueue は以前に作成されている必要があり、コピー前に何らかのアイテムが含まれていた場合、それらは最初にクリアされます（つまり、この情報は失われます）。

構文です。

ds_queue_copy(id, source);

論証 タイプ 商品説明
id DSQueue ID The id of the new queue.
source DSQueue ID The original queue to copy from.

リターンです。

N/A

例

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
    queue = ds_queue_create();
    ds_queue_copy(queue, other.queue);
}

上記の関数は新しいインスタンスを作成し、そのインスタンスで新しいDSキューを作成し、コードブロックを実行しているインスタンスのqueue の内容を、新しく作成したインスタンスqueue にコピーしています。
