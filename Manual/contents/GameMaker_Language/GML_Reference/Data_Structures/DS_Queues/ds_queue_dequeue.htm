ds_queue_dequeue

この関数は、DSqueue の先頭の値をデキューして、queue から取り除き、変数に格納する値を返します。queue が空の場合、この関数は定数 undefined を返し、そうでない場合はqueue に含まれる値を返します。

構文です。

ds_queue_dequeue(id);

論証タイプ 商品説明
id The id of the queue to dequeue from.

リターンです。

Variable (Data type value stored in the queue) or undefined

例

if !ds_queue_empty(move_queue)
{
    var xx = ds_queue_dequeue(move_queue);
    var yy = ds_queue_dequeue(move_queue);
    move_towards_point(xx, yy, 4);
}

上記のコードは、変数 "move_queue" でインデックスされた DS キューが空かどうかをチェックし、空でなければqueue の先頭から 2 つの値をデキューし、それを使って移動の方向を設定しています。
