ds_queue_create

この関数は、新しいqueue データ構造を作成し、インデックス値を返します。この値は変数に格納され、queue に関連する以降のすべての関数呼び出しで使用される必要があります。

重要!データ構造を作成するとき、それを識別するためのインデックス値は0から始まる整数値です。 これは、異なるタイプのデータ構造でも同じインデックス値を持つことができることを意味します。 ds_exists()関数を使用してからアクセスします。また、インデックスは再利用されるため、破壊されたデータ構造のインデックス値が、その後に新しく作成されたデータ構造で使用される可能性があることに注意してください。

構文です。

ds_queue_create();

リターンです。

DS Queue ID

例

queue = ds_queue_create();

これは、新しいqueue を作成し、そのインデックス ID をインスタンス変数 "queue" に代入します。
