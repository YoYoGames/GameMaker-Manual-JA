ds_queue_clear

この関数を使うと、与えられたqueue データ構造からすべてのデータをクリアすることができます。これは，データ構造を破壊するものではありません（破壊するには ds_queue_destroy()) からすべてのデータを消去し、空のqueue を返すだけです。

構文です。

ds_queue_clear(id);

論証タイプ 商品説明
id The id of the data structure to clear.

リターンです。

例

if (count = 15) && (!ds_queue_empty(command_queue))
{
    ds_queue_clear(command_queue);
    alarm[0] = room_speed;
    ai_count = 0;
}

上記のコードは、ある変数が特定の値に達したかどうかをチェックし、達した場合は変数 "command_queue" でインデックスされた DS キューをクリアし、アラームを設定し、変数を 0 にリセットしています。
