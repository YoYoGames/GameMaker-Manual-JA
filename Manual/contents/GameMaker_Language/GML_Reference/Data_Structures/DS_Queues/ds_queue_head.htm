ds_queue_head

この関数はqueue の最初の値（"先頭 "の値）だけを読み取ります。つまり、将来、この関数や ds_queue_dequeue().queue が空の場合、この関数は定数 undefined を返し、それ以外の場合はstring に含まれる実数値またはqueue を返します。

構文です。

ds_queue_head(id);

論証タイプ 商品説明
id The id of the data structure to read from.

リターンです。

(Data type value stored in the queue)

例

num = ds_queue_head(control_queue);

上記のコードは、変数 "control_queue "にインデックスされたqueue から値を読み取り、その戻り値を変数 "num "に格納します。
