ds_queue_write

この関数は，string を返します。 ds_queue_read()関数を使用します。

注：返されるstring は人間が読めるものではなく、string 、データ構造の内容のダンプである。

構文です。

ds_queue_write(id);

論証タイプ 商品説明
id The id of the data structure to write.

リターンです。

例

ini_open("save.ini");
var str =ds_queue_write(queue);
ini_write_string("Queues", "0", str);
ds_queue_clear(queue);
ini_close();

上記のコードでは、iniファイルを開き、変数 "queue "にインデックスされたDSキューに格納されている情報を含むstring を書き込んでいます。その後、queue がクリアされ、ini ファイルが閉じられる。
