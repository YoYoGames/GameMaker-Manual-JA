ds_map_replace_list

この関数を使うと、与えられた "key" に格納されているDS リストを、以前に作成した別のリストと置き換えることができます。この関数は、JSON 互換のマップを作成するために設計されており、それを json_encode()という機能があり、その機能との組み合わせでのみ使用されるべきです。

構文です。

ds_map_replace_list(id, key, value)

論証 タイプ 商品説明
id DSマップID The id of the map to use.
key 文字列 The key to replace.
value DSリストID The id of the ds_list to use to replace the one previously stored in the given key.

リターンです。

N/A

例

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
ds_map_replace_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_list_destroy(j_list);

上記のコードでは、 DS List を作成し、さまざまなグローバル変数の値を入力してから、 DS Map "j_map" に以前格納されていたリストを置き換えます。
