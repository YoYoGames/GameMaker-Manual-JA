ds_map_copy

この機能を使うと、あるマップの内容を、あらかじめ ds_map_create() を使って作成した別のマップにコピーすることができます。コピー先のDSマップが空でない場合。 この関数は、コピーする前にまずそれをクリアします。この処理では、元のDSマップは変更されません。

構文です。

ds_map_copy(id, source);

論証タイプ 商品説明
id The id of the map you are copying to
source The id of the map you are copying from

リターンです。

例

inventory_2 = ds_map_create();
ds_map_copy(inventory_2, inventory_1);

上記のコードでは、新しいマップを作成し、変数「inventory_2」に代入しています。そして、変数 "inventory_1" にインデックスされている DS マップの内容を、この新しいマップにコピーします。
