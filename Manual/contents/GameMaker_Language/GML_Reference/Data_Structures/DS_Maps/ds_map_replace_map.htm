ds_map_replace_map

この関数を使うと、与えられた "key "に格納されている DS Map を、以前に作成された別のマップに置き換えることができます。この関数は、JSON互換のマップを作成するために設計され、それを json_encode()という機能があり、その機能と組み合わせて使用する必要があります。

構文です。

ds_map_replace_map(id, key, value)

論証 タイプ 商品説明
id DSマップID The id of the ds_map to use.
key 文字列 The key to replace.
value DSマップID The id of the ds_map to use to replace the one previously stored in the given key.

リターンです。

N/A

例

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

上記のコードでは、 DS Map を作成し、配列と DS List を入力した後、 DS Map "j_map" にあらかじめ格納されているマップを置き換えます。
