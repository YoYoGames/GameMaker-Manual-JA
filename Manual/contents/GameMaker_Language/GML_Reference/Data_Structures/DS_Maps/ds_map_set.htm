ds_map_set

この関数を使うと、指定されたDSマップ内のキーの値を設定することができる。DS マップの ID を指定します (これは関数 ds_map_create())、次に設定したいキーとその値を指定します。キーは整数かstrings で、指定したキーが存在しない場合は、代わりにそのキーが作成され、値が設定されます。この関数は、DS map アクセサを使用してマップのキーと値のペアを設定/作成するのと同じです。この関数は何も返しませんので、キー値が置換されたかどうか、あるいは新しいキーが作成されたかどうかをチェックする必要がある場合は、関数 ds_map_replace() を使用してください。

構文です。

ds_map_set(id, key, value)

論証タイプ 商品説明
id The id of the map to use.
key The key to set.
value The value to set the key to.

リターンです。

例

if is_undefined(ds_map_find_value(map, "score"))
{
    ds_map_set(map, "score", 0);
}

上記のコードでは、与えられたキーが存在するかどうかを確認し、存在しない場合は、キーを作成して設定します。
