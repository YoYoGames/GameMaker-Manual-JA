ds_map_empty

この関数は、指定された（以前に作成された）DSマップが何らかのキー/値ペアを含んでいる場合は単に false 、含んでいない場合は true を返すだけです。

構文です。

ds_map_empty(id);

論証タイプ 商品説明
id The id of the data structure to check.

リターンです。

例

if ds_map_empty(inventory)
{
    weight = 0;
}

上記のコードでは、変数「inventory」でインデックスされたDSマップにキーと値のペアがあるかどうかをチェックし、なければ変数「weight」に0をセットしています。
