ds_map_add_map

この関数を使うと、与えられた DS マップ内のキーに (以前に作成した) DS マップを割り当てることができます。この関数は、JSON 互換のマップを作成するために設計されており、 それを json_encode()この機能と共に使用する必要があります。DS マップにこの方法で別のマップを追加した場合、親マップを破壊すると 含まれるマップも破壊されてメモリが解放され、その後に ds_map_clear()は、フラグが立っているマップも破壊し、クリーンアップします。

構文です。

ds_map_add_map(id, key, value)

論証 タイプ 商品説明
id DSマップID The id of the map to use.
key 文字列 The key for the added map.
value DSマップID The id of the map to add.

リターンです。

N/A

例

var j_map = ds_map_create();
var j_list = ds_list_create();
var sub_map = ds_map_create();
ds_map_add_list(sub_map, "list", j_list);
ds_map_add(sub_map, "array", j_array);
ds_map_add_map(j_map, "map", sub_map);
var j = json_encode(j_map);
ds_map_destroy(j_map);

上記のコードでは、2つのDSマップを作成し、1つにリストと配列を代入してから2つ目に追加し、それをJSONstring にエンコードしています。その後、マップは破棄され、マップとそれに含まれる他のマップやリストがメモリから削除されます。
