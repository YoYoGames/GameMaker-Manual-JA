ds_map_create

この関数は、新しい空の DS マップを作成し、その ID を返します。この ID は、他のすべての DS マップ関数でデータ構造にアクセスするために使用されます。

重要!データ構造を作成するとき、それを識別するためのインデックス値は0から始まる整数値です。 これは、異なるタイプのデータ構造でも同じインデックス値を持つことができることを意味します。 ds_exists()関数を使用してからアクセスします。また、インデックスは再利用されるため、破壊されたデータ構造のインデックス値が、その後に新しく作成されたデータ構造で使用される可能性があることに注意してください。

構文です。

ds_map_create();

リターンです。

DS Map ID

例

inventory = ds_map_create();

上記のコードでは、新しい空のDSマップを作成し、そのidインデックスを変数 "inventory "に格納します。
