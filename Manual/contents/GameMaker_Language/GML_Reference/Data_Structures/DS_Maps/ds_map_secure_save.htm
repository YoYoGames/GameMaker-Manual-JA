ds_map_secure_save

この関数は、与えられたDSマップの内容を、それが作成されたデバイスにリンクされたファイルに保存する（つまり、他のデバイスに転送しても読めないということ）。ファイル自体は、ほぼすべてのextension (たとえば、 *.dat.) を持つことができます。 *.json, *.bin, etc...）、難読化され、ターゲットプラットフォームのローカルストレージに保存されます。その後，ds_mapを再ロードするには，関数 ds_map_secure_load().ただし 保存される DS マップに配列が含まれている場合、この配列は保存時に DS リストに変換されます。

重要!セキュアセーブファイルの特徴として、作成されたデバイスにロックされるため、あるデバイスで保存されたファイルを他のデバイスで実行中のプロジェクトに読み込むことができません。

構文です。

ds_map_secure_save(map, filename);

論証タイプ 商品説明
map The id of the data structure to use
filename The name of the file to save the map to

リターンです。

例

ds_map_secure_save(purchase_map, "p_data.dat");

上記のコードは、変数 "p_data "にインデックスされたDSマップを、後で検索できるように、与えられたファイルに保存するものである。
