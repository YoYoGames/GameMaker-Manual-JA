ds_map_find_last

この関数は、DS マップに格納されている最後のキーを返します。これは、追加した順番で最後のキーになるわけではありません!DS マップは線形に保存されるわけではなく、DS リストを使用するため、この関数が行うのはコンピュータに保存されている最後のキーを見つけることだけです。これは、DS マップを繰り返し検索する必要がある場合に便利ですが、 時間がかかるので、できれば避けたいところです。

なお、この関数は、与えられたDSマップが空の場合、 undefined を返す。

構文です。

ds_map_find_last(id);

論証タイプ 商品説明
id The id of the map to use.

リターンです。

Variable or undefined

例

var size = ds_map_size(inventory);
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--)
{
    if (key != "gold")
    {
        key = ds_map_find_previous(inventory, key);
    }
    else break;
}

上記のコードでは、いくつかの一時変数を作成した後、DSマップのサイズを取得し、コンピュータがマップに保存している最後のキーを見つけます。そして、 for loop を使って DS マップを繰り返し検索し、キー値「gold」を探します。もし見つかったら、loop を実行する。
