ds_map_find_next

この関数は、DS マップに格納されているキーのうち、関数で指定したキーの次に格納されているキーを返します。これは、何かを探すために DS マップを繰り返し処理する必要がある場合に便利ですが、 時間がかかるので可能な限り避けるべきです。そのようなキーが存在しない場合、この関数は undefined を返します。これを確認するには、常に is_undefined()関数を使用します。

構文です。

ds_map_find_next(id, key);

論証タイプ 商品説明
id The id of the map to use.
key The key to find the next one to.

リターンです。

Variable or undefined

例

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if (key != "gold")
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

上記のコードでは、いくつかの一時変数を作成した後、DSマップのサイズを取得し、コンピュータがマップに保存している最初のキーを見つけます。次に、 for loop を使って DS マップを繰り返し、キーの値「gold」を探している。もし見つかったら、loop を展開する。
