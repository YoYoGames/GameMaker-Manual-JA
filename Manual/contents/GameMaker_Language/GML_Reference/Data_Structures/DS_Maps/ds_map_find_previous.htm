ds_map_find_previous

この関数は、DS マップに格納されているキーのうち、関数で指定したキーより前に格納されていたキーを返します。これは、何かを探すために DS マップを繰り返し処理する必要がある場合に便利ですが、 時間がかかるので、できれば避けたいところです。そのようなキーが存在しない場合、この関数は undefined を返します。これを確認するには、常に is_undefined()関数を使用します。

構文です。

ds_map_find_previous(id, key);

論証タイプ 商品説明
id The id of the map to use.
key The key to find the previous one to.

リターンです。

Variable or undefined

例

var size = ds_map_size(inventory) - 1;
var key = ds_map_find_last(inventory);
for (var i = size; i > 0; i--;)
{
    if (key != "gold")
    {
        _key = ds_map_find_previous(inventory, _key);
    }
    else break;
}

上記のコードでは、いくつかの一時変数を作成した後、ds_mapのサイズを取得し、コンピュータがマップに保存している最後のキーを見つけます。そして、 for loop を使って、ds_map を繰り返し検索し、キー値「gold」を探します。もし見つかれば、loop を取り出す。
