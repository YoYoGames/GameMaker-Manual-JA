ds_map_keys_to_array

この関数を使うと、ある DS マップが含むすべてのキーを取得することができる。で返されるような）キーを取得するための DS マップ ID を指定します。 ds_map_create()この配列の各エントリが DS マップのキーとなります。この場合、マップのキーデータは配列内の既存のデータに追加されます。この関数は、指定した配列を直接変更するだけでなく、 その配列への参照も返すことに注意しましょう (何も指定しなかった場合は、新しい配列への参照も返します)。

構文です。

ds_map_keys_to_array(id, [array])

論証 タイプ 商品説明
id DSマップID The id of the map to use.
[array] アレイ OPTIONAL DS マップのキーを追加するための配列。

リターンです。

Array

例

map_keys = ds_map_keys_to_array(inventory);

上記のコードは、DSマップのキーを取得し、将来使用するために配列として保存します。
