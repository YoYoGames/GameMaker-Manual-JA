ds_map_values_to_array

この関数を使うと、DS マップに含まれるすべての値を、マップの各キーに対して取得することができます。で返されるような) キーを取得するための DS マップ ID を指定すると、この関数はその DS マップの値を配列として返します。 ds_map_create()(たとえば、マップに 10 個のキーがある場合は、 10 個のエントリの配列が返され、各エントリがキーの値に対応します) この関数は、オプションの二番目の引数を指定します。）この場合、マップのキーの値データは、 配列内の既存のデータに追加されます。この関数は、指定した配列を直接変更するだけでなく、 その配列への参照も返すことに注意しましょう (何も指定しなかった場合は、新しい配列への参照も返します)。

構文です。

ds_map_values_to_array(id, [array])

論証 タイプ 商品説明
id DSマップID The id of the map to use.
[array] アレイ OPTIONAL DS マップの値を追加するための配列。

リターンです。

Array

例

var _values = ds_map_values_to_array(lvl_score);
var _total = 0;
var _length = array_length(_values);

for (var i = 0; i < _length; i ++)
{
    _total += _values[i];
}

draw_text(32, 32, "Total score for all levels: " + string(_total));

上記は、DS マップからマップの各キーに対応するすべての値を取得し、ローカル変数に配列として格納します。次に、loops 配列を走査して、各値をローカル変数に追加し、それを使って全値の合計を表示する。
