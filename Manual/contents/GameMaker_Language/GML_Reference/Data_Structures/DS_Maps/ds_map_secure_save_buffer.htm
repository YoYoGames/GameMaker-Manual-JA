ds_map_secure_save_buffer

この関数は，以前に作成した DS マップをbuffer に保存する。 DS マップの ID 値 (関数 ds_map_create()) と，書き込むbuffer の ID (関数 buffer_create()).保存される DS マップに配列が含まれている場合、保存時にこれは DS リストに変換されることに注意しましょう。

重要!セキュアセーブされたDSマップファイルは、それを作成したデバイスにしか読み込めませんので、別のデバイスに保存されたファイルを読み込もうとしても、それは機能しません。

注：この機能は、HTML5およびUWPではサポートされていません。

構文です。

ds_map_secure_save_buffer(filename);

論証 タイプ 商品説明
id DSマップID The DS map ID value.
buffer バッファID The buffer to save to.

リターンです。

N/A

例

buff = buffer_create(128,buffer_grow,4);
var map = ds_map_create();
ds_map_add(map,"bob","ajob");
ds_map_add(map,"money",10);
ds_map_secure_save_buffer(map, buff);
ds_map_destroy(map);

上記のコードは、buffer と DS マップを作成し、マップにいくつかの値を入れてbuffer に書き込んでからマップを削除します。
