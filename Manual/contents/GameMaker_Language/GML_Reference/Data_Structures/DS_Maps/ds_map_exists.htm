ds_map_exists

この関数は、指定されたキーが（以前に作成された）DSマップに存在する場合は true を、存在しない場合は false を返す。

構文です。

ds_map_exists(id, key);

論証 タイプ 商品説明
id DSマップID The id of the data structure to check
key 文字列 The key to check for

リターンです。

Boolean

例

if !ds_map_exists(inventory, "potions")
{
    ds_map_add(inventory, "potions", 1);
}

上記のコードは、変数「inventory」でインデックスされたDSマップに「potions」というキーがあるかどうかをチェックし、存在しなければマップに追加するものである。
