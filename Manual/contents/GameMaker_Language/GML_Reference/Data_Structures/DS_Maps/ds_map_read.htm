ds_map_read

この関数は、あらかじめ関数で作成されたstring を取得します。 ds_map_write()を作成し、それを以前に作成した DS マップに読み込む。string が読み込まれるマップにキー/値のペアが含まれている場合、保存されたマップが再構築される前に、まずそれらがクリアされます。

なお、指定されたDSstring がGameMakerで書かれたものである場合。Studio 1.2.xruntime (またはそれ以前) の場合、その後string のフォーマットが変更されたため、オプション引数 "legacy" を指定して true に設定する必要があります。

構文です。

ds_map_read(id, str, [legacy]);

論証 タイプ 商品説明
id DSマップID The id of the data structure to read the string into
str 文字列 The string to read
legacy ブーリアン OPTIONAL true または のいずれか、または完全に省略することができます。 false

リターンです。

N/A

例

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
    ds_map_read(inventory, t_string);
}
ini_close();

上記のコードでは、新しいDSマップを作成し、そのidインデックスを変数 "inventory "に格納している。次に、ini ファイルを開き、そのファイルからstring を読み込んで一時変数 "t_string" に格納する。最後に、string が有効かどうか (ini のデフォルト値である "" ではないか) をチェックし、有効な場合はstring を新しく作成した DS マップに読み込んでから ini を再び閉じます。
