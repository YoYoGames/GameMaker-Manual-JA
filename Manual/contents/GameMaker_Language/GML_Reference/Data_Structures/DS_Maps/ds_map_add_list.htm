ds_map_add_list

この関数を使うと、与えられた DS マップ内のキーに (以前に作成した)DSリストを割り当てることができます。この関数は、JSON 互換のマップを作成するために設計されており、それを json_encode()この機能と共に使用する必要があります。DS マップにこの方法でリストを追加した場合、親マップを破壊すると 含まれているリストも破壊されてメモリが解放され、同時に ds_map_clear()は、フラグ付きのリストも破棄してクリーンアップします。

構文です。

ds_map_add_list(id, key, value)

論証 タイプ 商品説明
id DSマップID The id of the map to use.
key 文字列 The key for the added list.
value DSリストID The id of the list to add.

リターンです。

N/A

例

var j_list = ds_list_create();
ds_list_add(j_list, health);
ds_list_add(j_list, lives);
ds_list_add(j_list, score);
var j_map = ds_map_create();
ds_map_add_list(j_map, "list", j_list);
var j = json_encode(j_map);
ds_map_destroy(j_map);

上記のコードでは、リストを作成し、そこにグローバル変数のさまざまな値を入力します。このリストは DS マップの中に "入れ子" にされ、マップは JSONstring にエンコードされた後、マップが破棄され、マップとそれに含まれるすべてのリストがメモリから削除されます。
