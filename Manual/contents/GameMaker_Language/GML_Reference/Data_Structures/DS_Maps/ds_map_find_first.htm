ds_map_find_first

この関数は、DS マップに格納されている最初のキーを返します。これは、追加した順番に最初のキーを返すのではありません!DS マップは線形に保存されるわけではなく、DS リストを使用するため、この関数が行うのはコンピュータが保存している最初のキーを見つけることだけです。これは、DS マップを繰り返し検索する必要がある場合に便利ですが、 時間がかかるので、できれば避けたいところです。

なお、この関数は、与えられたDSマップが空の場合、 undefined を返す。

構文です。

ds_map_find_first(id);

論証タイプ 商品説明
id The id of the map to use.

リターンです。

Variable or undefined

例

var size = ds_map_size(inventory) ;
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
    if key != "gold"
    {
        key = ds_map_find_next(inventory, key);
    }
    else break;
}

上記のコードでは、いくつかの一時変数を作成した後、DSマップのサイズを取得し、コンピュータがマップに保存している最初のキーを見つけます。次に、 for loop を使って DS マップを繰り返し、キーの値「gold」を探している。もし見つかったら、loop を展開する。
