ds_map_find_value

この関数を使用すると、指定したキーから値を取得することができます。この関数の入力値は、使用する（事前に作成した）DSマップと、チェックするキーである。

注：もしそのようなキーが存在しない場合、この関数は undefined を返します。これを確認するには、常に is_undefined()関数

構文です。

ds_map_find_value(id, key)

論証タイプ 商品説明
id The id of the map to use.
key The key to find.

リターンです。

Variable or undefined

例

amount = ds_map_find_value(inventory, "food");

または、マップアクセサー「?

amount = inventory[? "food"];

上記のコードでは、キー「food」の値を取得し、変数「amount」に格納します。
