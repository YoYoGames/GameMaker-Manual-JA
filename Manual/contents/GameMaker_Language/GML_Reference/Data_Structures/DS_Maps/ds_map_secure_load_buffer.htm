ds_map_secure_load_buffer

この関数は、buffer から安全に保存された DS マップをロードします。 事前にbuffer をメモリにロードしておく必要があります (これは buffer_load()この関数に渡すと、buffer の内容を取り込んだ DS マップが返されます。 この関数が正しく動作するためには、関数 ds_map_secure_save_buffer() を使ってbuffer が作成されていなければならないことに注意してください。 また、読み込む DS マップに配列が含まれていた場合、読み込み時にこれが DS リストに変換されることに注意してください。

重要!セキュアセーブファイルの特徴として、作成されたデバイスにロックされるため、あるデバイスで保存されたファイルを他のデバイスで実行中のプロジェクトに読み込むことができません。

注：この機能は、HTML5およびUWPではサポートされていません。

構文です。

ds_map_secure_load_buffer(buffer);

リターンです。

DS Map ID

例

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

上記のコードは、安全に保存された DS マップをbuffer からロードし、そのインデックス値を変数に保存して、将来使用するものです。
