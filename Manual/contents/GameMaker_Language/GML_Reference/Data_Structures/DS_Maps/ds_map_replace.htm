ds_map_replace

この関数を使うと、DSマップ内の指定されたキーの値を変更することができます。この関数が返すマップのインデックスを指定します。 ds_map_create()) そして、置換するキー (string または整数) とキー値を置換する値を指定します。指定されたキーが存在しない 場合はそのキーが作成され、存在する場合は現在の値が新しい の値です。この関数は、キーが存在し、値が置換された場合は true を、キーが存在せず、その値で新しいキーが作成された場合は false を返します。

構文です。

ds_map_replace( id, key, val );

論証タイプ 商品説明
id The id of the map to change.
key The key with the value that should be replaced by the new one
val The new value to replace the given value with

リターンです。

例

ds_map_replace(inventory, "torso", 55);

上記のコードでは、キー「torso」のDSマップを検索し、見つかったら（存在しない場合は作成される）現在の値を指定された値に置き換えています。
