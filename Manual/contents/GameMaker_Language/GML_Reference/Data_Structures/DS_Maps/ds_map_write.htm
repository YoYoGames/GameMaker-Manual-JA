ds_map_write

この関数は、指定されたマップの DS マップデータをstring 形式に変換し、*.ini または *.txt ファイルに書き込んで簡単に保存できるようにするものである。このstring は後で新しい ds_map に読み込むことができます。 ds_map_read().

注：返されるstring は，人間が読めるものではなく，データ構造の内容のダンプであるstring 。

構文です。

ds_map_write(id);

論証タイプ 商品説明
id The id of the data structure to use

リターンです。

例

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

上記のコードでは、書き込み可能なiniファイルを開いています。次に、 ds_map_write() を使ってstring を生成し、一時変数 "t_string" に格納する。最後に、string を ini ファイルに書き込んでから、ファイルを閉じます。
