ds_map_add

この関数は、指定されたDS マップにキー/値のペアのセットを追加するために使用されるべきである。この関数が成功したかどうかを確認することができます (成功した場合は true を、失敗した場合は false を返します)。 DS マップに同じキーが既に存在したり、追加するマップの ID に存在しない DS マップを指定したりすると、失敗する可能性があるからです。指定するキーと値は、任意のデータ型の組み合わせで構成することができる。 などが考えられる（ただし、実際にはキーにstring を使うのが最も一般的であろう）。

ds_map_add(map, 5, undefined);
ds_map_add(map, "level", 1);
ds_map_add(map, 89.6, "hello world");
ds_map_add(map, 5, buffer_get_address(buff));

また、以下のようにアクセッサ"?" を使用して、地図に追加することもできます。

map[? 5] = undefined;
map[? "level"] = 1;
map[? 89.6] = "hello world";
map[? 5] = buffer_get_address(buff);

注：GameMakerの他のデータ構造とは異なり、このキーはDSマップの開始点（または終了点）には行かず、DSマップの どこかに入るだけである。

構文です。

ds_map_add(id, key, val);

論証タイプ 商品説明
id The id of the map to add to.
key The key of the value to add.
val The value to add to the map.

リターンです。

例

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

これは新しいマップを作成し、そのidを変数 "inventory "に代入します。そして、マップに「hp potion」と「gold」という2つの新しいキーを追加し、それらの初期値を1と100に設定しています。
