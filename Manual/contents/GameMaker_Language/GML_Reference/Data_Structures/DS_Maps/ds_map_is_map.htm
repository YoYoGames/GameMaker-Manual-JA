ds_map_is_map

この関数は、与えられたマップキーにDSマップが格納されているかどうかを確認することができる。与えられたキーに DS マップ ID が含まれている場合，この関数は true を返し，そうでない場合は false を返します．

ds_map_add_map() 関数を使用して追加されたマップのみを検出することに注意してください。

構文です。

ds_map_is_map(id, key)

論証タイプ 商品説明
id The id of the map to use.
key The key to replace.

リターンです。

例

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if (ds_map_is_map(inventory, key))
    {
        ds_map_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

上記のコードはloops DS マップを通して、その中のキーが他の DS マップのものであるかどうかをチェックするものである。もしそうであれば、保存されている DS マップは破棄され、loop の最後でメインの DS マップも破棄される。
