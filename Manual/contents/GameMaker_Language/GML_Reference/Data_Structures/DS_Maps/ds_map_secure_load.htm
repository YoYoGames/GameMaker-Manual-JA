ds_map_secure_load

この関数は、与えられたファイルから以前に保存された DS マップをロードする。このファイルは、以前に ds_map_secure_save()この関数は、読み込まれたデータから作成された DS マップのインデックスを返します。この DS マップのインデックスは変数に格納され、 以降このマップに対するすべての関数呼び出しに使用される。ロードされる DS マップが配列の値をキーとして保存されていた場合、 ロード時にこの配列は DS リストに変換されることに注意すること。

重要!セキュアセーブファイルの特徴として、作成されたデバイスにロックされるため、あるデバイスで保存されたファイルを他のデバイスで実行中のプロジェクトに読み込むことができません。

構文です。

ds_map_secure_load(filename);

論証タイプ 商品説明
filename The name of the file to load the map data from.

リターンです。

DS Map ID

例

p_map = ds_map_secure_load("p_data.dat");

上記のコードは、安全に保存されたDSマップをロードし、そのインデックス値を変数に保存して、将来使用するものです。
