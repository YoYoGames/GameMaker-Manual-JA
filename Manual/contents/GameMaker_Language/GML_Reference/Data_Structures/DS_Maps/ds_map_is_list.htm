ds_map_is_list

この関数を使うと、与えられたマップキーにDSリストが格納されているかどうかを確認することができます。与えられたキーに DS リスト ID が含まれている場合、この関数は true を返します。 そうでない場合は、 false を返します。

ds_map_add_list() 関数を使って追加されたリストのみを検出することに注意してください。

構文です。

ds_map_is_list(id, key)

論証タイプ 商品説明
id The id of the map to use.
key The key to check.

リターンです。

例

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++)
{
    if ds_map_is_list(inventory, key)
    {
        ds_list_destroy(inventory[? key]);
    }
    key = ds_map_find_next(inventory);
}
ds_map_destroy(inventory);

上記のコードはloops DS マップを走査し、その中のキーが DS リストのものであるかどうかをチェックします。もしそうであれば、DS リストが破壊され、loop の最後に DS マップも破壊される。
