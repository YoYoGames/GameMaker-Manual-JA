ds_exists

この関数は、与えられた型のデータ構造が存在するかどうかをチェックすることができます。index" の値 (変数に格納されているもの) と DS "type" (以下に示す定数のいずれか) を与えると、この関数はデータ構造が存在すれば true を、そうでなければ false を返します。

DSタイプ定数
定数 商品説明
ds_type_map 地図データ構造
ds_type_list リストデータ構造
ds_type_stack スタックデータ構造
ds_type_grid グリッドのデータ構造
ds_type_queue 待ち行列のデータ構造
ds_type_priority 優先度データ構造

同じインデックス番号が異なるタイプの複数のデータ構造で使用されることがあるため、データ構造のタイプを確認するためにこの関数を使用することはできません。

構文です。

ds_exists(ind, type);

論証 タイプ 商品説明
ind 任意のDS ID The variable index to check for the data structure
type DSタイプ定数 The type of data structure to check for (see the list of constants above)

リターンです。

Boolean

例

if !ds_exists(ai_grid, ds_type_grid)
{
    ai_grid = ds_grid_create(room_width / 32, room_height / 32);
}

上記のコードでは、（あらかじめ初期化しておいた）変数 "ai_grid" が DS グリッドタイプのデータ構造のインデックスを持つかどうかをチェックし、持たない場合は作成してそのインデックスを変数に格納しています。
