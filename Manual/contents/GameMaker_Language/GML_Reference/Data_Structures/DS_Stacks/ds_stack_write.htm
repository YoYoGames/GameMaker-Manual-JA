ds_stack_write

この関数は，string を返します。 ds_stack_read()関数を使用します。

注：返されるstring は人間が読めるものではなく、string 、データ構造の内容のダンプである。

構文です。

ds_stack_write(id);

論証タイプ 商品説明
id The id of the data structure to write.

リターンです。

例

ini_open("save.ini");
var str = ds_stack_write(stack);
ini_write_string("Stacks", "0", str);
ds_stack_clear(stack);
ini_close();

上記のコードは、iniファイルを開き、変数 "stack "でインデックスされたDSスタックに格納されている情報を含むstring 。その後、スタックはクリアされ、ini ファイルは閉じられる。
