ds_stack_clear

この関数を使用すると、指定されたスタックデータ構造からすべてのデータをクリアすることができます。これは，データ構造体を破壊しません（破壊するためには ds_stack_destroy()のみで、空のスタックを返します。

構文です。

ds_stack_clear(id);

論証タイプ 商品説明
id The id of the data structure to clear.

リターンです。

例

if (ai_count = 15 && !ds_stack_empty(AI_stack))
{
    ds_stack_clear(AI_stack);
    alarm[0] = room_speed;
    ai_count = 0;
}

上記のコードは、ある変数が特定の値に達したかどうかをチェックし、達した場合は変数「AI_stack」でインデックスされたDSスタックをクリアし、アラームを設定し、変数を0にリセットしています。
