ds_stack_top

この関数は、スタックの最初の値（「上」にある値）だけを読み取ります。つまり、将来、この関数や ds_stack_pop().スタックが空の場合、この関数は定数 undefined を返し、それ以外の場合はスタックに含まれる実数値またはstring を返します。

構文です。

ds_stack_top(id);

論証タイプ 商品説明
id The id of the data structure to read from.

リターンです。

Variable (Data type value that is stored in the stack) or undefined

例

num = ds_stack_top(control_stack);

上記のコードは、変数「control_stack」にインデックスされたスタックから値を読み出し、その戻り値を変数「num」に格納します。
