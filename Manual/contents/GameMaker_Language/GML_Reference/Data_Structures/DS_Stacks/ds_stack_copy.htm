ds_stack_copy

この関数は、あるスタックの内容を別のスタックにコピーするために使用することができます。これは、元のスタックから内容を削除したり、元のスタックを破壊したりするものではないことに注意してください。この関数を使用する場合、コピー先のスタックは以前に作成されている必要があり、コピー前に何らかのアイテムが含まれていた場合、それらが最初にクリアされます（つまり、この情報は失われます）。

構文です。

ds_stack_copy(id, source);

論証 タイプ 商品説明
id DSスタックID The id of the new stack.
source DSスタックID The original stack to copy from.

リターンです。

N/A

例

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
{
stack = ds_stack_create();
ds_stack_copy(stack, other.stack);
}

上記の関数は、新しいインスタンスを作成し、そのインスタンスで新しいDSスタックを作成し、コードブロックを実行しているインスタンスのスタックの内容を、新しく作成されたインスタンススタックにコピーしています。
