ds_grid_read

この関数は、以前に関数によって作成されたstring を変換するために使用することができます。 ds_grid_write()を DS グリッドに戻す。DSグリッドは以前に作成されている必要があります（以下の例を参照してください）。

なお、指定されたDSstring がGameMakerで書かれたものである場合。Studio 1.2.xruntime (またはそれ以前) の場合、その後string のフォーマットが変更されたため、オプション引数 "legacy" を指定して true に設定する必要があります。

構文です。

ds_grid_read(index, string [, legacy]);

論証 タイプ 商品説明
index DS Grid ID The index of the grid to read.
string 文字列 The string to read into the DS grid.
legacy ブーリアン OPTIONAL true または のいずれか、または完全に省略することができます。 false

リターンです。

N/A

例

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close();

上記のコードでは、room のサイズに基づいて DS グリッドを作成し（32x32 の正方形のピクセルが 1 つのグリッドセルを表します）、ini ファイルから以前に保存したグリッドデータのセットを新しい DS グリッドに読み込んでいます。
