ds_grid_shuffle

この関数は、グリッド内の全セルの全値の位置をランダムにするために使用することができます。

注意この関数は、GameMakerが毎回同じ初期ランダムシードを生成して、debugging のコードをはるかに容易にするため、ゲームを新たに実行するたびに、グリッドセルを同じ位置にシャッフルします。これを避けるには この動作は randomise()をゲーム開始時に指定します。これはゲームのテストとdebugging 、最終的な実行ファイルである パッケージはこの挙動を示さず、毎プレイランダムになります。

構文です。

ds_grid_shuffle(index);

論証タイプ 商品説明
index The index of the grid to shuffle.

リターンです。

例

ds_grid_shuffle(grid)

これは、変数 "grid" でインデックスされた DS グリッド内のすべての値を取り、それらをシャッフルして異なる順序を得るものである。
