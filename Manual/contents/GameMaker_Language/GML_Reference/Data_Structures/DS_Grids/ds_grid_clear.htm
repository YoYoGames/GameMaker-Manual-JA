ds_grid_clear

この関数は、与えられたDSグリッドを特定の値にクリアするために使用することができます。グリッド内のすべてのセルにこの値が入ります。この値は実数またはstring です。以下の画像は、これがどのように動作するかを示しています。

構文です。

ds_grid_clear(index, val);

論証タイプ 商品説明
index This index of the grid to clear.
val The new value for all grid cells.

リターンです。

例

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上記のコードは、グローバル変数 "Grid" でインデックスされた DS グリッドのサイズを変更し、各セルに値 -1 が保持されるようにクリアします。
