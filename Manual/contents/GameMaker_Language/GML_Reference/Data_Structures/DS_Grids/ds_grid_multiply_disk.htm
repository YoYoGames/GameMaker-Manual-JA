ds_grid_multiply_disk

この関数は、DSグリッドの与えられた円盤状の領域にあるすべての値を取り、それぞれに与えられた量を乗算する。

注意この関数は実数に対してのみ動作し、strings は動作しません。

構文です。

ds_grid_multiply_disk(index, xm, ym, r, val);

論証 タイプ 商品説明
index DS Grid ID The index of the grid.
xm リアル The x position of the disk on the grid.
ym リアル The y position of the disk on the grid.
r リアル The radius of the disk on the grid.
val リアル The value to multiply the cells within the disk with.

リターンです。

N/A

例

ds_grid_multiply_disk(mygrid, 5, 5, 5, 2)

上記のコードは、円形のグリッド領域内で見つかったすべての値を取り、それぞれを2倍するものです。
