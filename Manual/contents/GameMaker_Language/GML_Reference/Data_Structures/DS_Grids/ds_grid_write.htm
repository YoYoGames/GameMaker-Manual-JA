ds_grid_write

この関数は，与えられた ds_grid をstring に変換し，外部ファイル（例えば）に格納することができる。この関数から返されたstring を ds_grid に読み戻すには，次の関数を使用する． ds_grid_read().

注 string返されるstring は，人間が読めるものではなく，データ構造の内容のダンプである。

構文です。

ds_grid_write(index);

論証タイプ 商品説明
index The index of the grid to write.

リターンです。

例

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

上記のコードは、ini ファイルを開き（まだ存在しない場合は作成）、与えられた ds_grid をそのファイルにstring として書き込みます。
