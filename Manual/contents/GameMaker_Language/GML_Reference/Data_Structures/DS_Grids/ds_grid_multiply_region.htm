ds_grid_multiply_region

この機能により、各セルの値を任意の量で乗算するグリッドの領域を指定することができます。

注意この関数は実数に対してのみ動作し、strings は動作しません。

構文です。

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

論証 タイプ 商品説明
index DS Grid ID The index of the grid.
x1 リアル The x position of the left of the region in the grid.
y1 リアル The y position of the top of the region in the grid.
x2 リアル The x position of the right of the region in the grid.
y2 リアル The y position of the bottom of the region in the grid.
val リアル The value to multiply with the region cells.

リターンです。

N/A

例

ds_grid_multiply_region(mygrid, 5, 5, 10, 10, 2);

上記のコードは、定義された長方形のグリッド領域内で見つかったすべての値を取り、それぞれに2を掛けます。
