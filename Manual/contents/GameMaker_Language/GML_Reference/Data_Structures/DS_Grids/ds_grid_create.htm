ds_grid_create

この関数を使用すると、指定したセルの幅と高さの新しい DS グリッドデータ構造を作成することができます。この関数は ID を返します。この ID は、この DS グリッドを扱う以降のすべての関数で使用する必要があります。

重要!データ構造を作成するとき、それを識別するためのインデックス値は0から始まる整数値です。 これは、異なるタイプのデータ構造が同じインデックス値を持つことができることを意味します。 ds_exists()関数を使用してからアクセスします。また、インデックスは再利用されるため、破壊されたデータ構造のインデックス値が、その後に新しく作成されたデータ構造で使用される可能性があることに注意してください。

構文です。

ds_grid_create(w, h);

論証タイプ 商品説明
w The width of the grid to be created.
h The height of the grid to be created.

リターンです。

DS Grid ID

例

mygrid = ds_grid_create(10, 10)

これにより、縦10セル、横10セルのグリッドが作成されます。
