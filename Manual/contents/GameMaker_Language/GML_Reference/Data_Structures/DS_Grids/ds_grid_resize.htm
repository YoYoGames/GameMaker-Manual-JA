ds_grid_resize

この関数は、与えられたDSグリッドの幅や高さを変更することができます。グリッドサイズが現在のグリッドより大きい場合，新しいセルの基本値は 0 となり，サイズが小さい場合は は、新しいサイズに収まらなくなり、失われます。それ以外のセルはそのまま残ります。

構文です。

ds_grid_resize(index, w, h);

論証タイプ 商品説明
index This index of the grid to resize.
w The new width of the grid.
h The new height of the grid.

リターンです。

例

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1)

上記のコードは、グローバル変数 "Grid" でインデックスされた DS グリッドのサイズを変更し、各セルに値 -1 が保持されるようにクリアします。
