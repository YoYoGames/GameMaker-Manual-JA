ds_grid_width

この関数は、与えられたグリッドの幅を返します。この値は、下の画像に示すように、グリッドが X 軸に沿って持つセルの数であり、常に整数です。

構文です。

ds_grid_width(index);

論証タイプ 商品説明
index This index of the grid to find the width of.

リターンです。

例

for (var i = 0; i < ds_grid_width(grid); ++i)
{
    for (var j = 0; j < ds_grid_height(grid); ++j)
    {
        if (ds_grid_get(grid, i, j) == 1)
        {
            instance_create_layer(i * 32, j * 32, "Walls", obj_Wall);
        }
    }
}

上記のコードは、変数 "grid" でインデックスされた DS グリッドをloop で検索し、特定のセルで値が 1 に等しければ、room 内の適切な位置に "obj_Wall" のインスタンスを作成します ...。
