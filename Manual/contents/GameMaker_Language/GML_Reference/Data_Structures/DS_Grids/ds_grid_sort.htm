ds_grid_sort

この機能は、与えられたカラムの値に基づいて DS グリッドをソートするために使用されます (OS のファイルエクスプローラでファイルを日付、サイズなどでソートするのと同じです)。次の画像は、その例である。

構文です。

ds_grid_sort(index, column, ascending);

論証タイプ 商品説明
index The index of the grid to sort.
column The column to use for sorting the rows
ascending Whether to sort lowest to highest (true)、または最高値から最低値(false)。

リターンです。

例

ds_grid_sort(grid, 3, false)

これは、変数 "grid" でインデックス化された DS グリッド内のすべての値を取得し、グリッドの 3 列目で見つかった値に従って並べ替えます (上の画像に示されているように)。
