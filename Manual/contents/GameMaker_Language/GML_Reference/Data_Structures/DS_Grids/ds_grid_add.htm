ds_grid_add

この関数は、グリッド内の与えられたセルの値に与えられた値（実数またはstring ）を追加するために使用することができます。追加される値は、グリッドセル内の値と同じ型でなければなりません。つまり、string を実数に追加したり、その逆をしたりすることはできません。 strings これは連結に相当する。

構文です。

ds_grid_add(index, x, y, val);

論証タイプ 商品説明
index The index of the grid.
x The x position of the cell in the grid.
y The y position of the cell in the grid.
val The value to add to the cell.

リターンです。

例

ds_grid_add(grid, 5, 5, 6)

この場合、変数 "grid" で参照される DS グリッド内の指定されたセルに 6 が追加される。
