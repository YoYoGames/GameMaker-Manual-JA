ds_list_add

この関数は、リストに新しい値（real またはstring ）を追加し、最後に追加するために使用することができます。この関数はさらにオプションの引数（必要な数だけ）を取ることができ、一度の呼び出しで複数の値をリストに連続して追加することができます。

構文です。

ds_list_add(id, val1 [, val2, ... max_val]);

論証 タイプ 商品説明
id DSリストID The id of the list to add to.
val バリアブル The value to add to the list.
[val2, ... max_val] バリアブル Optional values to be added to the list.

リターンです。

N/A

例

ds_list_add(sc_list, score);

上記のコードは、変数「score」に格納されている値を、変数「sc_list」でインデックス付けされたリストに追加するものである。
