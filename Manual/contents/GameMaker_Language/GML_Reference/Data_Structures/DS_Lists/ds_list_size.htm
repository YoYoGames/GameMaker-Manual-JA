ds_list_size

この関数は、リストの「サイズ」、すなわち、リストに追加された項目の数を返します。

構文です。

ds_list_size(id);

論証タイプ 商品説明
id The id of the data structure to check.

リターンです。

例

if !ds_list_empty(control_list)
{
    num = ds_list_size(control_list);
}

上記のコードは、DSリストが空かどうかをチェックし、空でなければ、その中に含まれる項目の数を取得し、変数に値を格納しています。
