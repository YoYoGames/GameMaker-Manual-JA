ds_list_set

この関数は、以前に追加されたリストエントリを設定するために使用できます。リスト ID (リストを作成したときに返されたもの) と、設定するリスト内の位置、そして設定する値を指定します。設定する項目がリストの範囲外である場合 (たとえば、リスト項目 20 を設定したが、現在のリストには 10 の項目しかない) は、 指定した位置までリストが埋められ、各項目が 0 に設定されます。 この関数は、DS リストアクセサと同じです。

構文です。

ds_list_set(id, pos, val);

論証 タイプ 商品説明
id DSリストID The id of the list to add to.
pos バリアブル The position within the list to set.
[val2, ... max_val] バリアブル The value to set in the list.

リターンです。

N/A

例

for (var i = 0; i < ds_list_size(list); i++;)
{
    ds_list_set(list, i, -1);
}

上記のコードは、変数「score」に格納されている値を、変数「sc_list」でインデックス付けされたリストに追加するものである。
