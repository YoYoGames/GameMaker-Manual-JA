ds_list_write

この関数は，string を返します。 ds_list_read()関数を使用します。

注：返されるstring は，人間が読めるものではなく，データ構造の内容のダンプであるstring 。

構文です。

ds_list_write(id);

論証タイプ 商品説明
id The id of the data structure to write.

リターンです。

例

ini_open("save.ini");
var str = ds_list_write(list);
ini_write_string("Lists", "0", str);
ds_list_clear(list);
ini_close();

上記のコードでは、iniファイルを開き、変数 "list "にインデックスされたDSリストに格納されている情報を含むstring を書き込んでいます。その後、リストがクリアされ、ini ファイルが閉じられる。
