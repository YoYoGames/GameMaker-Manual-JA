ds_list_create

この関数は、新しいリストデータ構造体を作成し、インデックス値を返します。この値は変数に格納し、そのリストに関連する以降のすべての関数呼び出しで使用する必要があります。もしリストに値が入る前、つまりリストが "空" の状態でアクセスしようとすると、返り値が 0 になる可能性があることに注意しましょう。 ds_list_size()関数は、新しく作成されたリストに対しても 0 を返します)。もし、リストの作成時に「本当に」空であることを確認したい場合は、リストの作成時に ds_list_clear()これは、未入力のリストスロットに返されるすべての値が undefined になることを意味します。

重要!データ構造を作成するとき、それを識別するためのインデックス値は0から始まる整数値です。 これは、異なるタイプのデータ構造でも同じインデックス値を持つことができることを意味します。 ds_exists()関数を使用してからアクセスします。また、インデックスは再利用されるため、破壊されたデータ構造のインデックス値が、その後に新しく作成されたデータ構造で使用される可能性があることに注意してください。

構文です。

ds_list_create();

リターンです。

DS List ID

例

list = ds_list_create();

これは、新しいリストを作成し、そのインデックスIDをインスタンス変数 "list "に代入します。
