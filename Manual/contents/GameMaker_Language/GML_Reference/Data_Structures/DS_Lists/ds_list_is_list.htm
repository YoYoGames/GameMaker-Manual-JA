ds_list_is_list

この関数を使うと、ある DS リスト内の指定した位置に別の DS リストが格納されているかどうかを確認することができます。指定した位置に DS リスト ID が含まれている場合、この関数は true を返します。 そうでない場合は、 false を返します。

ds_list_mark_as_list() 関数を使って手動でマークしたリストのみを検出することに注意してください。

構文です。

ds_list_is_list(id, pos);

論証タイプ 商品説明
id The id of the list to check.
pos The position within the list to check.

リターンです。

例

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if ds_list_is_list(ships, i)
    {
        ds_list_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

上記のコードは、loops DS リストを通して、いずれかのエントリが他のリスト ID を含んでいるかどうかをチェックします。もし含まれていれば、それらのリストが破棄され、loop が終了した後にメインリストが破棄されます。
