ds_list_is_map

この関数を使うと、DS マップがDS リスト内の指定された位置に格納されているかどうかを調べることができる。与えられた位置に DS マップ ID が含まれている場合、この関数は true を返します。 そうでない場合は、 false を返します。

ds_list_mark_as_map() 機能を使用して手動でマークされたマップのみが検出されることに注意してください。

構文です。

ds_list_is_map(id, pos);

論証タイプ 商品説明
id The id of the list to check.
pos The position within the list to check.

リターンです。

例

var size = ds_list_size(ships);
for (var i = 0; i < size; i++)
{
    if (ds_list_is_map(ships, i))
    {
        ds_map_destroy(ships[| i]);
    }
}
ds_list_destroy(ships);

上記のコードはloops DS リストを通して、いずれかのエントリがマップ ID を含んでいるかどうかをチェックします。もし含まれていれば、これらのマップは破壊され、loop が終了した後にメインリストが破壊される。
