ds_list_sort

この関数を使うと、リスト内のすべての値を昇順または降順に並べ替えることができます。リストにstrings が含まれる場合は、英語の26文字のアルファベットに基づき、アルファベット順に並べ替えられます。

構文です。

ds_list_sort(id, ascend);

論証タイプ 商品説明
id The id of the list to sort.
ascend Whether the values should be ascending (true) または降順 (false) とする。

リターンです。

例

if newgame
{
    ds_list_sort(name_list, true);
}

上記のコードは、変数 "newgame "にフラグが立っている場合、変数 "name_list "にインデックスされているリストをソートします（ true ） .
