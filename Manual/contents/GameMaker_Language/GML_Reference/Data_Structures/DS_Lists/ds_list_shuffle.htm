ds_list_shuffle

この関数はリストをシャッフルし、すべての構成要素の値を、リストに最初に追加された位置からランダムに並べ替えることができます。

注：この関数は、ゲームが新しく実行されるたびにリストアイテムを同じ位置にシャッフルします。これは、GameMakerがdebugging のコードをはるかに簡単にするために、毎回同じ初期ランダムシードを生成しているためです。この挙動を避けるには randomise()をゲーム開始時に指定します。これは、ゲームのテストとdebugging 、最終的な実行可能パッケージはこの動作を示さず、毎回のプレイがランダムになるためです。

構文です。

ds_list_shuffle(id);

論証タイプ 商品説明
id The id of the list to shuffle.

リターンです。

例

if restart
{
    ds_list_shuffle(card_list);
}

上記のコードは、変数 "restart "に true というフラグが立てば、変数 "card_list "にインデックスされたリストをシャッフルする。
