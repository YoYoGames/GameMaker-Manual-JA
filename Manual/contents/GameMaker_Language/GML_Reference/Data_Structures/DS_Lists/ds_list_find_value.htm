ds_list_find_value

この関数では、与えられたリストの位置をチェックし、その位置のリスト内の値を返すことができます。指定した位置がリストのサイズから外れている場合 (たとえば 10 個の値を持つリストの 11 番目の位置)、 この関数は undefined あるいは0 を返すことに注意しましょう。 これは、リストを作成する際に、最初に項目を追加する際のパフォーマンスの問題を最小限に抑えるために、 リストの最初の数項目が内部的に 0 に設定されているからです (しかしながら ds_list_size()関数は、新しく作成されたリストでも0を返します）。もし、リストの作成時に「本当に」空であることを確認したい場合は、リストの作成時に ds_list_clear()これは、未入力のリストスロットに返されるすべての値が undefined であることを意味し、その後に is_undefined()関数があります。

構文です。

ds_list_find_value(id, pos);

論証 タイプ 商品説明
id DSリストID The id of the list to use.
pos リアル The position to look at, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.

リターンです。

Variable or undefined

例

val = ds_list_find_value(list, ds_list_size(list) - 1);

上記のコードは、変数 "list "にインデックスされたリストの最後の位置をチェックし、返された値を変数 "val "に格納します。
