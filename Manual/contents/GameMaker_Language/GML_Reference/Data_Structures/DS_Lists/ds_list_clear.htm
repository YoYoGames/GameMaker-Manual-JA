ds_list_clear

この関数を使用すると、与えられたリストデータ構造からすべてのデータをクリアすることができます。これは，データ構造体を破壊しません（破壊するためには ds_list_destroy()この場合、すべてのデータが消去され、リストが空（サイズゼロ）になるだけです。）リストを消去すると、そこに格納されているすべてのデータ構造の参照が解除され、メモリリークが発生することに注意してください。したがって、これを防ぐには、リストを消去する前にすべてのデータ構造項目を手動で破棄する必要があります。この作業が不要なのは、リスト内のアイテムに別のDS リストやDS マップとしてのフラグを立てた場合のみで、 その場合は親をクリアする際に、これらのアイテムは破棄され (クリアされない!) 自動的にメモリがクリーンアップされます。

構文です。

ds_list_clear(id);

論証タイプ 商品説明
id The id of the data structure to clear.

リターンです。

例

if (count == 15) && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    ai_count = 0;
}

上記のコードは、ある変数が特定の値に達したかどうかをチェックし、達した場合は変数「command_list」でインデックスされたDSリストをクリアし、アラームをセットし、変数を0にリセットしています。
