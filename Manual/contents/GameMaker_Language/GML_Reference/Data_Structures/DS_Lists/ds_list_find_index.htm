ds_list_find_index

この関数を使用すると、指定されたリストの値をチェックし、その値のリスト内での位置を返すことができます。リスト内に同じ値を持つ項目が複数ある場合は、そのうちのいずれかの位置が返され、その値が存在しない場合は -1 が返されることに注意してください。

構文です。

ds_list_find_index(id, val);

論証 タイプ 商品説明
id DSリストID The id of the list to use.
val バリアブル The value to find.

リターンです。

Real

例

pos = ds_list_find_index(list, "Player1");

上記のコードは、変数listにインデックスされたリストの値「Player1」をチェックし、返された位置を変数posに格納しています。
