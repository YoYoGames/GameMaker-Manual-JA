ds_list_replace

この関数は、指定された位置の値を別の値に置き換えます。

構文です。

ds_list_replace(id, pos, val);

論証 タイプ 商品説明
id DSリストID The id of the list to change.
pos リアル The position to replace the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val バリアブル The new value to replace the given value with.

リターンです。

N/A

例

ds_list_replace(n_list, 3, name);

上記のコードは、リストの3番目に格納されている値を、変数 "name "に格納されている値に置き換えます。
