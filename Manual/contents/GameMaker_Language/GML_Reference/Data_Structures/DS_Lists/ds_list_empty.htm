ds_list_empty

この関数を使うと、与えられたDSリストが空かどうか（ true を返す）、あるいは空でないか（ false を返す）をチェックすることができます。

構文です。

ds_list_empty(id);

論証 タイプ 商品説明
id DSリストID The id of the data structure to check.

リターンです。

例

if (count == 15( && (!ds_list_empty(command_list))
{
    ds_list_clear(command_list);
    alarm[0] = room_speed;
    count = 0;
}

上記のコードは、ある変数が特定の値に達したかどうかをチェックし、達した場合は変数「command_list」でインデックスされたDSリストをクリアし、アラームをセットし、変数を0にリセットしています。
