ds_list_insert

この関数は、与えられた値を与えられた位置のリストに追加します。もしリストが与えられた位置の後にさらに値を含んでいれば、それらの位置は1つ上にシフトされ、room 、リストが1つ大きくなるようにします。

構文です。

ds_list_insert(id, pos, val);

論証 タイプ 商品説明
id DSリストID The id of the list to add to.
pos リアル The position to add the value, where 0 corresponds to the very beginning of the list and the final position is ds_list_size(id)-1.
val バリアブル The value to add to the list.

リターンです。

N/A

例

ds_list_insert(list, 9, score);

上記のコードは、変数「score」に格納されている値を、変数「list」でインデックス付けされたリストの10番目の位置に追加します（リストは0から数えるので、10個の値のリストは0〜9の番号が振られます）。
