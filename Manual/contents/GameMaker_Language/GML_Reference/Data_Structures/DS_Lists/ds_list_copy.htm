ds_list_copy

この機能を使うと、あるリストの内容を別のリストにコピーすることができます。どちらのリストも以前に作成されている必要があり、コピー先のリストがすでに情報を持っている場合は、まずそのリストがクリアされます。最終的には、同じ情報を含む独立した2つのリストができあがります。

構文です。

ds_list_copy( id, source );

論証タイプ 商品説明
id The id of the list being copied to.
source The id of the list to be copied from.

リターンです。

例

if !ds_list_empty(main_list)
{
old_list = ds_list_create();
ds_list_copy(old_list, main_list);
ds_list_clear(main_list);
}

上記のコードは、DSリストが空であるかどうかをチェックします。もし空でなければ、それは別の DS リスト (以前に作成されたもの) にコピーされ、その後、元のリストが消去されます。
