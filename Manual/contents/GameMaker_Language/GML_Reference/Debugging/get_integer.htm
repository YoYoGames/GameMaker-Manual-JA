get_integer

これはカスタムメッセージを表示するポップアップウィンドウを作成し、"Ok "というラベルのついたボタンを表示し、ユーザに整数値を入力するように促します。この関数は入力された整数を返し、何も入力されていない場合はデフォルト値を返します。

注意この関数はWindows ターゲット上でのみデバッグ用に使用され、他のすべてのターゲットでは非推奨となります。

構文です。

get_integer(str, def);

論証 タイプ 商品説明
str 文字列 The string to show in the pop-up message.
def リアル The default value in the text box.

リターンです。

Real

例

global.level = get_integer("Level to test?", 1);

上記のコードでは、テストするレベルを選択するようユーザに促すメッセージが表示されます。戻り値は、グローバル変数 "global.level" に格納されます。
