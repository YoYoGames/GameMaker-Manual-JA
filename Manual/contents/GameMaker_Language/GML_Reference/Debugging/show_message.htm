show_message

この関数は、与えられたstring と、それを閉じるための "Ok" とマークされたボタンを表示するポップアップメッセージボックスを作成します。

注意この関数はWindows ターゲット上でのみデバッグ用に使用され、他のすべてのターゲットでは非推奨とされています。

構文です。

show_message(str);

論証 タイプ 商品説明
str 文字列 The string to show in the pop-up message.

リターンです。

N/A

例

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
    tot += inv[i];
}
show_message("Total = " + string(tot));

上記のコードは、配列「inv」に格納されている値をloop 、変数「tot」に追加してから、合計値をメッセージで表示します。
