コードがコンパイルされているかどうか

この関数は、YYC や JS プラットフォームなど、仮想マシンの外でコンパイルする場合、 true を返します。

構文です。

code_is_compiled();

リターンです。

例

if code_is_compiled()
{
    show_debug_message("Compiler okay!");
}

上記のコードは、ゲームがYoYo Compiler を使ってコンパイルされたかどうかをチェックし、コンパイルされた場合はデバッグメッセージを表示します。
