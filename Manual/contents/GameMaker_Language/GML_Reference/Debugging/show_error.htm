ショーエラー

この関数は、エラーメッセージとしてカスタムstring を表示します。

重要第2引数は、古いプロジェクトとの後方互換性のためにのみ存在し、表示されるエラーダイアログには何の影響も与えません。

注意事項この機能はデバッグ用です。

構文です。

show_error(str, abort);

論証 タイプ 商品説明
str 文字列 The string to show in the pop-up message.
abort ブーリアン UNUSED エラー発生時にゲームを中断させるか（true）、プレイヤーがエラーを無視できるようにするか（false）を指定します。

リターンです。

N/A

例

if (room != rm_Dungeon)
{
    show_error("Error: Went to wrong area. Aborting game.", true);
}

上記のコードは、現在のroom がrm_Dungeon であるかどうかをチェックし、そうでない場合はエラーメッセージを表示します。
