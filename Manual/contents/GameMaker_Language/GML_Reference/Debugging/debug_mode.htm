デバッグモード

この読み取り専用変数は、ゲームがデバッグモードでプレイされているときは true 、通常通りプレイされているときは false を返します。

構文です。

debug_mode

リターンです。

例

if debug_mode
{
    ini_open("Cheats.ini");
}
else
{
    ini_open("Game.ini");
}

上記のコードでは、読み取り専用変数 debug_mode の値によって、異なる ini ファイルを開いています。
