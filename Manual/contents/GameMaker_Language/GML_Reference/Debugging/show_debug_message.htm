show_debug_message

この関数は、カスタムデバッグメッセージを作成し、コンパイラーウィンドウにruntime で表示します。string メッセージはstring でも実数でも構いませんが、もし両方が必要な場合は、最初に string()また、小数点以下が2桁以上ある数値の場合は string_format()というのは、デフォルトでは小数は小数点以下2桁に丸められるからです（つまり、「1.2468」は出力ウィンドウに「1.25」と表示されます）。

この機能で表示されるデバッグメッセージは、デバッグモードでゲームを実行すると、IDE の下部にあるCompiler Output Windowと、debugger のグラフビューに表示されます。もし、デバッグモードで表示されるメッセージだけを見たい場合は debug_event()の代わりに

構文です。

show_debug_message(string);

論証タイプ 商品説明
string The custom debug message to show.

リターンです。

例

if !instance_exists(obj_Exit)
{
    show_debug_message("Exit not created!");
    show_debug_message("Error Num: " + string(global.error));
    game_end();
}

上記のコードでは、インスタンスが存在するかどうかをチェックし、存在しない場合はコンパイルウィンドウにデバッグメッセージを表示し、ゲームを終了しています。
