ショー・クエスチョン

この関数は、"Yes "と "No "の2つのボタンを持つポップアップメッセージボックスを作成する。ユーザがどちらのボタンを押したかによって、 true または false を返します。

注意この関数はWindows ターゲット上でのみデバッグ用に使用され、他のすべてのターゲットでは非推奨となります。

構文です。

show_question(str);

論証 タイプ 商品説明
str 文字列 The string to show in the pop-up question.

リターンです。

Boolean

例

if (score > 500) && debug_mode
{
    if show_question("Continue to next room?")
    {
        room_goto(rm_Level2);
    }
    else game_end();
}

上記のコードは、スコアをチェックし、500点以上であれば、ユーザーに続けるかどうかを尋ね、「はい」ボタンがクリックされれば、別のroom 、しかし「いいえ」ボタンが選択されれば、ゲームを終了します。
