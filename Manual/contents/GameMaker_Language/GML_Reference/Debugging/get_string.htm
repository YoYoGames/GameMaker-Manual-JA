get_string

これは、標準的なメッセージを表示するポップアップウィンドウを作成し、「Ok」というラベルの付いたボタンで、ユーザーにstring を入力するように促します。 この関数は、入力されたstring を返し、何も入力されていない場合はデフォルト値を返します。

注意この関数はWindows ターゲット上でのみデバッグ用に使用され、他のすべてのターゲットでは非推奨となります。

構文です。

get_string(str, def);

論証 タイプ 商品説明
str 文字列 The string to show in the pop-up message.
def 文字列 The default string in the text box.

リターンです。

String

例

global.test_name = get_string("Test highscore name:", "Anonymous");

上記のコードでは、グローバル変数「test_name」に格納される名前を入力するようユーザーに促します。もし何も入力せず「OK」を押した場合は、デフォルトの「Anonymous」が返されます。
