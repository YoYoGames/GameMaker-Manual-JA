fps

GameMakerでは、ゲームの実行速度を伝えるために、主に2つの方法があります。それは game speed(ゲームオプションで指定)と fps (frames per second)です。これらの値はよく混同されますが、基本的に一方はGameMakerが1秒間に完了するはずのゲームステップ数であり、もう一方（fps）はGameMakerが1秒間に実際に完了しているCPUステップ数で、最大値は room スピードそのものになります。本当のfps、つまり1ゲームステップあたりの実際のCPUステップ数を得るには、以下のようにする。fps_real変数を使用します。

この読み取り専用変数は、現在のfpsを整数値で返します。この関数はゲームの各ステップごとに一度だけ更新されるので、ある値から別の値に「飛ぶ」ように見えるかもしれませんが、これはごく普通のことであることに注意してください。

構文です。

fps

リターンです。

例

if debug_mode
{
    draw_text(32, 32, "FPS = " + string(fps));
}

上記のコードは、ゲームがデバッグモードであるかどうかを確認し、デバッグモードであれば現在のfpsを画面に表示します。
