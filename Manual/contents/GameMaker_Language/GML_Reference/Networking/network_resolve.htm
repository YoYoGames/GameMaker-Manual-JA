この関数は、指定された URL の IP アドレスを返します。

構文です。

network_resolve(url);

論証タイプ 商品説明
url The URL to get the IP of (a string).

リターンです。

例

game_ip = network_resolve("www.macsweeneygames.com");

上記のコードは、与えられたURLのIPアドレスを返し、変数 "game_ip "に格納します。
