network_connect_raw_async

この関数を用いて，server に接続要求を送ることができます． この関数は，接続するソケットIDを受け取ります( network_create_socket()を指定し、接続するIPアドレスを指定する必要があります。 (string) と接続するポートに接続し、接続に失敗した場合は 0 未満の値を返します。この関数と network_connect_async()は、この 関数は任意のserver に接続でき、生データには何もしません。つまり、server 側でプロトコルを自分で実装しなければならないのです。この関数は非同期であり、network_type_non_blocking_connect タイプのAsynchronous Networkingイベントを生成することに注意してください。

構文です。

network_connect_raw_async(socket, url, port);

論証タイプ 商品説明
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

リターンです。

例

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

上記のコードでは、新しいTCPソケットを作成し、そのソケットを通して与えられたURLのポート6510に「生の」非同期接続を作成しようと試みます。
