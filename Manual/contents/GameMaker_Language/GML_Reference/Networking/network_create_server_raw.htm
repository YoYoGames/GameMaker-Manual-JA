ネットワークサーバを作成する

この関数は、許可された接続プロトコルのいずれかを使用して、ゲーム用に新しいネットワークserver を作成するために使用されます（以下の定数を参照）。server のタイプを指定し、使用するポートを指定し、最後にserver に一度に許可する最大接続数を指定します（この数は任意ですが、接続クライアントが多すぎるとネットワークが飽和したり、デバイスの CPU がその数のプレイヤーの処理を処理しきれなくなりますので、注意して使用してください）。この関数は、変数に格納され、以降のすべてのネットワーク機能でserver を識別するために使用されるべき固有のIDを返すか、接続に失敗した場合は 0 未満の値を返します。

これは「生の」server を作成するので、通信のために組み込まれたGameMaker データヘッダを受け付けたり使用したりしません。 network_send_raw()と network_send_udp_raw()を使用して、作成されたserver にフォーマットされていないデータを送信します。

構文です。

network_create_server_raw(type, port, max_client);

論証タイプ 商品説明
type The type of server to create (see the constants listed below).
port The port that the server will use.
max_client The maximum number of clients that can connect at once.

リターンです。

Network Socket ID

例

var port = 6510;
server = network_create_server_raw(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
{
    port++
    server = network_create_server_raw(network_socket_tcp, port, 32);
}

上記のコードは、6510番ポートのTCPを使用して、server を作成しようとします。もしそのポートが利用できない場合は、loop 、利用できるポートを探します。
