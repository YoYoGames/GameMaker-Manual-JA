network_send_udp_raw

この関数は、UDPを使用してネットワーク経由でデータをserver に送信します。 この関数は、接続するソケットID、接続先URL、使用するポートを受け取ります。次に、送信するデータを格納するバッファID（buffers の詳細については、リファレンス - バッファを参照してください）と、最後にデータのサイズ（バイト単位）を指定する必要があります。UDP は「コネクションレス型」で、実際には接続を行わず、IP に直接パケットを送信します。server は IP アドレスから受信データを受け取り、それを「そのまま」扱わなければなりません。送信されたデータはGameMakerでは一切フォーマットされず、受信側のデバイスはデータをストリームとして取得するため、自分で処理する必要がある。この関数は送信されたデータのバイト数を返すが、送信に失敗した場合は0より小さい値を返す。

注）この機能は、HTML5ターゲット上で動作するプロジェクトで使用した場合は動作しません。また、HTML5プロジェクトでもUDPを受信することはできません。

構文です。

network_send_udp_raw(socket, url, port, bufferid, size);

論証タイプ 商品説明
socket The id of the socket to use.
url The url or IP to connect to (a string).
port The port to connect to.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

リターンです。

Real

例

network_send_udp_raw(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff));

上記のコードは、URLで定義されたserver に、ポート6510で生のUDPパケットを送信します。データは変数 "buff "にインデックスされたbuffer から取得される。
