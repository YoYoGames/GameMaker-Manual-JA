ネットワーク_送信_udp

この関数を使うと、UDPを使ってネットワーク経由でデータをserver に送ることができます。 この関数は、接続するソケットID、接続先URL、使用するポートを受け取ります。次に、送信するデータを格納するバッファID（buffers の詳細については、リファレンス - バッファを参照してください）と、データのサイズ（バイト単位）を指定する必要があります。UDP は「コネクションレス型」で、実際には接続を行わず、単にパケットを IP に直接送信します。server は IP アドレスから受信データを受け取り、それを「そのまま」処理しなければなりません。この関数は、送信されたデータのバイト数を返し、送信に失敗した場合は0より小さい数値を返します。この関数が返す送信データの最終サイズには、さらに12バイトのヘッダー情報（GameMaker ）も含まれることは注目に値する。

この機能は、HTML5ターゲット上で動作するプロジェクトで使用した場合、動作しません。また、HTML5プロジェクトでもUDPを受信することはできません。

構文です。

network_send_udp(socket, url, port, bufferid, size);

論証 タイプ 商品説明
socket ネットワークソケットID The id of the socket to use.
url 文字列 The url or IP to connect to (a string).
port リアル The port to connect to.
bufferid バッファID The id of the buffer to get the data from.
size リアル The size (in bytes) of the data.

リターンです。

Real

例

network_send_udp(sock, "www.macsweeneygames.com", 6510, buff, buffer_tell(buff));

上記のコードは、URLで定義されたserver に、ポート6510でUDPパケットを送信します。データは変数 "buff "にインデックスされたbuffer から取得される。
