network_create_socket_ext

この関数は、あなたのゲームがネットワーク上で通信するための新しいクライアントソケットを作成するために使用されます。ソケットの種類(以下の定数リストを参照)と使用するポートを定義する必要があります。この関数は、そのソケットに対する以降のすべての関数呼び出しで使用されるべき一意のID、または接続に失敗した場合は 0 未満の値を返します。

構文です。

network_create_socket_ext(protocol, port);

論証 タイプ 商品説明
protocol ソケットタイプ一定 The network protocol to use
port リアル The port to use

リターンです。

Network Socket ID

例

client = network_create_socket_ext(network_socket_udp, 6510);

上記のコードでは、ポート6510に新しいUDPソケットが作成されます。
