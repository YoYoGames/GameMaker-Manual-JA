ネットワークコネクトアシンク

この関数を用いて，server に接続要求を送ることができます． この関数は，接続するソケットIDを受け取ります( network_create_socket()を指定し、接続するIPアドレスを指定する必要があります。 (string) に接続し、接続に失敗した場合は 0 未満の値が返されます。この接続は特殊なプロトコルを使用しており、GameMakerのゲームだけが互いに接続できるようになっています。 を使用して、GameMakerのゲームではないserver に接続することができます。 network_connect_async_raw().この関数は非同期式であり、network_type_non_blocking_connect 型のAsynchronous Networkingイベントを生成することに注意してください。

構文です。

network_connect_async(socket, url, port);

論証タイプ 商品説明
socket The id of the socket to use.
url The URL or IP to connect to (a string).
port The port to connect to.

リターンです。

例

client = network_create_socket(network_socket_tcp);
network_connect_async(client, "192.134.0.1", 6510);

上記のコードでは、新しいTCPソケットを作成し、そのソケットを介して、指定されたIPアドレスのポート6510に非同期接続を試みます。
