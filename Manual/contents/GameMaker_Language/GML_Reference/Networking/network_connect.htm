ネットワーク接続

この関数を使うと、server に接続するためのリクエストを送ることができます。 この関数は、接続するためのソケットID(参照)を受け取り、接続先のIPアドレス(参照)と接続するためのポートを要求します。 network_create_socket()) を受け取り、接続先のIPアドレス (string) と接続するポートを指定する必要がある。接続に失敗した場合は0未満の値が返される。この接続は、GameMakerゲームだけが互いに接続できるように特別なプロトコルを使用していますが、GameMakerゲームでないserver に接続する必要がある場合は、次のように使用します。 network_connect_raw().デフォルトでは、この機能は同期型であるため、接続中にゲームがハングアップしているように見えることがあることに注意してください。接続のタイムアウト値を設定するか、関数 network_set_config() を使って非同期で作成するか、あるいは代わりに関数 network_connect_async() を使うことができます。

注意HTML5ではこの関数は使用できません。WebSocketの場合は、Async関数を使用してください。

構文です。

network_connect(socket, url, port);

論証 タイプ 商品説明
socket ネットワークソケットID The id of the socket to use.
url 文字列 The URL or IP to connect to (a string).
port リアル The port to connect to.

リターンです。

Real or Network Socket ID

例

client = network_create_socket(network_socket_tcp);
network_connect(client, "192.134.0.1", 6510);

上記のコードでは、新しいTCPソケットを作成し、そのソケットを通して、指定されたIPアドレスのポート6510への接続を試みます。
