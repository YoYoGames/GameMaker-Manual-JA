ネットワーク作成ソケット

この関数は、ネットワーク上で通信するための新しいクライアントソケットを作成するために使用されます。ソケットの種類を定義し(以下の定数リストを参照)、そのソケットの一意なIDを返します(そのソケットに対する以降のすべての関数呼び出しで使用する必要があります)。

ヒントIP "127.0.0.1" を使って、ゲームを実行しているのと同じデバイスに接続し直すことができます。

構文です。

network_create_socket(type);

論証 タイプ 商品説明
type ソケットタイプ一定 The type of socket connection to create (see the constants listed above).

リターンです。

Network Socket ID

例

if os_browser == browser_not_a_browser
{
    client = network_create_socket(network_socket_tcp);
    network_connect( client, "192.134.0.1", 6510 );
}
else
{
    client = network_create_socket(network_socket_ws);
    network_connect_raw_async( client, "192.134.0.1", 6520 );
}

上記のコードは、ゲームがブラウザ上で実行されているかどうかをチェックし、新しいTCPまたはWebソケットを作成してから、それを介して与えられたIPアドレスと与えられたポートに接続しようとします。
