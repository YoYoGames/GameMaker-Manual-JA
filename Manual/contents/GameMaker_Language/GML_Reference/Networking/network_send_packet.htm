ネットワーク送信パケット

この関数を使うと、ネットワークを通じてデータの「パケット」を送信することができます。この関数は接続するソケット IDを受け取り、次に送信するデータを格納するバッファ ID(buffers に関するより詳しい情報はReference - Buffers を参照)、最後にデータパケットのサイズ (バイト単位) を指定する必要があります。この関数で送信されたパケットは、データを受け取るGameMakerのゲームがパケットを正しく「分割」できるようにフォーマットされ、送信したデータのバイト数、または送信に失敗した場合は0未満の数値が返される。この関数が返す送信データの最終サイズには、さらに12バイトのGameMaker ヘッダー情報が含まれることに注意する必要がある。

構文です。

network_send_packet(socket, bufferid, size);

論証タイプ 商品説明
socket The id of the socket to use.
bufferid The id of the buffer to get the data from.
size The size (in bytes) of the data.

リターンです。

Real

例

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

上記の内容は、あらかじめ保存しておいたbuffer のデータをメモリにロードし、変数 "buff" に格納するbuffer id を返します。この完全なbuffer は、変数 "sock "で識別されるソケットを使用してネットワーク上でパケットとして送信されます。
