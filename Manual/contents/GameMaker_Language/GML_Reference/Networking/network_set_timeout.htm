network_set_timeout

この関数は、指定されたソケットを介してserver との間でデータの読み書きを行う際のタイムアウトを設定することができます。タイムアウトはいかなる種類のイベントも発生させないので、アラームなどを使ってタイムアウトを自分で処理する必要があることに注意してください。 この値はデータの送受信にのみ影響しますので、接続タイムアウト値を変更したい場合は、関数 network_set_config().

構文です。

network_set_timeout(socket, read_timeout, write_timeout);

論証タイプ 商品説明
socket The id of the socket to use.
read_timeout The milliseconds in which a transfer from a server will timeout.
write_timeout The milliseconds in which a transfer to a server will timeout.

リターンです。

例

network_set_timeout(sock, 3000, 3000);

上記のコードでは、変数 "sock "にインデックスされたソケットを介したデータの読み書きのタイムアウトを3秒に設定します。
