network_send_broadcast

この機能を使用すると、buffer からのデータを IP アドレスの範囲にローカルにブロードキャストすることができます (buffers の詳細については、参考 - バッファを参照してください)。この範囲は、server を実行しているデバイスの範囲に限定されます。例えば、デバイスの IP が 92.168.11.130 である場合、データは 92.168.11.* の範囲にブロードキャストされます。この関数は、送信されたデータのバイト数を返し、送信に失敗した場合は0より小さい値を返します。

この機能は UDP で使用した場合のみ動作します。 は TCP で、クライアントは UDP でクライアントソケットを作成する必要があります。server network_create_socket_ext()serverでUDPクライアントソケットを作成する必要があります。

HTML5ターゲットで動作するプロジェクトで使用する場合は、本機能は動作しません。

構文です。

network_send_broadcast(socket, port, bufferid, size);

論証 タイプ 商品説明
socket ネットワークソケットID The id of the socket to use.
port リアル The port that the server will use.
bufferid バッファID The id of the buffer to get the data from.
size リアル The size (in bytes) of the data.

リターンです。

Real

例

buffer_seek(broadcast_buffer, buffer_seek_start, 0);
buffer_write(broadcast_buffer, buffer_string, global.server_name);
network_send_broadcast(server, 6511, broadcast_buffer, buffer_tell(broadcast_buffer));

上記のコードでは、現在のstring の名前server (「global.ServerName」に格納) を、id が「broadcast_buffer」のバイナリbuffer に書き込んでいます。このデータは、ポート6511にIPの範囲（ブロードキャストベースの範囲として、現在デバイスIPが暗示されている）にローカルにブロードキャストされます。
