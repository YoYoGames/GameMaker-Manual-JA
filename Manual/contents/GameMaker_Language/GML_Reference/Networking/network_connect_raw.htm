ネットワークコネクトロー

この関数を使うと、server に接続するためのリクエストを送ることができます。 この関数は、接続するためのソケットID(参照)を受け取り、接続先のIPアドレス(参照)と接続するためのポートを要求します。 network_create_socket()) を受け取り、接続先のIPアドレス (string) と接続するポートを要求します。接続に失敗した場合は、0未満の値が返されます。この関数と network_connect()との違いは、この関数は任意のserver に接続でき、生データには何もしないことです。つまり、server 側でプロトコルを自分で実装しなければならないのです。デフォルトでは、この関数は同期型であることに注意してください。つまり、接続が行われるときに、ゲームが「ハング」するように見えるかもしれません。接続のタイムアウト値を設定するか、あるいは、関数 network_set_config()を使うか、あるいは代わりに関数 network_connect_async_raw()を使うこともできます。

注意HTML5ではこの関数は使用できません。WebSocketの場合は、Async関数を使用してください。

構文です。

network_connect_raw(socket, url, port);

論証 タイプ 商品説明
socket ネットワークソケットID The id of the socket to use.
url 文字列 The URL or IP to connect to (a string).
port リアル The port to connect to.

リターンです。

Real or Network Socket ID

例

client = network_create_socket(network_socket_tcp);
network_connect_raw(client, "www.macsweeneygames.com", 6510);

上記のコードでは、新しいTCPソケットを作成し、そのソケットを通して指定されたURLのポート6510に「生の」接続を試みています。
