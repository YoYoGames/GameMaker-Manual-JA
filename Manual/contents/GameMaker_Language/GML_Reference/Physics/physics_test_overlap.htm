物理テストのオーバーラップ

この関数は、物理的なボディ（つまり、インスタンスの固定具）が回転してroom の与えられた位置に置かれたときに重なるかどうか、あるいは重なるかどうかをチェックするために使用できます。「angle」引数は、チェックされる位置で呼び出したインスタンスが持つ（あるいは持つ予定の）回転角度で、「obj」引数は単一のインスタンス ID、およびobject インデックス、あるいは、その両方を使用できます。 キーワードのすべて、またはその他を指定します。

構文です。

physics_test_overlap(xpos, ypos, angle, obj);

論証 タイプ 商品説明
xpos リアル The x position in the room to check
ypos リアル The y position in the room to check
angle リアル The angle to check (of the calling instance)
obj オブジェクトのアセットまたはインスタンスID The object to check for

リターンです。

Boolean

例

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
    alarm[0] = room_speed;
    ignited = true;
}

上記のコードは、物理的な固定具の重なりがある位置をチェックし、重なりがあれば変数とアラームをセットするものです。
