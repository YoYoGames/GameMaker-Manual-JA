physics_joint_delete

物理表現を持つ2つのインスタンスがジョイントによって結合されると、これはいつでも再び削除することができます。通常は、2つのインスタンスのうち1つが破壊されたときや、room が終了したときに自動的に行われますが、手動で行いたい場合もあります。そのような場合は、この機能を使うことになります。

また、ギアジョイントの一部であるインスタンスが破壊されるときは、ギアの形成に関与したどのインスタンスよりも先に、この関数を使用してギアジョイントを削除する必要があります（ただし、残りのジョイントは自動的に削除されます）、例えばインスタンスの破壊イベントなどで。

構文です。

physics_joint_delete(joint)

論証 タイプ 商品説明
joint 物理学ジョイントID The index of the joint that you wish to delete

リターンです。

N/A

例

var reactionForceX, reactionForceY, reactionForce;
if (shipJoint)
{
    _reaction_force_x = physics_joint_get_value(ship_joint, phy_joint_reaction_force_x);
    _reaction_force_y = physics_joint_get_value(ship_joint, phy_joint_reaction_force_y);
    _reaction_force = sqrt((_reaction_force_x + _reaction_force_x) + (_reaction_force_y + _reaction_force_y));
    if (_reaction_force > 2)
    {
        physics_joint_delete(ship_joint);
        ship_joint = -1;
    }
}

上記のコードでは、変数「shipJoint」がジョイントのインデックスを保持しているかどうかをチェックし、保持している場合は、2つの定数を使用してそのジョイントにかかる力を計算します。最後に、力の合計が2より大きい場合、そのジョイントは削除されます。
