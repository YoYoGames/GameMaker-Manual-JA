physics_joint_wheel_create

ホイールジョイントは、ピストンとリボルートジョイントを組み合わせたもので、自動車のショックアブソーバーにホイールを取り付けるようなものです。最初のインスタンスはジョイントの「ボディ」として機能し、2番目のインスタンスは「ホイール」として機能し、ジョイント軸の周りに自由に動くことができます。軸ベクトルも設定できます。この軸ベクトルは、ジョイントが「バネ」として機能する仮想線で、車輪が障害物に遭遇すると、この軸に沿って上下にバウンドします（ピストンジョイントのようなものです）。次の画像は、これを示しています。

モーターを有効にすると、2番目のインスタンスがアンカー位置の周りを回転します。希望のモーター速度を達成するために使用する最大モータートルク（N/m）と、モーターが回転する速度を設定することができます。ホイールジョイントにも軸があるので、ジョイントの振動数（Hz）と減衰比を設定できます。これらの値を微調整する必要があるかもしれませんが、最初は小さな値から始めて、望む効果が得られるまで値を増やしていくことをお勧めします。

他のジョイントと同様に、「col」の値を true に設定すると、2つのインスタンスは相互に作用して衝突することができますが、衝突イベントがある場合のみです。しかし、 false に設定すると、何があっても衝突することはありません。

構文です。

physics_joint_wheel_create(inst1, inst2, anchor_x, anchor_y, axis_x, axis_y, enableMotor, max_motor_torque, motor_speed, freq_hz, damping_ratio, col)

論証 タイプ 商品説明
inst1 インスタンスID The first instance to connect with the joint
inst2 インスタンスID The second instance to connect with the joint
anchor_x リアル The x coordinate where the joint is anchored, within the game world
anchor_y リアル The y coordinate where the joint is anchored, within the game world
axis_x リアル The x component of the wheel axis vector
axis_y リアル The y component of the wheel axis vector
enableMotor ブーリアン Whether the motor should be active (true) or not (false)
max_motor_torque リアル Sets the maximum motor torque used to achieve the desired motor speed (in Newtons per meter)
motor_speed リアル This is the speed at which the motor should rotate
freq_hz リアル This is the oscillation frequency for the joint, in hertz
damping_ratio リアル This damping ratio for the joint
col カラー Whether the two instances can collide (true) or not (false)

リターンです。

Physics Joint ID

例

var mainFixture = physics_fixture_create();
physics_fixture_set_box_shape(mainFixture, 128, 32);
var wheelFixture = physics_fixture_create();
physics_fixture_set_circle_shape(wheelFixture, 32);
var o_id = instance_create_layer(x+25, y + 100, "Instances", obj_Wheel);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(wheelFixture, o_id);
physics_joint_wheel_create(id, o_id, o_id.x, o_id.y, 0, 1, true, 10, 1, 15, 6, false);
physics_fixture_delete(mainFixture);
physics_fixture_delete(wheelFixture);

上記のコードでは、2つの新しいフィクスチャを作成し定義しています。これらは、呼び出し元のインスタンスと、それらをつなぐホイールジョイントが適用される前に作成された新しいインスタンスにバインドされ、その後フィクスチャはメモリから削除されます。
