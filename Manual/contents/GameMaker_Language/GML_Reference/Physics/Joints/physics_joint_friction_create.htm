物理接合部摩擦の作成

摩擦ジョイントは、物理シミュレーションの他のジョイントとは少し異なり、作成された接続がインスタンスの位置や動きを拘束するのではなく、その速度と回転を拘束するものです。これは、力とトルクの最大入力値を取り、それを2番目の固定具に適用して、速度と角運動量を最初のインスタンスのものと同じ値にすることで機能します。つまり、静止しているインスタンスと動いているインスタンスがあり、それらを摩擦ジョイントで接続すると、動いているインスタンスは徐々に減速し、自分も静止することになります。両方のインスタンスが動いている場合、2番目のインスタンスの移動速度は1番目のインスタンスの移動速度に一致するように変更されます。

他のほとんどのジョイントと同様に、結合するインスタンスと、ジョイントを作成する位置をroom 内で指定します。次に、最大力と最大トルク（方向と回転の摩擦）を指定します。「col」値を true に設定すると、2 つのインスタンスは相互に作用して衝突することができますが、衝突イベントがある場合のみです。しかし false に設定すると、何があっても衝突することはありません。

構文です。

physics_joint_friction_create(inst1, inst2, anchor_x, anchor_y, max_force, max_torque, col)

論証 タイプ 商品説明
inst1 インスタンスID The first instance to connect with the joint
inst2 インスタンスID The second instance to connect with the joint
anchor_x リアル The x coordinate for the joint, within the game world
anchor_y リアル The y coordinate for the joint, within the game world
max_force リアル The maximum frictional force that will be applied
max_torque リアル The maximum rotational force that will be applied
col ブーリアン Whether the two instances can collide (true) or not (false)

リターンです。

Physics Joint ID

例

var mainFixture, o_id;
mainFixture = physics_fixture_create();
physics_fixture_set_circle_shape(mainFixture, sprite_get_width(sprite_index) / 2);
o_id = instance_create_layer(x+300, y, "Instances", obj_Rudder);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(mainFixture, o_id);
physics_joint_friction_create(id, o_id, x, y, 10, 0.3, true);
physics_fixture_delete(mainFixture);

上記のコードでは、固定具を作成し、それを2つのインスタンスにバインドしてから、フリクションジョイントを使って両者を接続します。
