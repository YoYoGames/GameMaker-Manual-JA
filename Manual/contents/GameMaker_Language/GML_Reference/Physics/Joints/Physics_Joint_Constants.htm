物理学的接合定数

GameMaker GML には、物理世界のフィクスチャ間のジョイントに特化した定数が多数含まれています。これらは、物理シミュレーションの実行中にリアルタイムで情報を設定したり取得したりするために、多くの異なるジョイント関数と組み合わせて使用することができます。しかし、これらを呼び出すと複雑な計算が行われるので、必要なときだけ注意して使用する必要があります。また、多くは特定のタイプのジョイントに固有のものであることに注意してください。また、これらの値は適切な関数ですべて取得できますが、設定できるのは読み取り専用でないとマークされているものだけであることに注意してください。

一般に、これらの定数は、以下の関数と組み合わせて使用される。

  • physics_joint_get_value
  • physics_joint_set_value

以下の定数は、利用可能なすべてのジョイントタイプに適用することができます。

物理学共同定数
定数 商品説明 読み取り専用
phy_joint_anchor_1_x 部屋内のジョイントの最初のアンカーポイントのx座標 Yes
phy_joint_anchor_1_y 部屋内のジョイントの最初のアンカーポイントのy座標 Yes
phy_joint_anchor_2_x 室内にあるジョイントの2つ目のアンカーポイントのx座標 Yes
phy_joint_anchor_2_y 室内にある関節の2つ目のアンカーポイントのy座標 Yes
phy_joint_reaction_force_x これは、xのアンカー位置で関節の第2インスタンスに加わっている反力です Yes
phy_joint_reaction_force_y これは、yアンカー位置で関節の第2インスタンスに加わっている反力です Yes
phy_joint_reaction_torque これは、アンカー位置で関節の第2インスタンスに加えられているトルクである Yes


これらの定数は、モーターが付属している関節（レボリュート、プリズム、ホイール）のためのものです。

物理学共同定数
定数 商品説明 読み取り専用
phy_joint_max_motor_force 最大モーター力のジョイント作成時に指定された値 No
phy_joint_max_motor_torque 最大モータトルクをジョイント作成時に指定した値 No
phy_joint_motor_force 現在のモーター力 Yes
phy_joint_motor_speed 現在のモータ回転数 No
phy_joint_motor_torque 現在のモータートルク Yes


レボリュートジョイントの場合、以下の定数（モーター定数が追加されている場合はモーター定数も）を使用することができます。

物理学共同定数
定数 商品説明 読み取り専用
phy_joint_angle 関節の2つのアンカーポイントを結ぶ線が作る角度。物理世界の座標を使用して計算されます
( GameMaker room の座標ではありません)単位はラジアンです。		 はい
phy_joint_angle_limits ジョイントの角度制限を有効または無効にします。有効にする場合は true に、無効にする場合は false に設定します。 いいえ
phy_joint_upper_angle_limit ジョイントの角度の上限を度数で指定します。 いいえ
phy_joint_lower_angle_limit ジョイントの下限角度を度数で指定します。 いいえ


角柱のジョイントの場合、以下の定数を使うことができます。

物理学共同定数
定数 商品説明 読み取り専用
phy_joint_translation アンカーX/Y座標とローカルX/Y座標の距離を取得します。 Yes
phy_joint_speed 現在の関節の移動速度です。 Yes


ディスタンス、ウェルド、ホイールジョイントには、以下の定数が使えます（プーリージョイントの定数も同様）。

物理学共同定数
定数 商品説明 読み取り専用
phy_joint_damping_ratio 減衰比は無次元で、関節の「バネ性」を定義する。この定数の値は通常0から1の間ですが、もっと大きくすることも可能で、1の場合、減衰は臨界に達し、すべての振動が消滅するはずだという意味です。 No
phy_joint_frequency これは関節の発振周波数をヘルツ単位で返す（設定する）もので、通常、周波数は関数 physics_world_update_speed(). No
phy_joint_length_1 これは、最初のローカル x/y 座標から最初のアンカー x/y 座標までのジョイントの長さを返します（距離ジョイントのみ、読み出しのみ可能）。 Yes
phy_joint_length_2 これは、2番目のローカルX/Y座標から2番目のアンカーX/Y座標までのジョイントの長さを返します（距離ジョイントのみ、書き込みのみ可） No
