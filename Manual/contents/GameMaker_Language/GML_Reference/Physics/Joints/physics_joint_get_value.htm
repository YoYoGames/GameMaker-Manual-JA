物理ジョイントの値を取得する

あらかじめ定義された定数を使うことで、GameMakerに任意のジョイントの状態について、様々なことを教えてもらうことができます。これは非常に便利で、ジョイントを削除したり、その時の必要に応じてインスタンスの動作を変更したりすることができます。この関数で使用できる定数はいくつかあり、ここにあります。しかし、これらを呼び出すと複雑な計算が行われるので、必要なときだけ注意して使用する必要があり、多くは特定のタイプのジョイントに固有のものであることに注意してください。

プロパティが存在しない場合（例えば、プーリージョイントのプリズム移動をチェックする場合）には、0が返されます。

構文です。

physics_joint_get_value(joint, value)

論証 タイプ 商品説明
joint 物理学ジョイントID The index of the joint that you wish to test
value 物理学共同定数 The constant for the joint property that you wish to test

リターンです。

Real

例

if shipJoint
{
    var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
    var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
    var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
    if (reactionForce > 2)
    {
        physics_joint_delete(shipJoint);
        shipJoint = -1;
    }
}

上記のコードでは、変数「shipJoint」がジョイントのインデックスを保持しているかどうかをチェックし、保持している場合は、2つの定数を使用してそのジョイントにかかる力を計算します。最後に、力の合計が2より大きい場合、そのジョイントは削除されます。
