physics_joint_set_value

特定のジョイントプロパティは、ジョイント作成後でも変更・設定することが可能です。この機能で使用できる定数はいくつかあり、ここに記載されています。物理 ジョイント定数(Physics Joint Constants)。

構文です。

physics_joint_set_value(joint, field, value)

論証 タイプ 商品説明
joint 物理学ジョイントID The index of the joint that you wish to change
field 物理学共同定数 The constant for the joint property that you wish to change
value リアル The new value for the joint property

リターンです。

N/A

例

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
{
    physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
}

上記のコードでは、ジョイントの最大モータトルクが2未満に設定されているかどうかを確認し、もし設定されていれば、それを2に設定しています。
