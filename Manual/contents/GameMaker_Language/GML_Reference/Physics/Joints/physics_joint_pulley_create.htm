物理ジョイントのプーリー作成

プーリージョイントは、物理世界内の2つのインスタンスを、一方を動かすと他方の動きに直接影響を与えるように接続するために使用します。これらのジョイントは、まず世界空間の2点に固定され、それぞれがインスタンスの重心に接続されます。これは、ローカルアンカーの x/y 座標をインスタンスの原点に対して設定することで変更できます。つまり、インスタンスから物理世界のアンカーポイントへの実際の接続は、原点以外の場所にオフセットすることができます。次の画像は、これを示しています。

inst2を見てみると、そのローカルアンカーポイントは、原点から20ピクセル左にオフセットされていることがわかります（覚えておいてください！）。ワールドアンカーポイントはroom 座標を使って定義され、ローカルアンカーポイントはインスタンスの原点を基準にして定義されます)。プーリージョイントに比率を指定することもできます。これは、片方またはもう片方がより速く動くようにするもので、1 以下の値では最初のインスタンスがより速く動き（例えば 0.5 では 2 倍の速さになります）、1 以上の値では次のインスタンスがより速く動きます（例えば 2 では 2 倍の速さになります）。

他のジョイントと同様に、「col」の値を true に設定すると、2つのインスタンスは相互に作用して衝突することができますが、衝突イベントがある場合のみです。しかし、 false に設定すると、何があっても衝突することはありません。

構文です。

physics_joint_pulley_create(inst1, inst2, w_anchor1_x, w_anchor1_y, w_anchor2_x, w_anchor2_y, l_anchor1_x, l_anchor1_y, l_anchor2_x, l_anchor2_y, ratio, col)

論証 タイプ 商品説明
inst1 インスタンスID The first instance to connect with the joint
inst2 インスタンスID The second instance to connect with the joint
w_anchor1_x リアル The x coordinate where the first part of joint is anchored, within the game world
w_anchor1_y リアル The y coordinate where the first part of the joint is anchored, within the game world
w_anchor2_x リアル The x coordinate where the second part of the joint is anchored, within the game world
w_anchor2_y リアル The y coordinate where the second part of the joint is anchored, within the game world
l_anchor1_x リアル The local x coordinate where joint 1 is anchored to the first instance
l_anchor1_y リアル The local y coordinate where joint 1 is anchored to the first instance
l_anchor2_x リアル The local x coordinate where joint 2 is anchored to the second instance
l_anchor2_y リアル The local y coordinate where joint 2 is anchored to the second instance
ratio リアル Set the velocity ratio between two instances
col ブーリアン Whether the two instances can collide (true) or not (false)

リターンです。

Physics Joint ID

例

var t_fix, inst1, inst2;
t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, 20);
inst1 = instance_create_layer(150, room_height - 90, "Instances", obj_Block);
inst2 = instance_create_layer(300, room_height - 90, "Instances", obj_block);
physics_fixture_bind(t_fix, inst1);
physics_fixture_bind(t_fix, inst2);
physics_joint_pulley_create(inst1, inst2, 150, room_height - 140, 300, room_height - 140, 0, 0, 0, 0, 2, 0);
physics_fixture_delete(t_fix);

上記のコードでは、新しいフィクスチャを作成して定義し、「obj_Block」のインスタンスを2つ作成して、このフィクスチャを2つバインドしています。次に、これらのインスタンスの間にプーリージョイントを定義します。ジョイントのオフセットはなく、比率は 2:1 です（インスタンス 2 の方が速く動くという意味です）。プーリーでは、2つのインスタンスの間に衝突は起こりません。
