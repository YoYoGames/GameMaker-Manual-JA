物理の粒子密度

この関数で、物理シミュレーションの粒子の電流密度を知ることができます（この値を設定するには physics_particle_set_density()).

構文です。

physics_particle_get_damping()

リターンです。

例

if (physics_particle_get_density() < 1)
{
    physics_particle_set_density(physics_particle_get_density() + 0.01);
}

上記のコードは、システム内の全パーティクルの現在の密度値をチェックし、それが1より小さい場合は、それに0.01を追加することになります。
