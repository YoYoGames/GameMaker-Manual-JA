physics_particle_get_damping

この関数により、物理シミュレーションのパーティクルに対する現在の線形ダンピングを知ることができます（この値を設定するには physics_particle_set_damping()).

構文です。

physics_particle_get_damping()

リターンです。

例

if (physics_particle_get_damping() < 1)
{
    physics_particle_set_damping(physics_particle_get_damping() + 0.01);
}

上記のコードは、システム内の全パーティクルの現在のダンピング値をチェックし、それが1より小さい場合は、それに0.01を追加することになります。
