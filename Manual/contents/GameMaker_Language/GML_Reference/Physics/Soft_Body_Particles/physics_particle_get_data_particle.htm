物理粒子データ

この関数は、物理シミュレーション中の一つの粒子に関する様々な情報を、与えられたflags をチェックすることで返します。粒子のインデックス（ID）は関数 physics_particle_create()で、使用されるbuffer は、以前に関数 buffer_create().また、型は "grow "でなければならず、そのサイズは期待される戻りデータのサイズとほぼ同じでなければならない。flags 自体は以下に示す定数で設定され、ビット単位または"|" を使って、目的の情報を得るために一つのflag 値を作成することになる。

定数 商品説明 データ型
phy_particle_data_flag_typeflags パーティクルのflags の値です。 buffer_u32
phy_particle_data_flag_position パーティクルのx、y位置。 2 x buffer_f32
phy_particle_data_flag_velocity 水平・垂直方向の速度です。 2 x buffer_f32
phy_particle_data_flag_colour 色とアルファ値（16進数）です。 buffer_f32
phy_particle_data_flag_category パーティクルカテゴリ（パーティクル作成時に定義したもの）。 buffer_u32

構文です。

physics_particle_get_data_particle(ind, buffer, flags)

論証 タイプ 商品説明
ind 物理学パーティクルID The index (ID) of the particle to get the data from.
buffer バッファID The (previously created) buffer to use to store the data.
flags 物理 粒子データFlag 定数(s) The flags to use to extract data about specific particle types.

リターンです。

N/A

例

var part = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1)
var flags = phy_particle_data_flag_position | phy_particle_data_flag_velocity;
var buffer = buffer_create(16, buffer_grow, 4);
physics_particle_get_data_particle(part, buffer, flags);
px = buffer_read(buffer, buffer_f32);
py = buffer_read(buffer, buffer_f32);
pvelx = buffer_read(buffer, buffer_f32);
pvely = buffer_read(buffer, buffer_f32);
buffer_delete(buffer);

上記のコードでは、変数 "part "でインデックスされた粒子の位置と速度を取得し、そのデータをいくつかの変数に格納しています。
