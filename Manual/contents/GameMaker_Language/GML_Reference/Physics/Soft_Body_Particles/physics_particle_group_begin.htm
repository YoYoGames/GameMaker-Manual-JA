physics_particle_group_begin

room粒子群とは、ある形を作るのに必要な粒子をまとめて作るもので、flags によって様々な特性を持ったソフトボディのシミュレーションを行うことができます。フラグ "はパーティクル定数の組み合わせの戻り値です（関数 physics_particle_create()は，以下のグループ定数の組み合わせであり，"groupFlags" はflags を含まない場合は 0 となります。

物理学 粒子群Flag 定数
定数 商品説明
phy_particle_group_flag_solid 固体粒子群は、他の什器がその内部に留まることを防ぐ。万が一、貫通した場合でも、固体粒子群がその表面へ押し戻すため、固体粒子群は特に強い反発力を持っています。
phy_particle_group_flag_rigid リジッドパーティクルグループとは、他のフィクスチャと衝突しても形状が変化しないパーティクルのことです。

これらのグループflags はビットマスキングを使用して最終的な出力値を作成し、それをチェックして、パーティクルflags によって設定されるものとは別に、グループのさまざまな基本的なプロパティを設定するのです。ビットワイズまたは"|"を使用して適切なビットをマスクすることで、これらの定数のどれも、一方または両方を使用できます（以下の例で示すように）。

flags 以外に、room でパーティクルグループを作成する位置（指定した位置が完成したグループの重心になります）、水平・垂直方向の初期速度、グループの角速度を設定できますので、移動・回転するグループを作成することも可能です。

また、グループ内の全パーティクルの初期色とアルファ値を設定することができ、その値は関数 physics_particle_draw().その後、グループ内のパーティクルの凝集力を設定します。これは、パーティクル同士がどれだけ強く「くっつく」かを別の言い方で表現したものです。強さは 0 から 1 の間の値で、0 は凝集力がない状態、1 は最大凝集力を意味します。

最後に、パーティクルグループのカテゴリーを指定します。この値は任意の整数値で、似たようなプロパティを持たせたいパーティクルグループに与えます。この値は、後でその特定のカテゴリに対してflags を設定したり、その他様々なことに利用することができます。

注：カテゴリーには、シミュレーションで予約され、他の機能（描画など）ですべてのカテゴリーを選択するために使用される0以外の任意の整数値を使用することができます。

注意すべきは、この関数を呼び出しても、物理学room にグループが作成されないことです。この関数は単に処理を開始し、グループの基本的なパーティクルプロパティを設定するだけです。実際にパーティクルグループを作成するには、それにフォームを与え（下の例参照）、それからこの関数 physics_particle_group_end()この時点で作成されます。

注：設定したパーティクルタイプflags はパーティクルグループの凝集力にも影響します。例えば、パーティクルを水タイプとしてflag した場合（例）、グループが他のフィクスチャと衝突するとバラバラになりますが、パーティクルをスプリングタイプとしてflag した場合、衝突しても凝集力が維持されます。

構文です。

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

論証 タイプ 商品説明
flags 物理学 粒子Flag 定数(s) The flags to set on the particle.
groupflags 物理学 粒子群Flag 定数(s) The group flags to set on the particle group (see table at the top).
x リアル The x position to create the particle group at.
y リアル The y position to create the particle group at.
ang リアル The angle (in degrees) to rotate the group to on create.
xv リアル The initial horizontal velocity.
yv リアル The initial vertical velocity.
ang_velocity リアル The initial angular velocity.
col カラー The base colour to use for the particle group.
alpha リアル The base alpha to use for the particle group.
strength リアル Set the cohesion strength between particles in the group.
category リアル The user defined category to which the group belongs.

例

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

flags 上記のコードでは、パーティクルの種類とパーティクルグループのプロパティを変数に格納し、それらを使ってマウス位置で半径100pxの円形パーティクルグループを作成しています。
