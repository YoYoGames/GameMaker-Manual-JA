physics_particle_set_gravity_scale

物理シミュレーションのパーティクルに対する重力スケールファクターを設定することができる機能です。この機能は、物理シミュレーションの不安定性を防ぐために設計されており、特に、予測できない振る舞いをする可能性がある非常に小さな粒子を使用する場合 (爆発などのシナリオで（運動量保存を破るなど）。重力スケールを小さくしてこれらのパーティクルの速度を遅くすることで、挙動を安定させることができます。この関数はグローバルなもので、重力スケールを変更するのは は、変更後に作成された新しい粒子だけでなく、シミュレーションに既に存在する粒子にも適用されます。

注：1ステップあたりの更新反復回数を調整する（関数 physics_world_update_iterations() は、パーティクルに対する重力の影響にも影響を与えることができます。イテレーションサイズを大きくすると、重力に対する耐性が強くなります。

構文です。

physics_particle_set_gravity_scale(scale)

論証タイプ 商品説明
scale The scaling factor to be applied to gravity for all particles.

リターンです。

例

physics_particle_set_radius(15);
physics_particle_set_density(0.5);
physics_particle_set_damping(1);
physics_particle_set_gravity_scale(1);

上記のコードでは、シミュレーション内のすべてのパーティクルの基本プロパティを設定します。
