物理学上の粒子の半径を取得する

この関数を使うと、物理シミュレーションのパーティクルの現在の半径（ピクセル単位）を知ることができます（この値を設定するためには physics_particle_set_radius()).

構文です。

physics_particle_get_radius()

リターンです。

例

if (physics_particle_get_radius() < 32)
{
    physics_particle_set_radius(physics_particle_get_radius() + 1);
}

上記のコードは、シミュレーション中の粒子の現在の半径をチェックし、それが32未満であれば、そのサイズを1だけ大きくします。
