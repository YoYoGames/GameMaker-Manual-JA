physics_particle_set_max_count

物理シミュレーションで許容されるパーティクル数の合計を設定する関数です。この値を0に設定すると、作成されるパーティクルに上限がなくなり、他の値に設定すると パーティクルの総数がこの値と同じになります。なお、シミュレーションで作成できるパーティクルの最小数は128個なので、これ以下の値を設定しても（0に設定しない限り）効果はありません。

構文です。

physics_particle_set_max_count(count)

論証タイプ 商品説明
count The maximum number of particles to permit.

リターンです。

例

physics_particle_set_max_count(500);

上記のコードでは、物理シミュレーションのパーティクルキャップを500に設定します。
