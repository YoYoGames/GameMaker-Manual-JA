物理粒子削除領域ポリゴン

この関数により、現在のroom の物理シミュレーションから、定義されたポリゴン領域の境界内にあるすべてのパーティクルを削除（除去）することができます。この関数は、（あらかじめ作成された） ds_list()このリストの偶数番目の位置が x 座標、奇数番目の位置が y 座標です。例えば、三角形の場合、リストには 6 つのエントリがあり、エントリ 0、2、4 が x 座標、1、3、5 が y 座標となります。このポリゴンを使って、定義された領域内にあるパーティクルをすべて削除します。

なお、定義される多角形は少なくとも3点、多くても8点でなければならない。

構文です。

physics_particle_delete_region_poly(pointList)

論証タイプ 商品説明
pointList A ds_list of points to use to create the polygon.

リターンです。

例

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
    ds_list_add(p_list, mx[i]);
    ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

上記のコードは、 DS List に追加されたポイントによって定義されるポリゴン領域で見つかったすべてのパーティクルを削除します。
