物理学_粒子描画

この関数は、ユーザー定義のカテゴリー値を共有し、かつマスクに設定された値と一致するsprite を持つすべてのパーティクルの位置にflags を描画します。マスクの値は、パーティクルを作成するときにflags を定義するのと同じ方法で定義します（関数 physics_particle_create()すなわち、ビット単位または"|"を使用して、各flag の適切なビットをマスクします。

注意：この関数は、ユーザーカテゴリが同じであれば、物理シミュレーションのすべてのパーティクルを、個々のパーティクルでもグループでも、マスクのものと一致するflags で描画します。

描画されたパーティクルは、作成時に設定した色とアルファ値が使用され、パーティクルの角運動量に合わせて回転します。しかし、sprite を拡大縮小したり、その他の描画されたプロパティを変更したい場合は、関数 physics_particle_draw_ext().

構文です。

physics_particle_draw(typemask, category, sprite, subimg)

論証 タイプ 商品説明
typemask 物理学 粒子Flag 定数(s) The mask to compare the particles flags with.
category リアル The user defined category to draw (use 0 to draw all categories).
sprite スプライトアセット The sprite to use for each particle.
subimg リアル The image_index 描画するスプライトの

リターンです。

N/A

例

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw(flags, 2, sprBlob, 0);

上記のコードは、与えられたマスクと同じsprite を持つユーザー定義のカテゴリー "2" のすべての粒子についてflags を描画します。
