物理パーティクル作成

この関数を使うと、ゲーム内の任意の場所にパーティクルを1つ作成することができますroom 、特定のflags とプロパティを設定します。フラグ "は以下の定数の組み合わせの戻り値です。

物理学 粒子Flag 定数
定数 商品説明
phy_particle_flag_water ソフトボディパーティクルのデフォルトのプロパティです。
phy_particle_flag_zombie ゾンビパーティクルは、このように他のすべてのフラグを立てて1ステップで破壊されるパーティクルです。
phy_particle_flag_wall これはパーティクルを静的と定義し、本質的に物理シミュレーションで不動のobject 、何が衝突しても固定された位置に留まるように作成します。密度を0に設定するよりも、このflag を使用する必要があります。
phy_particle_flag_spring スプリングパーティクルは、バネのように互いにくっついたような効果を生み出します。このパーティクル（flag ）で作られたパーティクルは、ペアで「接続」され、それぞれのパーティクルは、作成時に最も近くにいたパーティクルと接続されます。一旦対になった粒子は「相手」を変えず、外力に引っ張られるほど、その外力を取り除いた時に衝突する力が大きくなる。なお、対になった粒子がどんなに離れても、粒子間の結合が切れることはない。
phy_particle_flag_elastic 弾性体パーティクルは、物理シミュレーションで他の剛体と衝突すると変形したり、バウンドしたりすることがあります。
phy_particle_flag_viscous 粘性粒子とは、油のように「くっつきやすさ」「粘り強さ」を発揮する粒子のことです。粘性のある粒子は、より固まり、よりくっつきます。
phy_particle_flag_powder 粉体は、砂やホコリのような散乱効果を生み出します。
phy_particle_flag_tensile 引張粒子は、表面張力、つまり液体表面の張ったような曲面の効果を出すために使われます。例えば、水滴に見られるような表面張力を発生させるために使用されます。表面張力がなくなると、粒子は弾力性があるように跳ねますが、互いに引き合うようになります。その結果、粒子は弾みながらクラスターを形成する傾向があります。
phy_particle_flag_colourmixing カラーミキシングパーティクルは、衝突した他のパーティクルの色の一部を取り込みます。ただし、衝突する2つのパーティクルのうち1つだけが混色パーティクルの場合、もう1つのパーティクルは衝突前の色を保持します。

これらのflags はビットマスキングを使用して最終的な出力値を作成し、それをチェックして粒子の異なる基本特性を設定します（基本特性は常に phy_particle_flag_water のものです）。例えば、表面張力を持つ粘性液体をシミュレートしたい場合、以下の例に示すように、ビットワイズまたは「｜」を使用して適切なビットをflag します。このようにして、作成された個々のパーティクルごとに異なるプロパティ（グローバルプロパティ以外）を設定することができます。

flags 以外に、パーティクルを作成するroom の位置、水平・垂直方向の初期速度、初期色とアルファ、ユーザーカテゴリを設定することができます。色とアルファの値は、関数を使用してパーティクルを描画するときにのみ使用されます。 physics_particle_draw()しかし、カテゴリの値は他の複数の関数で使用される。この値は、似たような性質を持たせたい各パーティクルに与える任意の整数値で、後でその特定のカテゴリにflags を設定したり、その他様々なことに利用することができます。例えば、「水」と「油」のパーティクルを定義した場合、「水」としてフラグを立てたパーティクルにユーザーカテゴリ1を与え、「油」として作成したパーティクルにユーザーカテゴリ2を与え、後で一つのカテゴリをターゲットにしてflags を変更したり特定のデータを取得したりすることが可能になるのです。

注：カテゴリーには、シミュレーションで予約され、他の機能（描画など）ですべてのカテゴリーを選択するために使用される0以外の任意の整数値を使用することができます。

この関数はパーティクルの一意なインデックス（または ID）値を返します。この値は変数に格納することができ、パーティクルを直接参照する他の関数で使用できます。

構文です。

physics_particle_create(flags, x, y, xv, yv, col, alpha, category)

論証 タイプ 商品説明
flags 物理学 粒子Flag 定数(s) The flags to set on the particle.
x リアル The x position to create the particle at.
y リアル The y position to create the particle at.
xv リアル The initial horizontal velocity.
yv リアル The initial vertical velocity.
col カラー The base colour to use for the particle.
alpha リアル The base alpha to use for the particle.
category リアル The user defined category to which the particle belongs.

リターンです。

Physics Particle ID

例

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var x_vel = lengthdir_x(5, image_angle);
var y_vel = lengthdir_y(5, image_angle);
var p = physics_particle_create(flags, x, y, x_vel, y_vel, c_white, 1, 1);

上記のコードでは、パーティクルの種類と初速を設定する変数を作成し、それを使ってインスタンスの x/y 位置にパーティクルを作成しています。
