physics_particle_get_group_flags

この関数を使用すると、パーティクルグループのグループflags を取得することができます。 group の値は、この関数でパーティクルグループを作成したときに返された値です。 physics_particle_group_end()で、この関数は、現在設定されているflags を組み合わせた値を返します。

構文です。

physics_particle_get_group_flags(group)

論証 タイプ 商品説明
group 物理粒子グループID The particle group to get.

リターンです。

Real

例

var flags = physics_particle_get_group_flags(group, flags);
if flags != (phy_particle_group_flag_solid | phy_particle_group_flag_rigid)
{
    flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
    physics_particle_set_group_flags(group, flags);
}

上記のコードでは、flags の値を格納する変数を作成し、それを使って変数 "group" でインデックスされたグループのflags をチェックします。もし、同じでなければ、このflags を使ってグループが設定されます。
