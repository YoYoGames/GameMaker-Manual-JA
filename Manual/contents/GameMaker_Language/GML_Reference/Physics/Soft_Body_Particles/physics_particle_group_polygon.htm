物理粒子グループポリゴン

この関数は、作成中のパーティクルグループの形状を設定します。まず最初に関数 physics_particle_group_begin()で、その後に この関数は、ポリゴン形状を定義するためのおおよそのポイントを追加することをGameMakerに伝えるために使用します。 は physics_particle_set_radius()物理シミュレーションは、定義された形状にできるだけ多くのパーティクルを「はめ込もう」とするからです。

次に、ポリゴンの点を定義するために、関数 physics_particle_group_add_point()room での x/y 位置を設定します（最小 3、最大 8。 は凹形状として定義できるが、形状の内側の点は無視される)。最後に physics_particle_group_end()でパーティクルグループを作成し、room 。

構文です。

physics_particle_group_polygon()

リターンです。

例

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_polygon();
physics_particle_group_add_point(200, 200);
physics_particle_group_add_point(300, 300);
physics_particle_group_add_point(100, 300);
mLastGroup = physics_particle_group_end();

上記のコードでは、パーティクルの種類とパーティクルグループのプロパティ（flags ）を変数に格納し、それを使ってマウス位置の3辺を持つポリゴンパーティクルグループを作成しています。
