physics_particle_draw_ext

この関数は、ユーザー定義のカテゴリー値を共有し、かつマスクに設定された値と一致するsprite を持つすべてのパーティクルの位置にflags を描画します。マスクの値はパーティクルを作成するときにflags を定義するのと同じ方法で定義します（関数 physics_particle_create()すなわち、ビット単位または"|"を使用して、各flag の適切なビットをマスクします。

注意：この関数は、ユーザーカテゴリが同じであれば、物理シミュレーションのすべてのパーティクルを、個々のパーティクルでもグループでも、マスクのものと一致するflags で描画します。

描画されるパーティクルは、作成時に設定した色とアルファ値を無視し、代わりにこの関数で指定したスケール、角度、色、アルファ値のパラメータを使って描画されます。これは純粋にパーティクルの位置にsprite を描画するためのもので、パーティクルの挙動には一切影響を与えないことに注意してください。

構文です。

physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha)

論証 タイプ 商品説明
typemask 物理学 粒子Flag 定数(s) The mask to compare the particles flags with.
category リアル The user defined category to draw (use 0 to draw all categories).
sprite スプライトアセット The sprite to use for each particle.
subimg リアル The image_index 描画するスプライトの
xscale リアル The scale factor to apply to the sprite along the x axis.
yscale リアル The scale factor to apply to the sprite along the y axis.
ang リアル The angle (in degrees) to draw the sprite at.
col カラー The colour to blend with the sprite (default is c_white).
alpha リアル The alpha value to use when drawing (0 - 1).

リターンです。

N/A

例

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5);

sprite 上記のコードは、ユーザー定義のカテゴリー "2 "のすべてのパーティクルについて、与えられたマスクと同じflags 、両軸に沿って2でスケーリングされ、アルファが0.5であるものを描画します。
