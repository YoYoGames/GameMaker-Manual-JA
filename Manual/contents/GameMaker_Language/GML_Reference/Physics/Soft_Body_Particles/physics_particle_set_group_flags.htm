physics_particle_set_group_flags

この関数を使用すると、パーティクルグループのグループflags を変更することができます。 group の値は、関数 physics_particle_group_end()であり、flags は以下の定数の組み合わせの戻り値である。

定数 商品説明
phy_particle_group_flag_solid 固体粒子群は、他の什器がその内部に留まることを防ぐ。万が一、貫通した場合でも、固体粒子群がその表面へ押し戻すため、固体粒子群は特に強い反発力を持っています。
phy_particle_group_flag_rigid リジッドパーティクルグループとは、他のフィクスチャと衝突しても形状が変化しないパーティクルのことです。

これらのグループflags はビットマスキングを使用して最終的な出力値を作成し、それをチェックして、パーティクルflags によって設定されるものとは別に、グループのさまざまな基本的なプロパティを設定するのです。ビットワイズまたは"|"を使用して適切なビットをマスクすることで、これらの定数のどれも、一方または両方を使用できます（以下の例で示すように）。

構文です。

physics_particle_set_group_flags(group, flags)

論証 タイプ 商品説明
group 物理粒子グループID The particle group to set.
flags 物理学 粒子群Flag 定数(s) The flags to set on the particle group.

リターンです。

N/A

例

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

上記のコードでは、flags の値を格納する変数を作成し、それを使って変数 "group" でインデックスされたグループ内で作成されたすべてのパーティクルのflags を設定することになります。
