physics_pause_enable

room通常、物理シミュレーションが機能するためには、連続的でなければならず、停止や開始、インスタンスが突然ある場所から別の場所に移動することはできません。しかし、（例えば）デバイスOSが一時停止したようにシミュレーションを一時停止する必要がある場合があるので、この機能を使用することになります。flag が true に設定されるとシミュレーションを一時停止し、flag が false に設定されるまでそれ以上の物理計算が行われません。

注：この機能は、room のインスタンスをすべて無効化する場合に特に有効です。無効化しても、物理世界での計算とシミュレーションは継続されます。

構文です。

physics_pause_enable(flag)

論証タイプ 商品説明
flag This can be set to true でシミュレーションを一時停止し、 で再スタートします。 false

リターンです。

例

if os_is_paused()
{
    physics_pause_enable(true);
    instance_deactivate_all(true);
    instance_create_layer(x, y, "Controllers", obj_pause_menu);
}

上のコードは、OS が一時停止したかどうかをチェックし、一時停止していれば、物理世界を一時停止し、自分以外のすべての機能を停止し、一時停止メニューのインスタンスを作成します。
