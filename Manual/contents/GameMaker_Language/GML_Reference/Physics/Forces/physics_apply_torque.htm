物理適用トルク

トルクというと、一般的には力や衝動によってインスタンスをその重心を中心に回転させる傾向のことを指しますが、GameMakerではインスタンスに独立してトルクをかけ、任意の方向に回転させることも可能です。望む回転を得るためにどれだけのトルクを適用するかは、インスタンスに割り当てた固定具のサイズと密度に依存します。これらのプロパティは質量を定義することにつながり、トルクに影響するイナーシャを定義します一般的な経験則では、インスタンスが多くの慣性を持つ場合、回転を開始させるために多くのトルクが必要になります。また、重力や衝突などの他の要因もインスタンスのトルクに影響を与えることに留意する必要があります。以下は、説明のための画像です。

ご覧のように、object を回転させるのに必要なトルクの量は、それが与える基礎的な質量と慣性に依存します。これはすべて、インスタンス作成時に割り当てられた固定具のサイズと密度に基づいて、GameMakerが計算してくれるものです。また、反時計回りの回転の場合、値は負でなければならないことに注意してください。

構文です。

physics_apply_torque(torque)

論証タイプ 商品説明
torque The torque (in Newton metres) to be applied

リターンです。

例

if keyboard_check(vk_left)
{
    physics_apply_torque(-100);
}

上記のコードは、左矢印キーを押すと、100Nmのトルクで反時計回り（左回り）にインスタンスを回転させます。
