物理学的応用力

物理の世界で力を使うもう一つの方法は、力をインスタンスに局所的に適用することである。これは、力の強さと方向が、インスタンスの原点（sprite がない場合は位置）に基づいて計算されることを意味し、ゲームroom や物理世界での方向や回転を考慮する必要はありません。この関数では、力はobject の質量中心ではなく、インスタンスに相対する点に適用され、object に作用する他の力（重力など）に依存するため、瞬時に適用されないことに注意する必要があります。この図は、ローカルフォースがどのように機能するかを示しています。

ご覧のように、xlocal と ylocal の座標は、軸の位置を常にインスタンスの原点 (0,0) とみなして計算しているので、room object のどこにいても、xlocal と ylocal は常に同じになるのです。次に、力のベクトル成分（xforce と yforce）を xlocal/ylocal の位置に対してどこに配置するかを決定する必要があります。つまり、力は xlocal/ylocal から xforce/yforce までの距離（ニュートン）、方向は xlocal/ylocal から xforce/yforce までの角度として計算されるのです。少しわかりにくいかもしれませんが、イラストを見れば、この仕組みがわかりますし、複雑な計算の手間が省けますよ。

注：物理の世界では、すべての計算にメートルを使用します。1メートルあたりのピクセルの比率は、room エディタで定義するか、または physics_world_create().

構文です。

physics_apply_local_force(xlocal, ylocal, xforce, yforce)

論証タイプ 商品説明
xlocal The x coordinate relative to the origin where the force will be applied
ylocal The y coordinate relative to the origin where the force will be applied
xforce the x component of the force vector
yforce the y component of the force vector

リターンです。

例

if keyboard_check(vk_right)
{
    physics_apply_local_force(-25, 0, -50, 0);
}

上記のコードは、右矢印キーを押している間、原点に対して左から右へ50ニュートンの力をインスタンスに加えます。
