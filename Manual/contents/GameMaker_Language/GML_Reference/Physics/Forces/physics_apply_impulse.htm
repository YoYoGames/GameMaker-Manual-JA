物理学_インパルスを適用する

GameMakerの物理演算では、object に力と重力を加えることができるだけでなく、衝動を加えることもできます。これは力とは少し違って、衝撃を加えるとすぐに 速度に影響し、特に衝撃を加えるために選んだ点が質量中心（注：質量中心は必ずしも原点と同じではありません！）と一致していない場合は、object のトルク（または「スピン」）に影響を与える可能性があります。以下はその例です。

このように、プレイヤーがボールをクリックすることで、ボールが回転しながら右上に移動するインパルスが発生します。正確な衝動は、xpos/ypos座標に対するximpulse/yimpulseの成分から得られるベクトルで定義されます。つまり、衝動はxpos/yposからximpulse/yimpulseまでの距離をニュートンで計算し、方向はxpos/yposからximpulse/yimpulseまでの角度とする、という単純なことです。

注：物理の世界では、すべての計算にメートルを使用します。1メートルあたりのピクセルの比率は、room エディタで定義するか、または physics_world_create().

構文です。

physics_apply_impulse(xpos, ypos, ximpulse, yimpulse)

論証タイプ 商品説明
xpos The x coordinate in the room where the impulse will be applied
ypos The y coordinate in the room where the impulse will be applied
ximpulse the x component of the impulse vector
yimpulse the y component of the impulse vector

リターンです。

例

if mouse_check_button_pressed(mb_left)
{
    with (place_meeting(mouse_x, mouse_y, all))
    {
        physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20));
    }
}

上のコードは、左ボタンが押されたときにマウスの位置にあるインスタンスに対して、ランダムなベクトルを持つインパルスを適用するものです。
