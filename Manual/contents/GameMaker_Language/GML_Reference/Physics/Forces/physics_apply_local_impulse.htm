physics_apply_local_impulse

GameMakerの物理演算では、object に力と重力を加えることができるだけでなく、衝動を加えることもできます。objectこれは力とは少し違い、衝撃を加えるとすぐにスピードや、トルク（または "スピン"）に影響を与えます。特に、衝撃を加えるために選んだ点が、質量中心と一致しないベクトルを持っている場合（注意：質量中心は必ずしも原点と同じではありません！）。この関数は、インスタンスに局所的にインパルスを適用します。これはつまり、衝動の強さと方向は、インスタンスの原点（またはsprite がない場合は位置）に基づいて計算され、ゲームroom や物理世界での方向や回転は考慮されません。以下はその例です。

ご覧のように、画像のカエルは下から衝撃を受けて「ジャンプ」しているのです。正確な衝撃は、xpos/ypos座標に対するximpulse/yimpulseの成分から得られるベクトルで定義されます。つまり、衝撃はxpos/yposからximpulse/yimpulseまでの距離をニュートンで計算し、方向はxpos/yposからximpulse/yimpulseまでの角度となる、という単純な話です。

注：物理の世界では、すべての計算にメートルを使用します。1メートルあたりのピクセルの比率は、room エディタで定義するか、次のようにします。 physics_world_create().

構文です。

physics_apply_local_impulse(xpos, ypos, ximpulse, yimpulse)

論証タイプ 商品説明
xpos The x coordinate relative to the origin where the impulse will be applied
ypos The y coordinate relative to the origin where the impulse will be applied
ximpulse the x component of the impulse vector (relative to the xpos coordinate)
yimpulse the y component of the impulse vector (relative to the ypos coordinate)

リターンです。

例

if jump
{
    physics_apply_local_impulse(0, 20, 0, 50);
}

上記のコードは、変数 "jump "がtrueの場合、力30ニュートン、インスタンスの原点に対して下から上に向かう方向の衝撃を加えるものである。
