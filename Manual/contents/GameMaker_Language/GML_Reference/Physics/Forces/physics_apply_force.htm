物理適用力

物理を扱う場合、objects に重力だけでなく、力も適用する必要があります。力とは、object に速度の変化、方向の変化、形状の変化を引き起こすあらゆる影響のことで、GameMaker の物理学の世界では重力の定義と同じように、2つの xforce/yforce 成分からベクトルを得て定義します。力は、object の重心にかかるのではなく、room のある点にかかり、object に作用する他の力（重力など）に依存して、方向や速度、トルク（スピン）が変化することに注意する必要があります。以下は、説明のための画像例である。

上の例では、下部のバネ（object ）により、ボール（object ）に力が発生し、異なる方向に動くようになりました。正確な力は xpos/ypos 座標に対する xforce/yforce の成分から得られるベクトルで定義されます。つまり、力は xpos/ypos から xforce/yforce までの距離をニュートン (1 キロの質量を毎秒 1 メートルの速度で加速するために必要な正味の力の量として定義) で計算し、方向は xpos/ypos から xforce/yforce への角度とする、という単純なことです。上記のような衝突のコードは、springobject の衝突イベント内にあり、以下の例で提供されるようなものになります。

注：物理の世界では、すべての計算にメートルを使用します。1メートルあたりのピクセルの比率は、room エディタで定義するか、次のようにします。 physics_world_create().

構文です。

physics_apply_force(xpos, ypos, xforce, yforce)

論証 タイプ 商品説明
xpos リアル The x coordinate in the room where the force will be applied
ypos リアル The y coordinate in the room where the force will be applied
xforce リアル The x component of the force vector
yforce リアル The y component of the force vector

リターンです。

N/A

例

with (other)
{
    physics_apply_force(x, y, 0, -30);
}

上記のコードは、30ニュートンの力と方向を持つ衝突の他の object にコードを適用します。
