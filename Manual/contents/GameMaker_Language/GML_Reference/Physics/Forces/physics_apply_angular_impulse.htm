物理学_応用_角形インパルス

この関数は、物理を有効にしたインスタンスに角度のインパルスを与えます。このインパルスは、現在のトルクを無視して、与えられた量によって角の回転を設定し、本質的にフィクスチャが持っている "スピン "の量を設定します。トルクを使用してインスタンスに角力を適用したい場合は、関数 physics_apply_torque().

構文です。

physics_apply_angular_impulse(impulse)

論証タイプ 商品説明
impulse The impulse (in Newton metres) to be applied

リターンです。

例

if keyboard_check(vk_up)
{
    physics_apply_angular_impulse(10);
}

上記のコードは、キーが押された場合、物理学が有効な器具に角度のインパルスを与えます。
