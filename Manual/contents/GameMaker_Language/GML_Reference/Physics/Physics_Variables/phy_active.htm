フィアクティブ

この変数は、インスタンスが現在「アクティブ」であるか否かを制御する。これを false にセットすると、インスタンスは物理世界に参加できなくなり、 true にセットすると再び参加できるようになります。インスタンスはまだ画面に表示されており、コードで変更することができますので、これはインスタンスを非アクティブにすることとは違うことに注意してください。

構文です。

phy_active;

リターンです。

(or undefined if the instance is not physics enabled)

例

if keyboard_check_pressed(ord"P")
{
    global.Pause = !global.Pause
    with (obj_Parent)
    {
        phy_active = !global.Pause;
    }
}

上記のコードは、文字「P」のキープレスを検出し、グローバル変数「Pause」を true から false へとトグルし、また元に戻すものです。この変数を使って、変数 "obj_Parent "でインデックスされたobject の子インスタンスで物理がアクティブかどうかが設定される。
