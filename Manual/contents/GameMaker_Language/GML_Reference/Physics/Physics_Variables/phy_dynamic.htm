phy_dynamic

ダイナミックインスタンスとは、物理世界の中で完全にシミュレーションされているインスタンスのことで、この読み取り専用の変数は、チェックされているインスタンスが完全にシミュレーションされている場合は true を、そうでない場合は false を返します。

構文です。

phy_dynamic

リターンです。

(or undefined if the instance is not physics enabled)

例

if other.phy_dynamic
{
    with (other)
    {
        var dir;
        dir = point_direction(x, y, other.x, other.y);
        physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir));
    }
}

上記のコードでは、新しいインスタンスを作成し、それを物理学の世界で「弾丸」であると定義してから、y軸に沿って大きな衝撃を与えています。
