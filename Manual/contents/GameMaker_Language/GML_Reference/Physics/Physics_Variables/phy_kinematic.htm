フィー・キネマティック

objectこのリードオンリー変数は、インスタンスがキネマティックに分類される場合は true を、そうでない場合は false を返します。キネマティックインスタンスとは、質量は無限（密度は0）だが、動くことができるインスタンスのことである。つまり、インスタンスを運動学的にするには、まず静的インスタンスを作成し、次に動きに関連するインスタンス変数を1つ以上設定します（例． phy_speed_x, phy_speed_yまたは phy_angular_velocity)

構文です。

phy_kinematic;

リターンです。

(or undefined if the instance is not physics enabled)

例

if !phy_kinematic
{
    phy_speed_x = 5;
}

上記のコードでは、インスタンスがキネマティックかどうかをチェックし、キネマティックでない場合は水平方向の速度を5に設定しています。
