この読み取り専用変数は、物理を有効にしたインスタンスの現在の速度を、1ステップあたりのピクセル数で定義して返します。この値を変更する必要がある場合は、変数 phy_speed_xと phy_speed_y.

構文です。

phy_speed;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if phy_speed > 10
{
    phy_linear_damping += 0.01;
}
else
{
    phy_linear_damping = 2;
}

上記のコードは、物理演算を有効にしたインスタンスの速度をチェックし、返された値に基づいて線形減衰を変更するものです。
