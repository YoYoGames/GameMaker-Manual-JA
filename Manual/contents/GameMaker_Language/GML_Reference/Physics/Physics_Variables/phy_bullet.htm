フィーバレット

この変数は、インスタンスが極端に速い動き（例えば弾丸）であるかどうかを定義する。デフォルト値は false ですが、 true に設定すると、インスタンスが高速で移動するため、他のインスタンスを検知できずに通過しないように、より高価な衝突検知が必要になることをGameMakerに通知します。

構文です。

phy_bullet;

リターンです。

(or undefined if the instance is not physics enabled)

例

with (instance_create_layer(x, y, "Bullets", obj_Shoot))
{
    phy_bullet = true;
    physics_apply_local_impulse(0, 10, 0, 200);
}

上記のコードでは、新しいインスタンスを作成し、それを物理学の世界で「弾丸」であると定義してから、y軸に沿って大きな衝撃を与えています。
