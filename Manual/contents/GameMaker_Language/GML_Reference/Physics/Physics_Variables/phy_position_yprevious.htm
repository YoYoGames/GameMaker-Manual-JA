phy_position_yprevious

この変数を使って、ゲームroom 物理世界内でのインスタンスの前の y 位置を取得（または設定）することができる。これは、物理世界内でのインスタンスの位置が、現在のステップより前のステップにあることを意味する

構文です。

phy_position_yprevious;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

xx = phy_position_xprevious;
yy = phy_position_yprevious;

上記のコードでは、物理を有効にしたインスタンスの以前のxとyの位置が2つの変数に格納されます。
