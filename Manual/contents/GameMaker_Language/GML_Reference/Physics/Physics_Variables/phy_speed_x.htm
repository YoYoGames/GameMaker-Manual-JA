ファイスピードエックス

この変数は、インスタンスの線形速度ベクトルの x 成分を取得または変更するために使用でき、1 ステップあたりのピクセル数で定義されます（1 秒あたりのピクセル数については、以下を参照してください）。 phy_linear_velocity_x).静的なインスタンス（つまり密度0のインスタンス）に対してこれを変更すると、キネマティックインスタンスに変化します。

構文です。

phy_speed_x;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if phy_speed_x != 0
{
    phy_speed_x = 0;
}

上記のコードは、リニアスピードベクトルのx成分をチェックし、それが0以外であれば0に設定します。
