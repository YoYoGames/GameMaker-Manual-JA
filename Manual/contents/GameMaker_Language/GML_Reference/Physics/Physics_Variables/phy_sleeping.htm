フィースリーピング

このリードオンリー変数は、インスタンスが現在「スリーピング」（true ）かそうでないか（false ）を返します。「スリーピング」インスタンスは、いかなる物理シミュレーションにもアクティブに従事していないインスタンスです。GameMakerは、インスタンスが休んでいて、他のインスタンスと衝突していないとき、シミュレーション・サイクルを節約するために、objects をスリープさせます。

構文です。

phy_sleeping;

リターンです。

(or undefined if the instance is not physics enabled)

例

if phy_sleeping
{
    instance_destroy();
}

上記のコードでは、object がアクティブにシミュレートされているかどうかをチェックし、そうでない場合は破棄しています。
