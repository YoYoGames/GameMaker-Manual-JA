フィーコムイー

この読み取り専用変数は、インスタンスの重心の y 位置を返す。これは、適切な関数で定義されたインスタンスの密度、慣性、質量に基づいて自動的に計算されます。

構文です。

phy_com_y;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

xx = phy_com_x;
yy = phy_com_y;

上記のコードでは、インスタンスの重心のxとyの位置に2つの変数を設定しています。
