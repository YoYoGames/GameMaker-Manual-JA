phy_position_y

この変数は、ゲーム内のインスタンスのy位置を取得（または設定）するために使用することができますroom 物理世界。この変数を直接設定してインスタンスを移動させると、物理世界はエラーになる可能性があることに注意してください。 連続的なシミュレーションを行うことができます。この変数は、インスタンス変数 y.

構文です。

phy_position_y;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

xx = phy_position_x;
yy = phy_position_y;

上記のコードでは、インスタンスのxとyの位置が2つの変数に格納されます。
