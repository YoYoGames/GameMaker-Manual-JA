phy_angular_damping

この変数は、インスタンスの角度のダンピングを設定するために使用することができ、また、現在の角度のダンピングを取得するために使用することもできる。ダンピングは、物理的に有効なインスタンスが持つ角の回転に対する「抵抗」の量です。低い値はインスタンスが衝突の後に速く回転することを可能にし（例えば）、高い値はより強い押しを必要とし、ゆっくりと回転することを可能にします。

構文です。

phy_angular_damping

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_angular_damping = 10;
}
else
{
    phy_angular_damping = 3;
}

上記のコードでは、呼び出したインスタンスと「obj_Water」のインスタンスとの衝突をチェックし、それに応じて角度のダンピングを変更します。
