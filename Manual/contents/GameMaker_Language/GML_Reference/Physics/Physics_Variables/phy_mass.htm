フィー質量

この読み取り専用変数は、インスタンスの質量をキログラム単位で返す。この値は、割り当てられたフィクスチャの表面積とその密度の値に基づいて自動的に計算されますが、関数 physics_mass_properties().

構文です。

phy_mass;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if (phy_mass < other.phy_mass)
{
    instance_destroy();
}

上記のコードは、インスタンスと他のインスタンスの衝突イベントのもので、それぞれのインスタンスの質量を比較し、質量が少ない方を破壊しています。
