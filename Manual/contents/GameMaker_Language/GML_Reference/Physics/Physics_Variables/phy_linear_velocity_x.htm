phy_linear_velocity_x。

この変数は、インスタンスの直線速度ベクトルの x 成分を取得または変更するために使用でき、ピクセル毎秒で定義されます（ピクセル毎秒のステップについては phy_speed_x).静的なインスタンス（つまり密度0のインスタンス）に対してこれを変更すると、キネマティックインスタンスに変わります。

構文です。

phy_linear_velocity_x;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if (phy_linear_velocity_x != 0)
{
    phy_linear_velocity_x = 0;
}

上記のコードは、直線速度ベクトルのx成分をチェックし、それが0以外であれば0に設定します。
