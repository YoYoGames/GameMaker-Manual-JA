ファイブローテーション

この変数は、インスタンスフィクスチャの角度を度数で取得（または設定）するために使用されます。 image_angle.ただし、物理学の世界では、通常のGameMakerのゲームの世界とは逆の方法で回転が計算されることに注意してください。 point_direction()は、その戻り値を修正する必要があります (単純に正を負にすれば解決します)。

構文です。

phy_rotation;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if (phy_speed_x > 0) || (phy_speed_y > 0)
{
    phy_rotation += sqrt(sqr(phy_speed_x) + sqr(phy_speed_y)) /10;
}

上記のコードでは、直線速度をチェックし、どちらかのベクトルが0でなければ、次に実際の速度を計算し、それを使って回転を設定します。
