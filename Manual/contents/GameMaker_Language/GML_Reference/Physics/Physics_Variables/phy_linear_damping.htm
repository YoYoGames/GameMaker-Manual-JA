phy_linear_damping

この変数は、インスタンスの線形ダンピングを設定するために使用することができ、また、現在の線形ダンピングを取得するために使用することもできます。ダンピングは、物理的に有効なインスタンスが持つ、前方への移動に対する「抵抗」の量です。低い値ではインスタンスはより速く移動し加速し、高い値ではより力強く押す必要があります

構文です。

phy_linear_damping;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
    phy_linear_damping = 10;
}
else
{
    phy_linear_damping = 3;
}

上記のコードは、呼び出したインスタンスと「obj_Water」のインスタンス間の衝突をチェックし、それに応じてリニアダンピングを変更します。
