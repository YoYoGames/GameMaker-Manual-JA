物理コリジョン

この読み取り専用配列は、2 つの物理有効インスタンス間の衝突で検出されたすべての点の y 位置を返します。

注：この変数は、物理を有効にしたインスタンスの衝突イベントでのみ利用可能です。

構文です。

phy_collision_y;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

for(var i = 0; i < phy_collision_points; i += 1;)
{
    part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5);
}

上記のコードは、物理演算が可能な2つのインスタンス間の衝突の定義されたすべてのポイントにパーティクルを作成します。
