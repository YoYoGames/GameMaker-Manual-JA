phy_col_normal_x

この読み出し専用変数は，衝突法線の x 成分に対応するものを返す。 phy_collision_x配列の値です。各接触点には、関連する接触法線があります（通常、衝突のすべての接触点に対して同じ法線になります）。この接触法線は、衝突のあるインスタンスから別のインスタンスを指す単位ベクトルであり、例えば、衝突を解決するための正しい「プッシュ」方向を計算するために使用することができます。

注：この変数は、物理を有効にしたインスタンスの衝突イベントでのみ利用可能です。

構文です。

phy_col_normal_x;

リターンです。

(single precision floating point value, or undefined if the instance is not physics enabled)

例

repeat(5 + irandom(5))
{
    with (instance_create_layer(x, y, "Effects", obj_Debris)
    {
        physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]);
    }
}

上記のコードでは、物理演算が可能な2つのインスタンス間の衝突イベントで生成されたインスタンスの移動方向を接触法線を使って設定しています。
