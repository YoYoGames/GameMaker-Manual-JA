物理変数

従来の」衝突や移動のシステムと同様に、物理システムにも、object のインスタンスが作成されたときに、各インスタンスに割り当てられる一連のインスタンス変数があります。これらのプロパティのいくつかは、GML でユーザーが定義できますが、多くは読み取り専用で、さらに衝突イベントでのみ適用されるものがあります。

以下の変数は、物理学が有効なすべての objects に存在し、インスタンスとscript 関数のコードを使用して読み書きができます（つまり、変更できます）。

  • phy_active
  • phy_angular_velocity
  • phy_angular_damping
  • phy_linear_velocity_x
  • phy_linear_velocity_y
  • phy_linear_damping
  • phy_speed_x
  • phy_speed_y
  • phy_position_x
  • phy_position_y
  • phy_position_xprevious
  • phy_position_yprevious
  • phy_rotation
  • phy_fixed_rotation
  • phy_bullet

一部の変数は読み取り専用で、インスタンスコードやscript 関数でその情報を変更することはできません（ただし、GameMakerの内部では変更可能なので、これらは定数ではなく変数になっています）。以下のリストは、すべての物理演算可能なオブジェクトに存在する読み取り専用の変数を示している。

  • phy_speed
  • phy_com_x
  • phy_com_y
  • phy_dynamic
  • phy_kinematic
  • phy_inertia
  • phy_mass
  • phy_sleeping

最後に、物理学が有効なobjects に存在し、衝突イベントの時だけ作成される一時変数がいくつかあります。これらの変数は上に挙げたものと同様で、読み取り専用です。

  • phy_collision_points
  • phy_collision_x
  • phy_collision_y
  • phy_col_normal_x
  • phy_col_normal_y
