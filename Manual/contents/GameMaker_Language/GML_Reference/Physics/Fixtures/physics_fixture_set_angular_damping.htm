physics_fixture_set_angular_damping

現実世界」で回転するobject を考えてみると、モーターがあるか宇宙空間でない限り、外力（周囲の空気との摩擦など）の影響で時間の経過とともに減速していきます。GameMakerの関数 physics_fixture_set_angular_damping() 、この効果をシミュレートして、物理世界のインスタンスの回転速度を下げることができます。これがなければ、回転するインスタンスは無限に回転し続けることになります。ダンピング・パラメータは 0 から無限大の間である必要があります。 0はダンピングなし、無限大は完全なダンピングを意味します。通常、0から1の間のダンピング値を使用しますが、必要であればマイナスでない任意の値を使用することができます。

構文です。

physics_fixture_set_angular_damping(fixture, damping)

論証タイプ 商品説明
fixture the index of the fixture
damping the angular damping of the fixture, usually between 0 and 1

リターンです。

例

physics_fixture_set_angular_damping(fix_Ball, 0.1);

上記のコードでは、"fix_ball "でインデックスされたフィクスチャの角度ダンピングを0.1に設定します。
