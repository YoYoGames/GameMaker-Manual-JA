physics_fixture_set_polygon_shape

この関数は、フィクスチャのポリゴン形状を設定しますが、そのためには physics_fixture_add_point()を使用して、フィクスチャの原点に対するこのポリゴンの形状を実際に定義します。このポリゴン は、フィクスチャがインスタンスにバインドされたときに閉じられます。また、インスタンスにバインドする前に、ポリゴンに対して少なくとも3つのポイントを定義しておく必要があります。 はエラーになります。

構文です。

physics_fixture_set_polygon_shape(fixture)

論証タイプ 商品説明
fixture the index of the fixture

リターンです。

例

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

上記のコードでは、変数 "fix_Ship "でインデックスされたフィクスチャーにポリゴン形状を適用し、3点を定義して三角形の形状を作成します。
