物理密度

を使ってインスタンスにフィクスチャをバインドすると physics_fixture_bind()は、バインドされたフィクスチャの "id" を返します。この id を使って、バインドされたフィクスチャの密度値を得ることができます（バインドされたフィクスチャではなく この関数を使えば、いつでも "ベース "フィクスチャを使うことができます。

構文です。

physics_get_density(fixture)

論証タイプ 商品説明
fixture the id of the bound fixture

リターンです。

例

var density = physics_get_density(fix_id);
physics_set_density(fix_id, density - 0.1);

上記のコードでは、インスタンスのバインドされた物理プロパティの電流密度値を取得し、それを別の値に設定しています。
