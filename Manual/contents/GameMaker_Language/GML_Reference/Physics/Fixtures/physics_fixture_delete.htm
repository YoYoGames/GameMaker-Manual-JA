physics_fixture_delete

いったんフィクスチャがインスタンスにバインドされると、そのインスタンスはフィクスチャのプロパティを維持するので、特にそれを再び使う予定がない場合は、この関数でフィクスチャをメモリから削除することができます。フィクスチャの削除に失敗した場合は そのため、フィクスチャを定義してバインドした後は、適切にクリーンアップを行う必要があります。

構文です。

physics_fixture_delete(fixture)

論証タイプ 商品説明
fixture the fixture that is to be deleted from memory

リターンです。

例

var fix, inst;
fix = physics_fixture_create();
physics_fixture_set_circle_shape(fix, 16);
physics_fixture_set_density(fix, 1.0);
inst = instance_create_layer(x, y, "Instances", genericBodyObject);
physics_fixture_bind(fix, inst);
physics_fixture_delete(fix);

上記のコードでは、固定具を作成し、そのインデックスを変数「fix」に代入しています。次に、固定具の形状と密度を定義してから、変数 "inst" に格納されたインデックスで作成されたインスタンスにバインドしています。 最後に、フィクスチャは不要になったので、メモリリークを防ぐために削除される。
