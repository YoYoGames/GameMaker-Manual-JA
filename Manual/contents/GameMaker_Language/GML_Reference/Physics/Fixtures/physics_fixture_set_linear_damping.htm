physics_fixture_set_linear_damping

ダンピングはインスタンスの物理世界の速度を下げるために使用され、摩擦とは、フィクスチャを持つ2つのインスタンスが接触したときにのみ発生する点で異なります。また、ダンピングは摩擦よりもずっと安価にシミュレーションできますが、ダンピング は摩擦の代わりではなく、この2つの効果は一緒に使うことができますし、使うべきです。以下は、ダンピングを説明するための画像です。

ダンピングパラメータは0〜∞の間で指定します。0はダンピングなしを意味します。 無限大は完全なダンピングを意味します。通常、0から1の間のダンピング値を使用しますが、必要に応じて負でない任意の値を使用することができます。

構文です。

physics_fixture_set_linear_damping(fixture, damping)

論証タイプ 商品説明
fixture the index of the fixture
damping the damping of the fixture, usually between 0 and 1

リターンです。

例

physics_fixture_set_linear_damping(fix_Ball, 0.1);

上記のコードでは、"fix_ball "でインデックスされたフィクスチャの線形減衰を0.1に設定します。
