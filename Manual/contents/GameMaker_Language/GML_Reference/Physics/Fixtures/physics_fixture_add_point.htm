physics_fixture_add_point

この関数は、あらかじめフィクスチャに設定されたポリゴンのポイントを、以下のいずれかの方法で定義します。 physics_fixture_set_polygon_shape()または、エッジのチェーンにポイントを追加するために使用することができます。 という関数を使って、フィクスチャを作成します。 physics_fixture_set_chain_shape().ポリゴン形状の場合、ポイントはフィクスチャの原点からの相対的なもので、x/y軸の0,0の位置が中心となります。 で、それ以降の座標はGameMakerで通常通り、下/右が+x,+y、上/左が-x,-yで計算されます。以下は、この処理をイメージしやすいように図解したものです。

ご覧の通り、単純な凸多角形はこの方法でモデル化することができます。 凹型のポリゴンは、現時点ではサポートされていません。最後に、ポリゴンを閉じるには、インスタンスにフィクスチャをバインドする必要があります。このとき、ポリゴンに少なくとも3つの点が定義されていなければなりません。 また、事前にポリゴン形状が設定されていないフィクスチャにポイントを追加してもエラーになります。また、1つのフィクスチャーに許される点の最大数は8であることにも注意してください。

チェーンシェイプ（単純にエッジシェイプをつなげたもの）の場合、この関数はエッジをチェーンでつなぐための「接続点」としてポイントを設定します。したがって、フィクスチャをインスタンスにバインドする前に、少なくとも2つのポイントを定義しておく必要があります。 でないとエラーになります。ただし、ポリゴンとは異なり、最大8点までという制限はなく、チェーンを作成するために必要な数だけ点を追加することができ、点は、ポリゴンに対する相対座標ではなく、実際のroom の座標で定義されます。 のインスタンスを作成します。

構文です。

physics_fixture_add_point(fixture, xpos, ypos)

論証タイプ 商品説明
fixture the index of the fixture
xpos the x position of the point (relative to the fixture origin for polygon shapes)
ypos the y position of the point (relative to the fixture origin for polygon shapes)

リターンです。

例

physics_fixture_set_polygon_shape(fix_Ship);
physics_fixture_add_point(fix_Ship, 0,0);
physics_fixture_add_point(fix_Ship, -40, 100);
physics_fixture_add_point(fix_Ship, 40, 100);

上記のコードでは、変数 "fix_Ship "でインデックスされたフィクスチャーにポリゴン形状を適用し、3点を定義して三角形の形状を作成します。
