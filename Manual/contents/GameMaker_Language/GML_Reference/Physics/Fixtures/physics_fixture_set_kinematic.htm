physics_fixture_set_kinematic

物理ベースのゲームでは、動き回りたいけれども、重力のような力や、動的なobjects との衝突によって発生する力を受けたくない場合があります（例えば、プラットフォームゲームでの移動プラットフォームを考えてみてください）。 など）。そのようなobjects に対して、単に固定具の密度を0に設定することは、物理がobject が静的であることを意図していると仮定することを意味します。しかし phy_speed_x, phy_speed_yまたは phy_angular_velocityこのようなフィクスチャがバインドされているインスタンスに対して、そのインスタンスは をキネマティックobject に変換し、移動を開始します。シミュレーションの途中でインスタンスをスタティックからキネマティックに変更すると、（物理世界でのインスタンスの状態に依存して）処理に時間がかかることがあります。 そうすれば、衝突や力（重力など）の影響を受けずに、一定の線速度や角速度で移動することができます。

構文です。

physics_fixture_set_kinematic(fixture)

論証タイプ 商品説明
fixture the index of the fixture

リターンです。

例

physics_fixture_set_kinematic(fix_Cloud);

上記のコードは、"fix_Cloud "でインデックスされたフィクスチャがキネマティックとしてマークされるべきことを示します。
