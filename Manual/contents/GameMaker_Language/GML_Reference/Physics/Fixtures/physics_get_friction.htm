物理学_摩擦を得る

を使ってインスタンスにフィクスチャをバインドすると physics_fixture_bind()は、バインドされたフィクスチャの "id "を返します。この id を使って、バインドされたフィクスチャ（「ベース」フィクスチャではない）の摩擦値を、この関数でいつでも取得することができます。

構文です。

physics_get_friction(fixture)

論証 タイプ 商品説明
fixture 物理フィクスチャID the id of the bound fixture

リターンです。

Real

例

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

上記のコードは、インスタンスのバインドされた物理プロパティの現在の摩擦値を取得し、それらを異なる値に設定します。
