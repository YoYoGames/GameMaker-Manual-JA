physics_remove_fixture

この関数は、インスタンスまたはインスタンスからフィクスチャを削除（または「アンバインド」）します。この関数には、バインドされているフィクスチャの一意な "id" が必要です (この関数が返すものです)。 physics_fixture_bind()そして、インスタンスに対して現在定義されているすべての物理プロパティを削除し、新しいフィクスチャを再定義してそれをインスタンスにバインドすることができるようにします。この方法で、インスタンスを破壊して再作成することなく、インスタンスの物理プロパティを変更することができます。

構文です。

physics_remove_fixture(id, fixture)

論証 タイプ 商品説明
id インスタンスID The ID of the instance to remove the fixture from
fixture 物理フィクスチャID The ID of the fixture that is to be removed from the instance

リターンです。

N/A

例

physics_remove_fixture(id, my_fix);

上記のコードは、変数 "my_fix" に格納された "id" を持つフィクスチャをインスタンスから削除します。
