physics_fixture_create

フィクスチャを設定する最初のステップは、この関数でフィクスチャを作成することです。返されたインデックスは、このフィクスチャを定義して使用するために使われるそれ以降のすべての関数で使われる変数に保存されなければなりません。

構文です。

physics_fixture_create()

リターンです。

Physics Fixture ID

例

fix_Ball = physics_fixture_create();

上記のコードでは、フィクスチャが作成され、そのインデックスが変数 "fix_Ball" に格納されます。
