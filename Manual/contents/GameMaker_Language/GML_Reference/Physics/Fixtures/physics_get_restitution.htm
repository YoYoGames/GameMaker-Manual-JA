物理学的返還

を使ってインスタンスにフィクスチャをバインドすると physics_fixture_bind()は、バインドされたフィクスチャの "id" を返します。このidを使って、復元（"bounciness "プロパティ）を取得することができます。 の値は、この関数を使っていつでも、"ベース"フィクスチャではなく、バインドされたフィクスチャの値です。

構文です。

physics_get_restitution(fixture)

論証タイプ 商品説明
fixture the id of the bound fixture

リターンです。

例

var rest = physics_get_restitution(fix_id);
physics_set_restitution(fix_id, rest * 2);

上記のコードは、インスタンスのバインドされた物理プロパティの現在の返還値を取得し、それらを異なる値に設定します。
