physics_fixture_set_chain_shape

この関数は、「チェーン」フィクスチャーの形状を定義します。チェーンフィクスチャはいくつかの点で構成され、それらはエッジシェイプを使って結合されます。この関数自体はフィクスチャのインデックス（ID）を受け取り、フィクスチャにloop かどうかを指示することができます。これを true にセットすると、最後の点と最初の点がエッジで接続され、 false にセットすると、接続されません。この関数は、チェーンを作るために必要な点を受け入れるためにフィクスチャを準備することに注意してください。 physics_fixture_add_point()ポリゴンフィクスチャを作るときと同じように。

チェーンフィクスチャーを作成する場合、少なくとも2つのポイントを与えなければなりませんが、それ以降のポイントを追加して完成したフィクスチャーを構成することは、特に制限されるものではありません。

構文です。

physics_fixture_set_chain_shape(fixture, loop)

論証タイプ 商品説明
fixture The index of the fixture
loop Whether to close the chain (true) か、そうでないか (false) です。

リターンです。

例

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
{
    physics_fixture_add_point(_fix, _xx, _yy);
    _xx += 50 + random(150);
    _yy = (room_height / 2) - 250 + irandom(500);
}
physics_fixture_bind(fix, id);

上記のコードでは、room の長さにわたってさまざまな高さの「エッジ」フィクスチャーのラインが作成されます。
