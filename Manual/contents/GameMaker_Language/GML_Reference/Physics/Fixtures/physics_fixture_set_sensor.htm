physics_fixture_set_sensor

時には、フィクスチャを持つ2つのインスタンスが衝突したときに、ゲーム側でそれを検知・認識したいことがあります。これは、基本的に衝突イベントを発生させますが、物理的な反応はないので、これらのインスタンスをゲーム内で起こる他のイベントの "トリガー "として使用することができますroom.どんな固定具でもセンサーとしてフラグを立てることができ、それがバインドされているインスタンスが静止しているか動いているかは関係ありません。

NOTE: センサーフィクスチャーは最初に衝突が起こったときだけ衝突イベントを発生させます。つまり、2つのボディが重なり続けても衝突イベントのストリームは発生しません（これは伝統的に起こることです）。つまり、2つのボディが重なり続けてもコリジョンイベントは発生しません。

構文です。

physics_fixture_set_sensor(fixture, state)

論証 タイプ 商品説明
fixture 物理フィクスチャID the index of the fixture
state ブーリアン whether a fixture is a sensor (true) or not (false)

リターンです。

N/A

例

physics_fixture_set_sensor(fix_Cloud, 1);

上記のコードは、"fix_Cloud "でインデックスされたフィクスチャのセンサーの状態をtrueにするものです。
