ショップ_リーブ_レーティング

この機能は、OSに依存したダイアログを開き、ユーザーに特定のページへの評価やコメントの投稿を依頼することができます。ダイアログに表示するテキスト、2つのボタンに表示するテキスト、コメントを投稿するURLを定義することができます。

注：この機能は、Android およびiOS のターゲットにのみ適用されます。

重要! iOSでTestFlightを使用している場合、開発中にプロンプトは表示されますが、実際には何もできません。アプリがApp Storeに掲載されると、このボタンがアクティブになり、ユーザはレビューを書いたり評価を残したりするオプションを得ることができます。TestFlightを通じてゲームを配布する場合、このウィンドウが表示されないことがあることに注意してください。

構文です。

shop_leave_rating(text, yes_string, no_string, url)

論証 タイプ 商品説明
text 文字列 Text that appears on the dialog.
yes_string 文字列 Text that appears on the "yes" button.
no_string 文字列 Text that appears on the "no" button.
url 文字列 The full URL where the comment has to be sent.

リターンです。

N/A

例

if timer <= 0
{
    shop_leave_rating("Thanks for playing the game! Why not leave a comment?", "Okay!", "Not Today!", "http://macsweeneygames.com/comments");
    timer = 10000000;
}
else
{
    timer -= 1;
}

上記のコードは、変数「timer」のカウントダウンが0になった場合に、ユーザーにコメントを残すように求めるものです。
