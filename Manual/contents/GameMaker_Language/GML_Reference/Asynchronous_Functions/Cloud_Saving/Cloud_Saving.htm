クラウドセービング

ゲームを作る場合、ゲームの状態に関する情報を何らかのファイルに保存する必要がある場合が多いのですが、この情報を端末に保存しておくと、プレイヤーがゲームを削除した後に再インストールした場合に、この情報が失われる可能性があり、必ずしも最適な方法とは言えません。これを防ぐには、インターネット上にデータを保存し、いつでも検索・変更できる様々なクラウドサービスを利用することができます。ただし、この機能は1つのゲームにつき1つのデータ「ブロブ」に限られるため、string 、ファイルであれ、新しいセーブをクラウドサービスに送信するたびに、以前に保存したデータが上書きされることに注意してください。

注意事項 現在、これらの機能を使用しているのはAndroidターゲットのみで、Android Game Optionsの SocialセクションにあるEnable Google Cloud Savingのチェックボックスにチェックを入れる必要があります。これにより、extension のクラウド保存が機能するために必要なGoogle サービス Android をダウンロードするよう促されます。クラウド保存の詳細については、次のヘルプデスク記事を参照してください。Androidのクラウド保存

GameMakerは、Asynchronous Cloud Eventと連動するいくつかの簡単な関数でこれをサポートします。これらの関数については、次のページで説明します。

  • cloud_synchronise
  • cloud_string_save
  • cloud_file_save
