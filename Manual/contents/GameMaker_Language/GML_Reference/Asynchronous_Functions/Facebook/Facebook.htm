Facebookはインターネット上で最も重要なソーシャルメディアサイトの一つであり、現在多くのゲームや開発者がゲームやメディアを投稿するための独自のページを持っています。GameMakerには extension 、Facebookと接続し、そのAPIで許可された特定のアクションやタスクを実行することができます。これらの機能を最大限に活用するためには、Facebook Graphの仕組みや利用可能なコールとパーミッションについてしっかりと理解することが不可欠であるため、extension を使い始める前にFacebook Developersページを訪問することをお勧めする。

Facebookextension の設定方法と使用方法については、以下のリンクからすべての利用可能なプラットフォームで確認できます。

  • Android、iOS 、HTML5。Facebookとの連携

以下は、レガシーなFacebook APIでのみ有効であった廃止された関数です。これらの関数が含まれる古いプロジェクトをインポートした場合に備えて、ここにリストアップしています。現在の Facebook の機能については、ターゲットプラットフォームの適切なゲームオプションでFacebook APIを有効にする必要があり、extension をダウンロードしてインストールするよう促されます。extension には、すべての関数定義と例を示しながら、使い方を説明した *.pdf 形式の「ミニ・マニュアル」が付属しています。古い facebook_*() 関数への参照はすべて、同等のextension fb_*() を使用するように置き換える (および更新する) 必要があります。
