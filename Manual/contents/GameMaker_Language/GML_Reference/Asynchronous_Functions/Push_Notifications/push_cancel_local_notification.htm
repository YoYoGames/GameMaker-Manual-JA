push_cancel_local_notification（ローカル通知）。

説明： 指定されたプッシュ通知をキャンセルします。キャンセルする通知のIDは、関数 push_get_first_local_notification()と push_get_next_local_notification()で、この関数は成功した場合は true を、そうでない場合は false を返します (たとえば、通知が存在しないか、すでにトリガーされている場合など)。

注：この機能は、iOSおよびAndroidのターゲットモジュールに限定されます。

構文です。

push_cancel_local_notification(ID);

論証 タイプ 商品説明
ID プッシュ通知ID The ID of the notification to cancel.

リターンです。

Boolean

例:

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
{
    var data = ds_map_find_value(map, "data");
    if data == "Daily_Reward"
    {
        push_cancel_local_notification(ntf);
    }
    ntf = push_get_next_local_notification(map);
}
ds_map_destroy(map);

上記のコードは、 DS Map を作成し、queue で最初に利用可能な通知のデータを要求します。通知が存在する場合、マップが入力され、コードは次に特定のstring のための「データ」キーをチェックします。string が見つかった場合、通知はキャンセルされ、queue のすべてがチェックされるまで、次の通知をチェックするために移動します。
