この関数は log2(n)や log10(n) 関数のように、対数の基本値を指定するだけです。例えば、 logn(5,25) は、5をいくつ掛ける必要があるかを返します。 で25（これは2）。

構文です。

logn(n, val);

論証タイプ 商品説明
n The log base.
val The input value.

リターンです。

例

logval = logn(5, num);

上記のコードは、"num "に格納されている値の log5 を取得します。
