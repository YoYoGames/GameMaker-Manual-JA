出鱈目

この非常に便利な関数は、整数（全数）のみを返します。したがって、たとえば 0 から 9 までの乱数を得るには、 irandom(9) を使えば、0 から 9までの数値を返します。浮動小数点も使えますが、ポイントより上の値は除外されるので、 irandom(9.7) は 0 から 9 までの値だけを返すことに注意してください。この関数の上限は$7fffffffffffffffLLなので、非常に大きな数値を使用する場合は注意が必要です。

注意これは、GameMakerが毎回同じ初期ランダムシードを生成し、debugging のコードをより簡単に作成するためである。この挙動を避けるには randomise()をゲーム開始時に指定します。

構文です。

irandom(n);

論証タイプ 商品説明
n The upper range from which the random number will be selected.

リターンです。

例

if (irandom(9) == 1)
{
    score += 100;
}

この場合、10分の1（0を含むので）の確率で100が加算されることになる。
