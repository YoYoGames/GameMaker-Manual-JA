log10

この関数は log2(n)関数ですが、 log10(n) は、n を得るために掛け合わせる必要のある 10 の数を取得します。したがって、 log10(100) は、10 を掛けると 100、 log10(1000) は 10x10x10=1000 として 3 を返すなど...。しかし、10で割り切れる数でなくても、適切な値を返します。例えば、 log10(101) の場合、2.00432137...となり、これは 101を10で何回割れるか。

構文です。

log10(n);

論証タイプ 商品説明
n The input value.

リターンです。

例

logval = log10(num);

上記のコードは、"num "に格納されている値の log10 を取得します。
