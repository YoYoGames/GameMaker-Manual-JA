ランダムセットシード

GameMakerは乱数を生成するために、まず乱数の種となる数値を設定します。この関数を使うと、そのシードを既知の値に設定することで、その後のすべてのランダムイベントの結果が、プログラムを実行するたびに同じになるように「強制」することができます。例えば、この関数は次のように使うことができる。 random_get_seed()を使えば、手続き的に生成されたコンテンツを作成し、膨大なセーブゲームをせずに結果を保存できます（シードだけを保存し、他のものは必要ありません）。本当にランダムな結果が必要な場合には randomise()関数を使用します。

注このシードは、各ターゲットプラットフォームで一貫した結果を得ることができますが、各ターゲットの動作方法が異なるため、プラットフォーム間で結果が異なる場合があります。

構文です。

random_set_seed(val);

論証タイプ 商品説明
val The seed to set.

リターンです。

N/A

例

if debug
{
    random_set_seed(1);
}

上記のコードでは、変数 "debug "がtrueの場合のみ、乱数シードを1に設定します。
