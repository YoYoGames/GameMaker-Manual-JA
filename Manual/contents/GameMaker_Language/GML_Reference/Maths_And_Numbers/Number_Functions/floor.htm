floor

n関数は、nの階数、つまりnを切り捨てて整数にしたものを返します。これは round()関数ですが、小数点以下の数値に関係なく切り捨てるだけなので、 floor(5.99999) は 5 を返しますが、同様に floor(5.2), floor(5.6457) etc...

構文です。

floor(n);

論証タイプ 商品説明
n The number to floor.

リターンです。

例

val = floor( 3.9 );

これにより、valが3に設定されます。
