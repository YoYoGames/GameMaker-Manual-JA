シリル

この関数は、任意の実数を受け取り、最も近い整数に切り上げます。この関数でよくある間違いは、ランダムな値を丸め、それが常に1より大きくなることを期待することです、つまり、この関数では注意が必要です。

int = ceil(random(5));

このコードは常に1から5までの整数を返すと思われますが、そうとは限りません。ランダム関数が0を返す可能性は非常に低く、0を切り上げても0になってしまうからです。これはごくまれな 可能性がありますが、この関数を使用する際には考慮する必要があります。

構文です。

ceil(val);

論証タイプ 商品説明
x The number to change.

リターンです。

例

val = ceil( 3.4 );

これにより、valが4に設定されます。
