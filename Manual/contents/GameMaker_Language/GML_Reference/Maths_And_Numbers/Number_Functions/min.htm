ミニ

この関数は入力値の最小値を返しますが、引数はいくつでもかまいません（引数が多いと解析が遅くなることに注意してください）。例えば、 min(12, 96, 32, 75) は 12 を返します。 すべての入力値の中で最も小さい値です。

構文です。

min(val1, val2, ... max_val);

論証タイプ 商品説明
val0 ... max_val The values to compare.

リターンです。

例

x = min(x, room_width);

これは、プレイヤーがroom の右側から出るのを阻止します。 これは、常にその x を自分自身か room_width 変数のどちらか小さいほうに設定することで機能します。もしプレイヤーが右から出ると、そのxはroom_widthより大きくなってしまうので をセットし直します。
