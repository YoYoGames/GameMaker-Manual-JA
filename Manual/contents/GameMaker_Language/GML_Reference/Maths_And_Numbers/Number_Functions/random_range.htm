ランダムレンジ

この関数は指定された範囲内の乱数を返すが、この返り値は整数である必要はない。例えば、 random_range(20,50) は 20 から 50 までの乱数を返しますが、その値は 38.65265 のような実数であってもかまいません。 実数は入力引数として使用することもできます。

注：この関数は、ゲームが新しく実行されるたびに同じ値を返します。これは、GameMakerが毎回同じ初期ランダムシードを生成して、debugging のコードをはるかに簡単な作業にするためです。この挙動を避けるには randomise()をゲーム開始時に指定します。

構文です。

random_range(n1, n2);

論証タイプ 商品説明
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

リターンです。

例

score += random_range(500, 600);

これにより、500点から600点、合計点数に加算されます。
