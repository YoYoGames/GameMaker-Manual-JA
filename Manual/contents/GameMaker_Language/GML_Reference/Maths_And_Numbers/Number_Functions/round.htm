まろやか

その言葉通り、 round() は実数を受け取り、最も近い整数に切り上げたり切り捨てたりします。与えられた数値がちょうど半整数の特別な場合（1.5, 17.5, -2.5, など）、数値は最も近い偶数値に丸められ、例えば 2.5 は 2 に、3.5 は 4 になります。この種の丸めはバンカー丸めと呼ばれ、多数の反復処理や浮動小数点数使用時には、統計的に優れた丸めが行われます 従来の「0.5を超えたら切り上げ、それ以外は切り捨て」よりも

これはどういうことかというと、ある値の端数が0.5の場合、丸めた結果は入力値に最も近い偶数整数になるということです。つまり、例えば23.5は24となり、24.5もそうであり、-23.5は-24となり、-24.5もそうである。この方法は，正 さらに重要なことは、合理的な値の分布では、丸められた数値の期待値（平均値）は元の数値のそれと同じであるということである。

構文です。

round(n);

論証タイプ 商品説明
n The number to round.

リターンです。

例

score += round(hp / 5);

上記のコードでは、スコア値に丸められた整数が追加されます。
