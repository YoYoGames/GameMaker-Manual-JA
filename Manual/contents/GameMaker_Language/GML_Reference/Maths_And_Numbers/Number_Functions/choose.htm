選ぶ

乱数選択のために数字以外のものを指定したい場合や、欲しい数字が実際の順序や設定された範囲内にない場合があります。このような場合、 choose() を使ってランダムな結果を生成することになります。例えば、先頭にランダムなobject を持つsprite を作成したい場合、この関数を使用してsprite のインデックスを与えられたsprites のセットのうちの1つに設定することができます。引数はいくつでも指定できます(ただし、引数が多くなると解析に時間がかかることに注意してください)。

注意この関数は、ゲームを新たに実行するたびに同じ値を返します。これは、GameMakerが毎回同じ初期ランダムシードを生成し、debugging のコードをはるかに簡単にするためです。この挙動を避けるには randomise()をゲーム開始時に指定します。

構文です。

choose(val0, val1, val2... max_val);

論証 タイプ 商品説明
val0... max_val バリアブル Any type of value(s).

リターンです。

Variable (One of the given arguments)

例

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Helen");

上記のコードでは、chooseを使ってインスタンスのプロパティをいくつも設定しています。
