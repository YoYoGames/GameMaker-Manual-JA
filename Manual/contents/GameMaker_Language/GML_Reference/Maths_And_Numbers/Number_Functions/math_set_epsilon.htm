math_set_epsilon

イプシロンとは、丸め誤差のある2つの数値が「等しい」と見なすのに十分近いかどうかを判断するための値である。浮動小数点演算を扱う際に、特定の演算で不正確な値や期待とは異なる値を返してしまう「丸め誤差」を減らすことができるため、便利な機能です。たとえば、sprite の画像インデックスに値を追加して、その結果が1つの整数になることを期待しても、浮動小数点演算の性質上、実際の最終値は 5.0000002 のようなものになってしまうので、次のようなチェックをすることになります。

if image_index == 5
{
    //do something
}

このコードは期待通りの動作をせず、 false を返します。しかし、εの値を0.000001に設定すると、 image_index の値は元の値の±0.000001の最も近い実数に丸められ、上記の比較は真を返します。

イプシロン値は、以下のすべての整数演算子で使用されます。

  • < :未満
  • > :より大きい
  • == :に等しい。
  • <= :以下であること
  • >= :以上であること
  • != :とは一致しない

なお、イプシロンに0を設定すると丸めが一切できなくなり、1を設定するとエラーになります。

構文です。

math_set_epsilon(epsilon);

論証 タイプ 商品説明
epsilon リアル The new epsilon value (from 0 to 0.999999999).

リターンです。

Real

例

math_set_epsilon(0.0001);

これにより、以降のすべての浮動小数点演算のイプシロン値が設定されます。
