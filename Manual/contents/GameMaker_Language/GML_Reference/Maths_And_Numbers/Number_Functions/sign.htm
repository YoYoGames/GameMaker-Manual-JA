この関数は、ある数値が正、負、またはそのどちらでもないかどうかを返し、それぞれ1、-1、0を返します。例えば、 sign(458) は1、 sign(-5) は-1、 sign(0) は0を返します。

構文です。

sign(n);

論証タイプ 商品説明
n The number to get the sign of.

リターンです。

例

y += sign(y - mouse_y);

上記のコードは、 y - mouse_y の結果に応じて、y に 1、-1 または 0 を加算します。
