中央値

この関数は、入力値の中央値、すなわち、中間値を返す。引数の数が偶数のときは，中央の2つの値のうち小さい方を返し，関数は必要なだけの引数を持つことができる（より多くの引数 は関数の解析が遅くなることを意味します）、これらはすべて実数値でなければなりません。つまり、例えば median(43, 12, 25, 19, 6) は残りのすべての値の中間の値である 19 を返すことになります。

構文です。

median(val1, val2, ... max_val);

論証タイプ 商品説明
val0 ... max_val The values to compare.

リターンです。

例

x = median( 0, x, room_width );
y = median( 0, y, room_height );

これは、medianをクランプとして使うことで、プレイヤーがroom のどの側からも出られないようにするものである。例えば、プレイヤーがroom の境界の左側に移動した場合、その x は 0 よりも小さくなります。これは、最初の median の中間数 は0になるので、プレイヤーは(0,y)にジャンプすることになる。
