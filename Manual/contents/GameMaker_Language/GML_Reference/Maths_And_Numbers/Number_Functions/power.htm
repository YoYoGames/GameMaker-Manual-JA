けんせい

これは、ある数字にそれ自体を「n」回かけた値を返します。例えば、 power(5,3) は 5 にそれ自身を 3 回掛けて 125 を返す。これは 5*5*5=125 と言っているのと同じである。ただし、「x」の値 (変更する数値)を負の値にすることはできません。

構文です。

power(x, n);

論証タイプ 商品説明
x The number to change.
n How many times to multiply x by itself.

リターンです。

例

score += power(dmg, 3);

これにより、変数「dmg」に保持されている値が、変数「score」に3の累乗で加算されます。
