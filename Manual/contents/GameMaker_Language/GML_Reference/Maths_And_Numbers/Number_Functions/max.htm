マックス

この関数は、入力値の最大値（最大16個まで）を返します。例えば、 max(12, 96, 32, 75) は、すべての入力値の中で最も大きい値である96を返します。

構文です。

max(val1, val2, ... max_val);

論証タイプ 商品説明
val0 ... max_val The values to compare.

リターンです。

例

x = max(x, 0);

これは、プレイヤーがroom の左側から出るのを阻止します。 これは、常にその x を自分自身か 0 のどちらか大きい方に設定することで機能します。もしプレイヤーが左から抜けると、そのxは0より小さくなる（つまり負になる）ので、そのままセットされることになります。 に戻る。
