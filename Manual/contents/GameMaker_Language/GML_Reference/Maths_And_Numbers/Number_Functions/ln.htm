ln

自然対数 ln(n) は、あるレベルの連続的な成長に達するために必要な時間量であり、nは到達したレベルである。つまり、20の成長を得るために必要な時間単位を知りたい場合は、 ln(20) を使用します。 は、その成長量を得るために必要な時間を2.99単位で返します。

構文です。

ln(n);

論証タイプ 商品説明
n The input value.

リターンです。

例

alarm[0] = ln(age) * room_speed;

上記のコードでは、変数 "age "に格納された値の自然対数を用いて、アラームを設定しています。
