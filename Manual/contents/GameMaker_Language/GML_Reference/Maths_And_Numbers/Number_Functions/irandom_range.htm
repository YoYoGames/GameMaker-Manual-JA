イランダム・レンジ

この関数は random_range()のみで，入力は整数値である。この関数は，入力として整数の低い値と高い値を与えると，その範囲内（および範囲外）のランダムな整数を返します。 を含む) 範囲を指定します。例えば、 irandom_range(10, 35) は 10 から 35までの整数を返す。

と同様に irandom()この場合，最も近い整数に切り捨てられます。

注意この関数は、GameMaker.NETがゲームを起動するたびに同じ値を返します。GameMaker:Studioは毎回同じ初期ランダムシードを生成し、debugging のコードをはるかに簡単にするためです。この挙動を避けるには randomise()をゲーム開始時に指定します。

構文です。

irandom_range(n1, n2);

論証タイプ 商品説明
n1 The low end of the range from which the random number will be selected.
n2 The high end of the range from which the random number will be selected.

リターンです。

例

score += irandom_range(500, 600);

500から600の整数値がスコアに加算されます。
