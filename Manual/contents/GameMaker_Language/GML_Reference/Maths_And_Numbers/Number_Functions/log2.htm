この関数は基本的に「2を何回掛けたらnになるのか」を問うものです。プログラミングでよく使われる例としては、ある数を表現するのに必要なビット数を計算することです。 log2(x) 例えば、 log2(100) は 6.643856 を返しますが、これは切り捨てると 6 で、これに 1 を加えると 7 になります。つまり、100を表すには7ビットが必要なのです。また、指数関数的な伸びや縮みを表現する場合にも使われます。 減衰し、しばしば2進対数とも呼ばれる。

構文です。

log2(n);

論証タイプ 商品説明
n The input value.

リターンです。

例

colourbits = floor(log2(colour)) - 1;

上記のコードでは、変数 "color "に格納されている値を表現するために必要なビット数が格納されています。
