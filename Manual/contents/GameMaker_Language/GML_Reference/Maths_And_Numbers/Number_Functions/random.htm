ランダム

この関数は、整数値を返す必要がない確率に適しています。例えば、 random(100) は 0 から 99 までの値を返しますが、その値は 22.56473 になる可能性があります!また、この関数では整数ではなく実数を使うこともできます - random(0.5), これは 0 から 0.4999999 の間の値を返します。

注：この関数は、ゲームが新しく実行されるたびに同じ値を返します。これは、GameMakerが毎回同じ初期ランダムシードを生成して、debugging のコードをはるかに簡単な作業にするためです。この挙動を避けるには randomise()をゲーム開始時に指定します。

構文です。

random(n);

論証タイプ 商品説明
n The upper range from which the random number will be selected.

リターンです。

例

if (random(10) >= 9)
{
    score += 100;
}

これにより、およそ10分の1の確率で100点を加算することができます。
