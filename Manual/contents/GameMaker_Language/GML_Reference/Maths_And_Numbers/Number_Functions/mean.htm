平均値

この関数は、入力された値をすべて足し算して、それぞれの数値で割るという動作をします。引数は必要な数だけ与えることができます（引数が多いと関数の解析が遅くなることに注意してください）。つまり、 mean(2, 6, 9, 32) は 2+6+9+32=49 と 49/4=12.25 として 12.25 を返します。

構文です。

mean(val1, val2, ... max_val);

論証タイプ 商品説明
val0 ... max_val The values to compare.

リターンです。

例

xmiddle = mean(obj_player1.x, obj_player2.x, obj_player3.x);
ymiddle = mean(obj_player1.y, obj_player2.y, obj_player3.y);

これはxmiddleとymiddleを3人のプレーヤーobjects 、obj_player1, obj_player2, obj_player3の座標の平均のxとy座標に設定します。例えば、これを利用して、ゲームカメラの焦点を3人のプレイヤーに合わせることができます。 1人ではなく、2人です。
