dot_product_normalised（ドットプロダクトノーマライズ

内積とは、2つのベクトルの角度関係を表す値で、2つのベクトルを取り、掛け合わせ、足し合わせることで求められる。ドットプロダクトという名称は、中央のドット"-"に由来する。 スカラー積はベクトルではなくスカラーであることを強調した名称である）。

実際の数式はこのように書くことができる。

つまり、2Dではベクトル a[x1,y1] と b[x2,2] の内積は x1x2 + y1y2 、GameMakerでのdot_productは次のように計算されます。

a · b = (x1*x2)+(y1*y2);

正規化内積はどうなっていますか？正規化点積は、戻り値が-1から1の範囲になるように補正されています（詳しくは「ベクトル」の項を参照）。これは多くの状況、特に照明や他の3D機能を扱うときに非常に便利です。

構文です。

dot_product_normalised(x1, y1, x2, y2)

論証タイプ 商品説明
x1 The x coordinate of the first vector.
y1 The y coordinate of the first vector.
x2 The x coordinate of the second vector.
y2 The y coordinate of the second vector.

リターンです。

例

var x1, y1, x2, y2;
x1 = lengthdir_x(1, image_angle);
y1 = lengthdir_y(1, image_angle);
x2 = o_Player.x - x;
y2 = o_Player.y - y;
if dot_product_normalised(x1, y1, x2, y2) > 0 seen=true else seen=false;

上記のコードでは、インスタンスの画像の角度を用いてベクトルを作成し、プレーヤーobject "o_Player" のベクトルを自分自身に取得します。最後に、この2つのベクトルの内積を計算し、それが0より大きい場合は変数 "seen" を true に、0 以下なら "false" に設定する。
