ディコス

直角三角形の場合cos(val) = 隣接 / 斜辺 (val は3つの角のうちの1つ)です。この関数は常に 1 から -1 までの数を返します。

注：この関数は、ラジアンではなく、度数で値を取ります。

構文です。

dcos(val);

論証タイプ 商品説明
val The angle (in degrees) to return the cosine of.

リターンです。

例

val = dcos(45);

これにより、"val "は0.71に設定される。
