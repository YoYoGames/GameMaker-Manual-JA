distance_to_point

この関数は、呼び出したインスタンスのバウンディングボックスの端から、room で指定された x/y 位置までの距離を計算し、戻り値はピクセル単位で返す。呼び出し元のobject にsprite がない場合，あるいはマスクが定義されていない場合，結果は正しくな いことに注意すること．

構文です。

distance_to_point(x, y);

論証タイプ 商品説明
x The x position to check.
y The y position to check.

リターンです。

例

if (distance_to_point(obj_Player.x, obj_Player.y) < range)
{
    can_shoot = true;
}

上記のコードでは、プレイヤーobject の x/y 位置までの距離をチェックし、それが変数「range」に格納されている値より小さい場合、変数「canshoot」が true に設定されます。
