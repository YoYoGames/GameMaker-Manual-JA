ダルコス

x の逆コサインを返します。 dcos(val)=n, darccos(n)=val, および結果の数値は 180 と 0 の間になります。

注：これは-1から1までの数字しか受け付けません（それ以外はエラーになります）。

注：この関数は、ラジアンではなく、度数で値を返します。

構文です。

darccos(val)

論証タイプ 商品説明
x The value to return the inverse cosine of.

リターンです。

例

val = arccos(-1);

これにより、"val "は180に設定されます。
