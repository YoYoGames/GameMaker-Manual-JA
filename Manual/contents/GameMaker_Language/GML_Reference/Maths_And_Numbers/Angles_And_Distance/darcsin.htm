ダルクシン

x の逆正弦を返します。 dsin(x)=n, darcsin(n)=x, とすれば、結果の数値は -90 から 90 の間になります。

注：これは-1から1までの数字しか受け付けません（それ以外はエラーになります）。

注：この関数は、ラジアンではなく、度数で値を返します。

構文です。

darcsin(x);

論証タイプ 商品説明
val The value to return the inverse sine of.

リターンです。

例

val = darcsin(-1);

これにより、"val "は-90に設定されます。
