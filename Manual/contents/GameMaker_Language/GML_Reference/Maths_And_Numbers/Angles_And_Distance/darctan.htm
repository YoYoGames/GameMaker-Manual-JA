ダルクタン

xの逆正接を返します。 dtan() darctan() 漸近線はy軸上にあるので、決して数値が返されないことを意味します。 90より大きいか、-90より小さい。

注：この関数は、ラジアンではなく、度数で値を返します。

構文です。

darctan(x);

論証タイプ 商品説明
val The value to return the inverse tangent of.

リターンです。

例

val = darctan(1);

これにより、"val "は-45に設定されます。
