コサイン

直角三角形の場合cos(val) = 隣接 / 斜辺 (val は3つの角のうちの1つ)です。この関数は常に 1 から -1 までの数を返します。

注：この関数は、度数ではなくラジアン単位で値を取ります。

構文です。

cos(val);

論証タイプ 商品説明
val The angle (in radians) to return the cosine of.

リターンです。

例

val = cos( 0 );

これにより、valが1に設定されます。
