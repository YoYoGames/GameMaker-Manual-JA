arcsin

x の逆正弦を返します。 sin(x)=n, arcsin(n)=x, そして結果の数値は -pi/2 と pi/2 の間になります。

注：これは-1から1までの値しか受け付けません（それ以外のものはエラーを投げます）。

注：この関数は、度数ではなくラジアン単位で値を返します。

構文です。

arcsin(x);

論証タイプ 商品説明
x The value to return the inverse sine of.

リターンです。

例

val = arcsin(0);

これにより、valは0になります。
