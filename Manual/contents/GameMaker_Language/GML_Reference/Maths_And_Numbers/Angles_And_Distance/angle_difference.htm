角度差

この関数は、指定された2つの角度の最小の差を返します。差は、元の角度から先の角度に向かって計算されます。返される値は -180 から 180 度の間です（正の角度は反時計回り、負の角度は時計回りとなります）。

source" と "destination" の角度の関係は、∠difference(dest, src) を呼んで得られた差をsrc の値に足し戻すとdest の角度になる（正確な数値は異なるかもしれないが）、というものである。同様の手法で、ステップごとに元の角度を徐々に先の角度に近づけていくことができます。例えば、返された差分の10％をsrcに足し戻して、少しずつdest の方に移動させます（以下の例で説明します）。

構文です。

angle_difference(dest, src)

論証タイプ 商品説明
dest The destination or target angle
src The source angle

リターンです。

例

var _dir = point_direction(x, y, mouse_x, mouse_y);
var _diff = angle_difference(_dir, image_angle);
image_angle += _diff * 0.1;

上記のコードでは、インスタンスからマウスカーソルまでの方向を取得し、その角度と現在の角度の差を取得します。 image_angleこの値を用いて、インスタンスをマウスの方向に徐々に回転させる。
