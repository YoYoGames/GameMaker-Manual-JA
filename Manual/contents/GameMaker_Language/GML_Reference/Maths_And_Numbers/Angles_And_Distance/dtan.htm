だんご

直角三角形の場合dtan(val) = Opposite / Adjacentここで "val" は3つの角のうちの1つです。

注：膨大な数の値（例えば90や-90）は無限大を返すため、この関数でエラーとなる。これをグラフで表現すると、これらの値で漸近線ができる。

注：この関数はラジアンではなく度数で値を取ります

構文です。

dtan(val);

論証 タイプ 商品説明
val リアル The angle (in degrees) to return the tangent of.

リターンです。

Real

例

val = dtan(45);

これにより、"val "が1に設定されます。
