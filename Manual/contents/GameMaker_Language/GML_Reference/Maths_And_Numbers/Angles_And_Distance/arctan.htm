アークタン

tan() とは異なり、 arctan() の漸近線は y 軸上にあるので、これはどんな数値も受け入れることができます。 pi/2より大きいか、-pi/2より小さい。

注：この関数は、度数ではなくラジアン単位で値を取ります。

構文です。

arctan(x);

論証タイプ 商品説明
x The angle (in radians) to return the inverse tangent of.

リターンです。

例

val = arctan( 0 );

これにより、valは0になります。
