オブジェクトまでの距離

この関数は、呼び出したインスタンスのバウンディングボックスのエッジから、指定されたobject の最も近いインスタンスのエッジまでの距離を計算するものである。object には、object のインデックス、または特定のインスタンス ID を指定することができる。 キーワードであり、その距離はピクセル単位で返される。objects のいずれかにsprite がない場合、あるいはマスクが定義されていない場合、結果は正しくないことに注意。

構文です。

distance_to_object(obj);

論証 タイプ 商品説明
obj オブジェクトのアセットまたはインスタンスID The object to check for.

リターンです。

Real

例

if (distance_to_object(obj_Player) < range)
{
    canshoot = true;
}

上記のコードでは、プレイヤーまでの距離object をチェックし、それが変数「range」に格納されている値より小さい場合、変数「canshoot」がtrueに設定されます。
