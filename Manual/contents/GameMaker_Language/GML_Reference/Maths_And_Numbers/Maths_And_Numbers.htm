数学と数字

GameMakerには、数学の様々な分野、あらゆる種類の数字、日付と時刻を扱うための多くの関数が含まれています。これらの関数のほとんどは、以下のセクションで見ることができます。

  • 日付と時刻
  • ナンバーファンクション
  • 角度と距離
  • マトリクス機能

実数として評価できる組み込みの定数もいくつかあります（ただし、常に値ではなく定数をチェックする必要があります）。

定数 価値
true 1（ただし、1以上の値は true ）として評価されます。
false 0（ただし、1より小さい値も false ）として評価されます。
pi 3.141592653589793280など...（正確な値は、対象となるプラットフォームなど様々な要因に依存します。）
