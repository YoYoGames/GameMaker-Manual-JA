マトリックスセット

この関数は、与えられた4x4行列の型に対応する16個の値の配列を受け取り、要素 [0 - 3] は行1、要素 [4 - 7] は行2、...となります。(メインページの画像を参照)。このような行列を作成するには matrix_build()または matrix_get()関数に渡すか、あるいは自分で配列を作成し、それを関数に渡します。使用できる行列の種類はview,projection,world で、これには以下の定数のいずれかを使用します。

定数 商品説明
matrix_view 現在のビューマトリックス
matrix_projection 現在の投影行列
matrix_world 現在のワールドマトリックス

構文です。

matrix_set(type, matrix);

論証 タイプ 商品説明
type マトリクス型定数 The type of matrix to get the values of (see the constants listed above)
matrix マトリクスアレイ The matrix data as an array

リターンです。

N/A

例

matrix_set(matrix_world, m_array);

上記のコードは、現在のワールド行列の値を、配列行列「m_array」に格納されている値に設定します。
