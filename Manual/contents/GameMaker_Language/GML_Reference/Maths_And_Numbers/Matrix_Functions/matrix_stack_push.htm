マトリックススタックプッシュ

この関数は，新しい行列を現在の行列スタックの先頭にプッシュするために使用されます．これはまず，与えられた行列と現在スタックの先頭にある行列を掛け合わせ，その結果の行列をスタックにプッシュします．

この機能は、様々な行列を手動で掛け合わせることなく、ビジュアルに複数の行列変換を適用するのに便利です。

例えば、ある長方形の領域（例えば、ウィンドウ）にグラフィックを描画する基本行列をスタックにプッシュさせることができます。その行列を使ってグラフィックを描画した後、別の行列をスタックにプッシュして、サブ領域（ウィンドウのボタンのようなもの）の中にグラフィックを描画することができます。そのサブ領域に何かを描画した後、 matrix_stack_pop() を呼び出すと、そのマトリックスがスタックから削除され、メインウィンドウ領域にビジュアルを描画し続けることができます。

構文です。

matrix_stack_push(matrix);

論証タイプ 商品説明
matrix The matrix to push to the stack

リターンです。

例

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(spr_tyre, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

上記のコードは、2 つの新しい行列を構築し、それらを行列スタックにプッシュします。その後、ワールド行列がスタックの最上部に設定され、sprite が描画され、変換がスタックからポップされます。
