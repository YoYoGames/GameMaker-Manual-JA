マトリックスマルチプライ

この関数を使用すると，2 つの行列配列を掛け合わせて新しい変換行列を作成することができます．この関数は，新しい行列のインデックスを返します．このインデックスは，将来使用するために変数に保存しておく必要があります．

注意：この関数では、行列定数を引数として使用することはできません。したがって、（たとえば）ビュー行列と、自分で作成したカスタム行列を掛け合わせたい場合は、最初に matrix_get()を作成し、ビュー行列の値を配列変数に代入し、それをカスタム行列で乗算してから、選択された行列（ビュー、投影、ワールド）を設定します。

構文です。

matrix_multiply(matrix1, matrix2);

論証タイプ 商品説明
matrix1 The first matrix index to use.
matrix2 The second matrix index to use.

リターンです。

Matrix Array

例

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

上記のコードは、現在のビュー行列を取得し、それにカスタム行列を乗算し、結果の行列インデックスを使用して再びビュー行列を設定します。
