行列のビルドプロジェクションの正方形

この関数は、以下の指定されたパラメータに基づいて正投影行列を構築する（これは、GameMakerでroom を作成する際に何も変更せずに使用されるデフォルトの投影方法である）。時々、switch 透視投影から正投影にする必要がありますが、この関数はそのためのものです。これは通常、3Dゲームで描かれた要素の「平面」ビューを提供するため、例えばスコアやその他の側面を表示するためのオーバーレイを描くために使用されます（つまり、遠近法がない）。正投影法と透視投影法の違いについては、以下の画像を参照してください。

注：また、現在の深度値に関係なく情報を描画したい場合は、一時的に非表示サーフェス除去をオフにする必要があります（switch ）。

構文です。

matrix_build_projection_ortho(width, height, znear, zfar);

論証タイプ 商品説明
w The width of the projection.
h The height of the projection.
znear The near clipping plane.
zfar The far clipping plane.

リターンです。

Matrix Array

例

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

上記のコードでは、新しいルックアット行列とオルソグラフィック行列を作成し、それらのIDをローカル変数に格納し、それらを使ってビューポート[0]に割り当てられたカメラのビュー行列と投影行列を設定します。
