マトリックスビルドプロジェクションパースペクティブ

この関数は、以下に示す指定されたパラメータを用いて、ニアクリッピング平面の次元に基づく透視投影行列を構築する。

引数 znear で指定された Z 位置に投影幅と高さが配置されるため、指定された width, height, znear の値によってカメラの視野が変化することに注意すること。

例えば、640x480の一定サイズであれば、 znear がカメラに近いと視野は広くなり、 znear がカメラから遠いと視野は狭くなります。この挙動は、次のビジュアルで実証されています。

このため、投影幅と同じ znear を使用し、等価な正投影図と一致する視野を得ることを推奨する。

構文です。

matrix_build_projection_perspective(width, height, znear, zfar);

論証タイプ 商品説明
w The width of the projection at the near Z position.
h The height of the projection at the near Z position.
znear The near clipping plane.
zfar The far clipping plane.

リターンです。

Matrix Array

例

var projmat = matrix_build_projection_perspective(640, 480, 640.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

上記のコードでは、透視投影を作成し、返された行列を使用してビューポート[0]に割り当てられたカメラを設定します。
