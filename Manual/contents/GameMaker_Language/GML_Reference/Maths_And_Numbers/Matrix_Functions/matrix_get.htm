matrix_get

この関数は，与えられた 4x4 行列の型に対応する 16 個の値の配列を返します。(メインページの画像を参照).利用可能な行列はview,projection,world で、これには次の定数のいずれかを使用します。

マトリクス型定数
定数 商品説明
matrix_view 現在のビューマトリックス
matrix_projection 現在のプロジェクションマトリックス
matrix_world 現在のワールドマトリックス

構文です。

matrix_get(type);

論証 タイプ 商品説明
type マトリクス型定数 The type of matrix to get (see the constants listed above)

リターンです。

Matrix Array

例

v_array = matrix_get(matrix_view);

上記のコードは、現在のビューマトリックスの値を取得し、配列 "v_array" にその値を格納します。
