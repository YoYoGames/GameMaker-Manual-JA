マトリクス機能

行列とは、一定数の行と列に並べられた数の集まりです。通常は実数であるが、必ずしもそうである必要はない。次の画像は、4行4列の行列を構成する方法を一般的に示しています。

一番上の行は1行目、一番左の列は1列目、この行列は4行4列なので4x4行列です（他の大きさの行列も作れます）。行列を記述する場合、形式は常に行／列であり、行列を構成する各数値を行列の要素と呼びます。行列の要素は特定の位置にあり、行と列の位置で表されます。行列の左上隅は行1列1です。上の行列の例では、1行目1列目の要素は値「1」です。2行目3列目の要素は "4.8 "という値です。

例えば、カメラビューの設定やモデルの平行移動・変形に使用されます。GameMakerには、行列を扱うための以下の関数があります。

  • matrix_set
  • matrix_build
  • matrix_multiply
  • matrix_build_identity
  • matrix_build_lookat
  • matrix_build_projection_ortho
  • matrix_build_projection_perspective
  • matrix_build_projection_perspective_fov
  • matrix_transform_vertex
  • is_matrix
