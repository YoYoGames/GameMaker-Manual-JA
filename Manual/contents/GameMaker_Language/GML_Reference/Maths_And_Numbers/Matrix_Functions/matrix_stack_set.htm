マトリックススタックセット

この関数は，行列スタックの現在の先頭を指定された行列で上書きします．

構文です。

matrix_stack_set(matrix);

論証タイプ 商品説明
matrix The matrix index to use.

リターンです。

例

var m = matrix_build(x, y, 0, 0, 0, 0, 1, 1, 1);
matrix_stack_set(m);

上記のコードは、新しい行列を構築し、その結果の行列インデックスを変数「m」に格納してから、行列スタックの先頭をそれで置き換えます。
