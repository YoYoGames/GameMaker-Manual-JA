日付設定タイムゾーン

残りのすべての日付と時刻の関数で使用する基本タイムゾーンを設定します。このタイムゾーンは、ローカルタイム（システムで設定される）かUTCのどちらかを選択することができます。

定数 商品説明
timezone_local システムで設定されたローカルタイムゾーンを使用する
timezone_utc 協定世界時を使う

構文です。

date_set_timezone(timezone);

論証 タイプ 商品説明
timezone タイムゾーン定数 The time zone to use for the base time.

リターンです。

N/A

例

if (date_get_timezone() != timezone_utc)
{
    date_set_timezone(timezone_utc);
}

このコードは、ゲームの基本タイムゾーン設定をチェックし、それがUTCでない場合はそれを変更します。
