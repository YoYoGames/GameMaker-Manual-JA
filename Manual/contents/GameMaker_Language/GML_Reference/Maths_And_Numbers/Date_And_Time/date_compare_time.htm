日付比較時間

この関数を使うと、2つの日付と時刻を比較して、どちらが時間成分を早く、または遅く持っているかを調べることができます。この関数は、datetime1が早ければ-1、datetime1が遅ければ1、同じであれば0を返し、日付は無視されます。 ということで、文字通り、どの時刻が指定された日よりも進んでいるかということだけです。

構文です。

date_compare_time(datetime1, datetime2);

論証タイプ 商品説明
datetime1 The first datetime.
datetime2 The datetime to compare the first one to.

リターンです。

例

d = date_compare_time(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

これは、どの時刻が最も早いかによって、"d "に対応する値を設定することになる。基本的には、現在の時刻が午前11時4分より遅ければ、1を返す。それ以前の場合は-1、現在時刻が11:04の場合は0を返す。
