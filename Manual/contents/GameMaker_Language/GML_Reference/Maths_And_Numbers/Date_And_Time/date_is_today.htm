日付が今日であること

この関数は、与えられた datetime 値がチェックされた日 (つまり今日) であれば true を、そうでなければ false を返します。これは、ゲームのイースターエッグや、季節のコンテンツのアンロックなどに便利な関数です。この関数は、設定されているタイムゾーン（デフォルトはローカルタイム）に影響されることに注意してください。 date_set_timezone()関数を使用します。

構文です。

date_is_today(date);

論証タイプ 商品説明
date The datetime to use.

リターンです。

例

if date_is_today(global.Halloween)
{
    global.Max_Levels = 200;
}

上記のコードは、グローバル変数「Halloween」に格納されたdatetimeが今日かどうかを調べ、今日であれば別のグローバル変数に新しい値を設定します。
