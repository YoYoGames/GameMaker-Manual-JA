日付_うるう年

この関数は、与えられた datetime 値の年成分がうるう年である場合は true を、そうでない場合は false を返します。これは、ゲームのイースターエッグや、特別なコンテンツのアンロックなどに便利な関数です。

構文です。

date_leap_year(date);

論証タイプ 商品説明
date The datetime to use.

リターンです。

例

if date_leap_year(date_current_datetime())
{
    if (!global.ExtraContent)
    {
        global.ExtraContent = true;
    }
}

上記のコードは、現在のdatetimeがうるう年かどうかをチェックします。もしそうなら、グローバル変数を設定します。
