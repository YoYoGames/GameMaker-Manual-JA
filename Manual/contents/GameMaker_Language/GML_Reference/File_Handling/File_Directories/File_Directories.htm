ファイル・ディレクトリー

ファイルを扱うとき、ファイルが格納されているディレクトリを参照する必要がある場合があります。GameMakerには、さまざまなディレクトリに関連する特殊な関数や変数が用意されているので、必要な場合があります。

  • directory_exists
  • directory_create
  • directory_destroy
  • temp_directory
  • working_directory
  • program_directory
  • game_save_id(ゲームコントロール全般より)

注 HTML5では、ディレクトリの作成と破棄はできません。
