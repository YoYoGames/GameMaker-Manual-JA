プログラムディレクトリ

これは、ゲームの実行ファイルが格納されているディレクトリを返します。しかし、これは必ずしも有用とは限りません。特に、デバイスによっては*.zipファイルから実行されるものもあるので、実際にゲームがどこで実行されていても同じ結果が返されます。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

注意：サンドボックスを無効にした場合でも、必要なゲームファイルを削除してしまう問題を防ぐために、このディレクトリへの書き込みはできません。

構文です。

program_directory

リターンです。

例

dir = program_directory;

これにより、実行ファイルが格納されているディレクトリが変数に格納されます。
