temp_directory（テンポラリーディレクトリ

これは、ゲームを実行するたびに作成される一時ディレクトリを返すために使用できます（ルートには最後の "second "が含まれません）。このディレクトリはファイルを保持し、ゲームの実行中にアクセスすることができますが、(ルートと一緒に)削除されます。 ゲームを閉じたときに、その中に含まれるすべてのファイルとともに。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

temp_directory

リターンです。

例

ini_open(temp_directory + "\temp_ini.ini");

これは、ゲームの一時ディレクトリにiniファイルを開く（まだ存在しない場合は作成する）ものです。
