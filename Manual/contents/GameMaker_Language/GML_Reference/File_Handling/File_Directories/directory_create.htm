ディレクトリ作成

この関数は、保存領域に指定された名前のディレクトリを作成します。

注）HTML5では、ブラウザのローカルストレージにディレクトリを作成したり削除したりできないため、この機能は動作しません。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

directory_create(dname)

論証タイプ 商品説明
dname The name of the directory to create.

リターンです。

例

if !directory_exists("Games")
{
    directory_create("Games");
}

これは、指定されたディレクトリがローカルのデータフォルダに存在するかどうかを確認し、存在しない場合は作成してくれるものです。
