directory_destroy

説明： 保存領域内の指定された名前のディレクトリを削除する。

注）HTML5では、ブラウザのローカルストレージにディレクトリを作成したり削除したりできないため、この機能は動作しません。

警告!この機能はGameMakerがサンドボックス化されているため、期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

directory_destroy(dname)

論証タイプ 商品説明
dname The name of the directory to remove.

リターンです。

例

if directory_exists("DLC")
{
    directory_destroy("DLC");
}

これは、指定されたディレクトリがローカルデータフォルダに存在するかどうかを確認し、存在する場合は削除されます。
