directory_exists

この関数は、指定されたディレクトリが存在する場合は true を、存在しない場合は false を返します。path 指定された名前は、相対パスではなく完全パスpath でなければなりません。また、デフォルトでは、すべてのゲームはサンドボックス化されているため、ゲームバンドル外からディレクトリにアクセスできません（詳しくは、ファイルシステムのセクションを参照してください）。

構文です。

directory_exists(dname)

論証タイプ 商品説明
dname The name of the directory to look for.

リターンです。

例

if directory_exists(working_directory + "Saves/")
{
    file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

指定されたディレクトリが存在するかどうかを確認し、存在する場合はそこに移動して、最初に見つかった「読み取り専用」の doc ファイルを返します。
