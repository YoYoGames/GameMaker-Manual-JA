md5_string_utf8

暗号技術において、MD5 (Message-Digest algorithm 5) は128ビットのハッシュ値を持つ暗号ハッシュ関数として広く使われており、様々なセキュリティアプリケーションで採用されている。また、ファイルやstrings の整合性をチェックするためにもよく使われています。 この関数は、入力の UTF-8string (1文字あたりのバイト数は可変) を受け取り、そのstring に固有の 32 文字の 16 進法 MD5 ハッシュを返します。このようにして、安全な鍵を生成し、それを保存して使用することができます。 に送信する（または受信する）情報の完全性をチェックするために、外部のserver （例）。

注：MD5エンコードには、UTF-8とunicodeの2つの形式があります。どちらも異なるserver セットアップでの通信を容易にするために提供されていますが、最も一般的に使用されるのは unicode です。

注：MD5は完全に安全ではなく、破られる可能性があります。詳しくはこちらのページをご覧ください。

構文です。

md5_string_utf8(string)

論証タイプ 商品説明
string The string to hash.

リターンです。

例

var hash, str;
str = json_encode(hiscore_map);
hash = md5_string_utf8(str);
ini_open("local.ini");
ini_write_string("info", "0", hash);
ini_close();
get[0] = http_post_string("http://www.MacSweeney Games.com/CatchTheHaggis?game_hiscores=" + string(global.game_id), str)

上記のコードは、DSマップをJSON（string ）にエンコードするものです。MD5 ハッシュが生成され、ini ファイルに保存されます。これは、後でserver から同じ情報を受け取ったときに、JSON の整合性をチェックするために使用できるようにするためです。 が送信される。
