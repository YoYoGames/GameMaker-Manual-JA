zip_unzip（ジップアンジップ

この関数は、保存されている zip ファイルを開き、その内容を指定されたディレクトリに展開します。ZIP ディレクトリに完全なpath を指定しない場合、現在のドライブルートが使用されることに注意してください。また、ゲームバンドルの作業ディレクトリから相対的にpath に配置したい場合は working_directory変数をpath の一部として使用します（"." や "..." を使用した相対的なpaths は期待通りに動作しないので避けるべきです）。zipはゲームバンドルに含まれているか（つまりインクルードファイル）、http_get_file()を使ってストレージにダウンロードされている必要があることにも注意してください。

この関数は、抽出されたファイルの数を示す値を返し、抽出に失敗した場合は 0 以下を返す。

構文です。

zip_unzip(zip_file, target_directory)

論証タイプ 商品説明
zip_file The zip file to open
target_directory The target directory to extract the files to

リターンです。

例

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if (num <= 0)
{
    show_debug_message("Extraction Failed!");
}

上記のコードは、downloadsディレクトリに格納されているzipファイルを開き、その内容をextractedディレクトリに展開し（まだ存在しない場合はそのディレクトリを作成）、展開が正しく行われたかどうかをチェックし、失敗した場合にはデバッグメッセージを表示します。
