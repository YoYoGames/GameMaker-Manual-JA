base64_encode

この関数は、string を base64 フォーマットでエンコードされたstring に変換します。これは、テキストとしてインターネット上で保存または転送する必要があるあらゆるメディアでよく使われるエンコーディング方式で、出力が読めなくなるようにします。 人間の目

構文です。

base64_encode(string)

論証タイプ 商品説明
string The string to encode.

リターンです。

例

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

上記のコードは、"game_data "に格納されているstring をbase64エンコードしたstring に変換し、外部のテキストファイルに格納します。
