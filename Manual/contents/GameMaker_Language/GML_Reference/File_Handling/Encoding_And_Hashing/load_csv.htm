ロードシーブイ

この関数は、CSV 形式のファイルを読み込んで DS グリッドに変換し、作成されたグリッドの一意の ID 値を返す。

CSVファイルは、CSVの仕様に沿ったものである必要があります。CSVの構文が正しくない場合、ファイルを読み込む際に予期せぬ結果を得ることになります。

構文です。

load_csv(filename)

論証 タイプ 商品説明
filename 文字列 The name of the file to open (as a string)

リターンです。

DS Grid ID

例

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
    for (var j = 0; j < hh; j++;)
    {
        draw_text(xx, yy, file_grid[# i, j]);
        yy += 32;
    }
    yy = 32;
    xx += 32;
}

上記のコードでは、与えられた CSV ファイルを開き、返された DS グリッドを変数 "file_grid" に格納する。このグリッドは for のループでパースされ、その内容が画面に描画される。
