md5_file

暗号技術において、MD5 (Message-Digest algorithm 5) は128ビットのハッシュ値を持つ暗号ハッシュ関数として広く使われており、様々なセキュリティアプリケーションで採用されている。また、ファイルの整合性チェックにもよく使われ、strings 。 この関数は、与えられたファイルを受け取り、そのファイルに対して一意のMD5を生成し、後で使用するために保存することができます。

注：MD5は完全に安全ではなく、破られる可能性があります。詳しくはこちらのページをご覧ください。

構文です。

md5_file(filename)

論証タイプ 商品説明
filename The file to generate the MD5 hash for.

リターンです。

例

hash = md5_file(working_directory + "game_data.ini")

上記のコードは、指定されたファイルに対してMD5ハッシュを生成し、返された値を変数「hash」に格納するものである。
