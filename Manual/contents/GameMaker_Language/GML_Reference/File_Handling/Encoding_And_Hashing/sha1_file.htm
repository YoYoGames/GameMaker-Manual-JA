sha1_file

暗号技術において、SHA-1は米国国家安全保障局によって設計された暗号ハッシュ関数で、ファイルの変更をチェックするために使用される人気の高いGitや、TLSとSSL、PGP、SSH、S/MIME、IPsecなどのプロトコルに広く採用されています。この関数は、入力ファイルを受け取り、そのファイルに固有の160ビットのメッセージダイジェストをASCII形式で返します。

注意：SHA-1は完全に安全ではなく、破られる可能性があります。詳しくはこのページをご覧ください。

構文です。

sha1_file(filename)

論証 タイプ 商品説明
filename 文字列 The file to generate the sha1 hash for.

リターンです。

String

例

hash = sha1_file(working_directory + "game_data.ini")

上記のコードは、指定されたファイルに対してsha1ハッシュを生成し、返された値を変数「hash」に格納するものである。
