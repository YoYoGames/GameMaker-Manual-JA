md5_string_unicode

暗号技術において、MD5 (Message-Digest algorithm 5) は128ビットのハッシュ値を持つ暗号ハッシュ関数として広く使われており、様々なセキュリティアプリケーションで採用されている。また、ファイルやstrings の整合性をチェックするためにもよく使われています。 この関数は、入力のunicodestring (各文字は16ビット) を受け取り、そのstring に固有の32文字の16進数MD5ハッシュを返します。このようにして、安全な鍵を生成することができます。 の情報を外部のserver （例）に送信する（または受信する）ことができます。

注意事項 MD5のエンコードには、UTF-8とunicodeの2つの形式があります。どちらも異なるserver セットアップでの通信を容易にするために提供されていますが、最も一般的に使用されるのは unicode です。

注：MD5は完全に安全ではなく、破られる可能性があります。詳しくはこちらのページをご覧ください。

構文です。

md5_string_unicode(string)

論証タイプ 商品説明
string The string to hash.

リターンです。

例

var hash, str;
str = base64_encode(game_data);
hash = md5_string_unicode(str);
http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str);

上記のコードでは、string を base64 でエンコードし、MD5 ハッシュを生成する。最後に、そのハッシュとエンコードされたstring の両方がserver に送られる。
