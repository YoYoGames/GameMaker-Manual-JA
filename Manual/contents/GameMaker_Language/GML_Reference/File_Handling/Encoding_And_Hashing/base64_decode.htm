base64_decode

この関数は、以前 base64 フォーマットでエンコードされたstring を標準テキストに変換します。Base64は一般によく使われる符号化方式で、インターネット上でテキストとして保存または転送する必要があるあらゆるメディアでよく使われ、また、そのようなメディアをレンダリングします。 は、人間の目には読めない出力です。

構文です。

base64_decode(string)

論証タイプ 商品説明
string The string to decode.

リターンです。

例

var str, file;
str = base64_encode(game_data);
file = file_text_open_read("save.txt");
str = file_text_read_string(file); level_data = base64_decode(str);
file_text_close(file);

上記のコードでは、テキストファイルを開き、そこからstring をローカル変数 "str" に読み込んでいます。次にこのstring をデコードして、その結果をインスタンス変数 "level_data" に格納する。
