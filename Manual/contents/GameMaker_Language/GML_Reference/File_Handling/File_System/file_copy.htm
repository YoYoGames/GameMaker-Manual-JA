ファイルコピー

この関数は、指定されたファイルをコピーし、指定された新しい名前を付けます。

Android でインクルードファイル領域（APK 内）からファイルをコピーする場合、まずメモリにコピーされます。したがって、APK から非常に大きなファイルをコピーすると、デバイスのメモリ不足を引き起こす可能性がありますので、注意してください。HTML5 ターゲットでは、クライアントサイドの保存に HTML5Local Storage APIを使用しており、この関数はバイナリファイル操作をサポートしていないため、テキストファイルをコピーすることを想定しています。

newnameファイルがすでに存在する場合、この関数は失敗します。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステムの制限」の項をご覧ください。

構文です。

file_copy(fname, newname);

論証タイプ 商品説明
fname The name of the file to copy.
newname The name of the new, copied file.

リターンです。

N/A

例

if file_exists("level1.txt")
{
    file_copy("level1.txt", "level2.txt");
}

これは、ファイルがあるかどうかをチェックし、存在すればコピーするものです。
