ファイル名_dir

この関数は，指定されたファイル名のディレクトリ部分を返す。これは通常，最後のバックスラッシュを除いてpath と同じである。

構文です。

filename_dir(fname);

論証タイプ 商品説明
fname The file to use.

リターンです。

例

dir = filename_dir("Test.ini");

上記のコードは、指定されたファイルのディレクトリを（string ）として取得します。
