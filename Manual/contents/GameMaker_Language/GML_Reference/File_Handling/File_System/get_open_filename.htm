get_open_filename（ファイル名

この関数は、ダイアログを開き、与えられたフィルタで開くファイル名をプレイヤーに尋ねます。フィルタは "name1|mask1|name2|mask2|... " という形式で、マスクはセミコロンで区切られた異なるオプションを含むことができ、また "*" は任意のstring を意味することができます。例えば、"bitmaps|*.bmp;*.wmf" は、システムからビットマップファイルの名前を選択するようにユーザに要求しますが、指定した2つのextensions (bmp と png) を持つものだけです。この関数は、ファイル自体を開くわけではありません!ファイルのフルネームとstring を含むpath を返すだけです。ユーザーが「キャンセル」を押した場合は、空のstring "" が返されます。

この機能を使うときの注意点は、ゲームの間、そのファイルに対して一定の権限を付与することです。そのため、一度ユーザーからファイルpath を取得すれば、何度でも問い合わせることなくアクセスすることができます。ただし、この機能はファイルを読むためのものなので、付与される権限はあくまで読むためのものであり、保存はOSが許可してくれるわけではありません。万が一、ファイルの読み書きが必要な場合は、以下の関数で許可を得ることができます。 get_save_filename().

注：この機能は、Windows およびmacOS のターゲットでのみ有効です。

構文です。

get_open_filename(filter, fname);

論証タイプ 商品説明
filter The file type filter and/or name.
fname The suggested filename to use.

リターンです。

例

var file;
file = get_open_filename("text file|*.txt", "");
if (file != "")
{
    file_text_open_read(file);
}

上記のコードは、ユーザーにテキストファイルを開くかどうかを尋ね、返されたファイル名とpath 、それが存在する場合は、読み取りのためにファイルを開くことを確認します。
