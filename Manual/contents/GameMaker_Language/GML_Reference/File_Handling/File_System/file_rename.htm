ファイル名変更

説明： 指定されたファイルを指定された名前にリネームする。この関数は，ファイル名の変更に成功した場合は true を，それ以外の場合は false を返す。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

file_rename(oldname, newname);

論証タイプ 商品説明
oldname The name of the file to change.
newname The new name to give the file.

リターンです。

例

if file_exists("level1.txt")
{
    file_rename("level1.txt", "level.txt");
}

これは、ファイルの有無を確認し、存在すれば名前を変更するものです。
