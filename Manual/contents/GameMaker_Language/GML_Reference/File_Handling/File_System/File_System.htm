ファイルシステム

GameMakerには、あらゆる種類のファイルをさまざまな方法で操作するための関数が数多く存在します。

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

ファイル名を変更するには、次の機能を使用します。

注：これらの関数は実際のファイルには作用しません。これらの関数は、異なるstrings 要素に関連するpath を扱うだけであり、他のファイル関数と組み合わせて使用する必要があります。

  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

最後に、ファイルタイプフィルターでファイルの読み込みや保存に使用できるファイル名を取得するための特別な関数がいくつかあります。これらの関数はエクスプローラーウィンドウを開くので、ゲームのフォーカスが外れます。このため、ゲームが依存している入力が正しく登録されない可能性があります。そのため、（これらの関数をキーボードイベントで使用する場合は）pressed や down イベントではなく、keyupイベントを使用することをお勧めします。

NOTEこれらの機能は、Windowsと macOSのみです。

警告 GameMakerはサンドボックス化されているため、これらの機能は期待通りに動作しない可能性があります。詳しくは「ファイルシステム」の項をご覧ください。

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
