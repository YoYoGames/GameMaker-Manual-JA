get_save_filename

この関数はダイアログを開き、指定されたフィルタで保存するファイル名をプレイヤーに尋ねます。フィルタは "name1|mask1|name2|mask2|... " という形式で、マスクはセミコロンで区切られた異なるオプションを含むことができ、また "*" は任意のstring を意味することができます。例えば、"bitmaps|*.bmp;*.wmf" は、システムからビットマップファイルの名前を選択するようにユーザに要求しますが、指定した2つのextensions (bmp と png) を持つものだけです。この関数はファイル自体を保存しません!ファイルのフルネームとstring を含むpath を返すだけです。ユーザーが「Cancel」を押すと、空のstring "" が返されます。

この機能を使うときの注意点は、ゲームの間、そのファイルに対して特定のパーミッションを与えることです。つまり、一度ユーザーからファイルpath を取得すれば、問い合わせることなく何度でもアクセスすることができ、そのパーミッションは読み書きに等しく適用されます。

注：この機能は、Windows およびmacOS のターゲットでのみ有効です。

構文です。

get_save_filename(filter, fname);

論証タイプ 商品説明
filter The file type filter and/or name.
fname The suggested filename to use.

リターンです。

例

var file;
file = get_save_filename("screenshot|*.png", "");
if (file != "")
{
    screen_save(_file);
}

上記のコードは、保存する画像ファイル名を選択するようユーザーに求め、返されたファイル名とpath 、それが空でなければstring 、スクリーンショットを取り、与えられた名前を使用してそれを保存することを確認することになります。
