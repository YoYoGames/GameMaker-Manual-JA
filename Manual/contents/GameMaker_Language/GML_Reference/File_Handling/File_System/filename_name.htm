ファイル名

この関数は、指定されたファイルの名前部分を、extension を付けて、path を付けずに返します。

構文です。

filename_name(fname);

論証タイプ 商品説明
fname The file to use.

リターンです。

String

例

name = filename_name(file_find_first("C:/Games/*.doc", 0));

上記のコードは、指定されたディレクトリで最初に見つかった「doc」タイプのファイルの名前を（string ）取得します。
