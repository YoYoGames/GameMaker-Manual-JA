ファイル検索

この関数は，マスクと属性を満足する最初のファイルの名前を返す。そのようなファイルが存在しない場合，空のstring が返される。

マスクには、path 、C: \temp*.doc のようなワイルドカードを使用できます。

属性は見たい追加ファイルを与えるので、通常のファイルはマスクを満たせば必ず返されます。以下の定数を足すことで、見たいファイルの種類を見ることができます（｜を使用）。属性を追加したくない場合は、0を使用します。

ファイル属性定数
定数 商品説明
fa_readonly 読み取り専用ファイル
fa_hidden 隠しファイル
fa_sysfile システムファイル
fa_volumeid ボリュームIDファイル
fa_directory ディレクトリー
fa_archive アーカイブされたファイル

属性はWindows でのみ使用可能です。その他のプラットフォームでは、0を使用してください。

警告 GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

注）この機能は、HTML5ターゲットでは全く機能しません。

構文です。

file_find_first(mask, attr);

論証 タイプ 商品説明
mask 文字列 The mask to use for searching.
attr ファイル属性定数(s) The specific file attribute to look for.

リターンです。

String

例

if directory_exists("\User Content")
{
    fileA = file_find_first("/User Content/*.doc", fa_readonly);
    fileB = file_find_next();
    fileC = file_find_next();
    file_find_close();
}

このコードは、指定されたディレクトリが存在するかどうかをチェックし、存在する場合はそこに行き、最初に見つかった「読み取り専用」 .doc ファイルを返します。その後、さらに2つのファイルを探し、ファイルファインダーを閉じます。

を使用すると、任意の数のファイルを検索することができます。 whileをループさせる。

var files = [];
var file_name = file_find_first("/User Content/*.doc", fa_readonly);

while (file_name != "")
{
    array_push(files, file_name);

    file_name = file_find_next();
}

file_find_close();

上記のコードでは、見つかったすべてのファイル名を格納する空の配列を作成し、読み取り専用のファイル（ .doc ）の検索を開始します。そのファイル名が空のstring でない場合、それを files 配列にプッシュし、次のファイルを探そうとします。loop は、空のstring が見つかる（一致するファイルがもうないことを意味する） まで繰り返され、その時点でloop が終了し、最後にファイルファインダーが閉じられる。
