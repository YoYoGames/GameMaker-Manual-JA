filename_ext

この関数は、指定されたファイル名の先頭のドットを含むextension 部分を返します。

構文です。

filename_ext(fname);

論証タイプ 商品説明
fname The file to use.

リターンです。

例

ext = filename_ext(file_find_first("*.*", 0));

上記のコードは、指定されたファイルのextension （string ）として取得します。
