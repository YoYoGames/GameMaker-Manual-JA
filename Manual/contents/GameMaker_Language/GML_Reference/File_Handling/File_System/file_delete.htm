ファイル削除

この関数は、指定されたファイルをシステムから削除します。この関数はGameMakerが作成し解析できるファイル、すなわちiniファイル、テキストファイル、バイナリファイル、またはsprites やサーフェスなどのゲーム作成リソースを格納するために作られたファイルのみを削除することに注意する必要がある。ただし、それ以外のファイルは削除されない。また、この関数は、ファイルが正常に削除された場合は true を、それ以外の場合は false を返す。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステムの制限」の項をご覧ください。

構文です。

file_delete(fname);

論証タイプ 商品説明
fname The name of the file to delete.

リターンです。

例

if file_exists("level.txt")
{
    file_delete("level.txt");
}

これは、ファイルがあるかどうかをチェックし、存在すれば削除するというものです。
