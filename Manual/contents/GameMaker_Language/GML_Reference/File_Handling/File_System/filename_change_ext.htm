ファイル名変更

extension この関数は，指定されたファイル名を，ドットも含めて新しいextension に変更したものを返します． 新しいstring として空のextension を使用すれば，extension の部分をすべて削除することができます．

構文です。

filename_change_ext(fname, newext);

論証タイプ 商品説明
fname The file to use.
newext The new extension to use.

リターンです。

例

ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), "");

上記のコードは、string の部分を削除したファイル名（extension として）を取得します。
