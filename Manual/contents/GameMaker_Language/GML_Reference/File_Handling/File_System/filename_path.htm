ファイル名パス

この関数は、指定されたファイルpath のpath の部分を、最後のバックスラッシュを含めて返します。

構文です。

filename_path(fname);

論証タイプ 商品説明
fname The file name and path to use.

リターンです。

例

path = filename_path(working_directory + "Test.ini");

上記のコードは、指定されたファイルのpath （string ）として取得します。
