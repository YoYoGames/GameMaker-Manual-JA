get_save_filename_ext

この関数は、ダイアログを開き、与えられたフィルタで保存するファイル名をプレイヤーに尋ねます。フィルタは "name1|mask1|name2|mask2|... " という形式で、マスクはセミコロンで区切られた異なるオプションを含むことができ、また "*" は任意のstring を意味することができます。例えば、"bitmaps|*.bmp;*.wmf" は、システムからビットマップファイルの名前を選択するようにユーザに要求しますが、指定した2つのextensions (bmp と png) を持つものだけです。この関数はファイル自体を保存しません!ファイルのフルネームとstring を含むpath を返すだけです。ユーザーが「キャンセル」を押した場合は、空のstring "" が返されます。また、検索を開始するディレクトリや、ウィンドウの上部に表示するキャプションを、デフォルトのキャプションの代わりに追加することができます。

この機能を使うときの注意点は、ゲームの間、そのファイルに対して特定のパーミッションを与えることです。つまり、一度ユーザーからファイルpath を取得すれば、問い合わせることなく何度でもアクセスすることができ、そのパーミッションは読み書きに等しく適用されます。

注：この機能は、Windows およびmacOS のターゲットでのみ有効です。

構文です。

get_save_filename_ext(filter, fname, directory, caption);

論証タイプ 商品説明
filter The file type filter and/or name.
fname The suggested filename to use.
directory The directory to start the search from.
caption The caption for the window.

リターンです。

例

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
{
    screen_save(file);
}

上記のコードは、作業ディレクトリに保存する画像ファイル名ファイルを選択するようユーザーに求め、返されたファイル名をチェックし、path 空でなければstring スクリーンショットを撮影し、与えられた名前を使用して保存します。
