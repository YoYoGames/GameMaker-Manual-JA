file_attributes

この関数は、指定されたファイルの属性を確認するために使用することができます。以下の定数を足すことで、ファイルの種類を確認することができます。

定数 商品説明
fa_readonly 読み取り専用ファイル
fa_hidden 隠しファイル
fa_sysfile システムファイル
fa_volumeid ボリュームIDファイル
fa_directory ディレクトリー
fa_archive アーカイブされたファイル

注意：これはWindows のみの機能です。

構文です。

file_attributes(fname, attr);

論証 タイプ 商品説明
fname 文字列 The name of the file to check.
attr ファイル属性定数 The attributes to check for.

リターンです。

Boolean

例

if !file_attributes(file, fa_hidden)
{
    file_delete(file);
}

これは、ファイルが隠されているかどうかをチェックし、隠されていない場合は削除するものです。
