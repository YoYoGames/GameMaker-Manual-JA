ファイルエグジスタンス

この関数は、指定されたファイルが存在する場合は true を、存在しない場合は false を返します。この関数はローカルファイルのチェックにのみ使用でき、リモートのserver に保存されているファイルはチェックできないことに注意してください。

構文です。

file_exists(fname);

論証タイプ 商品説明
fname The name of the file to check for.

リターンです。

例

if file_exists("level.txt")
{
    file = file_text_open_read("level.txt");
}

これは、ファイルがあるかどうかをチェックし、存在すれば読み込みのためにオープンするものです。
