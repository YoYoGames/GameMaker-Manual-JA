ファイル_bin_seek

この関数は、ファイル内の現在の読み取り位置を、指定された位置に移動させる。この関数が返すファイルIDの値を指定します。 file_bin_open()を移動させ、ファイルを追加する場合は、その位置を の大きさにしてから書き込む。

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_seek(binfile, pos);

論証タイプ 商品説明
binfile The ID of the file to read from.
pos The position in the file to move to.

リターンです。

例

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

これは、ゲームと同じディレクトリにあるファイルを開いて、そのインデックスを変数「file」に代入し、ファイルのサイズを取得して、次の書き込み位置をそのサイズに設定するというものです。
