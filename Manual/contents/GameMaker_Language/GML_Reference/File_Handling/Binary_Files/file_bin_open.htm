ファイル_bin_open

この関数は、指定された名前のバイナリファイルをオープンする。モードは，そのファイルに対して何ができるかを示す。

  • 0 = 読み取り
  • 1 = 書き込み
  • 2 = 読み書き両方

ファイルが存在しない場合は作成され、他の関数で使用する必要があるファイルのIDが返されます。同時に複数のファイルを開くことができますが（最大32個）、終了したらファイルを閉じることを忘れないでください。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_open(fname, mode);

論証 タイプ 商品説明
fname 文字列 The name of the file to read from.
mode リアル The indicator of what can be done with the file.

リターンです。

Binary File ID

例

file = file_bin_open("myfile.bin", 2);

これは、ゲームと同じディレクトリにあるファイルを開き、そのインデックスを変数「file」に代入するものです。
