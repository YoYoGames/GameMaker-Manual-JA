ファイル_bin_read_byte

この関数は、与えられたファイルIDを持つファイル内の現在の位置から1バイトのデータを返す。この関数が返すファイルIDの値を指定します。 file_bin_open().

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_read_byte(binfile);

論証タイプ 商品説明
binfile The ID of the file to get the data from.

リターンです。

例

file = file_bin_open("myfile.bin", 2);
data = file_bin_read_byte(file);
file_bin_close(file);

これは、ゲームと同じディレクトリにあるファイルを開き、そこから1バイトのデータを取得してから、再び閉じるというものです。
