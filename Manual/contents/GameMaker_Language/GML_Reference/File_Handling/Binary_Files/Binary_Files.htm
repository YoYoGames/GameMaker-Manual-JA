バイナリファイル

まれに、バイナリファイルからデータを読み込む必要がある場合があります。一般的には、バイナリデータの作成、保存、読み込みには、これらの関数ではなく、専用のバッファ関数を使用することをお勧めいたしますが、これらの関数はその目的のために使用することができます。

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

このために以下の低レベルルーチンが存在します。

  • file_bin_open
  • file_bin_rewrite
  • file_bin_close
  • file_bin_size
  • file_bin_position
  • file_bin_seek
  • file_bin_write_byte
  • file_bin_read_byte
