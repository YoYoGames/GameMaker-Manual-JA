ファイル_bin_close

以前に開いたバイナリファイルを閉じる。ファイルIDの値は、この関数によって返されます。 file_bin_open().

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_close(binfile);

論証タイプ 商品説明
binfile The ID of the file to close.

リターンです。

例

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

これは、ゲームと同じディレクトリにあるファイルを開き、そのインデックスを変数「file」に代入するものです。その後、ファイルを書き換えて（データを消去して）、再び閉じます。
