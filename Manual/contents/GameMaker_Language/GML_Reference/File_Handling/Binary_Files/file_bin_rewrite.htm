ファイル_bin_rewrite

この関数は、関数 file_bin_open()を書き直し、それまでのデータをすべてクリアして、ファイルの先頭から書き込みを開始します。

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_rewrite(binfile);

論証タイプ 商品説明
binfile The ID of the file to rewrite.

リターンです。

例

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);

これは、ゲームと同じディレクトリにあるファイルを開き、そのインデックスを変数「file」に代入するものです。そして、そのファイルを書き換えます（データを消去します）。
