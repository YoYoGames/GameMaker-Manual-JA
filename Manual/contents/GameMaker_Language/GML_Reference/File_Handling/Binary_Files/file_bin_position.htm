file_bin_position

この関数は、与えられたファイルIDのファイルの現在位置をバイト単位で返します（0は最初の位置）。この関数が返すファイルIDの値を指定します。 file_bin_open().

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_position(binfile);

論証タイプ 商品説明
binfile The ID of the file to get the position in.

リターンです。

例

pos = file_bin_position(file);

そうすると、現在の位置が変数 "pos "に格納される。
