file_bin_write_byte

この関数は、現在の書き込み位置のファイルIDで識別されるファイルに1バイトのデータを書き込みます。この関数が返すファイルIDの値を指定します。 file_bin_open()と、書き込むデータのバイト数を指定する。

注）対象モジュールがHTML5の場合、これらの機能は動作しません。

構文です。

file_bin_write_byte(binfile, byte);

論証タイプ 商品説明
binfile The ID of the file to write to.
byte The data to write.

リターンです。

例

file_bin_write_byte(file, data);

これは、選択されたファイルに1バイトを書き込むことになります。
