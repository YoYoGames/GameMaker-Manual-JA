ini_open

ini_fileを読み書きのためにオープンします。チェックした場所にini_fileが存在しない場合、GameMakerはini_fileを作成するかもしれませんが、それはあなたがデータを書き込む場合のみです。iniファイルから情報を読み取るだけであれば、以下のようになります。 read関数のデフォルト値は返されるが、iniファイルは実際には作成されない。

一度に開くことができるのは1つのiniファイルのみであることに注意してください。 ini_close().iniファイルからの読み書きが終了しても、その情報は実際には保存されないので ディスクに保存されます（ファイルを閉じるまでメモリにも保存されます）。

警告!この機能はGameMakerがサンドボックス化されているため、期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

ini_open(name);

論証タイプ 商品説明
name The filename for the .ini file.

リターンです。

例

ini_open("Settings/savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

これは、「savedata.ini」を開き、「save1」というセクションの「score」というキーがある部分のスコア値を読み込み、再度.iniを閉じます。save1"、"score "の下に値がない場合、または "savedata.ini "が存在しない場合は、"savedata.ini "を閉じます。 ファイルが存在する場合、scoreは0（デフォルト値）に設定されます。なお、このiniファイルは、Asset Browserの付属ファイルの中で、サブディレクトリ "Settings "に配置されています。
