ini_open_from_string

この関数は、string から一時的な ini ファイルを作成し、それを開いて読み書きを行います。string は、ini ファイルとして正しくフォーマットされている必要があります (つまり、セクション、改行キー、値がある)。そうでない場合、ini ファイルは正しく作成されません。この ini ファイルは一時的なもので、閉じた瞬間にメモリから削除され、保存されていたすべての情報が失われることに注意してください。 ini_close()関数は、完全なiniファイルのstring を返し、server またはディスクに保存することができます。

構文です。

ini_open_from_string(string);

論証タイプ 商品説明
string The string containing all the ini information.

リターンです。

N/A

例

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上記のコードでは、string "str "からの情報を保持するために一時的なiniを設定し、iniから読み出してから再び閉じています。
