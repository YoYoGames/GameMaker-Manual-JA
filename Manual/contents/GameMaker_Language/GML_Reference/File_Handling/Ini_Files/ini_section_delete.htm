ini_section_delete

この機能を使うと、iniファイルのセクション全体を削除することができ、それに関連するすべてのKey-Valueペアも削除されます。

構文です。

ini_section_delete(section);

論証タイプ 商品説明
section The section to delete.

リターンです。

例

ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
ini_section_delete("save1");
ini_close();

この例では、"savedata.ini "を開き、"save1" > "Score "に値を書き込みます。そして、"save1 "セクションを削除し、.iniファイルを閉じます。
