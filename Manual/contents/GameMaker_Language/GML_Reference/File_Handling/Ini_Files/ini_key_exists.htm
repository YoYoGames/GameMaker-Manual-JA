ini_key_exists

この関数は、現在開いている ini にキーが存在するかどうかを調べ、存在する場合は true を、そうでない場合は false を返します。キーが存在しない場合、存在しないキーから読み込んでもデフォルト値が返されるだけなので、エラーを防ぐために必要なチェックではありません。しかし、iniファイルが特定のデータを保存しているかどうかを確認したり、その他のいくつかのことを確認するのに便利です。

構文です。

ini_key_exists(section, key);

論証タイプ 商品説明
section The section in the open .ini file to check a key in.
key The key to check for.

リターンです。

例

ini_open("savedata.ini");
if (!ini_key_exists("save1", "name"))
{
    global.name = "Player1";
}
ini_close();

name "というキーが存在しない場合、変数global.nameに "Player1 "が設定されます。
