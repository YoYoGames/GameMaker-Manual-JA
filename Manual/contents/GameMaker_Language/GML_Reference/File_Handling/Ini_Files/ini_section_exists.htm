ini_section_exists

この関数は、現在開いている ini にセクションが存在するかどうかを調べ、存在する場合は true を、そうでない場合は false を返します。セクションが存在しない場合、存在しないセクションから読み込んでもデフォルト値が返されるだけなので、エラーを防ぐために必要なチェックではありませんが、iniファイルに特定のデータが保存されているかどうかを確認するのに便利です。

構文です。

ini_section_exists(section);

論証タイプ 商品説明
section The section in the open .ini file to check for.

リターンです。

例

ini_open("savedata.ini");
if !ini_section_exists("save1")
{
    global.savegame = "1";
}
ini_close();

save1 "が見つからない場合は、変数 global.savegame に "1" が設定されます。forloop を使えば、1つのゲームにいくつのセーブゲームがあるかを調べる基本的な方法として使うことができます (たとえば、1つのゲームに複数のセーブゲームを同時に行うシステムを作ることができます)。
