ini_write_real

iniデータファイルに値（数値）を書き込むときに使用します。Iniファイルはセクションに分割され、その後、各セクションはキーと 値のペアに分割されます。ですから、典型的なiniファイルは、次のようなものです。

構文です。

ini_write_real(section, key, value);

論証タイプ 商品説明
section The section of the .ini to write to.
key The key within the relevant section of the .ini to write to.
value The real value to write to the relevant destination.

リターンです。

例

score = 1000;
ini_open("savedata.ini");
ini_write_real("save1", "Score", score );
score2 = ini_read_real("save1", "Score", 0 );
ini_close();

これは、scoreを1000に設定し、savedata.iniを開いて、この値をsave1 > Scoreに書き込むというものです。そして、score2をその中の「save1」>「score」に実際の値である1000に設定し、.iniを閉じます。 を再度実行します。この値が見つからなければ、score2は0に設定されたまま終わっていたでしょう。
