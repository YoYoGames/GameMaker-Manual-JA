ini_read_string

ini データファイルからstring (テキスト)を読み込むことができます。Ini ファイルはセクションに分割され、その後各セクションはキーと 値のペアに分割されます。ですから、典型的なiniファイルは、次のようなものです。

構文です。

ini_read_string(section, key, default);

論証 タイプ 商品説明
section 文字列 The section of the .ini to read from.
key 文字列 The key within the relevant section of the .ini to read from.
default 文字列 The string to return if a string is not found in the defined place (or the .ini file does not exist). Must be a string.

リターンです。

String

例

ini_open("savedata.ini");
global.name = ini_read_string("player", "name', "Player1");
ini_close();

savedata.iniを開き、global.nameにsave1>Nameのstring を設定し、再度.iniを閉じると、savedata.iniが開きます。save1" > "Name "にstring がない場合、または savedata.ini がない場合は、global.name は "Player1" に設定されます。
