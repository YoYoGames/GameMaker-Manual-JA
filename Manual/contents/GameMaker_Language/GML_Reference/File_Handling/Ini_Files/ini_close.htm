イニクローズ

この関数は、開いているiniファイルの読み書きを終了した時点で呼び出す必要があります。iniの書き込み関数を使った後にこの関数を使わないと、この関数が呼ばれて書き込みが行われるまで、ファイル情報がメモリに保持されるため、ディスクには何も書き込まれません。別のini（または同じもの）を閉じた後でiniを開こうとすると、これもエラーになります。

また、この関数は ini ファイルをエンコードしたstring を返します。このstring はserver に保存することができます。 ini_open_from_string()を使って、ini を再作成することができます。

構文です。

ini_close();

リターンです。

String

例

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

savedata.iniを開き、save1というセクションからscoreというキーを持つ値を読み込んでiniファイルを閉じます。save1 "の下に "score "という値がない場合、または "savedata.ini "が存在しない場合は0が返され、新しいiniファイルが作成されます。
