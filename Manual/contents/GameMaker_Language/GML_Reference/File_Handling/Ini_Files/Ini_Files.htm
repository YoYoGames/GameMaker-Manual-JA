Iniファイル

INIファイルは、ほとんどのプラットフォームと互換性のある、小さくて軽いファイルです。インターフェイスの設定、ローカルハイスコア、レベルデータなど、小さな情報を保存するのに最適で、ゲームから情報を保存するのに最適で簡単な方法です。もし、より高度なファイル保存方法をお探しなら、テキストファイル、バイナリファイル、バッファなどの他のファイル機能をご覧になってみてください。

重要!すべてのGameMakerプロジェクトには "options.ini" という名前のファイルがあるので、その名前を付けてはいけません（ゲームプロジェクトをコンパイルしようとすると、uid failure が発生します）。

ローカライゼーションのあるゲームでは、アクセント付きの文字を外部ファイルから読み込めるようにすることが非常に重要であることに注意してください。つまり、UTF8形式のiniファイルを先に作成し、GameMakerにインクルードファイルとして追加しておくと、ゲーム実行時にエクスポートされ、ini関数で作成されるデフォルトのANSI形式のiniファイルの代わりに使用されます（事前に何も指定されていない場合）。こうすることで、アクセントやローマ字以外の文字が維持されたまま、正しく読み書きができるようになります。

iniファイルには、以下のような機能があります。

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
