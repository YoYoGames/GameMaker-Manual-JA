ini_write_string

ini データファイルにstring (テキスト)を書き込むことができます。Ini ファイルはセクションに分割され、その後各セクションはキーと 値のペアに分割されます。ですから、典型的なiniファイルは、次のようなものです。

inistrings を記述する際に引用符やエスケープ文字を使用すると、後でデータを読み返す際に、早々に切り捨てられる可能性があり、問題が発生する可能性があることに留意する必要があります。例えば、このように書くと問題になります。

ini_write_string("test2", "section", "hello \"Fritz\"");

構文です。

ini_write_string(section, key, value);

論証 タイプ 商品説明
section 文字列 The section of the .ini to write to.
key 文字列 The key within the relevant section of the .ini to write to.
value 文字列 The string to write to the relevant destination.

リターンです。

N/A

例

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

上記のコードは、読み書きのためにiniファイルを開き、グローバル変数「Name」に格納されているstring を「Player」をキーに「Save」セクションに書き込んでから、再びファイルを閉じています。
