ini_key_delete（キー削除

この機能を使うと、選択したキー（とそれに対応する値）をiniファイルから削除することができます。

構文です。

ini_key_delete(section, key);

論証タイプ 商品説明
section The section to delete a key from.
key The key to delete.

リターンです。

例

ini_open("savedata.ini");
ini_write_real("save1","Score",score);
ini_key_delete("save1","Score");
ini_close();

この例では、"savedata.ini "を開き、"save1">"Score "に値を書き込んでいます。そして、"Score "キーを削除し、.iniファイルを閉じます。
