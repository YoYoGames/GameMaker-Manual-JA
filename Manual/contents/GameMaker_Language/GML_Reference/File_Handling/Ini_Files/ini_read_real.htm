ini_read_real

この関数は、iniデータファイルから数値を読み取るために使用します。Iniファイルはセクションに分割され、その後、各セクションはキーと 値のペアに分割されます。ですから、典型的なiniファイルは次のようなものです。

構文です。

ini_read_real(section, key, default);

論証タイプ 商品説明
section The section of the .ini to read from.
key The key within the relevant section of the .ini to read from.
default The value to return if a value is not found in the defined place (or the .ini file does not exist). Must be a real number.

リターンです。

例

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0 );
ini_close();

これは、「savedata.ini」を開き、その中の「save1」>「score」の値をscoreに設定し、再度.iniを閉じます。save1" > "score "に値がない場合、または "savedata.ini "がない場合は、"save1" > "score "に値が設定されます。 が存在する場合、スコアは0に設定されます。
