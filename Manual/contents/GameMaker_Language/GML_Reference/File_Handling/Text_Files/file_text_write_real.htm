ファイル_テキスト_ライト_リアル

この関数を使うと、先に開いているテキストファイルに数値を書き込むことができます。ただし、書き込む値は実数であるため、小数点以下はすべて". "で区切られます。ファイルがすでに情報を持っている場合、その情報は消去され、string がファイルの先頭に書き込まれます（ただし、あなたがファイルを file_text_open_append().を使用することで回避することもできます。 file_text_readln()関数と一緒に file_text_eof()関数を使って、loop ファイルの中身を最後まで見てから、書き込みを開始します。

注意点としては、この関数を使って非常に大きな数値をテキストファイルに書き込んだ場合、「6.6624e+003」のような標準的な簡略形式に変換されてしまい、GameMakerに正しく読み込むことができないことがあります。このような問題を防ぐために、代わりに値をstring に変換して、関数 file_text_write_string()の代わりに

構文です。

file_text_write_real(fileid, val);

論証タイプ 商品説明
fileid The id of the file to edit.
val The real value to write to the file.

リターンです。

例

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

上記のコードは、書き込み用にファイルを開き、loops 、2つの配列を通して、配列の各値をファイルの新しい行に書き込んでいます。そして、loop が終了すると、ファイルは閉じられます。
