file_text_open_append

この関数は、指定されたファイル名のテキストファイルを書き込み用に開き（ファイルが存在しない場合は作成される）、そのファイルに関連する以降のすべての操作に使用されるため、変数に格納されるべきそのファイルの一意のIDを返す。ファイル内の書き込み位置は、そのファイルが含むテキストの最終行に設定されるので、既存のデータが上書きされることはない。

ファイルを作成できない場合（例えばファイル名が不正な場合）、この関数は -1 を返すことに注意してください。

注：一度に開くことができるファイルは最大32個までです。また、終了時には必ずファイルを閉じるようにしてください。これにより、ファイルの情報が書き込まれ、ファイルに関連するメモリが解放されます。

警告!この機能は、GameMakerがサンドボックス化されているため、期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

file_text_open_append(fname);

論証 タイプ 商品説明
fname 文字列 The name of the file to append to.

リターンです。

Text File ID or -1

例

file = file_text_open_append(working_directory + "save.txt");

これは、ゲームと同じディレクトリにある「save.txt」を開き、そのファイルIDを変数「file」に格納します。
