ファイル_テキスト_オープン_ライト

この関数は、指定されたファイル名のテキストファイルを書き込み専用に開き（ファイルが存在しない場合は作成される）、そのファイルに関連する以降のすべての操作に使用されるため、変数に格納されるべきそのファイルのユニークIDを返します。

ファイルを作成できない場合（例えばファイル名が不正な場合）、この関数は -1 を返すことに注意してください。

すでにファイルが存在する場合、この関数を使用すると、データ保存時に上書きされます。既存の内容を保持したまま、ファイルにデータを追加したい場合は file_text_open_append().

注：一度に開くことができるファイルは最大32個までです。また、終了時には必ずファイルを閉じるようにしてください。これにより、ファイルの情報が書き込まれ、ファイルに関連するメモリが解放されます。

警告!この機能は、GameMakerがサンドボックス化されているため、期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

file_text_open_write(fname);

論証 タイプ 商品説明
fname 文字列 The name of the file to write to.

リターンです。

Text File ID or -1

例

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

上記のコードは、書き込み用にファイル "level.txt" を開き、変数 "level_data" に格納されているstring を書き込んだ後、最後に再びファイルを閉じます。
