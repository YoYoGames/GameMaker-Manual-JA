ファイル_テキスト_オープン_文字列から

この関数は、string からテキストファイルを作成し、読み込み用にオープンします。このファイルから読み込むために、以降のすべてのファイル関数呼び出しで使用されるべきファイル「ハンドル」を返します。このファイルは一時的なものであり、読み込み専用であることに注意してください。

注：一度に開くことができるファイルは最大32個までです。また、終了時には必ずファイルを閉じるようにしてください。これにより、ファイルの情報が書き込まれ、ファイルに関連するメモリが解放されます。

警告! GameMakerはサンドボックス化されているため、この機能は期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

file_text_open_from_string(string);

論証タイプ 商品説明
string The string to create the file from.

リターンです。

File ID or -1

例

file = file_text_open_from_string(reset_str);

上記のコードでは、変数 "reset_str "に格納されたstring 、それを使って読み取り専用のテキストファイルを作成しています。このファイルの「ハンドル」は変数「file」に格納され、以降のすべてのファイル関数に使用される。
