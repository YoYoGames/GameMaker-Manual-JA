ファイル_テキスト_オープン_リード

この関数は、指定されたファイル名のテキストファイルを読み込み専用にオープンし、そのファイルに関連する以降のすべての操作に使用されるため、変数に格納されるべきそのファイルのユニークIDを返す。ファイルが存在しない場合、この関数は値 -1 を返します。

注：一度に開くことができるファイルは最大32個までです。また、終了時には必ずファイルを閉じるようにしてください。これにより、ファイルの情報が書き込まれ、ファイルに関連するメモリが解放されます。

警告!この機能はGameMakerがサンドボックス化されているため、期待通りに動作しない可能性があります!詳しくは「ファイルシステム」の項をご覧ください。

構文です。

file_text_open_read(fname);

論証タイプ 商品説明
fname The name of the file to read from.

リターンです。

Text File ID or -1

例

file = file_text_open_read(working_directory + "level.txt");

これは、ゲームと同じディレクトリにある「level.txt」を開き、そのファイルIDを変数「file」に格納するものです。
