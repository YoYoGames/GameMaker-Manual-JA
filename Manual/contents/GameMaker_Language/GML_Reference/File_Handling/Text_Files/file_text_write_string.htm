ファイル_テキスト_書き込み_文字列

この機能を使うと、あらかじめ開いておいたテキストファイルに、string を書き込むことができます。ファイルがすでに情報を持っている場合、その情報は消去され、string はファイルの先頭に書き込まれます。ただし、そのファイルを file_text_open_append().を使用することでこれを回避することもできます。 file_text_readln()関数と一緒に file_text_eof()関数を使って、loop ファイルの中身を最後まで見てから、書き込みを開始します。

構文です。

file_text_write_string(fileid, str);

論証タイプ 商品説明
fileid The id of the file to edit.
str The string to write to the file.

リターンです。

例

var file = file_text_open_write(working_directory + "hiscore.txt");
for (var i = 0; i < 10; ++i;)
{
    file_text_write_real(file, scr[i]);
    file_text_writeln(file);
    file_text_write_string(file, scr_name[i]);
    file_text_writeln(file);
}
file_text_close(file);

上記のコードは、書き込み用にファイルを開き、loops 、2つの配列を通して、配列の各値をファイルの新しい行に書き込んでいます。そして、loop が終了すると、ファイルは閉じられます。
