キーボード・セット・ナムロック

この機能は、switch キーパッドのナンバーロックのオン／オフ（オンにする場合はtrue、オフにする場合はfalseに設定）に使用できます。

注意事項 この機能はWindows ターゲットビルドでのみ利用可能であり、他のデバイスでは機能しません。

構文です。

keyboard_set_numlock(value);

論証 タイプ 商品説明
value ブーリアン Set this to true for "on" and false for "off"

リターンです。

N/A

例

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

上記のコード例では、ナンバーロックキーの状態を取得し、それがオン（true）であればオフ（false）に、逆であればオフ（false）に設定します。
