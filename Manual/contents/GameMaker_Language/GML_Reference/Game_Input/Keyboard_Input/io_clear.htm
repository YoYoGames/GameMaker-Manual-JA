io_clear

この関数は、現在のキーボードとマウスの状態をクリアします。この関数を呼び出した後、次のステップで何が起こるかは、何をチェックするのか、また、どのプラットフォームでチェックするのかに依存することに注意してください。マウス入力の場合、マウスボタンは しかし、キーボードの場合は、それほど単純ではありません。一般的には、キーボードはそのステップだけクリアされ、次のステップで再びキーを検出します。 また、マウスと同じようにキーを持ち上げて再度押す必要がある場合もあります。

構文です。

io_clear();

リターンです。

例

io_clear();

これにより、すべてのキーボードとマウスの状態がクリアされます。
