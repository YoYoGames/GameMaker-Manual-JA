キーボードクリア

この機能を使うと、現在のキーボードの状態をクリアすることができます。つまり、キーが押されたままだと、再び離すまで（この場合、キーボードキー解放イベントも発生しません）、そのキーは認識されないということです。 を使用し、再度押された場合。この関数は、キーコードの値を関数 ord()(A-Zの大文字、0-9の数字のみ)、または vk_* の定数のいずれかになります。 キーボード入力のメインページにあります。

構文です。

keyboard_clear(key);

論証タイプ 商品説明
key The key to clear.

リターンです。

例

keyboard_clear(vk_space);

上記のコードは、スペースバーの状態をクリアするものです。
