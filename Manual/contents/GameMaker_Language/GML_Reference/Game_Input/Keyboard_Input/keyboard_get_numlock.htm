keyboard_get_numlock

この関数は、キーパッドの番号ロックの状態を調べるために使用します。オンにするとtrueが返され、オフにするとfalseが返されます。

注意事項 この機能は、Windows exe buildsでのみ利用可能で、他のデバイスでは機能しません。

構文です。

keyboard_get_numlock();

リターンです。

例

if keyboard_get_numlock()
{
    keyboard_set_numlock(false);
}
else
{
    keyboard_set_numlock(true);
}

上記のコード例では、ナンバーロックキーの状態を取得し、それがオン（true）であればオフ（false）に、逆であればオフ（false）に設定します。
