キーボードの最後の文字

この変数には、最後に押されたキーのstring が格納される。この変数は読み取り専用ではないので、例えば、すでに処理した場合は""（空のstring ）に設定するなど、変更することができます。

構文です。

keyboard_lastchar;

リターンです。

例

if keyboard_lastkey != -1
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

上記のコードでは、変数lastkeyが-1より大きいかどうかをチェックし、大きい場合は最後のキーが何であったかを変数 "str "にstring として追加しています。
