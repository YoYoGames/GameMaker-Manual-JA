keyboard_lastkey

この変数が参照する値は keyboard_keyは前のフレームにあり、そのキーのキーコードを返します (すべての標準キーコード定数が返されます)。この変数は読み取り専用ではないので、たとえば、すでに処理した場合は -1 に設定するなど、変更することができます。

構文です。

keyboard_lastkey;

リターンです。

Virtual Key Constant

例

if (keyboard_lastkey != -1)
{
    str += keyboard_lastchar;
    keyboard_lastkey = -1;
}

string 上記のコードでは、変数lastkeyが-1でないかどうかをチェックし、もし-1であれば、最後のキーが何であったかを変数 "str "に追加し、さらに入力を受け入れるために変数 keyboard_lastkey をリセットしています。
