キーボードチェック解除

この機能により、キーが解放されたかどうかを確認することができます。とは異なり keyboard_check()この関数は、キーが離されるたびに一度だけ実行されます。したがって、この関数を再度呼び出すには、キーが最初に押され、その後再び離される必要があります。この関数には，関数 ord()(A-Zの大文字、0-9の数字のみ）、またはキーボード入力のメインページに記載されている vk_* 定数のいずれかを使用します。

構文です。

keyboard_check_released(key);

論証タイプ 商品説明
key The key to check the released state of.

リターンです。

例

if keyboard_check_released(ord("P"))
{
    instance_create_layer(0, 0, "Controllers", obj_Pause);
}

上記のコードでは、「P」キーが離されたかどうかを確認し、離された場合は「obj_Pause」のインスタンスを生成します。
