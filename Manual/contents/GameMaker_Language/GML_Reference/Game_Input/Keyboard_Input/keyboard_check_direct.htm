keyboard_check_direct

この関数は、ハードウェアを直接チェックすることにより、特定のキーコードのキーが押されていれば true を、そうでなければ false を返します。特に、キーコードvk_lshift,vk_lcontrol,vk_lalt,vk_rshift,vk_rcontrol,vk_raltを使って、左または右シフト、コントロール、オルトキーが押されているかどうかをチェックすることが可能です。この関数は，キーコードの値として，関数 ord()(A-Zの大文字、0-9の数字のみ）、またはキーボード入力のメインページにある vk_* の定数のいずれかを指定します。

注意事項 この機能は、標準のWindows ターゲットでのみ使用可能で、どのアプリケーションにフォーカスがあっても結果は同じです。

構文です。

keyboard_check_direct(key);

論証タイプ 商品説明
key The key to check the down state of.

リターンです。

例

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
{
    crouch = true;
}

上記のコードでは、altキーの左右どちらかが押されたかどうかを確認し、押された場合は変数「crouch」をtrueにセットしています。
