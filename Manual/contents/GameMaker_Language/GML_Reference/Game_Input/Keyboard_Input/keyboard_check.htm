キーボードチェック

この機能により、キーが押されているかどうかを確認することができます。とは異なり keyboard_check_pressed()または keyboard_check_released()この関数はキーが押されたり離されたりしたときに一度だけ起動しますが、この関数はキーが押されている間、毎ステップ起動します。この関数には，関数 ord()(A-Zの大文字、0-9の数字のみ）、またはメインキーボード入力ページにある vk_* の定数のいずれかを使用します。

構文です。

keyboard_check(key);

論証 タイプ 商品説明
key 仮想キー定数(vk_*) The key to check the down state of.

リターンです。

Boolean

例

if keyboard_check(vk_left)
{
    x -= 5;
}

上記のコードは、矢印キーが押されているかどうかをチェックし、trueが返ってくるごとにインスタンスを左に5ピクセル移動させます。
