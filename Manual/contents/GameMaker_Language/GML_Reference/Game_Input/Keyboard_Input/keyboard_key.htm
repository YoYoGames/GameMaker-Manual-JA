キーボードキー

この変数で、現在押されているキーのキーコードを取得することができ、チェックが終了したときにキーが押されていなければ0を返します。

構文です。

keyboard_key;

リターンです。

Virtual Key Constant

例

switch (keyboard_key)
{
    case vk_numpad1: gun = weapon[0][0]; break;
    case vk_numpad2: gun = weapon[1][0]; break;
    case vk_numpad3: gun = weapon[2][0]; break;
}

上記のコードでは、変数 keyboard_key の値を使って、配列と同じ値を変数にセットしています。
