キーボードゲットマップ

マップされたキーのアスキーコードを取得したい場合があります（例えば、既にマップされているかどうかを確認するため）。この関数は，関数 ord()(A-Z の大文字と0-9 の数字のみ)、またはメインキーボード入力ページにある vk_* の定数のいずれかを使用します。

構文です。

keyboard_get_map(key);

論証タイプ 商品説明
key This is the key that you wish to get the mapped value from

リターンです。

例

if keyboard_get_map(ord("A")) = ord("A")
{
    keyboard_set_map(ord("A"), keyboard_lastkey);
}

上記のコード例では、まず「A」が他のキーにマッピングされているかどうかを確認し、マッピングされていない場合は、ユーザーが最後に押したキーにマッピングします。
