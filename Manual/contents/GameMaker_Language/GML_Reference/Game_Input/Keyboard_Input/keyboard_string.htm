キーボード_ストリング

この変数には、キーボードで入力された最後の（最大で）1024文字を含むstring が格納される。このstring は入力された印刷可能な文字だけを含むが、バックスペースキーを押すと最後の文字が消去されて正しく応答する。この変数は読み取り専用ではないので、例えば、すでに扱っている場合は""（空のstring ）に設定するなどの変更が可能で、String Functionsを使用して操作することもできます。なお、画面上の仮想キーボードを使用する場合は、この変数のみがキーボード入力で更新されます。

構文です。

keyboard_string;

リターンです。

例

if string_length(keyboard_string) > 15
{
    keyboard_string = string_copy(keyboard_string, 1, 15);
}

上記のコードは、キーボードstring の長さを15文字に制限し、最初の15文字を変数にコピーし直すことで制限を超えたものを削除します。
