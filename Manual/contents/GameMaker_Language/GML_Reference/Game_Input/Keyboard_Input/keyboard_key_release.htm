キーボードキーリリース

この関数を使うと、キーボードの任意のキーのリリースをシミュレートすることができます。この関数は，関数 ord()(A-Zの大文字、0-9の数字のみ)です。 または、キーボード入力のメインページに記載されている vk_* の定数のいずれかです。

構文です。

keyboard_key_release(key);

論証タイプ 商品説明
key The key to simulate a release of.

リターンです。

例

keyboard_key_release(vk_space);

これは、スペースバーを離すことをシミュレートしています。
