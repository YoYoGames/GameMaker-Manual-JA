keyboard_check_pressed

この機能により、あるキーが押されたかどうかを確認することができます。とは異なり keyboard_check()この関数は、キーが押されるたびに一度だけ実行されます。したがって、この関数を再度呼び出すには、まずキーを離し、次にもう一度押さなければなりません。この関数には，関数 ord()(A-Zの大文字、0-9の数字のみ）、またはメインキーボード入力ページに記載されている vk_* 定数のいずれかを使用します。

構文です。

keyboard_check_pressed(key);

論証 タイプ 商品説明
key 仮想キー定数(vk_*) The key to check the pressed state of.

リターンです。

Boolean

例

if keyboard_check_pressed(vk_anykey)
{
    room_goto_next();
}

上記のコードは、プレイヤーがキーボードのいずれかのキーを押せば、次のroom に進みます（「Press Any Key to Continue」プロンプトのように動作します）。
