キーボードキー押下

この機能を使うと、キーボードの任意のキーの押下をシミュレートすることができます。この機能を使うと，対応するリリース関数が呼ばれるまで，そのキーは押されたものとして認識される ( keyboard_key_release()). この関数は，キーコードの値を関数 ord()(A-Zの大文字、0-9の数字のみ）、またはメインページ「キーボード入力」に掲載されている vk_* 定数のいずれかを使用します。

この関数を呼び出した後、キーボード上でキーが物理的に押されると、キーが物理的に離されるまで（release イベントが発生し、この関数が停止するまで）、この押下イベントが再び登録されないことに注意すること。 または、対応するリリースキー関数が呼ばれ、そのキーが再び押された場合。

構文です。

keyboard_key_press(key);

論証タイプ 商品説明
key The key to simulate a press of.

リターンです。

例

keyboard_key_press(vk_space);

これは、スペースバーを押すことをシミュレートするものです。
