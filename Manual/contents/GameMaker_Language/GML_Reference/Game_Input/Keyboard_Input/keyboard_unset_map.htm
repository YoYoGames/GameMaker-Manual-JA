キーボード_アンセット_マップ

この関数を使うと、再マップされたキーをすべてクリアして、デフォルトの状態、つまりすべてのキーが自分自身にマップされる状態に戻すことができます。

構文です。

keyboard_get_map();

リターンです。

N/A

例

if keyboard_check_pressed(vk_escape)
{
    keyboard_unset_map();
}

上記のコード例では、ユーザーが「escape」キーを押した場合、マッピングされたすべてのキーをデフォルトの設定にリセットします。
