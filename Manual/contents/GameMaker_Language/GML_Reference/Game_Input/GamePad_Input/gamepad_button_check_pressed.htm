gamepad_button_check_pressed

この関数は、与えられたゲームパッドボタンが押されたことを検出したかどうかに応じて、 true または false を返します。この関数は、そのボタンが最初に押されたときに一度だけトリガーされることに注意してください。再度トリガーするには、まずボタンを離し、再度押さなければなりません。もし、ボタンが連続して押されていることを確認する必要がある場合は、以下の関数を使用する必要があります。 gamepad_button_check().

アナログボタンをチェックする場合、チェックは閾値の設定を考慮し、生の入力値が与えられた閾値を超えたときのみ true を返します（この生の値は関数 gamepad_button_value() を使って得ることができます）。

構文です。

gamepad_button_check_pressed(device, button);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
button ゲームパッドボタン定数 Which gamepad button constant to check for.

リターンです。

Boolean

例

if gamepad_button_check_pressed(0, gp_start)
{
    audio_play_sound(snd_Start, 0, false);
    room_goto(rm_Level_1);
}

上記のコードは、デバイス「スロット」0に接続されたゲームパッドの「スタート」ボタンが押されたかどうかを検出し、押された場合はroom を変更するものです。
