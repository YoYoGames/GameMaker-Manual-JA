gamepad_hat_value

この関数を使うと、ゲームパッドの "ハット "の現在値を取得することができます。各ハットの値は、異なる方向に対するビットマスクです。

  • アップ＝1
  • 右＝2
  • ダウン＝4
  • 左＝8

これらは組み合わせることができますが（例えば、帽子のインデックスを3に供給すると、上と右をチェックすることになりますし、｜演算子を使って様々な値を組み合わせることができます）、一度に押せるのは最大で2ビットだけであることに注意してください。この関数は 0 と 1 の間の実数値を返します。0 は押されていない状態、1 は完全に押された状態です (ゲームパッドが帽子のアナログ入力をサポートしているかどうかによって、その間の値もありえます)。

構文です。

gamepad_hat_value(device, hatindex);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
hatindex リアル Which gamepad hat (or hats) to check for.

リターンです。

Real

例

var _dir = gamepad_hat_value(global.PadIndex, 0);
switch (_dir)
{
    case 1: y -= 3; break;
    case 2: x += 3; break;
    case 3: y -= 3; x += 3; break;
    case 4: x -= 3; break;
    case 6: y += 3; x += 3; break;
    case 8: y += 3; break;
    case 9: y -= 3; x -= 3; break;
    case 12: y += 3; x -= 3; break;
}

上記のコードでは、ハット "0 "の状態をローカル変数に格納し、その戻り値が何であるかをチェックして、それに従って動作しています。
