gamepad_remove_mapping

この関数は、指定されたゲームパッドスロットインデックスから現在のデバイスマッピングを削除するために使用されます。一度呼び出されると、そのスロットインデックスは gamepad_test_mapping()ゲームパッド定数を使用して正しく入力を検出できるようにしたい場合は、 gamepad_axis/button/hat_count() と gamepad_axis/button/hat_value() 関数を一緒に使用して常に直接入力を取得できます)。

構文です。

gamepad_remove_mapping(index);

論証タイプ 商品説明
index Which gamepad index "slot" to remove the mapping from.

リターンです。

例

if remap == true
{
    gamepad_remove_mapping(global.PadIndex);
}

上記のコードは、変数の値に基づいて、指定されたゲームパッドのインデックススロットからマッピングを削除します。
