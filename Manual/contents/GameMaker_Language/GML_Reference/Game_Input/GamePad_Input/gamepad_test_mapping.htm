ゲームパッドテストマッピング

この関数は、ゲームパッドマッピングを許可しているターゲットにゲームパッドマッピングを設定するために使用することができます。設定するゲームパッドの "slot "インデックスと、以下のフィールドを持つSDLフォーマットで作成されたマップstring を指定する。

<guid>,<description>,platform:<platform-name>,<bindings>

各項目に記載すべき内容は、以下の通りです。

  • guid - デバイスの種類を一意に識別する数字と文字からなる (これはstring gamepad_get_guid()関数)
  • description - デバイスのAscii説明文（これは gamepad_get_description()関数)
  • platform-name - は "platform:android", "platform:windows", "linux", または "mac" のいずれかです (この関数を呼び出すと自動的に に追加さstring れるので、独自のマッピングを作成する際に指定する必要はないことに注意してください ) 。strings
  • bindings - 実際の入力を特定の 定数に結合するための、',' で区切られた一連のエントリGML

カンマで区切られたリストは、各エントリーの特定のフォーマットに従っています。

<abstract-gp-name>:<value-binding>

ここで、 <value-binding> （すなわち、ゲームパッドの直接入力）は、以下のいずれかになります。

  • a<N> - ゲームパッドの軸を表し、<N>は0〜のいずれかの軸を表します。 gamepad_axis_count()-1
  • b<N> - ゲームパッドのボタンを表し、<N>は0から gamepad_button_count()-1
  • h<N>.<mask> - ゲームパッドのハットを表し、<N>は0〜のハットを表します。 gamepad_hat_count()-1、 <mask> は 1、2、4、8 のいずれかになります。

各入力値は SDL 名にバインドされなければならない。SDL 名は <>abstract-gp-name> のように表現され、さらにGML 定数にバインドされる。以下の表は、SDL名とGameMaker定数の等価性を示している。

マッピングstring を構築すると、次のようになります。

"050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android"

構文です。

gamepad_test_mapping(index, mapping_string);

論証 タイプ 商品説明
index リアル Which gamepad index "slot" to set.
mapping_string 文字列 The map string to use (see the description for more information).

例

var mapping = gamepad_get_guid(global.padIndex) + "," + gamepad_get_description(global.padIndex);
var len = array_length(global.PadInstances);
for (i = 0; i < len; i += 2)
{
    var left = global.PadInstances[i];
    var right = global.PadInstances[i+1];
    mapping += "," + left.sdlLabel + ":" + right.binding;
}
gamepad_test_mapping(global.padIndex, mapping);

上記のコードは、loop 、いくつかのインスタンスを通じて、それらが含むさまざまな変数の値を使用して、マッピングstring 、与えられたスロットインデックスにゲームパッド上で使用するために設定されます。
