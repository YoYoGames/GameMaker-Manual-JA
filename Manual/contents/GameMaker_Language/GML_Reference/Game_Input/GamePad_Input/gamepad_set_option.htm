gamepad_set_option

この関数は、ゲームパッドのオプションを設定するために使用されます。オプションを設定するゲームパッドの「スロット」と、設定するオプション（string ）、および使用する値を指定します。利用可能なオプション（string ）は、プロジェクトが実行されているプラットフォームによって、以下のように異なります。

オプション プラットフォーム 商品説明 価値
"allow_rotation "です。 ティーブイオーエス Siri Remoteを回転ロック（垂直方向のみ使用可能）するかどうかを設定します。ロックが解除されている場合（値が true ）、コントロールを回転させると入力が変化し、左に90°回転するとタッチパッドの「右」側が「上」に戻るなど、すべての入力がデバイスの向きに応じて相対的に行われるようになります。ロックされている場合（値 false ）、入力はコントローラーの位置に対して変更されず、常にコントローラーが垂直の位置にあるものとして扱われます。 ブーリアン
"dpad_absolute "です。 ティーブイオーエス Siri Remoteのタッチパッドが絶対入力と相対入力のどちらを受け付けるかを設定します。絶対入力モードでは ( true) 、タッチパッドの中心が (0,0) 位置とみなされ、アナログスティックが静止している位置または dpad の中心が入力されます。相対（ false ）に設定すると、（0,0）位置はパッドに最初に触れた場所と見なされ、すべての入力はパッドの中心ではなく、その位置から生成されます。 ブーリアン

構文です。

gamepad_set_option(device, option_name, value);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to set.
option_name 文字列 The name of the option to set (a string, see the table above)
value Real,Boolean, またはString The value to set the option to (can be boolean, real or string - see the table above)

例

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

上記のコードloops は、すべてのゲームパッド・スロットに目を通し、tvOS プラットフォームに「Siri Remote」があるかどうかをチェックします。検出された場合、入力回転を許可するようにリモートを設定します。
