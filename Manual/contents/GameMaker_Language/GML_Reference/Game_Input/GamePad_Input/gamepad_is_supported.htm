gamepad_is_supported

この関数を使うと、ターゲットプラットフォームがゲームパッドをサポートしているかどうか（ true ）を調べることができます（ false ）を返します。

構文です。

gamepad_is_supported();

リターンです。

例

global.GP = gamepad_is_supported();

上記のコードは、ゲームパッドがサポートされているかどうかをチェックし、将来のチェックのために戻り値をグローバル変数に格納します。
