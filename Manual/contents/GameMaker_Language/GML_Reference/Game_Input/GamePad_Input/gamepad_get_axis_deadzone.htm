gamepad_get_axis_deadzone

この関数は、ジョイスティック軸の「デッドゾーン」値を取得するために使用できます。取得するデバイススロットを指定すると、この関数は0〜1の値を返します。valueはジョイスティック軸が0とみなされる閾値を反映しています。

構文です。

gamepad_get_axis_deadzone(device);

論証タイプ 商品説明
device Which gamepad device "slot" to check.

リターンです。

例

if gamepad_get_axis_deadzone(global.PadId) != 0.5
{
    gamepad_set_axis_deadzone(global.PadId, 0.5);
}

上記のコードは、グローバル変数に格納されたデバイス「slot」に接続されたゲームパッドのアナログ軸の閾値をチェックし、デフォルト値である0.5でなければその値に設定しています。
