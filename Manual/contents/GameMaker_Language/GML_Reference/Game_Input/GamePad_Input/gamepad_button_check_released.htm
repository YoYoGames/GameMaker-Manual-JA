gamepad_button_check_released

この関数は、指定されたゲームパッドボタンが離されたことを検出したかどうかに応じて、 true または false を返します。この関数は、ボタンが離された瞬間に一度だけトリガーされることに注意してください。再度トリガーするには、ボタンがまず押され、その後、もう一度離されなければなりません。アナログボタンをチェックする場合、閾値の設定を考慮し、生の入力値が与えられた閾値を下回ったときのみ、 true を返します（この生の値は、関数 gamepad_button_value() を使って得ることができます）。

構文です。

gamepad_button_check_released(device, button);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
button ゲームパッドボタン定数 Which gamepad button constant to check for.

リターンです。

Boolean

例

if gamepad_button_check_released(0, gp_select)
{
    audio_play_sound(snd_Button, 0, false);
    global.Pause = !global.Pause;
}

上記のコードは、デバイス「スロット」0に接続されたゲームパッドの「セレクト」ボタンが押されたかどうかを検出し、グローバル変数「ポーズ」をトグルさせます。
