gamepad_get_device_count

この機能は、接続されているゲームパッドの数、またはゲームパッドが接続可能な「スロット」の数を表示するものです。

実際の戻り値はプラットフォームとその内部設定に依存するため、この関数は関数 gamepad_is_connected()を併用し、正確なパッド接続数を確認する必要があります。

構文です。

gamepad_get_device_count();

リターンです。

Real

例

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        global.gp[i] = true;
    }
    else
    {
        global.gp[i] = false;
    }
}

上記のコードは、loops 利用可能なゲームパッド（またはゲームパッドスロット）を通して、それぞれゲームパッドが接続されているかどうかをチェックします。返された値は、今後のチェックに使用するために、グローバル配列に true または false を設定するために使用されます。
