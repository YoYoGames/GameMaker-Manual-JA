gamepad_get_option

この関数は、与えられたオプションの値を取得するために使用することができます。チェックするゲームパッドの「スロット」とオプション（string ）を指定すると、関数は関連する値を返します。利用可能なオプションstring は、プロジェクトが実行されているプラットフォームによって、以下のように異なります。

オプション プラットフォーム 商品説明 リターンズ
"allow_rotation "です。 ティーブイオーエス Siri Remoteを回転ロック（垂直方向にのみ使用可能）するかどうかを返します。ロックされていない場合、コントロールを回転させると入力が変化し、左に90°回転させるとタッチパッドの「右」側が「上」に戻るなど、すべての入力がデバイスの向きに応じて変化します。 ブーリアン
"dpad_absolute "です。 ティーブイオーエス Siri Remoteのタッチパッドが絶対入力、または相対入力のどちらを受け付けるかを返します。絶対入力モードでは、タッチパッドの中心を(0,0)位置とみなし、アナログスティックの静止位置またはdpadの中心を中心に移動すると入力が生成されます。相対モードに設定すると、(0,0)位置はパッドに最初に触れた場所と見なされ、すべての入力はパッドの中心ではなく、その位置から生成されます。 ブーリアン

構文です。

gamepad_get_option(device, option_name);

論証タイプ 商品説明
device Which gamepad device "slot" to check.
option_name The name of the option to get the value from (a string, see the table above)

リターンです。

Boolean, Real, String, or N/A (depends on platform and option)

例

for(var i = 0; i < 12; ++i;)
{
    if (gamepad_is_connected(i) && gamepad_get_description(i) == "tvOS Siri Remote")
    {
        if (gamepad_get_option(i, "allow_rotation") == false)
        {
            gamepad_set_option(i, "allow_rotation", true);
        }
    }
}

上記のコードloops は、すべてのゲームパッド・スロットに目を通し、tvOS プラットフォームに「Siri Remote」があるかどうかをチェックします。検出された場合、入力回転を許可するようにリモートを設定します。
