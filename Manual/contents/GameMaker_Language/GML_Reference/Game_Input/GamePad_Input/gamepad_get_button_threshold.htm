gamepad_get_button_threshold

この機能は、指定されたデバイスのアナログボタンの現在の閾値設定を検出するために使用することができます。アナログボタンのデフォルトの閾値は0.5で、範囲は0から1です。閾値は、デジタルボタンとして機能することを必要とするゲームにおいて、どの時点でボタンを「押した」と見なすかを定義します。

構文です。

gamepad_get_button_threshold(device);

論証タイプ 商品説明
device Which gamepad device "slot" to check.

リターンです。

例

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

上記のコードは、デバイス「スロット」0に接続されたゲームパッドのアナログボタンの閾値をチェックし、デフォルト値である0.5でなければその値に設定するものです。
