gamepad_get_guid

この関数は、指定されたスロットインデックスに接続されたゲームパッドのGUIDを取得することができます。調べるゲームパッドスロット(0〜11)を指定すると、この関数は以下の文字列のいずれかを返します。

  • GUIDがない場合、またはゲームパッドが接続されていない場合は、"none"となります。
  • ゲームパッドスロットのインデックスが範囲外の場合、"device index out of range" と表示されます。
  • GUIDstring, "050000005e040000fd020000ffff3f00" のようなもの。もしスロットにゲームパッドが割り当てられ、その GUID を取得できる場合。

この関数は、通常、以下の関数と組み合わせて使用されます。 gamepad_get_description()を使用して、string,Windows およびAndroid のコントローラをリマップするために必要な SDLmacOS の一部を生成します。

プラットフォーム固有の注意事項
プラットフォーム 備考
オペラGX Opera GXターゲットプラットフォームでは、この関数は非DirectInputデバイスに対して常に"none "を返します。

構文です。

gamepad_get_guid(index);

論証タイプ 商品説明
index Which gamepad "slot" index to check

リターンです。

例

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

上記のコードでは、GUID と名前の記述strings を取得し、それらを連結して、最終的なstring を将来の参照用にグローバル変数に格納しています。
