ゲームパッドボタン数

この関数は、指定されたデバイスの "スロット "に接続されたゲームパッドで使用可能なボタンの総数を返します。

構文です。

gamepad_button_count(device);

論証タイプ 商品説明
device Which gamepad device "slot" to check.

リターンです。

例

b_num = gamepad_button_count(0);

上記のコードは、デバイス「スロット」0に接続されたゲームパッドで利用可能なボタンの数を返し、その値を変数「b_num」に格納します。
