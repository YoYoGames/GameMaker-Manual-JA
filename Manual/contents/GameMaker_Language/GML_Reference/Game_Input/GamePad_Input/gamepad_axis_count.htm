ゲームパッド軸数

この関数は、チェック対象のデバイスにある「軸」コントロールの数を返します。これらのコントロールは、ほとんどのコントローラでアナログ方向の「サムスティック」です。

構文です。

gamepad_axis_count(device);

論証タイプ 商品説明
device Which gamepad device "slot" to check.

リターンです。

例

axis = gamepad_axis_count(0))

上記のコードでは、デバイス「スロット」0に接続されたゲームパッドが使用できる軸の数が変数「axis」に格納されます。
