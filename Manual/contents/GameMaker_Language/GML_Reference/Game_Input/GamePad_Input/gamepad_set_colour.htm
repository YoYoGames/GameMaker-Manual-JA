ゲームパッドの色設定

この関数は、PlayStationコントローラ内のLEDの色を設定するために使用することができます。色には、色定数、特定の色関数を使用して作成した色値、またはHEX値（$FFFFFFFFなど）を指定することができます。

構文です。

gamepad_set_colour(device, colour);

論証タイプ 商品説明
device Which gamepad device "slot" to set.
colour The colour to use.

リターンです。

例

if health < 10
{
    gamepad_set_colour(0, c_red);
}

上記のコードは、変数 health が10以下になると、PlayStationコントローラのLEDを赤に設定します。
