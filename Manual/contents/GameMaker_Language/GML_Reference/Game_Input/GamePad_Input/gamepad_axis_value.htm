gamepad_axis_value

この関数を使用すると、指定されたゲームパッドスロットから異なる軸の値を取得することができます。チェックするゲームパッドスロットのインデックスと、軸インデックス定数（ここにリストアップ）、または 0 と gamepad_axis_count()-1 の間の整数値を指定します。返される値は、利用可能な水平軸と垂直軸のそれぞれについて、 -1 と 1 の間となります。

返される値は、軸のデッドゾーンの設定値（関数 gamepad_set_axis_deadzone() を使用）を考慮して正規化されるため、常に -1 ～ 0、および 0 ～ 1 の間になることに注意してください。2 に設定すると、スティックを右に押して 0 ～ 1 の値を返すのは、生の軸値が 0.2 を超えたときだけということになります。

構文です。

gamepad_axis_value(device, axisIndex);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
axisIndex ゲームパッド軸定数 The axis index to check (see the constants list).

リターンです。

Real

例

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

上記のコードは、デバイス「スロット」0に差し込まれたゲームパッドの左アナログコントローラーの戻り値に基づいて、インスタンスの速度と方向を設定します。
