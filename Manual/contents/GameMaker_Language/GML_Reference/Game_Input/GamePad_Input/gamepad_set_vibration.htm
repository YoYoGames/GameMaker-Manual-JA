ゲームパッドの振動を設定する

ゲームパッドのモーターの振動を設定する機能で、どちらかのモーターを0（無振動）～1（フル振動）の値で設定することができます。switch この機能には時間制限がないので、変数やアラームを使って、一定時間経過後に振動を止める（モーターを0にする）必要があります。そうしないと、ゲームパッドは無限に振動し続けます。

注：この機能は現在、Windows、PS4、Xbox Oneの標準ターゲットモジュールでのみ利用可能です。

構文です。

gamepad_set_vibration(device, left_motor, right_motor);

論証タイプ 商品説明
device Which gamepad device "slot" to check.
left_motor The amount of vibration from the left motor from 0 to 1.
right_motor The amount of vibration from the right motor from 0 to 1.

リターンです。

例

if lives = 0
{
    gamepad_set_vibration(0, 1, 1);
    alarm[0] = room_speed / 2;
}

上記のコードを衝突イベントで使用すると（例）、「スロット」0に接続されたゲームパッドが半秒間振動し、その時間が過ぎると設定されたアラームが使用されて、再びswitch 、振動が停止します。
