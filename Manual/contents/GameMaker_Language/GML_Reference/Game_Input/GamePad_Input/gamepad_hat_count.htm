ゲームパッドハットカウント

この関数は、指定されたデバイス "スロット "に接続されたゲームパッドで利用可能な帽子の総数を返します。ハットとは一般的に上下左右のボタンのことで、Windows ターゲットでは、ハットは DirectInput のものしか利用できないことに注意してください。 コントローラ（つまり、スロット 4 以上）。

構文です。

gamepad_hat_count(device);

論証タイプ 商品説明
device Which gamepad device "slot" to check.

リターンです。

例

h_num = gamepad_hat_count(4);

上記のコードは、デバイス「スロット」4に差し込まれたゲームパッドで利用可能な帽子の数を返し、その値を変数「h_num」に格納します。
