gamepad_button_value

この関数では、アナログボタンの現在値を0から1までの値で取得することができます。チェックするゲームパッドスロットのインデックスと、ボタン定数（ここにリストアップ）、または0からgamepad_button_count() - 1の間の整数値を指定します。

これはボタンの生の値を返すので、閾値の設定は考慮されないことに注意してください (閾値の詳細については、ここを参照してください)。

構文です。

gamepad_button_value(device, button);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
button ゲームパッドボタン定数 Which gamepad button constant to check for.

リターンです。

Real

例

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

上記のコードでは、デバイス「スロット」0に接続されたゲームパッドからのアナログトリガーの値を使用して、インスタンスの速度を設定します。
