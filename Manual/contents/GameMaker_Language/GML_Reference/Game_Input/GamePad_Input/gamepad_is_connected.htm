ゲームパッドが接続されているかどうか

この関数は、与えられた「スロット」にゲームパッドが接続されているか（ true を返す）、されていないか（ false を返す）を返します。通常、この関数は gamepad_get_device_count()関数で、利用可能なゲームパッドやゲームパッドのスロット数を正しく取得することができます。ユーザーがゲームパッドを接続してからGameMakerが接続を検出するまで、若干の遅れがあることに注意してください（特にBluetooth接続のコントローラを扱う場合に顕著です）。

注意：一部のプラットフォーム（特にコンソール）では、ゲームパッドが接続／選択された直後にこの関数が false を返すことがあり、ゲームパッドを正しく検出するために数フレーム（ステップ）後にアラームで確認する必要がある場合があります。

構文です。

gamepad_is_connected(numb);

論証タイプ 商品説明
device Which gamepad "slot" to check.

リターンです。

例

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++)
{
    if gamepad_is_connected(i)
        global.gp[i] = true
    else
        global.gp[i] = false;
}

上記のコードは、loops 利用可能なゲームパッド（またはゲームパッドスロット）を通して、それぞれゲームパッドが接続されているかどうかをチェックします。返された値は、今後のチェックで使用するために、グローバル配列に true または false を設定するために使用されます。
