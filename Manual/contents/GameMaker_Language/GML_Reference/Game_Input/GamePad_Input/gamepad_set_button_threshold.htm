gamepad_set_button_threshold

この機能は、指定されたデバイスのアナログボタンの現在の閾値設定を設定するために使用することができます。すべてのアナログボタンのデフォルトの閾値は0.5で、範囲は0から1です。

デジタルボタンとしての機能を必要とするゲームでは、どの時点でボタンが「押された」と判断するかを閾値で定義しています。

この関数は、アナログボタンのチェック、押された状態、離された状態に影響を与えますが、関数 gamepad_button_value()が返す値には影響しません。

構文です。

gamepad_set_button_threshold(device, threshold);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
threshold リアル The new threshold value (from 0 - 1, default 0.5).

リターンです。

N/A

例

if gamepad_get_button_threshold(0) != 0.5
{
    gamepad_set_button_threshold(0, 0.5);
}

上記のコードは、デバイス「スロット」0に接続されたゲームパッドのアナログボタンの閾値をチェックし、デフォルト値である0.5でなければその値に設定するものです。
