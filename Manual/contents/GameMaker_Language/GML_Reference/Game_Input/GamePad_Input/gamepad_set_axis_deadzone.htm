gamepad_set_axis_deadzone

この機能は、ジョイスティック軸の「デッドゾーン」を設定するために使用できます。設定するデバイススロットを指定し、0〜1の値を設定し、入力量が指定した値より小さい場合、ジョイスティック軸は0になったものとみなします。 なお、これはグローバルな設定であり、指定したデバイススロットに接続されているすべてのジョイスティックの全軸に影響します。

この関数は、軸の値がデッドゾーンの値から始まることを意味するものではありません。 gamepad_axis_value()の間の正規化された値を返します。 例えば、デッドゾーンを0.2に設定すると、スティックを右に押して0～1の値を返し始めるのは、生の軸値が0.2以上になってからということになり、生の値が0.2のときは戻り値は0になります。 生の値が0.5であれば、戻り値は0.375となり、生の値が0.9であれば、戻り値は0.875となります。

構文です。

gamepad_set_axis_deadzone(device, deadzone);

論証タイプ 商品説明
device Which gamepad device "slot" to check.
deadzone The dead zone value from 0 to 1.

リターンです。

例

if gamepad_is_connected(0) gamepad_set_axis_deadzone(0, 0.05);
if gamepad_is_connected(1) gamepad_set_axis_deadzone(1, 0.05);
if gamepad_is_connected(2) gamepad_set_axis_deadzone(2, 0.05);
if gamepad_is_connected(3) gamepad_set_axis_deadzone(3, 0.05);

上記のコードは、接続された4台のデバイスのうち、いずれかのジョイスティック軸のデッドゾーンを0.05に設定します。
