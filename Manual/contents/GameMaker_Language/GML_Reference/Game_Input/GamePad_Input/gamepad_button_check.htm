ゲームパッドボタンをチェックする

この関数は、与えられたゲームパッドボタンが押されていると検知されたかどうかに応じて、 true または false を返します。アナログボタンをチェックしている場合、閾値の設定を考慮し、生の入力値が与えられた閾値を超えている間のみ true を返します (この生の値は関数 gamepad_button_value() を使って得ることができます)。

構文です。

gamepad_button_check(device, button);

論証 タイプ 商品説明
device リアル Which gamepad device "slot" to check.
button ゲームパッドボタン定数 Which gamepad button constant to check for.

リターンです。

Boolean

例

if gamepad_button_check(0, gp_face1)
{
    if (canshoot = true)
    {
        audio_play_sound(snd_Shoot, 0, false);
        instance_create_layer(x, y, "Bullets", obj_Bullet)
        canshoot = false;
        alarm[0] = room_speed / 2;
    }
}

上記のコードは、上面の一番下のボタン（PS3コントローラでは「X」）が押されているかどうかを検出し、押されていれば「弾丸」インスタンスを撃ち、アラームをセットします。
