gamepad_get_description

string この関数は、指定されたゲームパッドの説明をstring に返します。例えば、PS3 コントローラの場合、「PLAYSTATION(R)3 Controller」、XBox360 コントローラの場合、「Xbox 360 Controller (XInput STANDARD GAMEPAD) 」のような説明を返すことがあります。このstring はハードウェアに依存し、返される値は、チェック対象のデバイス「スロット」に差し込まれたゲームパッドに依存します。

構文です。

gamepad_get_description(device);

論証タイプ 商品説明
device Which gamepad "slot" to get the name of.

リターンです。

例

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
    if (gamepad_is_connected(i))
    {
        draw_text(32, 32 + (i * 32), gamepad_get_description(i));
    }
    else
    {
        draw_text(32, 32 + (i * 32), "No Gamepad Connected");
    }
}

上記のコードは、loop 、すべてのゲームパッド・スロットに接続されたデバイスをチェックし、ゲームパッドがスロットに接続されているかどうかに基づいて、画面にテキストを描画します。
