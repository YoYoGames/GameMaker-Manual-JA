device_mouse_check_button_released

この関数は、指定したデバイスのタッチが解除されたかどうかにより、 true または false を返す。検出できる最大タッチ数は、使用するデバイスとそのOSに大きく依存することに注意してください（ほとんどのデバイスは、少なくとも4つの同時タッチをサポートしています）。この関数は、実際のリリース動作によって一度だけ起動され、このページに記載されている定数を使用してマウスボタンのチェックを行うことができます。なお、 mb_right は、ダブルタップタッチが検出され、リリース前の 2 回目のタップで保持された場合のみ検出されます（この動作は関数 device_mouse_dbclick_enable() を使用して無効にすることができます）。

構文です。

device_mouse_check_button_released(device, button);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

リターンです。

例

if device_mouse_check_button_released(0, mb_left)
{
    press = false;
}

上記のコードでは、デバイスが押されたかどうかをチェックし、押された場合は変数をfalseにセットしています。
