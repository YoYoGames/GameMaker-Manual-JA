デバイス入力

マルチタッチが可能なプラットフォームをターゲットにする場合、これらの機能を使用する必要があります。PCに複数のマウスが接続されている場合と同じように動作し、チェックされる定数は標準のマウス関数に使用されているものと同じです。

定数 商品説明
mb_left マウスの左ボタン
mb_middle マウスの中央ボタン（ターゲットプラットフォームによっては有効でない場合があります。）
mb_right マウスの右ボタン
mb_side1* マウスサイドボタン1
mb_side2* マウスサイドボタン2
mb_any マウスの任意のボタン
mb_none マウスボタンがない

注意*mb_side1およびmb_side2 ボタンは、Windows,macOS,Ubuntu および HTML5 でのみ使用可能です。

この関数はすべて、デバイスのディスプレイへのタッチに対応する「タッチ」値を取ることができますが、検出できるタッチの最大数は、デバイスと実行中のOSに大きく依存します。たとえば、ローエンドのAndroid デバイスでは、同時に 2 つか 3 つのタッチしか検出できない場合がありますが、UWP デバイスでは最大 10 のタッチを検出できる場合があります。また、タッチスクリーンデバイスでは、ダブルタップはマウスの右ボタンと同等のイベントとして返されることに注意してください。したがって、たとえば、複数のタップを連続して検出する必要がある場合、 mb_left と mb_right の両方をチェックする必要があります。この機能のオン・オフは device_mouse_dbclick_enable().

以下は、デバイス固有の制御機能の一覧です。

  • device_mouse_check_button
  • device_mouse_check_button_pressed
  • device_mouse_check_button_released
  • device_mouse_dbclick_enable
  • device_mouse_x
  • device_mouse_y
  • device_mouse_raw_x
  • device_mouse_raw_y
  • device_mouse_x_to_gui
  • device_mouse_y_to_gui
  • device_is_keypad_open
  • device_get_tilt_x
  • device_get_tilt_y
  • device_get_tilt_z

モバイルデバイス(Android,iOS andWindows Phone)では、いつでも戻るボタンが使えます。これは、GameMaker キーボード定数 vk_backspace であるキーボードbackspaceにマッピングされているので、これらのデバイスでは、そのキーに対するチェックを行い、戻るボタンを使用したかのような動作を実行することができます。
