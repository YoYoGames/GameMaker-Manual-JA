device_get_tilt_y

この関数は、デバイスの「傾き」の角度に応じて、-1〜1の値を返す。実際の傾きと返される値の相関は、デバイスや使用しているOSによって異なりますが、一般的に1または-1の値は+/-90°と同じです。下図は、各機能とデバイスの関連性を示したものです。

構文です。

device_get_tilt_y()

リターンです。

例

if display_get_orientation() = display_landscape
{
    x += sign(device_get_tilt_y());
}
else
{
    x += sign(device_get_tilt_x());
}

上記のコードでは、ディスプレイの向きをチェックし、対応するチルト値を使ってプレーヤーをX軸方向に動かしています。
