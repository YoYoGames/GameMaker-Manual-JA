デバイスマウスラウズ

この関数は、デバイス上のタッチの生の x 位置を返します。つまり、GameMakerの 定義ではなく、タッチされた実際のデバイスのx位置を返すので、ビュー位置やスケーリングといったものは無視される。検出できるタッチの最大数は、使用するデバイスとOSに大きく依存することに注意してください。

注：この関数はデバイスに大きく依存するので、まず目的のモジュールとデバイスで実験し、正確に何が返されるかを確認する必要があります。

構文です。

device_mouse_raw_x(device);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked.

リターンです。

例

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1));
    y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1));
}

上記のコードは、デバイス1とデバイス2が押されているかどうかをチェックし、押されていれば、各押されたポイントの間のx/y座標の平均位置を計算する。
