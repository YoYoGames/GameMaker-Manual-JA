デバイスがキーパッドを開いているかどうか

これは、キーパッドがあるかどうかデバイスをチェックし、キーパッドがある場合は true を返し、ない場合は false を返します。この関数は、主にAndroid のデバイスで使用することに注意してください。

Sony Xperia Playをお使いの方は、GameMaker内にボタン/キーマップのセットアップがありますので、キーボード定数 vk_up,vk_down,vk_left,vk_rightをジョイパッドのキーに、三角は ord("T") 、正方は ord("S") 、円は vk_alt+vk_backspace、バックボタンは単に vk_backspace 、クロスは vk_space 、セレクトは vk_return、スタートは vk_rshift、Lトリガは ord("L") とRトリガは ord("R") とすることで使用可能です。

構文です。

device_is_keypad_open

リターンです。

例

if device_is_keypad_open()
{
     global.Setting = 2;
}
else
{
     global.Setting = 1;
}

上記のコードでは、キーパッドをチェックし、返された値に応じてグローバル変数を変更します。
