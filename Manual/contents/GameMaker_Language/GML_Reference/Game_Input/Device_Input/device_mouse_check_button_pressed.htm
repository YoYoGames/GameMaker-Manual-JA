device_mouse_check_button_pressed

この関数は、指定したデバイスが「タッチ」（クリック）されたかどうかによって、 true または false を返します。デバイスの引数はタッチ番号で、0からnまであり、検出できる最大タッチ数は、使用するデバイスとそのOSに大きく依存します（ほとんどのデバイスは少なくとも4つの同時タッチをサポートします）。この関数は、実際に押す動作によって一度だけ 起動され、このページに記載されている定数を使用してマウスボタンを確認することができます。なお、 mb_right は、ダブルタップタッチが検出され、2回目のタップでホールドされた場合のみ検出されます（この動作は、関数 device_mouse_dbclick_enable() を使用して無効にすることが可能です）。

構文です。

device_mouse_check_button_pressed(device, button);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked.
button The button of the device that is being checked.

リターンです。

例

if device_mouse_check_button_pressed(0, mb_left)
{
    press = true;
}

上記のコードでは、デバイスが押されたかどうかをチェックし、押された場合は変数をtrueにセットしています。
