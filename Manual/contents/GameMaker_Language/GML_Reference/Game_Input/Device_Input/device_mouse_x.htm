デバイスマウスエクス

この関数は、デバイス上のタッチの x 位置を返します。HTML5 または PC と Mac モジュールでこれを実行する場合、デバイス（通常はマウス）が接続されている限りこの値は常に更新されますが、モバイル デバイスの場合は、画面がタッチされている間のみ更新されます。

構文です。

device_mouse_x(device);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked.

リターンです。

例

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_x(0) > x-32 && device_mouse_x(0) < x+32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

上記のコードでは、デバイスが押されているかどうかを確認し、押されている場合はデバイスのX位置をポーリングして、それがパラメータ内にあるかどうかを確認しています。もしそうなら、変数 "pressed "に true をセットし、そうでなければ false をセットします。
