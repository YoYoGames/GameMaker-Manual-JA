device_mouse_dbclick_enable

mb_left (マウスの左ボタン) のダブルタップを mb_right (マウスの右ボタン) の入力として検出するように設定することができます。

デフォルトでは、 true に設定されています。つまり、ユーザーがデバイスの画面をすばやく2回連続してタップするたびに、マウスの右ボタンをクリックしたのと同じ値が返されることになります。この設定がオンの場合、1回目のタップは mb_left として、2回目は mb_right として検出されますので、使用するコードはこの点を考慮に入れてください。

構文です。

device_mouse_dbclick_enable(bool);

論証 タイプ 商品説明
bool ブーリアン Set double-click detection on (true) またはオフ (false) にします。

リターンです。

Boolean

例

if os_type == os_windows || os_type == os_mac
{
    device_mouse_dbclick_enable(false);
}

上記のコードでは、ゲームを実行しているデバイスがWindows PC か Mac かを確認し、どちらかであればダブルタップ機能を無効にしています。
