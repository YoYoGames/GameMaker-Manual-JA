デバイスのマウスをギにする

この関数は、GUIレイヤーを基準として、デバイス上のタッチのy位置を返す。GUIレイヤーは画面の解像度や room のサイズとは無関係に異なるサイズに設定できるため、ユーザーが画面にタッチしたGUIレイヤー上の正確な位置を取得するのは時に煩雑になることがある。この機能を使えば、実際のGUIサイズに基づいてその位置を得ることができるので、GUIボタンやHUDエレメントの作成とインタラクションが容易になります。HTML5またはPCとMacのモジュールでこれを実行している場合、デバイス（通常はマウス）が接続されている限り、この値は常に更新されますが、モバイルデバイスの場合は、画面がタッチされている間のみ更新されます。

構文です。

device_mouse_y_to_gui(device);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked.

リターンです。

例

if device_mouse_check_button(0, mb_left)
{
    if (device_mouse_y_to_gui(0) > 0 && device_mouse_y_to_gui(0) < 32)
    {
        pressed = true;
    }
    else
    {
        pressed = false;
    }
}

上記のコードでは、マウスが押されているかどうかをチェックし、押されていれば、GUIレイヤーのデバイスのXポジションをポーリングして、それがパラメータ内にあるかどうかを確認しています。もしそうなら、変数 "pressed "を true に設定し、そうでないなら false に設定します。
