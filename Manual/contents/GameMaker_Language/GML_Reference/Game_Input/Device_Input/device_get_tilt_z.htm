device_get_tilt_z

この関数は、デバイスの "傾き "の角度に応じて-1〜1の値を返します。実際の傾斜角度と返される値の相関は、使用するデバイスやOSによって異なりますが、一般的には1または -1は+/-90°と同じです。下図は、各機能とデバイスの関連性を示しています。

構文です。

device_get_tilt_z()

リターンです。

例

y += sign(device_get_tilt_z());

上記のコードでは、tiltの値を使ってy軸方向にプレイヤーを動かしています。
