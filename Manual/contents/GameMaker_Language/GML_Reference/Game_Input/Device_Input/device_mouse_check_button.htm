device_mouse_check_button

この関数は、与えられたマウスボタンが与えられたデバイス上で押されているかどうかに応じて、 true または false を返します。マウスボタンには、このページにリストアップされている定数のうちの1つを指定することができます。

この関数はタッチスクリーンに使用することができ、device 引数は0からnまで、nは一度に画面に触れることができる「指」の最大数である。 検出されるかどうかは、使用するデバイスとそのOSに大きく依存しますが、ほとんどのデバイスは少なくとも4まで検出するはずです）。なお、 mb_right はダブルタップが検出された場合のみ検出されます（この動作を無効にするには 機能 device_mouse_dbclick_enable()).

構文です。

device_mouse_check_button(device, button);

論証タイプ 商品説明
device The device (from 0 - n) that is being checked
button The button of the device that is being checked.

リターンです。

例

if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left)
{
    room_goto(rm_Menu);
}

上記のコードでは、2つのタッチが同時に押されているかどうかを確認し、押されている場合は、別のroom 。
