ゲーム入力

ゲームをゲームたらしめている根本的なものは、プレイヤーとのインタラクションです。コンピュータではキーボードやマウス、モバイルではタッチスクリーンやデバイスの傾き、さらにはすべてのプラットフォームでゲームパッドなど、さまざまな手段でこれを実現することができます。

そもそもGameMakerでは、PCでは専用のマウスやキーボードイベント、モバイルではジェスチャーイベントを使いますが、これらのイベント以上のコントロールが必要な場合や、ゲームパッドによるコントロールや複数の異なるコントロールオプションを組み込みたい場合があります。このような場合は、ゲーム入力関数を使用して、どのボタンやキーが押されたり離されたりしたかを取得したり、スクリーン上のタッチやゲームパッドのジョイスティックをどれだけ動かしたかを取得することができます。

ゲームの入力や操作に関する機能は非常に多いので、必要なものを見つけやすいように、特定の入力方法に関連するセクションに分けられています。利用可能なセクションは以下の通りです。

  • キーボード入力
  • マウス入力
  • デバイス入力
  • ジェスチャー入力
  • GamePad入力
  • 仮想キーと仮想キーボード
