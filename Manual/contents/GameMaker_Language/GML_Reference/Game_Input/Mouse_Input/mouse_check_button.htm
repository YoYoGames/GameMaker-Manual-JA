mouse_check_button

この関数は、チェック対象のマウスボタンが押されている場合は true を、押されていない場合は false を返します。チェックするマウスボタンは、以下の定数のいずれかから指定します。

構文です。

mouse_check_button(numb);

論証 タイプ 商品説明
numb マウスボタン定数 Which mouse button constant to check for.

リターンです。

Boolean

例

if mouse_check_button(mb_left)
{
    instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

上記のコードは、マウスの左ボタンをチェックし、それが押されるたびに、"obj_Star "でインデックスされたobject のインスタンスが作成されます。
