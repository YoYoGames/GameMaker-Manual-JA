マウス入力

マウス入力はすべてのプラットフォームで受け入れられ（モバイル機器では1画面のタッチとして受け入れられます - マルチタッチを使用する必要がある場合は、デバイス固有の機能を使用する必要があります）、押されているボタンを指定するために使用するいくつかの定数を持っています。これらの定数は次の表に示すとおりです。

マウスの標準的なボタン操作には、次のような機能があります。

  • mouse_button
  • mouse_check_button
  • mouse_check_button_pressed
  • mouse_check_button_released
  • mouse_clear
  • mouse_lastbutton
  • mouse_wheel_up
  • mouse_wheel_down
  • mouse_x
  • mouse_y

なお、デスクトップターゲットでのマウス操作に関連するウィンドウ関数群もある。

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
