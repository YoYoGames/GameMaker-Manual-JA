mouse_button

この読み取り専用変数は、現在押されているマウスボタン（現在、このステップのように）を返し、 mb_any 以外の任意の特別なマウス定数を返すことができます。

構文です。

mouse_button;

リターンです。

Mouse Button Constant (except mb_any)

例

if mouse_button = mb_left
{
    x -= 1;
}

これは、マウスの左ボタンが押されている場合、現在のインスタンスを左に移動します。
