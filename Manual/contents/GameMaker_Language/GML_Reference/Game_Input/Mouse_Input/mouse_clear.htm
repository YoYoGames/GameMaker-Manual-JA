マウスクリア

この関数は、与えられたマウスボタンの現在の状態をクリアします。つまり、プレイヤーがボタンを離し、再び押すまで、ボタンが押されているかどうかのチェックは true を返しません（ただし、マウスボタンが押されている間にクリアが行われた場合は、リリース状態が検出されます）。

このページに記載されているマウスボタン定数のいずれかを指定することができます。

構文です。

mouse_clear(button);

論証 タイプ 商品説明
button マウスボタン定数 Which mouse button constant to check for.

リターンです。

Boolean

例

mouse_clear(mb_any);

上記のコードで、すべてのマウスボタンのダウン状態がクリアされます。
