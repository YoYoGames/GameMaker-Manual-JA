mouse_lastbutton

この変数は、最後に押されたマウスボタンを返し、 mb_any を除く任意の特殊マウス定数を返すことができる（この変数に定数の一つを設定することもできる）。

構文です。

mouse_lastbutton;

リターンです。

Mouse Button Constant (except mb_any)

例

if mouse_lastbutton = mb_left
{
    x -= 1;
}

これは、最後に押されたボタンがマウスの左ボタンであった場合、現在のインスタンスを左に移動します。
