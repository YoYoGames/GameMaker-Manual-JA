mouse_check_button_pressed

この関数は、チェック対象のマウスボタンが押された場合は true を、押されていない場合は false を返します。この関数は、マウスボタンが最初に押されたときに一度だけトリガされ、再度トリガするためには、ボタンが離され、再度押されることが必要です。この関数は、ステップの間、そしてマウスイベントを持つすべてのインスタンス、またはこの同じ関数を使用するインスタンスでトリガーされたとみなされることに注意してください。

チェックするマウスボタンを以下の定数のいずれかから指定します。

構文です。

mouse_check_button_pressed(numb);

論証 タイプ 商品説明
numb マウスボタン定数 Which mouse button constant to check for.

リターンです。

Boolean

例

if mouse_check_button_pressed(mb_left)
{
    score += 50;
}

上記のコードは、マウスの左ボタンが押されたかどうかを確認し、押された場合はスコアに50を加算します。
