mouse_check_button_released

この関数は、チェック対象のマウスボタンが離された場合は true を、離されていない場合は false を返します。この関数は、マウスボタンが離されたときに一度だけ起動され、再度起動するには、ボタンが押されてから離される必要があります。チェックするマウスボタンは、以下の定数のうちの1つから指定します。

構文です。

mouse_check_button_released(numb);

論証 タイプ 商品説明
numb マウスボタン定数 Which mouse button constant to check for.

リターンです。

Boolean

例

if mouse_check_button_released(mb_right)
{
    speed = point_distance(x, y, mouse_x, mouse_y) / 10;
}

上記のコードは、マウスの右ボタンが離されたかどうかを確認し、離された場合はインスタンスの速度を現在のx/y位置とマウスのx/y位置の距離の10分の1に設定します。
