キーボード仮想状態

この関数は、ゲームを実行しているデバイスの仮想キーボードの状態を取得するために使用することができます。この関数は、OS の仮想キーボードが表示されている場合は true を、隠されている場合は false を返します。

シンタックス

keyboard_virtual_status();

リターンです。

例

var _status = keyboard_virtual_status();
if (_status == false)
{
    keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

上記のコードは、現在のステータスが false の場合、OSの仮想キーボードを表示します。
