virtual_key_show

デバイス用の仮想キーを作成したら、それが正しく配置され、正しく動作することをデバッグおよびテストする必要があるかもしれません。この場合、表示したい仮想キーのインデックスを指定して、この関数を呼び出します。 virtual_key_add()色付きの矩形が描かれ、画面上にキーが表示されます。満足したら、独自のグラフィックを作成し、その領域に配置することをお勧めします。

注意描画される矩形は、対応する描画関数で設定されたブレンディング、カラー、アルファオプションの影響を受けます。

シンタックス

virtual_key_show(index);

論証 タイプ 商品説明
index 仮想キーID The index of the virtual key to show.

リターンです。

N/A

例

if global.Debug
{
    virtual_key_show(global.Left);
}
else
{
    virtual_key_hide(global.Left);
}

上記のコードでは、グローバル変数 "global.Debug "をチェックし、それがtrueであれば、変数 "global.Left "でインデックスされた仮想キーが画面に描かれ、falseであれば、そのキーは隠されることになります。
