キーボード_virtual_hide

この関数は、ゲームを実行しているデバイスの仮想キーボードを隠すために使用することができます。この関数を呼び出すと、システム非同期イベントが発生し、 async_load DS マップに以下のキー/値ペアが入力されます。

  • "event_type" - 仮想キーボード機能によってトリガーされた場合、"virtual keyboard status" となります。
  • "screen_height" - 仮想キーボードの高さ (ピクセル単位)です。キーボードが見えない場合は、0 になります。
  • "keyboard_status" - キーボードの現在のステータスで、以下の文字列のうちの1つとして返されます。
    • "隠蔽"
    • "非表示"
    • "表示"
    • "可視"

シンタックス

keyboard_virtual_hide();

リターンです。

例

if input == true
{
    input = false;
    keyboard_virtual_hide();
}

上記のコードは、与えられた変数が false に設定されていない場合、OS の仮想キーボードを隠します。
