virtual_key_delete

あなたのゲームがメニューやゲームプレイなどを操作するために異なるrooms またはインスタンスを持っている場合、ある時点で仮想キーの位置とキーマップを変更する必要がある可能性が高いです。その際、この機能を使えば、新しいものを作る前に古いものを削除することができます（必要な場合）。この関数では、削除する仮想キーの ID を指定する必要があります。 virtual_key_add().

なお、仮想キーはroom を変更すると自動的にroom から削除されますので、この機能はroom 自体が終了する前に手動で削除する場合にのみ必要です。

シンタックス

virtual_key_delete(index);

論証 タイプ 商品説明
index 仮想キーID The index of the virtual key to delete.

リターンです。

N/A

例

virtual_key_delete(global.Left);

上記のコードは、グローバル変数 "global.Left "にインデックスされた仮想キーを削除するものです。
