keyboard_virtual_height

この関数は、ディスプレイのサイズに基づき、仮想キーボードの現在の高さをピクセル単位で返します。キーボードが表示されていない場合は、0が返される。

シンタックス

keyboard_virtual_height();

リターンです。

例

if keyboard_virtual_status() == true
{
    key_h = keyboard_virtual_height();
}

上記のコードでは、OSの仮想キーボードの状態を確認し、表示されていればキーボードの高さを変数にセットします。
