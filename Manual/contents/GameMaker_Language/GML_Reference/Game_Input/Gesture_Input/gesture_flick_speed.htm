ジェスチャーフリックスピード

タッチまたはクリックを離したときにフリックジェスチャーイベントを発生させるのに必要な速度を設定する機能です。速度は1秒あたりのインチ数で表され、デフォルト値は2.0です。

構文です。

gesture_flick_speed(speed);

論証タイプ 商品説明
speed The speed (in inches per second) to set for flick gesture event detection.

リターンです。

例

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

上記のコードでは、ジェスチャー用のフリック速度が2インチ/秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
