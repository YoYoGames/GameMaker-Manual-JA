ジェスチャータップ数

タップカウントの有効／無効を設定する機能です。 true に設定するとタップカウントが有効になり、 false に設定すると無効になりますが、どちらの状態になっているかは機能で確認することができます。 gesture_get_tap_count().例えば、2回タップするとシングルタップイベントとダブルタップイベントの両方が発生し、シングルタップイベントのタップカウントは1、ダブルタップイベントのタップカウントは2というように、有効な場合、すべてのタップイベントがトリガーされます。タップカウント値は、ダブルタップ検出のための設定時間が経過すると0にリセットされます（機能 gesture_double_tap_time()).タップカウントを無効にすると、ダブルタップ時間が経過して2回目のタップが検出されない限り、最初のタップは登録されません。なお、デフォルトでは有効になっています。

構文です。

gesture_tap_count(enable);

論証タイプ 商品説明
enable Enable (set to true) または無効にする ( false に設定) タップカウントを行います。

リターンです。

例

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

上記のコードでは、タップカウントが有効かどうかを確認し、有効でない場合はオンにします。
