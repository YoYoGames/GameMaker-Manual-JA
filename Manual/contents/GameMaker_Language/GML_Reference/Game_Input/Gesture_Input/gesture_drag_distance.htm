ジェスチャードラッグ距離

タッチまたはクリックの動きによってDragging Gestureイベントが発生するまでの距離を設定します。距離の単位はインチで、デフォルト値は0.1です。

構文です。

gesture_drag_distance(distance);

論証タイプ 商品説明
distance The distance (in inches) to set for drag gesture event detection.

リターンです。

例

if gesture_get_drag_distance() != 0.1
{
    gesture_drag_distance(0.1);
}

上記のコードでは、ジェスチャーのドラッグ距離が0.1インチに設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
