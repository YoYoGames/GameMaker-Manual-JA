gesture_get_drag_time

この関数は、タッチまたはクリックによってDrag Start Gestureイベントが発生するまでの時間を取得するために使用されます。この時間は、タップイベントのトリガーにも影響し、この時間が経過する前にタッチやクリックを離すとタップとみなされます。時間の単位は秒、初期値は0.16です。

構文です。

gesture_get_drag_time();

リターンです。

(time in seconds)

例

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

上記のコードでは、ジェスチャーのドラッグ時間が0.16秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
