gesture_get_pinch_angle_away

この機能は、ピンチアウトジェスチャーを開始する前に、タッチが他のタッチから離れる角度を取得するために使用されます。角度の単位は度で、初期値は45度です。

構文です。

gesture_get_pinch_angle_away();

リターンです。

(inches)

例

if gesture_get_pinch_angle_away() != 45
{
    gesture_pinch_angle_away(45);
}

上記のコードでは、ジェスチャーのピンチアウト角度が45°に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
