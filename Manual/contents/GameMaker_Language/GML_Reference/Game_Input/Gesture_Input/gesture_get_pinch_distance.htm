ジェスチャーの取得距離

この機能は、2本の指で画面をタッチ／クリックし、移動することでピンチ・ジェスチャーが発生するまでの距離を取得するために使用されます。距離はインチで測定され、デフォルト値は0.1です。

構文です。

gesture_get_pinch_distance();

リターンです。

(inches)

例

if gesture_get_pinch_distance() != 0.1
{
    gesture_pinch_distance(0.1);
}

上記のコードでは、ジェスチャーのピンチ距離が0.1インチに設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
