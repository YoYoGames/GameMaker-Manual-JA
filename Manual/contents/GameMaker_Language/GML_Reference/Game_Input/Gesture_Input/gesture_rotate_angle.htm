ジェスチャー回転角

この機能は、回転開始ジェスチャーを発動させるために、2つのタッチが超えなければならない角度を設定するために使用されます。角度は度単位で、初期値は5度です。

構文です。

gesture_rotate_angle();

リターンです。

(inches)

例

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

上記のコードでは、ジェスチャーの回転角度が5°に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
