ジェスチャー回転時間

この機能は、回転開始ジェスチャーが発生するために、2つのタッチが一定の方向に回転していなければならない時間を設定するために使用されます。時間は秒単位で、デフォルト値は0.16sです。

重要!現在、60fpsのゲームでは、最大1秒にしか設定できません。そうしないと、回転が検出されません。フレームレートが低い場合、この値は大きくなり、例えば30fpsのゲームでは最大2秒に設定できます。

構文です。

gesture_rotate_time();

リターンです。

(inches)

例

if gesture_get_rotate_time() != 0.1
{
    gesture_rotate_time(0.1);
}

上記のコードでは、ジェスチャーの回転時間が0.1秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
