ジェスチャー ゲット タップ カウント

この関数は、タップカウントが有効か無効かを確認するために使用されます。この関数は、有効であれば true を、そうでなければ false を返します。タップカウントの有効／無効は、関数 gesture_tap_count().例えば、2回タップすると、シングルタップイベントとダブルタップイベントの両方が発生し、シングルタップイベントのタップカウントは1、ダブルタップイベントのタップカウントは2になります。タップカウント値は、ダブルタップ検出のための設定時間が経過すると0にリセットされます（機能 gesture_double_tap_time()).タップカウントを無効にすると、ダブルタップ時間が経過して2回目のタップが検出されない限り、最初のタップは登録されません。なお、デフォルトでは有効になっています。

構文です。

gesture_get_tap_count();

リターンです。

例

if !gesture_get_tap_count()
{
    getsure_tap_count(true);
}

上記のコードでは、タップカウントが有効かどうかを確認し、有効でない場合はオンにします。
