ジェスチャーの回転時間

この関数は、回転開始ジェスチャーを発動するために、2つのタッチが超えなければならない角度を取得するために使用されます。角度の単位は度で、初期値は5度です。

構文です。

gesture_get_rotate_angle();

リターンです。

(inches)

例

if gesture_get_rotate_angle() != 5
{
    gesture_rotate_angle(5);
}

上記のコードでは、ジェスチャーの回転角度が5°に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
