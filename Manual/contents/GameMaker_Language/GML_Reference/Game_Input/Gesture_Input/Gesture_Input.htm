ジェスチャー入力

デバイス入力関数は、単純な画面タッチやデバイスの向きを検出するのに適しています。しかし、プロジェクトでは、より複雑なことを検出したい場合があります。例えば、インスタンスをドラッグしたり、room 「フリック」することができます。これらのことは、デバイス関数で実現できますが、より簡単にするためにGameMakerには、ダブルタップ、ドラッグ、フリック、ピンチ、回転を検出し、イベントに関する情報を返すジェスチャーイベントの全シリーズがインスタンスとグローバルレベルの両方で備わっています。ここで紹介する関数は、これらのイベントをいつ、どのようにトリガーするか、より細かくコントロールできるように設計されています。

重要これらの関数は、HTML5ターゲットプラットフォームでは動作しません。使用されてもエラーにはならず、単に無視されます。

以下のようなジェスチャー機能が用意されています。

  • gesture_get_drag_time
  • gesture_get_drag_distance
  • gesture_get_flick_speed
  • gesture_get_double_tap_time
  • gesture_get_double_tap_distance
  • gesture_get_tap_count
  • gesture_get_pinch_distance
  • gesture_get_pinch_angle_towards
  • gesture_get_pinch_angle_away
  • gesture_get_rotate_time
  • gesture_get_rotate_angle
