ジェスチャードラッグ時間

ドラッグジェスチャーイベントがタッチまたはクリックで発生するまでの時間を設定する機能です。この時間は、タップイベントのトリガーにも影響し、この時間が経過する前にタッチやクリックを離すとタップとみなされます。時間は秒単位で、デフォルト値は0.16です。

構文です。

gesture_drag_time(time);

論証タイプ 商品説明
time The time (in seconds) to set for drag gesture event detection.

リターンです。

例

if gesture_get_drag_time() != 0.16
{
    gesture_drag_time(0.16);
}

上記のコードでは、ジェスチャーのドラッグタイムが0.16秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
