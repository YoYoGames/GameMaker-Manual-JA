gesture_get_flick_speed

この関数は、タッチまたはクリックが解除されたときにフリックジェスチャーイベントが発生するために必要な速度を取得するために使用されます。速度は1秒あたりのインチ数で表され、デフォルト値は2.0です。

構文です。

gesture_get_flick_speed();

リターンです。

(inches per second)

例

if gesture_get_flick_speed() != 2
{
    gesture_flick_speed(2);
}

上記のコードでは、ジェスチャー用のフリック速度が2インチ/秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
