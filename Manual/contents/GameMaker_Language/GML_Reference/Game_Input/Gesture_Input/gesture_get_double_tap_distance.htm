ジェスチャーゲットダブルタップ距離

ダブルタップジェスチャーを発生させるために、1回タップした後に再度画面をタッチ／クリックしなければならない距離を取得するための機能です。距離はインチで測定され、デフォルト値は0.1です。

構文です。

gesture_get_double_tap_distance();

リターンです。

(inches)

例

if gesture_get_double_tap_distance() != 0.1
{
    gesture_double_tap_distance(0.1);
}

上記のコードでは、ジェスチャーのダブルタップ距離が0.1インチに設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
