gesture_double_tap_time

ダブルタップジェスチャーイベントを発生させるために、2回のタッチ／クリックの間にかかる時間を設定する機能です。時間の単位は秒、デフォルト値は0.16です。

構文です。

gesture_double_tap_time(time);

論証 タイプ 商品説明
time リアル The time (in seconds) to set for double tap gesture event detection.

リターンです。

N/A

例

if gesture_get_double_tap_time() != 0.16
{
    gesture_get_double_tap_time(0.16);
}

上記のコードでは、ジェスチャーのダブルタップ時間が0.16秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
