ジェスチャーゲットダブルタップタイム

この関数は、ダブルタップジェスチャーイベントを発生させるために、2つのタッチ/クリックの間にかかる時間を取得するために使用されます。時間の単位は秒、デフォルト値は0.10です。

構文です。

gesture_get_double_tap_time();

リターンです。

Real (seconds)

例

if gesture_get_double_tap_time() != 0.16
{
    gesture_double_tap_time(0.16);
}

上記のコードでは、ジェスチャーのダブルタップ時間が0.16秒に設定されているかどうかを確認し、設定されていない場合はその値に設定しています。
