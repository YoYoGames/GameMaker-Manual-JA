イベントデータ

この変数はグローバルなもので、適切なイベント（ジェスチャーイベントや ブロードキャストメッセージなど）で使用されるときはDSマップを保持し、それ以外のときは-1される。DSマップの実際の内容は、それをトリガーしたイベントの種類に依存するので、それらのイベントに関する個々のセクションを参照してください。

構文です。

event_data;

リターンです。

DS Map ID

例

isFlick = event_data[?"isflick"];

if (isFlick)
{
    flickVelX = event_data[?"diffX"];
    flickVelY = event_data[?"diffY"];
}
else
{
    flickVelX = 0;
    flickVelY = 0;
}

上記のコードは、Drag Endジェスチャーイベントから取得され、ベントが「フリック」イベントであるかどうかをチェックし、もしそうなら event_data DS マップから必要なデータを抽出し、それを使っていくつかの変数をセットしています。フリック」イベントが検出されない場合は、同じ変数が 0 に設定されます。
