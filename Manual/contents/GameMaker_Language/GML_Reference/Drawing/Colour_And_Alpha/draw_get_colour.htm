draw_get_colour

この関数は、フォーム、テキスト、プリミティブ、テクスチャのない 3D モデルの描画に使用される現在の描画色を返すものである。これは draw_set_colour()関数があります。

構文です。

draw_get_colour()

リターンです。

Colour Constant

例

var _cur_color = draw_get_color();
draw_set_color(text_color);
draw_text(x, y, text);
draw_set_color(_cur_color);

上記のコードでは、現在の描画色をローカル変数に格納し、インスタンス変数に基づいて描画色を変更します。テキストをいくつか描画した後、描画色をローカル変数に格納されている値にリセットしている。
