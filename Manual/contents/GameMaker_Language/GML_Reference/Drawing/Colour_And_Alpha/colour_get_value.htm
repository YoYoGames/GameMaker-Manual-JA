color_get_value

この関数は、与えられた色の値（明度）を返します。これは最終的な色に混ぜられる「光」の量であり、色を定義するための色相、彩度、明度法の一部となります。次の画像は この値が色のHSVスケールとどのように対応しているか。

構文です。

colour_get_value(col);

論証タイプ 商品説明
col The colour to check

リターンです。

例

col = make_colour_hsv(random(255), 255, colour_get_value(c_teal));

上記のコードは、色定数 "c_teal "を作るために使われた値を取得し、その値を使って同じ輝度を持つランダムな色を設定し、その結果を変数 "col "に格納しています。
