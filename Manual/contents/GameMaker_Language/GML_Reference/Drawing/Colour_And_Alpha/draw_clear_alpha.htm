draw_clear_alpha

この関数は、画面全体を指定した色でクリアし、保存先のアルファ成分を設定した値に設定します。 設定したデスティネーションアルファを使用するように設定されます。これはインスタンスの描画イベントでのみ使用します（他のイベントで使用すると表示されません）。また、サーフェスを新しく作成したときにクリアするのに非常に便利です。

HTML5モジュールを使用している場合、この関数を使用してキャンバスの背景を透明にすることができますので、ホストHTMLで使用されている画像は、その上でゲームが実行されていることを確認することができます。この効果を得るには まず、room エディターで、ルームエディターから 背景レイヤーを削除し、ビュープロパティーの クリアビューバックグラウンドのオプションのチェックを外す必要があります。 最後に、新しいobject を作成し、ゲームの最初のroom に配置し、非常に高い深度（例えば、10000）を割り当てる必要があります。そして、描画前のイベントで、この関数を使用して、キャンバスのアルファ値を、希望の色と透明度に設定します。 を持つようにします。

構文です。

draw_clear_alpha(col, alpha);

論証タイプ 商品説明
col The colour with which the screen will be cleared
alpha The transparency of the colour with which the screen will be cleared

リターンです。

例

draw_clear_alpha(c_red, 0.5);

これは、赤の色で50％（0.5）の透明度で画面をクリアします。
