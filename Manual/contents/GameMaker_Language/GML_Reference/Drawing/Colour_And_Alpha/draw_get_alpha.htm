draw_get_alpha

この関数は、描画アルファ値の現在値を返します。この値は、0（完全に透明）〜1（完全に不透明）の範囲です。ドローアルファは、すべての描画関数の透明度に影響し、 draw_set_alpha() 関数で設定することができます。

構文です。

draw_get_alpha()

リターンです。

例

var _cur_alpha = draw_get_alpha();
draw_set_alpha(text_alpha);
draw_text(x, y, text);
draw_set_alpha(_cur_alpha);

上記のコードでは、現在の描画アルファ値をローカル変数に格納し、インスタンス変数に基づいて描画アルファ値を変更します。テキストをいくつか描画した後、アルファ値をローカル変数に格納されている値にリセットします。
