color_get_hue

この関数は、与えられた色の色相を返します。これは、色を定義するための色相、彩度、値（明度）メソッドの一部である「純粋な」色調です。次の画像は、この値がどのように 色のHSVスケール

構文です。

colour_get_hue(col);

論証タイプ 商品説明
col The colour to check

リターンです。

例

col = merge_colour(colour_get_hue(c_teal), c_white, 0.5);

上記のコードは、色定数 "c_teal "を作るために使われた色相を取得し、それを白と50：50の割合で合成し、結果の色を変数 "col "に格納します。
