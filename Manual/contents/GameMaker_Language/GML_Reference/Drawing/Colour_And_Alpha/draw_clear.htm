draw_clear

この関数は、画面全体を（アルファブレンドなしで）指定した色にクリアするために使用でき、インスタンスの draw イベントでのみ使用できます（他のイベントで使用すると、表示されません）。また、サーフェスを新しく作成したときにクリアするのにも便利です。

構文です。

draw_clear(col);

論証タイプ 商品説明
col The colour with which the screen will be cleared

リターンです。

例

draw_clear(c_blue);

これにより、青色で画面がクリアされます。
