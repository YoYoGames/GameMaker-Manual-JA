color_get_green

この関数は、与えられた色を作るために使われた緑の量を返します。値は 0 から 255 の間で、0 は緑なし、255 はすべて緑です。次の画像はこれを示しています。

構文です。

colour_get_green(col);

論証タイプ 商品説明
col The colour to check

リターンです。

例

g_comp = colour_get_green(c_teal);

上記のコードでは、色定数「c_teal」の緑成分を取得し、その値を変数「g_comp」に格納します。
