draw_getpixel_ext

この関数を使用すると、現在のレンダーターゲットに描画されている任意のピクセルの完全なAGBR 32ビット値を取得することができます。つまり、結果はこの関数が呼び出されたイベントと、使用されているターゲットサーフェスに依存します。

重要この機能は、パフォーマンスに大きな影響を与えるため、絶対に必要な場合にのみ使用してください。

構文です。

draw_getpixel_ext(x, y);

論証タイプ 商品説明
x The x coordinate of the pixel to check
y The y coordinate of the pixel to check

リターンです。

Real

例

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

上記のコードでは、マウスの位置の32bitカラー値を取得し、それを構成する値に分割して変数に格納します。
