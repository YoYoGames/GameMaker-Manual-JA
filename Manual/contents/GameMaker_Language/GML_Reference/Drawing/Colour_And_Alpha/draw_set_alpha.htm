draw_set_alpha

この関数では、ゲームの基本的な描画アルファを設定することができます。この値は 0 から 1 の間で設定でき、0 は完全に透明、1 は完全に不透明です（デフォルト値）。この値は、背景、sprites 、fonts を含む、以降のすべての描画に影響します。 プリミティブと3D。

assets のいずれかを描画し、関数の一部としてアルファ引数を与える場合（例えば、以下のような場合）には、注意してください。 draw_sprite_ext()で設定されたアルファ値は draw_sprite_ext() を使用すると、 draw_set_alpha() で設定したアルファ値に関係なく、sprite をアルファ引数 1 で描画すると、sprite は完全に不透明に描画されます（この動作は draw_self() を設定し image_alphaインスタンスの)。また、インスタンスがデフォルトの描画である場合（つまり、sprite を持つが、Draw イベントがない場合）、そのインスタンスの image_alpha 変数は draw_set_alpha() で設定されたアルファ値にクランプされます。例えば、 image_alpha が 1 で、描画アルファ値が 0.5 の場合、sprite はアルファ値 0.5 で描画され、 image_alpha が 0.25 で、描画アルファ値が 0.5 の場合、 は 0.5 で描画されます。 の場合、sprite は0.25アルファで描画されます。

構文です。

draw_set_alpha(alpha);

論証タイプ 商品説明
alpha The alpha to set (between 0 and 1)

リターンです。

例

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上記のコードは、指定された位置にテキストを描画し、描画のアルファと色を変更することでシャドウ効果を作成するものです。
