color_get_red

この関数は、与えられた色を作るために使われた赤の量を返します。値は0から255の間で、0は赤なし、255はすべて赤です。次の画像はこれを示しています。

構文です。

colour_get_red(col);

論証タイプ 商品説明
col The colour to check

リターンです。

例

r_comp = colour_get_red(c_teal);

上記のコードでは、色定数「c_teal」の赤成分を取得し、その値を変数「r_comp」に格納します。
