make_colour_hsv

GameMakerでは、この機能を利用して、ユーザーが自分の好きな色を作ることができます。この関数は、色相、彩度、明度という3つの要素から色を作り出します。これらの値は0から255の間とみなされるので、これで16,777,216（256*256*256）色を作ることができるのです!下の画像は、これらのコンポーネントを分離したときの様子です。

左の画像は、関数の個々の構成要素を分解したもので、次に右の画像は、これらの構成要素を変更することで最終的な色にどのような影響を与えるかを説明したものです。このように、色相を変えなくても（0、赤のまま）、他の2つの設定で色を劇的に変化させることができます。彩度では色調がグレー（0）から純色（255）へ、値ではより暗い（0）から明るい（255）へ変化させることができます。

構文です。

make_colour_hsv(hue, sat, val);

論証タイプ 商品説明
hue The hue of the colour
sat How saturated the colour is
val How dark the colour is

リターンです。

Colour Constant

例

col = make_colour_hsv(100, 145, 255);

上記のコードでは、この関数を使って色を作成し、その値を変数「col」に保存して後で使用するようにしています。
