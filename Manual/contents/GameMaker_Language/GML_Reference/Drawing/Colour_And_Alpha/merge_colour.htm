マージカラー

この関数を使うと、2つの色を取り込んで、それらを合成して新しい色を作ることができます。各構成色の量は、引数 "amount" を変更することで定義できます。0 の値は最初の色 (col1) を返し、1 の値は 2 番目の色 (col2) を返し、その間の値は対応する混合を返します。例えば、0.5を指定すると、2つの色が均等に混合されます。次の画像は、赤と青の色を合成することで、この機能を説明しています。

構文です。

merge_colour(col1, col2, amount);

論証タイプ 商品説明
col1 The first colour to merge
col2 The second colour to merge
amount How much of each colour should be merged. For example, 0 will return col1, 1 will return col2, and 0.5 would return a merge of both colours equally

リターンです。

Colour Constant

例

col = merge_colour(c_lime, c_orange, 0.5);

上記のコードでは、この関数を使ってライムとオレンジを50対50で合成した色を作り、その値を変数「col」に格納して後で使用するようにしています。
