draw_set_colour

この機能により、ゲームのベースとなる描画色を設定することができます。

fonts 、フォーム、プリミティブ、3Dの描画に影響を与えますが、sprites （手動またはインスタンスによって描画）には影響を与えません。影響を受けるグラフィックスが独自の色値で描画される場合、この値は無視されます。

構文です。

draw_set_colour(col);

論証 タイプ 商品説明
col カラー The colour to set for drawing.

リターンです。

N/A

例

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

上記のコードは、指定された位置にテキストを描画し、描画のアルファと色を変更することでシャドウ効果を作成するものです。
