draw_get_font

この関数は、現在テキストを描画するために割り当てられているfont を取得します。この関数は、font が設定されていない場合は -1、またはfont asset に割り当てられた ID 値(正の整数)を返します。

構文です。

draw_get_font();

リターンです。

Font Asset or -1

例

var _cur_font = draw_get_font();
var _y_offset = 0;

switch (_cur_font)
{
    case ft_small:
        _y_offset = 10;
    break;

    case ft_medium:
        _y_offset = 22;
    break;

    case ft_big:
        _y_offset = 34;
    break;

    default:
        _y_offset = 8;
}

draw_text(room_width / 2, 200 + _y_offset, "MENU");

上記のコードは、現在適用されているfont を取得し、それに対してswitch ステートメントを実行し、font に応じて異なる Y オフセット値を適用します。 そして、そのオフセット値を使用して、いくつかのテキストを描画します。
