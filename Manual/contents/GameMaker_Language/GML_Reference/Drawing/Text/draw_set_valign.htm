draw_set_valign

この関数は、テキストを垂直軸に沿って配置するために使用します。垂直方向の配置を変更すると、以降のすべてのテキストの描画位置と方向が変更されます。デフォルト値は fa_top です。以下の定数を使用できます。

定数 アライメント
fa_top
fa_middle
fa_bottom

構文です。

draw_set_valign(valign);

論証 タイプ 商品説明
valign 垂直アライメント定数 Vertical alignment constant (from the table above).

リターンです。

N/A

例

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

上記のコードでは、スコアはテキストのまさに中央を中心に描画されます。
