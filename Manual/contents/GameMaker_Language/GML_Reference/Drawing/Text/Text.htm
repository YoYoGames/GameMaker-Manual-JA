テキスト

様々な形でテキストを描画するための関数が用意されています。なお、これらの関数の中には、描画時にテキストの色やサイズをダイナミックに変化させるものがありますが、HTML5やデバイスを扱う場合、これらの関数はゲームの速度を低下させたり、見栄えを悪くしたりするので、拡大縮小描画を使うよりも様々なサイズのfont のリソースを用意したり、大きなfont リソースを持って、それを拡大ではなく 縮小して描画するとよい場合があります。また、大きなテキストブロックに色をつける必要がある場合は draw_set_colour()と draw_set_alpha()関数と単純な draw_text 関数で構成され、色の要素を持つ拡張関数ではありません。

以下の機能は、描画時に使用するfont の設定と、描画時のテキストの位置決めを行うものです。

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

これらの関数は、単純に、あるいは変形や色を付けて、画面にテキストを描画するために使用することができます。

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

また、ビルトインハイコアシステムで作成したハイスコアを描画するための専用関数として、以下の関数が存在します。

  • draw_highscore
