ドローテキストエクストカラー

この機能は、ベース draw_text()関数に draw_text_ext()と draw_text_colour()この関数を使うと、テキストのグラデーションカラーや、行間、1行の最大幅を一括して定義することができます。

注意事項 グラデーション ブレンディングは、WebGL が有効になっていない限り、HTML5 ターゲットでは利用できません。ただし、ブレンディング カラーを設定すると、font を最初に指定した色でブレンディングします。ただし、この方法でブレンドすると、font の複製が作成され、cache に保存されて必要なときに使用されます。これは最適とは言えず、複数の色変更を使用するとゲームのパフォーマンスが低下することになります。font cache のサイズを設定することで、これを制限することができます。 font_set_cache_size().

構文です。

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

論証 タイプ 商品説明
x リアル The x coordinate of the drawn string.
y リアル The y coordinate of the drawn string.
string 文字列 The string to draw.
sep リアル The distance in pixels between lines of text.
w リアル The maximum width in pixels of the string before a line break.
c1 カラー The colour for the top left of the drawn text.
c2 カラー The colour for the top right of the drawn text.
c3 カラー The colour for the bottom right of the drawn text.
c4 カラー The colour for the bottom left of the drawn text.
alpha リアル The alpha for the text.

リターンです。

N/A

例

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

上記のコードは、ユーザーがキーボードで入力したテキストを、string の長さが 300 ピクセルを超えるたびに新しい行に分割して描画します。この場合、行間も 5 ピクセルの間隔を維持します。各行のテキストは青のグラデーションで色付けされ、上部が明るい青、下部が暗い青になります。
