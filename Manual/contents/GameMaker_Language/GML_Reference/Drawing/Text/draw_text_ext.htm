draw_text_ext

と同様の方法でテキストを描画します。 draw_text()ただし、テキストが2行以上ある場合は、各行の間隔を設定し、幅を (ピクセル) で制限することができます。 string この値を超えた行があった場合、GameMakerは自動的に次の行にテキストを分割します（テキストに空白がない場合は、この最大値を超えます）。 幅の値)。このため、string の最初に置かれた空白は、描画のために解析される前に取り除かれることに注意してください。また、行の区切りの引数に -1 を指定すると、デフォルトでは の高さは，選択されたfont の文字列の高さです．

構文です。

draw_text_ext(x, y, string, sep, w);

論証タイプ 商品説明
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.
sep The distance in pixels between lines of text.
w The maximum width in pixels of the string before a line break.

リターンです。

例

draw_text_ext(100, 50, keyboard_string, 3, 300);

上記のコードは、ユーザーがキーボードで入力したテキストを、string の長さが 300 ピクセルを超えるたびに新しい行に分割して描画します。この場合、行間も 3 ピクセルの間隔を維持するようになります。
