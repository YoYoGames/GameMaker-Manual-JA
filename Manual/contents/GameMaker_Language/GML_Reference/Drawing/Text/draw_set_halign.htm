draw_set_halign

この関数は、テキストを水平軸に沿って配置するために使用します。水平方向の配置を変更すると、以降のすべてのテキストが描画される位置と方向が変わります。デフォルト値は、 fa_left です。次の定数を使用できます。

定数 アライメント
fa_left
fa_center
fa_right

構文です。

draw_set_halign(halign);

論証 タイプ 商品説明
halign ホリゾンタルアライメント定数 Horizontal alignment constant (from the table above).

リターンです。

N/A

例

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

上記のコードは、同じ行に2つのstrings 、スコアは左寄せ、健康状態は右寄せで描画されます。
