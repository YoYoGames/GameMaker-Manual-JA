draw_text

この関数を使うと、string 内の任意の位置に任意のroom を描画することができます（実数を描画する場合は string()関数でテキストに変換します)。strings を結合するには+を使用します(下記の例を参照)。また、string の中で\nを使用すると、改行を追加して複数行に渡って描画されます (string を正しくフォーマットする方法と使用できるエスケープ文字については、ここを参照してください)。テキストの色とアルファは、次のように設定された現在の基本アルファと色の値によって支配されます。 draw_set_alpha()と draw_set_colour().

注：テキストの実際の位置は、以下の方法で設定したアライメント値に影響されます。 draw_set_halign()と draw_set_valign().

構文です。

draw_text(x, y, string);

論証タイプ 商品説明
x The x coordinate of the drawn string.
y The y coordinate of the drawn string.
string The string to draw.

リターンです。

例

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

上記のコードでは、インスタンスのx/yの位置にstring 。グローバル変数「Name」に格納されたstring 、2行に分割して描画します。
