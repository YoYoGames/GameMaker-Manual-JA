gpu_set_blendmode_ext_sepalpha

GameMakerがピクセルを描画しようとするとき、ソースカラー（描画しようとするピクセルの色）とデスティネーションカラー（描画先のピクセルにすでにある色）があります。したがって、ピクセルの新しい色を決定するとき、ソースカラーとデスティネーションの色は選択したブレンドモードに従って計算されます。色の各要素は0から1の間の浮動小数点値として保存され、新しい色は、ソースカラーの各要素に何らかの係数を掛け、デスティネーションカラーの各要素に何らかの係数を掛けて、その結果を要素ごとに加算することによって計算されます。

この関数は、カスタムブレンドモードを作成するために、RGB成分とアルファ成分に異なるブレンドモード係数の部分を選択し、一緒に因数分解すべき異なる構成要素を設定することを許可します。つまり、"src" と "dest" 引数は、最初の3つのブレンドモード値（RGB）を使用し、"alphasrc" と "alphadest" 引数は、ブレンドモードファクターの最後の部分をとります。例えば、RGBのソースを bm_zero とすると、(0, 0, 0) が使用されます。アルファのソースをbm_oneに設定すると、(1)が使用され、最終的なソースのブレンドモードは(0, 0, 0, 1)となります。

下図は、各ブレンドモードにおけるRGBA値のうち、（Rs、Gs、Bs、As）をソース値、（Rd、Gd、Bd、Ad）をデスティネーション値と見なしたものである。使用可能な11個の定数は、この機能で使用する場合、ソースまたはデスティネーション（またはその両方）のどちらかになります。

定数 ブレンド係数（赤、緑、青、アルファ）
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs、1-Gs、1-Bs、1-As)
bm_src_alpha (アス、アス、アス、アス)
bm_inv_src_alpha （1-As、1-As、1-As、1-As）
bm_dest_alpha （アド、アド、アド、アド）
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) ここで、f = min(As, 1-Ad)

重要!WebGLが有効でないHTML5では、以下のモードを正しく表示することができません。

  • bm_src_colour
  • bm_inv_src_colour
  • bm_dest_colour
  • bm_inv_dest_colour
  • bm_src_alpha_sat

この関数には、4つの個別の引数を指定することも、引数の配列を指定することもできます（この場合、関数 gpu_get_blendmode_ext_sepalpha()のように）。配列で指定する場合は、以下の4つの要素で構成されている必要があります。

  • [0] = ソースブレンドモードからアルファ成分を除いたもの（デフォルトは bm_src_alpha （As、As、As））。
  • [1] = デスティネーションブレンドモードからアルファ成分を除いたもの（デフォルトは bm_inv_src_alpha （1-As、1-As、1-As））。
  • [2] = ソースのアルファ成分（デフォルトは bm_src_alpha （As））。
  • [3] = 出力先アルファ成分（デフォルトは bm_inv_src_alpha （1-As））。

ブレンドモードを最大限に活用し、その仕組みと画面に描画される最終画像への影響を理解するために、以下のガイドをお読みになることをお勧めします。

  • Blendmodesの使い方ガイド

構文です。

gpu_set_blendmode(src, dest, alphasrc, alphadest);

論証 タイプ 商品説明
src ブレンドモードファクター定数 Source blend mode factor for RGB (see constants above).
dest ブレンドモードファクター定数 Destination blend mode factor for RGB (see constants above)
alphasrc ブレンドモードファクター定数 Source blend mode factor Alpha (see constants above).
alphadest ブレンドモードファクター定数 Destination blend mode factor Alpha (see constants above)

リターンです。

N/A

例

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

上記のコードでは、ローカル配列を作成し、現在の拡張ブレンドモードを個別のアルファ値で取得しています。この配列を操作して、ブレンドモードを設定するために返します。
