gpu_get_blendmode

この関数は、描画に使用されている現在のブレンドモードを取得するために使用することができます。返される値は、以下の定数のいずれかです（デフォルト値は bm_normal ）。

ブレンドモードコンスタント
定数 商品説明 拡張ブレンドモード
bm_normal 通常のブレンド（デフォルトのブレンドモード）。 (bm_src_alpha, bm_inv_src_alpha)
bm_add 加法混色。明るい部分の輝度値が加算されます。 (bm_src_alpha, bm_one)
bm_subtract 減算式ブレンディング。明るい部分の輝度値が減算される。 (bm_zero, bm_inv_src_colour)
bm_max マックスブレンディング加算ブレンディングに似ている。 (bm_src_alpha, bm_inv_src_colour)

構文です。

gpu_get_blendmode();

リターンです。

Blend Mode Constant (see above for constants)

例

if gpu_get_blendmode() != bm_normal
{
    gpu_set_blendmode(bm_normal);
}

上記のコードは、現在のブレンドモードを取得し、それが bm_normal でない場合は、その定数に設定されます。
