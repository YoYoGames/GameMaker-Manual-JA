gpu_set_fog

この機能により、フォグの描画を有効または無効にすることができます。3Dゲームでは、霧を利用して遠くのインスタンスを薄く見せたり、見えないようにしたりすることができ、雰囲気を出したり、遠くのインスタンスを描画していないことを隠したりすることができます。

有効 (true) か無効 (false) か、フォグのブレンドに使用する色、および描画の開始と終了の距離を設定します。距離の値は、基本的にピクセル単位の深度値であり、カメラの位置からの相対的なものです。

例えば、位置0はカメラに最も近く、増加するごとに遠ざかります。デフォルトでは、ビューのカメラは深度-16000に配置されているので、フォグを深度0から始めて深度1000で終了させたい場合は、開始距離を16000に、終了距離を17000に設定することになります。

この関数は、4つの個別引数（上にあげたもの）、または以下の構造を持つ1つの配列を取ることができます（下のコード例ではこの方法を示しています）。

  • [0] = 有効トグル (ブール値、 true または false)、デフォルト。 false
  • [1] = 色（実数）、デフォルト c_black
  • [2] = 開始距離（実数）、デフォルト0
  • [3] = 終了距離（実数）、デフォルト1

構文です。

gpu_set_fog(enable, colour, start, end);

論証 タイプ 商品説明
enable ブーリアン Enable or disable fog
colour カラー The fog colour
start リアル The distance to start applying fog from (relative to the camera)
end リアル The distance to end the fog (relative to the camera)

リターンです。

N/A

例

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上記のコードでは、現在のフォグ設定を取得し、配列の色要素を c_red に設定してから、変更後の配列を使って再度フォグを設定しています。
