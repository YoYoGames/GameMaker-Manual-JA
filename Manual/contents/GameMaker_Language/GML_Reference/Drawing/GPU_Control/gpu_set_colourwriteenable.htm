gpu_set_colourwriteenable

この機能で、switch 、以降のすべての描画のためのカラーチャネルとアルファチャネルをオンまたはオフにすることができます。例えば、この関数を使ってアルファチャンネルのみのサーフェイスを作成したり（switch サーフェイスを使用する前に RGB の書き込みをオフにする）、スクリーンに描画する際に特殊効果を作成したりすることが可能です。各構成要素のデフォルト値は true で、4 つのユニークな引数として、または true (有効) または false (無効) のいずれかになる次の要素を含む 4 つの要素の 1 次元配列として提供することができます。

  • [0] = 赤チャンネル有効/無効
  • [1] = グリーンチャンネル有効/無効
  • [2] = 青チャンネルの有効化/無効化
  • [3] = アルファチャンネル有効/無効

構文です。

gpu_set_colourwriteenable(red, green, blue, alpha);

論証 タイプ 商品説明
red ブーリアン Enable/disable the red channel
green ブーリアン Enable/disable the green channel
blue ブーリアン Enable/disable the blue channel
alpha ブーリアン Enable/disable the alpha channel

リターンです。

N/A

例

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上記のコードでは、現在の色の書き込み値を取得し、アルファ成分を false に設定しています。
