gpu_get_blendenable

この関数は、アルファブレンディングの状態を取得するために使用することができる。 true を返した場合はアルファブレンディングが有効であり、 false を返した場合は無効である。デフォルトでは、これはオンになっているので、この関数は以下を返します。 true

構文です。

gpu_get_blendenable();

リターンです。

例

if gpu_get_blendenable() == false
{
    gpu_set_blendenable(true);
}

上記のコードでは、アルファブレンディングの状態をチェックし、無効になっている場合は再び有効にします。
