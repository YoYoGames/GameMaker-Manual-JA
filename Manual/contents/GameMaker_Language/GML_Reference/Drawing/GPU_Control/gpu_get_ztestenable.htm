gpu_get_ztestenable

この関数は、z-testing が有効かどうか（この関数は true を返します）、または無効かどうか（この関数は false を返します）を取得するために使用することができます。デフォルト値は，z-testingが無効であるため，この関数は false を返します。

構文です。

gpu_get_ztestenable();

リターンです。

例

if gpu_get_ztestenable() == false
{
    gpu_set_ztestenable(true);
}

上記のコードでは、z-testingが有効かどうかをチェックし、無効になっている場合は再度有効にしています。
