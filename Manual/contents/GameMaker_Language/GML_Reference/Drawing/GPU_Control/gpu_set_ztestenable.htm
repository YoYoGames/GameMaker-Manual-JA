gpu_set_ztestenable

この関数は、Zバッファーテストのオン/オフを切り替え、描画方法に影響を与えます（一般に、3Dプロジェクトで作業する場合にのみ使用されます）。基本的に、デフォルトでは、z-testing がオフで、2 つのobjects が同じ空間に描画している場合、z-buffer-testing のオン/オフを切り替えることができます。 objects の両方が、一方がもう一方を過剰に描画するかどうかに関係なくレンダリングされ、結果として不必要な描画呼び出しが発生します。switch これをオンにすると、Z バッファがテストされ、object が「見える」かどうかが確認され、「見えない」場合は描画されません。 である。これはデフォルトの動作ですが、z-bufferのテストに使用される比較のタイプを変更することで変更できることに注意してください（関数 gpu_set_zfunc().

デフォルトでは、z-bufferテストはオフ（false ）です。

構文です。

gpu_set_ztestenable(enable);

論証タイプ 商品説明
enable Enable or disable z-buffer testing (true または )。 false

リターンです。

例

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上記のコードは、背景sprite を描画するために z-buffer テストをオンにし、描画を継続するために再びオフに切り替えています。
