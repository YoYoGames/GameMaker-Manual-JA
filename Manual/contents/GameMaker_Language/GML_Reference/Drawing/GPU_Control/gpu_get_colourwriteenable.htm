gpu_get_colourwriteenable

この関数は、現在の色の書き込み許可値を取得するために使用することができます。この関数は，次の要素を持つ4要素の1次元配列を返します。この配列は， true （有効）または false （無効）のいずれかとなります。デフォルトでは、すべての色書き込みは true に設定されています。

  • [0] = 赤チャンネル有効/無効
  • [1] = グリーンチャンネル有効/無効
  • [2] = 青チャンネルの有効化/無効化
  • [3] = アルファチャンネル有効/無効

構文です。

gpu_get_colourwriteenable();

リターンです。

(4 elements only; see above for details)

例

var cw = gpu_get_colorwriteenable();
cw[3] = false;
gpu_set_colorwriteenable(cw);

上記のコードでは、現在の色の書き込み値を取得し、アルファ成分を false に設定しています。
