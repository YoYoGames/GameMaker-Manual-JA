gpu_get_alphatestref

この関数を使用すると、アルファテストのリファレンスの現在の値を見つけることができます（デフォルトは0。 gpu_set_alphatestref()でこの値以外を設定することができます)。

構文です。

gpu_get_alphatestref();

リターンです。

(from 0 - 255)

例

if gpu_get_alphatestenable()
{
    if gpu_get_alphatestref() < 254
    {
        gpu_set_alphatestref(254);
    }
}

上記のコードでは、アルファテストが有効かどうかを確認し、有効な場合は、現在のアルファテストの基準値を確認し、254より小さい場合はそれを設定します。
