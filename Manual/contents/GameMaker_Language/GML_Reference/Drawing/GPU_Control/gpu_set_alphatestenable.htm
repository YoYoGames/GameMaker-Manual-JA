gpu_set_alphatestenable

この機能は、あなたのゲームのアルファテストを有効または無効にします（デフォルトでは無効になっています）。アルファテストをオンにすることで、コンパニオン関数 gpu_set_alphatestref()以下の画像は、アルファテストのオンとオフで見られる違いを示しています。

注意：この機能は、iOS および Android デバイスのパフォーマンスに悪影響を及ぼす可能性があります。

構文です。

gpu_set_alphatestenable(enable);

論証タイプ 商品説明
enable Enable or disable alpha testing (true / ) false

リターンです。

例

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

上記のコードは、アルファテストが有効かどうかを確認し、有効でない場合は、switch アルファテストをオンにし、テストの閾値を128に設定します（アルファ値が0.5を超えるピクセルだけが描画されます）。
