gpu_set_state

この関数は、渡されたDSマップを使用して現在のGPUの状態を設定します。渡されたマップは、関数 gpu_get_state().

構文です。

gpu_set_state(ds_map);

論証タイプ 商品説明
ds_map The GPU state to set as a DS Map.

リターンです。

例

gpu_set_state(gpu_map);

上記のコードでは、変数「gpu_map」に指定されたマップを使用して GPU の状態を設定します。
