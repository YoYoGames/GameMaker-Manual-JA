gpu_set_texfilter_ext

GameMakerで Shadersを使用する際、1つのサンプラー「スロット」に対して線形補間を設定することができる関数です。この設定を有効にすると (true) サンプラーtexture が平滑化され、無効にすると (false) 最も近いピクセルに基づいて画像が描画されます。デフォルト値は、ゲームのグローバルゲームオプションで設定された値、または関数 gpu_set_texfilter().

注：この設定は、関数 gpu_set_texfilter().

注：HTML5ターゲットでは、この機能はWebGLが有効な場合のみ動作します。

構文です。

gpu_set_texfilter_ext(sampler_id, enable);

論証タイプ 商品説明
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / ) false

リターンです。

例

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texfilter_ext(s_tex))
{
    gpu_set_texfilter_ext(s_tex, false);
}
else
{
    gpu_set_texfilter_ext(s_tex, true);
}

上記のコードは、特定のサンプラーID（ローカル変数に格納）に対して、texture フィルタリングがオンかオフかを確認し、それに応じて切り替えています。
