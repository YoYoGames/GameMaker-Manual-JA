gpu_get_fog

この関数は、フォグ設定を取得するために使用することができます。この関数は、以下の情報を持つ4要素の1次元配列を返します。

  • [0] = 有効トグル (ブール値、 true または false), デフォルト false
  • [1] =カラーコンスタント、デフォルト c_black
  • [2] = 開始距離（リアル）、デフォルト0
  • [3] = 終了距離（リアル）、デフォルト1

これらの値を変更して、配列全体を gpu_set_fog()関数を単一の引数として指定します。

構文です。

gpu_get_fog();

リターンです。

Array (4 elements only; see above for details)

例

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

上記のコードでは、現在のフォグ設定を取得し、配列の色要素を c_red に設定してから、変更後の配列を使って再度フォグを設定しています。
