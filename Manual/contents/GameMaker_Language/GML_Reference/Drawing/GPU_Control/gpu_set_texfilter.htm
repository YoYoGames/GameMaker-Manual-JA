gpu_set_texfilter

この関数は、ゲーム画面に描画されるすべての画像のtexture フィルタリング（線形補間）を設定するために使用することができます。この機能を有効にすると (true)textures はすべて描画時に平滑化されます (sprites もtextures とみなされるため)。つまり、拡大縮小や移動時にピクセル比率が 1:1 でない場合、さまざまなピクセルで「にじみ」が発生し、使用するアートスタイルによって画像がぼやけて見えることがあります。この設定を無効にすると (false) 、拡大縮小や移動の際に、最も近いピクセルに基づいて画像が描画されるため、「ブロック状」の画像になることがあります。デフォルト値は false で、これはグローバルゲームオプションで個々のターゲットプラットフォーム用に変更することもできます。

注 texture ：この設定は、shaders のステージ補間の設定を上書きします。 gpu_set_texfilter_ext().

注：HTML5ターゲットでは、この機能はWebGLが有効な場合のみ動作します。

構文です。

gpu_set_texfilter(enable);

論証タイプ 商品説明
enable Enable or disable texture filtering (true / ) false

リターンです。

例

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

上記のコードでは、texture フィルタリングがオンかオフかを確認し、それに応じて切り替えています。
