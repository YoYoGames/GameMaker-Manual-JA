gpu_pop_state

この関数は、直前のGPUの状態をスタックからポップアップして適用します。参照 gpu_push_state()を参照してください。

構文です。

gpu_pop_state();

リターンです。

例

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

上記のコードは、現在の GPU 状態をスタックに「保存」し、次に特定の GPU 設定を変更し、インスタンスのグループを描画してから、GPU 状態を以前の状態にリセットします。
