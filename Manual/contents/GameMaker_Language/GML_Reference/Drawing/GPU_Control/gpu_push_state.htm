gpu_push_state

この関数を使用すると、現在の GPU の状態をスタックにプッシュして、後で使用することができます。一般に、現在の GPU 状態を「保存」し（ブレンドモード、アルファ書き込み、カリングなど、すべてスタックにプッシュされます）、次に別の設定で何かを描き、GPU スタックを（ gpu_pop_state()).

構文です。

gpu_push_state();

リターンです。

例

gpu_push_state();
gpu_set_blendmode(bm_add);
gpu_set_blendenable(false);
gpu_set_cullmode(true);
with (obj_Effect_Parent)
{
    draw_self();
}
gpu_pop_state();

上記のコードは、現在の GPU 状態をスタックに「保存」し、次に特定の GPU 設定を変更し、インスタンスのグループを描画してから、GPU 状態を以前の状態にリセットします。
