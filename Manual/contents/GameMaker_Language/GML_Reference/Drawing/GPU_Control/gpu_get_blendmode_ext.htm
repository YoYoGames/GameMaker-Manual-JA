gpu_get_blendmode_ext

この関数は、描画に使用されている現在の拡張ブレンドモードを取得するために使用することができます。この関数は、以下の要素を持つ2要素の1次元配列を返します。

  • [0] = ソース ブレンドモードファクター定数(デフォルトは bm_src_alpha)
  • [1] = デスティネーション ブレンドモードファクター定数(デフォルトは bm_inv_src_alpha)

配列の値は、以下の定数のいずれかになります（"s "はソースから取得した値、"d "はデスティネーションから取得した値を表します）。

ブレンドモードファクター定数
定数 ブレンド係数（赤、緑、青、アルファ）
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs、1-Gs、1-Bs、1-As)
bm_src_alpha (アス、アス、アス、アス)
bm_inv_src_alpha （1-As、1-As、1-As、1-As）
bm_dest_alpha （アド、アド、アド、アド）
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) ここで、f = min(As, 1-Ad)

これらの値を変更して、配列全体を gpu_set_blendmode_ext()関数を単一の引数として指定します。

構文です。

gpu_get_blendmode_ext();

リターンです。

Array (2 elements only; see above for constants)

例

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

上記のコードでは、現在の拡張ブレンドモードを取得し、ソースを変更した後、拡張ブレンドモードを再度設定しています。
