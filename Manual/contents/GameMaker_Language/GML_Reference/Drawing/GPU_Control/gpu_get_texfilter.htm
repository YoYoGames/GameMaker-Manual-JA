gpu_get_texfilter

この関数を使うと、texture フィルタリング（線形補間）が有効か（ true を返す）、無効か（ false を返す）を確認することができます。texture フィルタリングの詳細については，関数 gpu_set_texfilter().

注：HTML5ターゲットでは、この機能はWebGLが有効な場合のみ動作します。

構文です。

gpu_get_texfilter();

リターンです。

Boolean

例

if gpu_get_texfilter()
{
    gpu_set_texfilter(false);
}
else
{
    gpu_set_texfilter(true);
}

上記のコードでは、texture フィルタリングがオンかオフかを確認し、それに応じて切り替えています。
