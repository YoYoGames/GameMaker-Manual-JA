gpu_set_alphatestref

この関数は、アルファテストの基準値を設定します（但し、アルファテストが gpu_set_alphatestenable()).これは、アルファを持つピクセルが描画されない「カットオフ」閾値である。

基本的に、この値を0に設定し、線形補間をオンにしている場合（ゲームオプションまたは gpu_set_texfilter()) の場合、オーバーラップしているobjects のエッジで視覚的なエラーが発生することがあります。これは、ほとんど完全に透明な領域があり、その後ろに描かれたものがブロックされるからです。この参照値をたとえば 254（つまり、最大値の 1 つ下）に設定すると、100% 不透明なピクセルだけが描画されることが保証されます。また、texture 補間をオフにすることもできます。この場合、通常はデフォルトの値で問題なく使用でき、この機能を使用する必要はありませんが、ソースアートワークに大きく依存することになります。

注意：これは高度な描画機能なので、必要かどうかわからない場合や、何をするのかよくわからない場合は、ゲームの描画に深刻な影響を与える可能性があるので、使わない方がよいでしょう。

構文です。

gpu_set_alphatestref(val);

論証タイプ 商品説明
val Sets the reference value for alpha testing from 0 to 255 (default value is 0)

リターンです。

例

if !gpu_get_alphatestenable()
{
    gpu_set_alphatestenable(true);
    gpu_set_alphatestref(128);
}

上記のコードは、アルファテストが有効かどうかを確認し、有効でない場合は、switch アルファテストをオンにし、テストの閾値を128に設定します（アルファ値が0.5を超えるピクセルだけが描画されます）。
