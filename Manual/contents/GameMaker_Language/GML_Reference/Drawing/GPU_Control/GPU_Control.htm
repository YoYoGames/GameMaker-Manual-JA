GPU制御

room で描画する場合、描画をどのように行うかをより細かく制御することが必要になることがよくあります。アルファブレンディングは不要、カリングは必要、switch texture 補間のオン/オフは必要、などです。これらはすべて可能であり、さらに多くのことが可能です。- 以下の GPU コントローラ関数を使えば、これらすべてが可能です。これらの関数のほとんどはグローバルなもので、一度設定または変更すると、それ以 降のすべての描画に影響を与えることに注意してください。

以下の関数を使用して、さまざまなGPUプロパティの値を取得することができます。

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_destalpha
  • gpu_get_colourwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphatestref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

  • gpu_set_blendenable

  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colourwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphatestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

  • Blendmodesの使い方ガイド

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
