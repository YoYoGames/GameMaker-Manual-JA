gpu_set_zfunc

この関数は、z-bufferテストの比較モードを設定するために使用することができます ( gpu_set_ztestenable()を参照してください)。使用可能な値は、以下の定数のいずれかです（デフォルト値は cmpfunc_lessequal ）。

定数 商品説明
cmpfunc_never 決して
cmpfunc_less 少ない
cmpfunc_equal イコール
cmpfunc_lessequal 以下、等しい
cmpfunc_greater より大きい
cmpfunc_notequal ノットイコール
cmpfunc_greaterequal 大きいか等しい
cmpfunc_always 常に

構文です。

gpu_set_zfunc(cmp_func);

論証 タイプ 商品説明
cmp_func 比較機能定数 The comparison mode to use (see list above)

リターンです。

N/A

例

gpu_set_ztestenable(true);
gpu_set_zfunc(cmpfunc_always);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

上記のコードは、背景sprite を描画する前に z-buffer テストをオンにし、その比較モードを設定し、その後、描画を続けるために再びオフにします。
