gpu_get_zwriteenable

この関数は、z-writing が有効か（この関数は true を返します）、無効か（この関数は false を返します）を取得するために使用することができます。デフォルト値はz-writingが有効であるため、この関数は true を返します。

構文です。

gpu_get_zwriteenable();

リターンです。

例

if gpu_get_zwriteenable() == false
{
    gpu_set_zwriteenable(true);
}

上記のコードでは、z-writingが有効かどうかをチェックし、無効になっている場合は再度有効にしています。
