gpu_set_cullmode

この関数は、バックフェイスカリングモードを設定するために使用することができます。ポリゴン（プリミティブ）には、表側と 裏側がある。表側とは、反時計回りの順に頂点が定義されている側と言われています。通常は両面を描きますが、ピラミッドのような閉じた図形を作ると、三角形の裏側が見えなくなるので、処理能力の無駄遣いになります。このような場合、switch バックフェイスカリングを使用すると、描画時間を約半分に短縮できます。ただし、問題を防ぐために、ポリゴンを正しい方法で定義する作業が必要になります。

使用するカリングモード値は、以下の定数のいずれかである必要があります（デフォルト値は cull_noculling ）。

定数 商品説明
cull_noculling 淘汰されることはない
cull_clockwise 時計回りの三角形はすべて淘汰される
cull_counterclockwise 反時計回りの三角形はすべて淘汰される

構文です。

gpu_set_cullmode(cullmode);

論証 タイプ 商品説明
cullmode カリングモード一定 The cull mode to use (see list above)

リターンです。

N/A

例

if gpu_get_cullmode() != cull_clockwise
{
    gpu_set_cullmode(cull_clockwise);
}

上記のコードは現在のカリングモードを取得し、 cull_clockwise でない場合はその定数に設定されます。
