gpu_set_texrepeat_ext

GameMakerでShaders を使用する際に、1つのサンプラー「スロット」が与えられたtexture を繰り返すかどうかを設定することができます。 true に設定すると、uv座標が0-1の範囲外であれば、texture を繰り返します。 false に設定すると、繰り返さないという意味になります。これらの関数の使用例としては、3Dでtexture を繰り返すことが考えられますが、texture ページの他の部分から画像を引っ張ってこないようにするためには、使用するsprite をTexture Editor で "SeparateSprite Page" としてマークしておく必要があります。

注意 この設定は、関数を呼び出す際に設定した値によって上書きされます。 gpu_set_texrepeat().

構文です。

gpu_set_texrepeat_ext(sampler_id, enable);

論証タイプ 商品説明
sampler_id The sampler id from the shader.
enable Enable or disable texture filtering (true / ) false

リターンです。

例

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if (gpu_get_texrepeat_ext(s_tex))
{
    gpu_set_texrepeat_ext(s_tex, false);
}
else
{
    gpu_set_texrepeat_ext(s_tex, true);
}

上記のコードは、特定のサンプラーID（ローカル変数に格納）に対して、texture フィルタリングがオンかオフかを確認し、それに応じて切り替えています。
