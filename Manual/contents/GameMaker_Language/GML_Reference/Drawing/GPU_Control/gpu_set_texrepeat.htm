gpu_set_texrepeat

この関数は、モデルやプリミティブに使用する際に、textures を繰り返すか (true) しないか (false) をGameMakerに指示するために使用できます。通常、texture の幅と高さは 0 - 1 の範囲にありますが、デフォルトで 1 より大きい値を指定すると、残りは描画されません (texture 全体で、1 を超える割合に相当する「空の」部分が描画されます)。つまり、texture の幅や高さに 2 を指定すると、texture の半分のスペースに描画され、残りの半分は空白になります）。しかし、この関数を使って repeat を true に設定すると、texture はすべてのモデルやプリミティブで自動的に繰り返されます。これらの関数の使用例としては、texture を 3D で繰り返すことが考えられますが、texture ページの残りの部分から画像を引っ張ってこないようにするためには、sprite を使用する場合、Texture Editor で「SeparateSprite Page」にマークする必要があります。

注意:Sprites と背景は常に繰り返し描画されないので、一度sprite または背景を描画すると、この値は false にリセットされます。 このデフォルト動作はtexture のステージ設定には影響しません（つまり、関数を呼ぶと影響しますが、内部設定を false にすると影響しません）。

注：この設定は、機能使用時にtexture に設定されたshaders のステージリピートに優先します。 gpu_set_texrepeat_ext().

構文です。

gpu_set_texrepeat(enable);

論証タイプ 商品説明
enable Enable or disable texture filtering (true / ) false

リターンです。

例

if gpu_get_texrepeat()
{
    gpu_set_texrepeat(false);
}
else
{
    gpu_set_texrepeat(true);
}

上記のコードでは、texture のリピートがオンかオフかを確認し、それに応じて切り替えています。
