gpu_get_zfunc

この関数は、z比較モードを取得するために使用することができます。返される値は、以下の定数のいずれかです（デフォルト値は cmpfunc_lessequal ）。

比較機能定数
定数 商品説明
cmpfunc_never 決して
cmpfunc_less 少ない
cmpfunc_equal イコール
cmpfunc_lessequal 以下、等しい
cmpfunc_greater より大きい
cmpfunc_notequal ノットイコール
cmpfunc_greaterequal 大きいか等しい
cmpfunc_always 常に

構文です。

gpu_get_zfunc();

リターンです。

Comparison Function Constant (see table above)

例

if gpu_get_zfunc() != cmpfunc_greater
{
    gpu_set_zfunc(cmpfunc_greater);
}

上記のコードでは、z-testingメソッドが cmpfunc_greater に設定されているかどうかを確認し、設定されていなければその定数を設定します。
