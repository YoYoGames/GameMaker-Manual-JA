gpu_set_blendenable

この関数は、アルファブレンディングのオン／オフを切り替えるために使用します。基本的に、これを false に設定すると、描画されるすべての画像は100%不透明に描画されます。つまり、sprite や背景の透明または半透明の部分は はベタ色で表示されます。アルファブレンディングをオフにすると、描画速度が大幅に向上するため、可能な限りオフにすることをお勧めします（特にモバイルデバイス向けに開発する場合）。

構文です。

gpu_set_blendenable(enable);

論証タイプ 商品説明
enable Enable or disable alpha blending value (true または )。 false

リターンです。

例

gpu_set_blendenable(false);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_blendenable(true);

上記のコードでは、アルファブレンディングをオフにして背景sprite を描画し、再びオンにして描画を継続します。
