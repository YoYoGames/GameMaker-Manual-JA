gpu_get_blendmode_src

この関数は、描画に使用されている現在のソース拡張ブレンドモード係数を取得するために使用することができます。返される値は、以下の定数のいずれかです（"s "はソースから取得した値、"d "はデスティネーションから取得した値を表します）。

定数 ブレンド係数（赤、緑、青、アルファ）
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs、1-Gs、1-Bs、1-As)
bm_src_alpha (アス、アス、アス、アス)
bm_inv_src_alpha （1-As、1-As、1-As、1-As）
bm_dest_alpha （アド、アド、アド、アド）
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) ここで、f = min(As, 1-Ad)

構文です。

gpu_get_blendmode_src();

リターンです。

Blend Mode Factor Constant (see above table)

例

var bm;
bm[0] = gpu_get_blendmode_src();
bm[1] = gpu_get_blendmode_dest();
gpu_set_blendmode_ext_sepalpha(bm[0], bm[1], bm_inv_src_alpha, bm_inv_dest_colour);

上記のコードでは、ローカル配列を作成し、現在のソースとデスティネーションのブレンディング係数を取得します。この配列を使って、ブレンディング係数のアルファ成分を操作します。
