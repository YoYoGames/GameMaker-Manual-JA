vertex_freeze

この関数はvertex buffer を「フリーズ」させることができます。このbuffer は読み取り専用になるため、変更が必要な場合はbuffer をすべて削除し、再作成する必要があります。フリーズしたbuffer は、通常のダイナミックなshader よりも速くbuffer に送信でき、レベルまたはゲームの期間中、不変のカスタムプリミティブを必要とするエフェクトに推奨されます。

指定されたvertex buffer のフリーズに成功した場合は 0 を返し，何らかの理由で失敗した場合は -1 を返します．

構文です。

vertex_freeze(buffer);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The vertex buffer to freeze.

リターンです。

Real (0 for success, -1 for failure)

例

// Create event
vertex_freeze(vbuff);

上記のコードは、Createイベントで指定されたvertex buffer をフリーズさせるので、他のイベントで「静的」buffer として使用すると、より速くアクセスできるようになります。
