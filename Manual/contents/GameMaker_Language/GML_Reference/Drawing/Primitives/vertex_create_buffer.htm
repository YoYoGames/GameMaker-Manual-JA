バーテックス作成バッファ

この関数を使用すると、新しいvertex buffer を作成することができます。これはGameMakerが作成する特別なgrow buffer で、プリミティブを作るときに使うためにあらかじめフォーマットされています（たとえばshaders で使うため）。この関数はbuffer のインデックスを返し、それ以降の呼び出しで使用する必要があります。

この関数で作成されたvertex buffer を使用する場合、単に vertex_begin()vertex を使用して、buffer のデータの割り当てを開始し、カスタムプリミティブの定義を開始します。その後、shader に保持され、buffer に送信できるようになります。 vertex_freeze()関数)を呼び出すたびに vertex_begin()は、前のbuffer のデータを消去し、新しいデータを受け入れる準備をします。

構文です。

vertex_create_buffer();

リターンです。

Vertex Buffer ID

例

v_buff = vertex_create_buffer();

上記のコードでは、新規にvertex buffer を作成し、そのハンドルを変数 "v_buff" に格納します。
