vertex_ubyte4

この関数は、vertex のデータに 4 つの符号なしバイト値（0 ～ 255）を追加します。を使用してこれを受け入れるには、vertex が正しくフォーマットされている必要があります。 vertex_format_add_custom()関数を使用します。

構文です。

vertex_ubyte4(buffer, byte, byte, byte, byte);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
byte リアル The first input value.
byte リアル The second input value.
byte リアル The third input value.
byte リアル The fourth input value.

リターンです。

N/A

例

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

上記のコードでは、定義されているvertex のデータに4つの値が追加されます。
