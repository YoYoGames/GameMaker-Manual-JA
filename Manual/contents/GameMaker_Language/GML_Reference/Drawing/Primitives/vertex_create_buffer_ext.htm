vertex_create_buffer_ext

関数と同様に vertex_create_buffer()この関数は、新しいvertex buffer を作成します。これはGameMakerが作成した特別なもので、buffer で使用するプリミティブを構築する際に使用するためにあらかじめフォーマットされたshaders を育てます。buffer の初期開始サイズ（バイト単位）を指定すると、buffer のインデックスが返され、それ以降buffer を呼び出す際に使用されるべきものです。

構文です。

vertex_create_buffer_ext(size);

論証タイプ 商品説明
size The initial size of the buffer (in bytes).

リターンです。

Vertex Buffer ID

例

v_buff = vertex_create_buffer_ext(1024 * 1024);

上記のコードは、新しいvertex buffer 、最初は1MBのサイズを作成し、そのハンドルを変数 "v_buff "に格納します。
