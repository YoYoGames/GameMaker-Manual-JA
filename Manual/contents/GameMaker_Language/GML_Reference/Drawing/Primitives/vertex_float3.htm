vertex_float3

この関数は、vertex データに 3 つの浮動小数点値を追加します。を使用してこれを受け入れるには、vertex が正しくフォーマットされている必要があります。 vertex_format_add_custom()関数を使用します。

構文です。

vertex_float3(buffer, float, float, float);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
float リアル The first input value.
float リアル The second input value.
float リアル The third input value.

リターンです。

N/A

例

vertex_float3(buff, 0.05, 0.01, room_width / x);

上記のコードは、定義されているvertex データに 3 つの浮動小数点値を追加するものです。
