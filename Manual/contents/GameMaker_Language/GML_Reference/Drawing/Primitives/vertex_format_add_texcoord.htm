vertex_format_add_texcoord

新しく作成されるtexture フォーマットの一部として、vertex 位置データ (u と v) を受け入れるようにGameMaker に指示します。

構文です。

vertex_format_add_texcoord();

リターンです。

例

vertex_format_begin();
vertex_format_add_texcoord();
vertex_format_add_position_3d();
my_format = vertex_format_end();

上記のコードは、vertex と position の値だけで新しいtexture フォーマットを作成し、フォーマット ID を変数 "my_format" に格納します。
