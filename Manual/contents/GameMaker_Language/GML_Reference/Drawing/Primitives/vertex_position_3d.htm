頂点位置3d

この関数は、カスタムプリミティブ用に現在定義されているvertex に3D位置データを追加します。データを書き込むbuffer と、描画のための x, y, z 位置を指定します。

構文です。

vertex_position_3d(buffer, x, y, z);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
x リアル The x position.
y リアル The y position.
z リアル The z position.

リターンです。

N/A

例

vertex_position_3d(buff, x - 100, y - 125, 0);

上記のコードで、定義されているvertex の位置が設定されます。
