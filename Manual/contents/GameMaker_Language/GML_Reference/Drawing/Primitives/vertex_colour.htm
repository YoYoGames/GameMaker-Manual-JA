バーテックスカラー

定義されているvertex フォーマットが色の値をとる場合、この関数を使用して、現在のプリミティブに定義されているvertex にそのデータを追加することができます。この関数は、データを格納するためのbuffer を必要とし、色定数か、16進数値 (GameMaker標準の BGR フォーマットで、たとえば青は $FF0000) と、0 (透明) から 1 (完全不透明) のアルファ値も受け取ります。

構文です。

vertex_colour(buffer, colour, alpha);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
colour カラー The colour for this vertex (can be a constant or a hex value).
alpha リアル The alpha value for the vertex (from 0 to 1).

リターンです。

N/A

例

vertex_colour(b, c_white, 1);

上記のコードでは、定義されているvertex をアルファ値 1 の白に設定します。
