draw_vertex_texture

この関数は、プリミティブのテクスチャ（vertex ）の位置を定義する。プリミティブの最終的な外観は、描画に選択されたプリミティブの種類によって異なります ( draw_primitive_begin()textureを呼び出す必要があります。プリミティブを終了して描画するためには draw_primitive_end().

また、サンプリングする適切な xtex/ytex 位置（これらの座標はUVポイントとも呼ばれます）を設定することで、プリミティブの表面上にtexture の一部だけを描画することも可能です。この点は0と1の間の値で定義され、使用するイメージの左上隅が(0,0)点、右下隅が(1,1)点となります。このようにして、元画像の一部だけを使ったプリミティブをtexture にすることができますasset 。

テクスチャ付きプリミティブを描画する場合、texture はdraw_set_colour で設定された色とブレンドされます。 つまり、ブレンドなしでtexture を描画するには、以下の例に示すように、これを c_white に設定する必要があります。

注意事項texture を繰り返すには、32x32、128x128 など、2のべき乗のサイズである必要があります。

構文です。

draw_vertex_texture(x, y, xtex, ytex)

論証 タイプ 商品説明
x リアル The x coordinate of the vertex.
y リアル The y coordinate of the vertex.
xtex リアル The width of the texture sample to use.
ytex リアル The height of the texture sample to use.

リターンです。

N/A

例

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上記のコードでは、変数 "tex" に保持されたvertex でテクスチャを貼った 4texture の三角形の帯（長方形を作る）を描き、そのtexture 全体で完成したプリミティブを覆うことになります。
