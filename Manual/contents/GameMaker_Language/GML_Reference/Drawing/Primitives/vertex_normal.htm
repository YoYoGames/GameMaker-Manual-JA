頂点法線

この関数は、カスタムプリミティブ用に現在定義されているvertex にサーフェス法線データを追加します。データを書き込むbuffer と、法線の x, y, z 成分部分を指定します。

構文です。

vertex_normal(buffer, nx, ny, nz);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
nx リアル The x component of the surface normal.
ny リアル The y component of the surface normal.
nz リアル The z component of the surface normal.

リターンです。

N/A

例

vertex_normal(buff, 0, 1, 1);

上記のコードでは、定義されているvertex の表面法線が設定されます。
