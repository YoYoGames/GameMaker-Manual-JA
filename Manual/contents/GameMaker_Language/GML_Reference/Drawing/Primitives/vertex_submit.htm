vertex_submit

この関数は、vertex buffer の内容をグラフィックスパイプラインに送信し、shader で使用するために使用します。使用するbuffer のインデックス、使用するベースプリミティブタイプ（以下の定数を参照）、使用するtexture を指定します。ベースプリミティブタイプは、buffer に保存した頂点の描画と接続の順序を割り当てるためにのみ使用され、各頂点に実際に使用されるデータはvertex buffer を作成する際に定義したものになります。

注：この機能は、描画イベントでのみ使用できます。

さまざまなベースプリミティブを利用できる視覚的な例として、以下の画像をご覧ください。

定数 商品説明
pr_pointlist 点のリストで構成されるプリミティブ。
pr_linelist リスト内の個々の行から構成されるプリミティブ。
pr_linestrip 連続した帯状の線からなるプリミティブ。
pr_trianglelist 個々の三角形をリスト化したプリミティブ。
pr_trianglestrip 連続した帯状の三角形で構成されるプリミティブ。

構文です。

vertex_submit(buffer, primitive, texture);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to use.
primitive プリミティブ型定数 The primitive base type.
texture テクスチャポインタ The texture to use (or -1 for no texture).

リターンです。

N/A

例

shader_set(shader_prim);
vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index));
shader_reset();

上記のコードは、変数 "buff" でインデックスされたvertex buffer を、shader "shader_prim" で描画するために投入します。ベースプリミティブとして三角リストを使い、コードを実行しているインスタンスのtexture のsprite を使用します。
