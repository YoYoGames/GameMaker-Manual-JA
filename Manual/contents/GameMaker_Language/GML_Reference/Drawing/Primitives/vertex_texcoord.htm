頂点頂点座標

この関数は、現在カスタムプリミティブに定義されているtexture に使用するvertex 座標を設定します。データを書き込むbuffer と、使用するtexture 内の UV 位置を指定します。

構文です。

vertex_texcoord(buffer, u, v);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
u リアル The first texture coordinate to use (0 - 1).
v リアル The second texture coordinate to use (0 - 1).

リターンです。

N/A

例

vertex_texcoord(buff, 0, 0);

上記のコードでは、定義中のvertex の UV 値を設定します。
