頂点端

この関数で、カスタムプリミティブのビルドを終了します。このコマンドを呼び出すと、そのプリミティブは vertex_submit()関数を使用してshader で使用するか、またはbuffer をフリーズさせる（使用するvertex buffer を読み取り専用にし、より高速にする）ことができます。 vertex_freeze().

構文です。

vertex_end(buffer);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to end writing to.

vertex_end(buff);
