draw_primitive_begin

この関数は、プリミティブを定義する前に呼び出す必要があります。プリミティブは6種類あり、以下の定数で定義することができます。

プリミティブ型定数
定数 商品説明
pr_pointlist ポイント一覧
pr_linelist 行のリスト
pr_linestrip ラインストリップ
pr_trianglelist 三角形のリスト
pr_trianglestrip 三角形の帯
pr_trianglefan 三角形のファン

以下の画像は、基本的にこれらがどのように見えるか、また、頂点を定義する順序を示したものです。

一部のプラットフォーム（Windows, UWP, XBox）では、 pr_trianglefan のタイプがネイティブにサポートされていないため、GameMakerはゲームをコンパイルする際に変換を行って動作させていることに注意してください。このため、これらのプラットフォームでは、 pr_trianglefan 型を使用すると、他の型よりはるかに遅くなることを意味します。また、HTML5でこの機能を使うには、Game OptionsでWebGLを有効にする必要があることに注意してください。

構文です。

draw_primitive_begin(kind)

論証 タイプ 商品説明
kind プリミティブ型定数 The kind of primitive you are going to draw.

リターンです。

N/A

例

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
    draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

上記のコードでは、プリミティブで作られた円の4分の3を描画します。
