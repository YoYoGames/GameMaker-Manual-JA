vertex_get_buffer_size

この関数を使うと、与えられたvertex buffer のサイズをバイト単位で得ることができます。

構文です。

vertex_get_buffer_size(buffer);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to get the size of.

リターンです。

Real

例

bufferBytes = vertex_get_buffer_size(buff);

上記のコードは、与えられたvertex buffer が使用したバイト数を取得し、その値を変数 "bufferBytes" に格納します。
