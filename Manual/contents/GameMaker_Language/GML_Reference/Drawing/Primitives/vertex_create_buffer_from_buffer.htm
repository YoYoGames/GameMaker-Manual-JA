vertex_create_buffer_from_buffer

関数と同様に vertex_create_buffer()この関数は、新しいvertex buffer を作成します。ただし、格納するvertex データは、ソースとして指定されている通常のbuffer からコピーされます。作成されるbuffer は，buffer で使用するプリミティブを構築するためのフォーマッ トであらかじめフォーマットされた，特別なgrow vertex です（例：shaders ）．この関数は、以前に作成された通常のbuffer へのポインタと、コピーされたデータに適用されるべきvertex フォーマットを指定する必要があります。

構文です。

vertex_create_buffer_from_buffer(buffer, format);

論証 タイプ 商品説明
buffer バッファID The buffer to create the vertex buffer from.
format プリミティブ型定数 The primitive vertex format to use.

リターンです。

Vertex Buffer ID

例

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat);

上記のコードでは、新しいvertex フォーマットを作成し、以前に作成した通常のvertex から新しいbuffer buffer を作成し、カスタムvertex フォーマットを適用します。
