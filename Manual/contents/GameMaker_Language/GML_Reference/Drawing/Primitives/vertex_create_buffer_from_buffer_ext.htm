バッファからバッファを作成します。

関数と同様に vertex_create_buffer()この関数は、新しいvertex buffer を作成します。ただし、保存するvertex データは、ソースとして指定された通常のbuffer からコピーされます。buffer は、(たとえば)vertex で使用するプリミティブを構築するためのshaders フォーマットであらかじめフォーマットされています。また、コピー元のbuffer 内のオフセットと、最終的にbuffer が持つべき頂点の数を指定することも可能です。もし頂点の数がコピーされるものと一致しない場合、vertex のデータが破損する可能性があることに注意してください。

構文です。

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);

論証 タイプ 商品説明
buffer バッファID The buffer to create the vertex buffer from.
format プリミティブ型定数 The primitive vertex format to use.
src_offset リアル The offset within the the source buffer to copy from.
vert_num リアル The number of vertices the buffer should have.

リターンです。

Vertex Buffer ID

例

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

上記のコードは、新しいvertex フォーマットを作成し、以前に作成した通常のvertex から新しいbuffer buffer を作成し、それにカスタムvertex フォーマットをオフセット 0 で適用します。この関数は、新しいvertex buffer に、512 個の頂点を期待することを伝えます。
