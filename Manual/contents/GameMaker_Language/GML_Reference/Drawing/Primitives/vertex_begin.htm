頂点開始

この関数でカスタムプリミティブの定義を開始します。プリミティブを書き込むbuffer と、使用するvertex フォーマット (vertex フォーマット関数を使用して事前に定義) を割り当てます。次に、プリミティブの各vertex に必要な点を定義してから、次の関数を呼び出します。 vertex_end()を実行して、プリミティブの作成を終了します。

構文です。

vertex_begin(buffer, format);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to be written to.
format バーテックスフォーマットID The vertex format to use.

リターンです。

N/A

例

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
vertex_format_add_textcoord();
v_format = vertex_format_end();
v_buff = vertex_create_buffer();
vertex_begin(v_buff, v_format);

上記のコードでは、新しいvertex フォーマットを定義し、新しいbuffer を作成し、新しいプリミティブの定義処理を開始します。
