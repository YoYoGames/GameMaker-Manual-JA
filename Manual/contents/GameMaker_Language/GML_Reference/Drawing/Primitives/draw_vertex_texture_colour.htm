ドローバーテックステクスチャカラー

この関数は、プリミティブのテクスチャ（vertex ）の位置を定義する。プリミティブの最終的な外観は、描画に選択されたプリミティブの種類に依存します ( draw_primitive_begin()をご覧ください。） 頂点を追加する順序、texture サンプルに与える始点と終点の位置、設定した色とアルファ値。texture の外観を維持しつつ、アルファ値のみを変更する場合は color 引数に -1 (または c_white) を使用することができます。プリミティブを終了して描画するためには draw_primitive_end().

注意事項texture を繰り返すには、32x32、128x128 など、2のべき乗のサイズである必要があります。

構文です。

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

論証タイプ 商品説明
x The x coordinate of the vertex.
y The y coordinate of the vertex.
xtex The x coordinate within the texture.
ytex The y coordinate within the texture.
col The colour to blend with the texture at this vertex (-1 or c_white はブレンドしない場合）。
alpha The alpha to draw this vertex with (0-1).

リターンです。

例

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

上記のコードでは、変数 "tex" に保持されたvertex でテクスチャを貼った 4texture の三角形の帯（長方形を作る）を描き、texture 全体で完成したプリミティブを覆って、4種類の色で混色します。
