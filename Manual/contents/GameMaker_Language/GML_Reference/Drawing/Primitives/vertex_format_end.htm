vertex_format_end

この関数は、新しいvertex フォーマットを定義した後に呼び出す必要があります。この関数は新しいフォーマットの「ハンドル」(インデックス)を返します。このハンドルは、この新しいフォーマットを参照する以降のすべてのvertex 関数で使用されなければなりません。

構文です。

vertex_format_end();

リターンです。

Vertex Format ID

例

vertex_format_begin();
vertex_format_add_position();
vertex_format_add_colour();
my_format = vertex_format_end();

上記のコードでは、色と位置の値だけで新しいvertex フォーマットを作成し、フォーマット ID を変数 "my_format" に格納します。
