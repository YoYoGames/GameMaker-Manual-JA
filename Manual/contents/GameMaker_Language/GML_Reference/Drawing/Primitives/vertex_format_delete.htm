vertex_format_delete

この関数は、作成されたベクターフォーマットの使用が終了するたびに呼び出す必要があります。フォーマットIDの値（関数 vector_format_end()この関数は，そのメモリに関連するメモリを解放します。この関数を呼び出した後に、このフォーマットを再び使おうとすると、エラーが発生することに注意してください。

構文です。

vertex_format_delete(formatID);

論証 タイプ 商品説明
formatID バーテックスフォーマットID The vertex format to delete

リターンです。

N/A

例

vertex_format_delete(myFormat);

上記のコードは、変数 "myFormat "に作成されたvertex フォーマットをメモリから削除します。
