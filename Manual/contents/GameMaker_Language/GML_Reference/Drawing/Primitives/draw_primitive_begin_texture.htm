draw_primitive_begin_texture

この関数は、テクスチャ付きプリミティブの頂点を定義する前に呼び出す必要があります。使用するプリミティブの種類を指定する必要があります ( draw_primitive_begin()texture sprite または背景画像 を使用することができます。このasset asset idは関数から得ることができます。 sprite_get_texture()を使用します (texture がない場合は -1 を使用します)。

注意事項texture を繰り返すには、32x32、128x128 など、2のべき乗のサイズである必要があります。

構文です。

draw_primitive_begin_texture(kind, tex)

論証 タイプ 商品説明
kind プリミティブ型定数 The kind of primitive you are going to draw.
tex テクスチャポインタ The texture to use with the primitive.

リターンです。

N/A

例

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

上記のコードでは、変数 "tex" に保持されたvertex でテクスチャを貼った 4texture の三角形の帯（長方形を作る）を描き、そのtexture 全体で完成したプリミティブを覆うことになります。
