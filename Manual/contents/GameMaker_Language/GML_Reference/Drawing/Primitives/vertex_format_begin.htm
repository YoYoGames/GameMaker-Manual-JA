vertex_format_begin

新しいvertex フォーマットを定義する前に、この関数を使用してGameMakerにその旨を伝える必要があります。まずこの関数を呼び出し、次に適切な関数を使ってフォーマットの値を定義し、最後に vertex_format_end() を実行して定義を終了し、新しいフォーマットの「ハンドル」を返します。

構文です。

vertex_format_begin();

リターンです。

例

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上記のコードでは、色と位置の値だけで新しいvertex フォーマットを作成し、フォーマット ID を変数 "my_format" に格納します。
