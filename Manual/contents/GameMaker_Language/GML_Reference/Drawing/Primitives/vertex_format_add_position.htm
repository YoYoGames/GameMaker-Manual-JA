vertex_format_add_position

新しく作成されるvertex フォーマットの一部として、2D 位置情報 (x と y) を受け入れるようにGameMaker に指示します。

構文です。

vertex_format_add_position();

リターンです。

例

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

上記のコードでは、色と位置の値だけで新しいvertex フォーマットを作成し、フォーマット ID を変数 "my_format" に格納します。
