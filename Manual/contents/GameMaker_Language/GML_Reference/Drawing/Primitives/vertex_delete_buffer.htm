vertex_delete_buffer

この関数は、以前に作成したvertex buffer ( vertex_create_buffer()) をシステムメモリから削除します。また、その中に含まれていたカスタムプリミティブのデータ（vertex ）もすべて削除されます。

構文です。

vertex_delete_buffer(buffer);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The vertex buffer to be removed.

リターンです。

N/A

例

vertex_delete_buffer(v_buff);

上記のコードは、変数 "v_buff "にインデックスされたbuffer をメモリから削除します。
