vertex_float1

この関数は、vertex データに浮動小数点値を追加します。を使用してこれを受け入れるには、vertex が正しくフォーマットされている必要があります。 vertex_format_add_custom()関数を使用します。

構文です。

vertex_float1(buffer, float);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
float リアル The input value.

リターンです。

N/A

例

vertex_float1(buff, 0.05);

上記のコードは、定義されているvertex データに浮動小数点値を追加するものです。
