バーテックス アルゴブ

この関数は、現在カスタムプリミティブに定義されているvertex のARGB値を設定します。データを書き込むbuffer と、使用したい赤、緑、青、アルファ値を、32 ビットの符号なし整数で指定します。16進数表記（$AARRGGBB）、実数、または make_colour_*() 関数を使用して色値を定義することができます。

構文です。

vertex_argb(buffer, argb);

論証 タイプ 商品説明
buffer バーテックスBuffer ID The buffer to write the information to.
argb カラー The ARGB value to set.

リターンです。

N/A

例

vertex_argb(buff, $FFFFFFFF);

上記のコードでは、定義されているvertex の ARGB 値を白に設定します。
