draw_point

この関数は、現在設定されている描画色とアルファ値を使用して、画面上の任意の場所に1ピクセルを描画することができます。

構文です。

draw_point(x, y);

論証タイプ 商品説明
x The x coordinate of the point to be drawn.
y The y coordinate of the point to be drawn.

リターンです。

例

draw_set_colour(c_yellow);
draw_point(100,100);

これにより、位置(100,100)に黄色の画素が描画されます。
