draw_line_width

この関数を使うと、ゲーム内の任意の2点間に指定した幅の線を引くことができます。room.描画される線は、各社の違いにより、希望する寸法で描画されるために異なる値（x、yに+/-1）が必要な場合があることに注意してください。 サポートされているさまざまなプラットフォームで

注：shader を使って図形を描く場合、ほとんどのshaders は次のような入力を期待していることに注意する必要があります。vertex texture, 色.しかし、この関数を使う場合、vertex と色のデータだけが渡されます。 shader は何も描画しない（あるいは、描画はするが正しく表示されない）かもしれません。このように図形を描画する必要がある場合は、shader をこの点を考慮してカスタマイズする必要があります。

構文です。

draw_line_width(x1, y1, x2, y2, w);

論証タイプ 商品説明
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width of the line in pixels.

リターンです。

例

draw_set_colour(c_red);
draw_line_width(100, 100, 200, 200, 6);

点(100,100)から点(200,200)まで、6ピクセル幅の赤い対角線を描きます。
