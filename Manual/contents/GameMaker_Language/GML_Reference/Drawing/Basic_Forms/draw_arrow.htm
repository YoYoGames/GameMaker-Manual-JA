draw_arrow

この関数は、点 (x1,y1) から点 (x2,y2) まで矢印を描画する。矢印の幹はこれらの点に沿って描かれ、実際の矢じりは最後に描かれる。矢じりの大きさは引数 "size" で定義され は、終点が x2,y2 で定義される位置に常に一致するように、茎の一部として計算される。矢印の頭の幅は、長さに比例して自動的に計算される。

注：shader を使って図形を描く場合、ほとんどのshaders は次のような入力を期待していることに注意する必要があります。vertex texture, 色.しかし、この関数を使う場合、vertex と色のデータだけが渡されます。 shader は何も描画しない（あるいは、描画はするが正しく表示されない）かもしれません。このように図形を描画する必要がある場合は、shader をこの点を考慮してカスタマイズする必要があります。

構文です。

draw_arrow(x1, y1, x2, y2, size);

論証タイプ 商品説明
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line (where the arrowhead ends).
y2 The y coordinate of the end of the line (where the arrowhead ends).
size The length of the arrow in pixels.

リターンです。

例

draw_arrow(x, y, mouse_x, mouse_y, 10);

上記のコードでは、インスタンスの位置からマウスの位置まで、矢印の頭の長さが10ピクセルの矢印を描きます。
