draw_line_width_colour

この機能を使うと、1点目の色1と2点目の色2をブレンドした色で、指定幅の線を描くことができます。色設定は、機能で設定した基本色よりも優先されます。 draw_set_colour(). なお、サポートするプラットフォームの違いにより、描画する線に異なる値（x,yに±1）が必要な場合がありますので、ご注意ください。

注：shader を使って図形を描く場合、ほとんどのshaders は次のような入力を期待していることに注意する必要があります。vertex texture, 色.しかし、この関数を使う場合、vertex と色のデータだけが渡されます。 shader は何も描画しない（あるいは、描画はするが正しく表示されない）かもしれません。このように図形を描画する必要がある場合は、shader をこの点を考慮してカスタマイズする必要があります。

構文です。

draw_line_width_colour(x1, y1, x2, y2, w, col1, col2);

論証タイプ 商品説明
x1 The x coordinate of the start of the line.
y1 The y coordinate of the start of the line.
x2 The x coordinate of the end of the line.
y2 The y coordinate of the end of the line.
w The width in pixels of the line.
col1 The colour of the start of the line.
col2 The colour of the end of the line.

リターンです。

例

draw_line_width_colour(50, 50, 300, 50, 4, c_red, c_blue);

この場合、(50,50)から(300,50)まで4ピクセル幅の水平線を描き、赤から青へのグラデーションがなめらかになるようにします。
