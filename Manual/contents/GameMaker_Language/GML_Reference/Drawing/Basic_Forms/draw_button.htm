draw_button

この関数は、現在選択されている描画色とアルファ値を用いて、非常に単純な長方形の「ボタン」を描画します。ここで、up引数は、下の画像に示すように、ベベルエッジ効果の見え方を定義するものです。

注：shader を使って図形を描く場合、ほとんどのshaders は次のような入力を期待していることに注意する必要があります。vertex texture, 色.しかし、この関数を使用する場合、vertex とカラーデータのみが渡されるため、shader は何も描画しない（または、何か描画するが正しく描画しない）可能性があります。このように図形を描画する必要がある場合は、shader をこの点を考慮してカスタマイズする必要があります。

構文です。

draw_button(x1, y1, x2, y2, up);

論証 タイプ 商品説明
x1 リアル The x coordinate of the left of the button
y1 リアル The y coordinate of the top of the button
x2 リアル The x coordinate of the right of the button
y2 リアル The y coordinate of the bottom of the button
up ブーリアン Whether the button is up (true) または下 (false)

リターンです。

N/A

例

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

これは、マウスの左ボタンを押したままだと押されたように見えるボタンを描画します。
