draw_point_colour

この機能を使うと、画面上の任意の場所に1ピクセルを定義した色で描画することができます。この色設定は、関数 draw_set_colour().

注：shader を使って図形を描く場合、ほとんどのshaders は次のような入力を期待していることに注意する必要があります。vertex texture, 色.しかし、この関数を使う場合、vertex と色のデータだけが渡されます。 shader は何も描画しない（あるいは、描画はするが正しく表示されない）かもしれません。このように図形を描画する必要がある場合は、shader をこの点を考慮してカスタマイズする必要があります。

構文です。

draw_point_colour(x, y, col1);

論証タイプ 商品説明
x The x coordinate of the point.
y The y coordinate of the point.
col1 The colour of the point.

リターンです。

例

draw_point_colour(50, 50, c_red);

この場合、(50,50)に赤い画素が描画されます。
