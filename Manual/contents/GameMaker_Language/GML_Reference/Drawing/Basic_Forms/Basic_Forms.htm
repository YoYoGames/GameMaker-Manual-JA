基本フォーム

GameMakerには、様々な形状の描画に利用できる関数が揃っています。これらの関数はインスタンスの draw イベントでのみ使用し（他の場所でも使用できますが、何も表示されません！）、純粋にグラフィック効果を生成します。つまり、衝突イベントはインスタンスに割り当てられたsprite やコリジョンマスクを通してのみ発生します。これらの関数はすべて、現在の描画色、アルファ、ブレンドモードの影響を受けます（「色」のセクションを参照してください）。

重要!線、点、形は、描画されるイベントに関係なく、常に画面解像度と1:1で描画されます。

重要!これらの関数は、ゲーム内で頂点のバッチ処理を解除し、texture のスワップ数を増加させます。したがって、1 つの描画フレームでこれらの関数を複数回呼び出すと、パフォーマンスの問題につ ながる可能性があります。

基本的な図形を描画するために、以下の描画関数が存在する。これらの関数は呼び出しに時間がかかり、基本的な描画ツールとしてのみ提供されていることに注意。

  • draw_arrow
  • draw_circle
  • draw_circle_colour
  • draw_ellipse
  • draw_ellipse_colour
  • draw_line
  • draw_line_colour
  • draw_line_width
  • draw_line_width_colour
  • draw_point
  • draw_point_colour
  • draw_rectangle
  • draw_rectangle_colour
  • draw_roundrect
  • draw_roundrect_colour
  • draw_roundrect_ext
  • draw_roundrect_colour_ext
  • draw_triangle
  • draw_triangle_colour

また、GameMakerには以下のような関数があり、描画精度を設定したり、非常に特殊なものを描画したりすることができます。

  • draw_set_circle_precision
  • draw_button
  • draw_healthbar
  • draw_path
