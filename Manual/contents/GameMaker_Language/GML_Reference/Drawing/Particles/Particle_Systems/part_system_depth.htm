パート_システム深度

この機能では、パーティクルシステムの描画深度を設定できます。room 内の異なるレイヤーのレンダリング深度を設定できるのと同じです。描画深度が低いと、描画深度の高いものの上に表示され、描画深度が高いと、描画深度の低いものの下に表示されることになります。

構文です。

part_system_depth( ind, depth );

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to change.
depth リアル The depth at which to set the particle system.

リターンです。

N/A

例

global.Sname = part_system_create();
part_system_depth(global.Sname, -1000 );

上記のコードでは、パーティクルシステムを作成し、そのインデックスをグローバル変数「Sname」に格納しています。このシステムには、-1000の低い深度が与えられ、描画深度の高いすべてのものの上に表示されることを意味しています。
