パートシステム更新

この機能を使うと、パーティクルシステムに現在のパーティクルをすべて画面上に描画させることができます。もし part_system_automatic_draw()がオフの場合、この関数はインスタンスの draw イベントで使用されたときにパーティクルを表示し、描画ターゲットがサーフェスに設定されているときに使用されます (これは surface_set_target()) を使って、システム内のパーティクルをそのサーフェスに描画します。

構文です。

part_system_drawit(ind);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to draw the particles of.

リターンです。

N/A

例

if surface_exists(surf)
{
    surface_set_target(surf);
    part_system_drawit(global.psys);
    surface_reset_target();
}

上記のコードでは、変数 surf で指定されたサーフェイスが存在するかどうかを確認し、存在する場合は描画ターゲットをサーフェイスに設定し、グローバル変数に格納された ID でパーティクルシステムを描画し、描画ターゲットをリセットして画面に通常の描画が再び表示されるようにしています。
