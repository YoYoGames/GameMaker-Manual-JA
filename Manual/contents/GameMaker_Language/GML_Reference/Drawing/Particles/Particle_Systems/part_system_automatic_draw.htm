パートシステム自動描画

この機能を使うと、パーティクルシステムの描画をswitch オフにし、システムに対して行われた更新（自動的またはそれ以外）を見えなくすることができます。これは純粋に視覚的なオプションで、 false に設定すると、パーティクルは描画されないので見ることができませんが、パーティクルはまだ存在し、通常と同じように位置や色などを変化させています。自動描画がオフの場合、GameMakerにパーティクルシステムの現在の状態を描画するよう明示的に命令するには、関数 part_system_drawit()また、この関数を再度 true に設定すると、switch 自動描画をオンに戻すことができます。

注意点としては、関数で作成したシンプルなエフェクトを使う場合は effect_create_above()または effect_create_below()の場合、パーティクルシステムのインデックスとして 0（以下の効果）または 1（上記の効果）を使用し、これらの自動描画を切り替えることができます（これはGML Visual particle effect の描画を切り替えるためにも機能します）。

構文です。

part_system_automatic_draw(ind, automatic);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to change.
automatic ブーリアン Whether automatic drawing is on (true) or not (false).

リターンです。

N/A

例

part_system_automatic_draw(global.Sname, false);

上記のコードは、グローバル変数 "Sname" でインデックスされたパーティクルシステム の自動描画をオフにするものです。
