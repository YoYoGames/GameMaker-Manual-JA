パート_システム_描画順

この機能により、画面上にパーティクルを作成する際の描画方法を設定することができます。 false デフォルトでは、古いパーティクルが新しいパーティクルよりも高い深さで描かれ、新しいパーティクルの「下」に表示されます（この関数は true に設定されています）。以下の画像はこれを説明するもので、左の画像はデフォルト値の true で、右の画像は false です。

注意：加算ブレンドモードでパーティクルを描画している場合、この機能の効果は必ずしも明らかでない場合があります。

構文です。

part_system_draw_order(ind, oldtonew);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to change.
oldtonew ブーリアン Whether old particles should be drawn behind newer ones (true) or vice versa (false).

リターンです。

N/A

例

mysystem = part_system_create();
part_system_draw_order(mysystem, true);

これは、新しいパーティクルシステムを "mysystem" というインデックスで作成し、古いパーティクルの上に新しいパーティクルを描画するようにパーティクルを設定するものです。
