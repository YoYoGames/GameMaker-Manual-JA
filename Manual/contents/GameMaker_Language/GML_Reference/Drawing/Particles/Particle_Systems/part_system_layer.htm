パート_システム_レイヤー

この関数は、パーティクルシステムを現在のレイヤーから新しいレイヤーへswitch するために使用します。パーティクルシステムの ID 値を指定します（関数 part_system_create_layer())、そしてユニークなレイヤーID (関数 layer_create()または、string として使用するレイヤーの名前 - たとえば "instance_layer" -room エディターで定義されています）、システムは新しいレイヤーに移動されます。

構文です。

part_system_layer(ps, layer);

論証 タイプ 商品説明
ps パーティクルシステムID The particle system ID
layer 文字列またはレイヤID The new layer ID

リターンです。

N/A

例

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
    part_system_layer(global.p_sys, "effects_layer";
}

上記のコードは、パーティクルシステムがどのレイヤーにあるかをチェックし、与えられたレイヤーにない場合は変更されます。
