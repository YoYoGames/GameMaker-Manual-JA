パートパティクルクリア

この関数を使うと、システムによって現在作成されているすべてのパーティクルをroom から消去することができます。これはパーティクルの種類自体をリセットしたり削除するのではなく、その視覚的な表現を削除するだけです。もし、エミッタからobject ストリーミングパーティクルがあれば、これらのパーティクルは消えますが、このコードを呼んだ次のステップから再び出現し始めることになります。

構文です。

part_particles_clear(ind);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system.

リターンです。

N/A

例

if (lives <= 0)
{
    part_particles_clear(global.Sname);
    room_goto(rm_intro);
}

上記のコードは、変数「lives」の値をチェックし、それが0に等しい場合、システムからすべてのパーティクルをクリアし、room を変更します。
