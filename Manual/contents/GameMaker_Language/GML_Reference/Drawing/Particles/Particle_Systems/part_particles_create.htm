part_particles_create

この機能は、パーティクルエミッタの機能を必要としないエフェクト（例えば、ミサイルの煙や単純な爆発エフェクトなど）に最適で、ゲーム内の任意の位置に素早く簡単にパーティクルを作成することができます。room.この機能を使用するには、事前にパーティクル システムとパーティクル タイプを作成しておく必要があります。

構文です。

part_particles_create(ind, x, y, parttype, number);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system.
x リアル The x coordinate of where to create the particles.
y リアル The y coordinate of where to create the particles.
parttype 粒子タイプID The index (type) of the particles to be created.
number リアル The number of particles to create.

リターンです。

N/A

例

if mouse_check_button(mb_left)
{
    part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

上記のコードは、マウスボタンが押されたかどうかをチェックし、 true を返した場合、マウス位置に5個のパーティクルを生成します。
