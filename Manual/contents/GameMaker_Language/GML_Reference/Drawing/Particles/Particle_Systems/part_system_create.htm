part_system_create

この関数は新しいパーティクルシステムを作成するために使用され、そのシステムに関連する以降のすべての関数で保存され使用されるべき固有のインデックス番号を返します。システムは管理レイヤーを割り当てられ、深さ0に設定されます。管理レイヤーはユーザーからはアクセスできず、深さをレイヤーの代わりに使用する場合にのみ内部管理のために使用されます。通常は，関数 part_system_create_layer()の代わりに、これを使用します。

構文です。

part_system_create();

リターンです。

Particle System ID

例

global.Sname = part_system_create();

これにより、新しいパーティクルシステムが作成され、そのインデックスがグローバル変数 "Sname "に格納されます。
