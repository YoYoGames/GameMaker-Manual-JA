part_system_position

この関数では、パーティクルシステムの基本位置をroom の (0,0) 位置に対して設定することができます。デフォルトでは、この位置は常に (0,0) ですが、非常に特殊な状況では、これを他のものに変更したい場合があります。

注意：この関数はパーティクルシステム内のすべてを変更するので、エミッタの位置を(100,100)に設定し、パーティクルシステムの位置を(0,100)に設定すると、エミッタは(100,200)で描画されるようになります。また、システムを移動させてからエミッターを作成した場合も同様で、(100,100)で作成したエミッターは(100,200)で描画されます。

構文です。

part_system_position(ind, x, y);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to change.
x リアル The new x coordinate of the particle system.
y リアル The new y coordinate of the particle system.

リターンです。

N/A

例

if mouse_check_button_pressed(mb_left)
{
    part_system_position(global.Sname, mouse_x, mouse_y);
}

上記のコードでは、マウスボタンが押されたかどうかをチェックし、押されたことを検出すると、グローバル変数 "Sname" にインデックスされたパーティクルシステムをマウスの x/y 位置に移動させます。
