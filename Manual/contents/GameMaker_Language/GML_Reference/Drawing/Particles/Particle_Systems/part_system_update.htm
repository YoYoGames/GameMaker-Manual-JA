パートシステム更新

この関数は、指定されたパーティクルシステムをゲーム時間内に1ステップ進めます。自動更新がオフの場合のパーティクルシステムの動作をシミュレートしたり、特定のポイントまでパーティクルシステムを一気に進ませるために使用します。例えば、雪のパーティクルシステムがあり、プレイヤーがroom に入った瞬間から雪が降っているように見せたい場合、インスタンスの作成イベントで雪のパーティクルを数秒進め、最初に描画されたときに画面全体にパーティクルがあるようにします（下のコード例はこれを示しています）。また、この機能は、システムの自動更新が関数 part_system_automatic_update()を使用して、システムを移動させます。

構文です。

part_system_update(ind);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to update.

リターンです。

N/A

例

repeat (room_speed * 3)
{
    part_system_update(global.RainSys);
}

上記のコードは、グローバル変数 "RainSys "にインデックスされたパーティクルシステムを3秒進めるものです。
