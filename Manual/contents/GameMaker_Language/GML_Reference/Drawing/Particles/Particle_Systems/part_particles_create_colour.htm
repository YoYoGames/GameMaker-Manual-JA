パート_パーティクル_クリエイト_カラー

この機能は、パーティクルエミッタの機能を必要としないエフェクト（例えば、ミサイルの煙や単純な爆発エフェクトなど）に最適で、ゲーム内の任意の位置に迅速かつ簡単にパーティクルを作成することができますroom.この機能で作成したパーティクルに「その場で」色を付けることもでき、この色はパーティクルの定義済みの色よりも優先されますが、パーティクルの寿命が尽きるまでこの色がブレンドされるわけではありません。この機能を使用するには、事前にパーティクルシステムとパーティクルタイプを作成しておく必要があります。

構文です。

part_particles_create_colour(ind, x, y, parttype, colour, number);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system.
x リアル The x coordinate of where to create the particles.
y リアル The y coordinate of where to create the particles.
parttype 粒子タイプID The index (type) of the particles to be created.
colour カラー The colour blending for the particles.
number リアル The number of particles to create.

リターンです。

N/A

例

if (speed > 1)
{
    var _c = choose(c_aqua, c_lime, c_fuchsia, c_orange);
    part_particles_create_colour(sname, x, y, p_Smoke, _c, 1);
}

上記のコードは、このコードを持つインスタンスの速度が1より大きいステップごとに、1つのパーティクルを生成します。これらのパーティクルはランダムな色をしています。
