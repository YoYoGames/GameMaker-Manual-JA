part_system_destroy

この関数を使用すると、指定したパーティクルシステムを破壊し、メモリから削除することができます。この関数は、room やインスタンスの destroy イベントなど、システムが不要になったときに呼び出す必要があります。そうしないと、後のrooms でパーティクルが表示され、それを削除する方法がなく、メモリリークが発生して、最終的にゲームがクラッシュしてしまう可能性があります。

注：この関数は、作成され、破壊されるシステムに関連付けられたエミッタも破壊し、削除します。

構文です。

part_system_destroy(ind);

論証 タイプ 商品説明
ind パーティクルシステムID The index of the particle system to destroy.

リターンです。

N/A

例

if (global.player_hp <= 0)
{
    part_system_destroy(p_sys);
    room_goto_next();
}

上記のコードは、グローバル変数の値が 0 以下であるかどうかをチェックし、0 であれば与えられた変数で参照されているパーティクルシステムを破壊し、次のroom へと進みます。
