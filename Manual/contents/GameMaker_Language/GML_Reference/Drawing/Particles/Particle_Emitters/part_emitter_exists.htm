part_emitter_exists

この関数を使うと、インデックスで指定されたパーティクルエミッターが与えられたシステムに存在するかどうかをチェックすることができます。チェックされる変数が初期化されていない変数（パーティクルエミッターのインデックスが代入されている変数）である場合は、エラーを投げることに注意してください。

構文です。

part_emitter_exists(ps, ind);

論証 タイプ 商品説明
ps パーティクルシステムID The particle system to check for an emitter.
ind パーティクルエミッターID The index of the emitter to search for.

リターンです。

Boolean

例

if part_emitter_exists(sname, p_emit1)
{
    part_emitter_burst(sname, p_emit1, part_1, 30);
}

上記のコードは、変数 "permit "でインデックスされたエミッタをチェックし、存在すれば、そこから30個のパーティクルをバーストさせるというものです。
