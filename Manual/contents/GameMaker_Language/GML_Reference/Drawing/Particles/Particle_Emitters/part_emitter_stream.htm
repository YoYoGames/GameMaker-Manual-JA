パート_エミッター_ストリーム

この関数は、特定の種類のパーティクルを流すエミッターを設定するもので、通常、createイベントで使用されます。これは、ゲームの各ステップで指定した数のパーティクルを生成するエミッターを設定するワンオフコードだからです。パーティクルは、この関数で設定した分布、形状、位置に従って放出されます。 part_emitter_region().また、この関数はパーティクルの量を負の数で指定することもでき、その場合エミッターはランダムな確率でパーティクルを生成します。例えば、パーティクル数を-5に設定した場合、1ステップごとに1：5の確率でパーティクルが発生します。パーティクルをステップごとではなく、一度に発生させたい場合は、関数 part_emitter_burst().

構文です。

part_emitter_stream(ps, ind, parttype, number);

論証 タイプ 商品説明
ps パーティクルシステムID The particle system that the emitter is in.
ind パーティクルエミッターID The index of the emitter to burst from.
parttype 粒子タイプID The index (type) of the particles to be created.
number リアル The number of particles to create per step.

リターンです。

N/A

例

part_emitter_stream(global. Sname, p_emit1, p1, 1);

上記のコードでは、エミッタが破壊されるかストリームが0になるまで、ゲームの各ステップごとに1つのパーティクルをストリームします。
