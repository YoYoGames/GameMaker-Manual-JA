part_emitter_destroy（パートエミッターデストロイ

この関数は指定されたエミッタを指定されたシステムから削除し、メモリから消去します（これは指定されたエミッタによって生成されるパーティクルを停止しますが、room からそれらを削除するわけではありません）。この関数は、メモリリークやエラーを防ぐために、指定されたエミッタがシステムで不要になったときに常に呼び出される必要があります。

構文です。

part_emitter_destroy( ps, ind );

論証 タイプ 商品説明
ps パーティクルシステムID The particle system to destroy the emitter from.
ind パーティクルエミッターID The index of the emitter to destroy.

リターンです。

N/A

例

if part_emitter_exists(global.Sname, p_emit)
{
    part_emitter_destroy(global.Sname, p_emit1);
}

上記のコードは、変数p_emitでインデックスされたパーティクルエミッターがgive particle systemに存在するかどうかを確認し、存在する場合は破壊します。
