part_emitter_destroy_all

この関数は、指定されたシステムから定義されたエミッタをすべて削除し、メモリから消去します（これは、指定されたエミッタによって生成されるパーティクルも停止しますが、room から削除されるわけではありません）。この関数は、メモリリークやエラーを防ぐために、システムでエミッタが不要になったときに常に呼び出す必要があります。

構文です。

part_emitter_destroy_all( ps );

論証タイプ 商品説明
ps The particle system to destroy all emitters from.

リターンです。

例

if lives == 0
{
    part_emitter_destroy_all(global.Sname);
    room_goto(rm_Menu);
}

上記のコードは、組み込みのグローバル変数「lives」をチェックし、それが0であれば、すべてのパーティクルエミッターを破壊してから、room を変更します。
