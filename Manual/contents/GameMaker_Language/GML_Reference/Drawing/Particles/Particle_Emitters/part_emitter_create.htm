part_emitter_create

この関数は、与えられたパーティクルシステムに新しいパーティクルエミッターを作成するために使用されます。また、メモリリークを防ぐため、使わなくなったらエミッタを破棄する必要があります（特定のエミッタを破棄する関数やエミッタが所属するパーティクルシステム全体を破棄することで実現できます）。

注この関数が返すエミッタIDは再利用されます。例えば、最初に3つのパーティクルエミッタを作成すると、IDは0、1、2が割り当てられます。次に、2番目のエミッター（IDは1）を part_emitter_destroyエミッタIDが2であるため、次のエミッタのIDは3というように、再利用されます。

構文です。

part_emitter_create(ps);

論証 タイプ 商品説明
ps パーティクルシステムID The particle system to create the emitter in.

リターンです。

Particle Emitter ID

例

p_emit1 = part_emitter_create(Sname);

これにより、新しいパーティクルエミッターが作成され、そのインデックスが変数 "p_emit" に格納されます。
