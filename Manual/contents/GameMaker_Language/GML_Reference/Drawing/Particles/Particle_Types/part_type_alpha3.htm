part_type_alpha3

この関数は、指定されたタイプのパーティクルを作成するたびに、3つのアルファ（透明度）値のグラデーションを設定するために使用できます。最初のアルファ値はすべてのパーティクルが開始する値で、2番目のアルファ値はパーティクルの寿命の半分までブレンドされる値、3番目のアルファ値はパーティクルが終了する値です。パーティクルの寿命が尽きるまで、アルファからアルファへの滑らかなグラデーション変化が起こります。

構文です。

part_type_alpha3(ind, alpha1, alpha2, alpha3);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
alpha1 リアル The starting alpha of the particle.
alpha2 リアル The halfway point alpha of the particle.
alpha3 リアル The ending alpha of the particle.

リターンです。

N/A

例

part_type_alpha3( part_Health, 0.5, 1, 0);

上記のコードは、変数 "part_Health "でインデックスされたタイプの作成されたすべてのパーティクルが、そのライフタイムにおいて0.5から1、0までのアルファブレンドを持つようにする。
