part_type_sprite

この関数は、パーティクルの種類を、ゲームsprite のカスタムassets を使用するように設定するために使用することができます。sprite を選択し、GameMakerにsprite を使ってアニメーションさせるかどうかを指示します。アニメーションを選択した場合、サブ画像を一定間隔で変更し、作成したパーティクルの寿命の始まりと終わりでアニメーションを開始・終了させることもできます（寿命を最小15ステップ、最大30ステップに設定すると、寿命がどの値であっても、短い寿命では速く、長い寿命では遅く、アニメーションはそれに合わせて行われるようになります）。また、ランダムなサブ画像を選択することもでき、sprite がアニメーションしない場合、別の画像がランダムに選択されてsprite が作成され、パーティクルがアニメーションする場合は、ランダムなポイントからアニメーションが開始されます。

構文です。

part_type_sprite(ind, sprite, animate, stretch, random);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
sprite スプライトアセット The index of the sprite to set the particle type to.
animate ブーリアン Whether to follow the sprite's animation normally (true) or not (false).
stretch ブーリアン Whether to stretch the sprite's animation to match the particle lifespan (true) or not (false).
random ブーリアン Whether to choose a random sub-image (true) or not (false).

リターンです。

N/A

例

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

上記コードでは、グローバル変数 "p1 "でインデックスされたパーティクルタイプを、"spr_Coins "でインデックスされたsprite の形状に設定し、その他様々なパーティクルプロパティを設定します。
