part_type_alpha2

この関数は、指定されたタイプのパーティクルを作成するたびに、アルファ値（透明度）のグラデーションを設定するために使用できます。最初のアルファ値はすべてのパーティクルが開始する値、2番目のアルファ値はパーティクルが終了する値で、パーティクルの寿命に応じてアルファ値に滑らかなグラデーションが発生します。これは0（透明）から1（不透明）までの値です。

構文です。

part_type_alpha2(ind, alpha1, alpha2);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
alpha1 リアル The start alpha of the particle.
alpha2 リアル The end alpha of the particle.

リターンです。

N/A

例

part_type_alpha2(global.Snow_Part, 0, 1);

上記のコードは、グローバル変数「Snow_Part」でインデックスされたパーティクルタイプで作成されたすべてのパーティクルのアルファ値を0（透明）に設定し、寿命が尽きるとアルファ値1（不透明）になるようにフェードインします。
