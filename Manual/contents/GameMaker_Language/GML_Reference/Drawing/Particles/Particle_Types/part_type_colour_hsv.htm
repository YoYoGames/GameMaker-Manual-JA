part_type_colour_hsv

この関数では、指定されたタイプのすべてのパーティクルに対して、色相、彩度、値の範囲を設定することができます。3つの要素それぞれに最小値と最大値を与えると、作成されるパーティクルは、指定されたパラメータの範囲に基づいてランダムな色になります。この方法で、同じ色相で異なる彩度のパーティクルや、異なる色相で同じ値（輝度）のパーティクルを作成することができます...すべての値は0から255の間でなければなりません。

構文です。

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
hmin リアル The minimum the final colour's hue component can be.
hmax リアル The maximum the final colour's hue component can be.
smin リアル The minimum the final colour's saturation component can be.
smax リアル The maximum the final colour's saturation component can be.
vmin リアル The minimum the final colour's value component can be.
vmax リアル The maximum the final colour's value component can be.

リターンです。

N/A

例

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

上記のコードでは、グローバル変数 "Stars "でインデックスされたパーティクルタイプの各発光粒子が、異なる色と彩度を持ち、同じ値（光度）を持つように設定されています。
