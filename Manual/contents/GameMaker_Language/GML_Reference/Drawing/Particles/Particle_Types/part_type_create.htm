part_type_create（パートタイプ作成

この関数で新しいパーティクル タイプを作成し、その戻り値を変数に格納して、以降のすべてのパーティクル 関数で使用できるようにする必要があります。

構文です。

part_type_create();

リターンです。

Particle Type ID

例

mypart = part_type_create();

これにより、新しいパーティクル タイプが作成され、そのインデックスが変数 "mypart" に格納されます。
