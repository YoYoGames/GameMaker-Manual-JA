パートタイプ速度

この関数は、パーティクル作成時に移動速度を設定するのに使用し、パーティクルの寿命が尽きるまで速度を増加または減少させるのにも使用できます。最小速度と最大速度の値のデフォルトは1で、これはパーティクルが1ステップごとに1ピクセル移動することを意味します。同じ値に設定すると、すべてのパーティクルの初期速度が同じになりますが、異なる値に設定すると、それぞれのパーティクルは与えられた範囲の中からランダムに速度を決めて開始します。例えば、最小値を3、最大値を6とした場合、すべてのパーティクルは1ステップあたり3〜6ピクセルの速度で作成されます。

また、パーティクルの寿命が尽きるまで、より速く（正の数の場合）またはより遅く（負の数の場合）移動するように増分値を設定することができ、与えられた量が各ステップでパーティクル速度に追加されます。この値は最小で(+/-)0.01です。

最後に、"wiggle "ファクターを設定することができます。これは、パーティクルの寿命の各ステップごとに、速度にランダムに加算または減算される値です。明らかに大きな値は小さな値よりも顕著で、この値はマイナスにもでき、最大範囲は-20から20です。

構文です。

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
speed_min リアル The minimum speed the particle can start at.
speed_max リアル The maximum speed the particle can start at.
speed_incr リアル How much the particle speed should increase or decrease per step.
speed_wiggle リアル How much should randomly be added or subtracted from the particle's speed per step.

リターンです。

N/A

例

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

上記のコードでは、パーティクルの値をいろいろと設定します。スピードは最低5、最高10で、1ステップごとに-0.2の値でスピードが遅くなるように設定します。また、スピードは0から20の間のランダムな値で一歩ごとに揺れます。
