part_type_direction

この関数は、パーティクル作成時にパーティクルの方向を決定するために使用され、また、パーティクルの寿命が尽きるまでその方向を増加または減少させるために使用することができます。最小値と最大値はデフォルトで0、つまりパーティクルは変化なく右に移動しますが、この値を変更することでGameMaker標準の0が右、90が上、180が左、270が下という方向に沿って移動方向を制限することが可能です。同じ値を設定すると、パーティクルは一方向にしか動きません。

また、インクリメント値を設定することで、有効期間中に方向に対してプラス（正の数の場合）またはマイナス（負の数の場合）の度数を追加することができます。この値は、最小で (+/-) 0.01 です。

最後に、"wiggle "ファクターを設定することができます。これは、パーティクルの寿命の各ステップごとに、方向に対してランダムに加算または減算される値です。明らかに大きな値は小さな値よりも顕著で、この値は負にすることもでき、最大範囲は-20から20の間です。

構文です。

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
dir_min リアル The minimum direction the particle can start at.
dir_max リアル The maximum direction the particle can start at.
dir_incr リアル How much the particle direction should increase or decrease per step.
dir_wiggle リアル How much should randomly be added or subtracted from the particle's direction per step.

リターンです。

N/A

例

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

上記のコードでは、パーティクルの方向を0度から359度の間のランダムな値にするなど、様々な値を設定します。また、1ステップごとに1度ずつ上昇し、0から20の間のランダムな量が追加されます。
