part_type_shape（パートタイプ・シェイプ

この関数では、粒子の種類に応じて使用するsprite の形状を設定できます。以下の定数を使用できます。

粒子形状定数
定数 商品説明
pt_shape_pixel 1x1 ピクセルです。(デフォルトで設定されています）。
pt_shape_disk 円形に塗りつぶしたもの。
pt_shape_square 正方形に塗りつぶしたもの。
pt_shape_line 8px幅の横線。
pt_shape_star 5点満点の星。
pt_shape_circle 3pxのアウトライン化された円。
pt_shape_ring 内側に光る円（泡のように見える）。
pt_shape_sphere 外側に光る円 ' 真ん中に固まり、外側に光る。
pt_shape_flare 厳しく光る点（夜になると実際の星のように見える）。
pt_shape_spark 火花効果「複数の点がフェードアウトする星のようなもの。
pt_shape_explosion 四角い煙のような「爆発」を表現するためには、複数の色が必要です。
pt_shape_cloud 薄い雲は、雲に似せるためにアップスケーリングと複数のパーティクルが必要です。
pt_shape_smoke 爆発エフェクトのスムースバージョンです。複数使用することで煙のような雲を作ることができます。
pt_shape_snow 一般的な雪の結晶の形。

構文です。

part_type_shape(ind, shape);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to set.
shape 粒子形状定数 The shape to choose.

リターンです。

N/A

例

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

上記コードでは、グローバル変数 p1 で指定されたパーティクルの種類を正方形に設定し、そ の他の様々なパーティクルプロパティを設定しました。
