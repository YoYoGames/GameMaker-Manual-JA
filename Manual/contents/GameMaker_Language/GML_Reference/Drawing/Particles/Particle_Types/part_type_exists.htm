part_type_exists

この関数を使うと、インデックスで指定されたパーティクルの種類がゲーム内に存在するかどうかをチェックすることができます。ただし、チェックする変数が初期化されていない変数（パーティクルタイプのインデックスが代入されている変数）である場合は、エラーがスローされます。

構文です。

part_type_exists(ind);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to check for.

リターンです。

Boolean

例

if !part_type_exists(global.p1)
{
    global.p1 = part_type_create();
}

上記のコードでは、グローバル変数 p1 にパーティクル型のインデックスが格納されてい るかどうかを確認し、格納されていない場合はパーティクル型を生成しています。
