part_type_blend（パートタイプ・ブレンド

この機能により、パーティクルを加算ブレンドモードで描画する(true)か、しない(false)かを選択することができます。Additive ブレンドは、各パーティクルの輝度値を重なり合うごとに加算していく特殊なブレンドモードで、明るい色は重なり合うごとに徐々に明るくなり（白く見えるまで）、暗い色はどんどん透明になって黒はほとんど見えなくなっていきます。この機能は、美しいパーティクル効果を与えることができますが、必ずしも必要ではないかもしれません。例えば、煙の軌跡は、効果を灰色に保つために加算ブレンドをオフにしますが、炎の効果は、粒子をより半透明にし、明るくするために、おそらくこれをオンにします。

構文です。

part_type_blend(ind, additive);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
additive ブーリアン Whether the particles should be blended additively (true) or normally (false).

リターンです。

N/A

例

part_type_blend(part_Fire, true);

上記のコードでは、変数 "part_Fire "でインデックスされたパーティクルタイプで作成されたすべてのパーティクルに、加算ブレンドモードが設定されます。
