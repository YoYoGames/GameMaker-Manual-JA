part_type_alpha1

この関数は、個々のパーティクルの寿命の合計の間、単一のアルファ値（透明度）を持つようにパーティクルタイプを設定するために使用され、これは0（透明）から1（不透明）までの値を取ることができます。

構文です。

part_type_alpha1(ind, alpha1);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
alpha1 リアル The alpha of the particle.

リターンです。

N/A

例

part_type_alpha1(global.Snow_Part, 0.5);

上記のコードでは、グローバル変数「Snow_Part」でインデックスされたパーティクルタイプで作成されたすべてのパーティクルのアルファ値が0.5（半透明）になるように設定されます。
