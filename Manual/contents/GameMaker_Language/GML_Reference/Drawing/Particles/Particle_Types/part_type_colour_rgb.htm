part_type_colour_rgb

この関数では、指定された種類のパーティクルを作成する際に、赤、緑、青の色の組み合わせを設定できます。3つの要素それぞれについて最小値と最大値を指定すると、作成されるパーティクルは指定されたパラメータの範囲に基づいてランダムな色になります。すべての値は0から255の間でなければなりません。

構文です。

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
rmin リアル The minimum the final colour's red component can be.
rmax リアル The maximum the final colour's red component can be.
gmin リアル The minimum the final colour's green component can be.
gmax リアル The maximum the final colour's green component can be.
bmin リアル The minimum the final colour's blue component can be.
bmax リアル The maximum the final colour's blue component can be.

リターンです。

N/A

例

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

上記のコードでは、グローバル変数 "Blood_Part" でインデックスされたパーティクルタイプの各発光粒子を、異なる色調の赤のみに設定しています。
