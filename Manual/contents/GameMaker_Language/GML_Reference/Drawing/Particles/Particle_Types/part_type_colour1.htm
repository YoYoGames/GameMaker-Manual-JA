パートタイプカラー1

この関数は、個々のパーティクルの寿命の合計の間、パーティクルタイプが単一色であることを設定するために使用します。

構文です。

part_type_colour1(ind, colour1);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
colour1 カラー The single colour to make the particle type.

リターンです。

N/A

例

part_type_colour1(global.Snow_Part, c_white);

上記のコードでは、グローバル変数 "Snow_Part "でインデックスされたパーティクルタイプで作成されたすべてのパーティクルを白のみに設定します。
