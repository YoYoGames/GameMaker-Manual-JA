part_type_destroy

この関数を使うと、指定したパーティクルの種類をゲームから削除することができます。この関数を使用すると、指定したタイプのパーティクルはすべてroom から消え、パーティクル自体もメモリから削除されます。したがって、この関数はパーティクルが不要になった場合にのみ使用してください。

構文です。

part_type_destroy(ind);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to destroy.

リターンです。

N/A

例

if part_particles_count(smoke_sys) = 0
{
    part_type_destroy(smoke_part);
    part_system_destroy(smoke_sys);
    instance_destroy();
}
else alarm[0] = 1;

上記のコードは、パーティクルシステムをチェックして、現在room にパーティクルが表示されているかどうかを確認し、表示されていない場合は、パーティクル、システム、インスタンスを破棄しています。
