パートタイプカラーミックス

この関数では、与えられたパーティクルの種類を、2色のランダムなブレンドに設定することができます。

構文です。

part_type_colour_mix(ind, colour1, colour2);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
colour1 カラー The first colour the blend will take from.
colour2 カラー The second colour the blend will take from.

リターンです。

N/A

例

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

上記のコードは、グローバル変数 "P_Damage "でインデックスされたパーティクルタイプの各パーティクルが放出する色を、赤と黄色の間のランダムな混合色に設定するものです。
