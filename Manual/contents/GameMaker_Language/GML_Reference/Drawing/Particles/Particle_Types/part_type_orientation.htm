part_type_orientation

この関数は、パーティクル作成時にパーティクルsprite の向きを決定するために使用します。また、パーティクルのライフタイム中にパーティクルの向きを増加または減少させるために使用することも可能です。初期値は0ですが、この値を変更することでGameMakerの標準的な方向である0度：右、90度：上、180度：左、270度：下に沿ってランダムに向きを変更することができます。同じ値に設定すると、パーティクルはすべて同じ向きで作成されます。

また、インクリメント値を設定することで、その有効期間中に方位の角度を加算（正の数値の場合）または減算（負の数値の場合）することができます。この値は、最小で (+/-) 0.01 です。

他のパーティクル関数と同様に、"wiggle "ファクターも設定することができます。これは、パーティクルの寿命の各ステップごとに、向きにランダムに加算または減算される値です。明らかに小さい値より大きい値の方が顕著で、この値は負にすることもでき、最大範囲は-20から20の間です。

最後に、向きを相対的にするかしないかを選択できます。これは、パーティクルに方向（および速度）がある間は、パーティクルsprite はそのベクトルを中心に方向づけられることを意味します。パーティクルの速度を下げる設定にしていて0になった場合、速度不足で方向がデフォルト値の0°に設定されるため、相対的な方向はパーティクルsprite が異なる角度に「ジャンプ」することに留意してください。

構文です。

part_type_orientation( ind, ang_min, ang_max, ang_incr, ang_wiggle, ang_relative );

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
ang_min リアル The minimum starting angle of the particles.
ang_max リアル The maximum starting angle of the particles.
ang_incr リアル The increase (or decrease if set negative) of the angle in degrees per step.
ang_wiggle リアル The amount the angle wiggles clockwise/counterclockwise per step.
ang_relative リアル Whether to set its angle relative to the direction of the particle's motion (true) or not (false).

リターンです。

N/A

例

part_type_shape(particle2, pt_shape_spark);
part_type_size(particle2, 0.10, 0.50, 0.01, 0);
part_type_scale(particle2, 0.30, 0.30);
part_type_colour1(particle2, 8454143);
part_type_alpha1(particle2, 0.50);
part_type_speed(particle2, 4, 4, -0.07, 1);
part_type_direction(particle2, 0, 359, 0, 20);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 5);

上記のコードでは、様々なパーティクルの値が設定され、方向も0°から359°の間のランダムな値になります。また、各ステップごとに0から20の間でランダムな量が追加され、向きは進行方向に対して相対的なものとなります。
