パートタイプカラー3

この関数は、指定されたタイプのパーティクルを作成するたびに、3色のグラデーションを設定するために使用することができます。最初の色はすべてのパーティクルが開始する色で、2番目の色はパーティクルの寿命の半分までブレンドされる色、3番目の色はパーティクルが終了する色です。パーティクルの寿命が尽きるまで、ある色から別の色への滑らかなグラデーションが発生します。

構文です。

part_type_colour3( ind, colour1, colour2, colour3 );

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
colour1 カラー The colour the particle will start at.
colour2 カラー The colour the particle will be halfway through its lifespan.
colour3 カラー The colour the particle will end at.

リターンです。

N/A

例

part_type_colour3( part_Health, c_red, c_white, c_maroon);

上記のコードは、変数 "part_Health "でインデックスされたタイプのパーティクルを作成すると、その寿命の間に赤、白、マルーンに色がブレンドされるようになります。
