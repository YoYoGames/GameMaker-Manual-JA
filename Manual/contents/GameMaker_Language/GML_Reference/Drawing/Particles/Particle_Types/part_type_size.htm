パートタイプサイズ

この関数は、パーティクル作成時のサイズを決定するために使用され、また、パーティクルの寿命が尽きるまでサイズを増加または減少させるために使用することができます。サイズの最小値と最大値のデフォルトは1です（"1 "を指定した後のベースパーティクルと同じ比率であることを意味します）。 part_type_scale()例えば、最小値を0.5、最大値を2に設定すると、すべてのパーティクルは基本スケールの半分から2倍の大きさで作成されます。

また、粒子の寿命の間にsprite を成長（正の数の場合）または収縮（負の数の場合）させる増分値を設定することができ、指定した量が各ステップで粒子サイズに追加されます。この値は、最小で (+/-)0.01 です。

最後に、"wiggle "ファクターを設定することができます。これは、パーティクルの寿命の各ステップごとに、サイズにランダムに加算または減算される値です。明らかに大きな値は小さな値よりも顕著で、この値は負にすることもでき、最大範囲は-20から20の間です。

構文です。

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);

論証 タイプ 商品説明
ind 粒子タイプID The index of the particle type to change.
size_min リアル The minimum size the particle can start at.
size_max リアル The maximum size the particle can start at.
size_incr リアル How much the particle should increase or decrease per step.
size_wiggle リアル How much should randomly be added or subtracted from the particles size per step.

リターンです。

N/A

例

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

上記のコードは、各パーティクルが作成されるときに2倍に設定されるサイズを含む様々なパーティクル値を設定し、さらに0.1のウィグルでステップごとに0.01ずつサイズを増加させるものです。
