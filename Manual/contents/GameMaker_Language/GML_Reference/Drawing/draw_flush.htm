draw_flush

この関数を使用すると、描画パイプライン全体をフラッシュすることができます。この関数はデバッグ専用で、通常、YoYo Games サポートのメンバーから指示がない限り使用しないでください。 無差別に使用すると、ゲームのパフォーマンスに重大な問題を引き起こすからです。

構文です。

draw_flush();

リターンです。

例

draw_flush();

上記のコードで描画パイプラインをフラッシュします。
