draw_enable_drawevent

この関数を使うと、ゲーム内のすべてのインスタンスの draw イベントを有効 (true) または無効 (false) にすることができます。これは単にインスタンスが画面に描画しないだけでなく、draw イベントを完全に抑制することになるので、そのイベントに存在するゲームロジックも実行されないので注意が必要です。

この関数について理解しておくべき重要な点は、ゲームの開始直後、最初のフレームがレンダリングされる前（つまり、ゲームの最初のobject の最初のroom の Create Event）に呼び出すと、ゲーム ウィンドウがレンダリングされないということです。これは、何もレンダリングする必要のない専用のserver を必要とするマルチプレイヤー プロジェクトには便利ですが、GameMaker がサポートしていないため、ヘッドレスにはならないことに留意してください -- したがって、グラフィカル インターフェイスを持たないserver では実行できません。

構文です。

draw_enable_drawevent(enable);

論証タイプ 商品説明
Enable Set to true または . false

リターンです。

例

frame_skip ++;
if (frame_skip mod 5) == 0
{
    draw_enable_drawevent(true);
}
else
{
    draw_enable_drawevent(false);
}

上記のコードは、変数をチェックして、 true を返したら、draw イベントを有効にし、そうでなければ、draw イベントを無効にしています。
