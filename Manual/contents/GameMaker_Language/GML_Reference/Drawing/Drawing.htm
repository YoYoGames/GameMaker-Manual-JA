このセクションには、ゲーム内の描画に関連するすべての関数が含まれていますroom 、および物事の描画方法（ブレンド、アルファ、カリングなど）を制御するためのものです。描画に関する関数は非常に多く、必要なものを見つけやすくするために、以下のカテゴリに分けられています。

  • 色とアルファ
  • GPU制御
  • ミップマッピング
  • 基本フォーム
  • スプライトとタイル
  • テキスト
  • プリミティブとVertex フォーマット
  • サーフェス
  • 照明
  • 粒子
  • テクスチャ
  • シェーダー
  • 動画再生

1つはプロジェクトの描画イベントを有効または無効にするために使用され、もう1つは描画パイプライン全体をフラッシュするために使用される2つの特殊関数もあります。

  • draw_enable_drawevent
  • draw_flush
