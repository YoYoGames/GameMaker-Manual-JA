ドローライトディファインアンビエント

この機能は、シーンの環境光（点光源や指向性光源を定義しなくても、シーンにある光）を制御するために使用されます。シーンの全体的な色と明るさ（または暗さ）を効果的に調整します。 デフォルトの色は、 c_black です。

構文です。

draw_light_define_ambient(col);

論証タイプ 商品説明
col The colour to use (either a constant, a real or a hex value).

リターンです。

例

draw_light_define_ambient(c_white);

上記のコードでは、アンビエントライトは白色と定義されます。
