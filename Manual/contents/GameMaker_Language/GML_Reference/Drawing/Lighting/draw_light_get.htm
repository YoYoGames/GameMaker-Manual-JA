draw_light_get

この関数は、指定された光源パラメータを、以下の6つの要素を持つ配列として取得します。

  • [0] = 有効 / 無効 (true / false)
  • [1] = ライトタイプ定数(下表参照)
  • [2] = x 位置
  • [3] = y 位置
  • [4] = z 位置
  • [5] = ライトの半径（ポイントライトのみ、ディレクショナルライトの場合は0になる）
  • [6] = 明るい色 (本物)

ライトタイプ（要素[1]）には、以下の2つの定数のいずれかを指定することができます。

光源種類 一定
定数 商品説明
lighttype_dir ライトは指向性ライト
lighttype_point 光は点光源

構文です。

draw_light_get(ind);

論証 タイプ 商品説明
ind リアル The index number of the light (from 0 to 7)

リターンです。

Array

例

light_a = draw_light_get(1);
if (light_a[5] < 200)
{
    light_a[5] += 5;
    draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
}

上記のコードは、"1 "としてインデックスされたライトを定義するために使用される値を取得し、次に半径をチェックし、それが200より小さい場合、それは増加し、光の半径は新しい値に設定されます。
