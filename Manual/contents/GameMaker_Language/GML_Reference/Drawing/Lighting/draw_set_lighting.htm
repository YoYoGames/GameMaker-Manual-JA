draw_set_lighting

すべての照明効果を有効にするための機能です。デフォルトは無効（false ）です。

構文です。

draw_set_lighting(enable);

論証 タイプ 商品説明
enable ブーリアン Enable or disable all lighting (true またはfalse)

リターンです。

N/A

例

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

上記のコードでは、シーン全体の照明を有効にし、次にroom の空間に白い指向性ライトを定義し、最後にそのライトを点灯させます。
