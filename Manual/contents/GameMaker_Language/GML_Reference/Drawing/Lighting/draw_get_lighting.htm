draw_get_lighting

この関数は、シーン全体のライティングが有効（true ）か無効（false ）かを返します。

構文です。

draw_get_lighting()

リターンです。

例

if !draw_get_lighting()
{
    draw_set_lighting(true);
}

上記のコードは、シーンに対してライティングが有効かどうかをチェックし、有効でない場合は有効にします。
