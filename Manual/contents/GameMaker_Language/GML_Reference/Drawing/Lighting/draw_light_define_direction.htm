ドローライトの定義方向

この関数は指向性ライトを定義するためのもので、方向と強度は x, y, z 単位ベクトルに入力された値によって設定される（つまり、-1 から 1 の間でなければならず、1 または -1 は全強度、0 は無強度である）。また これは、ある特定の色が他の色よりも「暗く」見えるため、光の強さの知覚にも影響します。また、他の関数で参照するために使用されるインデックス番号もライトに与える必要があります。 となります。

注：利用可能なハードウェア・ライトは8つしかないため、一度に有効にできるのは定義された8つのライトのみです（ただし、もっと多く定義することは可能です）。

構文です。

draw_light_define_direction(ind, x, y, z, col);

論証タイプ 商品説明
ind The index number of the light (arbitrary)
x The x component of the light vector
y The y component of the light vector
z The z component of the light vector
col The colour to use for the light (either a constant, a real or a hex value).

リターンです。

例

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

上記のコードでは、シーン全体の照明を有効にし、次にroom の空間に白い指向性ライトを定義し、最後にそのライトを点灯させます。
