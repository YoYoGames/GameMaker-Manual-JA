draw_light_enable

この機能は、定義されたライトを有効にするために使用されます。switch ポジションライトやディレクショナルライトを定義する際には、そのライトにインデックス番号を割り当てる必要があります。このインデックス番号は、この関数でライトのオン/オフを切り替えるために使用されます。デフォルトは無効（false ）です。

注：利用可能なハードウェア・ライトは8つしかないため、一度に有効にできるのは定義された8つのライトのみです（ただし、もっと多く定義することは可能です）。

構文です。

draw_light_enable(ind, enable);

論証 タイプ 商品説明
ind リアル The index number of the light (from 0 to 7)
enable ブーリアン Enable or disable all lighting (true または ) false

リターンです。

N/A

例

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

上記のコードでは、シーン全体の照明を有効にし、次にroom の空間に白い指向性ライトを定義し、最後にそのライトを点灯させます。
