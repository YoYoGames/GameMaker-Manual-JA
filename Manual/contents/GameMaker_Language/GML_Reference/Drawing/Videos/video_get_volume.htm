video_get_volume

この関数は、現在読み込まれているビデオの音量を 0～1 の値で返します。読み込まれているビデオの音量は、 video_set_volume() で変更することができます。

構文です。

video_get_volume();

リターンです。

Real

例

/// Step event
var _video_volume = video_get_volume();

if (_video_volume < 1) {
    _video_volume += 0.02;
    video_set_volume(_video_volume);
}

上記のコードは、Stepイベントで実行されます。現在再生中の動画の音量を取得し、それが1より小さい場合は0.02ずつ上げていき、最大音量になるまで徐々に上げていきます。
