video_get_status

この関数は、現在ロードされているビデオのステータスを返す。ステータスには、以下の定数のいずれかを指定することができる。

ビデオステータス定数
定数 商品説明
video_status_closed 現在、ビデオが読み込まれていないか、または、ビデオが video_close()
video_status_preparing 動画は現在準備中で、まだ再生が始まっていません
video_status_playing 動画は現在再生中です
video_status_paused 動画が一時停止されます（ video_pause() 参照）。

構文です。

video_get_status();

リターンです。

Video Status Constant

例

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

上記のコードでは、動画の状態を取得し、プレーヤーが Space を押したかどうかをチェックしています。この場合、動画が再生中であれば一時停止し、一時停止中であれば再開します。
