video_set_volume

この関数は、現在読み込まれているビデオの音声の音量を変更する。0から1 の間の値をとり、0は無音、1は最大音量となる。

このメソッドは、 video_open() 呼び出しの後、 video_close() 呼び出しの前にのみ呼び出されることを確認してください。そうでない場合は、何も行いません（ビデオが読み込まれないため）。

プラットフォームによっては、この機能はビデオが完全に読み込まれた後にしか動作しないため、非同期コールバックでビデオの準備ができたことを確認してから、音量を変更するようにしてください。

構文です。

video_set_volume(value);

論証 タイプ 商品説明
value リアル Volume value between 0 and 1

リターンです。

N/A

例

my_video = video_open("splash.mp4");
video_set_volume(0.5);

上記のコードでは、ゲームのIncluded Filesからsplash.mp4をロードし、その音量を0.5（最大音量の50%）に変更します。
