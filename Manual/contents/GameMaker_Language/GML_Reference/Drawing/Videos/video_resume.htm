ビデオレジュメ

この関数は、現在ロードされているビデオファイルをレジュームする。この関数を呼び出す前に、 video_pause() で一時停止しておく必要があります。

構文です。

video_resume();

リターンです。

N/A

例

var _status = video_get_status();

if (keyboard_check_pressed(vk_space))
{
    if (_status == video_status_playing)
    {
        video_pause();
    }
    else if (status == video_status_paused)
    {
        video_resume();
    }
}

上記のコードでは、動画の状態を取得し、プレーヤーが Space を押したかどうかをチェックしています。この場合、動画が再生中であれば一時停止し、一時停止中であれば再開します。
