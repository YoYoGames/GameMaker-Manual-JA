video_close

この関数は、現在ロードされているビデオファイルをクローズします。この関数は、 video_open() の呼び出しの後にのみ呼び出されるようにしてください。そうでない場合は何もしません。

構文です。

video_close();

リターンです。

N/A

例

if (keyboard_check_pressed(vk_escape))
{
    video_close();
}

上記のコードは、Escapeキーが押されたときにビデオを閉じます。
