video_get_duration

説明： 現在ロードされているビデオの再生時間をミリ秒単位で返す。

構文です。

video_get_duration();

リターンです。

Real

例

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

上記のコードは、右矢印キーが押されたときに、再生中のビデオを2秒先に移動させ、新しい位置はビデオの継続時間よりも小さくなっています。ただし、再生時間内でない場合は、ビデオを閉じます。

この例では、 video_get_duration() を条件として、新しい位置が返された期間より小さいかどうかをチェックしています。
