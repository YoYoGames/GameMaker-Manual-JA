ビデオオープン

この関数は、指定されたpath を使用して動画ファイルを開きます。path は、プロジェクトのIncluded Filesにあるファイル、ユーザーのデバイスにローカルに存在するファイル、またはインターネットからファイルを読み込むための URL を参照できます（プラットフォームによっては機能しない場合があります）。

一度にロードできるビデオは1つだけなので、この関数は指定されたビデオを唯一の利用可能な「ビデオスロット」にロードし、他のビデオ関数で操作することになります。

動画再生に関連する非同期コールバックが存在します。詳細は、「非同期コールバック」を参照してください。

構文です。

video_open(path);

論証 タイプ 商品説明
path 文字列 Path to the video file

リターンです。

N/A

例

my_video = video_open("splash.mp4");

上記のコードでは、ゲームのIncluded Filesからsplash.mp4をロードしています。
