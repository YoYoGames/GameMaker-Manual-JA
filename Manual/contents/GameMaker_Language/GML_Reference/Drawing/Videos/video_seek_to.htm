video_seek_to

この関数は、現在読み込まれているビデオ内の指定された位置にシークすることができる。シークする時間値をミリ秒単位で指定します。

この機能は、動画が一時停止しており、現在位置から新しいシーク位置までの距離があまり小さくない場合に有効です。

構文です。

video_seek_to(time);

論証 タイプ 商品説明
time リアル The time to seek to (in milliseconds).

リターンです。

N/A

例

if (keyboard_check_pressed(vk_right))
{
    var _video_position = video_get_position();
    _video_position += 2000;

    if (_video_position < video_get_duration()) 
    {
        video_seek_to(_video_position);
    }
    else
    {
        video_close();
    }
}

上記のコードは、右矢印キーが押されたときに、再生中のビデオを2秒先に移動させ、新しい位置はビデオの継続時間よりも小さくなっています。ただし、再生時間内でない場合は、ビデオを閉じます。

この例では、 video_seek_to() 、新しい位置にビデオをシークするために使用されています。
