ビデオ・イズ・ルーピング

この関数は、現在読み込まれているビデオがloop に設定されているか（true）、そうでないか（false）を返す。 video_enable_loop() を呼び出すことで、ビデオをloop に設定することができます。

構文です。

video_is_looping();

リターンです。

Boolean

例

if (video_is_looping()) {
    obj_player.control_enabled = false;
}

上記のコードでは、現在ロードされている動画がループしているかどうかをチェックし、その場合はカスタム インスタンス変数をfalse に設定してプレーヤのコントロールを無効にしています。
