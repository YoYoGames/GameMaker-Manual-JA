サーフェス_ゲッタテクスチャー

この関数は、サーフェスの特別なポインタ texture を返します。この値は、他の描画関数、特に一般的な3D関数や一部の2Dプリミティブ関数で使用することができます。

注意サーフェスで作業する場合、texture のメモリーに保存されているため、いつでもサーフェスが存在しなくなる可能性があります。サーフェスを直接参照する前に、サーフェスが存在するかどうかを常にsurface_exists()を使用して、サーフェスが存在するかどうかを必ず確認してください。

構文です。

surface_get_texture(surface_id);

論証 タイプ 商品説明
surface_id サーフェスID The ID of the surface to get the width of.

リターンです。

Texture

例

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

上記のコードは、変数 "tex" に保持されているvertex でテクスチャされた 4texture の三角形の帯を描画します。この帯は、それ自体、以前に作成した表面から取得されます。
