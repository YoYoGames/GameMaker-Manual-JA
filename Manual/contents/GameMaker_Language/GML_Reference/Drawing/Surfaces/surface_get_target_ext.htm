サーフェスゲットターゲットエクスト

この関数は、サーフェスが利用できる4つのレンダーターゲットのいずれかに割り当てられているサーフェスIDを取得します。チェックするレンダーターゲットのインデックスを指定すると、サーフェスが割り当てられていない場合は-1、割り当てられたサーフェスのIDを表す整数値 >= 0が返されます（関数 surface_create()).

構文です。

surface_get_target_ext(index);

論証タイプ 商品説明
index The render target index to check (from 0 to 3).

リターンです。

Surface ID or -1 (if no target surface is set)

例

if surface_get_target_ext(0) == -1
{
    surface_set_target_ext(0, global.Surf);
}

上記のコードでは、まずshader レンダーターゲット 0 がサーフェイスに設定されているかどうかを確認し、設定されていない場合はサーフェイスを割り当てます。
