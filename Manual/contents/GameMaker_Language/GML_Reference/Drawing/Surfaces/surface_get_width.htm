表面幅を取得する

この関数は、単にインデックスされた表面の幅をピクセル単位で返します。を使用してサイズを変更した直後に application_surface をチェックするためにこの関数を呼び出した場合は、注意が必要です。 surface_resize() というのは、変更が完全に処理されるには 1、2 ステップ必要だからです。1ステップ待つと、新しいサイズが正しく返されるはずです。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists() 直接参照する前に

構文です。

surface_get_width(surface_id);

論証タイプ 商品説明
surface_id The ID of the surface to get the width of.

リターンです。

例

sw = surface_get_width(surf);

上記のコードでは、変数 "surf "でインデックスされた面の幅が変数 "sw "に格納されることになる。
