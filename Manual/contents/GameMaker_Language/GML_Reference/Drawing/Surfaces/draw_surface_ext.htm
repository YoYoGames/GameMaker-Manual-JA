ドローサーフェスエクスト

この関数は、与えられた表面を、関数 draw_surface()しかし、描画される表面のスケール、ブレンド、回転、アルファを変更する追加オプションがあります。これらの値を変更しても、リソースには何の変更もありません（描画方法のみ）。

構文です。

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

論証 タイプ 商品説明
id サーフェスID The unique ID value of the surface to draw.
x リアル The x position of where to draw the surface.
y リアル The y position of where to draw the surface.
xscale リアル The horizontal scale.
yscale リアル The vertical scale.
rot リアル The rotation or angle to draw the surface.
col カラー The colour with which to blend the surface.
alpha リアル The alpha transparency for drawing the surface.

リターンです。

N/A

例

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

上記のコードは、変数 "surf "でインデックスされた表面を、room の (0,0) の位置に、元の縮尺の2倍で、赤と半透明の混色で描画します。
