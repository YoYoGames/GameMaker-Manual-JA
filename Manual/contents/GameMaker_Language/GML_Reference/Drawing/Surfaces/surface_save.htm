サーフェスセーブ

この関数は、指定されたファイル名で表面をディスクに保存します。サーフェスは *.png 形式のファイルとして保存する必要があります。

構文です。

surface_save(surface_id, fname);

論証タイプ 商品説明
surface_id The ID of the surface to set as the drawing target.
fname The name of the saved image file.

リターンです。

例

if keyboard_check_pressed(ord("S")
{
    surface_save(surf, "test.png");
}

上記のコードは、ユーザーがキーボードの「S」キーを押したかどうかを確認し、押した場合は変数「surf」でインデックスされた面をディスクに保存します。
