draw_surface

この関数を使うと、room 内の指定した位置に表面を描くことができ、表面の左上隅は指定した x/y 位置に描かれます。

構文です。

draw_surface(id, x, y);

論証 タイプ 商品説明
id サーフェスID The unique ID value of the surface to draw.
x リアル The x position of where to draw the surface.
y リアル The y position of where to draw the surface.

リターンです。

N/A

例

var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
draw_surface(surf, _vx, _vy);

上記のコードでは、"surf "で指定された面をカメラview[0]と同じ位置に描画しています。
