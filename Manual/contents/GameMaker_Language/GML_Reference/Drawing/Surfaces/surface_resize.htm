surface_resize

この関数は、指定された寸法（ピクセル単位）にサーフェイスをリサイズします。surface_id」は、以前に作成したサーフェスのもの、または application_sufaceで、この関数はそのサーフェイスのサイズを変更します。これはサーフェイスの内容を切り取ったり伸ばしたりするのではなく、現在のサーフェイスを破壊し、新しい寸法で同じハンドル（surface_id）で再作成することになるので、クリアして再度描画する必要があることに注意。（それが application_surface であればGameMakerが自動的にこれを行う場合を除く）。

注：アプリケーションサーフェスのサイズを変更する場合、これらの変更は次の描画フレームが始まるまで登録されません。 surface_get_width()または surface_get_height()関数は、同じイベントまたはステップの中で、以前の値を返します。

構文です。

surface_resize(surface_id, w, h);

論証タイプ 商品説明
surface_id The ID of the surface to change.
w The width of the new surface.
h The height of the new surface.

リターンです。

例

if view_wport[0] != surface_get_width(application_surface) || view_hport[0] != surface_get_height(application_surface)
{
    surface_resize(application_surface, view_wport[0],view_hport[0]);
}

上記のコードは、ビューポートのサイズとサーフェス "aplication_surface" のサイズを照合し、異なる場合はサーフェスのサイズを変更します。
