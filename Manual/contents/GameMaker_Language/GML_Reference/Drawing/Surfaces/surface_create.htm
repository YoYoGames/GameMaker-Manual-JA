サーフェスクリエイト

この関数はサーフェスを作成するために使用され、サーフェスのインデックスを返します。このインデックスは、将来の関数呼び出しに備えて変数に格納する必要があります。サーフェスが最初に作成されたとき、それは基本的にこの目的のために確保された単なるメモリ領域であるため、「ノイズ」を含むことがあります（そしてそのメモリにはまだ情報が含まれていることがあります）ので、次のような関数で使用する前にサーフェスをクリアすることが望ましいでしょう。 draw_clear_alpha().

すべてのサーフェスは2の累乗サイズ、つまり16、128、512、1024ピクセルサイズで作成することを強くお勧めします。これは特定のプラットフォーム（Windows や MacOS など）では必ずしも必要ではありませんが、これらのターゲットでは確実に互換性が高まります。一方、HTML5 やデバイスでは必須で、これを覚えておかないと後で問題に遭遇する可能性があります。

構文です。

surface_create(w, h);

論証タイプ 商品説明
w The width of the surface to be created.
h The height of the surface to be created.

リターンです。

Surface ID

例

if !surface_exists(surf)
{
    surf = surface_create(1024, 1024);
    surface_set_target(surf);
    draw_clear_alpha(c_black, 0);
    surface_reset_target();
    view_surface_id[0] = surf;
}

上記のコードでは、サーフェスが存在するかどうかをチェックし、存在しない場合は幅1024ピクセル、高さ1024ピクセルのサーフェスを作成し、変数「surf」にインデックスを代入しています。そして描画ターゲットが新しいサーフェスに設定され、クリアされて透明になった後、描画ターゲットがディスプレイにリセットされる。最後に、サーフェスはビューに割り当てられる。
