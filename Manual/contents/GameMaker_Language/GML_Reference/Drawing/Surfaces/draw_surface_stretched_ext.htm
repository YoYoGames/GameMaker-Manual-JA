ドローサーフェスストレッチエクスト

この関数は draw_surface_stretched()関数に、描画時に表面のカラーブレンドとアルファ値を設定する機能が追加されました (関数 draw_surface_ext()).

構文です。

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

論証 タイプ 商品説明
id サーフェスID The unique ID value of the surface to draw.
x リアル The x position of where to draw the surface.
y リアル The y position of where to draw the surface.
w リアル The width at which to draw the surface.
h リアル The height at which to draw the surface.
colour カラー The colour with which to colour the surface.
alpha リアル The alpha with which to blend the surface.

リターンです。

N/A

例

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

これは、指定された表面の左隅をインスタンスの x/y 位置に描画し、200x200 ピクセルの領域を占めるように、ブレンドなしで、部分的に透明にして伸張します。
