サーフェスリセットターゲット

この関数では、ターゲットサーフェスからの以降の描画をすべてリセットし、スクリーンに戻すことができます。サーフェスへの描画を開始するには、最初に関数 surface_set_target()さもないと、それ以降のすべての描画は (他のインスタンスであっても)サーフェス上で行われるため、スクリーンには何も描画されなくなります。また、インスタンスの draw イベントでサーフェス自体が画面に描画されないと、何も表示されないことも知っておく必要があります。

注意：事前にレンダリングターゲットを関数 surface_set_target()この関数を使用すると、そのイベントに関するすべてのコード実行をサイレントで（つまり、エラーメッセージなしで）終了します。

構文です。

surface_reset_target();

リターンです。

例

if view_current = 0
{
    surface_set_target(surf);
    with (obj_Effect)
    {
        draw_self();
    }
    surface_reset_target();
}
else
{
    draw_surface(surf, 0, 0);
}

上記のコードは、現在どのビューが描画されているかをチェックし、それが view[0] であれば、描画ターゲットをサーフェスに設定し、object "obj_Effect" のインスタンスをすべて描画してから描画ターゲットを再度リセットしています。ビューが view[0] でない場合、サーフェスはスクリーンに描画されます。
