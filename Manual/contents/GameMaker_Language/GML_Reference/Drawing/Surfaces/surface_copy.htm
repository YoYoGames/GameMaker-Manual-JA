サーフェス_コピー

この関数は、あるサーフェスから画像を取り出し、そのサーフェス内の指定された局所位置（ここで (0,0) 位置はコピー先サーフェスの左上隅です）にある別のサーフェスにコピーするだけです。コピー先のサーフェスがすでに情報を持っている場合、これはコピーによって上書きされます。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists()直接参照する前に

構文です。

surface_copy(destination, x, y, source);

論証 タイプ 商品説明
destination サーフェスID The ID of the surface to copy the other surface to.
x リアル The x position to copy to.
y リアル The y position to copy to.
source サーフェスID The ID of surface to be copied.

リターンです。

N/A

例

if view_current == 0
{
    surface_copy(surf, 0, 0, temp_surf);
}
else
{
    draw_surface(surf, 0, 0);
}

上記のコードは、現在描画されているビューをチェックし、それがview[0]であれば、変数temp_surfでインデックスされたサーフェイスを変数surfでインデックスされたサーフェイスにコピーします。もし現在のビューがview[0]以外の場合は、サーフェス "surf "が画面に描画されます。
