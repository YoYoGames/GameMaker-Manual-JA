サーフェス_セット_ターゲット_エクスト

この関数は、Shader Functionsで使用するためのもので、ネイティブレベルshaders (DX9, DX11, OpenGL) の MRT (0 - 3) を設定するものです。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists() 直接参照する前に

構文です。

surface_set_target_ext(index, surface_id);

論証タイプ 商品説明
index The render target index to use (from 0 to 3).
surface_id The ID of the surface to use.

リターンです。

例

surface_set_target_ext(0, surf);

上記のコードでは、レンダーターゲット0に、変数「surf」でインデックスされたサーフェスIDを設定します。
