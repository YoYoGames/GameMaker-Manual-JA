ドローサーフェスストレッチ

この関数は、単純に表面を受け取り、与えられた幅と高さに、その面積を占めるように引き伸ばすだけである。と同様に draw_surface()は、表面を指定し、次に描画される表面のroom における x / y 位置を指定し、最後に幅と高さ (これらはピクセル値でなければならない) を指定することができます。

構文です。

draw_surface_stretched(id, x, y, w, h);

論証 タイプ 商品説明
id サーフェスID The unique ID value of the surface to draw.
x リアル The x position of where to draw the surface.
y リアル The y position of where to draw the surface.
w リアル The width at which to draw the surface.
h リアル The height at which to draw the surface.

リターンです。

N/A

例

draw_surface_stretched(surf, 10, 10, 100, 100);

これは、変数 "surf "でインデックスされたサーフェイスを、その左角を(10,10)にして描画するものである。幅と高さはともに100に設定されている。これはサーフェスの実際の幅と高さに関係なく、どれだけのスペースを占有するかということである。
