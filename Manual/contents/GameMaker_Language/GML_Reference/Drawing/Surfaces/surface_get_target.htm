サーフェスゲットターゲット

この関数を使用すると、現在描画対象としているサーフェスのIDを取得できます。を使用してサーフェスを作成した場合、これはそのサーフェスのユニークなIDである正の整数になります。 surface_create()を使用して作成した場合、これはそのサーフェスのユニークなIDである正の整数になります。

カスタムサーフェスが設定されておらず、デフォルトの application_surface が有効になっている場合、この関数は通常 0 を返します。しかし、以前にapplication_surfaceを操作していた場合は、0でなくなった場合に備えて、取得したIDがapplication_surface変数の現在の値と一致しないことを確認した方がよいかもしれません。

カスタムサーフェイスが設定されておらず、デフォルトの application_surface も無効化されている場合、この関数は -1 を返します。

構文です。

surface_get_target();

リターンです。

Surface ID or -1 (if no target surface is set)

例

if surface_get_target() != -1
{
    surface_reset_target();
}
draw_surface(surf, 0, 0);

上記のコードは、現在のレンダリングターゲットがサーフェスであるかどうかを確認し、サーフェスであればターゲットをリセットしてからサーフェスを描画します。
