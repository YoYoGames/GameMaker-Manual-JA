surface_get_depth_disable

この関数は、サーフェスに対する深度buffer の自動生成が有効かどうかをチェックします。通常、すべてのサーフェイスは深度buffers を持っているので、3Dobjects を描けば、深度によって適切に分類されます。しかし、深度buffers を割り当てると、サーフェイスのサイズが実質的に 2 倍になるので、特にゲームが非常にメモリを食う場合や主に 2D の場合は、過剰かつ不要なオーバーヘッドになる可能性があります。このような場合、この関数でこれを確認し、必要であれば、関数を使ってサーフェスの深度buffer を無効にすることができます。 surface_depth_disable().

構文です。

surface_get_depth_disable();

リターンです。

例

if (!surface_exists(surf))
{
    if (surface_get_depth_disable() == false)
    {
        surface_depth_disable(true);
    }
    surf = surface_create(room_width, room_height);
}

上記のコードは、与えられたサーフェイスが存在するかどうかを確認し、存在しない場合は、サーフェイス深度buffer の現在の状態をチェックし、有効になっていれば、代わりに無効にし、最終的にサーフェイスを作成することになります。
