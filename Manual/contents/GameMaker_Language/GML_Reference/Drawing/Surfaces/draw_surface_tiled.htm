ドローサーフェスティルド

この関数は、ある表面を受け取り、関数で与えた座標を起点に、tile 全体にわたって繰り返しroom を行います。

構文です。

draw_surface_tiled(id, x, y);

論証 タイプ 商品説明
id サーフェスID The unique ID value of the surface to draw.
x リアル The x position of where to draw the surface.
y リアル The y position of where to draw the surface.

リターンです。

N/A

例

draw_surface_tiled(surf, x, y);

これは、インスタンス自身のxとyの位置に "surf "でインデックスされた表面を描画し、room のすべての方向にタイル状に配置します。
