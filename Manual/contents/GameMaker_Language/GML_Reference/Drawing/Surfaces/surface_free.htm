サーフェスフリー

サーフェスを使うときは、使い終わったときに必ずこの機能を使うようにしてください。サーフェスはメモリにスペースを取るので、削除する必要があります。通常はroom の終了時に削除しますが、使用状況によってはいつでも削除できます。これを怠ると、メモリリークが発生し、最終的にはゲームの動作が遅くなり、クラッシュする可能性があります。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists()直接参照する前に

構文です。

surface_free(surface_id);

論証タイプ 商品説明
surface_id The ID of the surface to be freed.

リターンです。

例

if keyboard_check_pressed(vk_escape)
{
    surface_free(surf);
    room_goto(rm_Menu);
}

上記のコードは、キーが押されたかどうかをチェックし、押されたことを検出すると、変数 "surf "でインデックスされた表面用に確保されたメモリを解放し、room を変更します。
