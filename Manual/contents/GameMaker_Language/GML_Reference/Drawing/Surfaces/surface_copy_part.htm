surface_copy_part

この関数は、あるサーフェスから画像を取り出し、そのサーフェス内の指定されたローカル位置（ここで、(0,0)の位置はコピー先のサーフェスの左上隅です）の別のサーフェスにコピーするだけです。コピー元のローカル x、y 位置と、セクションの幅と高さを指定できます。これらはサーフェスのサイズに基づいた座標であり、サーフェスがroom に描かれている位置ではないことに注意してください。コピー先のサーフェスにすでに情報がある場合、コピーによって上書きされ、この関数によってコピー元のサーフェスが変更されることは一切ありません。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists()直接参照する前に

構文です。

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

論証 タイプ 商品説明
destination サーフェスID The ID of the surface to copy the other surface to.
x リアル The x position to copy to.
y リアル The y position to copy to.
source サーフェスID The ID of the surface to be copied.
xs リアル The x position in the source surface to copy from.
ys リアル The y position in the source surface to copy from.
ws リアル The width of the area in the source surface to copy from.
hs リアル The height of the area in the source surface to copy from.

例

if view_current == 0
{
    var _cam1_x = camera_get_view_x(view_camera[1]);
    var _cam1_y = camera_get_view_y(view_camera[1]);

    surface_copy_part(surf, 0, 0, temp_surf, 0, 0, _cam1_x - mouse_x, _cam1_y - mouse_y);
}
else
{
    draw_surface(surf, 0, 0);
}

上記のコードは、現在描画されているビューをチェックし、それがview[0]の場合、変数temp_surfで指定されたサーフェイスを変数surfで指定されたサーフェイスにコピーしています。コピーされる領域は、view[1]で描画されるサーフェス内のマウスの相対位置で形成される矩形に相当します。現在のビューがview[0]以外の場合は、サーフェス "surf "がスクリーンに描画される。
