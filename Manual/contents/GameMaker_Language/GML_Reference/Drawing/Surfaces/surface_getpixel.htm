サーフェスゲットピクセル

この関数は、表面のローカル座標（(0,0)が左上隅）を使用して、表面から特定のピクセルの色を取得するために使用することができます。この関数は非常に遅く、ゲームを中断させる可能性があるため、あまり頻繁に使用するべきではありません。

注：サーフェスを使用する場合、texture のメモリーに保存されているため、サーフェスがいつでも存在しなくなる可能性があります。を使用して、サーフェスが存在するかどうかを常に確認する必要があります。 surface_exists()直接参照する前に

構文です。

surface_getpixel(surface_id, x, y);

論証タイプ 商品説明
surface_id The ID of the surface.
x The x position on the surface from which to get the pixel.
y The y position on the surface from which to get the pixel.

リターンです。

Colour Constant

例

col = surface_getpixel(surf, 56, 78 );

これは、変数 "surf" でインデックスされた表面の座標 (56,78) にあるピクセルの色を返すものである。
