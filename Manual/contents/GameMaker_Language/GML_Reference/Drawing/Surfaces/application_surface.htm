アプリケーションサーフェス

このグローバルな組み込み変数は、アプリケーションサーフェスにアクセスするために使用され、サーフェス関数のいずれでも使用することができます。このサーフェイスは常時使用可能で、GameMakerによってゲームが描画される場所である。

通常のDrawイベント（Draw Begin、Draw、Draw End）は、デフォルトでこのサーフェイスに描画されます。その他のDrawイベント（Pre/Post Drawや Draw GUIなど）は、アプリケーションサーフェスには描画されません。これらのイベントがどのように動作するかの詳細については、描画イベントを参照してください。

構文です。

application_surface;

リターンです。

Surface ID

例

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

上記のコードは、アプリケーションサーフェスとGUIレイヤーの比率が1:1になるようにリサイズされます。
