application_surface_enable

この機能により、アプリケーションサーフェスの有効・無効を設定することができます。デフォルトでは、アプリケーションサーフェスは有効になっており、Drawイベントのすべての描画はそのサーフェスに行われ、サーフェスはスクリーンに描画されます。しかし、一部の古いデバイスや特定のチップセットでは、パフォーマンスが低下することがあります。そのような場合は、switch アプリケーションサーフェスへの描画をオフにすると、すべてのDrawイベントの描画は、直接スクリーンに行われますbuffer.

この機能を使って、ゲーム中の任意のタイミングでアプリケーションサーフェスの有効・無効を切り替えることができますが、変更が反映されるまでに少なくとも1フレーム（ステップ）必要なことに注意が必要です。オフにする はswitch 全てのアスペクト比補正をオフにしますは、ターゲットプラットフォームのゲームオプションで設定されているとおりです。つまり、必要なデバイス、ウィンドウ、または画面の寸法にゲームをスケーリングする何らかの方法を用意しておく必要があります。switch 再度オンにすると、次のステップでGame Optionsが再び有効になります。

この機能は、画面への描画方法に影響を与えるため、慎重に使用することをお勧めします。一般的には、ゲームのメニューにオプションとして組み込むか、ゲームの開始時に一度だけ呼び出し、有効か無効かに応じてコードを記述するのがよいでしょう。

構文です。

application_surface_enable(enable);

論証タイプ 商品説明
enable Set to enable (true) またはアプリケーションサーフェスを無効にする (false) ことができます。

リターンです。

例

if keyboard_check_pressed(vk_space)
{
    if (application_surface_is_enabled())
    {
        application_surface_enable(false);
    }
    else
    {
        application_surface_enable(true);
    }
}

上記のコードは、キーが押されたことを確認し、その状態に応じてアプリケーションサーフェスのオン/オフを切り替えます（オプションメニューのようなものです）。
