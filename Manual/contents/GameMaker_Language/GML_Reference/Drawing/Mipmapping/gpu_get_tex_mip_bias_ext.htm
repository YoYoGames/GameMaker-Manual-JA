gpu_get_tex_mip_bias_ext

この関数を使うと、指定したshader サンプラーのミップマップバイアス値を取得することができます。shader のサンプラーのインデックスを指定します（関数 shader_get_sampler_index()この関数は、偏りがない場合は 0.0 を返し、それ以上の場合は 1 が最初のミップマップ、2 が 2 番目のミップマップ、といった具合になります。これはミップマップが交換される速度を制御し、一般的に値が高いほど、また見ている「距離」が大きいほどshader textures をよりぼやけさせることになります。これは負の値も返すことができることに注意してください。その場合、shader textures は、値が低いほど、より遠くまで鮮明になります。

構文です。

gpu_get_tex_mip_bias_ext(sampler_index);

論証タイプ 商品説明
sampler_index The index of the shader sampler to get

リターンです。

(default: 0)

例

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_bias_ext(_sampleIndex) != 0)
{
    gpu_set_tex_mip_bias_ext(_sampleIndex, 0);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上記のコードは、与えられたshader texture サンプラーに対して、ミップフィルタのバイアスがまだ 0 に設定されていない場合、0 に設定します。
