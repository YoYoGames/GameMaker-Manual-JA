gpu_set_tex_mip_filter

この関数では、現在のミップフィルタモードを、サポートされている3種類のうちの1つに設定することができます。必要な mip フィルタリングモードを示す定数を与え、ポイントフィルタリング（デフォルト設定、つまりフィルタリングなし）、線形フィルタリング（これはトリリニアフィルタリングとしても知られています）、異方性フィルタリングから選択できます（下の定数表を参照してください）。

定数 商品説明
tf_point これにより、ミップマップレベル間のフィルタリングが無効になり、texture の遷移が見えることがありますが、最高のパフォーマンスを得ることができます。
tf_linear これにより、ミップマップレベル間の線形フィルタリング（これはトリリニアフィルタリングとしても知られています）が有効になり、texture の遷移が滑らかになりますが、性能に若干の影響が出ます。
tf_anisotropic これはミップマップレベル間の異方性フィルタリングを有効にし、texture の遷移品質を大幅に向上させ、他のフィルタリングモードで見えるぼやけを減らすことができますが、パフォーマンスへの影響が最も大きいです。

構文です。

gpu_set_tex_mip_filter(filter);

論証 タイプ 商品説明
filter ミップマッピングフィルタ定数 The mip filter mode to use (a constant, default: tf_point).

リターンです。

N/A

例

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

上記のコードはキーボードをチェックし、指定されたキーが押された場合、現在のミップフィルタを取得し、値を次のものに切り替え、異なるモードを循環させるものです。
