gpu_set_tex_max_mip

この関数では、現在設定されている最大ミップマップレベルを設定できます。0は最高解像度、1は1番目のミップマップ、2は2番目のミップマップなどを使用します。textures例えば、texture のページボーダーを 8px に設定し、最大ミップマップレベルを 3 に設定すると、レンダリング距離が長くてもにじみの問題が発生しないようにすることができます。

構文です。

gpu_set_tex_max_mip(maxmip);

論証タイプ 商品説明
maxmip The maximum mipmap level (default: 16)

リターンです。

例

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

上記のコードでは、現在の最大ミップマップレベルをチェックし、それが4でなければ4に設定されます。
