gpu_get_tex_min_mip

この関数では、現在使用されているミップマップの最小レベルを取得できます。値 0 は最高解像度、1 は最初のミップマップ、2 は 2 番目のミップマップを使用します。

構文です。

gpu_get_tex_min_mip();

リターンです。

(default: 0)

例

if gpu_get_tex_min_mip() != 0
{
    gpu_set_tex_min_mip(0);
}

上記のコードでは、現在の最小ミップマップレベルをチェックし、それが0でなければ0に設定されます。
