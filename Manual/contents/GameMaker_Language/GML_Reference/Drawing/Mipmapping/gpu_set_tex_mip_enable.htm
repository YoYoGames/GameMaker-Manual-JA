gpu_set_tex_mip_enable

この機能を使うと、ミップマッピングをオフにするか、すべてに対してオンにするか、テクスチャグループマネージャで選択したtexture グループに対してのみオンにするかを変更することができます。この機能には、以下の定数のいずれかが必要です。

定数 商品説明
mip_off ミップマッピングは無効です。
mip_on すべてのtextures のミップマッピングが有効です。
mip_markedonly ミップマッピングは、textures グループオプション（デフォルト）でミップマッピングを有効にしているTexture に対して有効です。

構文です。

gpu_set_tex_mip_enable(setting);

論証 タイプ 商品説明
setting ミップマッピング定数 The mipmap setting (a constant, default: mip_markedonly)

リターンです。

N/A

例

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

上記のコードでは、ミップマッピングが有効かどうかを確認し、有効でない場合は有効にします。
