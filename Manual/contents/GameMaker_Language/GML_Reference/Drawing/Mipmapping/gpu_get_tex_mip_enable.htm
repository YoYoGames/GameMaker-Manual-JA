gpu_get_tex_mip_enable

この関数は、ミップマッピングがオフになっているか、すべてに対してオンになっているか、テクスチャグループマネージャで選択したtexture グループに対してのみオンになっているかを取得することができます。この関数は、以下の定数のいずれかを返します。デフォルトの設定は mip_markedonly です。

構文です。

gpu_get_tex_mip_enable();

リターンです。

Constant

ミップマッピング定数
定数 商品説明
mip_off ミップマッピングは無効です。
mip_on すべてのtextures のミップマッピングが有効です。
mip_markedonly ミップマッピングは、textures グループオプション（デフォルト）でミップマッピングを有効にしているTexture に対して有効です。

例

if gpu_get_tex_mip_enable != mip_on
{
    gpu_set_tex_mip_enable(mip_on);
}

上記のコードでは、ミップマッピングが有効かどうかを確認し、有効でない場合は有効にします。
