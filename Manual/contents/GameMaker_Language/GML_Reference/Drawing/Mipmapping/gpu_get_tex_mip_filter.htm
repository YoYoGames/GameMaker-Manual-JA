gpu_get_tex_mip_filter

この関数を使うと、現在のミップフィルタのモードを取得することができます。この関数が返すモード値は，以下に示す定数のうちの1つです。

構文です。

gpu_get_tex_mip_filter();

リターンです。

Mipmapping Filter Constant (listed below):

ミップマッピングフィルタ定数
定数 商品説明
tf_point これはミップマップレベル間のブレンドが無効であることを意味し、目に見えるtexture の遷移が発生することがありますが、最高のパフォーマンスを得ることができます。
tf_linear これはミップマップレベル間のブレンドが有効であることを意味し（これはトリリニアフィルタリングとしても知られています）、texture の遷移を滑らかにしますが、性能に若干の打撃を与えます。
tf_anisotropic これは、異方性フィルタリングが有効であることを意味し、texture の遷移品質を大幅に向上させ、他のフィルタリングモードで見えるぼやけを減らすことができますが、パフォーマンスへの打撃は最も大きいです。

例

if keyboard_check(vk_enter)
{
    switch(gpu_get_tex_mip_filter())
    {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
    }
}

上記のコードはキーボードをチェックし、指定されたキーが押された場合、現在のミップフィルタを取得し、値を次のものに切り替え、異なるモードを循環させるものです。
