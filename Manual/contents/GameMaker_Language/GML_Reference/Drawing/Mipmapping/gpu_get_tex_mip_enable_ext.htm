gpu_get_tex_mip_enable_ext

この関数は、ミップマッピングがオフになっているか、すべてに対してオンになっているか、texture のサンプラー上のテクスチャグループマネージャで選択したshader グループに対してのみオンになっているかを取得することができます。shader サンプラーのインデックス値を指定します（関数 shader_get_sampler_index()この関数は、以下の定数のいずれかを返します。

構文です。

gpu_get_tex_mip_enable_ext(sampler_index);

論証 タイプ 商品説明
sampler_index シェーダーサンプラーハンドル The index of the shader sampler to get

リターンです。

Constant

定数 商品説明
mip_off ミップマッピングは無効です。
mip_on すべてのtextures のミップマッピングが有効です。
mip_markedonly ミップマッピングは、textures グループオプションでミップマッピングを有効にしているTexture に対して有効です。

例

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if (gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on)
{
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
}
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

上記のコードは、指定されたshader texture サンプラーのミップマッピングがまだ有効になっていなければ、それを有効にします。
