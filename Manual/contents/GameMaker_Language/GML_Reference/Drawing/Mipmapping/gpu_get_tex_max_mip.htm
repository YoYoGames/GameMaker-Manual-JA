gpu_get_tex_max_mip

この関数では、現在設定されている最大ミップマップレベルを取得できます。値 0 は最高解像度、1 は最初のミップマップ、2 は 2 番目のミップマップを使用します。textures例えば、texture のページボーダーを 8px に設定し、最大ミップマップレベルを 3 に設定すると、レンダリング距離が長くてもにじみの問題が発生しないことが確認されます。

構文です。

gpu_get_tex_max_mip();

リターンです。

(default: 16)

例

if gpu_get_tex_max_mip() != 4
{
    gpu_set_tex_max_mip(4);
}

上記のコードでは、現在の最大ミップマップレベルをチェックし、それが4でなければ4に設定されます。
