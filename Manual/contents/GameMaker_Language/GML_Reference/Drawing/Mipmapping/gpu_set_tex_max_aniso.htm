gpu_set_tex_max_aniso

この機能により、 tf_anisotropic フィルタモード使用時の異方性の現在の最大レベルを設定することができます ( gpu_get_tex_mip_filter()を参照してください)。入力値は、1～16の範囲で設定してください。

構文です。

gpu_set_tex_max_aniso(maxaniso);

論証タイプ 商品説明
maxaniso The maximum level for anisotropic filtering (default: 16)

リターンです。

例

if gpu_get_tex_max_aniso() != 8
{
    gpu_set_tex_max_aniso(8);
}

上記のコードは、現在の最大異方性フィルタリングレベルをチェックし、それが8でない場合は8に設定されます。
