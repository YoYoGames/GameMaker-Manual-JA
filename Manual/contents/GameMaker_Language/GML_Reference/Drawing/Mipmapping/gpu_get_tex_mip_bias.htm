gpu_get_tex_mip_bias

この関数では、ミップマップバイアス値を取得することができます。値 0 はバイアスなし、1 は最初のミップマップ、2 は 2 番目のミップマップ、といった具合です。これはミップマップが交換される速度を制御し、一般に、値が高く、表示される「距離」が大きいほど、レンダーtextures をよりぼやけさせることになります。これは負の値も返すことができることに注意してください。この場合、レンダリングされたtextures は、値が低いほど、より遠くまで鮮明になります。

構文です。

gpu_get_tex_mip_bias();

リターンです。

(default: 0)

例

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

上記のコードは、現在のミップマップバイアスをチェックし、それが0でなければ0に設定されます。
