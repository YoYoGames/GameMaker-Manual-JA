gpu_set_tex_mip_bias

この関数では、ミップマップバイアス値を設定できます。0はバイアスなし、1は1番目のミップマップ、2は2番目のミップマップ...と等しくなります。これはミップマップが交換される速度を制御し、一般に、値が高く、表示される「距離」が大きいほど、レンダーtextures をよりぼやけさせます。この関数は負の値も取れることに注意してください。その場合、レンダリングされたtextures は、値が低いほど、より遠くまで鮮明になります。

構文です。

gpu_set_tex_mip_bias(bias);

論証タイプ 商品説明
bias The mipmap bias value to use (default: 0)

リターンです。

例

if gpu_get_tex_mip_bias() != 0
{
    gpu_set_tex_mip_bias(0);
}

上記のコードは、現在のミップマップバイアスをチェックし、それが0でなければ0に設定されます。
