texture_get_texel_height

テクセル（テクスチャ要素）は、コンピュータグラフィックスで使われるtexture 空間の基本単位です。Textures は、画像がピクセルの配列で表されるように、テクセルの配列で表されます。この関数は、使用した画像texture のasset ページから一つのテクセル高を返すものです。

構文です。

texture_get_texel_height(tex);

論証 タイプ 商品説明
tex テクスチャポインタ The texture page asset pointer to use

リターンです。

Real

例

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

上記のコードは、texture sprite から取得したasset のテキスルの幅と高さを取得するものです。
