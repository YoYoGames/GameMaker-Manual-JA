テクスチャデバッグメッセージ

この関数は、texture のデバッグメッセージを有効または無効にするために使用します。有効にすると ( true に設定)、texture ページの使用に関する追加情報が出力ウィンドウに送信されます。この出力を再び無効にするには、 false に設定します。

構文です。

texture_debug_messages(enable);

論証タイプ 商品説明
enable Enable or disable the texture debug messages

リターンです。

例

if global.debug
{
    texture_debug_messages(true);
}

上記のコードでは、グローバル変数の値をチェックし、それが true に設定されている場合、texture のデバッグメッセージが有効になります。
