テクスチャ

以下の関数はすべて「テクスチャ」に関するものです。つまり、VRAMに保存され、ゲーム内でsprites などとして使用される画像です。最初の関数は、特定の画像に関する情報を取得し、テクスチャページでの位置やサイズを設定し、texture の機能を設定するために使用されます。

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • texture_global_scale

また、GameMakerには、個々のtextures やtexture グループ全体の VRAM を管理する機能があります。

  • draw_texture_flush
  • texture_prefetch
  • texture_flush
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

最後に、主にプロジェクトのデバッグに使用し、texture メモリの効率的な使用を保証するための以下の関数があります。

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
