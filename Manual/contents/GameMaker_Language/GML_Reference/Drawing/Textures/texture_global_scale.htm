texture_global_scale

この関数は、最終的なゲーム実行ファイルの一部である WAD ファイルからロードする際のtexture ページのスケーリングを制御することができます。入力値は、2のべき乗の値でなければならず、次のように動作する。

  • texture ページが 1:1 になるように設定するには、次のようにします。 texture_global_scale(1)
  • ハーフtextures を常時使用する場合 texture_global_scale(2)
  • 四分の一にするには、 texture_global_scale(4), etc... を使用します。

こうすることで、メモリ不足の問題があるプラットフォームでtexture ページのメモリ使用量をよりよく制御することができます。textures ページは WAD からオンデマンドで作成されるので、ゲーム開始時にこれを呼び出してすべてのtextures をスケーリングして読み込むこともできます。 特定のタイミングで呼び出して、特定のtextures をロードすることができます。 draw_texture_flush()この関数を呼び出す前に、プリフェッチ関数を使用して、正しい texture ページをメモリに格納します。

構文です。

texture_global_scale(pow2integer);

論証タイプ 商品説明
pow2integer The scale factor to use (1, no scale, 2, half scale, 4, quarter scale, etc...)

リターンです。

例

draw_texture_flush();
texture_global_scale(2);
sprite_prefetch(spr_Trees);

上記のコードは、メモリからすべてのtextures をフラッシュし、次にtexture のスケーリングを 2 に設定し (従ってtexture ページは半分のサイズ)、最後にtexture の "spr_Trees" が含まれるsprite ページをプリフェッチします。
