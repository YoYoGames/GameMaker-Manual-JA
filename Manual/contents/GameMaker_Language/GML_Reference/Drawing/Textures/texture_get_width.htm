texture_get_width

指定された ID のtexture の幅を返します。これは常に 0 から 1 の範囲の値です。これは、textures をモデルやプリミティブにマッピングするときに使用できます。

構文です。

texture_get_width(tex);

論証 タイプ 商品説明
tex テクスチャポインタ The texture page asset pointer to use

リターンです。

Real

例

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

上記のコードは、texture sprite から取得したasset の幅を取得します。
