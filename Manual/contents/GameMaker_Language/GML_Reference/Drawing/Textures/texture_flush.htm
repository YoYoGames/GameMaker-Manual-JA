テクスチャーフラッシュ

この関数は、texture ページまたはtexture ページのグループを「フラッシュ」する、つまり、不要になったページを VRAM から削除するために使用できます。単一のページをフラッシュするには、（ texturegroup_* 関数によって返されるような）固有のテクスチャページ IDを与えるか、またはグループ内のすべてのtexture ページをフラッシュするには、（テクスチャグループ エディタで定義されるような）テクスチャグループ名を与えることができ ます。

構文です。

texture_flush(tex_id);

論証 タイプ 商品説明
tex_id テクスチャポインタまたは文字列 The texture page pointer or a texture group name (a string)

リターンです。

N/A

例

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_flush(_tex_array[i]);
}

上記のコードは、texture グループ "MainMenu" の下にあるすべてのtexture ページをフラッシュします。
