テクスチャグループ_get_textures

この関数は、texture グループを構成する個々のページのテクスチャページ IDを取得するために使用することができます。texture グループ IDstring (Texture Group Editor で定義) を与えると、この関数は 1 次元配列を返し、配列の各エントリは単一のtexture ページ ID となります。この関数が失敗した場合、つまり、無効なグループが指定された場合や、グループにtexture が割り当てられていない場合は、配列は空 (長さ 0) になります。

構文です。

texturegroup_get_textures(tex_id);

論証 タイプ 商品説明
tex_id 文字列 The name of the texture group to check (a string)

リターンです。

Array of Texture Pointers

例

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

上記のコードは、texture グループ "MainMenu" のtexture ページ ID を取得し、それらが解凍されているかどうかをチェックし、解凍されていれば VRAM に配置します。
