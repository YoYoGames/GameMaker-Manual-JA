draw_texture_flush

この機能により、ビデオメモリからすべてのtextures を削除することができ、その後、初回使用時に再読み込みされます。これは、ゲームに複数のtexture ページがある場合に、ビデオ メモリを管理する唯一の効果的な方法です。また、texture メモリをフラッシュする必要があります。 ゲームのレベル間で、texture グループ機能を使ってグラフィックを整理し、最小数のtextures が使用されるようにします。

texture すべてのターゲットにおいて、デフォルトでは sprites ページは必要なときだけロードされることに留意する必要があります。 を使用して、この関数と同様に管理することができます。

構文です。

draw_texture_flush();

リターンです。

例

draw_texture_flush();

上記のコードは、texture のデータをビデオメモリにフラッシュするもので、おそらくobject に配置されたroom の最初のインスタンスの create イベント内に配置されるでしょう。
