texture_get_height

指定された ID のtexture の高さを返します。これは常に 0 - 1 の範囲の値です。これは、textures をモデルやプリミティブにマッピングするときに使用できます。

構文です。

texture_get_height(tex);

論証 タイプ 商品説明
tex テクスチャポインタ The texture page asset pointer to use

リターンです。

Real

例

tex_h = texture_get_height(surface_get_texture(global.Surf));

上記のコードでは、あらかじめ作成したサーフェスから取得したtexture の高さを取得します。
