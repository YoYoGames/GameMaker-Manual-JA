テクスチャ

この関数は、特定のtexture ページが解凍されて使用可能かどうか、またはtexture ページのグループが解凍されて使用可能かどうかを確認するために使用することができます。一意のテクスチャページID（ texturegroup_* 関数によって返される）またはtexture グループIDstring （Texture グループエディタで定義される）を与えると、関数はそれらが解凍されている場合は true を、そうでない場合は false を返します。

構文です。

texture_is_ready(tex_id);

論証 タイプ 商品説明
tex_id テクスチャポインタまたは文字列 The texture page pointer or a texture group name (a string)

リターンです。

Boolean

例

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    if !texture_is_ready(_tex_array[i])
    {
        texture_prefetch(_tex_array[i]);
    }
}

texture 上記のコードは、texture グループ "MainMenu" のページ ID を取得し、それらがアンパックされているかどうかをチェックし、アンパックされていない場合は VRAM にプリフェッチされることになります。
