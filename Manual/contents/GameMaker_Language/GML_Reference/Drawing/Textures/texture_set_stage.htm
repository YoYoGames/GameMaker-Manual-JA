テクスチャーセットステージ

この関数は、与えられたステージ "スロット "をtexture に設定します。利用可能なステージ「スロット」の数は、コンパイルするプラットフォームによって異なります。Windows 、Mac、Linux では最大8個利用可能ですが、ローエンドのAndroid デバイス（例）では、この数は2個までとなります。また、最初のステージスロット（1）は常にGameMaker によって自動的に使用されることは特筆すべきことです。

注：この関数は、実行中のshader のコンテキスト以外では何も行いません !詳しくはシェーダーを 参照してください。

構文です。

texture_set_stage(stage, tex);

論証 タイプ 商品説明
stage リアル The texture "slot" to use.
tex テクスチャポインタ The texture to use.

リターンです。

N/A

例

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上記のコードでは、"GlassShader "というインデックスを持つshader 内のサンプラーのハンドルを取得し、そのshader 定数を与えられたsprite texture に設定します。
