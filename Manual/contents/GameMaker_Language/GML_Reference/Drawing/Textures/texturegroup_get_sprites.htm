テクスチャグループ名 get_sprites

sprite この関数を使うと、与えられたsprites グループ内のtexture ページに割り当てられたtexture のそれぞれのインデックスを取得することができます。texture グループ IDstring (texture グループエディタで定義) を与えると、この関数は 1 次元配列を返し、各エントリにsprite リソースのsprite インデックスが含まれます。この関数が失敗した場合、つまり、無効なグループが指定された場合や、グループにtexture が割り当てられていない場合は、配列は空 (0 length) になります。

構文です。

texturegroup_get_sprites(tex_id);

論証 タイプ 商品説明
tex 文字列 The name of the texture group to check (a string)

リターンです。

Array

例

var _tex_array = texturegroup_get_sprites( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    show_debug_message("Sprite " + string(i) + " Index:" + string(tex_array[i]));
}

上記のコードは、sprite グループ "MainMenu" のtexture インデックスを取得し、その ID をコンソール出力ウィンドウに表示するものです。
