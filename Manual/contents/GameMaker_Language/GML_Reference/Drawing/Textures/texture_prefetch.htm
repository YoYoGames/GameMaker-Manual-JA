テクスチャープリフェッチ

この関数は、texture ページまたはtexture ページのグループを「プリフェッチ」する、つまり、必要なときに VRAM にロードするために使用することができます。単一のページをプリフェッチするには、（ texturegroup_* 関数によって返される）固有のテクスチャページ IDを供給するか、またはグループ内のすべてのtexture ページをプリフェッチするには、（Texture Group Editor で定義される）テクスチャグループ名を供給することができ ます。

構文です。

texture_prefetch(tex_id);

論証 タイプ 商品説明
tex_id テクスチャポインタまたは文字列 The texture page pointer or a texture group name (a string)

リターンです。

N/A

例

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
    texture_prefetch(_tex_array[i]);
}

上記のコードでは、texture グループ "MainMenu" の下にあるすべてのtexture ページをプリフェッチします。
