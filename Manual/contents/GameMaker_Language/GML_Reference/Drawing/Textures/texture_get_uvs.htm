テクスチャ・ゲット・ユーブス

この関数は、与えられたtexture ポインタの UV 座標を表す 4 つの要素を持つ 1 次元配列を返し、配列に以下の値を記入します。

  • [0] = 左
  • [1] = トップ
  • [2] = 右
  • [3] = 下
  • [4] =asset コンパイラがsprites の左側から切り捨てたピクセル数 (sprite assets のみ)
  • [5] =asset コンパイラがsprites 上面から切り捨てたピクセル数 (sprite assets のみ)
  • [6] =sprites ページに保存された元のtexture 幅のピクセルデータの正規化されたパーセンテージ (sprite assets のみ )
  • [7] =sprites ページに保存された元のtexture の高さからのピクセルデータの正規化されたパーセンテージ (sprite assets のみ)

この値は、その後、他の描画関数、特に一般的な3D関数や一部の2Dプリミティブ関数、Shader 関数で使用することができます。もし、sprite の UVS が必要なら、 sprite_get_uvs() を、font の場合は , font_get_uvs().

構文です。

texture_get_uvs(texid)

論証 タイプ 商品説明
texid テクスチャー The texture pointer to get the UVS for

リターンです。

Array (4 - 8 elements)

例

var _tex = surface_get_texture(surf_back);
var _uvs = texture_get_uvs(tex);
var _uvs_left = _uvs[0];
var _uvs_top = _uvs[1];
var _uvs_right = _uvs[2];
var _uvs_bottom = _uvs[3];

上記のコードでは、まずsurf_back に格納されているサーフェイスのtexture を取得し、そのtexture の UV 座標を取得する。そして、返された配列から左、上、右、下のUV座標を取得し、ローカル変数に格納します。
