draw_tilemap

この関数は，与えられたタイルマップをroom の任意の場所に描画するために用いることができる． タイルマップ要素ID（これは layer_tilemap_create()を使用したとき、あるいは関数 layer_tilemap_get_id()) を指定し、次に描画する x/y 位置を指定します (room)。これは単に、関数を呼び出しているインスタンスのレイヤー深度を使用して、指定されたポイントにタイルマップを描画することに注意してください。これはタイルマップを移動せず、いかなる方法でも変更せず、またタイルマップが可視としてフラグされているかどうかは関係ない。

構文です。

draw_tilemap(tilemap_element_id, x, y);

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tilemap element to draw
x リアル The x position within the room to draw at
y リアル The y position within the room to draw at

リターンです。

N/A

例

var lay_id = layer_get_id("Tiles_Effects");
var map_id = layer_tilemap_get_id(lay_id);
draw_tilemap(map_id, mouse_x, mouse_y);

上記のコードでは、レイヤーIDを取得し、それを使って特定のタイルマップIDを取得し、それを使ってマウスの位置にタイルマップを描画しています。
