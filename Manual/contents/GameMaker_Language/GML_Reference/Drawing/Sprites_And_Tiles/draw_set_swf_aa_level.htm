draw_set_swf_aa_level

この関数は、SWF 形式のベクトルsprites のアンチエイリアス（AA）レベルを設定するために使用することができます。これは 0 から 1 までの実数値で、これらのsprites のエッジを "滑らかに "します。この効果を見るためには、まず関数 draw_enable_swf_aa().

構文です。

draw_set_swf_aa_level(AA);

論証タイプ 商品説明
AA The anti-aliasing value to use from 0 to 1.

リターンです。

例

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上記のコードは、SWFフォーマットsprites のAA値をチェックし、それが0であればAAを有効にして、値を0.5に設定します。
