draw_get_swf_aa_level

この関数は、SWF 形式のベクトルsprites のアンチエイリアス (AA) レベルを取得するために使用できます。戻り値は 0 から 1 の間で、これらのsprites のエッジがどれだけ「滑らかに」描画されるかを示します。AA レベルを設定するには、関数 draw_set_swf_aa_level().

構文です。

draw_get_swf_aa_level();

リターンです。

例

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上記のコードは、SWFフォーマットsprites のAA値をチェックし、それが0であればAAを有効にして、値を0.5に設定します。
