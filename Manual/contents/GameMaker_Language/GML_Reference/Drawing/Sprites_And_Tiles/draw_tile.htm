ドロータイル

この関数を使うと、tile から任意のtile set を描画し、回転、反転、ミラーを競い、フレーム（アニメーションの場合）とroom 内の位置を設定できます。tile set ID 値はtile set で設定したIDE インデックス、tile data はtile インデックスと適用した変換に関するすべての情報を含むデータ「blob」です。関数 tilemap_get()を使ってtile データを取得し、タイル関数で操作することもできます。また、以下のtile 定数と、作成するtile のインデックス値を使って、自分でデータを作成することもできます。

定数 商品説明
tile_rotate tile データブロブのローテートビットを設定/取得するために使用します。
tile_mirror tile データブロブのミラービットを設定/取得するために使用されます。
tile_flip tile データブロブのフリップビットを設定/取得するために使用されます。
tile_index_mask tile のインデックスを抽出するためにtile データブロブと "and "演算するための特別な定数です。


これらの定数とtile のインデックス値をビット単位で "or "することで、最終的なタイルデータを得ることができます。

var tiledata = 8 | tile_rotate | tile_flip;

上記のコードでは、tile のタイルデータを格納するローカル変数を作成し、tile set のインデックスを 8 とし、それを回転、反転させているだけです。これは、指定したtile から、そのインデックスでtile set を描画するために使用できます。これは、どのタイルマップレイヤーにもリンクされておらず、tile は描画を行うインスタンスのレイヤー深度で、room で指定した x/y 位置に描画されます。 また、フレームを設定することで、描画もできることに注意してください。インデックス付きtile がアニメーションセットの一部である場合、ここでフレーム番号を設定できます (0 からフレーム数 - 1 まで)。インデックス付きtile は単に、どのアニメーションtile set を使用しているかの指標となり、描画されるのは指定されたフレームになります。もしtile がアニメーションセットの一部でないなら、frame は 0 に設定されるべきです。

構文です。

draw_tile(tileset, tiledata, frame, x, y)

論証 タイプ 商品説明
tileset タイルセットアセット The ID of the tile set to use
tiledata タイルデータ The tile data to use
frame リアル The frame number for the animated tile to use (default 0 for non-animated tiles)
x リアル The x position within the room to draw at
y リアル The y position within the room to draw at

リターンです。

Real

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

上記のコードは、与えられたタイルマップからtile のデータを取得し、"flip"flag を設定し、このデータを使ってインスタンス位置で反転したtile を描画するものです。
