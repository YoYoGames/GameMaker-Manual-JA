draw_enable_swf_aa

この機能を使用すると、SWF 形式のベクターsprites のアンチエイリアス (AA) を有効または無効にすることができます。AA は、ベクターイメージのエッジを単に滑らかにし、見た目を良くします。使用される AAの量は、この関数を使用して設定された値に依存します。 draw_set_swf_aa_level().デフォルトでは、これは無効になっています。

構文です。

draw_enable_swf_aa(enable);

論証タイプ 商品説明
enable Enable (true) または無効にする (false) すべての SWFsprites の AA。

リターンです。

例

if draw_get_swf_aa_level() == 0
{
    draw_enable_swf_aa(true);
    draw_set_swf_aa_level(0.5);
}

上記のコードは、SWFフォーマットsprites のAA値をチェックし、それが0であればAAを有効にして、値を0.5に設定します。
