draw_sprite_pos

この関数を使うと、四隅の座標で定義された領域上に歪んだsprite を描画することができます。最初の2つの引数は，描画するsprite と，そのサブイメージであるsprite （関数 draw_sprite()) しかし，次のものは，与えられたsprite のバウンディングボックスの4つの角のそれぞれの位置を定義するものです．これらの座標は時計回りの順序で与える必要があり，最初の座標は左上，次に右上，そして下 右、最後に左下です。また、sprite のアルファ値に値を設定することで、透明性を持たせて描画することも可能です。下の画像は、この機能がどのように働くかを示しています。

警告!この関数をsprite で使用すると、texture がプリミティブの四角形から構成されているため、異なる結果が得られ、sprite の「せん断」が発生する可能性があります。

注：この機能はビットマップ sprites に対してのみ有効であり、SWF や JSON (Spine)sprites では動作しません。

注：この機能を使用する場合、texture のページの自動切り抜き設定を無効にしておく必要があります。

構文です。

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

論証タイプ 商品説明
sprite The index of the sprite to draw.
subimg The subimg (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x1 The first x coordinate.
y1 The first y coordinate.
x2 The second x coordinate.
y2 The second y coordinate.
x3 The third x coordinate.
y3 The third y coordinate.
x4 The fourth x coordinate.
y4 The fourth y coordinate.
alpha The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

リターンです。

例

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

上記のコードは、コードを実行しているインスタンスに関連するsprite を、インスタンスの x / y 位置を中心に歪ませ、完全に不透明なアルファ値で描画します。
