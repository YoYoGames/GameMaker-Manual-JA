draw_sprite

この関数は、与えられたsprite と副画像をゲーム内のある位置に描画しますroom 。sprite には、インスタンス変数 sprite_indexを使用して、コードを実行しているインスタンスに割り当てられている現在のsprite を取得することもできますし、他の任意のsprite asset を使用することもできます。サブイメージも同様に、インスタンス変数に設定します。 image_indexこれはサブイメージを現在のインスタンスで選択されたものに設定しますsprite (別のsprite を描いても、現在のインスタンスのサブイメージ値を使用することに注意してください) または、選択したsprite の特定のサブイメージを描くために、これに他の値を使用することもできます。この値がサブイメージの数よりも大きい場合、GameMakerは自動的にloop を指定して対応する画像を選択します (たとえば、描画中のsprite に 0 から 4 までの 5 つのサブイメージがあり、インデックス値を 7 に設定すると、この関数は 0 番のサブイメージ 3 を描画することになります)。最後に、x と y の位置は、room 内でsprite が描画される位置で、sprite のx オフセットと y オフセットを中心とした位置である。

注意この関数は、スケルトンアニメーションsprites を使用する場合、期待通りに動作しないことがあります。また、この関数はデフォルトポーズの最初のフレームしか描画しないことがあるようです。代わりに draw_skeleton_* の関数を使用する必要があります。

構文です。

draw_sprite(sprite, subimg, x, y);

論証 タイプ 商品説明
sprite スプライトアセット The index of the sprite to draw.
subimg リアル The sub-image (frame) of the sprite to draw (image_index or -1 correlate to the current frame of animation in the object).
x リアル The x coordinate of where to draw the sprite.
y リアル The y coordinate of where to draw the sprite.

リターンです。

N/A

例

draw_sprite(sprite_index, image_index, x, y);
draw_sprite(spr_Halo, 0, x, y-32);

これは、sprite (sprite_index) に割り当てられたインスタンスを、room 内のインスタンスの x と y の位置に現在のサブイメージで描画します。 次に、"spr_Halo" でインデックスされたsprite の最初のフレームを、同じ x と y 位置に、32ピクセル「上」を描画します。
