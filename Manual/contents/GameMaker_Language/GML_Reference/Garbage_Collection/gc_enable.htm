gc_enable

この関数で、ガベージコレクタを有効または無効にすることができます。引数に true を指定してこの関数を呼び出すと有効になり、 false を指定すると無効になります（推奨しません）。デフォルトでは有効になっています。

構文です。

gc_enable(enable);

論証タイプ 商品説明
enable enable (true) またはガベージコレクタを無効化 (false) します。

リターンです。

例

if (global.debug == true)
{
    gc_enable(false);
}

上記のコードでは、与えられたグローバル変数が true の場合、ガベージコレクションを無効にします。
