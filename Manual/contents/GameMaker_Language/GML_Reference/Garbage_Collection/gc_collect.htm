gc_collect

この関数により、ガベージコレクタをトリガーし、現在のフレーム（ステップ）の終了時に強制的に実行させることができます。注目すべきは、ガベージコレクタがアクティブでなくても 動作することです。ガベージコレクタを無効にした後にこの関数を呼び出すと（関数 gc_enable() を使用）、1 フレームだけガベージコレクタが有効になり、ガベージコレクタが再び無効になる前に、収集フラグが立てられたすべてのobjects がメモリから削除されます。

構文です。

gc_collect();

リターンです。

例

if (global.debug == true && keyboard_check_pressed(vk_f1))
{
    gc_collect();
}

上記のコードでは、グローバル変数と押されたキーをチェックし、それらが true であれば、フレーム（ステップ）の終わりに向けてガベージコレクションが発動されます。
