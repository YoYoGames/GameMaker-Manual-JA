weak_ref_any_alive

この関数を使用すると、さまざまな構造体の弱い参照をチェックして、それらがまだ「生きて」いるかどうかを確認することができます。チェックする弱参照の配列を指定します (それぞれの弱参照は、この関数を使用して作成されている必要があります)。 weak_ref_create()の場合、この関数は true を返します。 いずれも の構造体がまだどこかで参照されている場合は、 false 、そうでなく、ガベージコレクションされた場合は、 。もし，値のどれかが弱参照でない配列を指定した場合，この関数は undefined を返すことに注意してください．

最初の引数には，チェックする配列の最初のインデックスを指定し，さらにそのインデックスから何番目かを指定するための引数を指定します。この引数が指定された場合、これらの引数で指定された配列のインデックスのみがチェックされます。

構文です。

weak_ref_any_alive(array, [index], [length]);

論証 タイプ 商品説明
array 構造体弱参照配列 Array containing weak references to the structs that you want to check.
[index] リアル OPTIONAL チェックを開始する配列のインデックス。
[length] リアル OPTIONAL をチェックするために、与えられたインデックスから始まるポジションの数を指定します。

リターンです。

Boolean (or undefined)

例

if weak_ref_any_alive(inventory_ref_array)
{
    instance_destroy(obj_Inventory_Control);
}

上記のコードでは、弱参照の配列をチェックし、まだ生きているものがあれば、そのインスタンスを破棄しています。
