gc_is_enabled

この関数を使うと、ガベージコレクタが有効かどうかを確認することができます。この関数は、有効であれば true を、そうでなければ false を返します。

構文です。

gc_is_enabled();

リターンです。

例

if (!gc_is_enabled())
{
    gc_enable(true);
}

上記のコードでは、ガベージコレクタが有効かどうかを確認し、有効でない場合は有効にしています。
