gc_target_frame_time

この関数を使うと、ガベージコレクタに対して、各フレームの実行に指定された時間以上を費やさないようにすることを指示することができます。この関数は、マイクロ秒（1,000,000 マイクロ秒は 1 秒に相当）で指定された時間値を受け取り、デフォルトのターゲットフレーム時間は 100 マイクロ秒です。

なお、これは単なる 一部の処理は1フレームで完全に実行される必要があり、目標時間よりも長くかかる場合があるからです。また、ターゲット時間を増やすと、ガベージコレクタはメモリ使用量の急激な変化に対応できるようになり、メモリがより迅速に解放されるようになりますが、実際にはほとんどの場合、このようなことは必要ないと思われることに注意してください。また、ターゲットフレーム時間を 0 に設定しても 、コレクターは完全に停止するわけではなく、各フレームで最小限の作業を行うことになります。コレクタを完全に停止させるには、 gc_enable() 関数を使用します。

構文です。

gc_target_frame_time(time);

論証タイプ 商品説明
time The target time - in microseconds -  that the garbage collector should work each frame

リターンです。

N/A

例

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上記のコードは、ガベージコレクタの現在のフレーム時間目標をチェックし、それが50マイクロ秒でない場合は、この値に設定されます。
