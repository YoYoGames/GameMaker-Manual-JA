weak_ref_create

この関数で、struct への弱い参照を作成し、struct がゲーム内でまだ「生きている」（参照されている）かどうかをチェックするために使用することができます。追跡したいstruct への参照を指定すると、この関数はそのstruct への弱い参照である別のstruct を返します。強い参照は "struct" またはstruct を生成したコンストラクタ関数の名前を返し、弱い参照は "weakref" を返しますので、関数 instanceof() を使って参照が「強い」か「弱い」かをチェックすることができます。また、弱い参照であるstruct は変数 "ref" を持ち、これにアクセスすることで当該struct への強い参照を得ることができます。ただし、ガベージコレクションされている場合は undefined を返すことに注意してください。

構文です。

weak_ref_create(struct_to_track);

論証 タイプ 商品説明
struct_to_track 構造 The struct that you want to create a weak reference for

リターンです。

Struct Weak Reference

例

inventory_ref = weak_ref_create(inventory);

上記のコードでは、struct への弱い参照を作成し、後で使用するためにインスタンス変数に保存しています。
