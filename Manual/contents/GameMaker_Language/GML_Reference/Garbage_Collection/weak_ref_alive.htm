weak_ref_alive

この関数を使うと、struct への弱い参照がまだ「生きて」いるかどうかをチェックすることができます。チェックする弱参照を指定します (関数 weak_ref_create()) で、この関数はstruct がまだどこかで参照されている場合は true を、そうでなくガベージコレクションされた場合は false を返します。弱参照でない値を指定した場合，この関数は undefined を返すことに注意してください．

構文です。

weak_ref_alive(weak_ref);

論証 タイプ 商品説明
weak_ref 構造体弱めのリファレンス The weak reference to the struct you want to check.

リターンです。

Boolean (or undefined)

例

if weak_ref_alive(inventory_ref)
{
    inventory = -1;
}

上記のコードはstruct への弱い参照をチェックし、それがまだ生きていれば、それを参照している変数を -1 に設定し、struct の参照を解除し、ガベージコレクションを可能にするものです。
