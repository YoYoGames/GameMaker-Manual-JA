gc_get_target_frame_time

この関数を使うと、ガベージコレクタの現在のターゲットフレーム値を取得することができます。返される値はマイクロ秒単位です。(1,000,000マイクロ秒は1秒に相当)、ターゲットフレーム時間のデフォルトは100マイクロ秒です。この値を変更したい場合は、関数gc_target_frame_time() を使用する必要があります。

構文です。

gc_get_target_frame_time(time);

リターンです。

例

if (gc_get_target_frame_time() != 50)
{
    gc_target_frame_time(50);
}

上記のコードは、ガベージコレクタの現在のフレーム時間ターゲットをチェックし、それが50マイクロ秒でなければ、この値に設定されます。
