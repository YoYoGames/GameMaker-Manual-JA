gc_get_stats

この関数を使うと、ガベージコレクタの現在の状態に関する情報を取得することができます。この関数は、以下のメンバ変数を持つ構造体を返します。（ここでいう「オブジェクト」とは、ガベージコレクションが可能なものであり、object Browser で定義されている一般的なAsset インスタンスではないことに注意してください）。

GC統計構造
バリアブル タイプ 商品説明
objects_touched リアル これは、ガベージコレクタが前フレームで見つけたアクティブなobjects の数です。これは、どの世代が収集されたかによって異なります。
objects_collected リアル ガベージコレクタが前のフレームでアクティブでないと判断し、したがって削除することができたobjects の数。
traversal_time リアル これは、ガベージコレクタがどのobjects がアクティブであるかを把握するのに要した時間（マイクロ秒単位、メインスレッド上）です。
collection_time リアル これは、ガベージコレクタが非アクティブとみなされたobjects をクリーンアップするために要した時間（マイクロ秒単位、別スレッド）です。
gc_frame リアル これは、ガベージコレクションのパスが発生するたびに増分されるカウンタである。ガベージコレクションが無効の場合、これは増加しない。
generation_collected リアル 最後に収集された世代のインデックスです。0が最も若い世代で、3が現在最も古い世代です。
num_generations リアル ガベージコレクションの総世代数です。
num_objects_in_generation アレイの リアルs これは、各世代におけるobjects の数を含む配列（サイズ num_generations ）である。

注：HTML5ターゲットプラットフォームでは、ガベージコレクションはJavaScriptエンジンが処理するため、この関数をそのプラットフォームで使用した場合、上記のstruct のメンバー変数はすべて0を返します。

この機能を使用する場合、objects の表示情報は、全世代の処理が終了してから更新されますので、フレームタイムの設定により数フレームかかる場合がありますのでご注意ください（フレームタイムについてはこちらをご覧ください）。

構文です。

gc_get_stats();

リターンです。

GC Stats Struct

例

if (global.debug == true)
{
    var _s = gc_get_stats();
    var _t = _s.traversal_time;
    var _c = _s.collection_time;
    show_debug_message("Traversal time = " + string(_t))
    show_debug_message("Collection time = " + string(_c))
}

上記のコードは、グローバル変数をチェックし、 true 、ガラバージコレクターから情報を取得し、デバッグメッセージとしてコンソールに出力しています。
