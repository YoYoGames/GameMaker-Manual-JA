ルームハイト

この変数は、現在のroom の高さをピクセル単位で保持する。この変数を変更することで、いつでもroom の高さを変更することができます。原点は左上隅とみなされるため、変更はroom の下部に適用されます。つまり、例えばroom の高さが 480px で、これを 640px に設定すると、room は下に 180px 追加されて展開されることになります。

構文です。

room_height;

リターンです。

例

if bbox_bottom > room_height
{
    y += room_height - bbox_bottom;
}

上記のコードは、現在のインスタンスのsprite のバウンディングボックスがroom の高さより大きいかどうかを確認し、大きい場合はインスタンスを上に移動させます。
