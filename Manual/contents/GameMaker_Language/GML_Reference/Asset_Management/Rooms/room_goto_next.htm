ルームゴトー

この関数を使用すると、ゲームがコンパイルされた時点でルームマネージャにリストされている次のゲームに移動するようにすることができます。このroom が存在しない場合、エラーが発生し、ゲームは強制終了されます。room は、この関数が呼び出されたイベントが終了するまで変更されないので、この関数が呼び出された後のコードは、同じイベント内であれば、まだ実行されることに注意してください。

構文です。

room_goto_next();

リターンです。

例

if room_exists(room_next(room))
{
    room_goto_next();
}

上記のコードは、現在のコードの後に別のroom があるかどうかを確認し、ある場合はそのroom に移動します。
