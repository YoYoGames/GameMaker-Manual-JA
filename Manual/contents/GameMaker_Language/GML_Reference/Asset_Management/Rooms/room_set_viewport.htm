room_set_viewport

この関数を使用すると、現在のものを除く、ゲーム内の任意のroom のビューポートプロパティを設定することができます。ビューポートを設定するroom インデックス、ビューポートインデックス（0〜7）、そしてビューポートをroom の開始時に表示するかどうか（表示する場合は true に、それ以外は false にセット）を指定します。その後、xとyの座標（ビューポートの左上の位置に相当）、そして幅と高さをピクセル単位で設定します。

構文です。

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

論証 タイプ 商品説明
rm ルームアセット The index of the room to set
vind リアル The index of the view port to set
vis ブーリアン The visibility of the view port (true は可視、 は不可視) false
xport リアル The x position for the view port in the room
yport リアル The y position of the view port in the room
wport リアル The width (in pixels) of the view port
hport リアル The height (in pixels) of the view port

リターンです。

N/A

例

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

これは、変数 "global.myroom" でインデックスされたroom のビューポート [0] プロパティを設定するものです。
