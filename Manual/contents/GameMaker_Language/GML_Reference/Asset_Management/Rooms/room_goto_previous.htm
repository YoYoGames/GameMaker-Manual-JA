この関数を使うと、ゲームがコンパイルされた時点でルームマネージャにリストされている、前のゲームに移動するようにすることができます。このroom が存在しない場合、エラーがスローされ、ゲームは強制終了されます。room は、この関数が呼び出されたイベントが終了するまで変更されないので、この関数が呼び出された後のコードは、同じイベント内であれば、まだ実行されることに注意してください。この関数は、Room Endイベントもトリガーします。

構文です。

room_goto_previous();

リターンです。

例

if room_exists(room_previous(room))
{
    room_goto_previous();
}

上記のコードは、現在のコードの前に別のroom があるかどうかを確認し、ある場合はそのroom に移動します。
