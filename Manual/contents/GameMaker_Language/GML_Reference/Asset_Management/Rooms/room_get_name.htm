部屋名

この関数は、指定されたroom の名前をstring として返すことができます。 これはあくまで string であり、room を直接参照することはできませんので、ご注意ください。しかし、このstring を使って、返されたstring と関数を使用して部屋のインデックスを取得することはできます。 asset_get_index().

構文です。

room_get_name(index);

論証タイプ 商品説明
index The index of the room to check the name of.

リターンです。

例

var roomname = room_get_name(room);

draw_text(32, 32, _roomname);

上記のコードは、現在のroom の名前を取得し、画面に描画します。
