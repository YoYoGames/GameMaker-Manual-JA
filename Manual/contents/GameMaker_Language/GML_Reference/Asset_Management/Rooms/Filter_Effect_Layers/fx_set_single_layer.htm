fx_set_single_layer

フィルター/エフェクトのシングルレイヤーモードの有効/無効を設定する機能です。デフォルトでは、フィルター/エフェクトは、それが割り当てられているレイヤーとその下のすべてのレイヤーに適用されますが、FXstruct のシングルレイヤーモードを有効にすると、それが割り当てられているレイヤーにのみ適用されることを確認できます。

FXstruct ( fx_create() または layer_get_fx() から返される) と、シングルレイヤーモードの有効化 (true) または無効化 (false) のブーリアン値 を指定します。

次のビジュアルは、シングルレイヤーモードが無効の場合（すべてのFXレイヤーのデフォルト動作です）と、シングルレイヤーモードが有効の場合の同じフィルタの使用状態を示しています。

注：FXのシングルレイヤーモードを有効にする場合、FXレイヤー以外のレイヤーに適用してください。FXレイヤー自体には何も含まれていないため、フィルターやエフェクトは表示されません。

構文です。

fx_set_single_layer(filter_or_effect, enable);

論証タイプ 商品説明
filter_or_effect The FX struct to modify
enable A boolean value to enable (true) または無効化 (false) シングルレイヤーモード

リターンです。

例

shake_fx = fx_create("_filter_screenshake");
fx_set_single_layer(shake_fx, true);
layer_set_fx("ShakeyThings", shake_fx);

上記のコードでは、新しいスクリーンシェイクFXを作成し、その上でシングルレイヤーモードを有効にし、room レイヤーに適用しています。つまり、スクリーンセイクフィルタは「ShakeyThings」レイヤーにのみ適用されることになります。
