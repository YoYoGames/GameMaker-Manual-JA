fx_create

この機能は、指定されたエフェクトタイプに対応する新しいFXStruct を作成し、レイヤーに適用するために使用します。

この関数に渡すことができるフィルター／エフェクトの名称は、このページに記載されています。すべてのフィルター/エフェクトの種類。

使用方法

作成したエフェクトのパラメータを読み込むには、以下のようにします。 fx_get_parameter/sstructを返し、 fx_set_parameter/s を呼び出して新しい値を割り当てることができます。

この関数が返すstruct には可視のメンバーは含まれず、Filter/Effect Functions のいずれかを通してのみ変更できることに注意してください。

機能障害

何らかの理由でstruct の作成に失敗した場合、この関数は -1 を返します。これは、指定された FX の名前が間違っていたり、FX がゲームパッケージにロードされていなかったりした場合に起こります。

この機能が正しく動作するように、フィルタとエフェクト機能のページで説明されているように、必要なFXがゲームパッケージにロードされていることを確認してください。

これらの条件が満たされない場合、つまり、string リテラルを渡しておらず、フィルタ/効果がrooms のいずれにも存在しない場合、この関数はそのフィルタ/効果を読み込むことができず、-1 を返します。

構文です。

fx_create(filter_or_effect_name);

論証 タイプ 商品説明
filter_or_effect_name 文字列 The name of the filter/effect to create (get them from All Filter/Effect Types).

リターンです。

FX Struct or -1 (on failure)

例

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上記のコードは、"_filter_tintfilter"エフェクトから新しい FXstruct を作成します。これはRoom Editor にある "Colour Tint" フィルタです。これは、色合いの色である"g_TintCol"パラメータに値を割り当て、内部的には vec4 であるため、4つの値（赤、緑、青、アルファ値に対応する）を含む配列を取ります。次に、色合いのためのFXstruct 、FXレイヤーに適用されます。
