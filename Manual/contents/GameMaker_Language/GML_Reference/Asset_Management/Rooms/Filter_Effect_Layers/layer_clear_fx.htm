レイヤークリアーフォックス

この関数は、指定したレイヤーに適用されているFX（struct ）をクリアするために使用します。変更したいレイヤーのIDまたは名前を指定すると、この関数はそのレイヤーに適用されているFXstruct を削除し、その効果を完全に無効にします。

この機能を使って一時的にエフェクトを無効にし、レイヤーを元に戻す必要がある場合、まず layer_get_fx() を呼び出してレイヤーの FXstruct を保存し、次にこの関数を呼び出してレイヤー上でそれをクリアする必要があります。その後、 layer_set_fx() を呼び出すことで、レイヤにそれを適用し、エフェクトを再び有効にすることができます。

構文です。

layer_clear_fx(layer_name_or_id);

論証 タイプ 商品説明
layer_name_or_id 文字列または レイヤーID The name or ID of the layer to clear the FX for

リターンです。

N/A

例

// Disable effect
if (keyboard_check_pressed(ord("E")))
{
    if (layer_get_fx("EffectLayer") != -1)
    {
        temp_fx = layer_get_fx("EffectLayer");
        layer_clear_fx("EffectLayer");
    }
}
// Re-enable effect
else if (keyboard_check_pressed(ord("F")))
{
    if (temp_fx != -1)
    {
        layer_set_fx("EffectLayer", temp_fx);
        temp_fx = -1;
    }
}

上記のコードは、"E "キーが押されたかどうかをチェックし、レイヤー "EffectLayer "にフィルターやエフェクトが適用されているかを調べます。その場合、そのFXstruct を一時インスタンス変数に格納して、フィルターやエフェクトのあるレイヤーをクリアーします。F "キーが押されると、一時変数に-1以外が格納されているかどうかをチェックし、その場合はそのFXをレイヤーに適用し、一時変数を-1にリセットします。このように、エフェクトのパラメータを失うことなく、必要なときにいつでもエフェクトを有効／無効にすることができるのです。
