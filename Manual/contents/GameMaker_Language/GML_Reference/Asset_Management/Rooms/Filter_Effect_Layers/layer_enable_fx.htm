レイヤー有効化

Room レイヤーに割り当てられたフィルター／エフェクトの有効／無効を設定する機能です。変更したいレイヤーのIDまたは名前を指定し、FXを有効（true）または無効（false）にするかどうかをブーリアン値で指定します。

falseを渡すと、レイヤーからFXが削除されず、単に不可視になります。レイヤーからFXを削除するには、 layer_clear_fx() を使用します。

同様に、レイヤーにFXが適用されていない場合は、trueを渡しても何も起こりません。レイヤーに FX を適用するには、 layer_set_fx() を使用します。

構文です。

layer_enable_fx(layer_name_or_id, enable);

論証 タイプ 商品説明
layer_name_or_id 文字列または レイヤーID The name or ID of the layer to modify
enable ブーリアン Whether to enable or disable the FX

リターンです。

N/A

例

if (hp <= 3)
{
    layer_enable_fx("DesaturateLayer", true);
}
else
{
    layer_enable_fx("DesaturateLayer", false);
}

上記のコードは、インスタンスのHP値が3以下の場合、Desaturate FXレイヤーを有効にし、それ以外の場合は無効にしています。これは、画面の彩度を落とすことで、プレイヤーにHPが低いことを知らせます。
