fx_get_single_layer

この関数は、与えられたフィルタ／エフェクトがシングルレイヤーモード（true ）を使用しているかどうかを返します（false ）。参照 fx_set_single_layer()シングルレイヤーモードに関する情報はこちらをご覧ください。

FXstruct ( fx_create() または layer_get_fx() から返される) を指定すると、この関数は FX に対して Single Layer モードが有効かどうかを示すboolean 値を返します。

構文です。

fx_get_single_layer(filter_or_effect);

論証タイプ 商品説明
filter_or_effect The FX struct to read

リターンです。

Boolean

例

var _fx = layer_get_fx("Distort");

if (!fx_get_single_layer(_fx))
{
    fx_set_single_layer(_fx, true);
}

上記のコードは、"Distort "というレイヤーからFXstruct を取得し、シングルレイヤーモードが有効になっているかどうかをチェックします。もしそうでなければ、そのFXのシングルレイヤーモードを有効にします。
