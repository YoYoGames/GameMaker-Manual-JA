fx_get_parameters

この関数は、指定された FXstruct のすべてのパラメータを含む構造体を取得するために使用されます。FXstruct ( fx_create() または layer_get_fx() から返される) を指定すると、この関数はそのフィルタ/エフェクトに適用されるすべてのパラメータ用の変数を含むstruct を返します。ベクトル(すなわち値のグループ)を使用するパラメータは、配列として存在します。

fx_get_parameter_names() を呼び出すと、すべてのパラメータの名前を別の配列で取得することができます。FX タイプとパラメータ.

このstruct で値を変更しても、その変更はフィルター/効果に適用されません。変更を有効にするには、fx_set_parameters() を呼び出して、変更したstruct をフィルター/効果に適用し直す必要があります。

構文です。

fx_get_parameters(filter_or_effect);

論証 タイプ 商品説明
filter_or_effect FX構造体 The FX struct for which the parameters will be returned

リターンです。

Struct (or -1 on failure)

例

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上記のコードは、「カラーティント」フィルタが適用されていると想定されるレイヤーの FXstruct を取得し、そのパラメータstruct を fx_get_parameters() を呼び出すことで取得しています。その後、 sin() と current_time を使って振動する値を作成し、それが色合い効果の RGBA 値に使用されるので、アニメーションが継続されます。RGBA配列は、パラメータstruct の変数 g_TintCol に割り当てられ、struct を呼び出してFXstruct に戻して適用されます。 fx_set_parameters().
