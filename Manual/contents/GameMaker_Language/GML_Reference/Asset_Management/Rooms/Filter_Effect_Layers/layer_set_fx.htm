layer_set_fx

この関数は、FXstruct をRoom レイヤーに割り当てるために使用します。変更したいレイヤーのIDまたは名前を指定し、指定したレイヤーに適用されるFXstruct （ fx_create() または layer_get_fx() から返される）を指定して、room でエフェクトを表示させます。

フィルター／エフェクトレイヤ以外のレイヤ（インスタンスレイヤ、Asset レイヤ、Tile レイヤなど）を指定すると、そのレイヤの内容だけでなく、その下のレイヤにも効果が適用されます。

指定したレイヤーにすでにFXstruct 、新しいレイヤーに置き換えられます。

注 script のルートレベルでこの関数を呼び出しても、最初のscripts がロードされる前にroom が実行されるため、何も起こりません。最初にターゲットルームを設定してから、この関数を呼び出すとよいでしょう。

構文です。

layer_set_fx(layer_name_or_id, filter_or_effect);

論証 タイプ 商品説明
layer_name_or_id 文字列または レイヤID The name or ID of the layer to modify
filter_or_effect FX構造体 The FX struct to apply to the layer

リターンです。

N/A

例

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上記のコードは、"_filter_tintfilter"エフェクトから新しい FXstruct を作成します。これはRoom エディタにある "Colour Tint" フィルタです。これは、色合いの色である"g_TintCol"パラメータに値を割り当て、内部的に vec4 であるため、4つの値（赤、緑、青、アルファ値に対応）を含む配列を取ります。次に、色合いのためのFXstruct 、FXレイヤーに適用されます。
