fx_get_name

この関数は、フィルタ/エフェクトの名前をstruct から取得するために使用します。読み込むFXstruct ( fx_create() または layer_get_fx() から返される) を指定すると、この関数はその名前をstring として返します。 この名前をfx_create() に渡すと、同じフィルタ/エフェクト タイプを使用して新しいFXstruct を作成することができます。

構文です。

fx_get_name(filter_or_effect);

論証タイプ 商品説明
filter_or_effect The FX struct to get the name of

リターンです。

例

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上記のコードでは forroom に存在するすべてのレイヤーをループし、各レイヤーに FXstruct があるかどうかをチェックします。もしレイヤーがFXstruct を持っていれば、 fx_get_name() を呼び出して、そのフィルター／効果の名前をチェックします。もしそれが"_filter_tintfilter"と等しければ、つまり "Colour Tint" フィルターであれば、その色合いの色を青に変更します。
