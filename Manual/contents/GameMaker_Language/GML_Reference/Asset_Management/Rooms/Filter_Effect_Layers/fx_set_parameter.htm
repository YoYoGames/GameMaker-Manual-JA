fx_set_parameter

この関数は、指定された FXstruct のパラメータ値を変更するために使用されます。FXstruct ( fx_create() または layer_get_fx() から返されたもの)、パラメータ名string 、およびその新しい値を指定します。これにより、FXstruct が割り当てられているレイヤーで変更内容が表示されます。

パラメータがベクトルの場合、複数の引数を渡すことができますが、複数の値を含む唯一の引数として、単純に配列を指定することもできます。

フィルター/エフェクトのパラメーター名を探すページです。FXの種類とパラメーター

構文です。

fx_set_parameter(filter_or_effect, parameter_name, val, ...);

論証 タイプ 商品説明
filter_or_effect FX構造体 The FX struct to modify
parameter_name 文字列 The name of the parameter to change the value of (as a string)
val... リアルまたは 配列 The new value(s) for that parameter as a real or array

リターンです。

N/A

例

var _fx_tint = fx_create("_filter_tintfilter");
fx_set_parameter(_fx_tint, "g_TintCol", [1, 0, 0, 1]);
layer_set_fx("EffectLayer", _fx_tint);

上記のコードは、"_filter_tintfilter"エフェクトから新しい FXstruct を作成します。これはRoom Editor にある "Colour Tint" フィルタです。これは、色合いの色である"g_TintCol"パラメータに値を割り当て、内部的には vec4 であるため、4つの値（赤、緑、青、アルファ値に対応する）を含む配列を取ります。次に、色合いのためのFXstruct 、FXレイヤーに適用されます。

なお、vectorには配列ではなく、別の引数として値を渡すことができます。

fx_set_parameter(_fx_tint, "g_TintCol", 1, 0, 0, 1);
