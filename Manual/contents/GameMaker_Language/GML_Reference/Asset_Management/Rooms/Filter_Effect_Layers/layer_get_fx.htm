レイヤーゲット

この関数は、あるレイヤーの FXstruct を取得するために使用します。対象となるレイヤーの ID または名前を指定すると、この関数はその適用された効果に関する情報を含むstruct を返します。このstruct は、 fx_create() から取得したstruct と同様のもので、関数 fx_get_parameter/sおよび fx_set_parameter/s を使用して、そのパラメータを読み取ったり変更したりすることができます。

指定されたレイヤーにフィルターやエフェクトが適用されていない場合、この関数は -1 を返します。

構文です。

layer_get_fx(layer_name_or_id);

論証 タイプ 商品説明
layer_name_or_id 文字列またはレイヤID The name or ID of the layer to read

リターンです。

FX Struct (or -1 if not found)

例

var layers = layer_get_all();
for(var i = 0; i < array_length(layers); i ++)
{    
    var layer_fx = layer_get_fx(layers[i]);
    
    if (layer_fx != -1)
    {
        if (fx_get_name(layer_fx) == "_filter_tintfilter")
        {            
            fx_set_parameter(_fx_tint, "g_TintCol", [0, 0, 1, 1]);
        }
    }
}

上記のコードでは forroom に存在するすべてのレイヤーをループし、各レイヤーに FXstruct があるかどうかをチェックします。レイヤーにFXがある場合struct 、そのフィルター／エフェクトの名前をコールしてチェックします。 fx_get_name()もしそれが"_filter_tintfilter "であれば、つまり "Colour Tint "フィルタであれば、その色合いを青に変更します。
