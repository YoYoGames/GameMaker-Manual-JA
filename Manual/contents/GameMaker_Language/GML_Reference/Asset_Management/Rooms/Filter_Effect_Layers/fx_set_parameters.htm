fx_set_parameters

この機能は、フィルター／エフェクトのパラメータを変更するために使用されます。FXstruct （ fx_create() または layer_get_fx() から返されるもの）と、そのパラメータを含むstruct （ fx_get_parameters() から返されるもの）を指定します。これにより、FXstruct が割り当てられているレイヤーで変更内容が表示されます。

パラメータstruct には、フィルター/エフェクトに適用できるすべてのパラメータを含める必要はなく、与えられたstruct に存在する変数のみが、フィルター/エフェクトのために更新されます。

構文です。

fx_set_parameters(filter_or_effect, parameter_struct);

論証タイプ 商品説明
filter_or_effect The FX struct to modify
parameter_struct A struct containing the parameters for the filter/effect

リターンです。

例

var _fx_struct = layer_get_fx("TintEffect");

if (_fx_struct != -1)
{
    var _params = fx_get_parameters(_fx_struct);
    var _osc = sin(current_time / 1000);
    _params.g_TintCol = [_osc, 0.3 + _osc, 0.6 + _osc, 1];

    fx_set_parameters(_fx_struct, _params);
}

上記のコードでは、「Colour Tint」フィルターを適用したレイヤーのFXstruct を取得し、そのパラメーターstruct を呼び出して取得しています。 fx_get_parameters().その後、 sin() と current_time を使って振動する値を作成し、それが色合い効果の RGBA 値に使用されるので、アニメーションが継続します。RGBA配列はパラメータstruct の変数 g_TintCol に代入され、struct は fx_set_parameters() を呼び出すことによってFXstruct に戻されます。
