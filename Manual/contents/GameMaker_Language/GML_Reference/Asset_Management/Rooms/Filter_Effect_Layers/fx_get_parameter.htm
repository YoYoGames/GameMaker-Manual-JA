fx_get_parameter

この関数は、FXStruct からパラメータの値を取得するために使用します。読み込むFXstruct （ fx_create() または layer_get_fx() から返されるもの）とパラメータ名をstring として指定すると、この関数はその現在の値を返します。パラメータがベクトル（RGBA カラーを格納する vec4 のような値のグループ）である場合、これは実数値または配列になることがあります。

フィルター/エフェクトのパラメーター名を探すページです。FXの種類とパラメーター

構文です。

fx_get_parameter(filter_or_effect, parameter_name);

論証 タイプ 商品説明
filter_or_effect FX構造体 The FX struct to get the parameter from
parameter_name 文字列 The name of the parameter to get the value of (as a string)

リターンです。

Real or Array

例

var _fx_struct = layer_get_fx("TintEffect");
var _tint_colour = fx_get_parameter(_fx_struct, "g_TintCol");
show_debug_message("The currently active tint colour is: " + string(_tint_colour));

上記のコードは、「色合い」フィルタが適用されていると想定されるレイヤーから FXstruct を取得し、その「g_TintCol」パラメータの値を取得します。このパラメータは、色合いの色（赤、緑、青、アルファ）の 4 つの値を含む配列を返します。そして、その値を出力ログにプリントします。
