レイヤーフレックス有効

この関数は、レイヤーのフィルタ／エフェクトが有効であるかどうかを返します。

対象となるレイヤーのIDまたは名前を指定すると、そのレイヤーのFXが有効（true）か無効（false）かを示すブーリアン値が返されます。

構文です。

layer_fx_is_enabled(layer_name_or_id);

論証 タイプ 商品説明
layer_name_or_id 文字列または レイヤーID The name or ID of the layer to check

リターンです。

Boolean

例

x = xstart;
y = ystart;
hp = hp_max;

if (layer_fx_is_enabled("DesaturateLayer"))
{
    layer_enable_fx("DesaturateLayer", false);
}

上記のコードでは、インスタンスを元の位置に移動し、HPを再充填することで「リスポーン」しています。そして、脱色FXレイヤーが有効かどうかをチェックし、有効な場合は無効にします。
