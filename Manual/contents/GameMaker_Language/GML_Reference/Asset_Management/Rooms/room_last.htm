room_last

この読み取り専用変数は、ゲーム内の最後のroom のインデックスを返します (これは、rooms がルームマネージャに表示される順番で定義され、それらが作成された順番ではありません)。この変数は、room_add() やroom_duplicate() を使って動的に追加されたrooms を認識したり、考慮したりしないことに注意してください。

構文です。

room_last

リターンです。

Room Asset

例

if keyboard_check_pressed(ord("Q"))
{
    room_goto(room_last);
}

上記のコードは、キーが押されたかどうかをチェックし、押された場合はゲーム内の最後のroom に移動します。
