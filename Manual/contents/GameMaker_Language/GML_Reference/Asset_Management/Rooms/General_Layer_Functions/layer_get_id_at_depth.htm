レイヤーの深さを指定します。

この関数を使用すると、特定の深さに割り当てられたすべてのレイヤーのIDを取得することができます。チェックする深度を指定すると、 その深度のレイヤーが存在するかどうかによって 1 つ以上のエントリを含む配列を返します。指定した深さにレイヤーがない場合は、 配列の [0] 位置に値 -1 のエントリがひとつだけ作成されます。

構文です。

layer_get_id_at_depth(depth)

論証 タイプ 商品説明
depth リアル The depth to check and retrieve the layer IDs from.

リターンです。

Array

例

var a = layer_get_id_at_depth(0);
if (a[0] != -1)
{
    for (var i = 0; i < array_length(a); i++;)
    {
        layer_destroy(a[i]);
    }
}

上記のコードでは、深さ 0 のレイヤーに関するデータを取得し、その深さにレイヤーが存在するかどうかをチェックし、存在する場合は返された配列をパースして、見つかったレイヤーをそれぞれ破棄しています。
