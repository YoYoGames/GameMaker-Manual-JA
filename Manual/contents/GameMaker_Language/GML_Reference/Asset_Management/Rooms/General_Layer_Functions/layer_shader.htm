レイヤーシェーダー

この関数を使用すると、任意のレイヤーにshader リソースを割り当てることができ、そのレイヤーはshader を使用してレンダリングされます。レイヤーのユニーク ID 値またはレイヤー名（string - これはパフォーマンスに影響します）を、使用するshader の ID と共に指定します。shader はAsset Browser で以前に作成されている必要があり、shader インデックス (shader リソースの名前) がこの関数に渡されます。割り当てられたレイヤーにインスタンスが追加されている場合、shader はそのインスタンスが使用するすべての描画イベントに適用されます。たとえば、インスタンスが Draw GUI Begin イベントを持っている場合、shader は自動的に適用されます。shader は、sprite assets やtile マップのように、そのレイヤーに描画される他のグラフィック要素にも影響します。

なお、この関数は描画イベントやステップイベントで呼ばれるものではなく、ルーム作成コードやインスタンスの作成イベント／ ルーム開始イベントでroom の開始時に呼ばれるだけでよいことに注意してください。

構文です。

layer_shader(layer_id, shader)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
shader シェーダーアセット The shader index to assign to the layer

リターンです。

N/A

例

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

上記のコードでは、すべての描画について、与えられたレイヤーにshader リソース "shd_Sepia" を割り当てます。
