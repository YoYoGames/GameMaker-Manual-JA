layer_get_hspeed

この関数を使うと、現在スコープされているroom 内のレイヤーの水平速度（ゲームフレームあたりのピクセル数）を取得することができます。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると、この関数は水平方向の速度を実数で返します。デフォルトは0（room エディターで設定しない限り）です。

構文です。

layer_get_hspeed(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to get the horizontal speed from

リターンです。

N/A

例

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上記のコードは、与えられたレイヤーの水平速度と垂直速度をチェックし、両方とも0に設定されていない場合、それらを0に設定します。
