レイヤーデストロイ

この関数は、指定されたレイヤーを破壊します。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を入力すると、現在のroom から削除されます。レイヤーがroom エディタでデザインされたものである場合、次にroom を離れて戻ったとき、レイヤーは元の内容で再び作成されます。しかし、room が永続化されている場合、room の永続化を再度オフにしないと、レイヤーは除去されます。この方法でレイヤーを破壊すると、そのコンテンツもすべて削除されます。したがって、背景やtile マップなどの参照 ID は無効となり、レイヤーに割り当てられたインスタンスはすべて破壊されます（そのインスタンスがある場合は、同時に破棄イベントと クリーンアップ イベントを実行します）。

構文です。

layer_destroy(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to destroy (or the layer name as a string)

リターンです。

N/A

例

if !instance_exists(obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

上記のコードは、object "obj_Bullet_Parent" のインスタンスが存在するかどうかをチェックし、存在しない場合は、グローバル変数に格納されているIDでレイヤーを破壊します。
