layer_get_all_elements

この関数は、指定されたレイヤーの要素IDを取得するために使用します。レイヤー ID (レイヤーを作成するときに layer_create()) あるいはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると、 この関数は ID の配列を返します。配列の各エントリは、そのレイヤーの要素に対して一意の ID を表します。たとえば、レイヤーがAsset Layer の場合、配列にはそのレイヤーに割り当てられている各sprite asset の ID 値が格納されます。コードを使用してレイヤーを操作するということは、 同じレイヤーに異なる種類の要素を割り当てることができるということです。 たとえば、sprite assets をインスタンスと一緒に持つことができます。 layer_get_element_type()を使用して、そのIDが関連する要素の種類を取得します。

構文です。

layer_get_all_elements(layer_id)

論証 タイプ 商品説明
layer_id 文字列または レイヤーID The unique ID value of the layer to get the elements from (or the layer name as a string)

リターンです。

Array of Layer Element IDs

例

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i])
    }
}

上記のコードでは、コードを実行しているインスタンスのレイヤーに割り当てられているすべてのインスタンス要素のIDを取得しています。そして、返された要素の中にsprite assets があるかどうかを確認し、ある場合は破棄します。
