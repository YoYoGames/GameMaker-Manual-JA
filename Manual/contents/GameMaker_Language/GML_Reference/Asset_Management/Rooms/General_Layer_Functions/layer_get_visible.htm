layer_get_visible

この関数を使うと、レイヤーが表示されているかどうかを確認することができます。レイヤーID（レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を返し、この関数は、それが表示されている場合は true を、そうでない場合は false を返します。

構文です。

layer_get_visible(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)

リターンです。

Boolean

例

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
layer_set_visible(lay_id, false);
}
else
{
layer_set_visible(lay_id, true);
}

上記のコードは、room エディターで「Instances」というレイヤーのID値を取得し、そのIDを使ってレイヤーが表示されているかどうかをチェックし、返された値に応じてレイヤーの表示/非表示を切り替えています。
