レイヤー要素移動

この関数は、あるレイヤーから別のレイヤーに要素を移動するために使用します。要素IDを指定します。これは、要素を作成するために使用した関数、room エディタ、または関数 layer_get_all_elements()でレイヤーを作成したときに取得したレイヤーIDを指定します。 layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) を指定します。

構文です。

layer_element_move(element_id, layer_id)

論証 タイプ 商品説明
element_id レイヤー要素ID The unique ID value of the element to move
layer_id 文字列またはレイヤID The unique ID value of the layer to move the element to (or the layer name as a string)

リターンです。

N/A

例

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_element_move(a[i], asset_layer)
    }
}

上記のコードは、コードを実行しているインスタンスが割り当てられているレイヤー上の要素を取得し、それらがsprite assets であるかどうかをチェックし、もしそうなら、変数 "asset_layer" に格納されている ID のレイヤーに移動されます。
