layer_get_element_layer

この関数を使用すると、指定された要素のレイヤーIDを取得することができます。一意な要素 ID の値 (例えば、要素を作成した関数やroom エディターから返される値) を与えると、 この関数はその要素が存在するレイヤの一意な ID を返します。与えられた要素IDが有効なものでない場合、この関数は -1 を返します。

構文です。

layer_get_element_layer(element_id)

論証 タイプ 商品説明
element_id レイヤー要素ID The unique ID value of the element to get the layer of

リターンです。

Layer ID or -1 (if the element is invalid)

例

element = layer_get_element_layer(asset_1);

上記のコードでは、変数 "asset_1 "に格納されているIDを持つ要素のレイヤーIDを取得し、変数に格納しています。
