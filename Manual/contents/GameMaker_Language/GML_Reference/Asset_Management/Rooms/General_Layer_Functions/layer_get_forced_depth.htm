layer_get_forced_depth

この関数は、room 内のレイヤーをレンダリングするために設定された Z 深度を取得するために使用することができます。 layer_force_draw_depth()を参照してください。

構文です。

layer_get_forced_depth()

リターンです。

Real

例

if layer_get_forced_depth() != 0
{
    layer_force_draw_depth(true, 0);
}

上記のコードでは、レイヤーのZ深度が強制的に0に設定されているかどうかを確認し、設定されていない場合はレイヤーのZ深度を0に設定しています。
