layer_has_instance

この関数は、与えられたインスタンスが現在与えられたレイヤーに割り当てられているかどうかをチェックするために使用することができます。レイヤー ID (これは、レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) と、チェックするインスタンスのインスタンス ID を指定します。また、 object_index (例:object Browser のAsset の名前) を指定すると、そのobject のインスタンスが指定したレイヤー上にあるかどうかをチェックします。この関数は、そのインスタンスがレイヤー上にある場合は true を、ない場合は false を返します。

構文です。

layer_has_instance(layer_id, instance_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
instance_id インスタンスID The unique instance ID or the object index of the instance to check for

リターンです。

Boolean

例

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
{
    layer_destroy(global.Bullet_Layer);
}

上記のコードは、与えられたレイヤーにobject "obj_Bullet_Parent" のインスタンスが含まれているかどうかをチェックし、含まれていない場合はレイヤーを破壊します。
