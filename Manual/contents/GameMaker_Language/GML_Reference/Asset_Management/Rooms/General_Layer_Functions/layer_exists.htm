レイヤーの存在

この関数は、与えられたレイヤーが存在するかどうかを確認するために使用することができます。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると、この関数は、存在すれば true を、存在しなければ false のブール値を返します。

注：この関数は、現在のターゲットroom の範囲内で動作します。デフォルトでは、この関数が呼び出されたroom で、この範囲は関数 layer_set_target_room().

構文です。

layer_exists(layer_name)

論証 タイプ 商品説明
layer_name 文字列またはレイヤID The name of the layer (a string or ID value)

リターンです。

Boolean

例

if !layer_exists(global.tileLayer)
{
    global.tileLayer = layer_create(1000);
}

上記のコードでは、グローバル変数に格納されたレイヤーが実際に存在するかどうかを確認し、存在しない場合は作成します。
