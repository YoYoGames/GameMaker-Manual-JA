layer_hspeed

この関数を使うと、現在スコープされているroom 内のレイヤーの水平速度（ゲームフレームあたりのピクセル数）を設定することができま す。レイヤーID（レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) と設定する速度値で、正の値は右側、負の値は左側になります。

構文です。

layer_hspeed(layer_id, hspd)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to set the horizontal speed of
hspd リアル The horizontal speed (in pixels per game frame) to set

リターンです。

N/A

例

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上記のコードは、与えられたレイヤーの水平速度と垂直速度をチェックし、両方とも0に設定されていない場合、それらを0に設定します。
