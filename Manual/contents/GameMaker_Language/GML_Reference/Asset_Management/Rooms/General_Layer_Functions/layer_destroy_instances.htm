layer_destroy_instances

この関数は、指定されたレイヤーに割り当てられたすべてのインスタンスを破棄するために使用することができます。レイヤーの ID を指定します。 layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) で、そのレイヤー上にあるすべてのインスタンスがゲームから削除され、DestroyおよびClean Upイベントがトリガーされます。

構文です。

layer_destroy_instances(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer (or the layer name as a string)

リターンです。

N/A

例

if global.game_over
{
    layer_destroy_instances(layer);
}

上記のコードでは、グローバル変数をチェックし、それが true であれば、呼び出したインスタンスのレイヤーにあるすべてのインスタンスが破壊されます（呼び出したインスタンスも含む）。
