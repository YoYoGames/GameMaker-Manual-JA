レイヤーゲットシェーダー

この関数は、与えられたレイヤーにshader が割り当てられているかどうかを確認するために使用することができます。レイヤー ID を指定します (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると、この関数はshader 割り当てられたshader のインデックス、またはshader が割り当てられていない場合は -1 を返します。

構文です。

layer_get_shader(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)

リターンです。

Shader Asset or -1 (if no shader is assigned)

例

if layer_get_shader(layer) == -1
{
    layer_shader(layer, shd_Sepia);
}

上記のコードでは、コードを実行しているインスタンスのレイヤーにshader が割り当てられているかどうかを確認し、割り当てられていない場合は割り当てます。
