レイヤーx

この関数は、現在スコープされているroom 内のレイヤーの x 位置を設定するために使用できます。レイヤーID（これは layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると、この関数はroom の水平軸に沿って指定されたピクセル数だけレイヤーを移動させます。

構文です。

layer_x(layer_id, x)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to set the x position of
x リアル The x position in the room to set the layer to

リターンです。

N/A

例

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

上記のコードでは、与えられたレイヤーの位置をチェックし、(0, 0)に設定されていなければ、その位置に設定されます。
