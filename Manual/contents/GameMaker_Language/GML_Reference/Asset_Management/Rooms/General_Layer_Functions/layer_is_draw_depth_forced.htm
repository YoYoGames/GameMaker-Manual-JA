layer_is_draw_depth_forced

この関数は、プロジェクト内のレイヤーのレンダリングに強制Z深度が有効であるかどうかを確認するために使用します。参照 layer_force_draw_depth()を参照してください。

構文です。

layer_is_draw_depth_forced()

リターンです。

例

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

上記のコードでは、レイヤーのZ深度が強制されているかどうかを確認し、強制されていない場合はZ深度を0に設定し、有効にしています。
