layer_set_visible

この関数を使うと、レイヤーの可視性を切り替えることができます。レイヤーID（レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) と、レイヤーのトグル値 (visible はtrue, invisible はfalse) を指定します。

レイヤーが不可視の場合、レイヤーに割り当てられているものは何も描画されず、もしレイヤーにインスタンスが割り当てられていても、それらは描画イベントさえ実行されません（インスタンス変数 visible を false に設定した場合とほぼ同じです）。すでに不可視のフラグが立っているインスタンスは、そのレイヤーが可視に設定されている場合、可視のフラグが立たないので注意してください。

構文です。

layer_set_visible(layer_id, visible)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
visible ブーリアン Whether the layer should be visible (true) か、そうでないか (false)

リターンです。

N/A

例

var lay_id = layer_get_id("Instances");
if (layer_get_visible(lay_id))
{
    layer_set_visible(lay_id, false);
}
else
{
    layer_set_visible(lay_id, true);
}

上記のコードは、room エディターで「Instances」というレイヤーのID値を取得し、そのIDを使ってレイヤーが表示されているかどうかをチェックし、返された値に応じてレイヤーの表示/非表示を切り替えています。
