レイヤーの速度

この関数を使うと、現在スコープされているroom 内のレイヤーの垂直方向の速度（ゲームフレームあたりのピクセル数）を設定することができます。レイヤーID（レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) と設定する速度値で、正の値は下向き、負の値は上向きです。

構文です。

layer_vspeed(layer_id, vspd)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to set the vertical speed of
vspd リアル The vertical speed (in pixels per game frame) to set

リターンです。

N/A

例

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
    layer_hspeed(lay_id, 0);
    layer_vspeed(lay_id, 0);
}

上記のコードは、与えられたレイヤーの水平速度と垂直速度をチェックし、両方とも0に設定されていない場合、それらを0に設定します。
