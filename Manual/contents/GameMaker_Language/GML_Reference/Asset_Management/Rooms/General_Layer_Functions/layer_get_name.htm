レイヤーの名前

この関数は、指定されたレイヤーの名前を取得するために使用することができます。ユニークなレイヤー ID 値を与えると、そのレイヤーがroom エディターで作成された名前付きレイヤーのうちの 1 つである場合、この関数はそのレイヤー名を持つstring を返します。レイヤーがroom エディターで作成されたものでない場合 (すなわち、それは layer_create()) の場合、空のstring が返されます。

構文です。

layer_get_name(layer_id)

論証 タイプ 商品説明
layer_id レイヤーID The unique ID value of the layer to get the name of

リターンです。

String

例

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;)
{
    if layer_get_name(a[i]) != ""
    {
        ds_list_add(layer_list, a[i])
    }
}

上記のコードは、room にあるすべてのレイヤーの ID を取得し、loops にあるレイヤーのうち、名前のついたレイヤーがあるかどうかをチェックしています。もしそうなら、その ID がリストに追加されます。
