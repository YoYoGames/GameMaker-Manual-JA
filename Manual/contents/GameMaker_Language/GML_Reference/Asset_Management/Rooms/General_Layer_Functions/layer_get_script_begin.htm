layer_get_script_begin

レイヤー ID (レイヤーを作成するときに layer_create()string この関数は、そのレイヤーのレンダリング開始時に実行するよう割り当てられた関数のスクリプト関数インデックスを返すか、関数が割り当てられていない場合は-1を返します。script の関数をレイヤーに割り当てるには、layer_script_begin() と layer_script_end() を使用します。

構文です。

layer_get_script_begin(layer_id);

論証 タイプ 商品説明
layer_id 文字列または レイヤーID The unique ID value of the layer to target (or the layer name as a string)

リターンです。

Script Function or -1 (if no function is assigned)

例

if layer_get_script_begin(layer) == -1
{
    layer_script_begin(layer, scr_SetShaderValues);
}

上記のコードでは、コードを実行しているインスタンスのレイヤーにscript 関数が割り当てられているかどうかを確認し、割り当てられていない場合は割り当てます。
