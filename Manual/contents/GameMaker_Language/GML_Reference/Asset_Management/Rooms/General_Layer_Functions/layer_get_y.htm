レイヤーゲット

この関数は、現在スコープされているroom 内のレイヤーの y 位置を取得するために使用できます。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定すると，この関数はレイヤーの x 位置を，room の (0,0) 位置からの相対値として実数で返します． デフォルトは 0 です．

構文です。

layer_get_y(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to get the y position of

リターンです。

Real

例

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
}

上記のコードでは、与えられたレイヤーの位置をチェックし、(0, 0)に設定されていなければ、その位置に設定されます。
