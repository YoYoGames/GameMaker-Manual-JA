レイヤーゲットターゲットルーム

この関数は、レイヤー関数が対象としている現在のroom を返します。を参照してください。 layer_set_target_room()を参照してください。

構文です。

layer_get_target_room()

リターンです。

Room Asset

例

if layer_get_target_room() != room
{
    layer_reset_target_room();
}

上記のコードは、現在のターゲットroom をチェックし、それが現在のroom でない場合、レイヤーのターゲットroom をリセットしています。
