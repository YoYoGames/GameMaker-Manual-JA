layer_get_element_type

この関数は、与えられた要素の要素タイプを取得するために使用します。一意の要素ID値（例えば、要素を作成した関数やroom エディタから返される値）を与えると、この関数は以下の定数のうちの1つを返します（要素が存在しない場合やID値に誤りがある場合は-1）。

レイヤー要素タイプ定数
定数 商品説明
layerelementtype_background 要素は背景である。
layerelementtype_instance 要素はインスタンスである。
layerelementtype_sprite エレメントは、sprite asset 。
layerelementtype_tilemap 要素はタイルマップである。
layerelementtype_particlesystem 素子は粒子系です。
layerelementtype_tile この要素は、レガシーバックグラウンドtile です（これは、以前のバージョンのGameMaker からインポートされたプロジェクトにのみ有効です）。
layerelementtype_sequence エレメントは、sequence asset 。

なお、この関数は、1つのレイヤーに複数の異なる種類の要素が割り当てられている場合に最も有効で、任意のレイヤー上の全要素のリストを得るには、関数 layer_get_all_elements().

構文です。

layer_get_element_type(element_id)

論証 タイプ 商品説明
layer レイヤー要素ID The unique ID value of the element to get the type of

リターンです。

Layer Element Type Constant or -1 (if element does not exist or is invalid)

例

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
    if layer_get_element_type(a[i]) == layerelementtype_sprite
    {
        layer_sprite_destroy(a[i])
    }
}

上記のコードでは、コードを実行しているインスタンスのレイヤーに割り当てられているすべてのインスタンス要素のIDを取得しています。そして、返された要素の中にsprite assets があるかどうかを確認し、ある場合は破棄します。
