レイヤーの深さを取得する

この関数は、指定されたレイヤーに関連する深さの値を取得するために使用します。レイヤー ID を指定します（レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を入力すると、この関数はそのレイヤーの深度を実数で返します。深度は、カメラから「遠い」ほど高く、「近い」ほど低くなるように定義されていることに注意してください。したがって、深度 -100, 0, 100 の 3 つのレイヤーがある場合、レイヤーは 100, 0, -100 の順序で描画され、「最上位」レイヤー (つまり、カメラビューに最も近いので他のすべてに重ねて表示) が深度 -100 のレイヤーとなることに注意してください。

次の画像は、レイヤーの深さがどのように機能するかを示す概略図です。

無効なレイヤーIDの値を指定するとエラーになることに注意してください。

構文です。

layer_get_depth(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to get the depth of (or the layer name as a string)

リターンです。

Real

例

if layer_get_depth(global.temp_layer) != -100
{
    layer_destroy(global.temp_layer);
    global.temp_layer = layer_create(-100);
}

上記のコードは、グローバル変数に格納されたレイヤーIDの深さをチェックし、それが-100でなければ、レイヤーを破棄し、深さ-100で再作成しています。
