layer_instance_get_instance

この関数は、与えられたインスタンス要素のユニークインスタンスIDを取得するために使用することができます。インスタンス要素のID を与えると (取得方法は以下のコード例を参照ください)、 この関数はその要素の一意なインスタンス IDを表す実数値を返します。インスタンスでない場合は、-1 を返します。

構文です。

layer_instance_get_instance(element_id)

論証 タイプ 商品説明
element_id インスタンス要素ID The unique ID value of the instance element to get the ID of

リターンです。

Instance ID

例

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
     if (layer_get_element_type(elements[i]) == layerelementtype_instance)
     {
         var layerelement = elements[i];
         var inst = layer_instance_get_instance(layerelement);
         inst.x = inst.x + 10;
     }
}

上記のコードは、レイヤー上のすべてのインスタンス要素を取得し、そのユニークなID値を取得し、それを使ってそれらを10px右に移動します。
