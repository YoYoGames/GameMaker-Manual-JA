layer_add_instance

この関数は、あるインスタンスを現在いるレイヤーから別のレイヤーに移動させるために使用できます。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) と、レイヤー間を移動するインスタンスの ID を指定します。たとえば、プレーヤーが別のレイヤーより低い深さのレイヤーにあり、その2つのレイヤーの間にあるレイヤーの後ろに表示させたいとします。この関数を使用すると、switch 、プレーヤーを前景レイヤーから背景レイヤーに移動し、描画中の他のレイヤーの背後に表示させることができます。

注意この関数は、レイヤーに新しいインスタンスを追加するために使用することはできません。事前にインスタンスを作成し、そのIDを変数に保存しておく必要があります。

注：この機能は、現在のroom の範囲内でのみ有効であり、機能layer_set_target_room() で範囲を変更した場合は使用できません。

構文です。

layer_add_instance(layer_id, instance_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the instance layer to target (or the layer name as a string)
instance_id インスタンスID The unique instance ID value of the instance to move over to the target layer

N/A

例

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上記のコードでは、まず与えられたx/y位置に最も近いインスタンスのインデックスを取得し、ローカル変数に格納します。次に、"Instances Front "というレイヤーの一意のインスタンスレイヤーのIDを取得し、見つかったインスタンスをそのレイヤー上に移動させます。
