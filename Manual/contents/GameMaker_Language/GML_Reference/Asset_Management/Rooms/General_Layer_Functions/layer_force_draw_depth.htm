layer_force_draw_depth

この関数は、すべてのレイヤーを指定されたz深度で描画するように強制します。これはレイヤーの描画順を変更するものではなく、深度順に描画されることに変わりはありません。一般的には、これを気にする必要はありませんが、レイヤーの深さが法定範囲外 (-16000 から 16000) にある場合、レンダリングされないので、Z 深度を妥当な値 - 例えば 0 - に強制すると、すべてうまくレンダリングされるようになります。これは一般的に、以前のバージョンのGameMakerのレガシーなプロジェクトで、許容されるレイヤー範囲よりも高いまたは低い描画深度を持つことができる場合にのみ使用されることに留意してください。

構文です。

layer_force_draw_depth(force, depth)

論証 タイプ 商品説明
force ブーリアン Whether to enable (set to true または無効（ に設定） Zデプスフォースメント false
depth リアル The new Z depth

リターンです。

N/A

例

if !layer_is_draw_depth_forced()
{
    layer_force_draw_depth(true, 0);
}

上記のコードでは、レイヤーのZ深度が強制されているかどうかを確認し、強制されていない場合はZ深度を0に設定し、有効にしています。
