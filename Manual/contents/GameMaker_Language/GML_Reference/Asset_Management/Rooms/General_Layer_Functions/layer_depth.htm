レイヤーの深さ

この関数は、与えられたレイヤーの深さを変更し、その内容が画面にレンダリングされる順序を変更するために使用できます。レイヤー ID を指定します (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) を指定し、設定する新しい深さ (-16000 から 16000 までの整数値) を指定します。レイヤーの深さは、カメラから「遠い」ほど高く、「近い」ほど低くなるように定義されます。したがって、深さ -100, 0, 100 の 3 つのレイヤーがある場合、レイヤーは 100, 0, -100 の順序で描かれ、「トップ」レイヤー（つまり、カメラビューに最も近く、他のすべての上に描かれます）は、深さが -100 のレイヤーになります。

次の画像は、レイヤーの深さがどのように機能するかを示す概略図です。

なお、レイヤーの深さは、いつでも関数 layer_get_depth().また、レイヤーの最小および最大の深さは-16000から16000であり、これらの深さ以外のものはレンダリングされないことに注意してください。この範囲外の深さが必要な場合は、関数 layer_force_draw_depth().

レイヤーの深さを変更すると、適用されるフィルタやエフェクトが変わることがあります。FXレイヤーの深さより低く変更すると、そのレイヤーにエフェクトが適用されなくなります。

構文です。

layer_depth(layer_id, depth)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to set the depth of (or the layer name as a string)
depth リアル The new depth for the layer

リターンです。

N/A

例

if layer_get_depth(layer) != -100
{
    layer_depth(layer, -100);
}

上記のコードでは、コードを実行しているインスタンスが乗っているレイヤーの深さを取得し、それが-100でなければ深さを-100に設定します。
