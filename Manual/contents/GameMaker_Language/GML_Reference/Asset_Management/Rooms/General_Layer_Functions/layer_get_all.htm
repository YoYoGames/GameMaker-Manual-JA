レイヤーをすべて取得する

この関数は、room の各レイヤーの一意な ID 値を入力した配列を返します。

構文です。

layer_get_all()

リターンです。

Array (populated with Layer IDs)

例

var a = layer_get_all();
for (var i = 0; i < array_length(a); i++)
{
    layer_destroy(a[i]);
}

上記のコードは、room にあるすべてのレイヤーを取得し、その ID 値を配列に追加しています。この配列は、room のレイヤーを破棄するために解析されます。
