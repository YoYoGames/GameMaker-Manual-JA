layer_get_id

この関数は、指定されたレイヤーのユニークな ID 値を取得するために使用することができます。IDE では、すべてのレイヤーには名前とタイプがあり、コードで編集や変更ができるようにするには、レイヤー ID値を与える必要があります。この関数は、レイヤーの名前（astring ）（IDE ）を使って、この ID を取得するために使用されます。もし、コードで新しいレイヤーを作成する場合、関数 layer_create()の場合、その関数は代わりにユニークな ID 値を返します (動的に作成されたレイヤーは名前を取得しません)。現在のroom に存在しないレイヤーの名前を指定すると、エラーが発生し、プロジェクトがクラッシュすることに注意してください。

構文です。

layer_get_id(layer_name)

論証タイプ 商品説明
layer_name The name of the layer (a string)

リターンです。

Layer ID or -1 (if the layer specified doesn't exist)

例

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

上記のコードでは、まず与えられたx/y位置に最も近いインスタンスのインデックスを取得し、ローカル変数に格納します。次に、"Instances Front "というレイヤーの一意のインスタンスレイヤーのIDを取得し、見つかったインスタンスをそのレイヤー上に移動させます。
