ルームファースト

この読み取り専用変数は、ゲーム内の一番最初のroom のインデックスを返します (これは、rooms がルームマネージャに表示される順番で定義され、作成された順番ではありません)。

構文です。

room_first;

リターンです。

Room Asset

例

if lives < 1
{
    room_goto(room_first);
}

上記のコードは、変数「lives」をチェックして、それが1より小さい場合、ゲームの最初のroom 。
