ルームインスタンス追加

この関数を使うと、現在のインスタンス以外のroom に、その中の任意の位置にインスタンスを追加することができますroom 。この関数は、一意の idインスタンスのプロパティを設定するための関数呼び出しに使用することができます。現在のroom でインスタンスを作成したい場合は、関数 instance_create_layer().room Browser で作成したasset Asset でこの関数を呼び出すと、そのインスタンスが永久にその部屋に追加されます。また、 game_restart() を呼び出してもroom は元の状態に戻りません（ゲームを終了して再び開くとroom が再び元の状態で開始するだけ）ので、注意してください。

構文です。

room_instance_add(index, x, y, obj);

論証タイプ 商品説明
index The index of the room to add an object instance to.
x The x position of the new instance.
y The y position of the new instance.
obj The object to add an instance of.

リターンです。

Instance ID

例

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_add(global.rm, 100, 100, obj_Player);

上記のコードでは、ゲームに新しいroom を追加し、"rm_Base" というインデックスを持つroom の内容をそこにコピーしています。そして、object "obj_player" のインスタンスを、この新しいroom の 100,100 の位置に追加します。
