layer_tilemap_get_id

この関数は、レイヤー上のtile マップ要素の一意な ID 値を取得するために使用することができます。レイヤー ID を指定します (レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) を入力すると、この関数はレイヤー上のtile マップ要素に関連する ID 値を返します。tile この関数は、特にIDE で追加されたtile マップに使用するために設計されていることに注意してください。 layer_tilemap_create()この場合、追加されたtile の地図要素に対応するユニークな ID が返されます。与えられたタイルマップIDが正しくない場合，あるいはタイルマップが存在しない場合，この関数は -1 を返します．

構文です。

layer_tilemap_get_id(layer_id)

論証 タイプ 商品説明
layer_id 文字列または レイヤーID The unique ID value of the layer to target (or the layer name as a string)

リターンです。

Tile Map Element ID or -1

例

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

上記のコードでは、「tilemap_trees」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上のtile マップ要素のIDを取得します。この ID を使って、レイヤーからtile マップを削除しています。
