タイルセットインデックス

この関数は、タイルデータの集合に対して、tile のインデックス（tile 画像内のtile set の位置）を設定するために使用することができます。この関数で指定するタイルデータは，関数 tilemap_get()そして，tile 内のtile set の位置（インデックス）を表す整数値を返します． この関数は，修正したタイルデータセットを返し，それを tilemap_set()関数があります。

構文です。

tile_set_index(tiledata, index)

論証 タイプ 商品説明
tiledata タイルデータ the tile-data to set
index リアル The integer index value of the tile to use

リターンです。

Tile Data

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var ind = tile_get_index(data);
data = tile_set_index(data, irandom(23));
tilemap_set(map_id, data, mx, my);

上記のコードは、与えられたレイヤーからtile マップ ID を取得し、それを使ってマウス位置の下にあるセルのタイルデータを取得します。このデータを使って、tile のインデックスを乱数に設定する。
