layer_tilemap_destroy

この関数は、与えられたtile の地図要素を破棄します。tile マップ ID (tile マップを layer_tilemap_create()またはレイヤーIDを layer_get_tilemap_id()) を削除します。tile もしtile マップがroom エディターで追加されたものであれば、次にroom を離れて戻ってきたとき、tile マップが再び作成されることに注意してください。しかし、room が永続的であれば、tile の永続性を再びオフにしない限り、room のマップは削除されます。

構文です。

layer_tilemap_destroy(tilemap_element_id)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map to be destroyed

リターンです。

N/A

例

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if (layer_tilemap_exists(lay_id, tile_id))
{
    layer_tilemap_destroy(tile_id);
}

上記のコードは、レイヤー "Tiles_trees" に与えられたtile マップ要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（ただしレイヤーは破棄しません）。
