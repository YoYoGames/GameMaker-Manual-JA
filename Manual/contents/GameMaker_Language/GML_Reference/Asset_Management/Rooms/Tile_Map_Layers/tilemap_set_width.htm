タイルマップの幅を設定する

この関数は、tile の地図要素のサイズを変更するために使用することができます。tile マップエレメント ID (tile マップエレメントを作成する際に layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) と、tile マップの新しい幅 (tile セル) を指定します。

構文です。

tilemap_set_width(tilemap_element_id, width)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to set the width of
width リアル The width value (in "cells")

リターンです。

N/A

例

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_width(map_id) != room_width div 16)
{
    tilemap_set_width(map_id, room_width div 16);
}

上記のコードは、特定のtile マップの幅をチェックし、それが正しいサイズでない場合、幅が設定されます。
