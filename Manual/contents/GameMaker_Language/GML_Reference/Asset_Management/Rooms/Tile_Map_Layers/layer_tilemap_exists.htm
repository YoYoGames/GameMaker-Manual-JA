layer_tilemap_exists

この関数は、任意のレイヤにtile の地図要素が存在するかどうかを確認するために使用します。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) とtile マップ要素 ID (tile マップ要素を作成するときに layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) そして、この関数は、その要素が存在する場合は true を、存在しない場合は false を返します。

注：この関数は、現在のターゲットroom の範囲内で動作します。デフォルトでは、この関数が呼び出されたroom で、この範囲は関数 layer_set_target_room().

構文です。

layer_tilemap_exists(layer_id, tilemap_element_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to check

リターンです。

Boolean

例

var lay_id = layer_get_id("tilemap_trees");
if layer_tilemap_exists(lay_id, global.Treestilemap)
{
    layer_tilemap_destroy(lay_id, global.Treestilemap);
}

上記のコードは、レイヤー "tilemap_trees" に与えられたtile マップ要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（レイヤーは破棄しません）。
