タイルマップの取得

この関数を使うと、room マップエレメントの x 位置（tile 内）を取得することができます。tile マップエレメント ID (tile マップエレメントを作成するときに layer_tilemap_create() を使用した場合、または関数 layer_tilemap_get_id())、この関数はX軸の位置を返します。

構文です。

tilemap_get_x(tilemap_element_id);

論証タイプ 商品説明
tilemap_element_id The unique ID value of the tile map element to get the x position of

リターンです。

例

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_x(map_id);
var _y = tilemap_get_y(map_id);
tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10);

上記のコードでは、取得したtile のマップ ID を使ってtile の x、y 位置を取得し、その値を使ってtile のマップを移動しています。
