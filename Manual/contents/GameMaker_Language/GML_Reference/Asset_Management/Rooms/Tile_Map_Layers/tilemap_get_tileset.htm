tilemap_get_tileset

この関数を使用すると、レイヤー上の指定されたTile Set マップエレメントに割り当てられたasset tile のインデックス値を取得することができます。Tile マップエレメント ID (tile マップエレメントを作成するときに layer_tilemap_create()を使って作成したときや、関数 layer_tilemap_get_id()を使用してTile Set asset マップ要素を作成したとき、または関数を使用したときに取得します）、この関数は インデックスを返します。

構文です。

tilemap_get_tileset(tilemap_element_id);

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the Tile Map element to get the tile set from

リターンです。

Tile Set Asset

例

var lay_id = layer_get_id("Tiles_trees");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(map_id) != ts_Nighttime
{
    tilemap_tileset(map_id, ts_Nighttime);
}

上記のコードは、レイヤー "Tiles_trees" に割り当てられた現在のtile set をチェックし、それが "ts_Nighttime" でない場合、そのtile set はtile マップに割り当てられます。
