タイルマップの取得

この関数を使用すると、tile マップエレメントのセルからtile のデータを取得することができます。tile マップエレメント ID (tile マップエレメントを作成する際に layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) とセルのx と y の位置を指定すると、この関数はtile データの "blob" を返します。このデータは基本的にビットマスクで、tile インデックス、フリップ/ローテート/ミラーブーリアン、適用されたマスクデータ( tilemap_set_mask()を作成し、そのデータ値を tile_* 関数で使用して、tiles のプロパティを変更することができます。tile の位置としてではなく、room マップグリッドのサイズに基づいた x/y セル位置を使用していることに注意してください。したがって、「cell_x」は 0 からタイルマップ幅までの値、「cell_y」は 0 からタイルマップ高さまでの値です。もし、特定のtile の位置にあるroom のデータを取得する必要がある場合は、関数を使用する必要がある。 tilemap_get_at_pixel().

重要! tiles マップのtile が変更されていない場合 (すなわち、回転や反転などされていない場合)、tile set データの戻り値はtile 上のtile set のインデックスと正確に一致します。 したがって、たとえばtiles のインデックス 1 のtile を使用してtile set の「衝突マップ」を作成し、この関数で 1 または 0 (空のtile) をチェックして衝突を計算することができます。

構文です。

tilemap_get(tilemap_element_id, cell_x, cell_y);

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to get the tile-data of
x_cell リアル The cell along the x-axis to get the tile data from
y_cell リアル The cell along the y-axis to get the tile data from

リターンです。

Tile Data (-1 if there is an error)

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上記のコードでは、レイヤー「Tiles_Sky」のtile マップの ID を取得し、それを使ってtile のセル (0, 0) からデータを取得しています。このtile のデータを反転させてから、再びtile のマップにtile を設定するために使用します。
