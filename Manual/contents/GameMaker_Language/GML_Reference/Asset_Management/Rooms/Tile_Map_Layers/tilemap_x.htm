タイルマップ_x

この関数は、レイヤー上のroom asset マップエレメントのtile の x 軸に沿った位置を制御します。tile マップエレメント ID (これはtile マップエレメントを layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) を設定し、使用する x 値を設定します (room の座標に基づく)。

構文です。

tilemap_x(tilemap_element_id, x);

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to change
x リアル The x position for the tile map

リターンです。

N/A

例

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_x(map_id, irandom(room_width));

上記のコードは、レイヤー "Asset_sky" に割り当てられたtile マップasset の ID 値を取得し、その x 位置を 0 からroom の幅の間のランダムな値に設定するものである。
