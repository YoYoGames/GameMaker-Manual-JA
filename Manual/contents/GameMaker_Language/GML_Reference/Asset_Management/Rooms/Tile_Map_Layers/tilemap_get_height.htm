tilemap_get_height

この関数を使用すると、tile の地図要素の高さ（セル単位）を取得できます。tile マップエレメント ID (tile マップエレメントを作成する際に layer_tilemap_create()または を使用すると layer_tilemap_get_id())、この関数は高さ(セル幅の数)を返します。

構文です。

tilemap_get_height(tilemap_element_id);

論証タイプ 商品説明
tilemap_element_id The unique ID value of the tile map element to get the height of

リターンです。

例

var lay_id = layer_get_id("espadrilles");
var map_id = layer_tilemap_get_id(lay_id);
var _w = tilemap_get_width(map_id);
var _h = tilemap_get_height(map_id);
total_tiles = (_w * _h);

tile 上記のコードでは、取得したtile マップ ID を使ってtile マップの幅と高さを取得し、その値を使ってインスタンス変数を設定しています。
