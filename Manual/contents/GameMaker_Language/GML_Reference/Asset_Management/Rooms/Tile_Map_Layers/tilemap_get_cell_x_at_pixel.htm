タイルマップのセル座標を取得します。

この関数を使用すると、tile 内の相対的な x 軸位置を指定して、個々のroom マップセルの x 軸位置を取得できます。tile マップ要素 ID (tile マップ要素を作成する際に layer_tilemap_create() を使用した場合、または関数 layer_tilemap_get_id()) と、部屋内のx と y の位置を指定すると、この関数はその点のtile マップ内のセルの x 位置を返します。 なお、値がtile マップの領域外にあり、セルがない場合は、-1 を返します。

構文です。

tilemap_get_cell_x_at_pixel(tilemap_element_id, x, y);

論証タイプ 商品説明
tilemap_element_id The unique ID value of the tile map element to get the cell x position of
x The x position within the room to use for getting the cell
y The y position within the room to use for getting the cell

リターンです。

(x-axis cell position or -1 if there is an error)

例

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

上記のコードでは、取得したtile のマップ ID を使って、room の座標セットからセル位置を取得し、そこで見つかったtile のデータをインスタンス変数に格納しています。
