tilemap_get_at_pixel

この関数を使うと、tile マップエレメントの位置（room 内）からtile のデータを取得することができます。tile マップエレメント ID (tile マップエレメントを作成するときに layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) と、tile データを取得する部屋のx と y の位置を指定すると、この関数はtile データの "ブロブ" を返します。このデータは基本的にビットマスクで、tile のインデックス、フリップ/ローテート/ミラーブーリアン、および適用されたマスクデータ( tilemap_set_mask()を作成し、そのデータ値を tile_* 関数で使用して、tiles のプロパティを変更することができます。もし、特定のtile セルからtile データを取得する必要がある場合は、関数 tilemap_get()の代わりに

重要! tiles マップのtile が変更されていない場合 (すなわち、回転や反転などされていない場合)、tile set データの戻り値はtile 上のtile set のインデックスと正確に一致します。 したがって、たとえばtiles のインデックス 1 のtile を使用してtile set の「衝突マップ」を作成し、この関数で 1 または 0 (空のtile) をチェックして衝突を計算することができます。

構文です。

tilemap_get_at_pixel(tilemap_element_id, x, y);

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to get the tile-data of
x リアル The position along the x-axis to get the tile data from (in room pixels)
y リアル The position along the y-axis to get the tile data from (in room pixels)

リターンです。

Tile Data (-1 if there is an error)

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

上記のコードでは、レイヤー「Tiles_Sky」上のtile マップ「Clouds」の ID を取得し、それを使ってtile のデータを位置 (64, 128) から取得しています。このtile のデータを反転させてから、再びtile のマップにtile を設定するために使用します。
