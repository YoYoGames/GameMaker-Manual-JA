タイルマップセット

この関数は、レイヤー上のtile マップエレメント内の任意のセル（グリッドマス）を、新しいtile に設定するために使用します。tile マップエレメントの ID（これは、tile マップエレメントを layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) 、そして設定するtile データとtile マップ内の位置を指定します。tile のデータを取得するには、関数 tilemap_get()を作成し、 tile_ 関数で変更してから、この関数を使用してセルを設定します。この関数は、tile が正常に設定された場合は true を返し、問題があって設定されなかった場合は false を返します。

構文です。

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to change
tiledata タイルデータ (実数) The tile data to set
xcell リアル The cell (grid) position to set along the x-axis
ycell リアル The cell (grid) position to set along the y-axis

リターンです。

Boolean

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

上記のコードでは、レイヤー「Tiles_Sky」のtile マップの ID を取得し、それを使ってtile のセル (0, 0) からデータを取得しています。このtile のデータを反転させてから、再びtile のマップにtile を設定するために使用します。
