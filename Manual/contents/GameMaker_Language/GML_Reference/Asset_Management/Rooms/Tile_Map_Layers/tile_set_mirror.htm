tile_set_mirror

この関数は、与えられた tile-data のセットをtile にミラーリングするかどうか設定するために使用することができます。このタイルデータは，関数 tilemap_get()そして、mirror 引数に、tile をミラーリングしたい場合は true を、tile をデフォルトのミラーリングされていない状態にしたい場合は false を設定します。この関数は，修正されたタイルデータセットを返し，それを tilemap_set()関数があります。

構文です。

tile_set_mirror(tiledata, mirror)

論証タイプ 商品説明
tiledata the tile-data to set
mirror Set to true をミラーリングし、 そのままにしておく。 false

リターンです。

Tile Data (Real)

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_mirror(data);
data = tile_set_mirror(data, bool);
tilemap_set(map_id, data, mx, my);

上記のコードでは、与えられたレイヤーからtile のマップ ID を取得し、マウス下のtile の x と y のセル位置を取得します。この位置からタイルデータを取得し、それをミラーリングして、再びtile を設定するために使用します。
