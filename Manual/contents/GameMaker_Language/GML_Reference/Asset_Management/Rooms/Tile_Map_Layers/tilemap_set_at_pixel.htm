タイルマップの設定画素数

tileこの関数は、レイヤー上のtile マップエレメント内のセルを、tile の実際の位置を使用して、新しいroom に設定するために使用できます。tile マップエレメント ID (tile マップエレメントを作成する際に layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()tile )、設定するroom データ、およびtile 内の位置を指定します。 tilemap_get_at_pixel()を設定し、 tile_ 関数で変更した後、この関数で再度設定します。この関数は、tile が正常に設定された場合は true を返し、問題があって設定されなかった場合は false を返します。

構文です。

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to change
tiledata タイルデータ (実数) The tile set data to set
xcell リアル The x position (in the room)
ycell リアル The y position (in the room)

リターンです。

Boolean

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

上記のコードでは、レイヤー「Tiles_Sky」上のtile マップの ID を取得し、それを使ってtile の位置 (4, 4) からデータを取得しています。このtile のデータを反転させてから、再びtile のマップにtile を設定するために使用します。
