tilemap_get_frame

tiles はアニメーションが可能なので、現在どのフレームが描画されているかを知り、それに応じて反応するのが便利な場合があります。この関数を使用すると、指定したtile マップの現在のフレームインデックスを取得することができます。tile マップ要素 ID (これはtile マップ要素を layer_tilemap_create()を使用した場合、または関数 layer_tilemap_get_id()) で、この関数はフレームインデックスを返します。

構文です。

tilemap_get_frame(tilemap_element_id)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to get the frame index of

リターンです。

Real (between 0 (inclusive) and the maximum number of frames of animation (exclusive))

例

var lay_id = layer_get_id("Tiles_Traps");
var map_id = layer_tilemap_get_id(lay_id);
if (tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4)
{
    global.activate = true;
}
else
{
    global.activate = false;
}

上記のコードは、与えられたレイヤー上の与えられたtile マップの現在のアニメーションフレームをチェックし、その戻り値に基づいてグローバル変数を設定します。
