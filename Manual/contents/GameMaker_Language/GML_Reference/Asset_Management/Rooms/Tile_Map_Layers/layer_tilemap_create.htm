レイヤーマップクリエイト

この関数を使うと、プロジェクトでtile マップとして使用するレイヤにタイルセットリソースを割り当てることができます。レイヤー ID (これは layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します)、そしてtile マップをroom に追加する初期 (x, y) 位置、使用するtile set リソース、そしてtile マップの幅と高さをセルで指定します (たとえば、幅 20、高さ 10 とすると、幅 20 セル、高さ 10 セルの 200 セルのtile マップを作成します)、セルのサイズ自体はtile set で定義されているものを使用します。tiles マップ内の負の位置にtile を配置することはできないので、すべてのtiles はセルスペース 0 から幅 - 1、0 から高さ - 1 の範囲に配置しなければならないことに注意してください。

構文です。

layer_tilemap_create(layer_id, x, y, tileset, width, height)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
x リアル The x position of the tile map in the room
y リアル The y position of the tile map in the room
tileset タイルセットアセット The Tile Set asset to be used
width リアル The width tile map (in cells)
height リアル The height of the tile map (in cells)

リターンです。

Tile Map Element ID

例

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

上記のコードでは、新しいレイヤーを作成し、そこに新しいtile マップ要素を追加します。tile マップの位置を (0,0) に設定するとともに、使用するtile set とtile マップの幅と高さを設定します。
