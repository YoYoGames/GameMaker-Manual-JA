タイルセット回転

この関数は、与えられたtile-dataのセットを、tile 90度回転させるかどうかを設定するために用いることができる。このタイルデータは，関数 tilemap_get()そして、tile を回転させたい場合は true を、tile をデフォルトの回転していない状態にしたい場合は false を rotate 引数に設定します。この関数は，修正されたタイルデータセットを返し，それを tilemap_set()関数があります。

構文です。

tile_set_rotate(tiledata, rotate)

論証 タイプ 商品説明
tiledata タイルデータ The tile-data to set
rotate ブーリアン Set to true で回転させ、 でそのままにします。 false

リターンです。

Real

例

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

上記のコードでは、与えられたレイヤーからtile のマップ ID を取得し、マウス下のtile の x と y のセル位置を取得します。この位置からタイルデータを取得し、回転させた後、再びtile を設定するために使用されます。
