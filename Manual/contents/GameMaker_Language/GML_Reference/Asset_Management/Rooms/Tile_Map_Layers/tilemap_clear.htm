タイルマップクリア

この関数を使うと、指定したタイルマップのtiles をすべてクリア／セットすることができます。tile マップエレメント ID (tile マップエレメントを作成するときに layer_tilemap_create()または関数 layer_tilemap_get_id()) 、そしてレイヤーをクリアしたいtile データを指定します。デフォルトの値 0 は、レイヤーからすべてのtiles をクリアします (本質的にすべてのtiles を「空」にします)。一方、専用の tile_* 関数を使用して、独自のtile データを作成し、tile マップをクリアすることができます。

構文です。

tilemap_clear(tilemap_element_id, tiledata)

論証 タイプ 商品説明
tilemap_element_id タイルマップ要素ID The unique ID value of the tile map element to change
tiledata タイルデータ (実数) The tile data to use to clear the layer

リターンです。

N/A

例

var layer_id = layer_get_id("Forest");
var tile_id = layer_tilemap_get_id(layer_id);
tilemap_clear(tile_id, 0);

上記のコードは、tile エディターで作成したroom マップの ID 値を取得し、「空」tiles を使ってクリアしています。
