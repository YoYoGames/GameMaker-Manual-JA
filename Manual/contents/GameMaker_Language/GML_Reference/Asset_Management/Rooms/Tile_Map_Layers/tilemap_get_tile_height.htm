tilemap_get_tile_height

この関数を使うと、tile マップエレメントの各tile セルの高さ（ピクセル単位）を取得することができます。tile マップエレメント ID (tile マップエレメントを作成する際に layer_tilemap_create() を使用した場合、または関数 layer_tilemap_get_id()) で、この関数はtile set のセルの高さを返します。

構文です。

tilemap_get_tile_height(tilemap_element_id);

論証タイプ 商品説明
tilemap_element_id The unique ID value of the tile map element to get the tile cell height of

リターンです。

例

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

上記のコードでは、取得したtile のマップ ID を使ってtile のセルの幅と高さを取得し、それを使って 2 つのグローバル変数を設定しています。
