layer_sprite_xscale

この関数は、レイヤー上の指定されたsprite 要素を X 軸に沿って拡大縮小するかどうかを変更することができます。sprite 要素の ID (sprite 要素を作成するときに layer_sprite_create()または関数 layer_sprite_get_id()) を設定し、スケール値を設定します。スケール値 1 はスケーリングなし (1:1) を示し、小さい値はスケーリングダウン (たとえば 0.5 は使用するsprite の幅の半分)、大きい値はスケーリングアップ、負の値はsprite を反転してスケーリングします。使用する値がちょうど -1 (この場合、使用するsprite はスケーリングなしでその原点に対して単に右から左に反転されます) である場合は、この値は使用されません。

構文です。

layer_sprite_xscale(sprite_element_id, xscale);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
xscale The xscale value (default is 1)

リターンです。

例

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

上記のコードは、コードを実行しているインスタンスのレイヤーに割り当てられたsprite 要素をチェックし、それがいずれかの方向にスケールされている場合、X 軸のスケールと Y 軸のスケールを共に 1 に設定するものです。
