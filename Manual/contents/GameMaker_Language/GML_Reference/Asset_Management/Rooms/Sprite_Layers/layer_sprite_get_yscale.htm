layer_sprite_get_yscale

この関数は、sprite 要素の y 軸に沿った現在のスケール倍率を取得するために使用できます。sprite 要素 ID (sprite 要素を作成する際に layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id()この関数は，y 軸のスケールを実数で返します（1 はスケーリングなし）。

構文です。

layer_sprite_get_yscale(sprite_element_id);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to get the information from

リターンです。

例

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_yscale(spr_id) != 1)
{
    layer_sprite_yscale(spr_id, 1);
}

上記のコードでは、「sprite_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上のsprite 要素のIDを取得します。このIDを使ってyscaleの値を調べ、1以外であれば1に設定します。
