レイヤースプライトが存在する

この関数は、任意のレイヤーにsprite の要素が存在するかどうかを確認するために使用します。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) とsprite 要素 ID (sprite 要素を作成するときに layer_sprite_create()を使用したとき、または関数 layer_sprite_get_id()) そして、この関数は、その要素が存在する場合は true を、存在しない場合は false を返します。

注：この関数は、現在のターゲットroom の範囲内で動作します。デフォルトでは、この関数が呼び出されたroom で、この範囲は関数 layer_set_target_room().

構文です。

layer_sprite_exists(layer_id, sprite_element_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
sprite_element_id スプライトエレメントID The unique ID value of the sprite element to check

リターンです。

Boolean

例

var lay_id = layer_get_id("Asset_Trees");
var spr_id = layer_sprite_get_id(lay_id, "graphic_254367CB");

if (layer_sprite_exists(lay_id, spr_id))
{
    layer_sprite_destroy(lay_id, global.TreesSprites);
}

上記のコードは、レイヤー "Asset_trees" に与えられたsprite 要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（レイヤーは破棄しません）。
