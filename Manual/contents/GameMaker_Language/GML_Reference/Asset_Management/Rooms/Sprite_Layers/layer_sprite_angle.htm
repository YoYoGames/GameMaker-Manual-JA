レイヤースプライト角

この機能を使うと、レイヤー上の指定されたsprite 要素の角度を変更することができます。sprite 要素の ID (sprite 要素を作成するときに layer_sprite_create()または関数 layer_sprite_get_id()0は右、90は上、180は左、270は下を表します。360以上の値を設定すると、0〜359の範囲に収まるようにループされます。

構文です。

layer_sprite_angle(sprite_element_id, angle);

論証 タイプ 商品説明
sprite_element_id スプライトエレメントID The unique ID value of the sprite element to change
angle リアル The angle of the sprite (default is 0)

リターンです。

N/A

例

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if (layer_sprite_get_angle(asset_sprite) != 0)
{
    layer_sprite_angle(asset_sprite, 0);
}

上記のコードは、コードを実行しているインスタンスのレイヤーに割り当てられたsprite 要素をチェックし、その角度が 0 でない場合は 0 に設定されます。
