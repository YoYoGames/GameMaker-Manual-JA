レイヤースプライトチェンジ

この機能を使うと、レイヤー上の任意のsprite 要素に割り当てられたsprite リソースを変更することができます。sprite 要素の ID (これは、sprite 要素を layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id()) を指定し、sprite インデックス要素asset を指定すると、変更されます。sprite にサブ画像がある場合、それもアニメーションすることに注意してください (この制御には layer_sprite_index()と layer_sprite_speed()関数を使用します)。なお、新しいsprite のインデックスに -1 を指定すると、sprite は表示されませんが、要素はまだ存在し、後で再び変更することが可能です。

構文です。

layer_sprite_change(sprite_element_id, sprite_index)

論証 タイプ 商品説明
sprite_element_id スプライトエレメントID The unique ID value of the sprite element to change
sprite_index スプライトアセット The new sprite index to use

リターンです。

N/A

例

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if (layer_sprite_get_sprite(s) != spr_Nighttime)
{
    layer_sprite_change(s, spr_nighttime);
}

上記のコードでは、グローバル変数 global.Asset_sprite に格納されているIDで要素のsprite インデックスをチェックし、それが「spr_Nighttime」でない場合は、そのsprite が要素に割り当てられます。
