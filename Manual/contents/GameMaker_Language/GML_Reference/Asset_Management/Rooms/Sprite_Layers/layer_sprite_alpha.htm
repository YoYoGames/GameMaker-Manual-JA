レイヤースプライトアルファ

sprite レイヤー上のasset のアルファ（透明度）を制御する機能です。sprite の要素 ID (sprite の要素作成時に layer_sprite_create()を使用した場合、または 関数 layer_sprite_get_id()) を設定し、使用するアルファ値を設定します。アルファ値は 0 (完全な透明) から 1 (完全な不透明) の間で指定でき、sprite 要素のデフォルトのアルファ値は 1 です。注意 sprite 要素が割り当てられているレイヤーが表示されていない場合、または要素自体が非表示になっている場合、そのレイヤーまたは要素が再度表示されるまで、この関数による違いは表示されません。

構文です。

layer_sprite_alpha(sprite_element_id, alpha);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
alpha The alpha for sprite sprite, from 0 to 1 (default is 1)

リターンです。

例

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

sprite asset 上記のコードでは、レイヤー「Asset_sky」に割り当てられた「Clouds」のID値を取得し、そのアルファ値に0〜1のランダムな値を設定しています。
