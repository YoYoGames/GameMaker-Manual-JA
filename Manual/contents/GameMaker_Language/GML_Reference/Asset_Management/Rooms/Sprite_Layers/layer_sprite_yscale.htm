レイヤースプライトアイスケール

この関数は、レイヤー上の指定されたsprite 要素を y 軸方向に拡大縮小するかどうかを変更することができます。sprite 要素の ID (sprite 要素を作成するときに layer_sprite_create()または関数 layer_sprite_get_id()) を設定し、スケール値を設定します。スケール値1はスケーリングなし（1:1）を示し、小さい値はスケーリングダウン（例えば0.5は使用されるsprite の高さの半分になる）、大きい値はスケーリングアップ、負の値はsprite をミラーリングしてスケーリングします。使用する値がちょうど-1（この場合、sprite はスケールなしでその (0, 0) 位置について上から下へミラーリングするだけ）。

構文です。

layer_sprite_yscale(sprite_element_id, yscale);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
yscale The yscale value (default is 1)

リターンです。

例

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
    layer_sprite_xscale(asset_sprite, 1);
    layer_sprite_yscale(asset_sprite, 1);
}

上記のコードは、コードを実行しているインスタンスのレイヤーに割り当てられたsprite 要素をチェックし、それがいずれかの方向にスケールされている場合、X 軸のスケールと Y 軸のスケールを共に 1 に設定するものです。
