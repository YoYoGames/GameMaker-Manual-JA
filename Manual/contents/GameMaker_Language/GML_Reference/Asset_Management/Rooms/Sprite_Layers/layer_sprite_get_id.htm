layer_sprite_get_id

この関数は、レイヤー上のsprite 要素の一意な ID 値を取得するために使用することができます。レイヤー ID を指定します (この値は layer_create()またはレイヤー名を layer_get_id()) と、sprite エディターで定義されたRoom 要素の名前です。この関数は、レイヤー上のそのsprite 要素に関連する ID 値を返します。

sprites この関数は、特にIDE レイヤーのasset に追加されたsprite で使用するために設計されていることに注意してください。 layer_sprite_create()その場合、その関数が返す ID を今後のすべての参照に使用する必要があります (そのsprite 要素にはこの関数に渡すべき名前がないため)。

指定されたレイヤーが存在しない場合、または指定されたsprite 要素が見つからない場合、この関数は -1 を返します。

構文です。

layer_sprite_get_id(layer_id, sprite_element_name)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target
sprite_element_name 文字列 The unique name of the sprite element on the layer as defined in the Room Editor

リターンです。

Sprite Element ID

例

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB");
layer_sprite_change(back_id, spr_Trees_Winter);

上記のコードでは、「Assets_trees」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上のsprite 要素「graphic_254367CB」のIDを取得することにしています。次に、取得したsprite 要素の ID を使って、要素のsprite を変更します。
