layer_sprite_get_sprite

この関数は、sprite 要素の現在のsprite インデックスを取得するために使用することができます。sprite 要素の ID (これは、sprite 要素を作成するときに layer_sprite_create()を使用したとき、または関数 layer_sprite_get_id()) 、この関数は表示されているsprite のインデックスを表す実数値を返します。もし、その要素にsprite が割り当てられていなければ、この関数は -1 を返します。

構文です。

layer_sprite_get_sprite(sprite_element_id);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to get the information from

リターンです。

Sprite Asset or -1

例

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_sprite(spr_id) != spr_Clouds
{
    layer_sprite_change(spr_id, spr_Clouds);
}

上記のコードでは、「sprite_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上のsprite 要素のIDを取得します。このIDを使って、その要素に割り当てられているsprite を確認し、sprite "spr_Clouds" でない場合は、それを設定します。
