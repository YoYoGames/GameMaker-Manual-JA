layer_sprite_x

この関数は、レイヤー上のroom asset 要素のsprite の x 軸に沿った位置を制御します。sprite 要素の ID (sprite 要素を作成するときに layer_sprite_create() を使用したとき、または関数 layer_sprite_get_id()) を設定し、使用する x 値を (room の座標に基づいて) 設定します。

構文です。

layer_sprite_x(sprite_element_id, x);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
x The x position for the sprite

リターンです。

例

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_x(spr_id, irandom(room_width));

上記のコードでは、レイヤー「Asset_sky」に割り当てられたsprite asset 「Clouds」のID値を取得し、そのx位置を0からroom の幅の間のランダムな値に設定しています .
