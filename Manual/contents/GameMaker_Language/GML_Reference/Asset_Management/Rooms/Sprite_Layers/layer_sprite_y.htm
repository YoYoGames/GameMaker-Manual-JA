layer_sprite_y

この関数は、レイヤー上のroom asset 要素のsprite の y 軸に沿った位置を制御します。sprite 要素の ID (sprite 要素を作成するときに layer_sprite_create() を使用したとき、または関数 layer_sprite_get_id()) を設定し、使用する y 値を (room の座標に基づいて) 設定します。

構文です。

layer_sprite_y(sprite_element_id, y);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
y The y position for the sprite

リターンです。

例

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_y(spr_id, 50 + irandom(room_height - 200));

上記のコードは、レイヤー「Asset_sky」に割り当てられたsprite asset 「Clouds」のID値を取得し、そのy位置を50からroom の高さから200を引いた値の間でランダムに設定するものです。
