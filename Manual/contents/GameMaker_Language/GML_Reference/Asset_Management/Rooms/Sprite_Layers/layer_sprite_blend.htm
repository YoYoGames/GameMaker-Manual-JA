レイヤースプライトブレンド

この関数は、sprite sprite のブレンド（または「色合い」）を制御します。デフォルト値は -1 です（定数 c_white を表し、これも使用可能です）。その他の値（ c_red のような内部色定数を含む。 または c_aqua) は、指定された色と元のsprite をブレンドします。sprite の要素 ID を指定します (sprite の要素作成時に layer_sprite_create()を使用した場合、または 関数 layer_sprite_get_id())、使用するブレンディングカラーを設定します。以下は、異なる色でブレンドされたsprite の例です。

なお、HTML5プラットフォームでは、（WebGLを使用していない限り）ブレンディングを制限するようにしてください。 ブレンドされたsprite はそれぞれ個別にキャッシュされるため、ブレンドされたsprites が多数あるとパフォーマンスに悪影響が出る可能性があるからです (cache のサイズを設定するには、関数 sprite_set_cache_size()).

構文です。

layer_sprite_blend(sprite_element_id, blend);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to change
blend The colour to blend with the sprite sprite (default is c_white)

リターンです。

例

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_blend(spr_id, c_gray);

sprite 上記のコードは、レイヤー「Asset_sky」に割り当てられた「Clouds」のID値を取得し、それを色に染めています。
