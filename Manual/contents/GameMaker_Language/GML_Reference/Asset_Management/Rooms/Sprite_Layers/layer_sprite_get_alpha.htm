layer_sprite_get_alpha

この関数は、sprite 要素のアルファ値を取得するために使用することができます。sprite 要素の ID (これは、sprite 要素を layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id()この関数は、0 (完全に透明) から 1 (完全に不透明) までの値を返します。）

構文です。

layer_sprite_get_alpha(sprite_element_id);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to get the information from

リターンです。

(from 0 to 1)

例

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_alpha(spr_id) < 0.1)
{
    layer_sprite_destroy(spr_id);
}

上記のコードでは、「sprite_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上のsprite 要素のIDを取得します。このIDを使って要素のアルファ値を調べ、0.1より小さい場合は、そのレイヤー要素を破壊しています。
