レイヤースプライトインデックス

この関数は、1つのレイヤーに複数のサブ画像を持つsprite asset の画像インデックスを設定するために使用することができます。sprite の要素 ID (sprite の要素作成時に layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id()) を設定し、使用する画像インデックスを設定します。サブ画像の範囲外の値を設定すると、画像インデックスがloop 周回します。

構文です。

layer_sprite_index(sprite_element_id, image_index);

論証 タイプ 商品説明
sprite_element_id スプライトアセット The unique ID value of the sprite element to set
index リアル The image index to use for the sprite

リターンです。

N/A

例

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

上記のコードは、「sprite_trees」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の与えられたsprite 要素のIDを取得します。この ID を使って、要素の画像インデックスを変更します。
