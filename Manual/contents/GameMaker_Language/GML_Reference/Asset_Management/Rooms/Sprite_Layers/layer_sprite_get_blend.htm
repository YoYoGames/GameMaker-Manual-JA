レイヤースプライト生成ブレンド

この関数は、sprite 要素のブレンドカラーを取得するために使用することができます。sprite 要素の ID (これは、sprite 要素の作成時に layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id())、この関数は割り当てられた色を表す実数値を返します。

構文です。

layer_sprite_get_blend(sprite_element_id);

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite element to get the information from

リターンです。

Colour Constant

例

var lay_id = layer_get_id("sprite_sky");
var back_id = layer_sprite_get_id(lay_id, "Clouds");
if (layer_sprite_get_blend(back_id) == c_white)
{
    layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上記のコードでは、「sprite_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤーにあるsprite の要素のIDを取得します。このIDを使って要素のブレンドカラーを調べ、定数 c_white と同じであれば、要素のブレンドカラーをランダムに設定します。
