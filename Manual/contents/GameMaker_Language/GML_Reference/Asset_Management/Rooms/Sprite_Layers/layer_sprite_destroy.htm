レイヤースプライトデストロイ

この関数は、与えられたsprite の要素を破棄します。sprite ID (これは、sprite を layer_sprite_create()またはレイヤーIDを layer_get_sprite_id()) を削除します。sprite sprite がroom エディタで追加されたものである場合、次にroom を残して戻ったとき、sprite が再び作成されることに注意してください。しかし、room が永続化されている場合、sprite の永続化を再びオフにしない限り、room は削除されます。

構文です。

layer_sprite_destroy(sprite_element_id)

論証タイプ 商品説明
sprite_element_id The unique ID value of the sprite to be destroyed

リターンです。

例

var lay_id = layer_get_id("Asset_Trees");
if (layer_sprite_exists(lay_id, global.Asset_spr_trees))
{
    layer_sprite_destroy(lay_id);
}

上記のコードは、レイヤー "Asset_Trees" に与えられたsprite 要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（レイヤーは破棄しません）。
