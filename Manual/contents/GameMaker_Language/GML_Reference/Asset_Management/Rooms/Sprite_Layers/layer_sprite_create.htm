レイヤースプライトの作成

この関数を使用すると、プロジェクトで使用するレイヤーにsprite リソースを割り当てることができます。レイヤー ID を指定します。 layer_create()または、レイヤー名を layer_get_id()) 、room 内の位置、sprite インデックス (sprite ブラウザで表示されるAsset の名前になります) を指定すると、レイヤに追加されます。この関数は、その要素の一意な ID 値を返し、それをsprites のさらなるレイヤー関数で使用することができます。

構文です。

layer_sprite_create(layer_id, x, y, sprite)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target
x リアル The x position to use
y リアル The y position to use
sprite スプライトアセット The sprite index to be used

リターンです。

Sprite Element ID

例

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
{
    var _x = random(room_width);
    var _y = room_height - 100;
    global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees);
}

上記のコードでは、新しいレイヤーを作成し、それに新しい10個の要素（sprite ）を追加し、各要素のIDを配列に格納しています。
