レイヤースプライトスピード

この機能は、sprite の要素に割り当てられたアニメーション画像の速度を設定するために使用します。sprite 要素の ID（sprite 要素を作成する際に layer_sprite_create()を使用した場合、または関数 layer_sprite_get_id()) を設定し、アニメーションの速度を設定します。IDE が使用するsprite のアニメーション速度を Frames Per Game Frame として計算するように設定している場合、与えられる速度値は乗数であり、1がデフォルト値で、0.5に設定するとアニメーション速度が半分になり、1.5に設定すると2倍になる。しかし、IDE が使用するsprite のアニメーション速度を Frames Per Second として計算するように設定している場合、与えられた速度値は文字通りの Frames Per Second 値であり、例えばこれを 5 に設定すると、ゲームが実行される毎秒にloop 5 フレームのアニメーションが設定されることになります。使用するsprite にサブ画像がない場合、この関数は何の効果もありません。

構文です。

layer_sprite_speed(sprite_element_id, speed);

論証 タイプ 商品説明
sprite_element_id スプライトアセット The unique ID value of the sprite element to change
speed リアル The speed multiplier/value to use for the element

リターンです。

N/A

例

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

上記のコードは、「Assets_water」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の与えられたsprite 要素のIDを取得します。このIDを使って、要素の画像速度を変更します。
