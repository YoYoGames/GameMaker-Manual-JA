部屋幅

この変数は、現在のroom の幅をピクセル単位で保持する。この変数を変更することで、いつでもroom の幅を変更することができる。

構文です。

room_width;

リターンです。

例

if bbox_right > room_width
{
    x += room_width - bbox_right;
}

上記のコードは、現在のインスタンスのsprite のバウンディングボックスがroom の幅より大きいかどうかを確認し、大きい場合はインスタンスを上に移動します。
