room_set_view_enabled

この機能を使うと、ゲーム内で現在表示されている以外のroom の表示を有効（true ）または無効（false ）にすることができます。

構文です。

room_set_view_enabled(index, val);

論証 タイプ 商品説明
index ルームアセット The index of the room to set.
val ブーリアン Whether to enable (true) または無効化 (false) されたビューを、指定されたroom で表示します。

リターンです。

N/A

例

global.myroom = room_add();
room_set_view_enabled(global.myroom, true);

これにより、"global.myroom "にインデックスされたroom のビューが有効になります。
