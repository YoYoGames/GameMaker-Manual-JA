room_persistent

この変数を使用して、現在のflag の persistentroom を取得および設定することができます。 true に設定すると、room は永続的とみなされ、room を離れて再び戻ってくるたびに、そのroom 内のインスタンスの状態は維持されます。しかし、 false に設定されている場合は、room に戻るたびに、初期状態にリセットされます。永続的なroom は、通常のroom よりもかなり多くのメモリを使用することに注意する必要があり、 ゲーム内であまり多くの永続的な を持つことは推奨されません。

構文です。

room_persistent;

リターンです。

例

if lives < 1
{
    room_persistent = false;
    room_goto(rm_start);
}

上記のコードは、変数「lives」をチェックし、それが1より小さい場合、room パーシスタンスを false に設定し、その後rooms を変更します。
