room_get_camera

この関数を使用すると、現在のビュー以外のroom で、特定のビューに割り当てられたカメラの固有のインデックス ID を取得することができます。使用するroom と、使用するビューポート（0から7まで）を指定すると、この関数はカメラのインデックスを返します。

構文です。

room_get_camera(rm, vind);

論証 タイプ 商品説明
rm ルームアセット The index of the room to get the view camera of
vind リアル The index of the view port to get the camera of

リターンです。

Camera ID

例

var cam = room_get_camera(rm_Game, 0);
if (cam != global.MainCam)
{
    room_set_camera(rm_Game, 0, global.MainCam);
}

上記のコードでは、新たに作成したroom のカメラをビューポート[0]に割り当てています。
