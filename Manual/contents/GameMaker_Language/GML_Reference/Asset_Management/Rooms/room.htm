部屋

この変数は、ゲームが現在実行されているroom のroom インデックスを保持します。これは読み取り専用の変数ではありませんが、これを変更しても、現在のroom のインデックス値は変更されず、変数を設定したインデックスに一致するように、room が変更されます。変数を変更したインデックスが有効でない場合、ゲームはエラーを投げて終了するので、この操作には注意が必要です。一般的には room_goto()を変更するには、rooms 。

注意：Room IDは、Asset BrowserやRoom Managerでの順番に基づくものではないので、数値の直接の供給は避けるべきです。代わりに、参照したいasset の部屋定数（これは単に名前である）を使用するか、関数を通じて取得するようにしてください。

構文です。

room;

リターンです。

Room Asset

例

if (room == rm_level1)
{
    audio_play_sound(snd_level1, 1, 1);
}

上記のコードは、現在のroom が最初のレベルかどうかをチェックし、その場合はそのレベルの音楽を再生します。
