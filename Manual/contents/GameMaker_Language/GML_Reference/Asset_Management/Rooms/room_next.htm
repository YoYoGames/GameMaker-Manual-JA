ルームネクスト

この関数を使うと、関数に入力されたroom の次の room のインデックスを取得することができます。例えば、変数 roomを使用して現在のroom のインデックスを取得し、この関数を使用して、ルームマネージャにリストされている、その後に続くroom を探します。入力されたものの後にroom がない場合、-1 が返されます。この関数は、room_add() またはroom_duplicate() を使用して動的に追加されたrooms を認識または考慮しないことに注意してください。

構文です。

room_next(numb);

論証タイプ 商品説明
numb The index of the room to get the next one after.

リターンです。

Room Asset

例

if room_next(room) != -1
{
    room_goto_next();
}

上記のコードは、次のroom が存在するかどうかを確認し、存在する場合はそこに移動します。
