ルームセット幅

この機能を使うと、ゲーム内の現在の幅以外のroom の幅を変更（または設定）することができます。

構文です。

room_set_width(index, w);

論証 タイプ 商品説明
index ルームアセット The index of the room to set the width of.
w リアル The new width of the room in pixels.

リターンです。

N/A

例

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

これは、新しいroom を作成し、そのインデックスを変数 "global.myroom" に格納します。そして、幅を640ピクセルに、高さを480ピクセルに、キャプションを「Game Room」に、永続性を「false」に設定します。
