room_set_persistent

この関数を使うと、ゲーム内の現在のもの以外のroom のパーシスタンスを変更（または設定）することができます。room の永続性を true と設定すると、プレイヤーが一度離れてから戻ってきても、そのroom 内のすべてのインスタンスの状態が維持されます。一方、永続性を false と設定すると、毎回初期状態にリセットされます。永続的なroom は、通常のroom よりもかなり多くのメモリを使用することに注意する必要があり、ゲーム内であまり多く持つことは推奨されません。

注意：この機能は、switch がすでに訪問されている場合、room のパーシステンスをオフにすることはできません！この機能は、 がすでに訪問されている場合、 のパーシステンスをオフにすることはできません。一旦訪問した永続的なroom はメモリに保持されます。switch 永続性をオフにするには、そのroom に行って room_persistent変数を false に設定し、再度room を終了してください。

構文です。

room_set_persistent(index, val);

論証 タイプ 商品説明
index ルームアセット The index of the room to set the persistence of.
val ブーリアン Whether the room should be persistent (true) か、そうでないか (false) です。

リターンです。

N/A

例

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

これは、新しいroom を作成し、そのインデックスを変数 "global.myroom" に格納します。そして、幅を640ピクセルに、高さを480ピクセルに、キャプションを「Game Room」に、永続性を「true」に設定します。
