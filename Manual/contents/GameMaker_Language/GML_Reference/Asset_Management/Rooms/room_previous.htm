前室

この関数を使うと、関数に入力されたroom の前にある room のインデックスを取得することができます。例えば、変数 roomを使用して現在のroom のインデックスを取得し、この関数を使用して、ルームマネージャにリストされている、その前にあるroom を探します。入力したものより前にroom がない場合、-1 が返されます。この関数は、room_add() またはroom_duplicate() を使用して動的に追加されたrooms を認識または考慮しないことに注意してください。

構文です。

room_previous(numb);

論証タイプ 商品説明
numb The index of the room to get the one before from.

リターンです。

Room Asset

例

if room_previous(room) != -1
{
    room_goto_previous();
}

上記のコードは、前のroom が存在するかどうかを確認し、存在する場合はそこに移動します。
