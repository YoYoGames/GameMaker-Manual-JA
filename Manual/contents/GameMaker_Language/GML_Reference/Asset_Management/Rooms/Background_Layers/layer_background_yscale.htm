layer_background_yscale

この関数は、背景要素の Y 軸に沿ったスケールを設定するために使用することができる。背景要素のID（これは layer_background_create()を使用した場合、または関数 layer_background_get_id()を設定し、次にスケール値を設定します。スケール値は、背景要素の表示方法を変更するために使用される乗数で、0.5 の値は要素を半分のスケールで表示し、2 の値は 2 倍のスケールで表示することになります。負の値も有効で、(0,0)の位置で要素を「反転」させるので、Xスケールを-1にすると画像が反転して表示されることに注意しましょう。

構文です。

layer_background_yscale(background_element_id, yscale);

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
yscale リアル The speed multiplier/value to use for the element

リターンです。

N/A

例

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_yscale(back_id, -1);

上記のコードでは、「Background_water」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って、要素のyscaleを変更します。
