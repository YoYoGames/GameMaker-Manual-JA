レイヤーバックグラウンドブレンド

この関数は、背景sprite のブレンド（または「色合い」）を制御します。デフォルト値は -1 です（定数 c_white を表し、これも使用可能です）。その他の値（ c_red のような内部色定数を含む。 または c_aqua) は、指定された色と元のsprite をブレンドします。背景要素のIDを指定します。 layer_background_create() または関数 layer_background_get_id())、使用するブレンディングカラーを設定します。以下は、異なる色でブレンドされたsprite の例です。

HTML5プラットフォームでは、（WebGLを使用していない限り）ブレンディングを制限するようにした方がよいことに注意してください。 ブレンドされたsprite は個別にキャッシュされるため、ブレンドされたsprites が多数あるとパフォーマンスに悪影響が出る可能性があります (cache のサイズを設定するには、関数 sprite_set_cache_size()).

構文です。

layer_background_blend(background_element_id, blend);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to change
blend The colour to blend with the background sprite (default is c_white)

リターンです。

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_blend(back_id, c_aqua);

上記のコードは、レイヤー「Background_sky」に割り当てられている背景のID値を取得し、それを色に染め上げています。
