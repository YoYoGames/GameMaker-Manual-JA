レイヤーの背景を取得する

この関数は、背景要素のアルファ値を取得するために使用することができます。背景要素のID（これは layer_background_create()を使用したとき、または関数 layer_background_get_id()この関数は，0（完全に透明）から1（完全に不透明）までの値を返します。）

構文です。

layer_background_get_alpha(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

(from 0 to 1)

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_alpha(back_id) < 0.1)
{
    layer_background_destroy(back_id);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを元に要素のアルファ値を調べ、0.1未満であればレイヤー要素を破棄します。
