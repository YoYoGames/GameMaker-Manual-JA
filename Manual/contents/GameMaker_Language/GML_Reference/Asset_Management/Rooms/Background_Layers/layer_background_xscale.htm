レイヤーバックグラウンドXスケール

この関数は、背景要素のX軸に沿ったスケールを設定するために使用することができます。背景要素のID（これは layer_background_create() を使用した場合、または関数 layer_background_get_id()を設定し、次にスケール値を設定します。設定したスケール値は、背景要素の表示方法を変更するために使用される乗数です。 0.5なら1/2倍、2なら2倍で表示されます。負の値も有効で、(0,0)の位置で要素を「反転」させるので、xスケールが-1の場合は、次のように表示されることに注意してください。 の画像が反転します。

構文です。

layer_background_xscale(background_element_id, xscale);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to change
xscale The scale value to use (1 is no scaling)

リターンです。

例

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

上記のコードでは、「Background_water」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って、要素のxscaleを変更します。
