レイヤーバックグラウンドストレッチ

この機能を使うと、背景要素sprite をroom に合わせて引き伸ばすか、解像度と 1:1 のままかを切り替えることができます。背景要素のIDを指定します。 layer_background_create()または関数 layer_background_get_id() true false true に設定すると、要素 は に合うように引き伸ばされ ( と のサイズに応じて拡大または縮小される)、 に設定すると、要素 は本来の解像度で表示されるようになります。sprite room sprite room false sprite

構文です。

layer_background_stretch(background_element_id, stretch)

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
stretch ブーリアン The stretch toggle, which can be true または false

リターンです。

N/A

例

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if (layer_background_get_stretch(back))
    {
        layer_background_stretch(back, false);
    }
    else
    {
        layer_background_stretch(back, true);
    }
}

上記のコードは、マウスボタンが押されたかどうかをチェックし、押されたことを検出すると、コードを実行しているインスタンスのレイヤーに割り当てられた背景要素sprite の伸縮をトグルするものです。
