レイヤーの背景を取得します。

この関数は、背景要素の現在のスケール倍率の値を取得するために使用することができます。背景要素のID（このIDは，"Background "関数を用いて背景要素を作成した際に得られる）を指定します。 layer_background_create()を使用したとき、あるいは関数 layer_background_get_id()この関数は，sprite の描画に使用されているスケール倍率を表す実数値を返します。 デフォルト値は1です。

構文です。

layer_background_get_yscale(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_yscale(back_id) != -1)
{
    layer_background_yscale(back_id, -1);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを元に要素のyスケールを確認し、-1でなければ-1に設定します。
