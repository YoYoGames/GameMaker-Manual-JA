レイヤーの背景を取得するインデックス

この関数は、背景要素の現在の画像インデックス値を取得するために使用することができます。背景要素のID（これは layer_background_create()を使用したとき、あるいは関数 layer_background_get_id()sprite背景となる要素が存在しない場合，あるいはその要素に有効なsprite が割り当てられていない場合，この関数は -1 を返します．

構文です。

layer_background_get_index(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

Real (the current sprite image index or -1)

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_index(back_id) < 4)
{
    layer_background_index(back_id, 4);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを元に、その要素の画像インデックスが4以下かどうかを調べ、4以下であれば4に設定します。
