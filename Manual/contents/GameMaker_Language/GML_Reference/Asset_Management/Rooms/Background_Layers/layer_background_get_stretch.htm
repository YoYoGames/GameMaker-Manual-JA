layer_background_get_stretch

この関数は，背景要素の伸縮状態を取得するために用いることができる。sprite.背景要素の ID (背景要素を layer_background_create()を使用した場合、または関数 layer_background_get_id()) 、この関数は、要素sprite が現在room に合うように引き伸ばされている場合は true を、そうでない場合は false を返します。

構文です。

layer_background_get_stretch(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_stretch(back_id))
{
    layer_background_stretch(back_id, false);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って、sprite の要素がroom に合うように引き伸ばされるかどうかを確認し、引き伸ばされる場合は、このプロパティを false に設定します。
