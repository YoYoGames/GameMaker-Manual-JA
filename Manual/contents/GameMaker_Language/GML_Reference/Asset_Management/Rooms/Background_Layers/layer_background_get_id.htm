レイヤーの背景を取得する

この関数は、レイヤー上の背景要素の一意の ID 値を取得するために使用できます。レイヤー ID を指定します (レイヤー名を layer_get_id()この関数は、そのレイヤーの背景要素に関連する ID 値を返します。この関数は、IDE で追加された背景を使用するために特別に設計されていることに留意してください。 layer_background_create()の場合、追加された背景要素の一意のIDが返されます。

構文です。

layer_background_get_id(layer_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target

リターンです。

Background Element ID

例

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_sprite(back_id, bck_Trees_Winter);

上記のコードでは、「Background_trees」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って、要素sprite を変更します。
