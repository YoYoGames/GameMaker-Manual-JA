レイヤーバックグラウンドスピード

この関数は、背景要素に割り当てられたアニメーション（sprite ）の速度を設定するために使用することができます。背景要素の ID (背景要素を layer_background_create()を使用した場合、または関数 layer_background_get_id()) を設定し、アニメーションの速度を設定します。IDE が使用するsprite のアニメーション速度を Frames Per Game Frame として計算するように設定している場合、与えられた速度値は乗数で、1がデフォルト値で、0.5に設定するとアニメーション速度が半分になり、2に設定すると2倍になります。しかし、IDE が使用するsprite のアニメーション速度を Frames Per Second として計算するように設定している場合、与えられた speed 値は文字通り Frames Per Second の値であり、例えばこれを 5 に設定すると、ゲームが実行されている毎秒にloop 5 フレームのアニメーションが設定されることになります。background 要素にsprite が付いていない場合や、使用されているsprite にサブ画像がない場合は、この関数は何の効果もありません。

構文です。

layer_background_speed(background_element_id, speed);

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
speed リアル The speed multiplier/value to use for the element

リターンです。

N/A

例

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_speed(back_id, 0.5);

上記のコードでは、「Background_water」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを利用して、要素の画像速度を変更します。
