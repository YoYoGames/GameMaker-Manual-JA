レイヤーの背景を作成する

この関数を使うと、sprite リソースをレイヤーに割り当てて、プロジェクトの背景として使用することができます。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string として - これはパフォーマンスに影響します) とsprite インデックス (sprite ブラウザで表示されるAsset の名前となります) を指定すると、レイヤーに追加されます。この関数は、その要素の一意な ID 値を返し、それを背景のためのさらなるレイヤー関数で使用することができます。

構文です。

layer_background_create(layer_id, sprite)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
sprite スプライトアセット The sprite index to be used

リターンです。

Background Element ID

例

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

上記のコードでは、新しいレイヤーを作成し、そこに新しい背景要素を追加し、使用する背景画像としてsprite を設定します。
