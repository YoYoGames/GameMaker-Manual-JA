レイヤーバックグラウンドインデックス

この関数は、複数のサブ画像を持つ背景sprite の画像インデックスを設定するために使用することができる。背景要素の ID (これは，"Background "関数を用いて背景要素を作成したときに得られる) を指定します。 layer_background_create()を使用した場合、または関数 layer_background_get_id()) を設定し、使用する画像インデックスを設定します。サブ画像の範囲外の値を設定すると、画像インデックスがloop 回ります。

構文です。

layer_background_index(background_element_id, image_index);

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
index リアル The image index to use for the background

リターンです。

N/A

例

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

上記のコードでは、「Background_trees」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを元に、要素の画像インデックスを変更します。
