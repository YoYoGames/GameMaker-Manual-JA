レイヤーバックグラウンドのブレンド

この関数は、背景要素のブレンドカラーを取得するために使用することができます。背景要素のIDを指定します。 layer_background_create()を使用したとき、または関数 layer_background_get_id())、この関数は割り当てられた色を表す実数値を返します。

構文です。

layer_background_get_blend(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

Colour Constant

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_blend(back_id) == c_white)
{
    layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って要素のブレンドカラーを調べ、定数 c_white と等しい場合は、レイヤーのブレンドカラーをランダムな色に設定します。
