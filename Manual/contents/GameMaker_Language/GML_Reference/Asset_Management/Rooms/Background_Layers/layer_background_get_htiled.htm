レイヤーの背景を取得する

この関数は、背景要素が水平方向にタイル状に配置されているかどうかを調べるために使用することができます。背景要素のIDを指定します。 layer_background_create()を使用したとき、あるいは関数 layer_background_get_id()この関数は、その要素が現在タイル状になっている場合は true を、そうでない場合は false を返します。

構文です。

layer_background_get_htiled(background_element_id);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to get the information from

リターンです。

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_htiled(back_id)
{
    layer_background_htiled(back_id, true);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを元に、その要素が水平軸にタイル状に並んでいるかどうかを確認し、並んでいない場合はタイル状になるように設定します。
