レイヤーバックグラウンドアルファ

この関数は、背景sprite のアルファ値（透明度）を制御します。背景要素の ID (背景要素を layer_background_create()または を使用する場合、関数 layer_background_get_id()) を設定し、使用するアルファ値を設定します。アルファ値は0（完全に透明）から1（完全に不透明）です。 の要素は1です。背景要素が割り当てられているレイヤーが表示されていない場合、または要素自体が非表示になっている場合、そのレイヤーまたは要素が表示されるまで、この関数による違いは見えませんので、ご注意ください。 をもう一度表示します。

構文です。

layer_background_alpha(background_element_id, alpha);

論証タイプ 商品説明
background_element_id The unique ID value of the background element to change
alpha The alpha for background sprite, from 0 to 1 (default is 1)

リターンです。

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

上記のコードは、レイヤー「Background_sky」に割り当てられた背景のID値を取得し、そのアルファ値を0〜1の間のランダムな値に設定しています。
