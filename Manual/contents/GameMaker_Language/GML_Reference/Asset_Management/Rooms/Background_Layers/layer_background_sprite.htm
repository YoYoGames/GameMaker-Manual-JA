layer_background_sprite

この関数を使用すると、背景要素のsprite インデックスを設定することができます。背景要素のIDを指定します。 layer_background_create()を使用した場合、または関数 layer_background_get_id())、そして使用するインデックスをsprite に指定します。背景要素の画像は、新しいsprite で置き換えられます。 値として -1 を指定すると、その要素にはsprite が割り当てられません (しかし、まだ存在し、後でsprite を再び割り当てることができます)。

この関数は、layer_background_change() という別名も持っています。

構文です。

layer_background_sprite(background_element_id, sprite_index)

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
sprite_index スプライトアセット The sprite index of the sprite to use for the background element

リターンです。

N/A

例

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if (layer_background_get_sprite(back_id) != spr_Clouds)
{
    layer_background_sprite(back_id, spr_Clouds);
}

上記のコードでは、「Background_sky」というレイヤーのレイヤーIDを取得し、それを使ってそのレイヤー上の背景要素のIDを取得しています。このIDを使って、その要素に割り当てられているsprite を確認し、sprite "spr_Clouds" でない場合は、それを設定します。
