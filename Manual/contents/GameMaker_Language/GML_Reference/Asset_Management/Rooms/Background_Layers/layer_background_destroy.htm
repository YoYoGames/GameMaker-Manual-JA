レイヤーバックグラウンド破壊

説明： 指定された背景要素を破棄する。背景のIDを指定します。 layer_background_create()またはレイヤーIDを layer_get_background_id()) を削除します。もし背景がroom エディタで追加されたものであれば、次にroom を離れて戻ってきたとき、背景は再び作成されることに注意してください。しかし、room が永続化されている場合、room の永続化を再度オフにしない限り、背景は削除されます。

構文です。

layer_background_destroy(background_element_id)

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to be destroyed

リターンです。

N/A

例

var lay_id = layer_get_id("Background_trees");
var bck_id = layer_background_get_id("Forrest");
if layer_background_exists(lay_id, bck_id)
{
    layer_background_destroy(bck_id);
}

上記のコードは、レイヤー "Background_trees" に与えられた背景要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（レイヤーは破棄しません）。
