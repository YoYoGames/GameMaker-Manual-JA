レイヤーの背景が存在する

この関数は、任意のレイヤーに背景要素が存在するかどうかを確認するために使用します。レイヤー ID (レイヤーを作成するときに layer_create()) またはレイヤー名 (string - これはパフォーマンスに影響します) と背景要素 ID (背景要素を作成するときに layer_background_create()を使用したとき、または関数 layer_background_get_id()) そして、この関数は、その要素が存在する場合は true を、存在しない場合は false を返します。

注：この関数は、現在のターゲットroom の範囲内で動作します。デフォルトでは、この関数が呼び出されたroom で、この範囲は関数 layer_set_target_room().

構文です。

layer_background_exists(layer_id, background_element_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the layer to target (or the layer name as a string)
background_element_id 背景要素ID The unique ID value of the background element to check

リターンです。

Boolean

例

var lay_id = layer_get_id("Background_trees");
if layer_background_exists(lay_id, global.TreesBackground)
{
    layer_background_destroy(lay_id, global.TreesBackground);
}

上記のコードは、レイヤー "Background_trees" に与えられた背景要素が存在するかどうかをチェックし、存在する場合はそれを破棄します（レイヤーは破棄しません）。
