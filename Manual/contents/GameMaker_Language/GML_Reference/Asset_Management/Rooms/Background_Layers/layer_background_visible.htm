レイヤーバックグラウンドビジブル

この関数を使用すると、背景要素の表示/非表示を切り替えることができます。背景要素のIDを指定します。 layer_background_create()または関数 layer_background_get_id() true false true に設定すると要素が表示され、 に設定すると要素が表示されな くなります。これはレイヤーの可視性に依存し、背景要素が可視とフラグされていても、それがあるレイヤーが非可視とフラグされている場合は描画されないことに注意してください。 false

構文です。

layer_background_visible(background_element_id, visible)

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
visible ブーリアン The visibility toggle, which can be true または false

リターンです。

N/A

例

if mouse_check_button_pressed(mb_left)
{
    var back = layer_background_get_id(layer);
    if layer_background_get_visible(back)
    {
        layer_background_visible(back, false);
    }
    else
    {
        layer_background_visible(back, true);
    }
}

上記のコードは、マウスボタンが押されたかどうかをチェックし、押されたことを検出すると、このコードを実行しているインスタンスのレイヤーに割り当てられた背景要素の背景の可視性をトグルします。
