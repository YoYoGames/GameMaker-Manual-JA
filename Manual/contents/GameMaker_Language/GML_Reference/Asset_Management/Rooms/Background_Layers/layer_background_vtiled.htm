レイヤーバックグラウンド

この関数は、あるレイヤーの背景要素を縦に並べるかどうかを変更することができます。背景要素の ID (背景要素の作成時に layer_background_create()または関数 layer_background_get_id()) を設定し、タイリングの値を設定します。 true に設定すると、その要素はroom の下に垂直にタイル状に表示され、 false に設定すると、その要素はタイル状に表示されません。この機能は2Dプロジェクト専用で、3Dカメラ投影の場合は正しく動作しません。

構文です。

layer_background_vtiled(background_element_id, tiled);

論証 タイプ 商品説明
background_element_id 背景要素ID The unique ID value of the background element to change
tiled ブーリアン The vertical tiling toggle, which can be true または false

リターンです。

N/A

例

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
    layer_background_htiled(back, true);
    layer_background_vtiled(back, true);
}

上記のコードは、コードを実行しているインスタンスのレイヤーに割り当てられている背景要素をチェックし、それがどちらの方向にもタイル状になっていない場合、それを垂直方向と水平方向にtile に設定します。
