room_exists

この関数を使うと、指定したroom が存在するかどうかを確認することができます。この関数は、room の名前 (room)ではなく、string のインデックス(実数) を受け取ります。この関数が最も役に立つのは，rooms を関数 room_add()を使用することもできますが、読み取り専用の変数 room_firstと room_lastまたは関数 room_next()と room_previous()特定のroom のインデックスを取得したり、ゲーム内の他のroom のインデックスを格納した変数を提供したりすることができます。

構文です。

room_exists(index);

論証タイプ 商品説明
index The index of the room to check.

リターンです。

例

if room_exists(global.rm[0])
{
    room_goto(global.rm[0]);
}

上記のコードでは、配列 "global.rm[]" のインデックスであるroom が存在するかどうかを確認し、存在する場合はそのroom に移動します。
