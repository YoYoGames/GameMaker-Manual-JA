room_goto

room この関数を使うと、コードで作成したものでも、ブラウザで作成したものでも、ゲームプロジェクト内の任意のAsset に移動することができます。room のインデックスを指定します（room の名前の変数に格納されているか、この関数から返される変数として格納されています）。 room_add()).room は、この関数が呼び出されたイベントが終了するまで変更されないので、この関数が呼び出された後のコードは、同じイベント内であればまだ実行されることに注意してください。この関数は、ルームエンドイベントのトリガーにもなります。

注 Room IDは、Asset BrowserやRoom Managerでの順番に基づくものではないので、数値の直接の供給は避けるべきです。代わりに、参照したいasset の部屋定数（これは単に名前である）を使用するか、関数を通じて取得するようにしてください。

構文です。

room_goto(index);

論証タイプ 商品説明
index The index of the room to go to.

リターンです。

例

switch (global.level)
{
    case 0: room_goto(rm_level1); break;
    case 1: room_goto(rm_level2); break;
    case 2: room_goto(rm_level3); break;
}

上記のコードでは、グローバル変数をチェックし、その変数が持つ値に基づいてroom を変更します。
