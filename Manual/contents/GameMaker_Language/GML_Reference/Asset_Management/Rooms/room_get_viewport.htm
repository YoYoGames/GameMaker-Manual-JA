room_get_viewport

この関数を使用すると、現在のビューポート以外のroom にあるビューポートの詳細を取得することができます。room ID と、取得したいビューポートのインデックス (0 から 7 まで) を指定すると、この関数は 5 つのインデックスの配列を返します。

  • [0] = visible (Boolean: true / false)
  • [1] = x 位置 (実数)
  • [2] = y 位置 (実数)
  • [3] = 幅 (実数)
  • [4] = 高さ (実数)

構文です。

room_get_viewport(rm, vind);

論証 タイプ 商品説明
rm ルームアセット The index of the room to get viewport data from
vind リアル The index of the view port to get

リターンです。

Array (5 elements: visible, x, y, width, height)

例

v_vals = room_get_viewport(rm_Game, 0);

if v_vals[0] == false
{
    room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

上記のコードは、与えられたroom のビューポートデータを取得し、そのポートに可視フラグが立っているかどうかをチェックします。もしそうでなければ、ビューポートデータを設定し、見えるようにします。
