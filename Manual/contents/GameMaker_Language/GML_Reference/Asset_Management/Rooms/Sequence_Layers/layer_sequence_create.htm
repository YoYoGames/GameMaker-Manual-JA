レイヤー・シーケンス・クリエイト

この関数を使うと、与えられたレイヤー上にsequence asset のインスタンスを作成することができます。レイヤーIDは、レイヤー名（string エディターで定義されたもの）のroom 、またはユニークレイヤーID（関数 layer_get_id()また、room を作成するためのsequence での X と Y の位置、そして最後に作成するsequence の ID を指定します。sequence ID はAsset ブラウザでsequence に対して定義した名前定数です。 この関数はsequence 要素のユニーク ID を返します。これはsequences の以降のすべてのレイヤ関数で使用できます。また、この ID はsequence インスタンスstruct を関数で取得するために使用できます。 layer_sequence_get_instance().

注目すべきは、sequence にobject のトラックが含まれている場合、そのsequence のキーがDope Sheet 上のどこに配置されているかに関わらず、asset 要素自体が作成されると同時にそのインスタンスが作成されることです。Sequence コントローラは、トラックのasset キーの位置と時間に応じて、インスタンスの可視性を非表示と表示に切り替えるだけで、インスタンスの作成と破棄を繰り返すことは ありません。その結果、インスタンスは、sequence 要素が作成されたときに Create イベントを実行し、そのアセットキーが開始されたときには実行されません。

構文です。

layer_sequence_create(layer_id, x, y, sequence_id)

論証 タイプ 商品説明
layer_id 文字列またはレイヤID The unique ID value of the instance layer to target
x リアル The x position in the room to create the sequence at
y リアル The y position in the room to create the sequence at
sequence_id シーケンスアセット The sequence asset to use, as defined in the Asset Browser

リターンです。

Sequence Element ID

例

var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(_s);

上記のコードでは、レイヤー「背景」上に新しいsequence を作成し、それを一時停止しています。
