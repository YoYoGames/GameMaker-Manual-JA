layer_sequence_get_headdir

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって、現在のプレイヘッドの方向を返します。

構文です。

layer_sequence_get_headdir(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

定数 商品説明 価値
seq_dir_right sequence は、左から右へインクリメンタルな順序でフレームを再生しています。 1
seq_dir_left sequence 、右から左へ減数法でフレームを再生しています。 -1

例

if layer_sequence_get_headdir(title_sequence) != seq_dir_left
{
    layer_sequence_headdir(title_sequence, seq_dir_left);
}

上記のコードでは、変数 "title_sequence" に格納されているsequence の要素 ID の現在の再生ヘッド方向をチェックし、 seq_dir_left に設定されていない場合は、この値が設定されます。
