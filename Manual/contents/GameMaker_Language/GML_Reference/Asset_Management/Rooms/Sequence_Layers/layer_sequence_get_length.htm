レイヤー配列の長さを取得する

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって、sequence の長さを返します。 これは、sequence が実行されるフレームの数です。

構文です。

layer_sequence_get_length(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

var _frames = layer_sequence_get_length(my_seq);
alarm[0] = frames;

上記のコードでは、sequence が実行されるフレーム数を取得し、この値を使ってアラームを設定しています。
