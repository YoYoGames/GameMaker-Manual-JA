layer_sequence_is_finished

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()この関数は、sequence が再生を終了しているかどうかを調べ、終了している場合は true を、終了していない場合は false を返します。なお、これはsequence が再生モードでloop または ping-pong に設定されていない場合のみ適用可能です。

構文です。

layer_sequence_is_finished(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

if layer_sequence_is_finished(title_sequence) != 0
{
    alarm[0] = room_speed * 3;
    layer_sequence_play(title_sequence);
}

上記のコードは、変数 "title_sequence" に格納されているsequence の要素 ID が終了しているかどうかをチェックし、終了していれば再生を再開し、アラームを設定するものです。
