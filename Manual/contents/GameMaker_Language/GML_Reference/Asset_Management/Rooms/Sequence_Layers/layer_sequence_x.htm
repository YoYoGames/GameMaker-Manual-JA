レイヤーシーケンスx

この関数を使うと、指定したroom の要素について、sequence の X（水平）軸に沿った位置を設定することができます。で返されるsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって設定されたX位置と一緒にsequence が新しい位置に移動されます。

構文です。

layer_sequence_x(sequence_element_id, pos_x)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target
pos_x The X position to move the sequence element to

リターンです。

例

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

上記のコードでは、変数 "my_seq" で参照されているsequence の要素が存在するかどうかを確認し、存在する場合は x/y 位置をコードを実行しているインスタンスのものに設定します。
