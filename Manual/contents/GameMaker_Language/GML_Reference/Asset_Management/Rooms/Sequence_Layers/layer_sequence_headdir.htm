レイヤー・シーケンス・ヘッドディール

この関数は、指定されたsequence のプレイヘッドの方向を設定することができます。で返されたsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって指定され、次にプレイヘッドの方向を指定します。この方向は以下の定数のいずれかでなければなりません。

定数 商品説明 価値
seq_dir_right sequence は、左から右へインクリメンタルな順序でフレームを再生します。 1
seq_dir_left sequence は、右から左へ向かって減数法でフレームを再生します。 -1

構文です。

layer_sequence_headdir(sequence_element_id, direction)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target
direction シーケンス方向定数 The playhead direction, a constant, listed above

リターンです。

N/A

例

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_headdir(_seq, seq_dir_left);

上記のコードでは、新しいsequence を作成し、その ID をローカル変数に格納しています。このIDを使って、プレイヘッドの方向をデクリメントフレーム（右から左への再生）に設定します。
