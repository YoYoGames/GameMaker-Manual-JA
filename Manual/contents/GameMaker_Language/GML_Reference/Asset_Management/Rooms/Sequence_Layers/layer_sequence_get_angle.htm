レイヤーシーケンス取得角度

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって、ゲーム内のsequence 要素の現在の角度を返しますroom 。角度は度数で返され、0ºが右、90ºが上、180ºが左、270ºが下であることに注意してください。

構文です。

layer_sequence_get_angle(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

var _ang = layer_sequence_get_angle(title_sequence)
if (_ang > 0)
{
    _ang -= 1;
    layer_sequence_angle(title_sequence, _ang);
}

上記のコードは、変数 "title_sequence" に格納された ID を持つsequence 要素の現在の角度を取得し、それが 0 でない場合は、現在の角度から 1 を差し引くものである。
