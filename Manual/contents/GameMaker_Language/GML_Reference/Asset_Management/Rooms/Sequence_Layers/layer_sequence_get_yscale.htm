レイヤー・シーケンス・ゲット・イスケール

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって、ゲーム内のsequence 要素のY軸に沿った現在のスケールを返しますroom 。

構文です。

layer_sequence_get_yscale(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

var _ys = layer_sequence_get_yscale(title_sequence)
if _ys < 1
{
    _ys += 0.01;
    layer_sequence_yscale(title_sequence, _ys);
}

上記のコードは、変数 "title_sequence" に格納された ID を持つsequence 要素の Y 軸に沿った現在のスケールを取得し、それが 1 より小さい場合、現在の Y スケールに 0.01 を追加します。
