layer_sequence_y

この関数を使うと、指定したroom 要素のsequence の Y（縦）軸方向の位置を設定することができます。で返されるsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって設定されたY位置と一緒にsequence が新しい位置に移動されます。

構文です。

layer_sequence_y(sequence_element_id, pos_y)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target
pos_y The Y position to move the sequence element to

リターンです。

例

if layer_sequence_exists(my_seq)
{
    layer_sequence_x(my_seq, x);
    layer_sequence_y(my_seq, y);
}

上記のコードでは、変数 "my_seq" で参照されているsequence の要素が存在するかどうかを確認し、存在する場合は x/y 位置をコードを実行しているインスタンスのものに設定します。
