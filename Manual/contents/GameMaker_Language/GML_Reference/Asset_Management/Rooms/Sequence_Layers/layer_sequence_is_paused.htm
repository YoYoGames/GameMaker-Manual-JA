この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって、sequence が現在一時停止しているかどうかをチェックし、一時停止している場合は true を、一時停止していない場合は false を返します。

構文です。

layer_sequence_is_paused(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

if layer_sequence_is_paused(title_sequence) != 0
{
    layer_sequence_play(title_sequence);
}

上記のコードは、変数 "title_sequence" に格納されたsequence の要素 ID が一時停止中かどうかを調べ、一時停止中であれば再生を開始します。
