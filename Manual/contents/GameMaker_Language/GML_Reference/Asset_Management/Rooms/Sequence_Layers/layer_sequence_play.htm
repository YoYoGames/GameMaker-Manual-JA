この関数は、与えられたsequence の再生を開始することができます。で返されたsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって再生されます。この関数はsequence を再生し，必要ならば関数 layer_sequence_pause().

重要! sequence にインスタンスがある場合、これらのインスタンスはimage_xscale /image_yscale / image_angle /x /y の変数を変更してはいけません。再生開始後にsequence が各ステップを更新するときに上書きされてしまうからです。

構文です。

layer_sequence_play(sequence_element_id)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target

リターンです。

N/A

例

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    var a = layer_get_all_elements(layer);
    for (var i = 0; i < array_length(a); i++;)
    {
        if layer_get_element_type(a[i]) == layerelementtype_sequence
        {
            if (global.Pause)
            {
                layer_sequence_pause(a[i]);
            }
            else
            {
                layer_sequence_play(a[i]);
            }
        }
    }
}

上記のコードでは、キーが押されたときに、ゲームが一時停止しているかどうかをチェックしています。ゲームが一時停止している場合は、現在のレイヤー（呼び出し元のインスタンスのレイヤー）上のすべてのloops 要素をsequence して再生を一時停止し、ゲームが一時停止していない場合は、loop の再生を再開します。
