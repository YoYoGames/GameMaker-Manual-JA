layer_sequence_speedscale

この関数を使うと、指定したsequence の再生速度を変更することができます。で返されるsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって指定され、さらに速度スケールの値が指定されます。この値は乗数で、1がデフォルトの再生速度、1より小さい値は遅く、1より大きい値は速くなります。例えば、0.5の値は半分の再生速度、2の値は2倍の再生速度になります。

構文です。

layer_sequence_speedscale(sequence_element_id, speedscale)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target
speedscale リアル The speed scale to use (a multiplier), default is 1

リターンです。

N/A

例

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_speedscale(_seq, 0.75);

上記のコードでは、呼び出したインスタンスのレイヤーに新しいsequence を作成し、その速度スケールの値を0.75（デフォルトの再生速度の4分の3）に設定します。
