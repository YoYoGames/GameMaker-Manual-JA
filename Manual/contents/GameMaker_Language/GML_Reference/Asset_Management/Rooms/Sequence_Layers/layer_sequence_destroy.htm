レイヤー・シーケンス・デストロイ

この関数を使うと、room の要素を破棄（sequence から削除）することができます。で返されたsequence 要素の ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって、sequence が破壊されます。

構文です。

layer_sequence_destroy(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_destroy(a[i])
    }
}

上記のコードでは、このコードを実行しているインスタンスのレイヤーに割り当てられているすべての要素のIDを取得しています。そして、返された要素の中にsequence assets があるかどうかを確認し、ある場合は破棄します。
