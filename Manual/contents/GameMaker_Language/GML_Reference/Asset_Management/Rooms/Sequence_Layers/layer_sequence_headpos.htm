レイヤー・シーケンス・ヘッドポス

この関数を使用すると、sequence 要素の再生ヘッド位置を特定のフレームに設定することができます。で返されるsequence 要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって設定された新しい位置と一緒に。位置はフレーム単位であり、総フレーム数より大きい値 (または 0 より小さい値) を設定すると、実際の最終的な再生ヘッド位置は、以下の規則に従って選択されたsequence 再生のタイプに依存することに注意してください。

  • ループしない。再生ヘッドは、loop の終わり（指定された値が負の場合は、始まり）にクランプします。
  • ループする。再生ヘッドは、sequence の始まりに余分なフレームを加えて巻き戻ります。例えば、30 フレームのsequence で再生ヘッドを 45 フレームに設定した場合、実際の再生ヘッド位置は 15 になります。負の値を指定すると、sequence の再生ヘッドが逆方向にラップされます。
  • ピンポン・ルーピング再生ヘッドが最後まで進み、全フレーム長を超えるフレーム数だけ戻り、再生方向が反転します。例えば、sequence が 30 フレームで、プレイヘッドを 40 に設定した場合、最後のプレイヘッド位置は 20 になり、方向は逆になります（0 に下がる）。なお、sequence のピンポン再生に負の値を与えてはいけません。undefined のような結果になることがあります。

このようにヘッド位置を設定しても再生は停止せず、sequence 、一時停止しない限り新しいプレイヘッド位置から再生が継続されるだけです。

構文です。

layer_sequence_headpos(sequence_element_id, position)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target
position リアル The position within the sequence (in frames) to set the playhead position to (can be a decimal value)

例

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++)
{
    if layer_get_element_type(a[i]) == layerelementtype_sequence
    {
        layer_sequence_headpos(a[i], 0)
    }
}

上記のコードは、このコードを実行しているインスタンスのレイヤーに割り当てられているすべてのエレメントのIDを取得します。このコードでは、返された要素のいずれかがsequence assets であるかどうかを確認し、その場合は再生ヘッド位置が 0 に設定されます。
