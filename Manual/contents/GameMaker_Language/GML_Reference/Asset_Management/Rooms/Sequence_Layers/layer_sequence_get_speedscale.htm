layer_sequence_get_speedscale

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数の1つによって、現在の再生速度スケールを返します。これは、再生速度を遅くしたり速くしたりするために使用される乗数値です。1 がデフォルト値で、1 より小さい値は再生速度が遅くなり、1 より大きい値は再生速度が速くなることを意味します。

構文です。

layer_sequence_get_speedscale(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

if layer_sequence_get_speedscale(title_sequence) != 1
{
    layer_sequence_speedscale(title_sequence, 1);
}

上記のコードは、変数 "title_sequence" に格納されているsequence の要素 ID の現在のプレイヘッドスピードスケールをチェックし、1 に設定されていなければこの値に設定されます。
