layer_sequence_get_instance

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかにより、sequence インスタンス 構造体が返されます。sequence インスタンス構造体のフォーマットについては、こちらのページで詳しく説明しています。

構文です。

layer_sequence_get_instance(sequence_element_id)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target

リターンです。

Sequence Instance Struct

例

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _struct = layer_sequence_get_instance(_seq);
_struct.speedScale = 0.5;

上記のコードでは、新しいsequence 要素を作成し、そのインスタンス ID をsequence から取得しています。これを利用して、sequence インスタンスの再生速度スケールのプロパティを変更します。
