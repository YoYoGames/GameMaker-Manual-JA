layer_sequence_yscale

この関数を使用すると、指定されたsequence 要素の Y スケールを設定することができます。で返されたsequence 要素の ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって、Y軸に設定する新しいスケールとともに、sequence はこの量だけスケールされます。スケール 1 はスケーリングなし (1:1) を表し、小さい値はスケーリングダウン (たとえば 0.5 はsequence の幅の半分)、大きい値はスケーリングアップ、負の値はsequence をその原点で反転してスケーリングします (使用する値がちょうど -1 であればsequence はスケーリングなしでその原点で反転されます)。

回転を使用するインスタンスを含むsequence 要素に不均等なスケーリング（例：X軸を3倍、Y軸を2倍）を適用すると、インスタンスの描画、衝突、カリング、その他様々な問題が発生することがありますので、非常に重要なポイントです。基本的に、sequence がインスタンスのプロパティに依存している場合、不均等なスケーリングとインスタンスの回転を組み合わせることはお勧めしません。

また、この関数を使用しているsequence のインスタンスの変数 image_xscale ￭ image_yscale ￭ image_angle ￭ x ￭ y は、関数呼び出し後sequence が更新されると同時に上書きになるため変更しないように注意 してください。

構文です。

layer_sequence_yscale(sequence_element_id, xscale)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target
yscale The new Y axis scale value to apply to the sequence element

リターンです。

例

if seq_scale < 2
{
    seq_scale += 0.01;
    layer_sequence_xscale(my_seq, seq_scale);
    layer_sequence_yscale(my_seq, seq_scale);
}

上記のコードでは、変数 seq_scale の値をチェックし、2 よりも小さい場合はその値を加算し、その値を使って変数 my_seq で参照されているsequence 要素の X と Y のスケールを設定しています。
