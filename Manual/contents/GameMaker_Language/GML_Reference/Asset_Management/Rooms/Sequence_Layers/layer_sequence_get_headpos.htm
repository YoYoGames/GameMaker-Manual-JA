layer_sequence_get_headpos

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって、現在の再生ヘッド位置（再生ヘッドが乗っている現在のフレーム）が返されます。

構文です。

layer_sequence_get_headpos(sequence_element_id)

論証タイプ 商品説明
sequence_element_id The unique ID value of the sequence element to target

リターンです。

例

if layer_sequence_get_headpos(title_sequence) != 0
{
    layer_sequence_headpos(title_sequence, 0);
}

上記のコードでは、変数 "title_sequence" に格納されたsequence の要素 ID の現在の再生ヘッド位置を確認し、0 になっていなければこの値を設定します。
