レイヤーシーケンス取得

この関数では、sequence の要素 ID を指定します。 layer_sequence_create()room またはレイヤー要素関数のいずれかによって、ゲーム内の現在の X 位置を返します。sequence 。

構文です。

layer_sequence_get_x(sequence_element_id)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target

リターンです。

Real

例

if layer_sequence_get_x(title_sequence) != room_width / 2
{
    layer_sequence_x(title_sequence, room_width / 2);
}

上記のコードでは、変数 "title_sequence" に格納されているsequence の要素 ID の X 位置をチェックし、room の幅の半分に設定されていなければ、この値が設定されます。
