この関数を使うと，指定したsequence の再生を一時停止することができます。で返されたsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤーエレメント関数の1つによって，この関数はsequence を一時停止し，関数 layer_sequence_play().

構文です。

layer_sequence_pause(sequence_element_id)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target

リターンです。

N/A

例

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
if global.Pause
{
    layer_sequence_pause(_seq);
}

上記のコードでは、新しいsequence を作成し、その ID をローカル変数に格納しています。次に、ゲームが一時停止しているかどうかを確認し、一時停止している場合は、sequence の再生を一時停止しています。
