レイヤーシーケンスが存在する

この関数を使うと、指定したレイヤーにsequence の要素が存在するかどうかを確認することができます。レイヤー ID は、string エディターで定義されたレイヤー名（room ）、あるいは、関数 layer_get_id()で返されるsequence 要素 ID と同様に layer_sequence_create()またはレイヤー要素関数の1つで、与えられた要素が存在する場合は true を、そうでない場合は false を返します。

構文です。

layer_sequence_exists(layer_id, sequence_element_id)

論証 タイプ 商品説明
layer_ID 文字列またはレイヤID The unique ID or name of the layer to check
sequence_element_id 配列要素ID The unique ID value of the sequence element to target

リターンです。

Boolean

例

if !layer_sequence_exists("Background", my_seq)
{
    my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
    layer_sequence_pause(my_seq);
}

上記のコードは、与えられたsequence 要素が存在するかどうかを確認し、存在しない場合は与えられたレイヤーに新しいsequence を作成し、それを一時停止します。
