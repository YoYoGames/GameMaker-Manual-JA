レイヤーシーケンス角度

この関数を使うと、指定されたsequence 要素の回転角度を設定することができます。で返されたsequence の要素 ID を指定します。 layer_sequence_create()またはレイヤー要素関数のいずれかによって設定された新しい角度と、sequence はその原点を中心に新しい位置まで回転されます。GameMaker の角度は、右を 0º として反時計回りに計算されます。つまり、90º が上、180º が左、270º が下です。sequence のデフォルトの角度は 0º です。

なお、この関数を使用しているsequence のインスタンスの変数 image_xscale ￭ image_yscale ￭ image_angle ￭ x Ⓒ y は、関数呼び出し後sequence が更新されると同時に上書きされますので、変更しないでく ださい。

構文です。

layer_sequence_angle(sequence_element_id, angle)

論証 タイプ 商品説明
sequence_element_id 配列要素ID The unique ID value of the sequence element to target
angle リアル The new angle to rotate the sequence element to

リターンです。

N/A

例

if current_angle < 90
{
    current_angle += 1;
    layer_sequence_angle(my_seq, current_angle);
}

上記のコードでは、変数 current_angle の値をチェックして、90 よりも小さければ、その値を加算して、変数 my_seq で参照しているsequence の要素の角度を設定しています。
