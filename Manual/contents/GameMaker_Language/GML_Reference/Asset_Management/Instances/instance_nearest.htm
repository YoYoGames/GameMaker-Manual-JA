インスタンス・ニアレスト

この関数は、与えられたobject のすべてのインスタンスをチェックして、どれが与えられた x/y 点に最も近いかを確認します。すべてのチェックは、指定された x/y 位置から、指定されたobject のインスタンスの原点(x/y 位置) まで行われます。object のインスタンスが存在しない場合、この関数はキーワードを返します。 nooneを返しますが、存在する場合は、そのインスタンスの idのインスタンスが見つかりました。コードを実行しているインスタンスが、チェック対象のobject と同じobject インデックスを持つ場合、そのインスタンスもチェックに含まれることに注意してください (呼び出し元のインスタンスが親の子でもある場合、親のobjects のチェックも含まれます)。

構文です。

instance_nearest(x, y, obj);

論証タイプ 商品説明
x The x position to check from.
y The y position to check from.
obj The object to check for instances of.

リターンです。

Instance ID or noone

例

var inst, xx;
xx = x;
x -= 10000;
inst = instance_nearest(xx, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}
x += 10000;

上記のコードは、現在のインスタンスを10000ピクセル移動させ、その前の位置をチェックして、同じobject タイプの最も近いインスタンスを見つけます。そのインスタンスが同じものであれば、元の位置に戻るだけですが、見つかったインスタンスが異なるものであれば、2つの間に線を描画します。
