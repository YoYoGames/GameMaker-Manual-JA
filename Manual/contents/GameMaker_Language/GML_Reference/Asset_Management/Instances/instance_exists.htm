インスタンスの存在

この関数は、調べたい内容に応じて2通りの使い方ができます。 true objectこの場合、この関数は、指定されたroom のアクティブなインスタンスが現在のroom に存在すれば true を返します。

ただし、インスタンス停止機能で停止されたインスタンスは考慮されません。

構文です。

instance_exists(obj);

論証 タイプ 商品説明
obj インスタンスIDまたはオブジェクトアセット The object or instance to check for the existence of.

リターンです。

Boolean

例

if !instance_exists(obj_Enemy)
{
    score += 200;
    room_goto(rm_hiscores);
}

上記のコードは、object "obj_Enemy" のインスタンスが存在するかどうかをチェックし、存在しない場合は変数 "score" に追加し、room を変更しています。
