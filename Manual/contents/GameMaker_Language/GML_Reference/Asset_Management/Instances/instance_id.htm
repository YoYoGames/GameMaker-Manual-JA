instance_id

この読み取り専用の 配列は、room 内のすべてのアクティブな インスタンスのidを保持します。 つまり、インスタンスの非アクティブ化関数を使用した場合、非アクティブ化されたインスタンスはこの配列には含まれません (以前にこの配列の値を使用した場合、この配列からキーワードを返します)。 noone).

構文です。

instance_id[num];

リターンです。

Instance ID

例

for (var i = 0; i < instance_count; i ++;)
{
    with (instance_id[i]) speed += 0.1;
}

上記のコードは、loop 内のすべてのインスタンスにroom 、その速度に 0.1 を加算します。
