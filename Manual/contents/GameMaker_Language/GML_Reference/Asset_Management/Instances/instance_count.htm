インスタンスカウント

この読み取り専用変数を使用すると、room にあるすべてのアクティブなインスタンスの数を取得できます。 これにはコードを実行しているインスタンスが含まれますが、インスタンスの非アクティブ化関数を使用して非アクティブ化されたインスタンスは含まれません。この変数はステップ開始時のインスタンス数しか取得できないので、ステップ開始後にroom 内のインスタンスに加えられた変更は考慮されないことに注意してください。

構文です。

instance_count;

リターンです。

例

if (instance_count < 100)
{
    var dif = 100 - instance_count;
    while (--dif > 0)
    {
        instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
    }
}

上記のコードでは、object "obj_Star" のインスタンスを、合計インスタンス数が100になるまで、複数作成します。
