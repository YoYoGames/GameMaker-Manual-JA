instance_id_get

この関数を使うと、現在アクティブなインスタンスリストから、任意のインスタンスのユニークID値を取得することができます。インスタンスリストのインデックスを指定して ID を取得すると、その値を変数に格納して返します。

構文です。

instance_id_get(index);

論証 タイプ 商品説明
index リアル The index within the instance list from 0 をインスタンス数 - 1 に変更する。

リターンです。

Instance ID

例

for (var i = 0; i < instance_count; ++i;)
{
    var temp_id = instance_id_get(i);
    with (temp_id)
    {
        speed += 0.1;
    }
}

上記のコードは、loop 内のすべてのインスタンスにroom 、その速度に 0.1 を加算します。
