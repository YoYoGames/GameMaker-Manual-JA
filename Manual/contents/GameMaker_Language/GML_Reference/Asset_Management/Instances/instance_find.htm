インスタンス検索

すべてのインスタンスには一意の識別子(idしかし、特定のインスタンスの ID が何であるかを常に知っているわけではありません。object インスタンスリストのその位置にインスタンスがあれば、この関数はそのインスタンスの id を返し、なければ特別なキーワード noone を返します。また、キーワード all を使用して、room 内のすべてのインスタンスを反復処理し、親object を使用して、その親/子階層に属するすべてのインスタンスを反復処理できます。さらに、現在のroom に実際に存在するかどうかを確認するために、（その id がある場合）インスタンスを指定することも可能です。インスタンスは任意の方法でソートされるため、この関数でチェックされる順序は決まっておらず、どのインスタンスもどの位置にも存在しうることに注意してください。

この関数の "n "の最大値は次のようになります。

  • キーワードについて all: instance_count - 1
  • object インデックスの場合。 instance_number(OBJ) - 1

構文です。

instance_find(obj, n);

論証 タイプ 商品説明
obj オブジェクト・アセット The object to find the nth instance of
n リアル The number of the instance to find.

リターンです。

Instance ID or noone

例

for (var i = 0; i < instance_number(obj_Enemy); ++i;)
{
    enemy[i] = instance_find(obj_Enemy,i);
}

上記のコードでは、 for loop 、「obj_Enemy」のすべてのインスタンスを繰り返し処理し、そのidを配列「enemy[]」に格納します。
