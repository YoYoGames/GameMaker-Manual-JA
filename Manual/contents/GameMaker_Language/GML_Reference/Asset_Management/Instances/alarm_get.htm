alarm_get

この関数は、与えられたアラームの現在値を取得するために使用することができます。0から11までのアラーム番号を指定すると、この関数はそのアラームが現在かかっている値を返します。これはアラーム配列の値を直接取得する方法とは別の方法です。

構文です。

alarm_get(index);

論証 タイプ 商品説明
index リアル The alarm index to get, from 0 to 11.

リターンです。

Real (integer)

例

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

上記のコードでは、呼び出したインスタンスのすべてのアラームをチェックし、0より大きい場合は-1に設定し、それ以上カウントダウンしないようにします。
