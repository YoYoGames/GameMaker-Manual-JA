インスタンス

インスタンスは オブジェクトから作成され、作成後はGML コードとGML Visual アクションで変更、操作することができます。インスタンスに関連する主なインスタンス関数で、利用可能で最もよく使われるものは以下の通りです。

  • instance_create_layer
  • instance_create_depth
  • instance_destroy
  • instance_exists
  • instance_change
  • instance_copy
  • instance_find
  • instance_furthest
  • instance_nearest
  • instance_number
  • instance_place
  • instance_place_list
  • instance_position
  • instance_position_list

さらに2つの組み込み変数（これらはグローバルスコープの変数です）と、特定の詳細を取得するのに便利なインスタンスの使用に関連する1つの関数があることに注意してください。

  • instance_id_get
  • instance_id
  • instance_count

object の各インスタンスには、インスタンスの表示方法、他のassets の使用方法、衝突への反応など、さまざまな側面を制御するために使用できる組み込みの変数が多数あります。

  • インスタンス変数

インスタンスには、アラームイベントの値を取得または設定するために使用できる配列変数がありますが、同じことを行うために使用できる2つの関数もあります。

  • alarm_get
  • alarm_set

最後に、インスタンスを非アクティブにすることも可能です。つまり、ゲーム内にはまだ存在しますが、イベント処理は行われません。これに関する機能は、以下のページで紹介しています。
