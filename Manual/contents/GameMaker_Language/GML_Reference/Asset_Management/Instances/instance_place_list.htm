instance_place_list

この関数を使用すると、チェック用のコードを実行するインスタンスのコリジョンマスクを使用して、object のすべてのインスタンスとの衝突の位置を確認することができます。この関数を使うと、GameMakerにインスタンスを新しい位置に移動させ、衝突をチェックし、その後、移動して衝突があったかどうかを教えてくれるよう頼むことになります。これは正確なコリジョンに対して機能しますが、インスタンスとチェックされるobject のインスタンスの両方が正確なコリジョンマスクを選択している場合のみで、それ以外の場合はバウンディングボックスコリジョンのみが適用されます。

DSのリストも渡すので id呼び出したインスタンスと衝突しているインスタンスの値は、与えられたリストの末尾に追加されます。この関数を呼び出す前にリストをクリアして、この関数の呼び出しによる結果のみを含むようにすることができます。また、チェックを行うインスタンスの原点から、衝突が見つかったインスタンスの原点までの距離に基づいて、リストを並べるオプションもあります。

この関数は、特別なキーワードも受け付けることに注意してください。 allこの場合、衝突が見つかったすべてのインスタンスがリストアップされます。この関数は，見つかったインスタンスの数を返し，何も見つからなかった場合は0を返します．

構文です。

instance_place_list(x, y, obj, list, ordered);

論証 タイプ 商品説明
x リアル The x position to check for instances.
y リアル The y position to check for instances.
obj オブジェクトのアセットまたはインスタンスID The object to check for instances of.
list DSリストID The DS list to use to store the IDs of colliding instances.
ordered ブーリアン Whether the list should be ordered by distance (true) か、そうでないか (false) です。

リターンです。

Real (The number of instances found to be in collision)

例

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);

if _num > 0
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}

ds_list_destroy(_list);

上記のコードでは、呼び出したインスタンスの位置で見つかったすべてのインスタンスとの衝突をチェックします。これらのインスタンスは DS リストに追加され、ループして衝突したインスタンスをすべて破壊します。
