インスタンスコピー

この関数を使うと、同じ位置でコードを実行する新しいバージョンのインスタンスが作成されるので、インスタンスを「クローン」することができます。perf "引数は、この新しいインスタンスにcreateイベントを実行するかどうかを指示するために使用されます。この関数は idを変数に格納したり、そのインスタンスにアクセスするために使用することができます。

注：createイベントを実行しないように選択した場合、インスタンスがこのイベントで初期化された変数に依存する場合、エラーが発生する可能性があります。

構文です。

instance_copy(perf);

論証 タイプ 商品説明
perf ブーリアン Whether to perform the new instance's Create event (true) or not (false).

リターンです。

Instance ID

例

var inst = instance_number(object_index);
if (inst < 10)
{
    instance_copy(true);
}

上記のコードでは、ローカル変数を作成し、それを使ってコードを実行しているobject のインスタンスの数をroom に格納しています。 この数が 10 未満の場合、そのインスタンスは自分自身のコピーを作成します。
