インスタンス位置

この関数を使うと、他のインスタンスまたはobject のすべてのインスタンスとの衝突をチェックすることができます。 これを使うと、room の 1 つのインスタンスまたはobject の 1 つの点をチェックします。 このチェックは、インスタンスのバウンディングボックスに対して、またはそのインスタンスのマスクに対して行われ、もしそのインスタンスが精密衝突をチェックしている場合は、ユニークなインスタンスが返されます。 id.衝突したインスタンスの ID が不要な場合は、以下のようにします。 position_meeting()の代わりにこの関数は、特殊なキーワード all と other も受け付け、衝突が起きない場合はキーワード noone を、衝突が起きた場合は見つかったインスタンスのユニークな ID 値を返します。

構文です。

instance_position( x, y, obj );

論証 タイプ 商品説明
x リアル The x position to check for instances.
y リアル The y position to check for instances.
obj オブジェクトのアセットまたはインスタンスID The object to check for instances of.

リターンです。

Instance ID or noone

例

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if (inst != noone)
{
    with (inst)
    {
        image_index = 1;
    }
    instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
}

上記のコードは、マウス位置で "obj_Pause_Button" のインスタンスとの衝突をチェックし、衝突があった場合は、返された id を使ってその image_index を新しい値に設定してからobject "obj_Menu" の新しいインスタンスを生成します。
