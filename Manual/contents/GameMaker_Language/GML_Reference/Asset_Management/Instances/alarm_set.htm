アラームセット

この機能を使って、アラームを設定することができます。0 から 11 までのアラーム番号と、アラームを設定する値を指定します。この値は整数値でなければならず、-1 に設定するとアラームは停止します (整数以外の値は最も近い整数に丸められます)。これは、アラーム配列を直接設定する方法とは別の方法です。

構文です。

alarm_set(index, value);

論証 タイプ 商品説明
index リアル The alarm index to set, from 0 to 11.
value リアル The value (an integer) to set the alarm to.

リターンです。

N/A

例

for (var i = 0; i < 12; i++)
{
    if alarm_get(i) > 0 alarm_set(i, -1);
}

上記のコードでは、呼び出したインスタンスのすべてのアラームをチェックし、0より大きい場合は-1に設定し、それ以上カウントダウンしないようにします。
