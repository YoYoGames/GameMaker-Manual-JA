instance_position_list

この機能を使うと、ある位置が他のobject のすべてのインスタンスと衝突していないかどうかをチェックすることができます。この関数を使用すると、room の 1 つのポイントについて、object のすべてのインスタンスをチェックすることになり、チェックはインスタンスのバウンディング ボックスに対して、またはobject で精密衝突をチェックしている場合はインスタンスのマスクに対して実行されます。

DSのリストも渡すので id呼び出したインスタンスと衝突しているインスタンスの値は、与えられたリストの末尾に追加されます。この関数を呼び出す前にリストをクリアして、この関数の呼び出しによる結果のみを含むようにすることができます。また、チェックする位置から、衝突が見つかったインスタンスの原点までの距離に基づいてリストを並べるオプションもあります。

この関数は、特別なキーワードも受け付けることに注意してください。 allこの場合、衝突が見つかったすべてのインスタンスがリストアップされます。この関数は，見つかったインスタンスの数を返し，何も見つからなかった場合は0を返します．

構文です。

instance_position_list(x, y, obj, list, ordered);

論証 タイプ 商品説明
x リアル The x position to check for instances.
y リアル The y position to check for instances.
obj オブジェクトのアセットまたはインスタンスID The object to check for instances of.
list DSリストID The DS list to use to store the IDs of colliding instances.
ordered ブーリアン Whether the list should be ordered by distance (true) か、そうでないか (false) です。

リターンです。

Real (The number of instances found to be in collision)

例

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

上記のコードでは、マウス位置で見つかったすべてのインスタンスとの衝突をチェックします。これらのインスタンスは DS リストに追加され、ループして衝突したインスタンスをすべて破壊します。
