instance_activate_layer

この関数を使うと、以前に非アクティブにしたレイヤーをアクティブにすることができます。レイヤー ID は、コードエディターで書かれたレイヤー名 (string) か、または layer_create()で、そのレイヤー上のすべての非アクティブ化されたインスタンスが再びアクティブになります。永続的なインスタンスを持つレイヤーを非アクティブにした場合、room を変更する前に、この関数でレイヤーを再度アクティブにする必要があります。そうしないと、永続的なインスタンスは引き継がれず、破棄されます。この方法でアクティブにしたインスタンスは、この関数が呼び出されたイベントが終了するまでアクティブとはみなされないことに注意してください。

構文です。

instance_activate_layer(layer_id);

論証 タイプ 商品説明
layer_id レイヤーIDまたは文字列 The layer name string (or ID value) to be used

リターンです。

N/A

例

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false);
instance_activate_layer("Player_Layer");

上記のコードでは、room 内のすべてのインスタンスをアクティブにし、現在のカメラビューの制限外にあるインスタンスを非アクティブにします。ただし、レイヤー「Player_Layer」上のインスタンスは最後に再びアクティブにします。
