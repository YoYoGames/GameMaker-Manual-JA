インスタンスアクティベートリージョン

この機能により、room 内に、以前に非アクティブにしたインスタンスをアクティブにするための領域を定義することができます。この領域は、次の画像のように、"inside "または "outside "としてフラグを立てることができます。

上の画像で、"apple" インスタンスが常にアクティブであることがわかります。これは、sprite 自体が領域と重なっていなくても、バウンディングボックスが重なっているからです。つまり、インスタンスは、そのバウンディングボックスが領域と重なったときに指定された領域内にいるとみなされ、コリジョンマスクの状態（正確かどうか）は考慮されないのです。活性化は瞬間的なものではなく、この方法で活性化されたインスタンスは、この関数が呼び出されたイベントが終了するまで活性化されたとはみなされないことに注意してください。

構文です。

instance_activate_region(left, top, width, height, inside);

論証 タイプ 商品説明
left リアル The x coordinate of the left of the rectangular region to activate.
top リアル The y coordinate of the top of the rectangular region to activate.
width リアル The width of the region to activate.
height リアル The height of the region to activate.
inside ブーリアン Whether to activate instances on the inside of the region (true) or the outside (false).

リターンです。

N/A

例

instance_deactivate_all(true);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上記のコードでは、コードを実行しているインスタンス以外のすべてのインスタンスを非アクティブにし、room 内のリージョンをアクティブにしています。
