instance_deactivate_object

この機能を使用すると、以前にアクティブ化されたすべてのインスタンスの中から、単一のインスタンスまたは特定のobject のすべてのインスタンスを非アクティブ化することができます。永続化フラグが立っているインスタンスまたはobject を非アクティブにした場合、そのインスタンスまたは を非アクティブにすることに注意してください。 を使用して再度有効にする必要があります。 instance_activate_object()を変更する前に、room 。さもなければ、それは引き継がれず、代わりに廃棄されます。また，非アクティブ化 このようにして非アクティブになったインスタンスは、 この関数が呼び出されたイベントが終了するまで非アクティブとみなされません。

注：room の開始時にインスタンスを非アクティブにした場合（つまり、room の作成コード、またはroom 内のインスタンスの作成イベントから）、room エディタからroom 内に配置されたすべてのインスタンスは、非アクティブになる前に作成イベントをまだ実行することになります。

警告：物理が有効になっているインスタンスを無効化しても、そのフィクスチャーが物理シミュレーションの中で相互作用することを止めません。そのためには、インスタンスの phy_active変数 を true または false に設定し、インスタンスをアクティブ/非アクティブにします。

構文です。

instance_deactivate_object(obj);

論証タイプ 商品説明
obj The object or instance to deactivate (the keyword all can also be used).

リターンです。

例

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上記のコードでは、object "obj_Enemy" のインスタンスをすべて非アクティブにし、room 内のリージョンをアクティブにしています。
