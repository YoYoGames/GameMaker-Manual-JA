instance_deactivate_layer

この関数を使うと、特定のレイヤーに割り当てられたすべてのインスタンスを非アクティブにすることができます。レイヤー ID は、コードエディターで書かれたレイヤー名（string ）か、あるいは layer_create()また、この関数ではインスタンスレイヤーのみを非アクティブにすることができることに注意してください。もし、objects のインスタンスが永続化されているレイヤーを非アクティブにした場合、そのレイヤーを関数 instance_activate_layer()roomさもなければ、レイヤー上の永続的なインスタンスは持ち越されず、破棄されます。また、非アクティブ化は即座には行われず、この方法で非アクティブ化されたインスタンスは、この関数が呼ばれたイベントが終了するまで非アクティブとみなされないことに注意してください。

注：room の開始時にインスタンスを非アクティブにした場合（つまり、room の作成コード、またはroom 内のインスタンスの作成イベントから）、room エディタからroom 内に配置されたすべてのインスタンスは、非アクティブになる前に作成イベントをまだ実行することになります。

警告：物理が有効になっているインスタンスを無効化しても、そのフィクスチャーが物理シミュレーションの中で相互作用することを止めません。そのためには、インスタンスの phy_active変数を true または false に設定し、インスタンスをアクティブ/非アクティブにします。

構文です。

instance_deactivate_layer(obj);

論証 タイプ 商品説明
layer レイヤーIDまたは文字列 The layer name string (or ID value) to be used

リターンです。

N/A

例

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

上記のコードは、レイヤー「Enemy_Layer」に割り当てられたすべてのインスタンスを非アクティブにし、room 内のリージョンをアクティブにしています。
