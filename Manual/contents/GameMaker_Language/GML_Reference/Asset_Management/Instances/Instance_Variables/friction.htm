フリクション

Friction は、インスタンスが持つことのできる「組み込みの」プロパティのひとつで speedがゼロ以外の場合。この機能は、object の速度が 0 になるまで、1 ステップごとに速度から量を減算していきます。したがって、たとえば摩擦を 0.1 に設定し、インスタンスの速度を 1（1 ステップにつき 1 ピクセル）とすると、10 ステップ後に減速して停止することになります。摩擦は正の速度にも負の速度にも等しく適用され、最終的にはobject の速度が一定時間後に 0 になることに注意してください。

構文です。

friction;

リターンです。

(single precision floating point value)

例

if abs(speed) > 0
{
    friction = 0.05;
}
else
{
    friction = 0;
}

上記のコードは、インスタンスの絶対速度が0以上の場合にのみ摩擦を適用します。
