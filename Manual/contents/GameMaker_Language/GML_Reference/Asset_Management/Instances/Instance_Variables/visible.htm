目に見える

この変数を true (visible) または false (invisible) に設定することで、インスタンスに visible というフラグを立てるかどうかを決めることができます。これはGameMakerにこのインスタンスのdrawイベントをスキップするように指示することで機能します。この変数を使用する際は注意が必要で、インスタンスが表示されていない間はdrawイベントに置かれたコードは実行されないことになります。また、インスタンスが表示されていないときに layerインスタンスが割り当てられているレイヤーが不可視フラグを立てている場合、この変数を true に設定しても、レイヤー自体も可視フラグを立てるまで何の効果もありません。

構文です。

visible;

リターンです。

例

if other.visible = true
{
    x = xprevious;
    y = yprevious;
}

上記のコードは、衝突イベントで「他」のインスタンスの可視flag をチェックし、 true に設定されていれば、そのインスタンスを以前の位置に移動させるものです。
