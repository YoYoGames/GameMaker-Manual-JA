hspeed は、すべてのインスタンスが持つ「ビルトイン」プロパティの1つで、インスタンスの水平移動速度（X軸方向）を1ステップあたりのピクセル数で定義します。つまり、 が 3 の場合、3 ピクセル分右に移動することを意味します。 (+x)ごとに1ステップ、 が-3なら、1ステップごとに3ピクセル左(-x)に移動することを意味します。 hspeed hspeed

変数 speed および/または direction を設定すると、 hspeed の値が自動的に更新されることに注意してください。 同様に、 hspeed の値を変更すると、 speed と direction の値にも影響があります。また、以下の点にも注意してください。 gravity, gravity_directionおよび frictionはすべて、ゲーム内で使用されたときに、この変数の値を変更することができます。

構文です。

hspeed;

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_left) hspeed = -5;
if keyboard_check(vk_right) hspeed = 5;

上記のコードは、どのキーが押されたかによって、水平方向の速度が変化します。
