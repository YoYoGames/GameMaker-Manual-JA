イースタート

この変数は、インスタンスがroom で最初に作成されたときの初期 y 位置を格納する。 この変数は読み取り専用ではなく、読み取りだけでなく設定も可能である。

構文です。

ystart;

リターンです。

(single precision floating point value)

例

if place_meeting(x, y, obj_spike)
{
    score -= 100;
    x = xstart;
    y = ystart;
}

上記のコードは、「obj_spike」のインスタンスとの衝突をチェックし、衝突があった場合はスコアから100を差し引き、インスタンスを開始位置まで戻すというものです。
