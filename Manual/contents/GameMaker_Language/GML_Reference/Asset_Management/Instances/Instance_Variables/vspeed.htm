対速度

vspeed は、すべてのインスタンスが持つ「ビルトイン」プロパティの1つで、インスタンスの垂直方向の移動速度（Y軸方向）を1ステップあたりのピクセル数で定義するものです。つまり、 が 3 の場合、3 ピクセル分下へ移動することを意味します。 (+y)を1ステップごとに、 が-3なら上(-y)に3ピクセルずつ移動することを意味します。 vspeed vspeed

変数 speed および/または direction を設定すると、 vspeed の値が自動的に更新されることに注意してください。 同様に、 vspeed の値を変更すると、 speed と direction の値にも影響があります。また、以下の点にも注意してください。 gravity, gravity_directionおよび frictionはすべて、ゲーム内で使用されたときに、この変数の値を変更することができます。

構文です。

vspeed;

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_up) vspeed = -5;
if keyboard_check(vk_down) vspeed = 5;

上記のコードは、どのキーが押されたかによって、垂直方向の速度が変化します。
