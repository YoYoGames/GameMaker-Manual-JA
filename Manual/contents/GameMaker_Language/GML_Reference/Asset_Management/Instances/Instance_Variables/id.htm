アイド

この読み取り専用の変数は、インスタンスの一意な識別番号を保持します。作成するすべてのインスタンス（コードを通じて、またはroom エディタでRoom に追加して、作成します）には、このインスタンスを識別するために内部的に使用される番号が与えられ、変数 id は、それを参照するために使用するものです。また、 instance_create_layer() や instance_create_depth() などのインスタンス関数を使用してインスタンスを作成すると、 id が返されます（変数に格納することも可能です）。

この変数は、単なる実数ではなく、インスタンスへの参照になりました。インスタンス ID 番号を得るには real()または int64()を呼び出すことでインスタンスID番号を得ることができます。

注：この変数の値は、room エディタでインスタンスが与えられている識別子と同じではありません。 instance_idの配列とも異なります。この配列には、現在アクティブなインスタンスのすべての ID が含まれています。

構文です。

id;

リターンです。

Instance ID

例

for (var i = 0; i < instance_count; i++;)
{
    if (instance_id[i] != id)
    {
        instance_id[i].scr += 5;
    }
}

上記のコードは、コードを実行しているインスタンス以外のroom のすべてのアクティブなインスタンスの "scr" 変数に 5 を追加しています。これは、すべてのアクティブなインスタンスをループして（「instance_id」配列を使って各アクティブなインスタンスのIDを返す）、内蔵の「id」変数（コードを実行している元のインスタンスのID）と比較することによって行われます。
