layer

この組み込み変数は、room のインスタンスごとに作成され、そのインスタンスが割り当てられているレイヤーのレイヤー ID 値を含んでいます。この値は、次のような他の関数で使用することができます。 layer_get_depth()を変更し、インスタンスを別のレイヤーに移動させることができます。ただし、割り当てられるレイヤーが現在のroom に存在しない場合、エラーが発生し、ゲームが強制終了されることに注意してください。レイヤーを割り当てる際には、関数「ColorNavigator」が返す一意のレイヤーIDを指定する必要があります。 layer_get_id()(名前付きroom レイヤーを使用している場合)、または関数 layer_create()(ランタイムに独自のレイヤーを作成した場合)。

重要事項 instance_create_depth()関数を使用するか、手動で depthという変数がある場合、インスタンスに割り当てられたレイヤーは、GameMakerが自動的に制御・管理する「マネージド」レイヤーとなる。この場合、 layer 変数は -1 を返します。

構文です。

layer

リターンです。

Layer ID

例

layer = layer_create(-1000);

上記のコードでは、深さ -1000 の新しいレイヤーを作成し、インスタンス layer 変数を返されたレイヤー ID に設定し、インスタンスを現在いるレイヤーから新しく作成されるレイヤーに移動させます。
