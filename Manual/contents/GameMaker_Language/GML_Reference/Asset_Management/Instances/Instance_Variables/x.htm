x

インスタンスのx値は、現在のroom における水平方向の位置で、ピクセル単位で測定されます。この値は、0、プラス、マイナスのいずれかになります。0は、room の左側で、右に移動するとxが増加し、左に移動するとxが減少します（マイナスになると、xは減少します）。 x の値は、インスタンスがroom の左側からはみ出したことを意味します。）x に 12.345 のような実数値を与えることもできますが、1 ピクセルの .345 を得ることはできないので、インスタンスが移動したように見えないかもしれません。 であり、GameMakerの内部で使用されるものであり、全く問題ない。

object x（とy）の値を設定することで、room の周りを好きな位置にジャンプさせることができます。また、実際に内蔵されている speed と direction関数を使用します。ある種の動きは、他のゲームプレイスタイルよりも適していることが多いので、ゲームを作る際に考慮する必要があります。

構文です。

x;

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

上記のコードでは、矢印キーでxの位置に加算・減算することで、動いているような錯覚を起こします。
