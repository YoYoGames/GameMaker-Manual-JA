重力方向

gravity_direction は、すべてのインスタンスが持つ「ビルトイン」プロパティの1つで、インスタンスが「ビルトイン」である場合に移動の方向を設定するために使用することができます。 gravityGameMakerでの方向は通常、0°が右、90°が上、180°が左、270°が下として計算されることに注意してください。

構文です。

gravity_direction;

リターンです。

(single precision floating point value)

例

if place_meeting(x, y, obj_switch)
{
    gravity_direction += 180;
}

上記のコードは、プレイヤーobject が "obj_switch" のインスタンスに出会うと、gravity_direction を変更します。
