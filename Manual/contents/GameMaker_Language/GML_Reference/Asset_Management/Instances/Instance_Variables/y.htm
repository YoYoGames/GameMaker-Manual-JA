y

インスタンスのy値は、現在のroom における垂直方向の位置で、ピクセル単位で測定されます。この値は、0、プラス、マイナスのいずれかであり、0はroom のトップであり、下に移動するとyは増加し、上に移動すると減少する（yの値がマイナスであることは、yがマイナスであることを意味する）。 は、インスタンスがroom の上端からはみ出したことを意味します。）y に 12.345 のような実際の値を与えることもできますが、1 ピクセルの .345 を得ることはできないので、インスタンスは移動したように見えないかもしれません。 GameMakerは完全に有効です。

object y (および x) 値を設定することで、room の周りを好きな位置にジャンプさせることができます。また、実際に内蔵されている speed と direction関数を使用します。ある種の動きは、他のゲームプレイスタイルよりも適していることが多いので、ゲームを作る際に考慮する必要があります。

構文です。

y

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

上記のコードでは、矢印キーでy位置を加減することで、動いているような錯覚を起こします。
