さきばしり

この組み込み変数は、インスタンスの前のy 位置を返します。この変数は、ステップ開始イベントの直前に現在の x 位置に設定されますが、いつでもコードで設定することができますので、必要に応じて独自の値を与えることができます。

構文です。

yprevious;

リターンです。

(single precision floating point value)

例

if x != xprevious || y != yprevious
{
    moved = true;
}

上記のコードでは、 xprevious と yprevious の値と、現在の x と y の値に違いがあるかどうかをチェックしています。もし差があれば、変数をtrueに設定する。
