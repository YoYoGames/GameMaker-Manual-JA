重力

gravity は、すべてのインスタンスが持つ「内蔵」変数のひとつで、設定すると、そのインスタンスに一定の力を加える。 gravity_directionのインスタンスに影響を与え、インスタンス speed と direction の両方に影響を与えます。重力は累積的な力であり、最終的な速度に上限を設けないことを選択した場合、object が加速することに注意してください。通常、この変数は 0.01 のような小さな小数値に設定します。 gravity を 0 に設定すると、インスタンスに重力はかかりません (これがデフォルト値です)。

構文です。

gravity;

リターンです。

(single precision floating point value)

例

if !place_meeting(x, y + 1, obj_Ground)
{
    gravity = 0.01;
}
else
{
    gravity = 0;
}

上記のコードは、インスタンスの下に "obj_Ground "のインスタンスが見つからない場合にのみ、重力を適用します。
