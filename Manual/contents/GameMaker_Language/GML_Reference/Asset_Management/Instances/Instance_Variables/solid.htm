ソリッド

インスタンスは、Object Editor のobject プロパティ、またはこの組み込み変数の値を変更することで、ソリッドとフラグを立てることができます。 solid を true に設定すると、衝突が検出されたとき、衝突したインスタンスは自動的に衝突前のステップの位置に戻されます（その後、衝突イベントで任意のコードやアクションが実行されます）。 false に設定された場合、すべての位置決めは衝突イベントで処理される必要があります。

構文です。

solid;

リターンです。

例

if other.solid = true
{
    x = xprevious;
    y = yprevious;
}

上記のコードは、衝突した「もう一方の」インスタンスの solidflag をチェックし、それが true に設定されていれば、そのインスタンスを以前の位置に移動させるものです。
