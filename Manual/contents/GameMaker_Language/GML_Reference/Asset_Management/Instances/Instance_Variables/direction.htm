方向

GameMakerのすべてのインスタンスは、その外観や動作を制御するために使用・設定できる特定の「ビルトイン」プロパティを持っています。 Direction はこれらのプロパティの 1 つで、インスタンスの移動方向を設定するために使用することができます。 インスタンスが speedGameMakerでの方向は通常、0°が右、90°が上、180°が左、270°が下として計算されることに注意してください。 gravityと gravity_direction変数がゲーム内で使用されると、 direction の値を変更することができます。

構文です。

direction;

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上記のコードは、どちらのキー（左または右）が押されたかに応じて、インスタンスの移動方向を変更するものです。
