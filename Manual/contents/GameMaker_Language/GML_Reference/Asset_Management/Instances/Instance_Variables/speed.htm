GameMakerのすべてのインスタンスは、その外観や動作を制御するために使用・設定できる特定の「組み込み」プロパティを持っています。 speed はこれらのプロパティの 1 つで、インスタンスが 1 ステップごとに何ピクセル動くかを定義します。 とは異なり hspeedと vspeedによって制御されるため、 speed には方向性がありません。 directionインスタンスの値ですが この場合、インスタンスは direction 関数で設定した方向とは反対の方向に移動します (例: direction - 180°)。なお、 speed および/または direction を設定します。 は、変数 hspeed と vspeed の値も変更します。 gravity, gravity_direction と frictionはすべて、ゲーム内で使用されたときに、この変数の値を変更することができます。

構文です。

speed;

リターンです。

(single precision floating point value)

例

if keyboard_check(vk_up) speed = 2;
if keyboard_check(vk_left) direction += 5;
if keyboard_check(vk_right) direction -= 5;

上記のコードでは、矢印キーを使ってインスタンスの速度と方向を設定します。
