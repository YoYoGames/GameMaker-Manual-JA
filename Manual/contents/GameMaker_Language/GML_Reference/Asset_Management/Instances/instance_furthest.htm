インスタンス最遠

この関数は、与えられたobject のすべてのインスタンスをチェックして、どれが与えられた x/y 点から最も遠いかを確認します。すべてのチェックは、指定された x/y 位置から、指定されたobject のインスタンスの原点(x/y 位置) まで行われます。object のインスタンスが存在しない場合、この関数はキーワードを返します。 nooneを返しますが、存在する場合は、そのインスタンスの idのインスタンスが見つかりました。コードを実行しているインスタンスが、チェック対象のobject と同じobject インデックスを持つ場合、そのインスタンスもチェックに含まれることに注意してください (呼び出し元のインスタンスが親の子でもある場合、親のobjects のチェックも含まれます)。

構文です。

instance_furthest(x, y, obj);

論証タイプ 商品説明
x The x position to check for instances far from.
y The y position to check for instances far from.
obj The object to check for instances of.

リターンです。

Instance ID or noone

例

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
    draw_line(x, y, inst.x, inst.y);
}

上記のコードは、コードを実行しているインスタンスと同じobject インデックスを共有する最も遠いインスタンスを見つけ、その id を変数に格納します。この変数が呼び出し元のインスタンスの id と同じかどうかをチェックし、同じでない場合は 2 つのインスタンスの間に線を引きます。
