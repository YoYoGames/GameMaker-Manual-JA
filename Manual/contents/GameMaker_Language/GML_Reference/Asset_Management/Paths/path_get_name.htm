パス名

この関数は、path として参照されているstring の名前を返します。 この名前は、エディタでpath を定義するために使用されたものであるか、 (path がコード関数によって作成されている場合) " という形式のstring が返されます。 _newpathXX" ここで、"XX" は生成されたpath の番号で、0 から始まり、新しいpath が作成されるたびに 1 ずつ増加します。これはあくまで string path を直接参照することはできません。そのためには、パスインデックスIDが必要です。しかし、このstring を使って、返されたstring と一緒にパスのインデックス IDを取得することはできます。 という関数で asset_get_index().

構文です。

path_get_name(index);

論証タイプ 商品説明
index The index of the path to check.

リターンです。

例

path_name = path_get_name(path_array[0]);

これは、"path_name "に、与えられた配列の0番目の位置でインデックスされたpath の名前を設定します。
