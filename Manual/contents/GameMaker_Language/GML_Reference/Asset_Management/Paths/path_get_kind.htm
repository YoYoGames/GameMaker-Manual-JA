path_get_kind

パスは滑らかなものと直線的なものがあります（滑らかなpath は定義点の周りに曲線的なpath を計算しますが、直線的なpath はある点から別の点にまっすぐ進むだけです）。この 関数は、与えられたpath が滑らかであるか (true) 、そうでないか (false) を調べるために使用することができます。

構文です。

path_get_kind(index);

論証タイプ 商品説明
index The index of the path to check.

リターンです。

例

path_kind = path_get_kind(pth_Patrol);

path これは、与えられたpath のインデックスがどのような種類のものであるかによって、 "path_kind" 変数を true または false に設定するものである。
