パスゲットスピード

この関数は、path 上の任意の位置の速度係数を返します。 位置は 0 から 1 の間の値である必要があります (次のように使用できます。 path_position例えば、インスタンスの現在の位置が必要な場合）0が開始位置、1が終了位置です。返される値は、実際の path_speed が設定されている係数になります。例えば、0と1の2点しかないpath で、0点の速度係数が100、1点の速度係数が0である場合、関数 path_get_speed(path, 0.5) は50を返します。

構文です。

path_get_speed(ind, pos);

論証 タイプ 商品説明
index パスアセット The index of the path to check.
pos リアル How far through the path to check. Between 0 (start) and 1 (end), can be decimal.

リターンです。

Real

例

speed = path_get_speed(pth_Patrol, 0.5) / 20;

これは、呼び出し側のインスタンスの速度を、与えられたpath のちょうど半分の地点の速度係数、20 で割ったものに設定します。つまり、見つかったスピードファクターがデフォルトの100であれば、呼び出し元のインスタンスのスピードは5に設定されます。
