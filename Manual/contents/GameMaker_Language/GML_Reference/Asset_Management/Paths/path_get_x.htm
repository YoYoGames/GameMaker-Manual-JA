path_get_x

この関数を使うと、任意のpath 上の位置の x 座標を得ることができます。 0 はpath の開始点、1 はpath の終了点、その中間はpath の通過点に相当します。 これはpath の定義点である必要はなく、その上の任意の位置であっても構いません。

構文です。

path_get_x(ind, pos);

論証 タイプ 商品説明
index パスアセット The index of the path to check.
pos リアル How far through the path to check. Between 0 (start) and 1 (end).

リターンです。

Real

例

x = path_get_x(pth_WallWalk, 0.5);

これは、呼び出したインスタンスの x を、与えられたpath のちょうど半分の位置にある点の x 座標に設定します。
