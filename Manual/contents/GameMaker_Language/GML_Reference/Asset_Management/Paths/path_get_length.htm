path_get_length

この関数を使うと、path の正確な長さをピクセル単位で得ることができます。これは、点から点へのおおよその長さではなく、path の形状に沿った正確な長さで、path が滑らかである場合にも有効です。 は高い曲線精度を持つ。

構文です。

path_get_length(index);

論証タイプ 商品説明
index The index of the path to measure.

リターンです。

例

path_len = path_get_length(pth_AI);

これにより、"path_len "には "pth_AI "でインデックスされたpath の全長がピクセル単位で設定されます。
