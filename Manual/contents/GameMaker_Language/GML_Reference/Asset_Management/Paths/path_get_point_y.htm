path_get_point_y

この関数は、インデックスを指定したroom について、入力された点の y 位置 (path 座標) を返します。もしその点がpath の範囲外であった場合 (たとえばpath が 8 点で、10 点の y 位置を求める場合)、値 0 が返されます。

構文です。

path_get_point_y(index, n);

論証 タイプ 商品説明
index パスアセット The index of the path to check.
n リアル The point number to check.

リターンです。

Real

例

if (path_position == 1)
{
    var _num = path_get_number(pth_Patrol);
    var _pos = floor(random(_num));
    x = path_get_point_x(pth_Patrol, _pos);
    y = path_get_point_y(pth_Patrol, _pos);
    path_position = (1 / _num) * _pos;
}

上記のコードは、インスタンスがpath の末尾にあるかどうかをチェックします。 もし末尾にある場合は、path 上のランダムなポイントを選び、そのポイントにインスタンスを移動させます。
