パス

GameMakerには、ゲーム内でpaths を使用するための関数と組み込み変数が豊富に用意されています。このセクションでは、パスエディタで作成されたpaths を扱うときに使う基本的なものを扱いますが、他にもコードを使って独自のpaths を作成したり、様々な方法で操作できる関数もあります。

パスを扱う基本的な機能として、以下のものがある。

  • path_start
  • path_end
  • path_get_closed
  • path_get_kind
  • path_get_length
  • path_get_name
  • path_get_number
  • path_get_point_speed
  • path_get_point_x
  • path_get_point_y
  • path_get_precision
  • path_get_speed
  • path_get_x
  • path_get_y

debugging paths を補助するために、画面にも描画する機能があります。

  • draw_path

object の各インスタンスには、paths を使いやすくするために設計された一連の組み込み変数もあります。これらの変数を次のセクションにリストアップします。

  • パス変数

最後に、ゲーム実行中にpath の内容を操作したり、新しいpaths を作成することが可能です。これらのことを可能にする関数や、それ以上のことは、次のセクションから利用可能です。

  • パスマニピュレーション
