path_get_point_speed

この機能を使うと、ポイント（Path Editorで定義されたもの、またはpath のポイントを動的に追加するときに path_add_point()) パーセントで表示されます。つまり、path エディタでpath ポイントを 50 に設定している場合、この関数を使用すると 50 が返されます。

構文です。

path_get_point_speed(index, n);

論証 タイプ 商品説明
index パスアセット The index of the path to check.
n リアル The point number to check.

リターンです。

Real

例

if (path_get_point_speed(pth_Patrol, 0) != 100)
{
    var _px = path_get_point_x(pth_Patrol, 0);
    var _py = path_get_point_x(pth_Patrol, 0);
    path_change_point(pth_Patrol, 0, _px, _py, 100);
}

上記のコードは、"pth_Patrol" でインデックスされたpath のpath ポイント "0" をチェックし、速度が 100 に等しくないかどうかを確認します。もしそうでなければ、その点の速度を100に設定する。
