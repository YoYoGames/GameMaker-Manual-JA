path_get_closed

この関数は、path が closed (true) か open (false) か、つまりpath loops が明確な始まりと終わりを持っているかどうかを返すために使用することができます。

構文です。

path_get_closed(index);

論証 タイプ 商品説明
index パスアセット The index of the path to check.

リターンです。

Boolean

例

state = path_get_closed(pth_Patrol);

これは、"pth_Patrol "でインデックスされたpath の閉じた状態に応じて、"state "を true または false のいずれかに設定します。
