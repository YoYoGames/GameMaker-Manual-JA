path_get_precision

この関数は、与えられたpath がどの程度の精度で "スムージング" されているかを返すもので、1 から 8 までの整数値となる。この値は、直線的なpath に対して取得 (および設定) できますが、path の種類が "smooth" に設定されている場合にのみ関係するため、インスタンスがpath をどのように使用するかに影響を与えません。

構文です。

path_get_precision(index);

論証タイプ 商品説明
index The index of the path to check.

リターンです。

例

if (path_get_kind(pth_Patrol))
{
    if (path_get_precision(pth_Patrol)) != 8
    {
        path_set_precision(pth_Patrol, 8);
    }
}

上記のコードは、"pth_Patrol "でインデックスされたpath がどのようなものかをチェックし、それがスムースであれば、次にスムージングの精度をチェックするものである。もし8より小さければ、8という値が設定されます。
