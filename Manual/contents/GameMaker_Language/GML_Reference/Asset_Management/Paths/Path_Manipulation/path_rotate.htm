パスローテート

この関数を使うと、与えられたpath をその中心を軸に回転させることができます。GameMakerでは（物理を使わない限り）角度は反時計回りに計算されるので、path を 90 度回転させるとpath が左に回転することを覚えておいてください。この関数は実際のpath リソースを変更するので、この関数が使われた瞬間からゲームの終了まで、ゲーム内のすべてのインスタンスでpath がどのように使われるかに永久に影響を与えます。これが必要でない場合は、 path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出します（必要なくなったら path_delete() をasset に対して呼び出すのを忘れないでください）。

注意：path をインスタンスに割り当てた後 path_start()関数を使用すると、 path_orientation 変数を使用してpath の回転も変更できます。

構文です。

path_rotate(index, angle);

論証 タイプ 商品説明
index パスアセット The index of the path to flip.
angle リアル The angle of rotation degrees.

リターンです。

N/A

例

path_rotate(pth_Patrol, 90);

これは、与えられたものを反時計回りに90度回転させることになります。
