パス_アドポイント

この機能により、指定したpath にポイントを追加し、その速度係数を設定することができます。この点は、点の位置に関係なく、path の末尾に追加されます。 path_speedpathのこの点に到達したとき、後続のインスタンスが実際に進むことを示します。path の点を末端ではない他の位置に配置したい場合は、次のようにします。 path_insert_point().

path ブラウザからAsset に使用する場合、この関数は実際のアセットを変更するため、関数を使用した瞬間からゲーム終了まで、ゲーム内のすべてのインスタンスがpath を使用する方法に永続的に影響を与えることに注意してください。これが必要でない場合は、 path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出します（必要なくなったら path_delete() をasset に対して呼び出すのを忘れないでください）。

構文です。

path_add_point(index, x, y, speed);

論証 タイプ 商品説明
index パスアセット The index of the path to add the point to.
x リアル The x coordinate of the new point.
y リアル The y coordinate of the new point.
speed リアル The speed factor of the point (default value is 100).

リターンです。

N/A

例

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
    path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

上記のコードは、コードを実行しているインスタンスの x/y 位置を中心とした円形path を作成します。
