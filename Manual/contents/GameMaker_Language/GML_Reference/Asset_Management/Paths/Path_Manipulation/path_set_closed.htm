path_set_closed

この関数は、与えられたflag が開いている（false）か閉じている（true）かをpath するために使用される。閉じたpath は、始点と終点がつながっていて、loop を形成しています。開いたpath は、始点と終点がつながっていない、明確なものです。この関数は実際のpath アセットを変更するので、この関数を使用した瞬間からゲームの終了まで、ゲーム内のすべてのインスタンスがpath を使用する方法に永続的に影響を与えます。これが必要でない場合は、まず path_duplicate() などの関数を使ってpath のコピーを作成し、複製したasset に対してこの関数をコールします ( path_delete() が不要になったらasset に対してコールするのを忘れないでください)。

構文です。

path_set_closed(index, closed);

論証 タイプ 商品説明
index パスアセット The index of the path to change.
closed ブーリアン Whether the path is closed (true) or not (false).

リターンです。

N/A

例

path_set_closed(pth_Patrol, true);

これにより、与えられたpath が閉じたpath に設定されます。
