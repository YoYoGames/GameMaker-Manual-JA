パスが存在する

この関数は、与えられたインデックスを持つpath が存在するかどうかを返す。まだ宣言されていない変数を通してpath の存在を確認した場合、エラーを投げることに注意してください。

構文です。

path_exists(index);

論証 タイプ 商品説明
index パスアセット The index of the path to check for.

リターンです。

Boolean

例

if (path_exists(my_path))
{
    path_delete(my_path);
}

このコードは、与えられた変数にpath のインデックスが格納されているかどうかを確認し、格納されている場合はpath をメモリから削除します。
