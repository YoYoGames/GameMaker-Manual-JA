パスフリップ

この関数は、path のすべてのポイントを取り、横軸に沿って反転させます。この関数は実際のpath アセットを変更するので、この関数が使われた瞬間からゲームの終了まで、ゲーム内のすべてのインスタンスによるpath の使用方法に永続的に影響します。これが必要でない場合は、まず path_duplicate() などの関数を使ってpath のコピーを作成し、複製したasset に対してこの関数を呼び出します ( path_delete() が不要になったらasset に対して呼び出すのを忘れないでください)。

構文です。

path_flip(index);

論証タイプ 商品説明
index The index of the path to flip.

リターンです。

例

path_flip(mypath);

これは、変数 "mypath "でインデックスされたpath を横軸に反転させるものである。
