パスシフト

この関数を使うと、path を水平軸と垂直軸に沿って移動させることができます。path に続くインスタンスが（絶対的ではなく）相対的にそれに従う場合、ゲーム内では目に見える効果はないことに注意してください。 の位置は無視されます。この関数は実際のpath アセットを変更するので、この関数が使われた瞬間からゲーム終了まで、ゲーム内のすべてのインスタンスによるpath の使用方法に永続的に影響を与えます。もし、これが path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出す必要があります。 パス_デリート()が不要になったら、asset )。

構文です。

path_shift(index, xshift, yshift);

論証タイプ 商品説明
index The index of the path to change.
xshift The number of horizontal pixels to shift the path. Negative=left, positive=right.
yshift The number of vertical pixels to shift the path. Negative=up, positive=down.

リターンです。

例

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

上記のコードでは、path asset を複製し、その ID 値を変数に格納しています。次にこの変数を使って複製されたpath の位置を移動し、インスタンスはpath を開始します。
