パス変更点

この機能により、あらかじめpath で定義された任意の点の位置や速度係数を変更することができます（path はpath エディタで作成されたもの、または以下の方法でコード化されたものです）。 path_add()).path ブラウザからAsset に使用する場合、この関数は実際のアセットを変更するため、関数を使用した瞬間からゲーム終了まで、ゲーム内のすべてのインスタンスによるpath の使用方法に恒久的に影響することに注意してください。これが必要でない場合は、 path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出します（必要なくなったら path_delete() をasset に対して呼び出すのを忘れないでください）。

構文です。

path_change_point(index, n, x, y, speed);

論証 タイプ 商品説明
index パスアセット The index of the path to change a point of.
n リアル The defining point to change the attributes of.
x リアル The new x coordinate (relative to the path) of the point.
y リアル The new y coordinate (relative to the path) of the point.
speed リアル The new speed factor of the point.

リターンです。

N/A

例

for (var i = 0; i < path_get_number(path0); ++i;)
{
    var px = path_get_point_x(pth_Patrol, i) + 64 - random(128);
    var py = path_get_point_y(pth_Patrol, i) + 64 - random(128);
    path_change_point(pth_Patrol, i, px, py, 100);
}

loops 上記のコードは、path の "path0" というインデックスを持つすべてのポイントを通過し、128x128 ピクセルの領域内のランダムな位置にすべてのポイントを再配置します。
