パス・リスケール

この関数は、与えられたpath を縦軸と横軸の両方（または一方）に沿って再スケールするために使用できます。基本的に、path の各ポイントは、path の中心を中心にこのスケールに対応する新しい位置に移動します。この関数は、実際のpath アセットを変更します。 そのため、この関数が使われた瞬間からゲームの終了まで、ゲーム内のすべてのインスタンスでpath がどのように使われるかに永続的に影響します。これが必要でない場合は、 path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出します（必要なくなったら path_delete() をasset に対して呼び出すことを忘れないでください）。

構文です。

path_rescale(index, xscale, yscale);

論証タイプ 商品説明
index The index of the path to scale.
xscale What to multiply the current horizontal scale by. Default scale is 1.
yscale What to multiply the current vertical scale by. Default scale is 1.

リターンです。

例

path_rescale(pth_AI, 3, 3);

これは、与えられたpath を元のサイズの3倍に再スケールするものです。
