path_set_kind

この関数は、指定されたリソースの種類をpath に設定するために使用することができます。これは、直線的なpath (0 に設定) か、平滑化されたpath (1 に設定) のどちらかになります。この場合、path の精度も考慮されなければなりません (精度 を使用して設定することができます。 path_set_precision()).この関数は実際のpath リソースを変更するので、ゲーム内のすべてのインスタンスがpath を使用する方法に、その瞬間から永続的に影響を与えます。 この関数が必要でない場合は、 path_duplicate() のような関数を使って、まずpath のコピーを作成し、それからこの関数を呼び出す必要があります。 を複製されたasset に対して呼び出します (不要になったらasset に対して path_delete() を呼び出すことを忘れないでください)。

構文です。

path_set_kind(index, val);

論証タイプ 商品説明
index The index of the path to change.
val The kind of the path, 0 for straight or 1 for smooth.

リターンです。

例

path_set_kind(pth_Patrol, true);

これは、与えられたpath を「滑らかな」path に設定します。
