パス削除

このコードを使用して、path をメモリから削除することができます。もしこのpath が path_add()path path path がパスエディターで作成された場合、 は永久に削除されるため、path ゲーム全体ではアクセスできなくなります。

構文です。

path_delete(index);

論証タイプ 商品説明
index The index of the path to delete.

リターンです。

例

var t_path = path_add();
if (mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1))
{
    path_assign(mypath, t_path);
}
path_delete(t_path);

上記のコードでは、path を作成し、そのインデックスをローカル変数に格納します。このpath は、そのインデックスをローカル変数に格納するために使用されます。 mp_grid_path()生成されたpath は、ターゲットへの道を見つけることに成功した場合、"mypath" でインデックスされたpath に代入される。最後に "t_path "は削除される。
