パス複製

この関数は、path を受け取り、それを新しいpath にコピーします。その過程で新しいpath が作成され、そのインデックスが返されます。これは、この新しいpath を使用するために、以降のすべての呼び出しで使用されます。

構文です。

path_duplicate(index);

論証タイプ 商品説明
index The index of the existing path to duplicate.

リターンです。

Path Asset

例

mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4));

上記のコードは、4つのpath リソースの中から1つを選んで複製し、新しいpath のインデックスを変数 "mypath" に格納しています。
