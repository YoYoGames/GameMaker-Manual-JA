パスアド

この関数を使うと、path エディタを使わずにGameMakerでpath を作成することができます。この関数は、path のインデックスを返します。これを変数に保存して、以後path の参照先として使用する必要があります。作成されたpath は空、つまりポイントが定義されていないことに注意してください。そのため、他の利用可能な関数を使用してpath にポイントを追加するか、MP グリッドを使用してpath を生成する必要があります。path の使用を終了するか、新しいものを作成してそのインデックスを同じ変数に格納したい場合、まず、古いpath を次のようにして削除する必要があります。 path_deleteメモリリークでゲームがクラッシュするのを防ぐためです。

構文です。

path_add();

リターンです。

Path Asset

例

global.newpath = path_add();

これにより、新しいpath が作成され、そのインデックスが global.newpath に割り当てられます。
