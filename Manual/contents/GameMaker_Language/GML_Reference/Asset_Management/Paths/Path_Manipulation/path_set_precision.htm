path_set_precision

この関数では、path の「滑らかさ」を変更することができます。この値は 1 から 8 の間でなければなりません。低い値では、ポイント間の曲線がより鮮明でまっすぐなエッジになり、高い値ではポイントが丸まってpath がより「曲面的」になります。この関数は、path がpath エディタでスムースに設定されていない場合、または関数 path_set_kind().この関数は実際のpath アセットを変更するため、この関数を使用した瞬間からゲーム終了まで、ゲーム内のすべてのインスタンスでpath がどのように使用されるかに永久に影響を及ぼします。これが必要でない場合は、 path_duplicate() のような関数を使用して、まずpath のコピーを作成し、複製されたasset に対してこの関数を呼び出します（必要なくなったら path_delete() をasset に対して呼び出すのを忘れないでください）。

構文です。

path_set_precision(index, prec);

論証 タイプ 商品説明
index パスアセット The index of the path to change.
prec リアル The precision of the path. Must be an integer between 1 and 8.

リターンです。

N/A

例

path_set_precision(pth_Patrol, 2);

これは、与えられたパスの精度を2に設定します。
