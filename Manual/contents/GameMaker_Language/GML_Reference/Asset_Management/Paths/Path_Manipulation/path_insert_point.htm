path_insert_point

この機能を使うと、新しいポイントをpath に挿入することができます（path は、path エディタで作成されたもの、あるいは path_add()).この点は、関数で指定された点 "n" の前にpath に追加されます。

構文です。

path_insert_point(index, n, x, y, speed);

論証 タイプ 商品説明
index パスアセット The index of the path to insert the point into.
n リアル The defining point to insert the new point BEFORE.
x リアル The x coordinate (relative to the path) of the new point.
y リアル The y coordinate (relative to the path) of the new point.
speed リアル The speed factor of the point.

リターンです。

N/A

例

path_insert_point(mypath, 0, 50, 50, 100);

これは、変数 "mypath "でインデックスされたpath 、(50,50)の一番最初に点を挿入し、速度係数は100%になります。
