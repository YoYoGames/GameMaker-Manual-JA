パスオリエンテーション

この変数は、関数 path_start() が呼び出されたときにインスタンスに割り当てられたpath の現在の向きを保持する。path が作成されたとき、その向きはデフォルトの 0度ですが、この値を使って好きな値に設定することができます。GameMakerでは（物理を使わない限り）角度は反時計回りに計算されるので、path の向きを設定することを覚えておいてください。 を90°に設定すると、path が左に回転します。

構文です。

path_orientation;

リターンです。

例

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));
path_start(path, 4, path_action_reverse, 0);
path_orientation = 90;

上記のコードでは、あらかじめ作成したランダムなpath asset を変数「mypath」に複製しています。この新しいpath を起動し、90°回転させる。
