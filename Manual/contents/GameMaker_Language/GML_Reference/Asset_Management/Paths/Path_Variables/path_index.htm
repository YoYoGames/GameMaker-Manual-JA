パスインデックス

GameMakerのすべてのリソースは一意な識別番号を持っており、 path_index は読み取り専用の変数で、 path_start() 関数を使用してインスタンスに割り当てられた、与えられたpath asset のインデックス値を保持する。インスタンスにpath が割り当てられていない場合、この変数には -1 がセットされる。

構文です。

path_index;

リターンです。

Path Asset

例

if (path_index == -1)
{
    path_start(pth_enemy3, 4, path_action_reverse, 0);
}

上記のコードは、path がインスタンスに割り当てられているかどうかをチェックし、そうでない場合は新しいpath を開始します（同時に path_index に割り当てます）。
