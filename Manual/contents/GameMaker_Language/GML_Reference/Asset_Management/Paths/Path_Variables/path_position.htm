path_position

この関数は、path に沿ったインスタンスの位置を取得または設定するために使用します。値は 0 - 1 で正規化されているので、例えば 0.5 に設定すると、インスタンスはpath のちょうど真ん中に移動します。

構文です。

path_position

リターンです。

例

path_position = random(1);

上記のコードでは、インスタンスを現在のpath の任意の位置に設定しています。
