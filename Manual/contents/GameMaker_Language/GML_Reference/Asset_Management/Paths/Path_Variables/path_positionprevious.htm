path_positionprevious

この変数は、前のステップの現在のpath に沿ったインスタンスの位置を取得または設定するために使用することができ、0と1の間で正規化された値である、すなわち、0はpath の開始位置、1は終了位置となる。これは xpreviousと ypreviousこの変数は、paths にのみ対応しています。これは、path のフォロワーが邪魔な場合に一時的に停止させるような場合に便利です（以下のコード例を参照してください）。

構文です。

path_positionprevious;

リターンです。

例

var _x = x + lengthdir_x(16, direction);
var _y = y + lengthdir_y(16, direction);
if (collision_circle(_x, _y, 16, obj_player, false, true))
{
    path_position = path_positionprevious;
}

上記のコードは、インスタンスの前の領域でobject "obj_Player" との衝突をチェックし、衝突があれば、関数 path_start() で割り当てられたように、現在のpath でインスタンスが占めていた前の位置に戻す。
