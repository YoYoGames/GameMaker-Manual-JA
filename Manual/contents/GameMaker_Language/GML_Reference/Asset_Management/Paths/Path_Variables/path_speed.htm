この関数は、関数 path_start() を使って起動した後のpath の速度を取得したり、設定したりするために使用することができます。負の値を使用することで、インスタンスに従うことを意味します。 をpath に逆転させる。

構文です。

path_speed;

リターンです。

例

path_speed = -1 + random(2);

上記のコードでは、インスタンスがpath を -1 から 1 の間のランダムな速度で移動するように設定されます。
