パススケール

この値は、インスタンスに現在割り当てられているpath (関数 path_start() で設定) のスケールを取得または設定するために使用できます。 1は1:1の縮尺で、例えば2を設定すると2倍、0.5を設定すると半分の縮尺になります。

構文です。

path_scale;

リターンです。

例

path_scale = 1 + random(2);

上記のコードでは、現在のパスのスケールを1（元のサイズ）から3（元のサイズの3倍）の間のランダムなサイズに設定します。
