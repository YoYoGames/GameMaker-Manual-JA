パス_エンドアクション

この変数は、インスタンスが現在のpath の終わりに達したときの反応を取得したり、変更したりするために使用することができます。通常は、path を開始するときに、この変数を設定します。 path_start()が、ゲーム中の様々なイベントに応じて、この動作を変更したい場合があります。使用可能な値は、以下の定数で表されます。

パスエンドアクション定数
定数 商品説明
path_action_stop パスを停止する
path_action_restart スタート位置から継続し、path が閉じていない場合はジャンプする
path_action_continue path 現在位置から再スタート
path_action_reverse path の速度を逆にする（path を逆回転させる）。

構文です。

path_endaction;

リターンです。

Path End Action Constant

例

if (path_endaction == path_action_stop)
{
    path_endaction = path_action_reverse;
}

上記のコードでは、path のエンドアクションを確認し、stop に設定されていれば、エンドアクションを reverse に変更します。
