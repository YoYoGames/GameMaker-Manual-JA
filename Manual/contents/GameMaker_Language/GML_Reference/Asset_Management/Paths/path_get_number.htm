パスゲットナンバー

この関数は、path 上の点の数を返すために使用することができます。

構文です。

path_get_number(index);

論証タイプ 商品説明
index The index of the path to check.

リターンです。

例

if (path_get_number(pth_AI) > 1)
{
    path_start( pth_AI, 4, 3, 0);
}

上記のコードは、path 上に複数のポイントがあるかどうかをチェックし、ある場合はそのpath に沿ってインスタンスの移動を開始します。
