audio_sound_gain

一定時間ごとに音をフェードイン、フェードアウトさせたり、瞬時に音のゲインを設定したりする機能です。

時間はミリ秒単位で、この関数では、その時間が終了するまでにサウンドが到達すべき最終的なゲインレベルを入力する必要があります。このゲインは0（無音）から0より大きい値まで設定できますが、通常は最大音量を1と考えます。 1以上の値も使用できますが、使用するサウンドとコンパイルするプラットフォームによっては、サウンドを再生したときに歪みやクリッピングが発生する場合があります。なお、ゲインスケールはリニアであり、瞬時にゲインを変更するには、単にtime引数を0に設定します。

この関数は、インデックスがサウンドリソースの場合、room で現在再生中のサウンドのすべてのインスタンスに影響を与え、最終的な音量が、それ以降のすべてのサウンドインスタンスを再生する音量となります。しかし、次のような関数から返されたインデックスを使用した場合は、そのサウンドのそれ以降のすべてのインスタンスが再生される音量となります。 audio_play_sound()は、その1つのサウンドのインスタンスにのみ影響します。

構文です。

audio_sound_gain(index, volume, time);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to set the gain for.
volume リアル Value for the music volume.
time リアル The length for the change in gain in milliseconds.

リターンです。

N/A

例

if val
{
    var snd = audio_play_sound(snd_fountain);
    audio_sound_gain(snd, 0, 0);
    audio_sound_gain(snd, 1, 5000);
}

上記のコードでは、変数をチェックして、 true を返したら、再生する音のインデックスをローカル変数 "snd" に代入します。この変数を使って、その特定のサウンドの音量を0にし、5秒かけてフルボリュームまでフェードアップさせます。
