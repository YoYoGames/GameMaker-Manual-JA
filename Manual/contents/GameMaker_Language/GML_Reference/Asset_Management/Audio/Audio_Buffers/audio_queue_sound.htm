オーディオキューサウンド

この関数は、buffer のデータを、以前関数で作成したオーディオqueue に追加します。 audio_create_play_queue().追加するqueue のインデックスと、データを取得するbuffer の ID、さらにbuffer 内の位置（オフセット）と追加するバイト数を指定します。buffer 一旦、queue からbuffer にオーディオを追加すると、queue は、関数 audio_free_play_queue()また、buffer のプロパティは、サウンドを追加するqueue のプロパティと一致させる必要があります。

構文です。

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

論証 タイプ 商品説明
queueIndex オーディオQueue ID The index of the queue to add to.
bufferId バッファID The buffer to add to the queue.
bufferOffset リアル The offset within the source buffer to start from.
bufferLength リアル The length of the buffer (the number of the bytes in the buffer).

リターンです。

N/A

例

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

上記のコードは非同期のAudio Recordingイベントで呼び出され、録音されたデータをbuffer に割り当て、それをオーディオqueue に追加します。そして、これが再生されます。
