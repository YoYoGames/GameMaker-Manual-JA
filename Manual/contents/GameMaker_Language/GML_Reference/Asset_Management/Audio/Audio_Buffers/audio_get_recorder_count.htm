audio_get_recorder_count

この関数は、現在ゲーム内で利用可能なオーディオ録音ソース（マイクなど）の数を返します。例えば、戻り値が4であれば、0、1、2、3のソースにオーディオ入力があり、これらの値のうち1つは、この関数を使用して録音を開始するソースとなります。 audio_start_recording().この値は、マイクや他の入力ソースをデバイスに接続したり外したりすることで、いつでも変更することができます。機能 audio_get_recorder_info()を使うと、接続されている各機器の情報を得ることができます。

重要この関数は、Opera GX ターゲットでは常に 1 を返します。

構文です。

audio_get_recorder_count();

リターンです。

Real

例

if audio_get_recorder_count() > 0
{
    channel_index = audio_start_recording(0);
}

上記のコードは、利用可能なオーディオレコーダーをチェックし、1つ以上ある場合は、インデックスされた最初のもの（ソース0）から録音を開始します。
