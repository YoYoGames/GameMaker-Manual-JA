audio_create_buffer_sound

この関数では、buffer の内容から新しい音を作ることができます。buffer は以前に作成され（作成方法の詳細はバッファ関数を参照）、データが追加または読み込まれた状態になっています。この関数に、データフォーマット（現在サポートされているのは buffer_u8 または buffer_s16 のみ）、サンプルレート（1000hz から 48000hz の間）、データを取得するbuffer のオフセットを渡します。また、buffer のサンプル数と、サウンドが必要とするチャンネルを指定する必要があります。これらのチャンネルは、以下の定数のいずれかで定義されます。

オーディオ・チャンネル・タイプ
定数 商品説明
audio_mono モノラル（1チャンネル）オーディオ。
audio_stereo ステレオ（デュアルチャンネル）オーディオ。
audio_3D 3D（5.1）オーディオ。

サウンドを作成した後、そのサウンドに関連するポインタインデックスが不要になった場合は、関数 audio_free_buffer_sound().これを怠り、変数を再代入したり、rooms 等を変更すると、サウンドIDが失われ、メモリリークを起こします。また、buffer からサウンドが作成され、そのサウンドが最初に解放されないと、削除できないことに注意してください。ですから、まずサウンドを解放してから、 buffer を削除してください。 また、サウンドが作成された後にbuffer に何かを追加したり、buffer のサイズを変更したりすると、予期しない結果になるので、お勧めできません - 一度buffer からサウンドを作成し始めたら、その後、他の方法で操作するべきではありま せん。

構文です。

audio_create_buffer_sound(bufferId, bufferFormat, bufferRate, bufferOffset, bufferLength, bufferChannels);

論証 タイプ 商品説明
bufferId バッファID The ID of the buffer to use.
bufferFormat バッファデータ型定数 The format of the data in the buffer (buffer_u8 または )。 buffer_s16
bufferRate リアル The sample rate of the data in the buffer.
bufferOffset リアル The offset into the buffer to read the sample data from (in bytes).
bufferLength リアル The length of the buffer (the number of bytes of audio data, excluding the header).
bufferChannels オーディオ・チャンネル・タイプ The channels to use from one of the constants listed above.

リターンです。

Sound Asset

例

var rate = 44100;
var hertz = irandom_range(220, 880);
var samples = 44100;

var bufferId = buffer_create(rate, buffer_fast, 1);
buffer_seek(bufferId, buffer_seek_start, 0);

var num_to_write = rate / hertz;
var length = buffer_get_size(bufferId);
var val_to_write = 1;

for (var i = 0; i < (samples / num_to_write) + 1; i++)
{
    for (var j = 0; j < num_to_write; j++)
    {
        buffer_write(bufferId, buffer_u8, val_to_write * 255);
    }
    val_to_write = (1 - val_to_write);
}

soundId = audio_create_buffer_sound(bufferId, buffer_u8, rate, 0, length, audio_stereo);

上記では、buffer を作成し、そこにデータをプロシージャルに埋めています。このデータを使って新しいサウンドを作成し、変数 "soundId "に格納する。
