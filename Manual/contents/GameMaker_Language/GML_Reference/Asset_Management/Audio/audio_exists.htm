オーディオの存在

この関数は、音が存在するか(true)、しないか(false)を返します。検索するインデックスが以前に初期化されていない場合、この関数は存在しないasset インデックスを検索しているため、エラーになることに注意してください。チェックするサウンドは、単一のサウンドのインスタンスでもかまいません（再生中の個々のサウンドのインデックスは audio_play_sound()または audio_play_sound_at()関数) またはサウンドasset.

構文です。

audio_exists(index);

論証 タイプ 商品説明
index 資産 The index of the sound to check the existence of.

リターンです。

Boolean

例

if audio_exists(global.Music)
{
    audio_play_sound(global.Music, 0, true);
}

上記のコードは、サウンドが存在するかどうかを確認するものです。もし存在すれば、loop で再生するように設定されます。
