audio_get_type

GameMakerにオーディオを追加する場合、ストリーミングか メモリ内のどちらかになります。もし、指定されたサウンドインデックスがストリームオーディオかどうかを知りたい場合は この関数は、ストリームの場合は 1、メモリ内のサウンドの場合は 0 を返し、エラーが発生した場合やインデックスが有効なサウンドリソースを指していない場合は -1 を返します。

構文です。

audio_get_type(index);

論証タイプ 商品説明
index The index of the sound to check.

リターンです。

例

type = audio_get_type(snd_Music_1);

上記のコードは、変数 "snd_music_1 "でインデックスされたオーディオの種類をチェックし、返された値を変数 "type "に格納するものです。
