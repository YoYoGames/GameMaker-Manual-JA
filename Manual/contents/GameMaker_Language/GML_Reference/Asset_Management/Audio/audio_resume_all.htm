オーディオレジュームオール

一時停止している音をすべて再開することができます。

構文です。

audio_resume_all();

リターンです。

例

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if global.Pause
    {
        audio_pause_all();
    }
    else
    {
        audio_resume_all();
    }
}

上記のコードでは、キーボードキー "P "が押されたかどうかをチェックし、押されたことを検出すると、グローバル変数 "Pause "に true または false を設定し、すべてのサウンドを一時停止するか、以前に一時停止したサウンドすべてを再開します。
