audio_sound_get_gain

この関数は、与えられたサウンドの現在のゲイン値を返します。を使用したときに変数に格納された、再生中の個々のサウンドのインデックスから参照されるサウンドのいずれかである。 audio_play_sound()または audio_play_sound_at()関数、またはasset Browser から実際のサウンドAsset を選択します。ゲインは通常0から1までの値で計算されますが、プラットフォームによっては1以上のゲインを設定することもできます。ただし、1という値は「フルボリューム」とみなされ、それ以上になるとオーディオクリップが発生する可能性があります。

構文です。

audio_sound_get_gain(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to get the gain of.

リターンです。

Real

例

if audio_sound_get_gain(snd_Music) != 1
{
    audio_sound_gain(snd_Music, 1, 0);
}

上記のコードは、"snd_music "というインデックスを持つサウンドのゲインが1でない場合、そのサウンドから再生されるオーディオのゲインを変更します。
