audio_get_master_gain

この関数を使用すると、特定のリスナーに対するすべてのサウンドと音楽のグローバルボリュームの絶対値を取得することができます。デフォルトのリスナーインデックスは0ですが、この関数で audio_get_listener_info()を使用すると、ターゲットプラットフォームで利用可能なさまざまなインデックスを取得することができます。返されるゲイン値は、0（無音）から1（フルボリューム）までのリニアスケールに基づきます。プラットフォームによっては、ゲインを1より大きくすることができますが、1という値は "フルボリューム "とみなされ、それ以上にするとオーディオクリップが発生する可能性があることに注意してください。

構文です。

audio_get_master_gain(listenerIndex);

論証 タイプ 商品説明
listenerIndex リアルまたはオーディオリスナーID The index of the listener to get the gain of.

リターンです。

Real

例

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
{
    var info = audio_get_listener_info(i);
    var ind = info[? "index"];
    if audio_get_master_gain(ind) != 1
    {
        audio_set_master_gain(info[? "index"], 1);
    }
    ds_map_destroy(info);
}

上記のコードは、loops 利用可能なリスナーを通じて、そのゲインが1であるかどうかをチェックし、そうでない場合は、それぞれのリスナーに対してそれを1に設定します。
