audio_sound_get_listener_mask

この関数は、どのオーディオリスナーから音を再生するかを定義するビットマスクデータを返します。詳細はオーディオリスナーの項を参照ください。

構文です。

audio_sound_get_listener_mask(soundID);

論証 タイプ 商品説明
soundID サウンドインスタンスID The unique ID of the sound to get the mask of

リターンです。

Real

例

var snd = audio_play_sound(snd_PlayerDead, 10, false);
if audio_sound_get_listener_mask(snd) != global.PlayerMask
{
    audio_sound_set_listener_mask(snd, global.PlayerMask);
}

上記のコードでは、音を再生した後、その音のリスナーのマスクデータをチェックし、それがグローバル変数に格納されているものと同じでない場合、グローバル変数に格納されているマスクデータを使って、再生するリスナーを設定します。
