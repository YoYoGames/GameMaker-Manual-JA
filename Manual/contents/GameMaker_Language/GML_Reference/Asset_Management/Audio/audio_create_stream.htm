オーディオクリエイトストリーム

この関数を使用すると、新しいサウンドインデックスを作成して、通常のオーディオ関数で使用し、外部のOGGファイルソースから直接オーディオをストリーミングすることができます。この関数は、ファイル名（例えば、インクルードファイル）を要求し、使用するための新しいサウンドインデックスを返します。サウンドが不要になった後は，関数 audio_destroy_stream()さもないと、メモリリークが発生してゲームの動作が遅くなり、最終的にはクラッシュする可能性があります。

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_create_stream(filename);

論証 タイプ 商品説明
filename 文字列 Path to the file (OGG only) to stream the audio from.

リターンです。

Sound Asset

例

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

上記のコードでは、与えられたファイルから新しいサウンドインデックスを変数 "snd "に作成し、このサウンドを再生しています。
