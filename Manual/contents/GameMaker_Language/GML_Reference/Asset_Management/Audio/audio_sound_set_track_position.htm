audio_sound_set_track_position

この機能は、指定されたサウンドIDまたはasset の位置（秒）を設定します。

与えられた "index "は、サウンドの単一のインスタンス(index.phpから返されたもの)であることができます。 audio_play_sound()) またはサウンドasset (例:Asset Browser から追加したもの)。この関数の動作は、指定されたインデックスの種類によって異なります。

  • もし、あなたが使っているユニークなサウンドIDで、すでに再生中のものであれば、その位置は直ちに与えられた時間に変更されます。
  • もしそれが音asset であれば、与えられた音asset のそれ以降の再生はすべて新しい時刻に開始される。

この関数は、現在再生中のサウンド、またはその次の再生のトラック位置のみを変更します（サウンドasset が提供されている場合）。サウンドのループ再生が有効な場合、次の再生はこの関数で定義したトラック位置からではなく、常に先頭（0.0秒）から開始されます。例えば、loop を 5.7 秒から開始すると、その時点から最初の音が再生されますが、それ以降は、トラックの最初から最後までを繰り返し再生し続けることになります。

構文です。

audio_sound_set_track_position(index, time);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to change.
time リアル The time (in seconds) to set the start point to. Values longer than the length of the given sound are ignored.

リターンです。

N/A

例

var snd = audio_play_sound(snd_MainTrack, 0, false);
audio_sound_set_track_position(snd, 32);

上記のコードでは、サウンドを再生し、返されたID値を使って、サウンドの開始位置を32秒に設定します。
