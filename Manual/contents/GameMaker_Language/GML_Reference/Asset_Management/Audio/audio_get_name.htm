audio_get_name

この関数は、与えられたオーディオリソースの名前をstring として返す。"index "の値は、リソース自体のもの（Asset Browser で見られる）、あるいは、例えば、using sound を再生したときに与えられるユニークな ID 値とすることが可能である。 audio_play_sound().なお、返されるstring はリソース ID とは異なり、リソース自体へのアクセスには使用できないので、表示やエラーチェックにのみ使用すること。

構文です。

audio_get_name(index);

論証タイプ 商品説明
index The index of the sound to check.

リターンです。

例

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

上記のコードでは、3つの異なるサウンドリソースからランダムにサウンドを再生し、その名前をデバッグメッセージとして表示します。
