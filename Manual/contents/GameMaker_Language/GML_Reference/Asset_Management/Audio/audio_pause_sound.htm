オーディオ_ポーズ_サウンド

現在再生中のサウンドを一時停止することができる機能です。サウンドは、単一のインスタンスであることもできます（再生中の個々のサウンドのインデックスは audio_play_sound()または audio_play_sound_at()関数) またはサウンドasset の場合、指定したサウンドのすべてのインスタンスが一時停止されます。

構文です。

audio_pause_sound(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to pause.

リターンです。

N/A

例

if keyboard_check_pressed(ord("P"))
{
    global.Pause = !global.Pause;
    if (global.Pause)
    {
        audio_pause_sound(snd_Waterfall);
    }
    else
    {
        audio_resume_sound(snd_Waterfall);
    }
}

上記のコードでは、キーボードキー「P」が押されたかどうかをチェックし、押されたことを検出すると、グローバル変数「Pause」に true または false を設定し、変数「snd_Waterfall」で指定したサウンドを一時停止するか、一時停止状態からサウンドを再開しています。
