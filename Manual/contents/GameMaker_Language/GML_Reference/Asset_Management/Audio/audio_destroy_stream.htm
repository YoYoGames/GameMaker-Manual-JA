audio_destroy_stream

ファイルからオーディオストリームを作成した場合、関数 audio_create_stream()その音が不要になった場合、この関数を呼び出してメモリから削除することができます。破棄された後にさらにサウンドインデックスを呼び出すと、エラーが発生します。この関数はストリームを解放しますが、ターゲットプラットフォームではメモリマネージャに表示されない可能性があることに注意してください。これは、GameMakerがメモリ割り当てのオーバーヘッドを防ぐためにメモリリソースをプールしているためで、メモリは他の用途に使われるか新しいストリームに再利用されるまで割り当てられたままになります。

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_destroy_stream(filename);

論証 タイプ 商品説明
filename 文字列 Path to the audio file that was streamed.

リターンです。

N/A

例

audio_destroy_stream(snd);

上記のコードは、変数 "snd "にインデックスされているサウンドをメモリーから削除します。
