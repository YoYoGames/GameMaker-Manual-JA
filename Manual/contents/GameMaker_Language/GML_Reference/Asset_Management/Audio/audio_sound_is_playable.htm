オーディオ_サウンド_イズ_プレイアブル

この関数は、与えられたサウンドインデックスが現在再生可能かどうかをチェックするために使用することができる。これは、HTML5ターゲットプラットフォームにおいて、ストリーム再生と非ストリーミング再生が異なる方法で処理されるために必要であり、サウンドが再生可能な場合は true を、再生不可能な場合は false を返します。なお、HTML5 以外のプラットフォームでは、この関数は常に true を返します。

構文です。

audio_sound_is_playable(index);

論証 タイプ 商品説明
index サウンドアセット The index of the sound to check.

リターンです。

Boolean

例

if audio_sound_is_playable(snd_Shot)
{
    audio_play_sound(snd_Shot, 10, false);
}

上記のコードは、サウンド "snd_Shot "が再生可能であれば、それを再生します。
