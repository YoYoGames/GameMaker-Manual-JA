audio_sound_set_listener_mask

この関数は、サウンドのビットマスクを設定し、指定されたリスナーからのみ再生されるようにするために使用することができる。ビットマスクの作成には audio_get_listener_info()そして、ビット単位で or ("|") を使って、サウンドを再生する必要のあるリスナーでカスタムマスクを作成し、再生後にこのカスタムマスクをサウンドに適用します。このマスクは、デフォルトのマスクや、関数 audio_set_listener_mask().

この関数は、サウンドを再生するように設定された後にのみ使用でき、サウンドを再生するために使用される関数が返すサウンドIDを必要とします。

構文です。

audio_sound_set_listener_mask(soundID, mask);

論証 タイプ 商品説明
soundID サウンドインスタンスID The unique ID of the sound to set the mask of
mask リアル The bitmask data to set for the sound

リターンです。

N/A

例

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

上記のコードでは、音を再生し、グローバル変数に格納されたマスクデータを使って、再生するリスナーを設定します。
