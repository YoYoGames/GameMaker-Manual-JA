オーディオシステムが使用可能かどうか

この関数は、オーディオシステムが初期化されたかどうか、またはオーディオコンテキストが実行されているかどうかを確認するために使用することができます。全てのプラットフォームにおいて、 HTML5ターゲット以外では、オーディオエンジンが初期化されると、この関数はGame Start直後に真を返します。HTML5では、オーディオコンテキストの状態は、ユーザーの入力、使用しているブラウザ、バージョンによって随時変化するため、この関数を使用してオーディオが再生可能かどうかを確認することができる。この関数が false を返した場合（つまり、オーディオコンテキストステータスが実行されていない場合）、ストリームされていない音声のみが再生される可能性があり（ただし、保証されているわけではないので、音声が再生できないことを想定してください）、一方 true を返した場合は、すべての音声が再生されます。

構文です。

audio_system_is_available();

リターンです。

例

if audio_system_is_available()
{
    if audio_is_paused(global.Music)
    {
        audio_resume_sound(global.Music)
    }
    else
    {
        if !audio_is_playing(global.Music)
        {
            global.Music = audio_play_sound(snd_Music, 0, true);
        }
    }
}
else
{
    if audio_is_playing(global.Music)
    {
        audio_pause_sound(global.Music);
    }
}

上記のコードは、オーディオシステムが初期化され使用可能かどうかによって、オーディオトラックを一時停止/一時停止します。
