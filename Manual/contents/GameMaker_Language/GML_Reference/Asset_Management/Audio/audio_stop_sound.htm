audio_stop_sound

この関数は、与えられたサウンドが現在再生中であれば、それを停止させる。サウンドは単一のインスタンスであることも可能です（再生中の個々のサウンドのインデックスは audio_play_sound()または audio_play_sound_at()関数) またはサウンドasset の場合，与えられたサウンドのすべてのインスタンスが停止される。

構文です。

audio_stop_sound(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to stop.

リターンです。

N/A

例

if !global.SFX
{
    audio_stop_sound(snd_Waterfall);
}
else
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

上記のコードは、グローバル変数 "SFX" をチェックして、 false を返したら、現在再生中の変数 "snd_Waterfall" でインデックスされたサウンドを停止し、 true を返したら、サウンドをloop にします。
