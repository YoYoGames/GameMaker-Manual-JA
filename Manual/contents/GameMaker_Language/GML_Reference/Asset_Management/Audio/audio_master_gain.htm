オーディオマスターゲイン

この関数では、すべてのサウンドと音楽のグローバルボリュームの絶対値を設定することができます。0（無音）から0を超える値までのリニアスケールに基づきますが、通常は最大ボリュームを1と考えます。 1以上の値を使用することもできますが、使用するサウンドやコンパイルするプラットフォームによっては、サウンドを再生した時に歪みやクリッピングが発生する可能性があります。この関数は、ゲーム内で再生される全てのサウンドや音楽の相対的なボリュームに影響を与えます。

構文です。

audio_master_gain(gain);

論証 タイプ 商品説明
gain リアル Value for the global volume (0 to 1).

リターンです。

N/A

例

if keyboard_check(vk_up)
{
    if vol < 1 vol += 0.05;
    audio_master_gain(vol);
}
if keyboard_check(vk_down)
{
    if vol > 0 vol -= 0.05;
    audio_master_gain(vol);
}

上記のコードでは、上下の矢印キーが押されたことを確認し、変数 "vol "を増減させています。この変数は、ゲーム内のすべてのサウンドと音楽のグローバルゲインを設定するために使用されます。
