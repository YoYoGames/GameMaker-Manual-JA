audio_sound_pitch

この機能は、与えられた音のピッチを変更するために使用されます。を使用したときに変数に格納された、再生中の個々の音のインデックスから参照される音のいずれかを使用することができます。 audio_play_sound()または audio_play_sound_at()asset 関数で変数に格納された再生中のサウンドのインデックスから参照したサウンド、またはAsset Browserから取得した実際のサウンドのいずれかです。再生中のサウンドのインデックスであれば、そのインスタンスのみが変更されますが、asset ブラウザからサウンドAsset を使用する場合は、再生中のサウンドasset のすべてのインスタンスが変更されます。

pitch 引数はピッチ乗算器であり、入力値は現在のピッチにその量を乗算します。したがって、デフォルトの値である 1 はピッチを変更せず、1 未満の値はピッチを下げ、1 より大きい値はピッチを上げます。この機能では、0以下または5以上の値は、いずれにせよ聞き取れない可能性があるため、小さな増分を使用することが最善です。なお、許容されるピッチ変化の総量は（1/256）-256オクターブに固定されており、これを超える値や下回る値は登録されません。

上記のクランプされた値は、GameMakerが範囲をクランプしようとする値ですが、この値はすべてのターゲットプラットフォームで保証されるわけではありません。iOS例えば、GameMakerの場合、(1/256) - 8に固定されるため、オクターブ上または下の値が必要な場合は、ターゲットプラットフォームごとに実験し、それぞれ異なるバージョンのサウンドリソースを用意し、あらかじめシフトさせる必要があるかもしれません。

構文です。

audio_sound_pitch(index, pitch);

論証 タイプ 商品説明
index サウンドアセットまたは サウンドインスタンスID The index of the sound to change.
pitch リアル The pitch multiplier (default 1).

リターンです。

N/A

例

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
    case 1: audio_sound_pitch(s_engine, 0.8); break;
    case 2: audio_sound_pitch(s_engine, 0.9); break;
    case 3: audio_sound_pitch(s_engine, 0.95); break;
    case 4: audio_sound_pitch(s_engine, 1); break;
    case 5: audio_sound_pitch(s_engine, 1.2); break;
}

上記のコードは、変数「s_engine」でインデックスされたサウンドから再生される音声のピッチを、変数「gear」の値に基づいて変更するものです。
