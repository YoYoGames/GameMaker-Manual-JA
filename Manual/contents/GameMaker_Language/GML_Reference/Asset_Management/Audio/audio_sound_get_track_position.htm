audio_sound_get_track_position

この関数は、サウンドファイル内のサウンドの再生位置（秒）を取得します。サウンドは単一のインスタンスのみです（再生される個々のサウンドのインデックスを変数に格納するには audio_play_sound()または audio_play_sound_at()関数)。

構文です。

audio_sound_get_track_position(index);

論証 タイプ 商品説明
index サウンドインスタンスID The index of the sound to get the play position of.

リターンです。

Real

例

if audio_sound_get_track_position(global.Music) != 0
{
    audio_sound_set_track_position(global.Music, 0);
}

上記のコードは、トラックをチェックして開始位置を取得し、それが0秒でなければ0秒をセットします。
