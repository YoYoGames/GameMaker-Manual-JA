オーディオ作成同期グループ

同期グループを作成し、一意の ID 値を返します。この ID は、このグループに対する以降のすべてのオーディオ関数呼び出しで使用されます。グループをloop にしたい場合は true を、そうでない場合は false を渡します。ただし、loop にしたい場合は、後で追加するすべてのトラックの長さが同じである必要があることに注意してください。シンクグループを作成したら、使用しないときはメモリとサウンドを audio_destroy_sync_group()機能（例えば、Room Endや Destroyイベントなど）を使用します。

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_create_sync_group(loop);

論証 タイプ 商品説明
loop ブーリアン Whether the tracks in the group should loop (true) か、そうでないか (false) です。

リターンです。

Audio Sync Group ID

例

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

上記では、新しい同期グループを作成し、グループのインデックスを変数 "sg "に代入しています。そして、4つのサウンドがグループに追加され、そのうち3つのサウンドのゲインを0に設定します。
