オーディオ同期グループ再生中

この関数は、同期グループ内のオーディオが再生されているかどうかを確認するために使用することができます。この関数が返す同期グループIDを指定する必要があります。 audio_create_sync_group().

この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_sync_group_is_playing(group_index);

論証 タイプ 商品説明
group_index オーディオ・シンク・グループID The group index to check.

リターンです。

Boolean

例

if audio_sync_group_is_playing(group_one)
{
    audio_stop_sync_group(group_one);
}

上記のコードでは、同期グループ group_one が現在再生中かどうかを確認し、再生中であればグループを停止しています。
