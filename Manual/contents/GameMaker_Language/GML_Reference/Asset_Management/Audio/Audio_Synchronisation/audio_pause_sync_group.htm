オーディオポーズシンクグループ

この関数は、指定された同期グループが再生中であれば、それを一時停止します。 audio_create_sync_group().これは音を止めるわけではなく audio_resume_sync_group()を実行すると、一時停止したのと同じ位置から再び再生が開始されます。

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_pause_sync_group(group_index);

論証 タイプ 商品説明
group_index オーディオ・シンク・グループID The group index to pause.

リターンです。

N/A

例

if (keyboard_check_pressed((ord)"P"))
{
    global.Pause = !global.Pause
    if (global.Pause)
    {
        audio_pause_sync_group(sg);
    }
    else
    {
        audio_resume_sync_group(sg);
    }
}

上記のコードでは、"P "キーが押されたかどうかをチェックし、押された場合は "global.Pause "変数をトグルして、変数 "sg "でインデックスされた同期グループを一時停止または再開するかどうかをチェックします。
