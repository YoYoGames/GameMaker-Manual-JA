オーディオ同期グループデバッグ

この関数は、任意の同期グループに関するデバッグ情報を表示するために使用できます（グループインデックスは、グループが関数 audio_create_sync_group()).この関数を-1の値で呼び出すと、switch オーバーレイをオフにすることができます。

debugging オーディオシンクグループの場合、オーバーレイが表示されます。

  • どの同期グループからデバッグしているか
  • 何チャンネルあるのか
  • 同期グループが何回失敗したか（これは、あまりにも多くの作業を行おうとしたことを意味します）
  • チャンネル一覧
    • 赤は、このチャンネルが現在何もしていないことを意味します。
    • 黄色は、チャンネルが初期状態（まだデコードや再生をしていない）であることを意味します。
    • 緑色は、現在シーク/デコード中であることを意味する
    • 紫色は、チャンネルのデータがなくなったことを意味します
  • デコードスレッドがどのような作業を行っているかを示すプロファイリングバー
    • 青色は、ファイルが初めて読み込まれることを意味します。
    • 赤色は、スレッドがどこから読み込むかを見つけるためにファイルを読んでいることを意味します。
    • 緑色は、スレッドがデータをデコードしていることを意味します。
    • 紫色は、作業が必要になるまでスリープ状態であることを意味します。

なお、一般的な音声のdebugging 、以下のような機能もあります。 audio_debug().

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_sync_group_debug(group_index);

論証 タイプ 商品説明
group_index オーディオ・シンク・グループID The group index to debug, or -1 to switch off the debugging.

リターンです。

N/A

例

if (debug_mode)
{
    audio_sync_group_debug(sg);
}
else
{
    audio_sync_group_debug(-1);
}

上記のコードは、ゲームがデバッグモードで実行されているか否かによって、変数 "sg "にインデックスされた同期グループを示すオーディオ同期グループデバッグオーバーレイのオン/オフをswitch 。
