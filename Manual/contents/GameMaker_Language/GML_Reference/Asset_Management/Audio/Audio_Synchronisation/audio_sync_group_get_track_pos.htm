audio_sync_group_get_track_pos

この関数は、指定された同期グループの現在の再生位置を返します。グループインデックスは、関数 audio_create_sync_group()また、戻り値はトラックが再生されている時間（秒）である。

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_sync_group_get_track_pos(group_index);

論証 タイプ 商品説明
group_index オーディオ・シンク・グループID The group index to get the position of.

リターンです。

Real

例

var real_secs = audio_sync_group_get_track_pos(sg);
var secs = real_secs mod 60;
var mins = string(real_secs div 60);
if (secs > 9)
{
    secs = string(secs);
}
else
{
    secs = "0" + string(secs);
}

draw_text(32, 32, "Time = " + mins + ":" + secs);

上記のコードでは、変数 "sg "にインデックスされた同期グループの時間位置を取得し、この値を使って再生時間を画面に描画しています。
