オーディオスタートシンクグループ

この関数を使用すると、以前に作成した同期グループの再生を開始することができます。グループインデックスを指定する必要がありますが、これは関数 audio_create_sync_group().

注：この機能は、HTML5ターゲットプラットフォームでは使用できません。

構文です。

audio_start_sync_group(group_index);

論証 タイプ 商品説明
group_index オーディオ・シンク・グループID The group index to play.

リターンです。

N/A

例

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

上記では、新しい同期グループを作成し、グループのインデックスを変数 "sg "に代入しています。そして、4つのサウンドがグループに追加され、そのうち3つのサウンドのゲインを0に設定します。
