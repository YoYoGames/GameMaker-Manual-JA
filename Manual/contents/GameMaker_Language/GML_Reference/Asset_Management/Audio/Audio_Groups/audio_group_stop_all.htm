オーディオグループ停止

この関数は、現在再生中の指定されたオーディオグループの音をすべて停止させる。

構文です。

audio_group_stop_all(groupID);

論証 タイプ 商品説明
groupID オーディオグループID The index of the audio group to stop (as defined in the Audio Groups Window)

リターンです。

N/A

例

if keyboard_check_pressed(vk_space)
{
    audio_group_stop_all(audiogroup_level1);
}

上記のコードでは、"space "キーをチェックし、与えられたグループから再生されているすべてのオーディオを停止しています。
