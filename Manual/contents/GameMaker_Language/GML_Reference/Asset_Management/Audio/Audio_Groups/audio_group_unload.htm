audio_group_unload

この関数は、与えられたオーディオグループに属するとフラグ付けされたすべてのサウンドをメモリにアンロードする。この関数は、アンロードが開始された場合は true を、グループ ID が無効な場合、または既にアンロードのフラグが立っている場合は false を返す。この関数が呼ばれたとき、現在再生中のオーディオはすべて停止することに注意すること。

構文です。

audio_group_unload(groupID);

論証 タイプ 商品説明
groupID オーディオグループID The index of the audio group to unload (as defined in the Audio Groups Window)

リターンです。

Boolean

例

if audio_group_is_loaded(audiogroup_level1)
{
    audio_group_unload(audiogroup_level1);
}

上記のコードは、オーディオグループがロードされているかどうかをチェックし、ロードされていればそれをアンロードします。
