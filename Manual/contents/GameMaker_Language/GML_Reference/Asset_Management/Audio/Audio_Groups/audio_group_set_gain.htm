オーディオグループ設定ゲイン

グループ内の音を一定時間ごとにフェードイン／アウトさせたり、グループゲインを瞬時に設定したりする機能です。

時間はミリ秒単位で、この機能では、その時間が終了するまでにグループが到達すべき最終的なゲインレベルを入力する必要があります。このゲインは0（無音）から1（フルボリューム）の間で、スケールはリニアで、0.5の値はハーフボリュームとなります。プラットフォームによっては、1以上のゲインを設定できるものもありますが、1という値は「フルボリューム」とみなされ、それ以上になるとオーディオクリップが発生する可能性があることに注意してください。

構文です。

audio_group_set_gain(groupID, volume, time);

論証 タイプ 商品説明
groupID オーディオグループID The index of the audio group to stop (as defined in the Audio Groups Window)
volume リアル The final value for the group volume.
time リアル The length of the change in gain in milliseconds.

リターンです。

N/A

例

if keyboard_check_pressed(vk_space)
{
    audio_group_set_gain(audiogroup1, 0, 5000);
}

上記のコードは、「スペース」キーをチェックし、「audiogroup1」のすべてのオーディオを5秒かけて0にフェードアウトさせるものです。
