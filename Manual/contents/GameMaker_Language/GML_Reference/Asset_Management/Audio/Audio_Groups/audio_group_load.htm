オーディオグループのロード

この関数は、与えられたオーディオグループに属するとフラグ付けされたすべてのサウンドをメモリにロードする。この関数は、読み込みが開始された場合は true を、グループ ID が無効な場合、またはすでに読み込みのフラグが立っている場合は false を返します。この関数は非同期なので、バックグラウンドでオーディオがロードされている間、ゲームは実行され続けます。つまり、グループ全体がメモリにロードされ、サウンドが使用できるようになったことを知らせる非同期ロード/セーブイベントもトリガーされます。

構文です。

audio_group_load(groupID);

論証 タイプ 商品説明
groupID オーディオグループID The index of the audio group to load (as defined in the Audio Groups Window)

リターンです。

Boolean

例

if !audio_group_is_loaded(audiogroup_level1)
{
    audio_group_load(audiogroup_level1);
}

上記のコードは、オーディオグループがロードされているかどうかを確認し、ロードされていない場合はロードします。
