オーディオグループ名

この関数は、指定されたオーディオグループの名前を含むstring を返し、表示または確認します。ゲームオプションでオーディオグループを定義するとき、一意の "name "を与えますが、これは実際にはグループのID値として使用するための定数です。この関数が行うのは、IDを受け取り、指定されたID名のstring を返すだけです。

構文です。

audio_group_name(groupID);

論証 タイプ 商品説明
groupID オーディオグループID The index value constant of the audio group to check (as defined in the Audio Groups Window)

リターンです。

String

例

var name = audio_group_name(audiogroup_level1);
draw_text(32, 32, "Now Playing Group: " + name);

上記のコードは、与えられたオーディオグループ定数の名前を取得し、画面に表示するものです。
