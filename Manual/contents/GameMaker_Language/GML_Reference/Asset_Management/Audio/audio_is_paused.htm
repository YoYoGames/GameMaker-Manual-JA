オーディオが一時停止している

この関数は、与えられたサウンドが現在一時停止しているかどうかを確認する。サウンドは、単一のサウンドのインスタンス（再生中の個々のサウンドのインデックスは audio_play_sound()または audio_play_sound_at()assetこの場合、与えられたサウンドのすべてのインスタンスがチェックされ、そのいずれかが一時停止していれば、この関数は true を返し、そうでなければ false を返します。

構文です。

audio_is_paused(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to check.

リターンです。

Boolean

例

if audio_is_paused(snd_Waterfall)
{
    audio_resume_sound(snd_Waterfall);
}

上記のコードは、変数「snd_Waterfall」でインデックスされたサウンドが現在一時停止中かどうかをチェックし、 true を返したらサウンドの再生を再開します。
