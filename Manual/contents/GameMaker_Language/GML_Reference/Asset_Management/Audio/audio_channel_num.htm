オーディオチャンネル番号

GameMakerで音声を再生できるオーディオチャンネルの数を設定する機能です。基本的には、同時に再生できるサウンドの数を指定し、その数を超えると優先度の低いサウンドを停止して、優先度の高いサウンドのためにチャンネルを空ける、ということです。オーディオのチャンネル数が少ないほどパフォーマンスが向上するため、この機能を使ってゲームをデバイス用に最適化することができますが、一度に多くのサウンドを再生すると特定のサウンドの切断が目立つようになる可能性があることに注意してください。

構文です。

audio_channel_num(num);

論証タイプ 商品説明
num Number of available audio channels (default is 128).

リターンです。

例

switch (os_browser)
{
    case browser_not_a_browser:
        switch (os_type)
        {
            case os_windows:
            case os_macos:
                audio_channel_num(200);
            break;

            default:
                audio_channel_num(64);
            break;
        }
    break;

    default:
        audio_channel_num(16);
    break;
}

上記のコードでは、ゲームが動作しているプラットフォームをチェックし、利用可能なサウンドチャンネル数を変更してパフォーマンスを向上させています。
