audio_is_playing

この関数は、与えられたサウンドが現在再生中かどうかを確認する。サウンドは、単一のインスタンスであることも可能です（再生中の個々のサウンドのインデックスは audio_play_sound()または audio_play_sound_at()assetこの場合、与えられたサウンドのすべてのインスタンスがチェックされ、それらのいずれかが再生中であれば、この関数は true を返し、そうでなければ false を返します。 true この関数は、チェックされたサウンドが、以前に audio_pause_sound()関数があります。

構文です。

audio_is_playing(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to check.

リターンです。

Boolean

例

if !audio_is_playing(snd_Waterfall)
{
    audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

上記のコードは、変数「snd_Waterfall」でインデックスされたサウンドが現在再生中かどうかをチェックし、もし false を返したら、そのサウンドをroom の位置で、落下距離300、優先度低でループさせるものです。
