audio_sound_get_pitch

この関数は、指定された音の音程を取得するために使用することができます。を使用したときに変数に格納された、再生中の個々のサウンドのインデックスから参照されるサウンドのいずれかである。 audio_play_sound()または audio_play_sound_at()関数、またはasset Browser からAsset 実際のサウンドを選択します。

構文です。

audio_sound_get_pitch(index);

論証 タイプ 商品説明
index サウンドアセットまたはサウンドインスタンスID The index of the sound to get the pitch of.

リターンです。

Real

例

if audio_sound_get_pitch(snd_Explode) != 1
{
    audio_sound_pitch(snd_Explode, 1);
}

上記のコードは、"snd_Explode "としてインデックスされたサウンドのピッチが1でない場合に、そのサウンドから再生される音声のピッチを変更します。
