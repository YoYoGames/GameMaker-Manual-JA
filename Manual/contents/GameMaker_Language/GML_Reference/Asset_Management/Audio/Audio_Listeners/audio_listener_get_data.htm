audio_listener_get_data

この関数は、DSマップを作成し、与えられたリスナーの位置、速度、および向きの値を入力します。デフォルトのリスナーインデックスは 0 ですが、関数 audio_get_listener_info()を使用して、ターゲットプラットフォームで利用可能なさまざまなインデックスを取得することができます。リスナーインデックスが正しくない場合、この関数は -1 を返します。

注意事項 返却された DSマップは 、お客様の責任において適切な機能で破棄してください。

DSマップには、以下のキーが含まれることになる。

  • 「x" - リスナーのx方向の位置です。
  • 「y" - リスナーのy方向の位置です。
  • "z" - リスナーのz位置
  • 「vx" - リスナーのX軸方向の速度．
  • 「vy" - リスナーのY軸方向の速度です。
  • 「vz" - リスナーのz軸方向の速度．
  • "lookat_x" - リスナーの視線ベクトルの x 成分．
  • "lookat_y" - リスナーの視線ベクトルの y 成分．
  • "lookat_z" - リスナーの視線ベクトルの z 成分です．
  • "up_x" - リスナーのアップベクトルの x 成分です。
  • "up_y" - リスナーのアップベクトルのy成分です。
  • "up_z" - リスナーのアップベクトルのz成分です。

構文です。

audio_listener_get_data(index);

論証 タイプ 商品説明
index リアルまたはオーディオリスナーID The listener to get the data for (default is 0).

リターンです。

DS Map ID

例

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i)
{
    var info = audio_get_listener_info(i);
    var data = audio_listener_get_data(info[? "index"]);
    if data[? "x"] != 0
    {
        audio_listener_set_position(info[? "index"], 0, 0, 0);
    }
    ds_map_destroy(info);
    ds_map_destroy(data);
}

上記のコードは、利用可能なリスナーの数をチェックし、loops 、もし彼らのxの位置が0でなければ、彼らの位置を0, 0, 0に設定します。
