audio_get_listener_mask

この関数は、オーディオリスナーのための現在のデフォルト（グローバル）マスクを定義するビットマスクデータを返す。

構文です。

audio_get_listener_mask();

リターンです。

例

var g_mask = audio_get_listener_mask();
if g_mask != global.Audio_Mask
{
    var num = audio_get_listener_count();
    global.Audio_Mask = 0;

    for(var i = 0; i < num; ++i;)
    {
        var info = audio_get_listener_info(i);
        var m = audio_listener_get_data(info[? "mask"]);
        global.Audio_Mask = global.Audio_Mask | m;
        ds_map_destroy(info);
    }
    audio_set_listener_mask(mask);
}

上記のコードでは、現在のリスナーのマスクデータを取得し、グローバル変数に格納されているデータと比較しています。もし同じでなければ、コードは利用可能なリスナーの数をチェックし、loops 、それらのマスクビットを取得し、それらを組み合わせて1つのマスクを作成し、オーディオシステムに適用してグローバルリスナーを定義します。
