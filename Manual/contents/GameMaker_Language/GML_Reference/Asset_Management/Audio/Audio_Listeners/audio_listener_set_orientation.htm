audio_listener_set_orientation

この関数を使用すると、3Dオーディオ空間内での指定したリスナーの向きを変更することができます。デフォルトのリスナーインデックスは 0 ですが、関数 audio_get_listener_info()を使用して、ターゲットプラットフォームで利用可能なさまざまなインデックスを取得することができます。

ルックアットベクトル、アップベクトルは、与えられた相対的なx、y、zの位置から分解された値を基準とし、下図のようにルックアットベクトルは（0、0、1）、アップベクトルは（0、1、0）がデフォルトとなっています。

この関数でリスナーのオリエンテーションを変更すると、ゲームroom の周囲にあるオーディオエミッターによって作られたサウンドが、ゲームプレイヤーにどのように知覚されるかが変わります。下の例では、リスナーがデフォルトの位置にいる時にエミッタによって作られた音は、リスナーの下と右側から聞こえてくるように見えますが、リスナーの位置と向きを変更すると、上と右側から聞こえてくるように認識されるようになります。

構文です。

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

論証 タイプ 商品説明
index リアルまたは オーディオリスナーID The listener to set the orientation of.
lookat_x リアル The x look vector (default 0).
lookat_y リアル The y look vector (default 0).
lookat_z リアル The z look vector (default 1).
up_x リアル The x up vector (default 0).
up_y リアル The y up vector (default 1).
up_z リアル The z up vector (default 0).

リターンです。

N/A

例

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

上記のコードは、カメラビュー[0]のビューマトリックスを取得し、それを使ってグローバル変数 "Player_Listener "に格納されたIDのリスナーのオーディオリスナーの位置と向きを設定しています。
