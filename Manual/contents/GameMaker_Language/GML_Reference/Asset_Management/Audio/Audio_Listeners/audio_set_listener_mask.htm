audio_set_listener_mask

システムで複数のリスナーを使用する場合、ある音にビットマスクを設定し、フラグを立てたリスナーからしか聞こえないようにすることができます。しかし、この機能を使ってグローバルマスクを設定すれば、各サウンドに個別にマスクを設定しなくても、通常再生されるすべてのサウンドは、このマスクでフラグづけされたリスナーから聞こえるようになります。

構文です。

audio_set_listener_mask(mask);

論証 タイプ 商品説明
mask リアル The bit-mask data to set for the listeners

リターンです。

N/A

例

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

上記のコードは、利用可能なリスナーの数をチェックし、loops 、それらのマスクビットを取得し、それらを組み合わせて単一のビットマスクを作成し、グローバルリスナーに適用します。
