audio_get_listener_info

この関数は、DS マップを作成し、与えられたリスナーの情報を入力する。

注意事項 返却された DS Map は、お客様の責任において適切な機能で破棄してください。

DSマップには、以下のキーが含まれることになる。

  • "name" - リスナーの名前 (string) 。"default" はほとんどのターゲット・プラットフォームで標準のリスナー名です。
  • 「mask" - リスナーのビットマスク。
  • 「index" - リスナー独自のインデックス値。

マスク値は、サウンドやエミッタを複数のリスナーから一度に再生するように設定するために使用します。単純にビット単位の or "|" を使用してサウンド用のマスクを生成します（以下のコード例を参照）。一方、インデックスは、次のような関数を使用して、指定したリスナーの位置や速度などのプロパティを設定するために使用されます。 audio_listener_set_position().

構文です。

audio_get_listener_info(num);

論証 タイプ 商品説明
num リアルまたはオーディオリスナーID The listener number to get the data for.

リターンです。

DS Map ID

例

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i)
{
    var info = audio_get_listener_info(i);
    var m = audio_listener_get_data(info[? "mask"]);
    mask = mask | m;
    ds_map_destroy(info);
}
audio_set_listener_mask(mask);

上記のコードは、利用可能なリスナーの数をチェックし、loops 、それらのマスクビットを取得し、それらを組み合わせて単一のビットマスクを作成し、グローバルリスナーに適用します。
