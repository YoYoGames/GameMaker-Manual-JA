オーディオリスナーの方向性

この関数を使用すると、3Dオーディオ空間内でのリスナーの向きを変更することができます。ルックアット方向とアップ方向は、与えられた相対的なx、y、z位置から分解されるベクトルに基づき、下図のように、ルックアット方向は（0、0、1）、アップ方向は（0、1、0）がデフォルトとなります。

注意 複数のリスナーがいる場合は、関数 audio_listener_set_orientation().

この関数でリスナーのオリエンテーションを変更すると、ゲームroom の周囲にあるオーディオエミッターによって作られたサウンドが、ゲームのプレイヤーにどのように知覚されるかが変化します。下の例では、リスナーがデフォルトの位置にいる時にエミッタによって作られた音は、リスナーの下と右側から聞こえてくるように見えますが、リスナーの位置と向きを変更すると、上と右側から聞こえてくるように認識されるようになります。

構文です。

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

論証 タイプ 商品説明
lookat_x リアル The x look vector (default 0).
lookat_y リアル The y look vector (default 0).
lookat_z リアル The z look vector (default 1).
up_x リアル The x up vector (default 0).
up_y リアル The y up vector (default 1).
up_z リアル The z up vector (default 0).

リターンです。

N/A

例

dx = dcos(direction);
dy = - dsin(direction);
dz = - dsin(zdirection);
audio_listener_position(x, y, z);
audio_listener_orientation(dx, dy, dz, 0, 0, 1);

上記のコードでは、3Dオーディオリスナーの位置を、インスタンスのx、y変数と追加のz変数に設定します。次に、オーディオリスナーの向きを、インスタンスの directionと zdirection 変数に設定します。
