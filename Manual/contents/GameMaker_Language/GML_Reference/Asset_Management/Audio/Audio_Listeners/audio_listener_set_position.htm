audio_listener_set_position

この関数を使うと、3Dオーディオ空間内の指定されたリスナーの位置を変更することができます。デフォルトのリスナーインデックスは 0 ですが、関数 audio_get_listener_info()を使用すると、ターゲットプラットフォームで利用可能なさまざまなインデックスを取得できます。下の画像は、オーディオ空間におけるリスナーのデフォルトの位置を示しています。

ご覧の通り、デフォルトの位置は (0, 0, 0) ですが、通常はこの関数を使ってリスナーをゲーム内のプレイヤーobject と共に移動させ、エミッターによって作られたオーディオがプレイヤーに聞こえる方法を変更します。例えば、下の画像のトップダウンゲームでは、プレイヤーインスタンスがリスナーを設定し、プレイヤーがレベルを移動すると、様々なエミッターからのオーディオが「変化」するようになっています。

構文です。

audio_listener_set_position(index, x, y, z);

論証 タイプ 商品説明
index リアルまたはオーディオリスナーID The listener to set the position of (default 0).
x リアル The x position of the listener (default 0).
y リアル The y position of the listener (default 0).
z リアル The z position of the listener (default 0).

リターンです。

N/A

例

var _vmat = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _vmat[0], _vmat[1], _vmat[2]);
audio_listener_set_orientation(info[? "index"], _vmat[3], _vmat[4], _vmat[5], _vmat[6], _vmat[7], _vmat[8]);

上記のコードは、カメラビュー[0]のビューマトリックスを取得し、それを使ってグローバル変数 "Player_Listener "に格納されているIDのリスナーのオーディオリスナーの位置と向きを設定しています。
