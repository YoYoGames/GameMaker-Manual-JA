オーディオセット・マスターゲイン

この関数を使用すると、特定のリスナーに対して、すべてのサウンドと音楽のグローバルボリュームの絶対値を設定することができます。デフォルトのリスナーインデックスは0ですが、関数 audio_get_listener_info() を使って、ターゲットプラットフォームで利用可能なさまざまなインデックスを取得することができます。ゲイン値は0（無音）から1（フルボリューム）までのリニアスケールに基づいており、ゲーム内からそのリスナーを通じて再生される全てのサウンドや音楽の相対的なボリュームに影響を与えます。

構文です。

audio_set_master_gain(listenerIndex, gain);

論証 タイプ 商品説明
listenerIndex リアルまたはオーディオリスナーID The index of the listener to set the gain on.
gain リアル Value for the global volume (0 to 1).

リターンです。

N/A

例

var num = audio_get_listener_count();
for( var i = 0; i < num; i++;)
{
    var info = audio_get_listener_info(i);
    audio_set_master_gain(info[? "index"], 0.75);
    ds_map_destroy(info);
}

上記のコードは、loops 利用可能なリスナーを通して、それらのマスターゲインを0.75に設定します。
