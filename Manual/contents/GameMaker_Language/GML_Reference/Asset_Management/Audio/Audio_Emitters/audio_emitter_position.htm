オーディオエミッターポジション

この機能を使うと、3Dオーディオ空間内のオーディオエミッタの位置を変更することができます。この位置は、リスナーがオーディオ空間内のどこに位置しているかによって、音にさまざまな影響を与えます（デフォルトの位置は（0、0、0）です）。詳細は audio_listener_position()例えば、エミッタの位置が（100, 0, 0）で、現在のリスナーが（200, 0, 0）にいる場合、エミッタから流れる音声は、オーディオフィールドの左側にあるように見えるでしょう。下図は、エミッタとリスナーとの相対的な位置関係を視覚的に表したものです。

構文です。

audio_emitter_position(emitter, x, y, z);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to change.
x リアル The x position.
y リアル The y position.
z リアル The z position.

リターンです。

N/A

例

if speed > 0
{
    audio_emitter_position(s_emit, x, y, 0);
}

上記のコードは、インスタンスの速度が0を超えているかどうかをチェックし、0を超えていれば、変数 "s_emit "でインデックスされたオーディオエミッターを現在のx/y位置に更新しています。
