audio_emitter_get_listener_mask

この関数は、エミッタがどのオーディオリスナーから音を再生すべきかを定義するビットマスクデータを返します。詳細はオーディオリスナーのセクションを参照してください。

構文です。

audio_emitter_get_listener_mask(emitterID);

論証 タイプ 商品説明
emitterID オーディオエミッターID The unique ID of the emitter to get the mask of

リターンです。

Real

例

s_emit = audio_emitter_create();
if (audio_emitter_get_listener_mask(s_emit) != global.PlayerMask)
{
    audio_emitter_set_listener_mask(snd, global.PlayerMask);
}

上記のコードでは、エミッタを生成した後、リスナーのマスクデータをチェックし、それがグローバル変数に格納されているものと同じでない場合、グローバル変数に格納されているマスクデータを使用して再生するリスナーを設定します。
