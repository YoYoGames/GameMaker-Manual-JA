オーディオエミッターのピッチを取得する

説明： 指定されたオーディオエミッターに設定されている現在のピッチ値を返す。

構文です。

audio_emitter_get_pitch(emitter);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to use.

リターンです。

Real

例

if audio_emitter_get_pitch(emitter_player) != 1
{
    audio_emitter_pitch(emitter_player, 1);
}

上記のコードは、与えられたエミッタの現在のピッチをチェックし、それが1に等しくない場合、1に設定されます。
