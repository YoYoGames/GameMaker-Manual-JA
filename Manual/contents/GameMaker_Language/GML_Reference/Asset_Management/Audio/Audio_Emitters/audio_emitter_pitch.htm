オーディオエミッターピッチ

この関数は、指定されたエミッタから発せられるすべての音のピッチを変更するために使用することができます。ピッチの乗算器であり、入力値は現在のピッチにその量を乗算します。したがって、デフォルトの1ではピッチの変化はなく、1より小さい値ではピッチは下がり、1より大きい値ではピッチは上がります。この機能では、0以下や5以上の値はいずれにせよ聞き取れない可能性があるため、小さな増分を使用するのが最善です。

ピッチの変更を聞くために、すでにオーディオエミッタで再生されている音を再起動する必要はありません。ピッチの変更は即座に適用されます。

構文です。

audio_emitter_pitch(emitter, pitch);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to change.
pitch リアル The pitch multiplier (default 1).

リターンです。

N/A

例

switch (gear)
{
    case 1: audio_emitter_pitch(s_emit, 0.8); break;
    case 2: audio_emitter_pitch(s_emit, 0.9); break;
    case 3: audio_emitter_pitch(s_emit, 0.95); break;
    case 4: audio_emitter_pitch(s_emit, 1); break;
    case 5: audio_emitter_pitch(s_emit, 1.2); break;
}

上記のコードは、変数「s_emit」でインデックスされたエミッタから再生される音声のピッチを、変数「gear」の値に基づいて変更するものです。
