オーディオ・エミッター・フリー

この関数を使うと、指定されたエミッタをメモリから削除することができます。この操作は、room やゲーム内でエミッタが使われなくなった時、つまりインスタンスのDestroy イベントやRoom End イベントで行うべきです。そうしないと、メモリリークが発生してゲームが遅くなり、最終的にはクラッシュする可能性があります。

構文です。

audio_emitter_free(emitter);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to free.

リターンです。

N/A

例

if lives = 0
{
    audio_emitter_free(s_emit);
    room_goto(rm_Menu);
}

上記のコードは、グローバル変数 "lives "の値をチェックし、0を返したら、変数 "s_emit "にインデックスされているエミッタを破壊し、変数 "rm_Menu "にインデックスされているroom に移動します。
