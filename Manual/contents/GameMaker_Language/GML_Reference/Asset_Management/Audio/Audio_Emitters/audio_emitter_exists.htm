audio_emitter_exists

この関数は、オーディオエミッタが存在するか (true) 、存在しないか (false) を返します。検索するインデックスが以前に初期化されていない場合、この関数は存在しないインデックスを検索しているため、エラーになることに注意してください。

構文です。

audio_emitter_exists(index);

論証 タイプ 商品説明
index オーディオエミッターID The index of the emitter to check the existence of.

リターンです。

Boolean

例

if audio_emitter_exists(s_emit)
{
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}
else
{
    s_emit = audio_emitter_create();
    audio_play_sound_on(s_emit, snd_Explode, false, 1);
}

上記のコードでは、変数 "s_emit "のインデックスを持つエミッタが存在するかどうかをチェックしています（以前に初期化済み）。存在する場合は音を鳴らし、存在しない場合はエミッタを生成して音を鳴らす。
