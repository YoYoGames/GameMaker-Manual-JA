オーディオエミッターゲイン

音の最大ゲイン（音量）を設定する機能です。音の音量は、フォールオフ値や リスナーとの位置関係によって変化しますが、この関数でゲインを設定することで、全音量が指定したゲイン値を超えることはありません。下の画像は、フォールオフが0より大きい場合、ゲインがエミッタの音量にどのように影響するかを示しています。

この関数は、再生中の音と、指定したエミッタで再生される後続のすべての音の音量を変更します。プラットフォームによっては、ゲインを1より大きくすることができますが、1という値は「フルボリューム」とみなされ、それ以上の値はオーディオのクリッピングや歪みを引き起こす可能性があることに注意してください。

注 最終的な音量は、機能で設定されたグローバルオーディオゲインの影響も受けます。 audio_master_gain().

構文です。

audio_emitter_gain(emitter, gain);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to change.
gain リアル The maximum gain (default 1).

リターンです。

N/A

例

if up
{
    gain += 0.05;
    if gain >= 1 up = false;
}
else
{
    gain += 0.05;
    if gain <= 0 up = true;
}

audio_emitter_gain(s_emit, gain);

上記のコードでは、変数「gain」に異なる値を設定し、その同じ変数を使って変数「s_emit」でインデックスされたエミッタのゲインを設定します。
