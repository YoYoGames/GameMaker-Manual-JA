オーディオ・エミッター・ゲット・ワイ

この関数は、与えられた音声エミッタの現在の y 位置を返す。

構文です。

audio_emitter_get_y(emitter);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to use.

リターンです。

Real

例

if audio_emitter_get_y(emitter_player) != x
{
    audio_emitter_position(emitter_player, x, y, 0);
}

上記のコードでは、与えられたエミッタの現在のy位置をチェックし、それがインスタンスのy位置と等しくない場合、インスタンスの位置に設定されます。
