オーディオエミッターベロシティ

この関数は、エミッタにドップラー 効果を与え、与えられた相対的なx、y、z位置から分解されるベクトルに基づいて、オーディオの動きをシミュレートするために使用することができます。しかし、例えば、敵が右からやってきて左へスクロールするようなスクロールシューティングゲームを作る場合、敵のcreateイベントでこの値を(hspeed, 0, 0)に設定します（そしてstepイベントで各エミッターを audio_emitter_position()敵のインスタンスから発せられる音がプレイヤーのインスタンスを通過するときに正しいドップラーを与えるためです（プレイヤーは、関数 audio_listener_position()を使用して、リスナーを正しい位置に設定します)。

構文です。

audio_emitter_velocity(emitter, vx, vy, vz);

論証 タイプ 商品説明
emitter オーディオエミッターID The index of the emitter to change.
vx リアル The x vector value (default 0).
vy リアル The y vector value (default 0).
vz リアル The z vector value (default 0).

リターンです。

N/A

例

s_emit = audio_emitter_create();
audio_emitter_position(s_emit, room_width, 0, 0);
audio_emitter_velocity(s_emit, -5, 0, 0);

上記のコードでは、オーディオエミッターを作成し、そのインデックスを変数 "s_emit "に代入しています。このエミッターは、room 内のある位置に置かれ、x軸に沿って1ステップあたり5ピクセルの速度が与えられます（リスナーに対して、1ステップあたり5ピクセルの水平速度を持っているかのように正しくドップラー効果が得られます）。
