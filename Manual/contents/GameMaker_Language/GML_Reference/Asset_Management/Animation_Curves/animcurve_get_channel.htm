animcurve_get_channel

animation curve asset で指定されたチャンネルのデータを含む構造体を返す。animation curve ID またはstruct と，チャンネル名またはインデックスを指定すると，以下のフォーマットでstruct を返す。

変数名 データ型 商品説明
name 列 これはチャンネルの名前です。
type アニメーションカーブ補間タイプ定数 これは、ポイント間の線形補間のための定数 animcurvetype_linear 、キャットマル・ロム補間を用いたポイント間の「滑らかな」補間のための定数 animcurvetype_catmullrom 、ベジェ補間のための定数 animcurvetype_bezier のいずれかになります。
iterations 整数 このチャンネルが catmull-rom ("smooth") 補間を使用している場合、曲線の各セグメントに対して生成されたポイント数を保持します（これらの余分なポイントは関数内部で使用され、runtime の計算にのみ使用されることに注意してください）。チャンネルが線形補間を使用している場合、この値はまだ存在しますが、カーブの補間方法には関係しないため、無視することができます。
points 配列ポインタ これは配列であり、配列の各項目は点であるstruct 。


animation curve ID は、animation curve Browser で定義されたAsset の名前、または、関数が返すstruct ポインタの名前です。 animcurve_create().チャンネル名は、アニメーションカーブstring で定義されたチャンネルを参照するasset 、またはインデックス値を指定できます。インデックス値は 0 からn で、nはカーブの最後のチャンネルですasset （例：animation curve に 4 つのチャンネルがある場合、0 から 3 のインデックスになります）。インデックス値を渡すと、チャンネル名を渡すよりも少ない処理で済むことに注意してください。この関数が失敗した場合 (つまり、指定した名前あるいはインデックスを持つチャンネルが存在しなかった場合) には、エラーが発生します。

1つのチャンネルの点は，structs として配列に格納される．配列の各項目は，1つの点であるstruct である．点struct は以下の変数を持つ。

変数名 データ型 商品説明
posx じっさいにある 点の時間的位置（0から1まで正規化）。
value じっさいにある ポイントの値です。

構文です。

animcurve_get_channel(curve_struct_or_id, channel_name_or_index);

論証 タイプ 商品説明
curve_struct_or_id アニメーションカーブ構造 The ID or struct pointer of the animation curve to target
channel_name_or_index ストリングまたは リアル The channel name (a string) or the channel index (an integer).

リターンです。

Animation Curve Channel Struct

例

var _channeldata = animcurve_get_channel(ac_ButtonTween, 0);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

上記のコードは、カーブstruct "ac_ButtonTween" のチャンネル 0 のチャンネルasset を取得し、チャンネルカーブのポイントをloops 経由して、その値に 1 を加算しています。
