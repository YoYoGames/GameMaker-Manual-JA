animcurve_point_new

この関数は、animation curve チャンネルに追加する新しいポイント構造体を作成するために使用することができます。ポイントstruct は作成時、空であり、ポイントデータを生成するために以下の変数を設定する必要があります。

アニメーションカーブポイント構造
変数名 データ型 商品説明
posx じっさいにある 点の時間的位置（0から1まで正規化）。
value じっさいにある ポイントの値です。

作成した各ポイントstruct は配列に追加する必要があり、この配列はanimation curve チャンネルstruct 「チャンネル」変数に追加されます（以下の例のように）。

構文です。

animcurve_point_new();

リターンです。

Animation Curve Point Struct

例

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上記のコードでは、新しいanimation curve struct を作成し、変数 "my_curve" に格納しています。このstruct には、名前とチャンネル配列が入力される。チャンネル配列には、3 つのポイントを持つ 1 つのチャンネルが含まれます。
