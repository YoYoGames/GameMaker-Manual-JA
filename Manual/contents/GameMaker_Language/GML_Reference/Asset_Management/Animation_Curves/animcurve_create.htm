アニメーションカーブクリエイト

この関数は、空のanimation curve 構造体を作成し、チャンネルデータを入力する準備ができます。この関数は，struct へのポインタを返し，これを用いてanimation curve にチャンネルやその他のデータを追加します． animcurve_get()を使用して、後でカーブからデータを取得することができます。

この関数を使用する場合、作成されるstruct には何もデータが入っていません。使用するには、少なくとも1つのチャンネルを追加し、そのチャンネルにポイントが追加されている必要があります。チャンネルを追加するには，関数 animcurve_channel_new()また、チャンネルにポイントを追加するには、関数 animcurve_point_new().さらに、変数「name」を設定することで、animation curve struct に名前を付けることができます（以下のコード例に示すように）。新しいanimation curve に追加するチャンネルは配列として追加し、配列の各項目はチャンネルを参照しますstruct 、チャンネル内のポイントも配列として追加し、配列の各項目はポイントを参照しますstruct 。

このようにして作成されたanimation curves は、不要になったら削除しなければならないことに注意してください。そうしないと、メモリ上のスペースを占有し、メモリリークが発生し、ゲームの動作が遅くなり、最終的にはクラッシュする可能性があるからです。不要になったanimation curve を削除するには、次の関数を使用します。 animcurve_destroy().カーブ内のチャンネルやポイントデータは、カーブ自体が破棄されるときにガベージコレクタによって自動的に処理されるため、クリーンアップする必要はありません。

構文です。

animcurve_create();

リターンです。

Animation Curve Struct

例

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上記のコードでは、新しいanimation curve struct を作成し、変数 "my_curve" に格納しています。このstruct には、名前とチャンネル配列が入力される。チャンネル配列には、3 つのポイントを持つ 1 つのチャンネルが含まれます。
