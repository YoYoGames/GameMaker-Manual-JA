animcurve_get

この関数は、与えられたanimation curve の全データを含む構造体を返す。animation curve asset ID (Asset Browser で定義されているもの) を与えると、この関数は以下の変数を持つstruct を返します。

アニメーションカーブ構造
変数名 データ型 商品説明
name 文字列 これは、animation curve の名前です。
channels アレイの アニメーションカーブチャンネル構造s これは配列であり、配列の各項目はチャンネルであるstruct 。


struct の変数 channels は配列であり、配列の各項目は アニメーションカーブチャンネル構造カーブ内のチャンネルに関連するデータで構成される。チャンネルstruct については、このページで説明しています。

チャンネルと同様に、1つのチャンネルのポイントは、配列にstructs として格納されます。配列の各項目は、1つのポイントstruct です。点struct については、このページで説明しています。

なお、この関数が失敗した場合（例えば、指定したアニメーションカーブasset が存在しない場合）には、-1 が返されることに注意してください。

構文です。

animcurve_get(curve_id);

論証 タイプ 商品説明
curve_id アニメーションカーブアセット The asset browser ID (index) of the animation curve to get.

リターンです。

Animation Curve Struct or -1

例

var _curve = animcurve_get(ac_ButtonTween);
var _channel = _curve.channels[0];
if (_channel.type != animcurvetype_linear)
{
    _channel.type = animcurvetype_linear;
}

上記のコードでは、カーブanimation curve "ac_ButtonTween" のstruct asset を取得し、最初のチャンネルのカーブタイプをチェックし、それが線形でない場合は線形に設定します。
