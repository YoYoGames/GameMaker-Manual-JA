animcurve_channel_new

この関数は、animation curve チャンネル構造体を新規に作成するために使用することができます。チャネルstruct は、以下の変数を持つ。

アニメーションカーブチャンネル構造
変数名 データ型 商品説明
名前 列 これはチャンネルの名前です。
タイプ アニメーションカーブ補間タイプ定数 これは、以下の表で示される定数のいずれかになります。
反復 整数 このチャンネルが catmull-rom ("smooth") 補間を使用している場合、曲線の各セグメントに対して生成されたポイント数を保持します（これらの余分なポイントは関数内部で使用され、runtime の計算にのみ使用されることに注意してください）。チャンネルが線形補間を使用している場合、この値はまだ存在しますが、カーブの補間方法には関係しないため、無視することができます。
点 勢揃い これは配列であり、配列の各項目は アニメーションカーブポイント構造.

アニメーションカーブ補間タイプ定数
定数 商品説明
animcurvetype_linear 点間の線形補間に使用する。
animcurvetype_catmullrom キャットマル・ロム補間により、点間を滑らかに補間するために使用します。
animcurvetype_bezier 点間のベジェ補間に使用する。


新しいチャンネルを作成するときのデフォルトでは、struct 、"name "変数が空のstring 、"type "が animcurvetype_linear 、"iterations" が16になります。これらの変数はすべて、必要な値に設定することができます（「iterations」の値は、線形カーブタイプには効果がないことに注意してください）。一度作成したら、曲線にポイントを追加する必要があります。これは、異なるポイントstructs を「ポイント」配列に追加することで行われます。これらの点structs は、関数 animcurve_point_new().

構文です。

animcurve_channel_new();

リターンです。

Animation Curve Channel Struct

例

my_curve = animcurve_create();
my_curve.name = "My_Curve";
var _channels = array_create(1);
_channels[0] = animcurve_channel_new();
_channels[0].name = "alpha";
_channels[0].type = animcurvetype_catmullrom;
_channels[0].iterations = 8;
var _points = array_create(3);
_points[0] = animcurve_point_new();
_points[0].posx = 0;
_points[0].value = 0;
_points[1] = animcurve_point_new();
_points[1].posx = 0.5;
_points[1].value = 1;
_points[2] = animcurve_point_new();
_points[2].posx = 1;
_points[2].value = 0;
_channels[0].points = _points;
my_curve.channels = _channels;

上記のコードでは、新しいanimation curve struct を作成し、変数 "my_curve" に格納しています。このstruct には、名前とチャンネル配列が入力される。チャンネル配列には、3 つのポイントを持つ 1 つのチャンネルが含まれます。
