アニカーブデストロイ

この関数は、以前に関数で作成されたanimation curves を破棄するために使用できます。 animcurve_create().この関数を呼び出すと、animation curve がメモリから削除され、それに含まれるチャネルやポイントもクリーンアップされます（これらは決して手動でクリーンアップする必要はありません）。 animation curves ブラウザで作成されたasset は破壊できません。runtime で作成されたものだけであることに注意してください。

構文です。

animcurve_destroy(curve_struct);

論証 タイプ 商品説明
curve_struct アニメーションカーブ構造 The pointer to the curve struct to destroy.

リターンです。

N/A

例

animcurve_destroy(my_curve);

上記のコードは、変数 "my_curve" にインデックスされた（以前に作成された）animation curve struct を破棄します。
