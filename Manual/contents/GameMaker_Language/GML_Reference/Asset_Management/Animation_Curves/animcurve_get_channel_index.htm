animcurve_get_channel_index

この関数は、任意の与えられたanimation curve チャンネルのインデックス値を返します。animation curve ID またはstruct 、カーブ ID はanimation curve Browser で定義されたAsset の名前、または関数 animcurve_create() で返されたstruct ポインタを指定します。次に、チャンネルの名前をstring として指定すると、関数はそのチャンネルに関連するインデックス値を返します。なお，曲線やチャンネルが存在しない場合は，エラーになります．

構文です。

animcurve_get_channel_index(curve_struct_or_id, channel_name);

論証 タイプ 商品説明
curve_struct_or_id アニメーションカーブ構造 The ID or struct pointer of the animation curve to target
channel_name 文字列 The channel name (a string).

リターンです。

Real

例

var _channelindex = animcurve_get_channel_index(ac_ButtonTween, "x_pos")
var _channeldata = animcurve_get_channel(ac_ButtonTween, _channelindex);
var _points = _channeldata.points;
for (var i = 0; i < array_length(_points); ++i;)
{
    _points[i].value += 1;
}

上記のコードは、カーブstruct "ac_ButtonTween" の "x_pos" という名前のチャンネルasset を取得し、チャンネルカーブのポイントをloops 通して、その値に 1 を加算しています。
