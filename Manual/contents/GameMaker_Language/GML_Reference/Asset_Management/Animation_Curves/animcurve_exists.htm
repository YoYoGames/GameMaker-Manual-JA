アニメーションカーブの存在

この関数は、与えられた アニメーションカーブアセットまたは アニメーションカーブ構造が存在し、かつ有効なアニメーションカーブである場合、falseとなる。

構文です。

animcurve_exists(curve_struct_or_id);

論証 タイプ 商品説明
curve_struct_or_id アニメーションカーブ構造または アニメーションカーブアセット The Animation Curve that will be checked.

リターンです。

Boolean

例

if (animcurve_exists(spring_curve))
{
    apply_spring_animation(sprite_curve);
}

上記のコードでは、カスタム変数spring_curveに格納されているアニメーションカーブが存在するかどうかをチェックしています。存在する場合は、そのアニメーションカーブを使用するカスタムメソッドを実行します。
