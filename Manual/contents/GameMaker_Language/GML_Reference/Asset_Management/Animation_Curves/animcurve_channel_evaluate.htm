animcurve_channel_evaluate

この関数は、あるチャンネル構造体から特定の時点の値を取得するために使用します。チャンネルのstruct ポインタ（この関数が返すもの，あるいは，この関数から で返されるもの）を指定します． animcurve_get_channel()または，この関数からanimation curve struct で返される値）を指定します． animcurve_get()) と，評価する曲線チャンネルに沿った "x" 位置 (時間) を指定します．この位置は 0 から 1 の間でなければならず，この関数はその位置の曲線値を返します．また，与えられたチャンネルstruct が無効な場合は 0 を返します．

構文です。

animcurve_channel_evaluate(channel_struct, posx);

論証 タイプ 商品説明
channel_struct アニメーションカーブチャンネル構造 The struct pointer for the channel to evaluate.
posx リアル The position in time to check (from 0 to 1).

リターンです。

Real

例

var _channel = animcurve_get_channel(ac_AlphaCurve, 0);
var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000));
image_alpha = _val;

上記のコードでは、struct animation curve "ac_AlphaCurve" のチャンネルasset を取得しています。そして、返された評価値を使って、インスタンスの画像アルファを設定します。
