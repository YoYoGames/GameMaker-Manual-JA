スプライトセーブ

この機能は、sprite の任意の部分画像を、指定されたファイル名を付けてディスクに保存するために使用します。sprite はruntime で追加されている必要があり (sprites を通して追加されたIDE は保存できません)、ファイルは *.png extension で保存される必要があります。

注：選択されたターゲットプラットフォームによっては、ファイルの保存先や読み込み先に制限があります。詳細は、「ファイル操作」を参照してください。

構文です。

sprite_save(ind, subimg, fname);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to save.
subimg リアル The sub-image of the sprite to save.
fname 文字列 The filename for the saved sprite.

リターンズ

N/A

例

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

上記のコードでは、サーフェイスを作成し、そのインデックスをローカル変数「surf」に格納しています。次に、そのサーフェスをターゲットにしてクリアし、sprites をいくつか重ねて描画する。サーフェスに描かれた合成画像から新しいsprite を作成し、そのインデックスをローカル変数 "spr_Custom" に代入してから、サーフェスが使用していたメモリを解放する。最後に、出来上がったsprite をファイルに保存し、メモリからも削除する。
