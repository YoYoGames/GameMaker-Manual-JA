スプライトアドフロムサーフェイス

と全く同じ働きをします。 sprite_create_from_surface()は、選択したインデックス付きサーフェスの領域から新しいsprite を作成する代わりに、サーフェスの定義された領域を新しいサブイメージとして、以前に作成したsprite に追加します（これは、リソースツリーのsprite に直接追加することはできず、サーフェスから以前に作成したものか、リソースから複製されたsprite でしか追加できないことを意味します）。 sprite_duplicate()).

この方法でsprite を追加する場合、使用するメモリが予想以上に大きくなることに注意する必要があります。これは、GameMakerがsprite をtexture のページとして保存し、さらに GPU メモリにも保存するため、同じsprite の画像ファイルよりも総メモリ量が大きくなるためです。

構文です。

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

論証 タイプ 商品説明
index スプライトアセット The index of the sprite to add the new image to.
surface サーフェスID The index of the surface from which we get the image.
x リアル The x position to copy from.
y リアル The y position to copy from.
w リアル The width of the area to be copied (from the x position).
h リアル The height of the area to be copied (from the y position).
removeback ブーリアン Indicates whether to make all pixels with the background colour (left-bottom pixel) transparent.
smooth ブーリアン Indicates whether to smooth the edges.

リターンです。

N/A

例

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i += 1)
{
    sprite_add_from_surface(spr_custom, surf, i, 0, 32, 32, true, true);
}

上記のコードでは、変数 "surf "でインデックスされた表面からsprite を作成し、そのインデックスを変数 "spr_Custom "に割り当てます。次に for loop を使って表面上を移動し、さまざまな部分をキャプチャして、サブ画像としてsprite に追加しています。
