スプライトの存在

この関数は、実行中のプロジェクトに指定されたインデックスのsprite が存在するかどうかを返します。

構文です。

sprite_exists(index);

論証タイプ 商品説明
index The index of the sprite to check.

リターンズ

例

if sprite_exists(spr_array[0])
{
    sprite_index = spr_array[0];
}
else
{
    sprite_index = spr_BaseSprite;
}

上記のコードでは、配列に有効なsprite のインデックスが含まれているかどうかをチェックし、含まれていればそのsprite をインスタンスに割り当てますが、含まれていなければ含まれているリソースからsprite を割り当てています。
