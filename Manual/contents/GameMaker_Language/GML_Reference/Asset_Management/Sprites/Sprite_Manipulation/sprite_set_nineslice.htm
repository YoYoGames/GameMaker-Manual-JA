sprite_set_nineslice

この関数は、ナインスライス構造体をsprite に適用するために使用します。適用するために、sprite のインデックスとナインスライスstruct を指定します。 sprite_nineslice_create()または sprite_get_nineslice().

注意事項 この関数はスプライトアセットに影響を与えるので、この関数で変更した内容は、このsprite を使用しているすべてのインスタンスと、将来のすべてのインスタンスに影響します。

構文です。

sprite_set_nineslice(ind, nineslice);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to modify
nineslice ナインスライスストラクチャー The Nine Slice struct to apply

リターンです。

N/A

例

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上記のコードでは、新しいNine Slicestruct を作成し、それを有効にして、ガイドのオフセットを設定しています。次に、struct をsprite に適用し、そのナインスライスプロパティを変更します。
