sprite_set_cache_size_ext

画像のブレンドを行う場合、HTML5では実行ファイルと同じように動的に行うことができません。そのため、GameMakerはブレンドした画像のコピーを一時的に保存し、それを読み込む必要がある。この関数は、与えられたsprite のブレンドコピーを、古いものが上書きされる前に何枚キャッシュできるかを設定する。デフォルトは4です。これは、sprite の与えられた1つの部分画像に適用されます。

構文です。

sprite_set_cache_size_ext(ind, index, max);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to change the cache size of.
index リアル The sub-image of ind to change the cache size of.
max リアル The maximum number of cached copies of the sprite that can be stored.

リターンズ

N/A

例

sprite_set_cache_size_ext(sprite0, 0, 2);

これにより、sprite0 の最初のサブイメージのsprite cache が2コピーに設定されます。
