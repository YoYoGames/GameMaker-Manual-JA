スプライトデリート

この関数は、ゲームからsprite を削除し、そのために予約されていたメモリをすべて解放します。これは、外部からsprites を作成したり変更したりする必要がある場合（たとえば、sprite をファイルから読み込む場合など）に、非常に重要な関数です。 sprite_add() sprite sprite_duplicate() また、不要になったassets を削除するために常に使用する必要があります。 ゲームに必要な場合、またはインデックスされたasset を変数から消去してから、その変数に別のasset を再割り当てする場合です。これは、sprites ブラウザ内のassets の一部としてゲームに含まれているAsset を削除するために使用することはできません。

構文です。

sprite_delete(index);

論証タイプ 商品説明
index The index of the sprite to be deleted.

リターンズ

例

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

上記のコードでは、ローカル変数を作成し、アプリケーションサーフェスから作成されたsprite のインデックスを格納しています。このsprite は、変数 "spr_Player" のインデックスを持つasset と結合された後、再びメモリから削除されます。
