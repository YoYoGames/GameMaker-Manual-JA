sprite_save_strip

この関数は、sprite のすべてのサブ画像からストリップ画像を作成し、指定されたファイル名でディスクに保存します。sprite はruntime で追加されている必要があり (sprites を通して追加されたIDE は保存できません)、ファイルは *.png extension で保存されている必要があります。

注：選択されたターゲットプラットフォームによっては、ファイルの保存先や読み込み先に制限があります。詳細は、「ファイル操作」を参照してください。

構文です。

sprite_save_strip(ind, filename);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to save.
filename 文字列 The filename for the saved sprite strip.

リターンズ

N/A

例

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);

for (i = 1; i < 8; i +=1)
{
    sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}

sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

上記のコードは、変数「surf」でインデックスされた表面からsprite を作成し、そのインデックスをローカル変数「spr_Custom」に代入します。次に、 for loop を使用して表面を移動し、サブ画像としてsprite に追加されるさまざまな部分をキャプチャします。この新しいsprite はメモリから削除する前に png の帯として保存されます。
