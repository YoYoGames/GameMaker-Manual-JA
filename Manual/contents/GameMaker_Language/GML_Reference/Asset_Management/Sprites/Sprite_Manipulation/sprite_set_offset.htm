スプライトセットオフセット

この関数は、sprite の x と y の原点を設定するために使用でき、(0,0) の位置をsprite の左上隅とした相対値を取ります。 次の画像は、これを示しています。

注意この関数はスプライトアセットに 影響を与え、今後このsprite を持つすべてのインスタンスは同じオフセットを持つようになります。

構文です。

sprite_set_offset(ind, xoff, yoff);

論証タイプ 商品説明
ind The index of the sprite to change the offset of.
xoff The x position of the origin.
yoff The y position of the origin.

リターンズ

例

sprite_assign(spr_Custom, spr_Base);
sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base));

上記のコードでは、"spr_Base "にインデックスされたsprite を "spr_Custom "にインデックスされたsprite に割り当て、"spr_Base "のxおよびyオフセット値を使用して新しいスプライトの原点を設定します。
