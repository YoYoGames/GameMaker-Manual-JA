sprite_nineslice_create

ナインスライシングとは、矩形の画像を細部を保持するためにスケーリングする際に使用する手法で、スケーリング後も元の形状を保持させることができます。ナインスライシングについては、こちらのページをご覧ください。

この機能は、Nine Slice構造体を作成し、それを修正した上で、sprite に適用するために使用します。 このstruct の内容については、このページで詳しく説明します。

このstruct を変数に格納し、そのプロパティを変更し、 sprite_set_nineslice() を使って任意のsprites に適用することができます。

構文です。

sprite_nineslice_create();

リターンです。

Nine Slice Struct

例

var _nineslice = sprite_nineslice_create();

_nineslice.enabled = true;
_nineslice.left = 10;
_nineslice.right = 10;
_nineslice.top = 10;
_nineslice.bottom = 10;

sprite_set_nineslice(spr_box_0, _nineslice);

上記のコードでは、新しいNine Slicestruct を作成し、それを有効にして、ガイドのオフセットを設定しています。次に、struct をsprite に適用し、そのナインスライスプロパティを変更します。
