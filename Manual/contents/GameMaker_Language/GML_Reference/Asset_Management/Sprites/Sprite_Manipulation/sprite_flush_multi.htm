sprite_flush_multi

この関数を使うと、与えられたtexture の任意の数のsprites ページをtexture メモリから削除することができます。sprite のインデックスを配列として与えると、それらが割り当てられているtexture ページがtexture メモリから削除されます。もし、そのうちの1つが texture ページがroom の他の場所 (別のインスタンスsprite やバックグラウンドなど) で使用されている場合、そのページがすぐにメモリに再ロードされるため、若干のパフォーマンス低下が生じる可能性がありますので、この関数を使用する場合は注意が必要です。 この関数は、選択したリソースでフラッシュがサポートされていない場合は -1 を返し、すべてが正しく動作している場合は 0 を返すことに注意してください。

構文です。

sprite_flush_multi(array);

論証タイプ 商品説明
array Array with the sprite indices to flush

リターンです。

(-1 or 0)

例

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

上記のコードでは、各要素がsprite のインデックスを保持する配列を作成しています。この配列を使って、メモリからsprite textures を消去します。
