スプライトセットスピード

この関数は、sprite リソースの基本アニメーションの速度と種類を設定するために使用します。これは通常スプライトエディタで設定されますが、与えられたsprite リソースに対して手動で設定する必要がある場合があります。設定するsprite インデックス、アニメーション速度、適用するアニメーション速度の種類を指定する必要があります。アニメーションスピードの種類には、以下の定数のいずれかを指定します。

定数 商品説明
spritespeed_framespersecond sprite 、1秒あたりのフレーム数でアニメーションを定義した。
spritespeed_framespergameframe sprite は、ゲーム1フレームあたりのアニメーションで定義しました。


アニメーションの計算に使用する速度の種類を選択することは、2つのオプションの間に大きな違いがあるため、非常に重要です。次の2つの例は、この違いを説明するためのものです。

  • 1秒間に1フレーム描画するsprite 、画像速度を0.5に設定すると、1秒間に0.5フレームで描画されることになります。ゲームのフレームレートが 60 フレーム/秒の場合、sprite は 120 ゲームフレームごとに 1 フレーム描画することになります。
  • ゲームフレームごとに1フレームを描画するsprite 、画像速度を0.5に設定すると、ゲームフレームごとに0.5フレームを描画することになります。ゲームフレームレートが60フレーム/秒の場合、sprite は60ゲームフレームごとに30フレームを描画することになります。

sprite のアニメーションに使用された速度値が何であったかは、関数 sprite_get_speed()を使用すると、アニメーションの速度タイプを取得することができます。 sprite_get_speed_type().

重要!この関数はsprite リソースに影響を与えるので、この関数で行った変更はsprite を使用しているすべてのインスタンスと、将来のすべてのインスタンスにも影響を与えます。

構文です。

sprite_set_speed(index, speed, type)

論証 タイプ 商品説明
index スプライトアセット The index of the sprite to set the animation speed of
speed リアル The speed value to set
type スプライト速度定数 The speed type to use

リターンです。

Constant (see above)

例

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

上記のコードでは、インスタンスに割り当てられたsprite の基本アニメーション速度を 10 フレーム/秒に設定しています。
