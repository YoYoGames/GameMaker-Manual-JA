スプライトアサイン

この関数は、以前に作成された（またはインクルードされた）2つのsprite インデックスを受け取り、一方からもう一方に画像をコピーします。このようにして、あるsprite を別のインデックスにコピーする (あるいは「クローン」する) ことができます。ただし、ゲームリソースにコピーすることはできません。コピー先のsprite は sprite_add()または sprite_duplicate()関数を使用します。

構文です。

sprite_assign(index, sprite);

論証タイプ 商品説明
index The index of the sprite to be copied to (ie: overwritten).
sprite The sprite to be copied from.

リターンズ

例

if sprite_exists(global.Player_Sprite)
{
    var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
    sprite_assign(global.Player_Sprite, t_spr);
    sprite_delete(t_spr);
}
else
{
    global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

上記のコードでは、グローバル変数「Player_Sprite」にsprite があるかどうかをチェックし、もしあれば sprite_assign を使って外部ファイルからロードされたものに変更します。もし含まれていなければ、sprite がロードされ、そのインデックスが変数に保存される。
