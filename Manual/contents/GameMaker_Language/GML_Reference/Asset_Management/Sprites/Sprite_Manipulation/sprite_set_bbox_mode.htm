sprite_set_bbox_mode

この関数は、sprite のバウンディングボックスモードを設定するために使用します。sprite のインデックスと使用するモードを指定しますが、これは以下の定数のいずれかでなければなりません。

バウンディングボックスモード定数
定数 商品説明
bboxmode_automatic 自動 - スプライトのトレランス設定に基づき、バウンディングボックスが自動的に計算されます。
bboxmode_fullimage Full Image- 許容範囲や「空の」ピクセルに関係なく、バウンディングボックスがsprite の幅と高さを完全に使用するように設定されます。
bboxmode_manual 手動- ユーザー定義の値にバウンディングボックスを手動で設定するため、関数 sprite_set_bbox()

注：この関数はスプライトアセットに 影響し、今後このsprite を持つすべてのインスタンスが同じバウンディングボックスモードになるようにします。

構文です。

sprite_set_bbox_mode(ind, mode);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to change the mode of.
mode バウンディングボックスモード定数 The mode to set (a constant).

リターンズ

N/A

例

if sprite_get_bbox_mode(sprite_index) != bboxmode_automatic
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

上記のコードは、現在のsprite の bbox モードをチェックし、それが automatic でないなら、その値に設定されます。
