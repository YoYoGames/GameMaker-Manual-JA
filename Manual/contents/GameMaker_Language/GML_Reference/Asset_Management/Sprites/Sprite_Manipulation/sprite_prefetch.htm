スプライトプリフェッチ

この関数は、指定されたtexture のtexture ページをプリフェッチ（sprite メモリに配置）するために使用することができます。sprite sprite を作成したときに定義したasset のインデックスを指定すると、そのページがあるtexture がメモリにロードされます。なお，この 関数は、選択されたリソースでプリフェッチがサポートされていない場合やターゲットプラットフォームが HTML5 の場合は -1 を返し、すべてが正しく動作した場合は 0 を返します。

注：ほとんどのデバイスでは、texture がtexture のメモリにアップロードされるため、パフォーマンスが低下することがあります。 sprite_prefetchで 必要なグラフィックは、ストールを避けるために、レベルの開始時に表示します。

構文です。

sprite_prefetch(ind)

論証タイプ 商品説明
ind The sprite index to fetch

リターンです。

(-1 or 0)

例

sprite_prefetch(spr_Player_Aura);

上記のコードは、参照されたsprite をtexture のメモリに配置し、使用可能な状態にします。
