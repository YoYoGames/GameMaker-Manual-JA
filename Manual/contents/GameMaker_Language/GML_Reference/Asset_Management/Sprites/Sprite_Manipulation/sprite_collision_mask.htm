スプライトコリジョンマスク

この機能により、sprite が持つべきコリジョンマスクのプロパティを設定することができます。バウンディングボックスモードとして自動（0）またはフルイメージ（1）を選択した場合、個々のバウンディングボックスの値は0に設定できます。しかし、ユーザー定義のマスク（2）の場合は、これらの値を設定する必要があります。異なるバウンディングボックスの値は、常にsprite の左上隅（x と y の原点に関係なく）を基準としており、これは位置 (0, 0) と見なされます。

マスクの種類を設定すると、マスク自体の一般的な形状が設定されますが、矩形以外のマスクは、衝突を解決する際に多くの処理能力を必要とし、結果としてパフォーマンスが低下することに注意してください。一般に、矩形以外のマスクタイプは、絶対に必要な場合にのみ使用するようにしてください。

注：この機能は、回転した矩形のコリジョンマスクの種類を許可していません。

設定可能なマスクの種類は、以下の定数のいずれかとなる。

バウンディングボックス 種類(形状) 定数
定数 商品説明
bboxkind_rectangular 矩形（非回転）コリジョンマスク形状
bboxkind_ellipse 楕円形のコリジョンマスクの形状
bboxkind_diamond ダイヤモンドコリジョンマスクの形状
bboxkind_precise 正確なコリジョンマスク。マスクは、与えられた許容値（下記参照）に基づき、sprite の非透過ピクセルに適合するようになります。）


最後に、トレランスは、マスクの精度を定義するために使用されます（「フル画像」マスクで使用する場合、これは何の効果もありません）。トレランスが0であれば、マスクは透明度が0を超えるすべてのピクセルに追従し、他の値はピクセルの透明度によってコリジョンマスク周辺をシフトさせます。

注：この機能はビットマップ sprites に対してのみ有効であり、SWF や JSON (Spine)sprites では動作しません。

注：この機能は、追加されたsprites または複製されたsprites に対してのみ機能し、既製のリソースに直接作用することはありません。sprites を複製するには、次の関数を使用します。 sprite_duplicate().

構文です。

sprite_collision_mask(ind, sepmasks, bboxmode, bbleft, bbtop, bbright, bbbottom, kind, tolerance);

論証 タイプ 商品説明
ind スプライトアセット The index of the sprite to set the bounding box of.
sepmasks ブーリアン Whether to create collision masks for each sub-image of the sprite (true)、または1つのマスクですべてをカバーする(false)。
bboxmode リアル What kind of bounding box to use. 0 = automatic, 1 = full image, 2 = user defined.
bbleft リアル The maximum left position of the bounding box.
bbtop リアル The maximum top position of the bounding box.
bbright リアル The maximum right position of the bounding box.
bbbottom リアル The maximum bottom position of the bounding box.
kind バウンディングボックス 種類(形状) 定数 The kind of mask, a constant (see the table in the description).
tolerance リアル Indicates the tolerance in the transparency value (0=no tolerance, 255=full tolerance).

リターンズ

N/A

例

spr = sprite_add("player_5.png", 16, true, true, 0, 0);
sprite_collision_mask(spr, true, 1, 0, 0, 0, 0, 0, 0);

上記のコードでは、外部からsprite を読み込み、新しいインデックスを変数 "spr" に格納しています。次に、このコードでは、新しいsprite に、そのサブイメージのそれぞれについて正確なコリジョンマスクを設定します。
