スケルトンスキン取得

スケルトン・アニメーションsprites では、アニメーションに別々のtextures (「スキン」と呼ばれます) を割り当てることができ、それにより、1つのアニメーションを複数の異なるものに使用することができます。この関数は、インスタンスが使用しているスケルトン・アニメーションstring に現在割り当てられているスキンの名前を (sprite として) 返します。返される名前は、アニメーション・プログラムでsprite を作成したときに設定したものです。

構文です。

skeleton_skin_get();

リターンです。

例

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

上記のコードは、現在アニメーションに割り当てられているスキンをチェックし、それが「skin_Enemy1」であれば、3つのオプションの中から新しいスキンを選択して設定することになります。
