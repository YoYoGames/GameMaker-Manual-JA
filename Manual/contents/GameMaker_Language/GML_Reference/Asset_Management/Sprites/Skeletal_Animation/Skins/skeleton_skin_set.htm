スケルトン・スキン・セット

スケルトン・アニメーションsprite は、様々なtextures （「スキン」と呼ばれます）で描画することができ、1つのアニメーションを複数の異なる目的に使用することができます。例えば、一般的な歩行アニメーションがあり、これをゲーム内のNPCキャラクターや敵などに使用する場合、描画するスキンを変更するだけで使用できます。これらのスキンは、あらかじめアニメーションプログラムで作成し、名前を付けておく必要があります。この名前を関数に渡すと（string ）、描画時にそのスキンを使用するようにsprite が設定されます。

構文です。

skeleton_skin_set(skinname);

論証タイプ 商品説明
skinname The name (a string) of the skin to use.

リターンです。

例

if skeleton_skin_get() == "skin_Enemy1"
{
    skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

上記のコードは、現在アニメーションに割り当てられているスキンをチェックし、それが「skin_Enemy1」であれば、3つのオプションの中から新しいスキンを選択して設定することになります。
