skeleton_skin_list

この関数では、スケルトン・アニメーションの一部として含まれるすべてのスキンの名前を（事前に作成された）DSリストに入力することができますsprite 。この名前はstrings になり、他のスケルトン・アニメーション用スキン関数で、これらのタイプに使用することができますsprite.

構文です。

skeleton_skin_list(sprite, list);

論証 タイプ 商品説明
sprite スプライトアセット The sprite index of the Spine skeletal animation to get the list from.
list DSリストID The ID of the DS list to populate with the animation names.

リターンです。

N/A

例

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

上記のコードでは、DSリストを作成し、そこにスキン名を入力しています。そして、リストが破棄される前に、ランダムなものが選ばれ、インスタンスに適用されます。
