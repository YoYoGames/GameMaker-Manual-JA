スケルトン・スロット・カラー・セット

この関数を使うと、アタッチメントスロットの色とアルファを設定でき、それにアタッチメントされたすべてのsprites がこのブレンド値で描画されるようになります。インスタンスsprite は、ブレンドカラーとアルファ値を設定できることに留意してください (image_blendと image_alpha)、アタッチメントも同様です(関数 skeleton_attachment_create_colour()そのため、スロットに割り当てられたアタッチメントsprite が持つ最終的な色とアルファは、これらすべての値の合成値となります。

構文です。

skeleton_slot_colour_set(slot, colour, alpha);

論証 タイプ 商品説明
slot 文字列 The slot name to set, a string
colour カラー The colour to set, either an integer, a constant, or a hex value.
alpha リアル The alpha to set from 0 to 1.

リターンです。

N/A

例

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上記のコードでは、DS リストを作成し、そのリストにroom のマウスの位置で見つかったスロット名を入力します。次に、スロットリストをloops で検索し、見つかったスロットの色を白にリセットし、色がまだ白でない場合はアルファ値を 1 とします。
