skeleton_find_slot

この機能を使うと、現在のインスタンスに関連するSpine sprite で、指定した部屋空間位置にあるスロットを見つけることができます。DS リストを作成し、その ID とチェックする x/y 位置を指定すると、リストにはstring の利用可能なアタッチメントスロット (アタッチメントの変更も含む) それぞれについて、名前sprite が入力されることになります。リストは常に最上位のスロットから降順にソートされることに注意してください。

構文です。

skeleton_find_slot(x, y, list);

論証 タイプ 商品説明
x リアル The x position in the room to check.
y リアル The y position in the room to check.
list DSリストID The ID of the DS list to populate with the DS maps.

リターンです。

N/A

例

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

上記のコードでは、DS リストを作成し、room でマウスの位置で見つかったスロット名を入力します。
