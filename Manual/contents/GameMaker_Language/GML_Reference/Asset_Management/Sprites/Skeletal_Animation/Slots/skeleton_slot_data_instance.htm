skeleton_slot_data_instance

この関数を使用すると、現在のインスタンスに関連付けられたSpine sprite の利用可能な添付ファイルスロットの各データを (事前に作成された)DS リストに入力できます (添付ファイルの変更も含まれます)。このデータは、以下のキー/値のペアを含むDS マップの形で返されます。

  • "name"（名前）。スロットの名前。
  • "bone"（骨）。骨の名前。
  • "attachment"（添付ファイル）。添付ファイルがある場合はその名前、添付ファイルがない場合は"(none) "を指定する。

各キーの値はstrings となり、これらのタイプの他のスケルトン添付関数で使用することができますsprite 。作成された DS Map は破棄されないので、loop DS リストを通して、作成された各マップを自分で破棄する必要があることに注意してください。

構文です。

skeleton_slot_data_instance(list);

論証 タイプ 商品説明
list DSリストID The ID of the DS list to populate with the DS maps.

リターンです。

N/A

例

var list = ds_list_create();
var open = true;
slot_name = "";
skeleton_slot_data_instance(list);
for (var i = 0; i < ds_list_size(list); i++)
{
    var map = list[| i];
    if open
    {
        if map[? "attachment"] == "(none)"
        {
            open = false;
            slot_name = map[? "name"];
        }
    }

    ds_map_destroy(map);
}

ds_list_destroy(list);

上記のコードでは、DS リストを作成し、そのリストにインスタンスsprite のスロット データを入力します。このデータは次に解析され、スロットデータを持つ個々の DS マップが抽出されます。そして、空のスロットがあるかどうかを調べ、もしあれば、 DS マップを破棄する前に、変数 "slot_name" に空のスロット名が代入されます。最後に、DS リストが不要になったので、これを破棄します。
