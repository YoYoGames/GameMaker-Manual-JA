skeleton_slot_colour_get

この関数を使用すると、アタッチメントスロットの色を取得することができます。スロット名 (string) を指定すると、この関数はその色を整数値で返します。

構文です。

skeleton_slot_colour_get(slot);

論証 タイプ 商品説明
slot 文字列 The slot name to check, a string

リターンです。

Real

例

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_colour_get(slot_list[| i]) != c_white
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上記のコードでは、DS リストを作成し、そのリストにroom のマウスの位置で見つかったスロット名を入力します。次に、スロットリストをloops で検索し、見つかったスロットの色を白にリセットし、色がまだ白でない場合はアルファ値を 1 とします。
