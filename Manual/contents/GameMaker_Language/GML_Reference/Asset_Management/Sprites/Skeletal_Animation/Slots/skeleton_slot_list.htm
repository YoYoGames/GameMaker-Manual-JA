スケルトン・スロット・リスト

この関数で、スケルトン・アニメーションの一部として作成されたすべてのスロットの名前を（あらかじめ作成された）DSリストに入力することができますsprite 。この名前はstrings になり、他のスケルトンアニメーションスロット関数で、これらのタイプに使用することができます。sprite.

構文です。

skeleton_slot_list(sprite, list);

論証 タイプ 商品説明
sprite スプライトアセット The sprite index of the Spine skeletal animation to get the list from.
list DSリストID The ID of the DS list to populate with the bone names.

リターンです。

N/A

例

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

上記のコードでは、DSリストを作成し、後で使用するためにスロット名を入力しています。
