スケルトンスロットのアルファ値を取得します。

この関数を使用すると、アタッチメントスロットのアルファ値を取得することができます。スロット名 (string) を指定すると、0 から 1 までのアルファ値の実数値を返します。

構文です。

skeleton_slot_alpha_get(slot);

論証 タイプ 商品説明
slot 文字列 The slot name to check, a string

リターンです。

Real

例

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if (!ds_list_empty(slot_list))
{
    for (var i = 0; i < ds_list_size(slot_list); ++i)
    {
        if skeleton_slot_alpha_get(slot_list[| i]) != 1
        {
            skeleton_slot_colour_set(slot_list[| i], c_white, 1);
        }
    }
}

上記のコードでは、DS リストを作成し、そのリストにroom のマウスの位置で見つかったスロット名を入力します。次に、スロットリストをloops で調べて、見つかったスロットの色を白にリセットし、アルファがまだ 1 でない場合はアルファを 1 にします。
