スケルトン・アタッチメント・セット

スケルトン・アニメーションsprite は、他の画像を添付ファイルとして追加することができます。これらの画像は、名前の付いたスロット（アニメーション・プログラムで添付スロットを作成するときに名前が付けられます）に追加され、現在のsprite のアニメーションと一緒に描画されます。この機能を使うと、指定したスロットに添付ファイルを設定することができます。このとき、スロットと添付ファイルの名前（strings ）を指定する必要があります。これらの名前は、使用するアニメーションプログラムによって定義されるか、または (添付ファイルの場合) skeleton_attachment_create().

なお、添付ファイルとして sprite_index を渡すと、そのsprite が使用されます。また、-1 を使用すると、スロットから添付ファイルを削除することができます。引数としてsprite インデックスを渡すと、sprite の名前をstring としてスロットに名前をつけてアタッチメントスロットを作成します (したがって、たとえば spr_sword を使用するとアタッチメントスロット "spr_sword" が作成されます)。スロットではsprite の最初の画像インデックス (0) とその x および y 原点オフセットを使用して、スケールは (1,1) と 0 回転となります。

構文です。

skeleton_attachment_set(slot, attachment);

論証 タイプ 商品説明
slot 文字列 The slot name (a string) to get the attachment of.
attachment 文字列またはスプライトアセット The name (as a string or a sprite_index) of the attachment image.

リターンです。

String

例

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

上記のコードは、「slot_leftHand」という名前のスロットに対して現在割り当てられているアタッチメント名をチェックし、空のstring が返された場合、新しいsprite をアタッチすることになります。
