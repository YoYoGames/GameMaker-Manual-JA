スケルトンアタッチメント作成

スケルトンアニメーションを作成するときsprite 、アタッチメントスロットとそこに入れるアタッチメントを割り当てることができます。これらは単純にアニメーションから離れた画像ですが、添付されると、添付されたボーンとともに動きます。通常、これらのアタッチメントはアニメーションプログラム (Spine) で割り当てますが、この関数を使用すると、ゲームのsprite asset を使用してランタイムに独自のアタッチメントを作成できます。この関数では、アタッチメントに名前を付け (string)、使用する sprite_index と image_index、および x と y の原点を設定し (sprite で定義したものとは異なる場合がありますsprite )、添付時に画像に適用したい変形も設定する必要があります。添付ファイルが正常に作成された場合、この関数は 1 を返し、そうでない場合 (sprite のインデックスが無効であるか、ベースsprite がSpine sprite でない場合) は、 -1 を返します。

構文です。

skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot);

論証 タイプ 商品説明
name 文字列 The name (as a string) of the attachment to create.
sprite スプライトアセット The sprite_index to get the attachment image from.
ind リアル The image_index to get the attachment image from.
xorigin リアル The x origin for the image being used.
yorigin リアル The y origin for the image being used.
xscale リアル The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale リアル The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot リアル The rotation of the image. 0=normal, 90=turned 90 degrees counter-clockwise etc.

リターンです。

Real (1 if successful, -1 if not)

例

skeleton_attachment_create("sword", spr_Weapons, 0, 0, 80, 1, 1, 90);
skeleton_attachment_create("knife", spr_Weapons, 1, 0, 45, 1, 1, 90);
skeleton_attachment_create("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

上記のコードは、「slot_leftHand」という名前のスロットに対して現在割り当てられているアタッチメント名をチェックし、空のstring が返された場合、新しいsprite をアタッチすることになります。
