skeleton_attachment_get

スケルトン・アニメーションsprite は、他のsprites をアタッチメントとして追加することができます。これらのsprites は、名前の付いたスロット（アニメーション・プログラムでアタッチメントスロットを作成するときに名前が付けられます）に追加され、現在のsprite のアニメーションと一緒に描画されます。この関数で、現在割り当てられているstring の指定されたスロットのアタッチメントの名前 (sprite) を取得することができます。アタッチメントsprites は、Spine で指定された名前文字列を通して参照されることに注意してください。 skeleton_attachment_create().

構文です。

skeleton_attachment_get(slot);

論証 タイプ 商品説明
slot 文字列 The slot name (a string) to get the attachment of.

リターンです。

String

例

if skeleton_attachment_get("slot_leftHand") == ""
{
    skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

上記のコードは、「slot_leftHand」という名前のスロットに対して現在割り当てられているアタッチメント名をチェックし、空のstring が返された場合、新しいsprite をアタッチすることになります。
