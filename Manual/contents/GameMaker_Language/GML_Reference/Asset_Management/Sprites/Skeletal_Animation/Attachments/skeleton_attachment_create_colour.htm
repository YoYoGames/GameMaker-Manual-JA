skeleton_attachment_create_colour

スケルトン・アニメーション（sprite ）を作成する際に、アタッチメント・スロットとそこに入れるアタッチメントを割り当てることができます。これらは単に画像 (sprites) で、アニメーションとは別物ですが、添付されると添付されたボーンと一緒に動きます。通常、これらのアタッチメントはアニメーションプログラム (Spine) で割り当てますが、この関数を使用すると、ゲームのsprite asset を使用して、ランタイムに独自のアタッチメントを作成できます。この関数では、アタッチメントに名前を付けて (string のように ) sprite_indexと image_indexsprite sprite また、画像に適用したい変形や、画像にブレンドする色、アルファ（透明度）値も設定できます。添付ファイルが正常に作成された場合、この関数は 1 を返し、そうでない場合 (sprite のインデックスが無効であるか、ベースsprite がSpine sprite でない) は、 -1 を返します。

注目すべきは、インスタンスsprite が、ブレンドカラーとアルファの設定(image_blendと image_alpha)、アタッチメントスロットも同様です(関数 skeleton_slot_colour_set()そのため、スロットに割り当てられたアタッチメントsprite が持つ最終的な色とアルファは、これらすべての値の合成となります。

構文です。

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, colour, alpha);

論証 タイプ 商品説明
name 文字列 The name (as a string) of the attachment to create.
sprite スプライトアセット The sprite_index to get the attachment image from.
ind リアル The image_index to get the attachment image from.
xorigin リアル The x origin for the image being used.
yorigin リアル The y origin for the image being used.
xscale リアル The horizontal scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale リアル The vertical scaling of the image, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot リアル The rotation of the image: 0 = normal, 90 = turned 90° counter-clockwise etc.
Colour カラー The colour value to use (A constant, integer or hex value).
Alpha リアル The alpha value to use (from 0 to 1).

リターンです。

Real (1 if successful, -1 if not)

例

skeleton_attachment_create_colour("sword", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("knife", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("crossbow", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

上記のコードは、「slot_leftHand」という名前のスロットに対して現在割り当てられているアタッチメント名をチェックし、空のstring が返された場合、新しいsprite をアタッチすることになります。
