skeleton_bone_data_set

骨格アニメーションはいくつかの "ボーン" で構成されています。アニメーションプログラムで定義し、名前をつけたボーンに、この関数を使用していつでも特定のデータを設定できます。このデータは、スケルトンのデフォルトポーズを参照しており、描画されている現在のポーズではないことに注意してください (そのためには、関数 skeleton_bone_state_set()また、事前に作成したDS マップから設定する必要があり、DS マップには以下のキーとそれに相当する値が必要である。

  • "x":親ボーンに相対するボーンのローカル x 位置。
  • "y":親ボーンに対するボーンのローカルy位置。
  • "angle "です。親ボーンに対するボーンの局所的な回転を指定します。
  • "xscale"。ボーンのローカルな水平スケール。
  • "yscale"。ボーンのローカル垂直スケール。
  • "parent"（親）。親骨の名前（string ）。

構文です。

skeleton_bone_data_set(bone, map);

論証 タイプ 商品説明
bone 文字列 The name (as a string) of the bone.
map DSマップID The (previously created) DS map that stores the bone data.

リターンです。

N/A

例

var bone_map = ds_map_create();
skeleton_bone_data_get("head", bone_map);
ds_map_replace(bone_map, "parent", "body");
skeleton_bone_data_set("head", bone_map);
ds_map_destroy(bone_map);

上記のコードでは、DSマップを作成し、"head "という名前のボーンからデータを取り込んでいます。そして、マップ内の "parent" ボーンキーを新しい値で置き換え、 "head" ボーンを新しいデータセットで再び設定します。
