スケルトンボーンリスト

この関数で、スケルトンアニメーションの一部として使用されるすべての骨の名前を、（あらかじめ作成された）DSリストに入力することができます。sprite 。この名前はstrings になり、他のスケルトンアニメーションのボーン関数で、これらのタイプに使用することができますsprite.

構文です。

skeleton_bone_list(sprite, list);

論証 タイプ 商品説明
sprite スプライトアセット The sprite index of the Spine skeletal animation to get the list from.
list DSリストID The ID of the DS list to populate with the bone names.

リターンです。

N/A

例

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

上記のコードでは、DS リストを作成し、後で使用するために骨の名前を入力しています。
