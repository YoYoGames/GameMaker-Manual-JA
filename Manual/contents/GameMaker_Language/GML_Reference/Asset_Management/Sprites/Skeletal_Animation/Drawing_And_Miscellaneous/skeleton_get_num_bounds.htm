スケルトンゲットナンバーバウンズ

この関数は、コードを実行しているインスタンスに割り当てられているスケルトン・アニメーションsprite に関連するバウンディング・ボックスの数を返します。この関数は skeleton_get_bounds()を使用して、各バウンディングボックスに関するデータを取得することができます。

構文です。

skeleton_get_num_bounds();

リターンです。

Real

例

var num = skeleton_get_num_bounds();
var yy = 60;
for(var i = 0; i < num; i++)
{
    var b_info = skeleton_get_bounds(i);
    if b_info[0] > 0
    {
        var data = b_info[1] + ":";
        for(var j = 0; j < b_info[0]; j++)
        {
            data += " (" + string(b_info[(j * 2) + 2]) + ", " + string(b_info[(j * 2) + 2 + 1]) + ")";
        }
        draw_text(20, yy, data);
        yy += 20;
    }
}

上記のコードは、現在割り当てられているloop に関連する各バウンディングボックスをsprite 通過し、そのデータをstring として画面に描画します。
