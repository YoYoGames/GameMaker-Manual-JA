スケルトンコリジョン描画セット

この機能により、インスタンスが現在使用しているスケルトン・アニメーションsprite のコリジョンデータを描画するオン (true) としない (false) を切り替えることができます。この機能をオンにすると、バウンディングボックスと正確なコリジョンマスクがsprite の周囲にアウトラインとして描画されます。

この機能は、スケルトンsprite の描画がobject によって処理されているときのみ動作します。 スケルトンsprite を手動で描画するときにコリジョンバウンディングボックスを描画するには、 draw_skeleton_collision() を使用します。

構文です。

skeleton_collision_draw_set(flag);

論証 タイプ 商品説明
flag ブーリアン Set to true で描画をオンにし、 でオフにします。 false

リターンです。

N/A

例

if debug_mode == true
{
    skeleton_collision_draw_set(true);
}

上記のコードは、ゲームがデバッグモードで実行されているかどうかをチェックし、実行されている場合は、表示するインスタンスのスケルトン衝突データをトグルしています。
