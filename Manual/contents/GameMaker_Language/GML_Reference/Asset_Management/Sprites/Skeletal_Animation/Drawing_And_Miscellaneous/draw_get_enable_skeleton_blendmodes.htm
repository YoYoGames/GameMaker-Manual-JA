draw_get_enable_skeleton_blendmodes

この関数は、スケルトンsprites のスロット毎のブレンドモードが有効（true）か無効（false）かを返します。

構文です。

draw_get_enable_skeleton_blendmodes()

リターンです。

Boolean

例

if (!draw_get_enable_skeleton_blendmodes())
{
    draw_enable_skeleton_blendmodes(true);
}

上記のコードは、スケルトン（sprites ）のスロットごとのブレンドモードが無効になっている場合、それを有効にします。
