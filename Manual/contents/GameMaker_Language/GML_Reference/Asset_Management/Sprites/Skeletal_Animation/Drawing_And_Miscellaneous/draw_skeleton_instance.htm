draw_skeleton_instance

この機能は、スケルトン・アニメーション・プログラムsprites を使って作成されたSpine が割り当てられたインスタンスにのみ使用します。これらのsprites は通常の描画関数 (たとえば draw_self()しかし、インスタンスのtexture プロパティを実際に変更することなく、1 つのフレームを描画したり、肌の変化sprite を表示したい場合があります。このような場合、この関数でインスタンスに割り当てられたsprite を描くことができます。一意のインスタンス ID を与え、フレームを取得するアニメーションの名前（string 、sprite を作成するために使用したプログラム内で設定された）を与えます。次に、使用するスキンの名前（これもsprite が作成されたときに設定されたもの）とフレーム値を指定します。フレーム値は、アニメーションのどの部分を表示するかをGameMakerに指示するためのもので、0から1までの値です。 image_number.残りの引数は、通常のsprite の描画に使用するものと同じで、最終的に描画されるアニメーションフレームのスケール、角度、ブレンドカラー、アルファを変更します。

警告!スケルトン・アニメーションは、さまざまな「キー」フレーム間で補間されるため、この呼び出しには追加のオーバーヘッドがあります。したがって、絶対に必要でない限り、このメソッドを使ってスケルトン・ベースsprites を描画しないことをお勧めします。

構文です。

draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, colour, alpha);

論証 タイプ 商品説明
instance インスタンスID The index of the instance to draw the sprite from.
animname 文字列 The name of the animation to get the frame from (a string).
skinname 文字列 The name of the skin to use (a string).
frame リアル The animation frame to draw (from 0 to image_number - 1).
x リアル The x coordinate of where to draw the sprite.
y リアル The y coordinate of where to draw the sprite.
xscale リアル The horizontal scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
yscale リアル The vertical scaling of the sprite, as a multiplier: 1 = normal scaling, 0.5 is half etc...
rot リアル The rotation of the sprite. 0=normal, 90=turned 90 degrees counter-clockwise etc.
colour カラー The colour with which to blend the sprite.
alpha リアル The alpha of the sprite (from 0 to 1 where 0 is transparent and 1 opaque).

リターンです。

N/A

例

draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1);

上記のコードは、指定されたインスタンスに割り当てられたスケルトン・アニメーションsprite の最初のフレームを、「jump」アニメーション・セットと「skin1」として定義されたスキンを使って描画するものです。
