スケルトン・ゲット・ミンマックス

この関数は、スケルトンアニメーションsprite に割り当てられた個々のバウンディングボックスの値の合計を配列で返します。返される配列の要素は 4 つです。

  • [0] - すべてのバウンディングボックスの最小のx位置です．
  • [1] - すべてのバウンディングボックスのy座標の最小値．
  • [2] - すべてのバウンディングボックスの最大 x 位置です．
  • [3] - すべてのバウンディングボックスの最大y位置．

構文です。

skeleton_get_minmax();

リターンです。

Array (4 elements: xMin, yMin, xMax, yMax)

例

var frame = skeleton_animation_get_frame(0);
if (frame > 0 && frame < 2)
{
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

上記のコードは、track 0 に割り当てられたアニメーションの現在のフレームをチェックし、sprite の全バウンディングボックスの中央にあるポイントからいくつかのパーティクルをバーストさせます。
