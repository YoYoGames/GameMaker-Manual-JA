スケルトン・アニメーション・セット・フレーム

この関数は、与えられたトラックに割り当てられているアニメーションを、指定されたフレームに設定します。フレームインデックスは 0 からアニメーションのフレーム数までの実数値である必要があります (これは関数 skeleton_animation_get_frames).この範囲外の値を指定した場合、アニメーションはその差を埋めるようにループします。例えば、アニメーションが 5 フレームで、フレームを 8 に設定した場合、実際に表示されるフレームは 3 になります。

構文です。

skeleton_animation_set_frame(track, index);

論証 タイプ 商品説明
track リアル The animation track to set the frame of.
index リアル The new index for the animation frame.

リターンです。

N/A

例

if global.pause
{
    skeleton_animation_set_frame(0, -image_speed);
}

上記のコードは、グローバル変数 "pause" が true のとき、トラック 0 のアニメーションのフレームインデックスを負の画像速度に設定します（実質的にアニメーションは一時停止します）。
