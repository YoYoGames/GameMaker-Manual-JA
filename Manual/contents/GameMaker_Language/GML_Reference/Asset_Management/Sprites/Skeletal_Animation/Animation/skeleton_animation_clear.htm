スケルトンアニメーションクリア

この関数は、指定されたアニメーショントラックのすべてのアニメーションを消去し、再割り当ての準備をします。

構文です。

skeleton_animation_clear(track);

論証 タイプ 商品説明
track リアル The animation track to clear.

リターンです。

N/A

例

if mouse_check_button(mb_right)
{
    skeleton_animation_clear(1);
}

上記のコードでは、マウスの右ボタンが押されると、アニメーショントラック1がクリアされます。
