skeleton_animation_is_finished

この関数により、現在の非ループスケルトンアニメーションが、与えられたトラックでの再生を終了したかどうかをチェックすることができます。

構文です。

skeleton_animation_is_finished(track);

論証 データ型 商品説明
線路 リアル アニメーショントラックのインデックス

リターンです。

Boolean

例

var _finished = skeleton_animation_is_finished(1);
if (_finished)
{
    show_debug_message("The animation has finished playing");
}

上記のコードでは、まずトラック 1 のアニメーションの再生が終了したかどうかをチェックし、その結果をローカル変数 _finished に格納しています。そして、アニメーションの再生が終了した場合には、デバッグメッセージを表示します。
