スケルトンアニメーションをループさせる

この関数は、与えられたトラックの現在のアニメーションがloop に設定されているかどうかを確認することができます。

構文です。

skeleton_animation_is_looping(track);

論証 タイプ 商品説明
線路 リアル アニメーショントラックのインデックス

リターンです。

Boolean

例

if skeleton_animation_is_looping(0)
{
    show_debug_message("Track 0 of the skeleton animation is looping");
}

上記のコードは、track 0 のアニメーションがloop に設定されているかどうかをチェックします。 もしそうであれば、デバッグメッセージを表示します。
