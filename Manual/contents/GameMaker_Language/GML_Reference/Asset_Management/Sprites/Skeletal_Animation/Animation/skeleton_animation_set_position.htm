スケルトンアニメーションの位置設定

この関数は、与えられたトラックに割り当てられているアニメーションを、指定された位置に設定します。もしこの範囲外の値を指定した場合は、トラックの位置はその値の小数部に設定されます。例えば、0.4、1.4、2.4、......とします。これらはすべて0.4の位置に設定されます。

構文です。

skeleton_animation_set_position(track, position);

論証 タイプ 商品説明
track リアル The animation track to set the frame of.
position リアル The new position for the animation frame.

リターンです。

N/A

例

skeleton_animation_set_position(1, 0.25);

上記のコードは、スケルトンアニメーションのトラック1の位置を0.25に、言い換えれば、アニメーションの開始から25％の位置に設定します。
