スケルトンアニメーションゲット

一つのスケルトン・アニメーションsprite は様々なアニメーション・セットを持つことができ、これらのセットは異なるトラックに割り当てることができるので、アニメーション・セットを「混ぜて使う」ことができます。この関数は、与えられたトラック番号で現在使われているアニメーションセットの名前を返します (関数 skeleton_animation_set_ext).

構文です。

skeleton_animation_get_ext(track);

論証 タイプ 商品説明
track リアル The track number to get the animation set name of.

リターンです。

String

例

if skeleton_animation_get_ext(1) != "bodyfight"
{
    skeleton_animation_set_ext("bodyfight", 1);
}

上記のコードは、トラック 1 で使用されているアニメーションセットを、"bodyfight" に変更します。
