スケルトンアニメーションリスト

この関数では、スケルトン・アニメーションの一部として含まれるすべてのアニメーションの名前を、（あらかじめ作成された）DSリストに入力することができますsprite 。この名前はstrings になり、他のアニメーション関数で、これらのタイプに使用することができますsprite.

構文です。

skeleton_animation_list(sprite, list);

論証 タイプ 商品説明
sprite スプライトアセット The sprite index of the Spine skeletal animation to get the list from.
list DSリストID The ID of the DS list to populate with the animation names.

リターンです。

N/A

例

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);

for (var i = 1; i < ds_list_size(list); i++;)
{
    skeleton_animation_mix(list[| 0], list[| i], 0.5);
}

ds_list_destroy(list);

上記のコードでは、インスタンスで使用されているsprite のすべてのアニメーション名の DS リストを作成しています。そして、loops 、最初のアニメーションを含むすべてのアニメーションの mix 値を 0.5 に設定します。
