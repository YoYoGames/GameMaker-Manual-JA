スケルトン・アニメーション・セット

インスタンスにスケルトン・アニメーションsprite を割り当てたら、この関数で、どのアニメーション・セットを使用するかを定義します。sprite を作成したとき (Spine で) は、異なる状態に対して複数のアニメーションセットを定義し、それぞれに適切な名前を付けていることでしょう。これらの名前は、ゲーム中にどのアニメーションセットを使用するかをGameMakerに伝えるために使用します。たとえば、sprite がプラットフォームゲームの場合、「jump」セット、「run」セット、「idle」セットをすべて1つのsprite に含めて、この関数を使って、ゲーム内でそれらの間でswitch を使用することができます。
アニメーションはデフォルトでループ再生されます。

この方法で新しいアニメーションを設定すると image_indexを 0 にリセットし、新しいアニメーションセットをそのアニメーションの最初から開始します。loop 。これが望ましい動作ではない場合、「混合」値を skeleton_animation_mix()を設定すると、新しいアニメーションを設定するときに、前のアニメーションとスムーズに補間されるようになります。また、別々のアニメーショントラックを使用する予定がない限り、前のアニメーションからデータが保持されるのを防ぐために、すべてのアニメーションの最初にkeyframe を設定する必要があります。

構文です。

skeleton_animation_set(animname, [loop]);

論証 タイプ 商品説明
animname 文字列 The name (a string) of the animation set to use.
loop ブーリアン アニメーションを にするかどうか（デフォルトはloop true)

リターンです。

N/A

例

if keyboard_check_pressed(vk_space)
{
    if skeleton_animation_get() != "jump"
    {
        skeleton_animation_set("jump");
    }
}

上記のコードは、スペースキーが押されると、使用中のアニメーションセットを「ジャンプ」セットに変更します。ただし、現在使用中のセットがまだ「ジャンプ」でない場合に限ります。
