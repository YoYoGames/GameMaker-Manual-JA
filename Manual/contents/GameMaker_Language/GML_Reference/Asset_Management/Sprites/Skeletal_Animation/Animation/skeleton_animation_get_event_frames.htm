スケルトンアニメーションゲットイベントフレーム

この関数は、与えられたアニメーションの、与えられたイベントのすべてのフレームを取得するために使用することができます。スケルトンアニメーションの名前（string として、アニメーションを作るために使われたプログラムで定義されたもの、あるいは関数 skeleton_animation_get() を使って返されたもの）とイベントの名前を指定します。この関数は、そのイベントのフレームを配列として返します。

指定されたイベント名が存在しない場合、この関数は最初の（そして唯一の）要素を-1とする配列を返す。

構文です。

skeleton_animation_get_event_frames(anim_name, event_name);

論証 タイプ 商品説明
anim_name 文字列 The animation name to get the frames of.
event_name 文字列 The event name to get the frames for.

リターンです。

Array of Reals

例

var _frames = skeleton_animation_get_event_frames(skeleton_animation_get(), "Footstep");

if (_frames[0] != -1)
{
    var _count = array_length(_frames);
    var _current_frame = skeleton_animation_get_frame(0);

    for (var i = 0; i < _count; i ++)
    {
        if (_frames[i] == _current_frame)
        {
            audio_play_sound(snd_footstep, 1, false);
            break;
        }
    }
}

上記のコードでは、「足音」イベントのフレームを取得しています。有効なフレームを含む配列が返された場合、loops 、現在のフレームが配列内のいずれかのフレームと等しいかどうかをチェックします。この場合、足音の効果音を再生し、loop をブレーク（停止）します。
