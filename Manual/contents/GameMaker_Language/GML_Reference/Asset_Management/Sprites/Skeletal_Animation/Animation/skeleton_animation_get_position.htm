スケルトンアニメーションの位置を取得します。

この関数は、指定されたアニメーショントラック上のアニメーションの位置を 0 から 1 までの値で返します。 この方法では、トラックの位置を、値 0 が 0%、値 1 が 100% に対応するパーセントとして取得することができます。

指定されたトラックにアニメーションが割り当てられていない場合や、インスタンスにsprite が設定されていない場合は、-1 を返します。

構文です。

skeleton_animation_get_position(track);

論証 タイプ 商品説明
track リアル The animation track to get the position of.

リターンです。

Real

例

var _position = skeleton_animation_get_position(0);
var _position_percent = _position * 100;
show_debug_message("The skeleton animation position on track 0 is at " + string(_position_percent) + "%.");

上記のコードでは、トラック 0 に割り当てられたスケルトン・アニメーションの現在の位置をチェックし、一時変数_position に格納しています。次に、その値を 100 倍してパーセンテージに変換し、その結果を別の一時変数_position_percent に格納しています。
最後に、パーセントの値を表示するデバッグ・メッセージを表示します。
