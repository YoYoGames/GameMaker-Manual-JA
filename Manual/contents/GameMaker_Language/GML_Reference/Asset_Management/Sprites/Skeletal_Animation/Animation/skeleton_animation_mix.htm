スケルトンアニメーションミックス

を使えば、簡単にswitch アニメーションセットを作成することができます。 skeleton_animation_set()関数がありますが、この場合、あるアニメーションから別のアニメーションに切り替わるときに、スキップやスタッターが発生することがあります。これを防ぐには、2つのアニメーションセットの間にミックス値を設定すれば、sprite がそれらの間を補間します。通常、この設定はスケルトンアニメーションのインスタンスの Create Event で行います。この設定は一度だけ行えばよく、GameMakerはそのインスタンスのアニメーションセットを使ってsprite に続くすべての変更を補間します。duration の値は 0 から 1 で、0.5 を指定すると、1つのセットから他のセットへ「半分ずつ」補間されることに注意してください。

注Spine 4.0では回転アニメーションの動作が変更され、最短経路で回転を補間するのではなく、keyframes の絶対値で補間されるようになりました。このため、Spine のアニメーションを古いバージョンからアップグレードする場合、予期しない動作をする可能性があります。また、回転が必要な動作をするためには、Spine プロジェクト内の変更が必要な場合があります。

構文です。

skeleton_animation_mix(animfrom, animto, duration);

論証 タイプ 商品説明
animfrom 文字列 The name (a string) of the first animation set to interpolate from.
animto 文字列 The name (a string) of the second animation set to interpolate to.
duration リアル The duration of the interpolation (from 0 to 1)

リターンです。

N/A

例

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

上記のコードは、スケルトン・アニメーションsprite を持つインスタンスの作成イベントに入り、2つのアニメーション・セット "walk" と "jump" の間を補間するためのアニメーション・ミックスの期間を設定します。
