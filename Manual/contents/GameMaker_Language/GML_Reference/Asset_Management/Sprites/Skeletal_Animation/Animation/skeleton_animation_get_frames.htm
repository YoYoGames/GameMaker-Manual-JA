skeleton_animation_get_frames

この関数は、任意のスケルトンアニメーションのフレーム数を取得するために使用することができる。スケルトンアニメーションの名前（string 、アニメーションを作るために使われたプログラムで定義されたもの、または関数 skeleton_animation_get() を使って返されたもの）を与えると、この関数はそれが持つフレームを整数値で返します。指定されたアニメーションが存在しない場合，この関数は 0 を返します。

構文です。

skeleton_animation_get_frames(anim_name);

論証 タイプ 商品説明
anim_name 文字列 The animation name to get the frames of.

リターンです。

Real

例

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

上記のコードは、アニメーションのフレーム数を取得し、sprite を最後のフレームに設定して、アニメーションを停止します。
