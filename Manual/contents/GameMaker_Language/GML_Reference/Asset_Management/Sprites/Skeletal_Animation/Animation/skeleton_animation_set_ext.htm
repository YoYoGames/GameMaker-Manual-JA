スケルトン・アニメーション・セット・テキスト

スケルトン・アニメーションsprite にアニメーション・セットを設定すると、内部的にはトラック 0が割り当てられます。しかし、さらに別のセットを別のトラック（1 以上）に割り当てることができます。このようにして、sprite で同時に 2 つ（またはそれ以上）のアニメーションを再生することができます。sprite は、このことを念頭に置いて、あなたのアニメーションプログラムで設計されている必要があります。トラック 0 の "base" アニメーションは、特定のボーンにだけkeyframes があり、次のトラックは他のボーンにkeyframes がある、というように。たとえば、"walk" アニメーションがあり、そのkeyframes がsprite の脚だけに影響する場合、頭のアニメーションと腕のアニメーションをそれぞれトラック 1 と 2 に設定し、ゲームのプレイ中にリアルタイムで基本動作とそれに適応できるようにすることが可能です。
アニメーションは、デフォルトでループ再生されます。

構文です。

skeleton_animation_set_ext(animname, track, [loop]);

論証 タイプ 商品説明
animname 文字列 The name (a string) of the animation set to use.
track リアル The track for the animation set to use.
loop ブーリアン アニメーションを にするかどうか（デフォルトはloop true)

リターンです。

N/A

例

if mouse_check_button(mb_left)
{
    skeleton_animation_set_ext("bodyfight", 1);
}
else
{
    skeleton_animation_set_ext("bodywalk", 1);
}

上記のコードは、マウスの左ボタンを押すと、トラック 1 が使用しているアニメーションセットを「bodyfight」セットまたは「bodywalk」セットに変更します。
