この関数は、スプライトエディタで sprite リソースに定義されているsprite の速度を取得するために使用できます。返された値は、例えば、次のような計算に使用することができます。 が異なる場合、何フレーム描画してもよい。 image_speed.この返り値は、スピードの種類によって大きく異なることに注意してください。 Sprite EditorでFrames Per SecondまたはFrames Per Game Frameのどちらかを適用した。次の2つの例で説明します。

  • 1秒間に1フレーム描画するsprite 、画像速度を0.5に設定すると、1秒間に0.5フレームで描画されることになります。ゲームのフレームレートが60フレーム/秒の場合、sprite は120フレームごとに1フレーム描画することになります。 ゲームフレーム
  • ゲームフレームあたり1フレームを描画するsprite 、画像速度を0.5に設定すると、ゲームフレームあたり0.5フレームを描画することになります。ゲームフレームレートが60フレーム/秒の場合、sprite は30フレームを描画します。 60ゲームフレームごとに

sprite を定義する際に、どのような種類のアニメーション速度が使われていたかは、関数 sprite_get_speed_type()また、アニメーションの速度と種類を設定することができます。 関数を使用して sprite_set_speed().

構文です。

sprite_get_speed(index)

論証タイプ 商品説明
index The index of the sprite to get the speed of

リターンです。

例

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

上記のコードは、sprite 速度とsprite アニメーションタイプを取得し、将来使用するためにインスタンス変数に格納します。
