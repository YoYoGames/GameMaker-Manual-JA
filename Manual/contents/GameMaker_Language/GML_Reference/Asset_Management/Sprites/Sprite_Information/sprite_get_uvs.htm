sprite_get_uvs

texture この関数は、sprite ページにあるtexture のサブイメージの UV 座標などのデータを配列で返します。この関数は、以下の 8 つの要素からなる配列を返します。

  • [0] = 左
  • [1] = トップ
  • [2] = 右
  • [3] = 下
  • [4] =asset コンパイラがsprites の左側から切り捨てたピクセルの量
  • [5] =asset コンパイラがsprites 上面から切り捨てたピクセル数
  • [6] =sprites ページに保存された元のtexture 幅のピクセルデータの正規化されたパーセンテージです。
  • [7] =sprites ページに保存されたオリジナルのtexture の高さからのピクセルデータの正規化されたパーセンテージです。

この配列は、他の描画関数、特に3D作業や2Dプリミティブ関数、シェーダー関数で使用することができます。

注意：この関数は、ベクターsprites またはスケルトンアニメーションsprites では動作しません。

構文です。

sprite_get_uvs(sprite, subimage);

論証タイプ 商品説明
sprite The index of the sprite to use.
subimage The sub-image of the sprite to use.

リターンズ

(1D, 8 elements)

例

var tex = sprite_get_uvs(sprite, 0);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上記のコードでは、与えられたsprite の UV 座標をローカル配列に格納し、その値をインスタンス変数に代入しています。
