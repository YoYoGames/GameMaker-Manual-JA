sprite_get_speed_type (スプライト・ゲット・スピード・タイプ)

この関数は、スプライトエディタで sprite リソースに定義されているsprite アニメーションタイプを取得するために使用できます。返される値は、以下の定数のいずれかです。

スプライト速度定数
定数 商品説明
spritespeed_framespersecond sprite 、1秒あたりのフレーム数でアニメーションを定義した。
spritespeed_framespergameframe sprite は、ゲーム1フレームあたりのアニメーションで定義しました。

次の2つの例は、スプライトのアニメーション速度を計算する際に生じる違いを示しています。

  • 1秒間に1フレーム描画するsprite 、画像速度を0.5に設定すると、1秒間に0.5フレームで描画されることになります。ゲームのフレームレートが 60 フレーム/秒の場合、sprite は 120 ゲームフレームごとに 1 フレーム描画することになります。
  • ゲームフレームごとに1フレームを描画するsprite 、画像速度を0.5に設定すると、ゲームフレームごとに0.5フレームを描画することになります。ゲームフレームレートが60フレーム/秒の場合、sprite は60ゲームフレームごとに30フレームを描画することになります。

sprite のアニメーションに使用された速度値が何であったかは、関数 sprite_get_speed()また、アニメーションの速度と種類を設定するには、関数 sprite_set_speed().

構文です。

sprite_get_speed_type(index)

論証 タイプ 商品説明
index スプライトアセット The index of the sprite to get the animation type of

リターンです。

Sprite Speed Constant (see above)

例

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

上記のコードは、sprite 速度とsprite アニメーションタイプを取得し、将来使用するためにインスタンス変数に格納します。
