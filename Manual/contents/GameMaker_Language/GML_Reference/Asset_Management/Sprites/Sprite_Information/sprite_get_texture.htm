sprite_get_texture

この関数はsprite texture ページへの特別なポインタを返します。この値は、他の描画関数、特に2D プリミティブ関数、およびシェーダ関数で使用することができます。返されたtexture ページに関する詳細な情報は、ここにあるさまざまな texture_ 関数を使用して得ることができます。

構文です。

sprite_get_texture(spr, subimg);

論証 タイプ 商品説明
spr スプライトアセット The index of the sprite to use.
subimg リアル The sub-image of the sprite to use.

リターンズ

Texture

例

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1);
draw_primitive_end();

上記のコードは、変数 "tex" に保持されたvertex でテクスチャされた 4texture の三角形の帯を描画します。
