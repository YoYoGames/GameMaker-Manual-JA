sprite_get_info

この関数は、指定されたsprite の情報を取得するために使用されます。sprite のインデックス（アセットブラウザで追加したasset やruntime で追加したスプライトでも可）を指定すると、この関数は以下の変数を持つ構造体を返します。

以下のセクションは、structs のタイプに基づき、返されるstruct に含まれる配列とsprite に関する情報を含んでいる。

一般的なSprite データ

このセクションでは、あらゆる種類のスプライトのためにstruct に含まれる変数に関する情報が記載されています。

変数 messages は、与えられたsprite に存在する放送メッセージの情報を格納する配列である。この配列の各放送メッセージは、以下の変数を含むstruct である。

変数名 データ型 商品説明
フレーム じっさいにある アニメーションの開始からのこの放送メッセージのタイミング（フレーム数）
メッセージ 列 放送メッセージの文字列

frame_info 変数は、スプライトのフレームのタイミングに関する情報を格納する配列です。この配列の各フレームは、以下の変数を含むstruct です。

変数名 データ型 商品説明
フレーム じっさいにある このフレームの開始タイミング（単位：フレーム）
存続期間 じっさいにある このフレームの継続時間（単位：フレーム）
画像インデックス じっさいにある このフレームの画像インデックス

ビットマップ用struct sprites (sprites でSpine や SWF 型sprites でないもの ) に含まれる変数に関する情報を記載します。

frames 変数は、スプライトのフレームtextures の情報を格納する配列です。この配列の各フレームは、以下の変数を含むstruct です。

変数名 データ型 商品説明
x じっさいにある このフレームのtexture ページにおける X 位置（ピクセル単位）。
y じっさいにある このフレームのtexture ページにおける Y 位置（ピクセル単位）。
w じっさいにある 内部で使用するフレームの論理幅（ピクセル）。
h じっさいにある 内部で使用するフレームの論理的な高さ（ピクセル単位
肌触り じっさいにある texture このフレームのページ ID
オリジナル幅 じっさいにある フレームの元の幅（ピクセル単位）
オリジナル高さ じっさいにある フレームの元の高さ（ピクセル単位）
クロップ幅 じっさいにある texture のページで、トリミングと拡大縮小を行った後の実際の枠の幅（GameMaker は画像の周りの空きスペースを自動的にトリミングし、またサイズが合わない場合は縮小するため）。
クロップハイト じっさいにある トリミング、スケーリング後のtexture のページにおけるフレームの実際の高さ

y_offset じっさいにある 元画像の上端から切り出し部分の上端までのYオフセット

シアーエックス じっさいにある (Internal toSpine) Xに関するせん断値。
せん断性Y じっさいにある (Internal toSpine) Yのせん断値

変数名 データ型 商品説明

構文です。

sprite_get_info(index);

論証 タイプ 商品説明
index スプライトアセット The index of the sprite to get the information for.

リターンズ

Sprite Info Struct (or undefined)

例

var _info = sprite_get_info(sprite_index);

if (_info != undefined && _info.type == 0)
{
    var _messages = _info.messages;
    var _messageCount = array_length(_messages);
    
    for (var i = 0; i < _messageCount; i ++)
    {
        var _message = _messages[i];
        show_debug_message("Message at frame " + string(_message.frame) + ": " + _message.message);
    }
}

struct上記のコードは、インスタンスの情報sprite を取得し、それが undefined でなく、そのタイプが 0（ビットマップであることを意味しますsprite ）であるかどうかをチェックします。この場合、そのsprite のブロードキャストメッセージの配列を取得し、forloop を実行して各ブロードキャストメッセージを（そのフレームとともに）出力ログに表示します。
