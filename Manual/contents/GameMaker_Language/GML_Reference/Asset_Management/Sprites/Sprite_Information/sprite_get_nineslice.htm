sprite_get_nineslice

この関数は、スプライトエディタまたはsprite_set_nineslice() を使用して設定された、そのすべてのナインスライスプロパティを含むsprite のナインスライス構造体を返します。このstruct の内容については、このページで詳しく説明しています。

提供されたsprite にナインスライスstruct が割り当てられていない場合、デフォルトのナインスライスプロパティを持つ新しいstruct がsprite に作成されて返されます。

このstruct の値を変更すると 、元のsprite の Nine Slice プロパティが変更され、そのsprite で今後行われる描画呼び出しに影響します。

構文です。

sprite_get_nineslice(ind);

論証タイプ 商品説明
ind The index of the sprite from which the Nine Slice struct will be retrieved

リターンです。

Nine Slice Struct (or -1 if the sprite doesn't exist)

例

var _box_nineslice = sprite_get_nineslice(spr_box_0);

_box_nineslice.enabled = true;
_box_nineslice.left = 10;
_box_nineslice.right = 10;
_box_nineslice.top = 10;
_box_nineslice.bottom = 10;

上記のコードは、struct からナインスライスsprite を取得し、そのナインスライスを有効にして、ガイドオフセットを設定します。
