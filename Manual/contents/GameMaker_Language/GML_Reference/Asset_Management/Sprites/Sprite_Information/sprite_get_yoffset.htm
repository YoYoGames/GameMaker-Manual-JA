sprite_get_yoffset

sprite エディタでsprite を定義するとき、そのsprite の原点を設定する必要があります。これは、インスタンスで使用されたときに、sprite が「接続」または「描画」されるポイントです。この関数は、原点の左上隅を 0,0 とし、そこから +y を下、-y を上として、y 軸の相対的なオフセットを返します。次の画像はその説明です。

構文です。

sprite_get_yoffset(index);

論証タイプ 商品説明
index The index of the sprite to find the yoffset of.

リターンズ

例

if y - sprite_get_yoffset(sprite_index) < 0
{
    y = _yoffset;
}

上記のコードでは、room に基づいてsprite 内でインスタンスが維持されるようになります (従って、sprite は常に表示されます)。
