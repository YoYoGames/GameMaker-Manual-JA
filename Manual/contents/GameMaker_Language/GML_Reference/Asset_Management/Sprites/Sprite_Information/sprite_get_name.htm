スプライトゲット名

この関数は、指定されたsprite の名前を文字列として返します。この名前はGameMaker のメインウィンドウのリソースツリーでsprite に指定されている名前である。これはあくまで string であり、sprite を直接参照することはできないことに注意してください - そのためには、スプライトのインデックスが必要です。しかし、このstring を使って、返されたstring と、関数 asset_get_index().

構文です。

sprite_get_name(index);

論証 タイプ 商品説明
index 資産 The index of the sprite to get the name of.

リターンズ

String

例

str = sprite_get_name(sprite_index);

上記のコードは、コードを実行しているインスタンスのsprite インデックス名を取得し、その戻り値を変数 "str "に格納します。
