スプライトゲットナンバー

sprite は少なくとも1つの副画像を持たなければなりませんが、この関数はそれがいくつあるか返します。sprite の部分画像は 0 から番号が振られることを覚えておいてください。この関数が 3 を返した場合 (sprite が 3 つの部分画像を持っている)、それらは 0、1、2です。

構文です。

sprite_get_number(index);

論証タイプ 商品説明
index The index of the sprite.

リターンズ

例

image_index = floor(random(sprite_get_number(sprite_index)));

上記のコードは、sprite が持つサブイメージの数に基づいて、ランダムなフレームにイメージインデックスを設定します。
