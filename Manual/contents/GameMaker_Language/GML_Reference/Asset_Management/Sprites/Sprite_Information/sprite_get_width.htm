スプライトゲット幅

この関数を使うと、変換を行わないベースリソースsprite の幅をピクセル単位で求めることができる。

構文です。

sprite_get_width(index);

論証タイプ 商品説明
index The index of the sprite to find the width of.

リターンズ

例

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

上記のコードは、現在のインスタンスにあるsprite の幅をチェックし、それと元のベースsprite との間に差がある場合、X 軸のスケールをリセットしています。
