sprite_get_bbox_mode

この関数は、バウンディングボックス計算の現在の "モード "を返す。チェックするsprite のsprite インデックスを指定すると、この関数は以下に示す定数のいずれかを返します。

構文です。

sprite_get_bbox_mode(ind);

論証タイプ 商品説明
ind The index of the sprite to check.

リターンズ

Bounding Box Mode Constant

定数 商品説明
bboxmode_automatic 自動 - スプライトのトレランス設定に基づき、バウンディングボックスが自動的に計算されます。
bboxmode_fullimage Full Image - 許容範囲や「空の」ピクセルに関係なく、バウンディングボックスがsprite の幅と高さを完全に使用するように設定されます。
bboxmode_manual 手動 - バウンディングボックスをユーザー定義の値に手動で設定（sprite エディターで設定するか、関数 sprite_set_bbox())

例

if sprite_get_bbox_mode(sprite_index) != 0
{
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
}

上記は、現在のsprite の bbox モードをチェックし、それが automatic でなければ、その値に設定しています。
