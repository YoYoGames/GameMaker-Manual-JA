sprite_get_tpe

この関数は、ゲームで使用するsprite ページにおける、与えられたtexture とサブイメージの位置に関連する特別な値を返します。このようにして、コンパイルしたゲームの外部で使用するためにextensions に値を渡すと、GameMaker の描画関数を効果的にバイパスして、sprite を DOM コンテンツで使用することができ、ゲームキャンバスを含むウィンドウ内のどこにでも描画することができます。この関数は、ホストページのGameMaker のcanvas 要素の外側にボタンなどのインタラクティブなメディアを作成したい場合に、特に有用です。 clickable_add().

注：この機能はHTML5のみです。

構文です。

sprite_get_tpe(sprite, index);

論証 タイプ 商品説明
sprite スプライトアセット The index of the sprite to find the texture page entry of.
index リアル The image index of the sprite.

リターンズ

Texture Page Entry

例

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

上記のコードでは、ゲームキャンバスが実行されているページの位置 (32, 32) に DOM ボタンを作成しています。このボタンはsprite ページから "spr_MS_Home" として参照されているtexture を使用し、クリックすると指定された URL の新しいウィンドウが、そのウィンドウに定義されたプロパティとともに開かれるようになっています。
