イメージ番号

この読み取り専用の変数は、インスタンスに割り当てられたsprite のサブイメージの数を取得するために使用できます（インスタンスに割り当てられたサブイメージ以外のsprite のサブイメージの数が必要な場合は、次のように使用します）。 sprite_get_number()).サブ画像が 1 個の場合、この変数は "1" を返しますが、そのサブ画像の image_index は 0 となることに注意してください。

構文です。

image_number;

リターンです。

(single precision floating point value)

例

if image_speed > 0
{
    if image_index > image_number - 1 instance_destroy();
}

上記のコードでは、sprite がアニメーションしているかどうかを確認し、アニメーションしている場合は、現在の image_index がサブイメージの数よりも大きいかどうかを確認し、大きい場合はインスタンスを破棄しています。
