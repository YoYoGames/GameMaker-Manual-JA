bbox_top

この読み取り専用変数は、インスタンスのバウンディングボックスの上端のroom 内の位置を (y 軸に沿って) 返します。バウンディングボックスは、インスタンスのマスクの最大幅と高さ (sprite_index または mask_index で設定される) で定義されます。sprite が正確なコリジョンマスクを持っている場合でも、バウンディングボックスは存在し、特定のことに使用されるので、この変数を使用して見つけることができます。インスタンスにsprite が割り当てられていない場合、返される値はインスタンスの Y 位置と同じであることに注意してください。

構文です。

bbox_top;

リターンです。

(integer)

例

if bbox_top < 0
{
    y = sprite_yoffset;
}

上記のコードでは、インスタンスのバウンディングボックスがroom の外にあるかどうかを確認し、外にある場合はインスタンスを新しい位置に設定しています。
