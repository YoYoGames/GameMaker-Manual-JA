画像_xscale

この値は、現在のインスタンスに割り当てられているsprite に適用される水平方向のスケーリングを設定します。1のスケールはスケーリングなし（1:1）を示し、小さい値はスケールダウン（例えば0.5はsprite の幅を半分にします）、大きい値はスケールアップ、負の値はsprite を反転してスケールします。使用する値がちょうど-1（この場合sprite はスケーリングなしでその原点を中心に単に反転します）である場合を除き、この値はスケールします。

構文です。

image_xscale;

リターンです。

(single precision floating point value)

例

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上記のコードでは、sprite を拡大縮小し、元のサイズの 5 倍に拡大したら、別のobject のインスタンスを新規に作成し、そのインスタンスを破棄しています。
