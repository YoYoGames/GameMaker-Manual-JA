bbox_left

この読み取り専用の変数は、インスタンスの左側のバウンディングボックスのroom 内の位置 (x 軸に沿って) を返します。バウンディングボックスは、インスタンスのマスクの最大幅と高さ (sprite_index または mask_index で設定) によって定義されています。sprite が正確なコリジョン マスクを持っている場合でも、バウンディング ボックスは存在し、特定のことに使用されるので、この変数を使用して見つけることができます。インスタンスにsprite が割り当てられていない場合、返される値はインスタンスの X 位置と同じであることに注意してください。

構文です。

bbox_left;

リターンです。

(integer)

例

if bbox_left < 0
{
    x = sprite_xoffset;
}

上記のコードでは、インスタンスのバウンディングボックスがroom の外にあるかどうかを確認し、外にある場合はインスタンスを新しい位置に設定しています。
