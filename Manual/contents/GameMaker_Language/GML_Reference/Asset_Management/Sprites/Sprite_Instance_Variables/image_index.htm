sprite は、1つまたは複数のサブイメージで構成されており、sprite がswitch のようにアニメーションで表示されたり、ウィンドウズのボタンのように、コードで切り替えて異なる「状態」を与えることができます。sprite がアニメーションしている場合、 image_index 変数をチェックすることでアニメーションの現在のフレームを取得できます。静的なsprite の状態を変更したい場合、この変数にsprite の希望のサブイメージを設定することで新しいサブイメージを選択できます。 この変数の変化が見えるようにするには、インスタンスが draw イベントを持っていないか（つまりGameMakerがsprite を初期描画します）、次のいずれかの描画関数を使用して描かれていなければならないことに注意してください：(a)インスタンスが描画されていない、(b)描画関数が使用されている、(c)インスタンスが描画されている。 draw_self()または draw_sprite_ext()(適切な引数に image_index を与えることで)。

この変数をフレームに直接設定しても、そのフレームに存在する可能性のあるブロードキャストメッセージはトリガーされないことに注意してください。

また、スケルトン・アニメーションsprites を使用している場合でも、 image_index の値を取得したり、設定したりすることができることに注意してください。 skeleton_animation_get_duration()を参照してください。

構文です。

image_index;

リターンです。

(single precision floating point value)

例

if (image_speed > 0)
{
    if (image_index >= image_number - 1) instance_destroy();
}

上記のコードでは、sprite がアニメーションしているかどうかを確認し、アニメーションしている場合は、現在の image_index が最後のフレームにあるかどうかを確認し、その場合はインスタンスを破棄しています。
