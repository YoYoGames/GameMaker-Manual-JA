イメージアングル

この値はsprite の角度（回転）を設定するもので、単位は度で、右が0º、上が90º、左が180º、下が270ºとなります。この変数を0に設定すると、sprite はsprite エディターで定義されたとおりに描画されるようにリセットされます。この変数を変更するためには、インスタンスに描画イベントがないか（つまりGameMakerがデフォルトでsprite を描画する）、拡張描画関数の1つを使って描画されなければならないことに注意してください。 draw_self()または draw_sprite_ext().

構文です。

image_angle;

リターンです。

Real

例

image_angle = point_direction(x, y, mouse_x, mouse_y);

上記のコードでは、インスタンスのsprite が常にマウスの位置を指すように回転されます。
