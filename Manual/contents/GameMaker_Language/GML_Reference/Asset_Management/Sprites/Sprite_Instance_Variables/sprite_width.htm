sprite_width

この読み取り専用変数は、インスタンスに割り当てられているsprite の幅を返す。この幅はピクセル単位で返され、インスタンスに割り当てられた image_xscale.もし、拡大縮小されていない幅が必要な場合は、. sprite_get_width().

構文です。

sprite_width

リターンです。

Real

例

if sprite_width != sprite_get_width(sprite_index)
{
    image_xscale = 1;
}

上記のコードは、コードを実行しているインスタンスに割り当てられたsprite の幅をsprite のリソースの幅と照合し、同じでない場合は image_xscale を 1 にリセットしています。
