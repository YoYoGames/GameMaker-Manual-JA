image_alpha

この変数はsprite のアルファ値を取得または設定するために使用されます。 アルファ値は常に 0 と 1 の間の値として計算され、0 は完全に透明、1 は完全に不透明になります。この変数の変化を見るためには、インスタンスが draw イベントを持たないか（つまりGameMakerがデフォルトでsprite を描画する）、または次のような拡張描画関数を使用して描画される必要があることに注意してください。 draw_self()または draw_sprite_ext().

構文です。

image_alpha;

リターンです。

Real

例

image_alpha = clamp(image_alpha - 0.01, 0, 1);

上記のコードは、0になるまでゆっくりとimage_alphaを減らしていきます。
