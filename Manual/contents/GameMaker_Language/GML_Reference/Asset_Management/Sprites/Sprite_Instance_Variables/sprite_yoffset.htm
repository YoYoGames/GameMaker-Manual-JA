sprite_yoffset

この読み取り専用変数は、インスタンスに割り当てられたsprite のローカル yoffset (sprite エディタで定義された原点の y 成分) を返します。このyoffsetはピクセル単位で返され、その値はインスタンスのサイズに依存します。 image_yscale例えば、32x32 の正方形sprite の y 原点が 16 と定義されている場合、そのsprite が 2 倍に拡大されると、この変数は 32 を返します。もし、スケーリングされていない yoffset が必要なら、次のようにします。 sprite_get_yoffset().

構文です。

sprite_yoffset

リターンです。

Real

例

if y - sprite_yoffset < 0
{
    y = sprite_yoffset;
}

上記のコードは、インスタンスのsprite の上端がroom の外にあるかどうかをチェックし、外にある場合はインスタンスを移動してsprite 全体が見えるようにします。
