スプライトインデックス

この変数は、インスタンスの現在のsprite のインデックスを返し、インスタンスにsprite が関連付けられていない場合は -1 を返します。リソースツリーからsprite の名前を与えるか、外部で読み込まれたsprite のインデックスを持つ変数を使用することで、インスタンスに異なるsprite を与えるように変更することができます。sprite を変更しても、現在表示されているフレームのインデックスは変更されないので、フレーム番号 3 のsprite を変更すると、新しいsprite はそのフレームが見える状態で描画されます（フレーム数が同じであると仮定しています）。

構文です。

sprite_index;

リターンです。

Sprite Asset

例

with (obj_Check)
{
    if !collision_line(x, y, other.x, other.y, obj_wall, false, true)
    {
        sprite_index = spr_spotted;
    }
    else
    {
        sprite_index = spr_clear;
    }
}

上記のコードは、loop 、「obj_Check」の全インスタンスとこのコードを実行しているインスタンスとの間に衝突線があるかどうかをチェックします。これらのインスタンスのsprite は、衝突線の戻り値 (真または偽) に応じて変更されます。
