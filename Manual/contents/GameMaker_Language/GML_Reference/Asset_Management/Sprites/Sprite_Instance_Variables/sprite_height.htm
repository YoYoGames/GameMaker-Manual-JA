スプライトハイト

この読み取り専用変数は、インスタンスに割り当てられているsprite の高さを返す。この高さはピクセル単位で返され、インスタンスに割り当てられている image_yscale.もし、拡大縮小されていない高さが必要な場合は、. sprite_get_height().

構文です。

sprite_height;

リターンです。

Real

例

if sprite_height != sprite_get_height(sprite_index)
{
    image_yscale = 1;
}

上記のコードは、コードを実行しているインスタンスに割り当てられたsprite の高さとsprite リソースの高さを照合し、同じでない場合は image_yscale を 1 にリセットしています。
