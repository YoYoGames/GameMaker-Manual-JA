マスクインデックス

GameMakerでobject を定義するとき、定義されたsprite に対応するマスクではなく、衝突に使用するマスクを割り当てることができます。この変数を使うことで sprite_indexsprite または、あるインスタンスのマスクを選択したsprite に設定します。マスクのインデックスを設定するということは、例えば、不規則な形状のインスタンスにsprite を設定し、別のsprite から取得した円形のコリジョン マスクを与えることができることを意味します。

構文です。

mask_index;

リターンです。

Sprite Asset

例

mask_index = spr_Round;

上記のコードでは、インスタンスのマスクをsprite "spr_Round "のマスクに設定しています。
