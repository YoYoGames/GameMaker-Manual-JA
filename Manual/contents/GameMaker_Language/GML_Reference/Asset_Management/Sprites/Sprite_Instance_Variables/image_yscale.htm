image_yscale

この値は、現在のインスタンスに割り当てられているsprite に適用される垂直方向のスケーリング（Y軸に沿って）を設定します。sprite 1のスケールはスケーリングなし（1:1）を示し、小さい値はスケールダウン（例えば0.5はsprite の高さの半分）、大きい値はスケールアップ、負の値はsprite をミラーリングしてスケールします。

構文です。

image_yscale

リターンです。

(single precision floating point value)

例

if image_xscale < 5
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上記のコードでは、sprite を拡大縮小し、元のサイズの 5 倍に拡大したら、別のobject のインスタンスを新規に作成し、そのインスタンスを破棄しています。
