イメージスピード

この変数は、GameMakerが現在のインスタンスsprite に対して、サブ画像を循環させる速度を決定する。0.5に設定すると、スプライトエディタや イメージエディターで設定したアニメーションスピードの半分になり、2に設定すると2倍になります。使用するsprite にサブ画像がない場合、この変数は何の効果もありません。

構文です。

image_speed;

リターンです。

(single precision floating point value)

例

with (instance_create_layer(x, y, "Effects", obj_Explosion))
{
    image_speed = 0.5;
}

上記のコードでは、object "obj_Explosion" のインスタンスを作成し、その image_speed を 0.5 に設定します（実質的に走行速度が半分になります）。
