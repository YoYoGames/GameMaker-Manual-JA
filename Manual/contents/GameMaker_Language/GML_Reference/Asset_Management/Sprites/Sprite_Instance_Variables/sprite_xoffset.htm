スプライトxオフセット

この読み取り専用変数は、インスタンスに割り当てられているsprite のローカル xoffset (sprite エディタで定義されている原点の x 成分) を返します。このxoffsetはピクセル単位で返され、その値はインスタンスに割り当てられているX座標に依存します。 image_xscale例えば、32x32 の正方形sprite の x 原点が 16 と定義されている場合、そのsprite が 2 倍に拡大されると、この変数は 32 を返します。もし、スケーリングされていない xoffset が必要なら、次のようにします。 sprite_get_xoffset().

構文です。

sprite_xoffset;

リターンです。

Real

例

if x - sprite_xoffset < 0
{
    x = sprite_xoffset;
}

上記のコードは、インスタンスのsprite の左端がroom の外に出ているかどうかをチェックし、出ていればsprite 全体が見えるようにインスタンスを移動させるものです。
