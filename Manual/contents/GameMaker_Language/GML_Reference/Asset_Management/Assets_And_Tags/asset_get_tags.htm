asset_get_tags

この関数を使うと、asset に割り当てられたすべてのタグをAsset Browser から取得することができます。asset の名前 (string) またはasset のインデックスを指定すると、この関数はそのasset に対応するタグの配列を返します。タグが見つからないか、エラーが発生した場合（たとえば、指定された名前string が存在しない場合）、返される配列は空になります。asset インデックス値を指定する場合、オプションでasset タイプ引数 (定数) を指定する必要があります。これは、異なるタイプのassets が、同じ名前を持つことはできなくても、同じインデックスを持つことができるからです。利用可能なasset のタイプは、以下の表のとおりです。

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を指します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。

構文です。

asset_get_tags(name_or_index, [asset_type]);

論証 タイプ 商品説明
name_or_index 文字列またはアセット The name of the asset (a string) or its index.
[asset_type] アセットタイプ 定数 OPTIONAL! The type of asset to get the tags for, only used when supplying an index for the first argument.

リターンです。

Array

例

my_tags = asset_get_tags(object_get_name(object_index));

上記のコードは、このコードを実行しているインスタンスが作成されたobject に割り当てられているすべてのタグを取得するものです。
