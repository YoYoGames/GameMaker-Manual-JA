asset_get_type

この関数を使うと、参照されているasset の種類をその名前（astring ）から取得することができます。以下の定数のいずれかが返されます。

構文です。

asset_get_type(name);

論証 タイプ 商品説明
name 文字列 The name of the game asset to get the type of (a string).

リターンです。

Asset Type Constant

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を意味します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。
asset_unknown 与えられた名前は、存在しないか、上記のいずれでもないasset を指します。

例

if asset_get_type("pth_Path_" + string(global.Game)) == asset_unknown
{
    show_debug_message("Path doesn't exist!!!");
}
else
{
    path_index = asset_get_index("pth_Path_" + string(global.Game));
}

上記のコードでは、動的に作成されたasset の名前をチェックして、asset が正しい型であるかどうかを確認しています。そうでない場合はデバッグメッセージが表示され、そうでない場合はasset の名前を使用してasset がインスタンスに割り当てられます。
