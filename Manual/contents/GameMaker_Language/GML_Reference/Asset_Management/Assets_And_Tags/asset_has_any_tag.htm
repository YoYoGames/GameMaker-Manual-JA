asset_has_any_tag

この関数を使うと、1つ以上のタグstrings がasset ブラウザから任意のasset に割り当てられているかどうかを確認することができます。asset の名前 (string) か、そのasset のインデックス、そして単一のタグstring か、各項目が単一のタグである配列string のいずれかを指定します。asset インデックス値を指定する場合、オプションでasset タイプ引数（定数）を指定する必要があります。これは、異なるタイプのassets は、同じ名前を持つことはできませんが、同じインデックスを持つことができるからです。利用可能なasset のタイプは、以下の表のとおりです。

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を指します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。

この関数が成功し、1つ以上のタグがasset に存在する場合、 true を返し、そうでない場合、 false を返します。もし、任意のタグやタグの集合に正確にマッチするかどうかを調べる必要がある場合は、関数 asset_has_tags().

構文です。

asset_has_any_tag(name_or_index, tags, [asset_type]);

論証 タイプ 商品説明
name_or_index 文字列または アセット The name of the asset (a string) or its index value (an integer).
tags 文字列または 配列文字列の A single asset tag string or an array with various asset tags.
[asset_type] アセットタイプ 定数 OPTIONAL! The type of asset to check the tags for, only used when supplying an index for the first argument.

リターンです。

Boolean

例

var _a = array_create(3);
_a[0] = "boss";
_a[1] = "main_boss";
_a[2] = "final_boss";

if asset_has_any_tag(object_index, _a, asset_object)
{
    instance_create_layer(0, 0, "Overlay", obj_Boss_Text);
}

上記のコードでは、タグの配列を作成し、そのいずれかが与えられたobject に適用されているかどうかを確認し、適用されている場合は、別のobject のインスタンスを作成します。
