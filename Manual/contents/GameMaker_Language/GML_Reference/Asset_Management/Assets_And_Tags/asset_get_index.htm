asset_get_index

この関数を使用すると、ゲーム名からasset の一意の識別インデックスを取得することができます。asset が見つからない場合、この関数は値 -1 を返します。そうでない場合は、チェック対象のasset の一意のインデックス ID を返します。この ID は、他の関数で他のインデックス値と同じように使うことができます。 sprite_indexまたは path_index例えばこの関数はassets からstrings を参照するために使用できますが (以下の例を参照)、使用する前に必ずasset が存在することを確認する必要があります。そうしないと、エラーが発生してゲームがクラッシュする可能性があります。

構文です。

asset_get_index(name);

論証タイプ 商品説明
name The name of the game asset to get the index of (a string).

リターンです。

Asset (any asset type)

例

var obj = asset_get_index("obj_Enemy_" + string(global.Level));

if (object_exists(_obj))
{
    instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

上記のコードは、object からstring のインデックスを取得し、そのインデックスが存在すれば、ゲーム内にobject のインスタンスを作成します。
