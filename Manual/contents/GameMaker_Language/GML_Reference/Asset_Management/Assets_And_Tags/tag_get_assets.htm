タグ_get_assets

この関数は、与えられたタグが割り当てられているすべてのassets の名前を取得することができます。指定するのは単一のタグstring か配列で、配列の各項目はタグstring です。この関数は配列を返し、各エントリには、指定したタグを持つstring の名前 (asset) が入ります。もしasset の一意なインデックスが必要なら、この関数を使うことができます。 asset_get_index()と共に返される。指定したタグを持つassets が存在しない場合、あるいはエラー (例: 指定したタグが存在しない) の場合は、空の配列が返されます。

構文です。

tag_get_assets(tags);

論証 タイプ 商品説明
tags 文字列または文字列の配列 A single asset tag string or an array with various asset tags.

リターンです。

Array

例

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
    if asset_get_type(_assets[i]) == asset_sprite
    {
        ds_list_add(backgrounds, asset_get_index(_assets[i]));
    }
}

上記のコードでは、リストを作成し、「background」というタグを持つすべてのassets の名前を取得しています。loops 返された名前の配列に目を通して、どれかがsprite assets であるかどうかを調べ、もしそうなら、asset の一意なインデックス値を今後使用するためにリストに追加します。
