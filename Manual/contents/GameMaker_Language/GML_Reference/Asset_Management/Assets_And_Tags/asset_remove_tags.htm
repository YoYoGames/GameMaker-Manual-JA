asset_remove_tags

この関数を使うと、strings ブラウザから、任意のasset に対する 1 つ以上のタグasset を削除することができます。asset の名前 (string) か、そのasset のインデックス、および単一のタグstring か、各項目が単一のタグである配列string のいずれかを指定します。asset インデックス値を指定する場合、オプションでasset タイプ引数（定数）を指定する必要があります。これは、異なるタイプのassets は、同じ名前を持つことはできませんが、同じインデックスを持つことができるからです。利用可能なasset のタイプは、以下の表のとおりです。

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を指します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。

タグの除去に成功した場合は、 true を返し、失敗した場合は、 false を返します。

構文です。

asset_remove_tags(name_or_index, tags, [asset_type]);

論証 タイプ 商品説明
name_or_index 文字列またはアセット The name of the asset (a string) or its index value (an integer).
tags 文字列または文字列の配列 A single asset tag string or an array with various asset tags.
[asset_type] アセットタイプ 定数 OPTIONAL! The type of asset to remove the tags from, only used when supplying an index for the first argument.

リターンです。

Boolean

例

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

上記のコードは、タグの配列を作成し、与えられたobject からタグを削除します。
