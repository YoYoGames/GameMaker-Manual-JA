asset_clear_tags

この関数を使うと、asset ブラウザから、任意のasset に存在するすべてのタグをクリアすることができます。asset の名前 (string) か、そのasset インデックスを指定します。asset インデックス値を指定する場合は、オプションでasset タイプ引数 (定数) を指定する必要があります。異なるタイプのassets は、同じ名前を持つことはできませんが、同じインデックスを持つことができるからです。利用可能なasset のタイプは、以下の表のとおりです。

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を指します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。

この関数がタグのクリアに成功した場合は、 true を返し、そうでない場合は、 false を返します。

構文です。

asset_clear_tags(name,[asset_type]);

論証 タイプ 商品説明
name_or_index 文字列またはアセット The name of the asset (a string) or its index value (an integer).
[asset_type] アセットタイプ 定数 OPTIONAL! The type of asset to clear the tags from, only used when supplying an index for the first argument.

リターンです。

Boolean

例

asset_clear_tags(obj_Enemy_Parent, asset_object);

上記のコードは、与えられたobject asset からすべてのタグをクリアします。
