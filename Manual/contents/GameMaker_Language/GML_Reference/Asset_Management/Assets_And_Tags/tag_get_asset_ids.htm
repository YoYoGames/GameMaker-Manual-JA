タグ_get_asset_ids

この関数を使うと、与えられたタグが割り当てられている、与えられた型のassets をすべて取得することができます。単一のタグ（string ）か、配列の各項目がタグ（string ）である配列、およびチェックするasset の型を指定します。asset の型は、以下の定数のいずれかである必要があります。

定数 商品説明
asset_object 指定された名前は、オブジェクトを指します。
asset_sprite 指定された名前は、スプライトを指します。
asset_sound 与えられた名前は、音を指します。
asset_room 与えられた名前から room.
asset_tiles 与えられた名前は、タイルセットを指します。
asset_path 指定された名前は、パスを参照している。
asset_script 与えられた名前から script.
asset_font 与えられた名前は、フォントを指します。
asset_timeline 与えられた名前は、時間軸を意味します。
asset_shader 指定された名前は、シェーダーを指します。
asset_animationcurve アニメーションカーブを意味する。
asset_sequence 与えられた名前は、Sequenceを指します。

この関数は配列を返します。配列の各項目は、asset のインデックス値 1 つになります。指定したタグを持つ型がassets に存在しない場合は、空の配列が返されます。

構文です。

tag_get_asset_ids(tags, asset_type);

論証 タイプ 商品説明
tags 文字列または文字列の配列 A single asset tag string or an array with various asset tags.
asset_type アセットタイプ 定数 An asset type constant (listed above)

リターンです。

Array

例

var _paths = tag_get_asset_ids("enemy", asset_path);
var _num = irandom(array_length(_paths) - 1);
path_start(_paths[_num], 1, path_action_reverse, false);

上記のコードでは、"enemy "というタグを使って、そのタグを持つすべてのpath assets 、ランダムに1つを選んで、コードを実行しているインスタンスに割り当てています。
