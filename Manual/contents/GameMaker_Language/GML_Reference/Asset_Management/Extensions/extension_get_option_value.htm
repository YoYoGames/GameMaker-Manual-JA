extension_get_option_value

この関数は、与えられたオプションの値を、与えられた名前のextension で返します。

提供されたextension の名前が無効な場合、この関数は未定義を返します。

構文です。

extension_get_option_value(extension_name, option_name);

論証 タイプ 商品説明
extension_name 文字列 The name of your extension asset as a string
option_name 文字列 The name of the option to read

リターンです。

Any

例

var _enabled = extension_get_option_value("MyExtension", "enabled");

このコードは、MyExtensionの enabledオプションの値を取得しますextension.
