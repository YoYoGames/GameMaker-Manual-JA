extension_get_options

この関数は、与えられた名前を持つstruct のすべてのオプションとその値を含むextension を返します。

提供されたextension の名前が無効な場合、この関数は未定義を返します。

構文です。

extension_get_options(extension_name);

論証 タイプ 商品説明
extension_name 文字列 The name of your extension asset as a string

リターンです。

Struct

例

var _options = extension_get_options("MyExtension");
var _enabled = _options.enabled;

このコードは、struct のオプションextension を取得し、そこから値を読み取ります。
