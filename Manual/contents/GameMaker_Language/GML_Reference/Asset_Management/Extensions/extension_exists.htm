extension_exists

この関数は、指定された名前のextension が存在するか（true）、存在しないか（false）を知ることができる。

構文です。

extension_exists(extension_name);

論証 タイプ 商品説明
extension_name 文字列 The name of your extension asset as a string

リターンです。

Boolean

例

if (extension_exists("MyExtension"))
{
    var _options = extension_get_options("MyExtension");
    var _enabled = _options.enabled;
}

このコードは、MyExtension extension が存在するかどうかをチェックし、存在する場合はそのオプションをstruct として取得し、そこから値を読み取ります。
