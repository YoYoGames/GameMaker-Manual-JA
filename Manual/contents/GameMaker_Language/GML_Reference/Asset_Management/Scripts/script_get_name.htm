script_get_name

この関数は、指定されたscript の名前を文字列として返す。この名前は、GameMaker のメインウィンドウのscript Browser でAsset に指定された名前である。scripts の詳細については、 Script Editor を参照してください。

構文です。

script_get_name(scr);

論証 タイプ 商品説明
scr スクリプトアセット The index of the script that you want to get the name of.

リターンです。

String

例

scr_name = script_get_name(Help_File);

上記のコード例では、指示されたscript インデックスの名前が変数 "scr_name" に格納されます。
