script_exists

この関数は、与えられたインデックスを持つscript またはスクリプト関数が存在するかどうかに応じて、 true または false を返します。これはstringではなく、asset (script で表示されるような) またはIDE script で定義されているようなscript 関数の一意のインデックスを保持するasset 名であることに注意してください (メソッド変数では機能しないことに注意してください)。scripts の詳細については、「 Script エディタ」を参照してください。

構文です。

script_exists(scr);

論証 タイプ 商品説明
scr スクリプトアセット The script index that you want to check.

リターンです。

Boolean

例

script[0] = -1;
script[1] = AI_Left;
script[2] = AI_Right;
var script_num = choose(0, 1, 2);
if script_exists(script[script_num])
{
    script_execute(script[script_num]);
}

上記の例では、2つのscript 関数と値を配列に追加し、乱数を取得して、それを使って実行するscript 関数を選択します。ただし、-1 を選択した場合は、何も起こりません。
