script_execute_ext

この関数は、関数script_execute()と同様に動作しますが、呼び出される関数/スクリプトに必要な引数を含む配列を指定することができます。関数にオプションの引数を 2 つ渡すと、引数を取得する配列のオフセットと 配列から使用する引数の数を指定できます (array_length - offset の最大値である必要があります)。

構文です。

script_execute_ext(scr, array_args, [offset], [num_args]);

論証 タイプ 商品説明
scr スクリプトアセットまたはスクリプトファンクション The function/script that you want to call
array_args アレイ The array containing the arguments for the function/script
[offset] リアル OPTIONAL 引数配列へのオフセット
[num_args] リアル OPTIONAL 使用する引数の数(オフセット以降)

リターンです。

Variable (Will depend on the return value from the function/script being called)

例

script_execute_ext(move_inst, move_array, floor(random(4)), 1);

上記のサンプルコードでは、 script_execute_ext 、4つの引数の配列を与えて、与えられた関数を呼び出しますが、そのうちの1つだけをランダムに使用します。

NOTE内蔵の機能とは併用できません。
