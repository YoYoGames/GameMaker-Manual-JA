tileset_get_name

この関数は、指定されたtile set asset の名前を文字列として返す。この名前はGameMaker のメインウィンドウのtile set Browser でAsset に指定されている名前である。これはあくまで string であり、tile set を直接参照することはできないことに注意してください - そのためには、タイルセットのインデックスが必要です。しかし、このstring を使って、返されたstring と、関数 asset_get_index().

構文です。

tileset_get_name(index);

論証 タイプ 商品説明
index タイルセットアセット The index of the tile set to get the name of.

リターンズ

String

例

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_get_tileset(_m);
tileset_name = tileset_get_name(_t);

上記のコードは、与えられたレイヤーのインデックス名tile set を取得し、そのリターンstring を変数 tileset_name に格納します。
