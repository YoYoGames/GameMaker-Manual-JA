tileset_get_uvs

この関数は、指定されたtexture のtile set の UV 座標などのデータを、texture ページに配列で返します。この関数は、以下の 8 つの要素からなる配列を返します。

  • [0] = 左
  • [1] = トップ
  • [2] = 右
  • [3] = 下
  • [4] =asset コンパイラがtile set の左側から切り捨てたピクセルの量
  • [5] =asset コンパイラがtile set 上面から切り捨てたピクセル数
  • [6] =tile set ページに保存された元のtexture 幅のピクセルデータの正規化されたパーセンテージです。
  • [7] =tile set ページに保存されたオリジナルのtexture の高さからのピクセルデータの正規化されたパーセンテージです。

この配列は、他の描画関数、特に3D作業や2Dプリミティブ関数、シェーダー関数で使用することができます。

構文です。

tileset_get_uvs(tileset);

論証タイプ 商品説明
tileset The index of the tile set to use.

リターンズ

(1D, 8 elements)

例

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

上記のコードでは、与えられたtile set の UV 座標をローカル配列に格納し、その値をインスタンス変数に代入しています。
