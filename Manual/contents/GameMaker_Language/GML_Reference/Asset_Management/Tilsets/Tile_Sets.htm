タイルセット

アセットブラウザで作成されたtile set assets を操作する場合、以下の機能が利用できます。これらの関数は、asset 自体を扱うためのものであり、tiles やtile のマップがゲーム内に存在する場合の作業用ではありません（ルームのセクションの適切なレイヤー関数を参照してください）。

使用できる機能は次のとおりです。

  • tileset_get_name
  • tileset_get_texture
  • tileset_get_uvs
