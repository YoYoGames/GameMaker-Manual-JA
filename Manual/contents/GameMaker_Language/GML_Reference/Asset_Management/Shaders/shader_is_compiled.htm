シェーダーがコンパイルされているかどうか

この関数は、実行時にshader が正常にコンパイルされたかどうかを確認します。コンパイルに成功した場合、この関数は true を返し、失敗した場合、 false を返します。この関数はゲーム開始時に使用し、ゲームを実行しているプラットフォームでshaders が正常にコンパイルされていることを確認する必要があります（特にWindows では、DX9 でShader Level 2.0 を使用し、shader Level 3.0 を使用する新しいバージョンではないコンピュータもあるかもしれません）。

もし、shader がコンパイルされていない状態で shader_set()はゲームがクラッシュするので、単純な GLSL ESshaders 以外を使用する場合は、常に何らかのチェックをする価値があります。

構文です。

shader_is_compiled(shader);

論証タイプ 商品説明
shader The shader to check.

リターンです。

Boolean

例

global.GFX = false;
if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
{
    global.GFX = true;
}

上記のコードでは、グローバル変数に false を設定し、チェック対象のshaders の両方が正しくコンパイルされていれば、 true に設定されます。
