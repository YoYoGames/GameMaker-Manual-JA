shader_set_uniform_f_array

この関数で、shader の定数を設定し、値の配列を保持することができます。この関数を使用するには、事前に関数 shader_get_uniform()また、事前に配列を初期化しておく必要があります。

注意：すべてのユニフォームは、関数 shader_set()を呼び出す前、そして shader_reset().

構文です。

shader_set_uniform_f_array(handle, array);

論証 タイプ 商品説明
handle シェーダーユニフォームハンドル The handle of the shader constant to set.
array アレイ A previously initialised array of floating point values.

リターンです。

N/A

例

shader_set(shader_Glass);
tex_array[0] = 0.5;
tex_array[1] = 0.1;
tex_array[2] = 0.25;
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f_array(shader_params, tex_array);
draw_self();
shader_reset();

上記のコードでは、shader 定数「u_vParams」のハンドルを取得し、その定数を与えられた配列に設定します。
