shader_set_uniform_matrix

この関数を使うと、shader 定数の値（または複数の値）を現在の変換行列（Matrix 関数で設定したもの）に設定できます。この定数の「ハンドル」は、事前に関数 shader_get_uniform().

注意：すべてのユニフォームは、関数 shader_set()を呼び出す前、そして shader_reset().

構文です。

shader_set_uniform_matrix(handle);

論証 タイプ 商品説明
handle シェーダーユニフォームハンドル The handle of the shader constant to set.

リターンです。

N/A

例

shader_set(shader_Glass);
shader_matrix = shader_get_uniform(shader_glass, "u_vMatrix");
shader_set_uniform_matrix(shader_matrix);
draw_self();
shader_reset();

上記のコードでは、shader 定数「u_vMatrix」のハンドルを取得し、その定数を現在の変換行列に設定します。
