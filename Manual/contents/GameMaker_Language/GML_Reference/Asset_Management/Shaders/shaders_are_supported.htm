シェーダがサポートされているかどうか

この関数は、選択されたターゲットプラットフォームがshaders をサポートしているかどうかをチェックし、サポートしている場合は true を、サポートしていない場合は false を返します。Android では、プロジェクトにshader リソースが定義されていない場合、デバイスがshaders をサポートしているかどうかに関係なく、この関数は常に false を返すことに注意することが重要です。

構文です。

shaders_are_supported();

リターンです。

例

global.GFX = false;
if (shaders_are_supported())
{
    if (shader_is_compiled(sh_glass) && shader_is_compiled(sh_warp))
    {
        global.GFX = true;
    }
}

上記のコードでは、グローバル変数に false を設定し、プラットフォームがshaders をサポートし、チェック対象のshaders の両方が正しくコンパイルされていれば、 true に設定されることになります。
