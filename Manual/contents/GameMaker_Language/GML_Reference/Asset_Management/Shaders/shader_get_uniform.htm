shader_get_uniform

shader 自体の中でshader 定数の値を変更することはできないので、利用可能なユニフォームセット関数のいずれかを使用して、shader を呼び出す前に設定する必要があります。しかし、これを行うには、まずこの関数を呼び出して、変更したいshader 定数の「ハンドル」を取得する必要があります。

注意：shader は2つの独立したプログラム (vertex と fragment ) から構成されていますが、この関数はその2つを区別せず、どちらかの定数からshader のハンドルを返します。

構文です。

shader_get_uniform(shader, uniform);

論証 タイプ 商品説明
shader シェーダーアセット The index of the shader to use.
uniform 文字列 The shader constant to get the handle of (a string).

リターンです。

Shader Uniform Handle

例

shader_params = shader_get_uniform(shd_glass, "u_vRefractColour");

上記のコードでは、shader の定数 "u_vRefractColour" のハンドルを取得します。
