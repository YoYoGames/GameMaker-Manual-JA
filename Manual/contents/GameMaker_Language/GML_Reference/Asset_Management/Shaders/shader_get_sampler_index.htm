shader_get_sampler_index

shader 自体の中でshader サンプラーの値を変更することはできないので、利用可能なユニフォームセット関数のいずれかを使用して、shader を呼び出す前に設定する必要があります。しかし、これを行うには、まずこの関数を呼び出して、設定したいshader サンプラーの「ハンドル」を取得する必要があります。

注 shader は2つの独立したプログラム（vertex と fragment ）から構成されていますが、この関数はその2つを区別せず、どちらか一方のサンプルshader のハンドルを返します。

構文です。

shader_get_sampler_index(shader, uniform);

論証 タイプ 商品説明
shader シェーダーアセット The index of the shader to use.
uniform 文字列 The shader sampler to get the handle of (a string).

リターンです。

Shader Sampler Handle

例

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

上記のコードでは、"GlassShader "というインデックスを持つshader 内のサンプラーのハンドルを取得し、そのshader 定数を与えられたsprite texture に設定します。
