シェーダーカレント

この関数は、現在レンダリングに使用されているshader のインデックス ID 値を返し、shader が使用されていない場合は -1 を返します。

構文です。

shader_current();

リターンです。

Shader Asset or -1 (if no shader is assigned)

例

if (shader_current() == -1)
{
    shader_set(sh_warp);
}

上記のコードでは、現在のshader が何であるかを確認し、-1 を返した場合（shader が使用されていない）、shader が設定されます。
