sequence_instance_override_object

この関数を使うと、object で使われているsequence のインスタンスをすべて別のものにオーバーライド（置き換え）できます。sequence インスタンスstruct ID (sequence インスタンスがroom で作成されたとき、またはroom レイヤー関数のいずれかを使用して返されたもの -ここを参照) と、オーバーライドしたいobject のasset インデックス (object ブラウザで定義されたもの) を指定します。最後に、object をオーバーライドするobject として使用するsequence インデックスまたはインスタンス ID を指定します (インスタンス ID を指定すると、そのインスタンスが作成されたobject をオーバーライドとして使用するだけになります)。これはsequence のインスタンスに対してのみ可能であり（sequence objects は不可）、sequence の再生が始まる前に行う必要があることに注意してください。

構文です。

sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id);

論証 タイプ 商品説明
sequence_instance_struct シーケンスインスタンス構造体 The sequence instance struct to modify.
object_id オブジェクト・アセット The object index of the object within the sequence to override.
instance_or_object_id オブジェクトのアセットまたはインスタンスID The object index or instance ID to use to override the sequence objects.

リターンです。

N/A

例

var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
var _seq_inst = layer_sequence_get_instance(_seq);
sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer);

上記のコードは、与えられたレイヤー上に新しいsequence インスタンスを作成し、object "obj_Trees_Winter" のインスタンスをすべてobject "obj_Trees_Summer" のインスタンスに置き換えるように変更します。
