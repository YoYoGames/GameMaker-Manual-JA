シーケンス作成

この関数で、新しいsequence object を作成し、それにトラックを追加することができます。この関数は構造体を返し、この構造体にアクセスして、作成した新しいsequence をセットアップすることができます。このstruct の内容については、このページで詳しく説明しています。sequence object struct を使って、sequence のレイヤーにroom のインスタンスを作成することができます。 layer_sequence_create().この方法でsequence objects を作成する場合、関数 sequence_destroy()さもないと、メモリーリークが発生してゲームの動作が遅くなり、最終的にはクラッシュする可能性があります。

構文です。

sequence_create();

リターンです。

Sequence Object Struct

例

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

上記のコードでは、新規にsequence object struct を作成し、その再生長さとloop モードを設定しています。
