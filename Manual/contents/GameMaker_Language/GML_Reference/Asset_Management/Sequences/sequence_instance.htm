sequence_instance

これはゲーム内の各インスタンス（object ）に作成されるインスタンス変数の一部である組み込み変数です。 undefinedインスタンスがsequence によって制御されている場合、この変数にはインスタンスを制御しているSequence のシーケンスインスタンス構造体が格納されます。これは読み取り専用の変数で、変更することはできません。

なお、この変数は、制御しているSequence が終了すると、そのSequence 要素がまだ存在していてもundefined となり、そのstruct 要素が再生されると再びSequence を取得する。

構文です。

sequence_instance

リターンです。

Sequence Instance Struct

例

if (in_sequence)
{
    sequence_instance.speedScale = 2;
}

上記のコードでは in_sequence変数が true (インスタンスがsequence によって制御されていることを意味します) であれば、そのSequence の速度スケールを 2 に変更します。
