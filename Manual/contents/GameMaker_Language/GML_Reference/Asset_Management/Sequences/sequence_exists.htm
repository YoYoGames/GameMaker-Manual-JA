シーケンスが存在する

この関数を使うと、sequence object が存在するかどうかを確認することができます。sequence object struct (この関数が返す)のどちらかを指定します。 sequence_create()または sequence_get()) またはsequence ID (関数 layer_sequence_get_sequence()またはsequence インスタンスstruct プロパティ sequence 、またはasset ブラウザのインデックスから）、この関数はsequence object が存在する場合は true を、存在しない場合は false を返します。

構文です。

sequence_exists(sequence_struct_or_id);

論証 タイプ 商品説明
sequence_struct_or_id Sequence AssetまたはSequenceObject Struct. The sequence to check for, can be the asset reference itself or its Sequence object struct.

リターンです。

Boolean

例

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

上記のコードでは、与えられたsequence object が存在するかどうかを確認し、存在する場合は破棄しています。
