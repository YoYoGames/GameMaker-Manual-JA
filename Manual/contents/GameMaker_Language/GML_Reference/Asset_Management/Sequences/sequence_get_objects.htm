sequence_get_objects

この関数は、指定したobject 内でインスタンスが生成されているすべてのsequence インデックスの配列を取得することができます。sequence object struct (この関数が返すもの) のいずれかを指定します。 sequence_create()または sequence_get()) またはsequence ID (関数 layer_sequence_get_sequence()またはsequence インスタンスstruct プロパティ sequence から取得し、この関数は配列を返します。 object_indexobjects がインスタンスを作成するために使用している異なるsequence のためのものです。

構文です。

sequence_get_objects(sequence_struct_or_id);

論証 タイプ 商品説明
sequence_struct_or_id Sequence AssetまたはSequenceObject Struct. The sequence object struct or ID to get the objects from.

リターンです。

Array

例

var _seq = sequence_get(seq_Logo);
obj_array = sequence_get_objects(_seq);

上記のコードは、struct sequence に対してobject を取得し、使用するobjects を取得し、その配列を変数に格納しています。
