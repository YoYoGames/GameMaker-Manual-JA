キーフレームデータ構造

トラックのキーフレーム構造を定義するとき、それに関連するデータも定義する必要があります。これは異なるチャンネルで構成され、各チャンネルは単純にKeyframe データStruct となります。チャンネルは、keyframe データを設定するトラックの種類によって、特定のタイプのデータを与えることができます。

Keyframe データStruct には、以下の変数が含まれます。

SequenceKeyframe データ構造
バリアブル タイプ 商品説明
channel リアル keyframe データを適用するチャンネルを指定します。0 から始まる正の整数値です。「position」または「scale」のパラメータトラックを作成する場合、keyframes 、非常に特殊なチャンネル値を使用する必要があることに注意する必要があります。チャンネル 0 は X ポジションまたは X スケール、チャンネル 1 は Y ポジションまたは Y スケールです。
spriteIndex スプライトアセット トラックに使用するsprite asset のインデックスです。このプロパティは、タイプ seqtracktype_graphic のトラックにのみ有効で、取得または設定することができます。
soundIndex サウンドアセット トラックに使用するオーディオasset のインデックスです。このプロパティは、タイプ seqtracktype_audio のトラックでのみ利用可能で、取得または設定することができます。
playbackMode シーケンスオーディオキー定数 サウンドの再生モード。このプロパティは、タイプ seqtracktype_audio のトラックにのみ有効で、取得または設定することができます。このプロパティは、この下の表で与えられている定数のいずれかでなければなりません。
curve アニメーションカーブ構造 このプロパティはanimation curve struct を必要とし（詳細はこちらをご覧ください）、タイプseqtracktype_real のトラックでのみ利用可能です。カーブstruct が使用されていない場合、このプロパティの値は -1 になります。
value リアル このプロパティは、単にkeyframe データチャンネルに関連付けられた値で、 curve struct が供給されない場合、 seqtracktype_real タイプのトラックでのみ利用可能です。これは、例えば、「位置」パラメータトラックの中に置かれた場合、トラックのXまたはY位置とすることができます。
colour アレイの リアルs このプロパティは、keyframe の色値を [A, R, G, B] のフォーマットで配列として返す（設定されている場合は必須）ものである。これは、 seqtracktype_colour タイプのトラックでのみ利用可能です。ここで、0 は HEX 値 #00 に、1 は HEX 値 #FF に対応します (Sequences Editor のカラートラックのカラーピッカーに表示される 0 - 255)。
sequence SequenceObject Struct このプロパティは、シーケンスobject 構造体を返し（設定されている場合は要求）、タイプ seqtracktype_sequence のトラックに対してのみ利用可能である。
objectIndex オブジェクト・アセット このプロパティは、object インデックスを返し（設定されている場合は要求）、タイプ seqtracktype_instance のトラックに対してのみ利用可能です。
events 配列の 文字列s このプロパティは、keyframe データstruct に関連するイベントと ブロードキャストメッセージにアクセスすることを可能にする。このプロパティは取得または設定することができ、取得時にはstrings の配列が返され、設定時にはstrings の配列を指定する必要がある。イベントの詳細については、「シーケンスイベントとモーメント」のセクションを参照してください。このプロパティは、タイプ seqtracktype_message のトラックでのみ利用可能です。
event 方法 このプロパティは、以下のものを返す（あるいは設定することができる）。 methodに関連付けられているkeyframe データstruct 。メソッドが指定されていない場合、またはメソッドを削除したい場合は、このプロパティは-1でなければなりません。 このプロパティは、 seqtracktype_moment 型のトラックでのみ利用可能です。

変数 playbackMode には、以下の定数のいずれかを指定することができます。

シーケンスオーディオキー定数
定数 商品説明 価値

テキストトラックデータ

struct は、テキストトラック(seqtracktype_text)に割り当てられると、以下の追加変数を含みます。

SequenceKeyframe データ構造
バリアブル タイプ 商品説明
text 文字列 これは、トラック上に描かれるテキストstring 。
wrap ブーリアン これは、テキストを折り返すか (true) しないか (false) を示すブール値である。
alignmentH シーケンステキストアライメント定数 これはテキストの水平方向のアライメントを表し、以下のいずれかになります。 シーケンステキストアライメント定数は以下に示す。
alignmentV シーケンステキストアライメント定数 これはテキストの垂直方向のアライメントで シーケンステキストアライメント定数は以下に示す。
fontIndex フォントアセット これは フォントアセットテキストトラックで使用されます。

alignmentH は、以下のいずれかの定数です。

シーケンステキストアライメント定数(alignmentH)
定数 商品説明
seqtextkey_left テキストは枠の左側に配置されます
seqtextkey_right テキストは枠の右側に整列されます
seqtextkey_center テキストはフレーム内の水平方向の中央に配置されます。
seqtextkey_justify テキストはフレーム内で両端揃えになります

alignmentV は、以下のいずれかの定数です。

シーケンステキストアライメント定数(alignmentV)
定数 商品説明
seqtextkey_top テキストはフレームの上部に整列されます
seqtextkey_bottom テキストはフレームの下に整列されます
seqtextkey_middle テキストはフレーム内の垂直方向の中央に配置されます。
