SequenceObject Struct

A SequenceObject Structは、struct Sequence から取得したasset に付けられた名前です。Sequence のすべてのデータは、このstruct にruntime で格納され、Sequence で作成されたroom のすべての要素から参照されます。 このことは、この構造体の変数（またはその入れ子のstructs 内部）が変更されると、元のSequence が（ゲームを閉じて再び開くまで）変更されて、その既存のSequence のすべての要素の影響を受けることを意味します。

この構造体は、以下のプロパティを持ち、変更することが可能である。

playbackSpeed リアル sequence の再生速度を指定する。 playbackSpeedType に応じて、frames-per-second または frames-per-game-frame として解釈される（後述）。この値は、取得または設定することができます。
playbackSpeedType スプライト速度定数 これは、 playbackSpeed の解釈方法を指定するもので、この値を取得または設定することができます。

備考

なお，sequence ブラウザで作成されたasset のプロパティにアクセスしたい場合は，まず，関数 sequence_get()をasset インデックスで検索し、sequence object struct を取得します。また、このsequence struct に加えられた変更は、このsequence asset の他のすべてのインスタンスにもその変更が加えられることを意味し、たとえ game_restart() 関数を呼び出したとしても、ゲームが実行されている限り、その変更は維持されることに注意してください。
