トラック構造

トラックstruct は、トップレベルのアセットトラックにも、サブトラックのパラメータトラックにも使用でき、トラックの動作は、名前と タイプの2つで定義されます。トラック構造体は、関数 sequence_track_new()で、Sequence Instance StructのactiveTracks プロパティ、またはSequencesequence Struct のtracks プロパティを使用してassets Object から取得することができます。

トラックstruct で利用できるプロパティは次のとおりです。

シーケンストラック構造

名称

name プロパティには、以下のいずれかの文字列を指定することができます。

  • "position" -seqtracktype_real タイプのトラックに設定された場合、このトラックの内容が位置データとして使用されることを示す。
  • "scale" -seqtracktype_real タイプのトラックに設定された場合、このトラックのコンテンツがスケーリングデータとして使用されるべきであることを示します。
  • "rotation" -seqtracktype_real タイプのトラックに設定された場合、このトラックの内容が回転データとして使用されることを意味します。
  • "blend_multiply" または "image_blend" -seqtracktype_colour タイプのトラックに設定された場合、このトラックのコンテンツがカラー・マルチプリケーション・データとして使用されるべきであることを示します。このようなトラックのkeyframe データは、[A, R, G, B]というフォーマットの4つの値の配列であるべきです。各成分の値は 0 から 1 で表され、0 は HEX 値 #00 に、1 は HEX 値 #FF（Sequences Editor のイメージブレンドトラックのカラーピッカーに示される 0 ～ 255）に対応することに注意してください。
  • "image_speed" -seqtracktype_real タイプのトラックに設定された場合、このトラックの内容がイメージスピード値として使用されるべきことを示します。
  • "image_index" -seqtracktype_real タイプのトラックに設定された場合、このトラックの内容がイメージインデックス値として使用されることを示します。
  • "image_angle" -seqtracktype_real タイプのトラックに設定された場合、このトラックの内容が画像角度の値として使用されるべきことを示します。


    テキストトラックのパラメータ名 （テキストトラックでのみ使用されます）。
  • "frameSize" -seqtracktype_real トラックに設定された場合、このトラックの内容がテキストのフレームサイズとして使用されるべきであることを示します。このトラックの各keyframe は 0 と 1 の 2 つのチャンネルを持っているはずで、0 にはフレームの X サイズが、1 には Y サイズが格納されます。
  • "characterSpacing" -seqtracktype_real トラックでも使用され、テキストの文字間隔として使用される。このトラックの各keyframe は、スペーシング値を含む単一のチャンネルを持つ必要があります。
  • "lineSpacing" -seqtracktype_real トラックでも使用され、テキストの行間として使用される。このトラックの各keyframe は、スペーシング値を含む 1 つのチャンネルを持つ必要があります。
  • "paragraphSpacing" -seqtracktype_real トラックでも使用され、テキストの段落間隔として使用されます。このトラックの各keyframe には、スペーシングの値を含む 1 つのチャンネルがあるはずです。
