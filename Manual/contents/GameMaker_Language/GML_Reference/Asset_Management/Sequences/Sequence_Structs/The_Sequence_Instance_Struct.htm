Sequence インスタンス構造

sequence 、room に配置されると、レイヤー上の要素として配置され、この要素にはシーケンスインスタンスが含まれます。このsequence インスタンスは、その構造体に以下のプロパティを持つことになる。

シーケンスインスタンス構造体
バリアブル タイプ 商品説明
sequence SequenceObject Struct これは、struct インスタンスの生成元であるシーケンスオブジェクト sequence の ID です（詳細はこちらを参照）。このstruct を取得または設定することで、インスタンスが参照しているベースsequence を変更することができる。
headPosition リアル これは、sequence インスタンスの現在の再生ヘッド位置（フレーム単位）です。この値は取得または設定できますが、現在定義されている再生領域より大きいまたは小さい値に設定すると、sequence に設定されている再生の種類によって異なる効果があることに注意してください。 詳細は、以下を参照してください。 layer_sequence_headpos()を参照してください。
headDirection シーケンス方向定数 sequence インスタンスの現在の再生方向です。この値は、次の表に示す定数で取得・設定できます（ただし、レイヤー関数 layer_sequence_get_headdir()と layer_sequence_headdir()を使えば、struct に直接アクセスしなくても、この値を取得したり設定したりできることに注意してください)。
speedScale リアル このプロパティは、再生スピードのスケールを取得または設定するために使用されます。スピードスケールは乗数で、1がデフォルトの再生スピードで、1より小さい値は再生が遅くなり、1より大きい値は速くなる。例えば、0.5の値は半分の再生スピードで、2の値は倍の再生スピードとなる。なお、レイヤー関数 layer_sequence_get_speedscale()と layer_sequence_speedscale()を使えば、struct に直接アクセスしなくても、この値を取得したり設定したりすることができます。
paused ブーリアン このプロパティを確認することで、sequence が一時停止されているかどうかを確認でき、一時停止されている場合は true に、そうでない場合は false になります。このプロパティは読み取り専用ですが、レイヤー関数 layer_sequence_pause()を使用して再生を一時停止し、必要に応じて layer_sequence_play()必要であれば、再生を再開することができます。また、このプロパティはレイヤー関数で確認することもできます。 layer_sequence_is_paused()struct のプロパティを直接チェックするのではなく、レイヤー関数を使ってこのプロパティをチェックすることもできます。
finished ブーリアン このプロパティで、sequence の再生が終了したかどうかを確認できます。再生が終了している場合は true を、そうでない場合は false を返します。このプロパティは読み取り専用で、loop または ping-pong に設定されていないトラックに対してのみ、 true を返すことになります。このプロパティは、レイヤー関数を使用して確認することもできます。 layer_sequence_is_finished()struct のプロパティを直接チェックするのではなく、レイヤー関数を使ってこのプロパティをチェックすることもできます。
elementID 配列要素ID このプロパティは、sequence 要素の ID を保持します。この ID は、room エディターでレイヤーに関連付けられた単純な識別値で、適切なレイヤー関数で使用すると、sequence が割り当てられているレイヤーを見つけたり、sequence の特定のプロパティをstruct を処理せずに変更したりすることができるようになります。
activeTracks アレイの シーケンスアクティブトラック構造s このプロパティは、structs の各アセットトラック(グラフィック、sequence 、オーディオなど) の現在の状態に関する情報を含む "evaluation"sequence のアレイを保持します。これらはトップレベルのトラックであり、返される配列は読み取り専用です。各配列エントリで返されるasset トラックstruct の内容については、以下のセクションを参照してください。

シーケンス方向定数
定数 商品説明 価値
seqdir_right sequence は、左から右へインクリメンタルな順序でフレームを再生します。 1
seqdir_left sequence は、右から左へ向かって減数法でフレームを再生します。 -1

activeTracks 構造

このプロパティは Create イベントで インスタンス に既に存在しますが、最初の Begin Step イベント以降にのみトラックで埋められます。従って、この に初めて値を見つけるのは、 Step Event の最初の反復の時です。sequence struct Sequence Sequence struct Sequence

以下のリストには、アクティブなトラックstruct にアクセスする際に利用できる可能性のあるプロパティが含まれています (Sequence Instancestruct の activeTracks 配列に含まれています、上記で説明しました)。各asset トラックstruct は、トラックが使用するasset のタイプに応じて、以下のいずれかのプロパティを持つことができます（読み取り専用として指定されている場合を除き、これらのプロパティはすべて変更できることに注意してください）。返される値はすべて、現在の再生ヘッド位置のものです。

シーケンスアクティブトラック構造
バリアブル タイプ 商品説明
matrix マトリクスアレイ これは、トラック（sprite 、インスタンス、sequence トラック）を描画するために使用される変換行列です。
posx リアル asset 現在のプレイヘッド位置のトラックのX軸に沿ったsequence の位置（すべてassets ）。
posy リアル asset 現在のプレイヘッド位置のトラックについて、sequence の Y 軸に沿った位置（すべてassets ）です。
scalex リアル 現在の再生ヘッド位置のトラック（asset 、インスタンス、sequence トラック）の X 軸に沿ったsprite のsequence のスケールです。
scaley リアル asset 現在の再生ヘッド位置のトラックのY軸に沿ったsequence のスケール（sprite 、インスタンスとsequence のトラック）。
xorigin リアル トラックのasset の X 原点（すべてassets ）。
yorigin リアル トラックのasset の Y 原点（すべてassets ）。
width リアル トラック（asset 、インスタンス、sprite トラック）のsequence の幅（ピクセル単位）です。
height リアル トラック（asset 、インスタンス、sprite トラック）のsequence の高さ（ピクセル単位）です。
imageindex リアル 現在の再生ヘッド位置のasset のトラック上のsequence の画像インデックス（sprite とインスタンストラック）です。
imagespeed リアル 現在の再生ヘッド位置のasset のトラック上のsequence のイメージ速度（sprite とインスタンストラック）。
colourMultiply カラー 現在の再生ヘッド位置のasset のトラックのsequence の色乗算値 (sprite, インスタンスとsequence のトラック) です。この値は、[A, R, G, B]のフォーマットを持つ4つのARGB値の配列になります。各コンポーネントの値は0と1の間で表され、0はHEX値#00に、1はHEX値#FFに対応します（Sequences エディタのカラーマルチトラックのカラーピッカーに表示される0〜255）ので、注意してください。
emitterIndex オーディオエミッターID このトラック（オーディオトラック）に属するオーディオエミッターのインデックスを指定します。
gain リアル 現在の再生ヘッド位置のトラックのゲイン（音量）（オーディオトラック）。
pitch リアル 現在の再生ヘッド位置でのトラックのピッチ（オーディオトラック）。
frameSizeX リアル テキストフレーム（テキストトラック）の水平サイズです。
frameSizeY リアル テキストフレーム（テキストトラック）の垂直方向のサイズです。
characterSpacing リアル 文字間隔の値（テキストトラック）。
lineSpacing リアル 行間の値（テキストトラック）。
paragraphSpacing リアル パラグラフスペーシングの値（テキストトラック）。
track シーケンストラック構造 このトラックが基づいているTrack Struct(allassets,read-onlyvalue)です。
parent シーケンスインスタンス構造体 トラックの親sequence インスタンス ID (allassets,read-onlyvalue)。
activeTracks アレイの シーケンストラック構造s これは、structs トラックが含む各パラメータトラックの評価asset の配列です（すべてassets 、読み取り専用の値です）。配列内の各struct の内容は、「パラメータトラック構造」セクションに記載されています。
