キーフレームストラクチャー

各トラック（トラック構造体についてのページで定義されています）には、1つ以上のkeyframe 構造体が割り当てられ（トラックstruct の keyframes プロパティから取得できます）、それぞれ以下のプロパティを持つことになります。

SequenceKeyframe Struct
バリアブル タイプ 商品説明
frame リアル timeline のkeyframe に沿った位置（フレーム単位）。 デフォルト値は 0。
length リアル keyframe の長さ。デフォルト値は 1 で、それ以上の値を設定すると、keyframe が参照する track プロパティは、与えられた長さの間、初期値のまま維持されます。なお、 stretch プロパティは、 true に設定すると、これをオーバーライドします。
stretch ブーリアン このプロパティを true に設定すると、keyframe はトラックの次のkeyframe のいずれか、または最後のkeyframe の場合はトラックの終端まで伸びる。この値は取得または設定することができ、デフォルト値は false です。
channels 配列の SequenceKeyframe データ構造s このプロパティは、トラックのチャンネルのキーフレームデータ構造体のリストにアクセスすることができる。このプロパティを取得する場合、keyframe データstructs の配列が返され、このプロパティを設定する場合、keyframe データstructs の配列が指定されるべきである。
