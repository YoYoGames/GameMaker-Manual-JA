シーケンスデストロイ

この関数を使うと、動的に生成されたsequence object を破棄することができます。sequence object struct (この関数が返す)のいずれかを指定します。 sequence_create()) またはsequence ID (この関数によって返される) layer_sequence_get_sequence()またはsequence インスタンスstruct プロパティ sequence から）。この関数は、動的に作成されたsequence が不要になったときに、それに関連するメモリを解放するために使用する必要があります。

構文です。

sequence_destroy(sequence_struct_or_id);

論証 タイプ 商品説明
sequence_struct_or_id Sequence AssetまたはSequenceObject Struct. The sequence object struct or ID to destroy

リターンです。

N/A

例

if sequence_exists(my_seq)
{
    sequence_destroy(my_seq);
}

上記のコードでは、与えられたsequence object が存在するかどうかを確認し、存在する場合は破棄しています。
