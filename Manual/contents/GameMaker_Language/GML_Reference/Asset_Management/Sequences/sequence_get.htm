sequence_get

この関数を使用すると、Object sequence のインデックスからSequenceasset Structを取得することができます。asset ブラウザで定義されているsequence のasset インデックスを指定すると、この関数はアクセス可能なsequence object struct を返します。sequence が存在しない場合や指定したasset インデックスがsequence でない場合は -1 を返します。

構文です。

sequence_get(sequence_index);

論証 タイプ 商品説明
sequence_index シーケンスアセット The sequence asset as defined in the Asset Browser.

リターンです。

Sequence Object Struct or -1

例

var _seq = sequence_get(seq_Logo);
seq.loopmode = seqplay_pingpong;

上記のコードはsequence object "seq_Loop" からstruct sequence object を取得し、loop のsequence モードを設定します（以降に作成されるすべてのsequence インスタンスはこのloop モードを使用するようになります）。
