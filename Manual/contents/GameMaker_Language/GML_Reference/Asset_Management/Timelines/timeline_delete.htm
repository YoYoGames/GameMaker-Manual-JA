タイムライン削除

この関数を使うと、ゲームから任意の時間軸を削除することができます。この時間軸がGameMaker Asset Browser で作成された場合、ゲームから完全に削除されるので、この関数を使った後にインデックス付きの時間軸を再度呼ぼうとするとエラーが発生することに注意してください。時間軸を動的に作成する場合は timeline_add()この関数は、メモリリークによるゲームのスローダウン（最終的にはクラッシュ）を防ぐために、時間軸が不要になった瞬間に使用する必要があります。

構文です。

timeline_delete(ind);

論証タイプ 商品説明
ind The index of the time line to delete.

リターンです。

例

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

上記のコードは、グローバル変数 "tl "にインデックスされた時間軸が存在するかどうかをチェックし、存在する場合はその時間軸を削除しています。
