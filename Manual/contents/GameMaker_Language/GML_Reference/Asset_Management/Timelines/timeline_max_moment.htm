タイムラインマックスモーメント

この関数は、timeline が最終アクションを実行した瞬間の値を返す。つまり、20秒間隔で8つのアクションが配置されたtimeline がある場合、この関数はそのtimeline の最後のアクションのモーメント値、つまり160を返すことになります。この関数は、timeline が実行中に最後のアクティブな瞬間を過ぎたかをテストする良い方法です。なぜなら、timeline の位置は、さらなるアクションがあるかどうかにかかわらず、ゲームの各ステップでインクリメントし続けるからです。

構文です。

timeline_max_moment(ind);

論証タイプ 商品説明
ind The index of the timeline to get the last moment of.

リターンです。

例

if timeline_position > timeline_max_moment(timeline_index)
{
    timeline_running = false;
}

上記のコードでは、現在のtimeline の位置と最大アクティブモーメントを照合し、その方が大きければtimeline を停止します。
