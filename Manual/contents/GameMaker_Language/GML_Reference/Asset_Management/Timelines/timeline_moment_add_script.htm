timeline_moment_add_script

この関数を使うと、時間軸上の任意の「瞬間」にスクリプト関数を動的に追加して、Timelines 。「瞬間」は1ゲームティック（またはステップ）に相当します。この方法で、新しい時間軸を作ることができます。 timeline_add()関数を使って、任意の時点で異なるビヘイビアを追加したり、以前に作成した時間軸リソースを新しいビヘイビアで単純に変更したりすることができます。また、この関数を使用してTimeline Browser に存在するasset Asset を変更すると、このtimeline を使用しているすべてのインスタンスが影響を受け、その変更はゲーム終了まで続きます（ game_restart() を呼んでも、変更はリセットされません）。

構文です。

timeline_moment_add_script(ind, step, script);

論証 タイプ 商品説明
ind タイムラインアセット The index of the timeline to add a moment to.
step リアル The moment (step) to add to.
script スクリプト機能 The index of the script function to add into the moment.

リターンです。

N/A

例

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
    timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
    i += room_speed * 60;
}

上記のコードでは、新しい時間軸を作成し、そのインデックスを変数「global.tl」に格納します。そして、1分間隔でランダムに選ばれた3つのscripts を時間軸に追加する。
