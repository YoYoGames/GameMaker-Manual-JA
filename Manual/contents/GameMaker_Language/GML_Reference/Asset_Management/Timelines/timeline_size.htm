タイムラインサイズ

この関数を使うと、timeline のアクティブなモーメントの総数を得ることができます (「アクティブな」モーメントとは、コードまたはGML Visual が追加されているモーメントのことです)。を使用して動的なtimelines を作成する際に便利です。 timeline_moment_add_script()と timeline_moment_clear()関数で、与えられたtimeline が正しい数のアクティブなモーメントを持っているか、まったく持っていないかを確認することができるからです。

構文です。

timeline_size(ind);

論証タイプ 商品説明
ind The index of the timeline get the information from.

リターンです。

例

if timeline_size(global.tl) == 0
{
    timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

上記のコードは、与えられたtimeline のサイズをチェックし、0 を返した場合 (すなわち、timeline にアクティブな瞬間がない場合)、グローバル変数 "tl" で指定されたscript の中のランダムな瞬間に使用するtimeline 関数を追加しています。
