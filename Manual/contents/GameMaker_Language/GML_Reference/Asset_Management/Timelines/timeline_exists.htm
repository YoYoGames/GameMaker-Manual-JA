タイムラインが存在する

この関数を使うと、時間軸が存在するか（ true を返す）、しないか（ false を返す）を確認することができます。これは特に、Timelines を使って動的に作成するときに便利です。 timeline_add()しかし、初期化されていない変数（時間軸のインデックスに割り当てられる）を探すと、エラーが発生することに注意してください。

構文です。

timeline_exists(ind);

論証タイプ 商品説明
ind The index of the time line to check for.

リターンです。

例

if timeline_exists(global.tl)
{
    timeline_delete(global.tl);
}

上記のコードは、グローバル変数 "tl "にインデックスされた時間軸が存在するかどうかをチェックし、存在する場合はその時間軸を削除しています。
