タイムラインポジション

この変数は、時間軸が現在置かれている位置（瞬間）を保持する。この値を変更することで、時間軸の一部をスキップしたり、一部を繰り返したり、時間軸を最初からやり直したりすることができる。

構文です。

timeline_position;

リターンです。

(single precision floating point value)

例

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

上記のコードは、インスタンスが時間軸を実行しているかどうかを確認し、実行していない場合は、割り当てられた時間軸を最初の瞬間にリセットして、開始します。
