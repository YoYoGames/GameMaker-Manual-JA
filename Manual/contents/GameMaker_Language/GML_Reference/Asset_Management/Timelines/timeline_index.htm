タイムラインインデックス

この変数は、現在インスタンスに関連するtimeline のインデックスを保持する。これを特定の時間軸に設定するとその時間軸を使用することができ、-1に設定するとインスタンスの時間軸の使用を停止することができる（インスタンスに時間軸が定義されていない場合は、-1も返される）。これは時間軸を開始させないことに注意してください。 timeline_running.

構文です。

timeline_index;

リターンです。

Timeline Asset

例

if timeline_exists(global.tl)
{
    timeline_index = global.tl;
}

上記のコードは、コードを実行しているインスタンスに、変数「global.tl」にインデックスされた時間軸が存在する場合、その時間軸を割り当てます。
