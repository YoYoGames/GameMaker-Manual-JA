タイムラインループ

この変数は、時間軸がループしているか（true）、していないか（false）を返す。この変数を変更することで、switch ループのオン・オフが可能で、負の時間軸速度で動作する（時間軸の位置が0以下になると、最後に定義した瞬間から再スタートする）。

構文です。

timeline_loop;

リターンです。

例

if !timeline_loop
{
    timeline_loop = true;
}

上記のコードは、あるインスタンスに現在割り当てられている時間軸がloop になるかどうかを確認し、ならない場合は true を設定します。
