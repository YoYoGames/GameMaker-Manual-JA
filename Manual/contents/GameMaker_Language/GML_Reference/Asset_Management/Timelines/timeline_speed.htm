timeline_speed

通常、各ステップで時間軸上の位置は1ずつ増加しますが、この変数を異なる値に設定することでこの量を変更することができます。実数（たとえば0.5とか2.4とか）を使うことができ、値が1より大きいと、同じタイムステップで複数の瞬間が起こることになります（これらはすべてタイムラインに定義されたのと同じ順序で実行されるので、アクションがスキップされることはありません）。

構文です。

timeline_speed

リターンです。

(single precision floating point value)

例

if !timeline_running
{
    timeline_running = true;
    timeline_speed = 0.5;
}

上記のコードは、インスタンスが関連する時間軸を実行しているかどうかをチェックし、実行していない場合は、時間軸を半分の速度で実行するようにします。
