タイムラインランニング

この変数は、割り当てられた時間軸の現在の状態を保持し、実行中であれば true を、実行中でなければ false を返す。また、この変数に true または false を設定すると、いつでも時間軸を開始したり停止したりできる。停止した時間軸はリセットされないので、後で再び開始すると、停止した時点から開始されることに注意する必要がある。

構文です。

timeline_running;

リターンです。

例

if !timeline_running
{
    timeline_position = 0;
    timeline_running = true;
}

上記のコードは、インスタンスが時間軸を実行しているかどうかを確認し、実行していない場合は、割り当てられた時間軸を最初の瞬間にリセットして、開始します。
