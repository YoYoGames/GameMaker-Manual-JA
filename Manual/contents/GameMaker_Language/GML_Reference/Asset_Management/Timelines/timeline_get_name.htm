timeline_get_name

この関数は，時間軸の名前をstring として取得するために用いることができる。 timeline_add()関数を使用すると、返される名前は次のようになります。 ここで、"N"は時間軸の番号(0から始まる)です。これはあくまで string であり、timeline を直接参照することはできないことに注意してください。 そのためには、タイムラインインデックスが必要です。string しかし、このstring を使って、タイムラインインデックスを取得することは可能です。 asset_get_index().

構文です。

timeline_get_name(ind);

論証タイプ 商品説明
ind The index of the time line to check the name of.

リターンです。

例

var _str = timeline_get_name(timeline_index);
show_debug_message("Current timeline = " + _str);

上記のコードは、現在割り当てられているtimeline の名前を取得し、コンソールに出力しています。
