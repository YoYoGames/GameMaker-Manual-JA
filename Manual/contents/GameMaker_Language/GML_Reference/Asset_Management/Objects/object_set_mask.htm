object_set_mask

この関数を使うと、特定のobject の mask_index を設定することができます (マスクの詳細については、TheObject Editor を参照してください)。これは、マスクが変更された後に作成されたこのobject のすべてのインスタンスが、この新しいマスクで作成されることを意味し、room にすでにあるインスタンスは影響を受けないかもしれません。これを-1に設定すると、sprite のマスクが削除され、sprite のobject に定義されたマスクがデフォルトになります (sprite が選択されていない場合はマスクがありません)。

これはインスタンス機能ではありません。個々のインスタンスのマスク・インデックスを設定することができます。 mask_indexobject この関数を使用してobject のマスクインデックスを変更しても、現在room にあるすべてのインスタンスはそのまま残り、この関数を呼び出した後に作成されたインスタンスだけが新しいマスクで開始されます。

構文です。

object_set_mask(index, spr);

論証 タイプ 商品説明
index オブジェクト・アセット The index of the object to change.
spr スプライトアセット The new sprite to use as the object's mask.

リターンです。

N/A

例

if (global.level == 10)
{
    object_set_mask(obj_Platform, spr_mask_10);
}

上記のコードは、グローバル変数の値をチェックし、object "obj_Platform" のマスクインデックスが10に等しければ変更します。
