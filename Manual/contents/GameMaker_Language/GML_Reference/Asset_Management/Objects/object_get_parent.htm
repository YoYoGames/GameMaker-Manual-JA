object_get_parent

この関数は、指定されたobject に割り当てられた親のobject インデックスを取得します。さもなければ、object に割り当てられた親がないことを示す -100、またはチェック対象のobject が存在しない場合は -1 が返されます。親についての詳細は、「オブジェクトエディタ」のセクションを参照してください。

構文です。

object_get_parent(obj);

論証タイプ 商品説明
obj The index of the object to check

リターンです。

Object Asset

例

par = object_get_parent(object_index);

上記のコードでは、コードを実行しているインスタンスのobject インデックスの親を取得し、その戻り値を変数「par」に格納します。
