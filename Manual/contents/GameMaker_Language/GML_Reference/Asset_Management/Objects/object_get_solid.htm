object_get_solid

この関数は、チェックしているobject が "solid "のフラグを立てているかどうかを教えてくれます。solidobject は、従来のコリジョンシステムを使用した場合（つまり、物理ワールドがオフの場合）、特別なコリジョンイベントを発生させます。これはインスタンス関数ではないことに注意してください!object つまり、同じobject のソリッドインスタンスとノーマルインスタンスを持つことができますし、その逆もしかりです。個々のインスタンスのソリッドflag を設定するには solidインスタンス変数

構文です。

object_get_solid(obj);

論証タイプ 商品説明
obj The index of the object to check.

リターンです。

例

if (!solid) && (object_get_solid(object_index))
{
    solid = true;
}

上記のコードは、実行しているインスタンスが固体であるかどうかをチェックし、さらにインスタンスのobject インデックスをチェックして、固体としてフラグが立っているかどうかを確認します。インスタンスが固体でないにもかかわらず、object インデックスが固体と判定された場合、そのインスタンスの "solid" を true に設定します。
