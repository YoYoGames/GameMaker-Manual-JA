object_is_ancestor

この関数は、あるobject が他のobject の祖先であるかどうかを調べるのに使うことができます。したがって、両方の引数はインスタンス IDではなく object のインデックスである必要があります。また、最初の引数は常に、2番目の引数の子であるかどうかを調べたいobject で、これは常に祖先（親）として調べたいobject となります。

構文です。

object_is_ancestor(obj, par);

論証 タイプ 商品説明
obj オブジェクト・アセット The object that is being checked as the child.
par オブジェクト・アセット The object that is being checked as the ancestor (parent).

リターンです。

Boolean

例

if object_is_ancestor(object_index, obj_Enemy)
{
    instance_destroy();
}

上記のコードでは、実行中のインスタンスがobject "obj_Enemy" の子であるかどうかをチェックし、もしそうであれば破棄しています。
