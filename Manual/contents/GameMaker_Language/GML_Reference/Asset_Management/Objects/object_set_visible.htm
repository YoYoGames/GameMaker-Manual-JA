object_set_visible

flag この関数を使用すると、特定のobject を可視または非可視にすることができます（「可視」flag の詳細については、「オブジェクトエディタ」のセクションを参照してください）。これは、visible が変更された後に作成されたこのobject のすべてのインスタンスが、この新しい状態で作成されることを意味し、既にroom にあるインスタンスには影響がない場合があります。

これはインスタンス機能ではありません。を使用して、個々のインスタンスの可視flag を設定することができます。 visibleobject flag IDEまた、この関数でflag の visibleobject を true に変更しても、room にあるインスタンスはすべてそのままで、この関数を呼び出した後に作られたインスタンスだけが visible が true になった状態でスタートします。

構文です。

object_set_visible( index, vis );

論証 タイプ 商品説明
index オブジェクト・アセット The index of the object to change.
vis ブーリアン The new visibility of the object (true=visible, false=invisible）。

リターンです。

N/A

例

if !object_get_visible(obj_Block)
{
    object_set_visible(obj_Block, true);
}

上記のコードは、flag "obj_Block" の可視object をチェックし、もし false ならば true のフラグを立てるように変更します。
