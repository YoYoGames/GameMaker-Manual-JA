オブジェクトセットスプライト

この関数を使用すると、特定のsprite のobject インデックスを設定することができます。これは，sprite_index が変更された後に作成されるこのobject のすべてのインスタンスが，この新しい sprite_index で作成されることを意味します。ただし，すでにroom にあるインスタンスは影響を受けないかもしれません。

これはインスタンス機能ではありません。を使って、個々のインスタンスのsprite インデックスを設定することができます。 sprite_indexsprite object この関数を使用してsprite のobject インデックスを変更しても、現在room にあるすべてのインスタンスはそのまま残り、この関数を呼び出した後に作成されたインスタンスだけが新しいsprite で開始されます。

構文です。

object_set_sprite( index, spr );

論証 タイプ 商品説明
index オブジェクト・アセット The index of the object to change.
spr スプライトアセット The sprite to assign to the object.

リターンです。

N/A

例

switch (room)
{
    case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
    case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
    case rm_end: object_set_sprite(obj_Player, spr_casual); break;
}
instance_create_layer(32, 32, "Instances", obj_Player);

上記のコードは、object "obj_Player"sprite のインデックスを、このコードを実行しているインスタンスが現在いるroom に応じて異なる値に設定します。そして、「obj_Player」のインスタンスを作成します。
