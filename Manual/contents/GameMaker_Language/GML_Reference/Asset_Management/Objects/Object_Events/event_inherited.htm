イベント継承

この関数は、インスタンスの親であるobject の現在のイベントを呼び出します。通常、インスタンスが親object を持つ場合、自動的に親と同じすべてのイベントをinherits します。しかし、（たとえば）親object に create イベントがあり、子object にそれを追加すると、子object のすべてのインスタンスは親object にあるイベントではなく、あなたが追加した新しい create イベントを実行することになります。同じタイプの親イベントobject と子イベントobject の両方を使用する必要がある場合は、この関数を使用してください。この関数は、子イベントが含む残りのコードやアクションを実行する前に、親イベントobject を実行します。

構文です。

event_inherited();

リターンです。

例

event_inherited();
switch (sprite_index)
{
    case spr_Enemy_1: dmg += 2; break;
    case spr_Enemy_4: dmg -= 1; break;
    case spr_Enemy_10: dmg +=10; break;
}

上記のコードでは、継承した親イベントを呼び出し（その中で変数 "dmg" を初期化しています）、次に変数 "dmg" を変更しています。このコードを実行しているインスタンスに親が指定されていない場合、 dmg が定義されていないため、「不明な変数」エラーが発生します。
