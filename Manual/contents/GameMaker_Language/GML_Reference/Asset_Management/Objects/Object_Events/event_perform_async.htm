この関数は、GameMakerに用意されている非同期イベントのうち、任意の1つを実行するために使用する。Asyncイベントの定数（下の表に示す）と、呼び出されたAsyncイベントで利用可能なDSマップを変数 async_load に与えます。

定数 商品説明

ev_async_system_event システムイベント

非同期イベントは、以下の方法で呼び出すことができます。 event_perform().

なお、第2引数で指定されたDSマップは、Asyncイベント実行後に関数によって自動的に破棄されるため、手動で破棄する必要はない。

構文です。

event_perform_async(type, ds_map);

論証 タイプ 商品説明
type 非同期イベントタイプ定数 The type of event to perform (see the table above).
ds_map DSマップID The DS map to use as async_load を呼び出されたイベントで使用します。

リターンです。

N/A

例

var _map = ds_map_create();

_map[? "id"] = "custom_async_event";
_map[? "result"] = true;
_map[? "data"] = { a: 13, b: 16 };

event_perform_async(ev_async_social, _map);

上記のコードでは、DSマップを作成し、非同期イベントで読み込まれるカスタムエントリをそのマップに入力します。そして、呼び出されたイベントの async_load として渡された新しく作成されたマップを使用して、Async Social イベントを実行します。
