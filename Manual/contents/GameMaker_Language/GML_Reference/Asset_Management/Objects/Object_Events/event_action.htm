event_action

この読み取り専用の変数は、現在実行中のアクションのインデックスを返す。以前のバージョンのGameMaker では 0 から始まる。 しかし、GameMaker ではこの変数は廃止 されている。レガシーサポートのために残されており、実行速度を向上させるためにすべてのアクションが連結されるため、常に0を返すことになる。

構文です。

event_action

リターンです。

Real

例

num = event_action;

上記のコードでは、現在のアクション番号を変数 "num "に格納しています。
