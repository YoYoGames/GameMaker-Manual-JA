イベント番号

このリード・オンリー変数は現在呼び出されているイベントの番号を返します。この番号は実際にはイベントの「サブ・イベント」を参照しています。特定のサブイベントで使用可能な定数の全リストは、以下を参照してください。 event_perform()また、メインイベントそのものを知る必要がある場合には event_type.

構文です。

event_number;

リターンです。

Event Number Constant

例

switch (event_number)
{
    case ev_step_normal: show_debug_message("Step event!"); break;
    case ev_game_start: show_debug_message("Game Start""); break;
    case ev_room_start: show_debug_message("Room Start!"); break;
}

上記のコードは、script から呼び出され、現在どのイベントがトリガーされているかを知らせるデバッグメッセージを表示するために使用することができます。
