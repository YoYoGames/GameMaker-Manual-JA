イベントパーフォームオブジェクト

と同じ働きをします。 event_perform()ただし、今回は別のobject からのイベントを指定することができます。しかし、これは文字通りイベントのすべてのコードを実行するだけで、ゲームは手動でトリガーするために何も修正しないことに注意してください。例えば、キーボードを押すイベントを選択した場合、イベントはトリガーされますが、関連するキーは押されたとは認識されません。また、アラームイベントを実行した場合、アラームカウンターは-1に設定されず、カウントダウンが継続されます。この関数が必要とする定数の完全なリストは、この関数のページで確認できます。 event_perform().

注意この関数で呼び出されたイベントのアクションは、現在のインスタンスに適用され、指定されたobject のインスタンスには適用されません。

構文です。

event_perform_object(obj, type, numb);

論証 タイプ 商品説明
obj オブジェクト・アセット The object that should have its event triggered.
type イベントタイプ定数 The type of event to perform.
numb 実数またはイベント番号定数 The specific event, if one is necessary (otherwise, just use 0).

リターンです。

N/A

例

event_perform_object(obj_Player, ev_keypress, ord("W"));

これは、現在のインスタンスのobject "obj_Player" から Keyboard Check Pressed > W キーに関連するイベントを実行します。
