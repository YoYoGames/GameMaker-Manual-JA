object_get_persistent

この関数は、チェックしているobject が「persistent」であるとフラグが立っているかどうかを教えてくれます。永続的なobject とは、そのインスタンスが明示的に破棄されない限り、room からroom へと引き継がれるようなものを指します。これはインスタンス関数ではないことに注意してください!object したがって、同じobject の永続的なインスタンスと非永続的なインスタンスを持つことができ、その逆も可能です。個々のインスタンスの永続的なflag を設定するには persistentインスタンス変数で指定します。

構文です。

object_get_persistent(obj);

論証タイプ 商品説明
obj The index of the object to check.

リターンです。

例

if (!persistent) && (object_get_persistent(object_index))
{
    persistent = true;
}

上記のコードは、実行しているインスタンスが永続的かどうかをチェックすると同時に、インスタンスのobject インデックスをチェックし、永続的であるとフラグが立っているかどうかを確認します。インスタンスが永続的でないにもかかわらず、object インデックスが永続的であると判定された場合、そのインスタンスに対して "persistent" を true にセットします。
