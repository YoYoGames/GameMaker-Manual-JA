object_get_name

この関数は、指定されたobject の名前を文字列として返す。この名前は、GameMaker のメインウィンドウのobject Browser でAsset に指定されている名前です。これはあくまで文字列であることに注意してください。 object を直接参照することはできません。そのためには、オブジェクトのインデックスが必要です。しかし、このstring を使って、返されたstring と関数を使用してオブジェクトのインデックスを取得することはできます。 asset_get_index().

構文です。

object_get_name(obj);

論証タイプ 商品説明
obj The index of the object to check.

リターンです。

例

str = object_get_name(object_index);

上記のコードは、コードを実行しているインスタンスのobject インデックス名を取得し、その戻り値を変数 "str "に格納します。
