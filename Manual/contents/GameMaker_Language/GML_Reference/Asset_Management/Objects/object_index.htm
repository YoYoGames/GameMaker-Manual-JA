オブジェクトインデックス

この読み取り専用変数は、インスタンスの作成元であるobject のインデックスを返します。これは、object の名前とは異なり、string で見つけることができます。 object_get_name()この関数はインデックス番号を返すが、これはGameMakerが作成時に全てのobject に割り当てる一意の値である。

構文です。

object_index;

リターンです。

Object Asset

例

obj_name = object_get_name(object_index);

上記のコードでは、 object_index 、現在のインスタンスが作成されたobject の名前を検索するために使用します。
