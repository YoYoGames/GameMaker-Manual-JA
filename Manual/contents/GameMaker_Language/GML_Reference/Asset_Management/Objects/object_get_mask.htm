object_get_mask

この関数は、チェックしているobject がマスクのインデックスを持っているかどうかを調べ、持っている場合はそのマスクのインデックス（これはsprite asset ）、持っていない場合は -1 を返します。これはインスタンス関数ではないことに注意してください!マスクを持たないobject があっても、同じobject のインスタンスはマスクを持つことができますし、その逆もまた可能です。個々のインスタンスのマスクインデックスを設定するには mask_indexインスタンス変数があります。

構文です。

object_get_mask(obj);

論証タイプ 商品説明
obj The index of the object to check

リターンです。

Sprite Asset

例

if (mask_index != object_get_mask(object_index))
{
    mask_index = object_get_mask(object_index);
}

上記の例では、インスタンスのマスクインデックスとインスタンスのobject_indexのマスクを照合しています。もしそれらが同じでなければ、object インデックスが持っているマスクと同じものをインスタンスに割り当てます。
