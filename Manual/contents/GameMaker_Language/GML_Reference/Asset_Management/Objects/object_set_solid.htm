object_set_solid

この関数を使用すると、特定のflag をソリッドであるかどうかobject することができます（「ソリッド」flag の詳細については、「オブジェクトエディタ」のセクションを参照してください）。これは、solid が変更された後に作成されるこのobject のすべてのインスタンスが、この新しい状態で作成されることを意味します。一方、すでにroom にあるインスタンスは影響を受けないかもしれません。

これはインスタンス機能ではないことに注意してください! solid 変数を使って個々のインスタンスの solidflag を設定することができるので、object のflag がIDE で false に設定されていても、10個のインスタンスがすべて solid と判定されます。この関数を使ってflag の solidobject を true にしても、現在room にあるすべてのインスタンスはそのままで、この関数を呼んでから作られたインスタンスだけが solid が true になって開始されます。

構文です。

object_set_solid( index, solid );

論証 タイプ 商品説明
index オブジェクト・アセット The index of the object to change.
solid ブーリアン The new solidity of the object (true=固体、 false=固体でない）。

リターンです。

N/A

例

if !object_get_solid(obj_Block)
{
    object_set_solid(obj_Block, true);
}

上記のコードでは、flag "obj_Block" の solidobject をチェックし、false ならば true のフラグを立てるように変更しています。
