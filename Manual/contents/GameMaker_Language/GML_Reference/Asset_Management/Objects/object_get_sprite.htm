object_get_sprite

この関数は、チェックしているobject がsprite を持つかどうかを調べ、持つ場合はそのsprite のインデックスを返し、持たない場合は -1 を返します。これはインスタンス関数ではないことに注意してください!object を持たないsprite があっても、同じobject のインスタンスには があり、その逆もあり得ます。また、異なるsprites を持つことさえあり得ます。個々のインスタンスにsprite を設定するには sprite_indexインスタンス変数です。

構文です。

object_get_sprite(obj);

論証タイプ 商品説明
obj The index of the object to check

リターンです。

Sprite Asset

例

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
    sprite_index = _spr;
}

上記の例では、インスタンスの sprite_index を、インスタンスの object_index のsprite と照合します。もしそれらが同じでなければ、sprite インデックスのものと同じobject をインスタンスに割り当てます。
