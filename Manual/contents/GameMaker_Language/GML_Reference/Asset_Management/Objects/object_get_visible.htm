object_get_visible

この関数は、チェックしているobject が、"visible"（描画イベントを実行する）と判定されたのか、そうでない（描画イベントを実行しない）のか、教えてくれます。これはインスタンス関数ではないことに注意してください!object したがって、同じobject の可視インスタンスと不可視インスタンスを持つことができますし、その逆も可能です。個々のインスタンスの可視性を設定するには visibleインスタンス変数で指定します。

構文です。

object_get_visible(obj);

論証タイプ 商品説明
obj The index of the object to check.

リターンです。

例

if (!visible) && (object_get_visible(object_index))
{
    visible = true;
}

上記のコードでは、実行中のインスタンスが可視状態かどうかを確認すると同時に、インスタンスのobject インデックスをチェックして、可視状態であるとのフラグが立っているかどうかも確認します。インスタンスが表示されていないにもかかわらず、object インデックスがオンになっている場合、そのインスタンスに対して "visible" を true に設定します。
