object_get_physics

この関数は、あなたがチェックしようとしているobject が「物理が有効」であるとのフラグが立っているか - その場合は true 、 - そうでない場合は -false を返すかを教えてくれます。

構文です。

object_get_physics(obj);

論証タイプ 商品説明
obj The index of the object to check.

リターンです。

例

if object_get_physics(object_index)
{
    phy_active = true;
}

上記のコードは、実行中のインスタンスに対して、作成元のobject が物理演算可能かどうかを確認し、可能な場合はインスタンスの物理シミュレーションを起動します。
