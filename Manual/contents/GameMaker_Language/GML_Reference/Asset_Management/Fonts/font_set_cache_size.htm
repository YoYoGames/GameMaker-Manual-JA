フォントキャッシュサイズを設定する

画像のブレンドやカラーリングを行う場合、HTML5では実行ファイルと同じように動的に行うことができません。そのため、GameMakerはブレンドされた画像のコピーを一時的に保存し 必要なときに呼び出す。この関数は、与えられたfont のブレンドコピーを、古いものが上書きされるまでにいくつキャッシュできるかを設定します。デフォルト値は 4 です。

注意 sprite fonts の場合は、sprites と同等の関数を使用する必要があります。 sprite_set_cache_size().

構文です。

font_set_cache_size(ind, max);

論証タイプ 商品説明
ind The index of the font to change the cache size of.
max The maximum number of cached copies of the font that can be stored.

リターンです。

例

font_set_cache_size(fnt_MainMenu, 2);

これにより、変数 "fnt_MainMenu "でインデックスされたfont のcache font が2枚に設定されます。
