font_add_enable_aa

を使用して追加したfonts のアンチエイリアス（AA）の有効／無効を設定することができます。 font_add().この関数はfonts を追加する前に呼び出す必要があり、この関数を true に設定すると AA が有効になり、 false に設定すると無効になります。デフォルトでは、AAが有効になっています。この関数が呼ばれる前に追加されたfonts には効果がないことに注意してください。また、この関数はfont が追加されたときに一度だけ呼ばれ、font が使用されているすべての描画/ステップで呼ばれる必要はないことに注意してください。

構文です。

font_add_enable_aa(enable);

論証タイプ 商品説明
enable Whether to enable (true) または無効化 (false) AA を追加した場合fonts 。

リターンです。

例

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上記のコードは、追加されたfonts のアンチエイリアシングの状態をチェックし、有効になっていない場合は有効にしています。
