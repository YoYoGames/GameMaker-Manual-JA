font_get_italic

この関数で、任意のfont asset がイタリック体 flag を持つかどうかをチェックすることができます。もしあれば、この関数は true を返し、そうでなければ、 false を返します。

構文です。

font_get_italic(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

例

if font_get_italic(fnt_Main)
{
    draw_set_font(fnt_Main);
}

これは、アクティブな描画font がfont のプロパティでイタリックとして設定されている場合、 fnt_Main に設定します。
