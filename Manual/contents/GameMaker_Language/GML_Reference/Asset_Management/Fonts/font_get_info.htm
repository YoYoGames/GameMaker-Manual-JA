font_get_info

この関数は、与えられたfont の情報を取得するために使用する。font のインデックス（アセットブラウザで追加されたasset またはfont で追加された runtime ）を指定すると、この関数は次の変数を持つ構造体を返します。

フォント情報構造
変数名 データ型 商品説明
ascenderOffset リアル ベースラインからfont の上端までの最大オフセット（単位：ピクセル）
size リアル font のおおよそのサイズ（ピクセル単位）。
spriteIndex スプライトアセット sprite スプライトから作成された場合はfont のインデックス，そうでない場合は -1
texture ポインタ font がsprite から作成された場合は -1、そうでない場合はフォントのtexture ID を指定します。
name 文字列 フォントの名前
bold ブーリアン true font が太字の場合、それ以外の場合は false (下記のNOTE を 参照)
italic ブーリアン true font がイタリックの場合、それ以外の場合は false (下記のNOTE を 参照)
glyphs フォントグリフ構造 struct の各グリフの情報を含むfont (詳細は後述します)

注 bold とitalic 変数は、ユーザーの設定をfont に反映するだけなので、デフォルトで太字や斜体に表示されるfonts の場合は正確ではないかもしれません。

与えられたfont が存在しない場合，この関数は undefined を返します．

また、これらの変数の値や、 glyphs struct に含まれる変数を変更しても、font のレンダリング方法は変わらないので、この情報は読み取り専用と考えるべきであることに注意してください。

グリフ構造

返されたstruct の変数 glyphs は、それ自体がstruct となり、font に含まれる各グリフの情報を含んでいます。 この構造体の各変数はグリフの文字名となり、そのグリフの情報を含むstruct を保持することになります。こ れ ら の入れ子にな っ たstructs は次の よ う な形式を想像で き ます （ メ イ ン のstruct か ら 始ま る ）。

{
    glyphs:
    {
        A: {},
        B: {},
        C: {},
        // ...other glyphs
    },
    // ...other font info
}

次のコードのように、グリフの文字名を使って、その情報をstruct から取得することができます。

var _font_info = font_get_info(Font1);

var _info_A = _font_info.glyphs[$ "A"]; // $ is a struct accessor

show_debug_message(_info_A);

上のコードは、「A」文字のグリフstruct を取得し、それを出力ログにプリントしています。また、string ("A") をstring を保持する変数に置き換えることで、その変数を通してruntime で任意のグリフの情報を取得することができます。

グリフに対して返されるstruct は、以下の変数を含む。

リターンです。

Font Info Struct (or undefined)

例

var _info = font_get_info(my_font);

if (_info != undefined && _info.bold)
{
    image_xscale *= 2;
    image_yscale *= 2;
}

このコードは、変数 my_font に格納されているfont の情報を取得する。返された値がundefined でなく 、与えられたfont が太字の場合、インスタンスのスケールは 2 倍になる。
