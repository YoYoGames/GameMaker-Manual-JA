フォント追加

この関数は、Included Filesに存在するfont ファイルからゲームにfont を追加するために使用することができます。font のサイズ（ポイント数）、font を太字にするか斜体にするか、また、含める文字の範囲も定義することができます。 -1 この関数はインデックス値を返しますが、これは変数に格納されるべきもので、今後このfont を参照するすべてのコードで必要になります。

なお、font ファイルは、 *.ttf または*.otf のいずれかでなければなりません。これは、アジアやアラビア語のような非標準のfonts を追加する場合にも便利です。fonts.

警告!I ncluded Filesにこのようなfont を含める場合、ゲームと一緒に配布するためのライセンスを取得する必要があります。

HTML5 ターゲットモジュールの場合、この関数はWeb フォントの追加にのみ使用でき、WebGL が有効でない場合にのみ動作します。WebGL が有効な場合は、一般的なfont リソース（例. draw_set_font()とともに、font ブラウザからAsset ) またはビットマップfonts ( font_add_sprite()).

重要!HTML5 をターゲットとする場合、斜体や太字はfont asset または Webfont 自体にこれらの文字が含まれている場合にのみサポートされます。これらは、ベースとなるfont から合成されることはありません。

ファイルからfont を読み込む場合、font のサイズは ピクセル単位で、最初と 最後の値は無視されます。つまり、font のすべてのグリフが追加されます。このようにして追加されたFonts は、それ自身のtexture ページに割り当てられるので、この機能を使うときは注意が必要です。描画時にtexture を交換する数が増加することになり ます。また、グリフがfonts のために定義された bounding box の外側に描画される部分を持つことがあるため、描画時にfont が若干大きく見えることがあることも知っておくとよいでしょう。 特定のfont に対して 1 フレームで描画できるグリフは約 200 という制限があることも知っておくとよいでしょう。

デフォルトでは、追加されたfonts はアンチエイリアスが有効になっています。しかし、AA の状態を確認するには、関数 font_add_get_enable_aa()という関数で、AAの有効・無効を切り替えることができます。 font_add_enable_aa().

注意事項 GameMakerに font を読み込んだら、忘れずに再度削除する必要があります。 font_delete()さもないと、メモリリークを起こし、ゲームが遅くなり、最終的にはクラッシュしてしまいます。

構文です。

font_add(name, size, bold, italic, first, last);

論証 タイプ 商品説明
name 文字列 The path of the font file in the Included Files (*.ttf または )。 *.otf
size リアル The size of the font - points for Web Fonts, pixels for file fonts.
bold ブーリアン Whether the font is bold (true) か、そうでないか (false) です。
italic ブーリアン Whether the font is italic (true) か、そうでないか (false) です。
first リアル The first character to include (if you're unsure, go for 32).
last リアル The last character to include (if you're unsure, go for 128).

リターンです。

Font Asset

例

new_font = font_add("gamefont.ttf", 24, true, true, 32, 128);

ゲームのIncluded Filesにある "gamefont.ttf" ファイルを24ptの大きさで読み込みます。font は太字と斜体になり、フォントの範囲には大文字と小文字、数字、すべての一般的な句読点が含まれます。font がロードされると、その ID が new_font 変数に格納されます。
