font_add_get_enable_aa

この機能は、fonts で追加されたアンチエイリアシング（AA）が有効かどうかを確認するために使用できます。 font_add().この関数は、AAが有効な場合は true 、有効でない場合は false を返します。AA はデフォルトで有効ですが，追加されたfonts の AA 状態は，関数 font_add_enable_aa()を追加する前に呼び出される限り、font 。

構文です。

font_add_get_enable_aa();

リターンです。

例

if !font_add_get_enable_aa()
{
    font_add_enable_aa(true);
}

上記のコードは、追加されたfonts のアンチエイリアシングの状態をチェックし、有効になっていない場合は有効にしています。
