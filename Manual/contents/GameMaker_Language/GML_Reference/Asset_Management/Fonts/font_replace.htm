フォント置換

この関数は、ゲーム内のfont を、実行中のシステムにインストールされているfonts から置き換えるために使用することができます。置き換えられるfont は、font ブラウザのAsset であってはならないことに注意してください。また、font は、関数 font_add().

この方法でfont を置き換える場合、font のサイズ（ポイント数）、font を太字にするか斜体にするか、また、含める文字の範囲を定義することができます。もしこの関数が失敗したら、黙って失敗し、与えられたfont は置換されません。

HTML5 ターゲットモジュールでは、この関数は Web フォントの置き換えにのみ使用でき、WebGL が有効でない場合にのみ動作します。WebGL が有効な場合は、一般的なfont リソース（つまり draw_set_font()とともに、font ブラウザからAsset ) またはビットマップfonts ( font_add_sprite()).

重要HTML5をターゲットとする場合、斜体や太字は、font asset または Webfont 自体にこれらの文字が含まれている場合にのみサポートされます。これらは、ベースとなるfont から合成されることはありません。

その他の（Web 以外の）ターゲットでは、この関数を使用して、ファイルからfont を置き換えることができます。ファイルはGameMaker のIncluded Files機能を使用してゲームバンドルに含まれている必要があり、 *.ttf または*.otf 形式のfont ファイルである必要があり、アジアやアラビア語などの非標準のfonts を追加するのに便利です。

TIP このようなfont ファイルをゲームに同梱する場合は、ゲームと一緒に配布するための適切なライセンスが付与されていることを確認してください。

この方法でファイルからfont を読み込む場合、font のサイズは ピクセル単位で、最初と 最後の値は無視されます。つまり、font のすべてのグリフが追加されます。この方法で追加されたFonts は、それ自身のtexture ページに割り当てられるので、この機能を使うときは注意が必要です。描画時にtexture を交換する数が増加することになり ます。また、グリフがfonts のために定義された bounding box の外側に描画される部分を持つことがあるため、描画時にfont が若干大きく見えることがあることも知っておくとよいでしょう。 特定のfont に対して1フレームで描画できるグリフは約 200 という制限があることも知っておくとよいでしょう。

注意事項 GameMakerに font を読み込んだら、忘れずに再度削除する必要があります。 font_delete()さもないと、メモリリークを起こし、ゲームが遅くなり、最終的にはクラッシュしてしまいます。

構文です。

font_replace(ind, name, size, bold, italic, first, last);

論証 タイプ 商品説明
ind フォントアセット The index of the (previously added) font to be replaced.
name 文字列 The name of the font to be used (eg 'Arial'), or the file path if the font is an included *.ttf または *.otf ファイル。
size リアル The size of the font - points for Web Fonts, pixels for file fonts.
bold ブーリアン Whether the font is bold (true) or not (false).
italics ブーリアン Whether the font is italic (true) or not (false).
first リアル The first character to include (if you're unsure, go for 32).
last リアル The last character to include (if you're unsure, go for 128).

リターンです。

N/A

例

font_replace(myfont, "Arial", 24, true, true, 32, 128);

これは、変数 "myfont" でインデックス化されたfont を、サイズが24ptで "Arial" を使い、太字と斜体の新しいfont で置き換えるものである。font の範囲には、大文字と小文字、数字、すべての一般的な句読点が含まれます。
