font_get_texture

この関数はfont texture ページのための特別なポインタを返します。この値は、他の描画関数、特にプリミティブを使用した一般的な描画や、シェーダ関数で使用することができます。返されたtexture ページに関する詳細な情報は、ここにあるさまざまな texture_ 関数を使用して得ることができます。

構文です。

font_get_texture(font);

論証 タイプ 商品説明
font フォントアセット The index of the font to use.

リターンです。

Texture

例

tex = font_get_texture(fnt_Main);

上記のコードでは、"fnt_Main "というインデックスを持つtexture のfont ポインタを取得することになります。
