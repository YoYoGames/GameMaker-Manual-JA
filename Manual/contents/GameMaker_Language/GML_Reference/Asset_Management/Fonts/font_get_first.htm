font_get_first (非推奨)

GameMakerで font を定義するとき、含める文字の範囲を定義することができます。これは、font 自体は実際にはゲームに含まれず（法的な理由から）、そのイメージは font はtexture ページに含まれており、あなたのゲームではそれが使用されます (他のグラフィックasset と同様です)。このため、使用する文字数は最小限に抑え、以下の範囲のみを指定する必要があります。 texture メモリを可能な限り最適に保つために、あなたのゲームが必要とするものです。この関数は、font asset がゲームに追加されたときに使用された開始文字（ASCII値）を見つけるために使用することができます。

注意 GameMakerはfonts にスペース文字(32)を強制するため、font の処理方法の変更により、この関数は常に32 を返します。 は32より低いコードの文字も許可します。同様の作業を行う新しい関数は、将来のアップデートで追加される予定です。

構文です。

font_get_first(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

例

num = font_get_first(fnt_Main);

上記のコードでは、"fnt_Main "にインデックスされたfont のfont 範囲の最初の文字の ASCII 値が格納されます。
