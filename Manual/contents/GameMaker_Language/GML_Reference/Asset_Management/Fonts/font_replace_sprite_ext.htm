フォント置換スプライト文字列

この関数は、以前関数を使って作成したsprite font を置き換えます。 font_add_sprite_ext().これは「スプライトストリップ」を使用し、各サブイメージは個々の記号または文字であり、イメージは関数の引数「string_map」に基づいて文字にマッピングされます。この引数はstring で、GameMakerにsprite font のサブイメージの順番を伝えると、テキストを書くときにこれらを自動的にマッピングしてくれる。例えば、"AaBbCcDdEeFfGgHh "という文字列マップがあれば、sprite font のサブイメージはこのように順番に並んでいなければならない。

また、font の間隔をプロポーショナル (true) にするかしないか (false) に設定できます。プロポーショナルfont は実際の文字の幅に基づいて間隔が決められ (従って、例えば「i」の文字は「w」の文字よりもroom が小さくなります)、非プロポーショナルfont は等幅font のようにサブイメージ幅に基づいて間隔が決められま す。最後に、書くときに各文字の間に空けるスペースを定義します。これは任意の整数値で、0 はスペースなし (プロポーショナルの場合、文字が「接触」します) です。この関数はインデックス値を返しますが、これは変数に格納する必要があります。この値は、今後このfont を参照するすべてのコードで必要になるからです。

この関数が失敗した場合，与えられたsprite font を置き換えることなく，静かに失敗します．

注意事項 sprite font この機能を使用するには、事前に font_add_sprite()というのは、これはそれを置き換えるものであり、ゲームリソースの一部として含まれているfonts には使用できないからです。

構文です。

font_replace_sprite_ext(font, spr, string_map, prop, sep);

論証 タイプ 商品説明
font フォントアセット The font to replace.
spr スプライトアセット The sprite to add a font based on.
string_map 文字列 String from which sprite sub-image order is taken.
prop ブーリアン Set as proportional font or not.
sep リアル The space to leave between each letter.

リターンです。

N/A

例

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

上記のコードは、変数 "global.Font" にインデックスされているsprite font asset を、変数 "spr_CalcFont" にインデックスされているsprite の画像に置き換えるものです。
