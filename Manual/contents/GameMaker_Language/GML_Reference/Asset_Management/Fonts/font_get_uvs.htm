font_get_uvs

この関数は、font ページにあるtexture texture の UV 座標を、以下の値で埋めた配列を返します。

  • [0] = 左
  • [1] = トップ
  • [2] = 右
  • [3] = 下

この値は、他の描画機能、特にプリミティブを使用した一般的な描画や、シェーダー機能で使用することができます。

構文です。

font_get_uvs(font);

論証タイプ 商品説明
font The index of the font to use.

リターンです。

例

var tex = font_get_uvs(fnt_Main);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

上記のコードでは、与えられた背景のUV座標をローカル配列に格納し、その値をインスタンス変数に代入しています。
