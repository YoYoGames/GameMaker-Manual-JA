font_get_fontname

この関数を使うと、与えられたfont asset の実際のシステム名を取得することができます。この関数はインデックスではなく 文字列を返し、返される名前は使用されているfont に依存します。たとえば、font Browser に "fnt_Main" というasset Asset があり、font 自体がWindows システムfont Arial であるような場合です。この場合、この関数はfont のシステム名である"Arial"を返します。Asset ブラウザに表示される名前が必要な場合は font_get_name().

上記の動作は、font を通して追加されたassets IDE にのみ適用されます。font で読み込まれたruntime （ font_add() を使用）に対して使用すると、この関数はそのpath 名ではなく、そのfont ファイルへの完全なfont を返します。

構文です。

font_get_fontname(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

String

例

fnt_Name = font_get_fontname(font0);

上記のコードでは、font リソースのシステム名を取得し、変数 "fnt_Name" にstring として格納します。
