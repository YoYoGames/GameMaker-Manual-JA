フォント置き換えスプライト

この関数を使うと、以前に作成したsprite font asset を「スプライトストリップ」で置き換えることができます。それぞれのサブイメージは個々のシンボルや文字になります。インデックスには、使用する最初の文字のUTF8マップ番号を指定します。使用する範囲は、通常のfont 、"!"で始まるものと同じですが、必要なものだけを含めるべきで、そのためには ord()関数を使って、開始文字に対応する正しい UTF8 値を取得します。

また、font の間隔をプロポーショナル (true) にするかしないか (false) に設定できます。プロポーショナルfont は実際の文字の幅に基づいて間隔が決められ (従って、例えば「i」の文字は「w」の文字よりもroom が小さくなります)、非プロポーショナルfont は等幅font のようにサブイメージ幅に基づいて間隔が決められま す。最後に、書くときに各文字の間に空けるスペースを定義できます。これは任意の整数値で、0はスペースなし（プロポーショナルなら文字が「接触」する）です。

この関数が失敗した場合，与えられたsprite font を置き換えることなく，静かに失敗します．

例えば、'.'と'0'を使って'/'を使わない場合、'/'を含めるか、使用する2文字の間に空白のサブイメージを残す必要があります。下の画像は、典型的なsprite font のレイアウトです。

構文です。

font_replace_sprite(ind, spr, first, prop, sep);

論証 タイプ 商品説明
ind フォントアセット The index of the font that shall be replaced (ie all references to this resource past this function will refer to the new font, not the original).
spr スプライトアセット The sprite to add a font based on.
first リアル The index of the first available character.
prop ブーリアン Whether white space around each letter should be removed (if false, the created font will be monospaced, otherwise it'll be like most regular fonts).
sep リアル The space between each letter, 0 meaning the right of one letter just touches the left of the next one.

リターンです。

N/A

例

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

上記のコードは、グローバル変数「Font」にインデックスされているfont asset を、外部の画像ファイルから新しいfont asset に置き換えるものです。
