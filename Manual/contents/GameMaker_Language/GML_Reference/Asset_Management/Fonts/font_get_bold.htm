font_get_bold

この関数で、任意のfont asset が太字の flag を持つかどうかを調べることができます。もし太字であれば、この関数は true を返し、そうでなければ false を返します。

構文です。

font_get_bold(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

例

if font_get_bold(fnt_Main)
{
    draw_set_font(fnt_Main);
}

これは、アクティブな図面font がfont のプロパティで太字として設定されている場合、 fnt_Main に設定されます。
