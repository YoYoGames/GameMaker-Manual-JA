フォント名

この関数は、GameMaker string Browser にfont として追加されたときに、asset につけられた名前（Asset ）を取得するために使用することができます。これはあくまで string で、font を直接参照することはできません。そのためには、フォントのインデックスが必要です。しかし、このstring を使って、返されたstring と共にフォントインデックスを取得することはできます。 機能 asset_get_index().

構文です。

font_get_name(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

例

fnt_Name = font_get_name(font0);

上記のコードは、font のリソースの名前をAsset Browser に表示されるように取得し、string として変数 "fnt_Name" に格納します。
