フォントが存在する

この関数は、指定されたインデックスを持つfont が存在するかどうかを返します。font Browser で定義されたAsset インデックスや、 font_add() などの関数で追加されたfonts インデックスを確認することができます。

構文です。

font_exists(ind);

論証タイプ 商品説明
ind Index of the font to check.

リターンです。

例

if font_exists(fnt_Main)
{
    draw_set_font(fnt_Main);
}

これにより、アクティブな描画font が存在する場合、 fnt_Main に設定されます。
