place_empty

この関数を使用すると、呼び出したインスタンスがobject の他のインスタンスまたはゲーム内のすべてのインスタンスと衝突するかどうかをチェックし、確認することができます。この衝突を正しく解決するには、コードを実行しているインスタンスが有効な衝突マスクを持っている必要があります（sprite 自体か、あるいは mask_index) で、有効なマスクを持っているインスタンスとのみ衝突を登録します。

この関数は，呼び出したインスタンスが特定の位置に配置されたときに衝突が起こらないかどうかを調べます．オプションの引数を指定すると，指定した型のインスタンスとの衝突が起こらないかどうかだけを調べることができます．オプションのobject ID が与えられない場合、チェックはroom 内のすべてのインスタンスに対して行われることに注意してください。衝突チェックは、どの種類のコリジョン マスクを選択したかによって、正確かインスタンスのバウンディング ボックスに基づくかのいずれかとなりますが、正確に衝突が動作するためには、チェック内のすべてのインスタンスが正確にコリジョン マスクを備えている必要があります。

なお、与えられたx/y座標は、チェックが行われる前に最も近い整数にフロアーされる。

構文です。

place_empty(x, y, [object_id]);

論証 タイプ 商品説明
x リアル The x position to check.
y リアル The y position to check.
[object_id] オブジェクト・アセット OPTIONAL 照合するオブジェクト。

リターンです。

Boolean

例

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
    x = mouse_x;
    y = mouse_y;
}

上記のコードは、呼び出したインスタンスがマウスと同じ位置に配置されているものとして、object "obj_Enemy" の他のインスタンスとの衝突をチェックします。衝突が検出されなかった場合、そのインスタンスの x/y 座標はマウスの座標に設定されます。
