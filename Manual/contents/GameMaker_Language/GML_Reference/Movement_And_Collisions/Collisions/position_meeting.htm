ポジション・ミーティング

この関数を使うと、ある位置が他のインスタンスまたはobject のすべてのインスタンスと衝突していないかチェックすることができます。 これを使うと、インスタンスまたはroom のobject の1点をチェックします。 このチェックは、インスタンスのバウンディングボックスに対して、またはそのインスタンスが精密衝突をチェックしている場合はインスタンスのマスクに対して実行されます。一意のインスタンスを取得する必要がある場合 イドobject を使用する必要があります。 instance_position().この関数は、特殊なキーワードである allとother も受け付けます。

構文です。

position_meeting(x, y, obj);

論証 タイプ 商品説明
x リアル The x position to check.
y リアル The y position to check.
obj オブジェクトのアセットまたはインスタンスID The object (or instance id, or the keywords "all" or "other") to check for a collision with.

リターンです。

Boolean

例

if mouse_check_button(mb_left)
{
    if (!position_meeting(mouse_x, mouse_y, all))
    {
        instance_create_layer(mouse_x, mouse_y, "Walls", obj_wall);
    }
}

上記のコードでは、マウスの左ボタンをチェックし、押されていれば、マウスのx/yの位置がいずれかのインスタンスと衝突していないかどうかをチェックします。もし何もなければ、"obj_Wall "のインスタンスが作成されます。
