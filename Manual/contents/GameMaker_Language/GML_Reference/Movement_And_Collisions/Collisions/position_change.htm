ポジションチェンジ

この関数は、ある位置で指定した地点のインスタンスとの衝突をチェックし、衝突があった場合は、衝突しているすべてのインスタンスを、選択したインスタンスに変更しますobject 。perf "引数を true に設定すると、GameMaker は見つかったインスタンスのDestroyとClean Upイベント、および新しいインスタンスのCreateイベントを実行します。また、 false に設定すると、これらのイベントを一切実行しません。Destroy、Clean Up、Create イベントを実行しない場合、create イベントで通常定義される変数を使用するインスタンスが作成されると、その変数が存在しなくなるため、ゲームがクラッシュすることに注意してください。

構文です。

position_change(x, y, obj, perf);

論証 タイプ 商品説明
x リアル The x coordinate of where to change colliding instances.
y リアル The y coordinate of where to change colliding instances.
obj オブジェクト・アセット The new object the calling object will change into.
perf ブーリアン Whether to perform that new object's Create event (true) or not (false).

リターンです。

N/A

例

position_change(200, 200, obj_Bird, true);

これは、点 (200,200) で衝突しているすべてのインスタンスを "obj_Bird" のインスタンスに変更し、その過程で "obj_Bird" の Create イベントをそれぞれ実行するものです。
