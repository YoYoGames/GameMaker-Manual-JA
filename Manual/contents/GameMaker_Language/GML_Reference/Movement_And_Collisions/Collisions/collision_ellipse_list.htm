collision_ellipse_list

この関数は collision_ellipse()関数は、一度に1つの衝突を検出する代わりに、複数の衝突を検出します。チェックする楕円形領域の左上と右下の x/y 位置と、チェックするobject を指定します。また、チェックを正確に行うか (この場合、チェックするすべてのインスタンスに正確な衝突マスクが必要です)、呼び出し側のインスタンスを含めてチェックするかどうかを設定することも可能です。

DSのリストも渡すので id呼び出したインスタンスと衝突しているインスタンスの値は、与えられたリストの末尾に追加されます。この関数を呼び出す前にリストをクリアして、この関数の呼び出しによる結果のみを含むようにすることができます。また、楕円領域の原点からインスタンスの原点までの距離に基づいてインスタンスを並べるオプションもあります。この関数は、見つかったインスタンスの数を返し、1つも見つからなかった場合は0を返します。

object インデックスの代わりにインスタンスキーワード all を指定すると、現在のroom にあるすべてのインスタンスをチェックすることができます。このインスタンスには、（ notme 引数に依存する）コードを実行しているインスタンスが含まれる場合があります。

構文です。

collision_ellipse_list(x1, y1, x2, y2, obj, prec, notme, list, ordered);

論証 タイプ 商品説明
x1 リアル The x coordinate of the left side of the ellipse to check.
y1 リアル The y coordinate of the top side of the ellipse to check.
x2 リアル The x coordinate of the right side of the ellipse to check.
y2 リアル The y coordinate of the bottom side of the ellipse to check.
obj オブジェクトのアセットまたはインスタンスID The object to check for instance collisions.
prec ブーリアン Whether the check is based on precise collisions (trueより遅い）、またはそのバウンディングボックス全般（false 、より速い）。
notme ブーリアン Whether the calling instance, if relevant, should be excluded (true) か、そうでないか (false) です。
list DSリストID The DS list to use to store the IDs of colliding instances.
ordered ブーリアン Whether the list should be ordered by distance (true) か、そうでないか (false) です。

リターンです。

Real (The number of instances found to be in collision)

例

var _list = ds_list_create();
var _num = collision_ellipse_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false);
if (_num > 0)
{
    for (var i = 0; i < _num; ++i;)
    {
        instance_destroy(_list[| i]);
    }
}
ds_list_destroy(_list);

上記のコードでは、インスタンスの呼び出し位置を中心に100ピクセルの楕円形の領域で、"obj_Enemy "のインスタンスとの衝突がないかどうかをチェックします。衝突があった場合、事前に作成されたリストがループされ、衝突した各インスタンスが破壊されます。
