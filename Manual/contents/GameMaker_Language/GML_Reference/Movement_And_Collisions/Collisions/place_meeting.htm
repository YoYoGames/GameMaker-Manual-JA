プレースミーティング

この関数を使用すると、チェック用のコードを実行したインスタンスのコリジョン マスクを使用して、ある位置で他のインスタンスまたはobject のすべてのインスタンスとの衝突をチェックできます。この関数を使うと、GameMakerにインスタンスを新しい位置に移動させ、衝突をチェックし、戻って衝突があったかどうかを教えてくれるよう頼むことになります。これは正確なコリジョンに対して機能しますが、チェックされるインスタンスとobject の両方が正確なコリジョンマスクを選択している場合のみで、それ以外の場合はバウンディングボックスコリジョンのみが適用されます。この関数は、特殊キーワードallおよびother も使用できます。衝突されるobject のユニークなインスタンスIDを取得する必要がある場合、この関数を使用する必要があります。 instance_place().

なお、与えられたx/y座標は、チェックが行われる前に最も近い整数にフロアーされる。

構文です。

place_meeting(x, y, obj);

論証 タイプ 商品説明
x リアル The x position to check.
y リアル The y position to check.
obj オブジェクトのアセットまたはインスタンスID The instance or object to check for.

リターンです。

Boolean

例

if keyboard_check(vk_left)
{
    if !place_meeting(x - 5, y, obj_wall) x -=5;
}

上記のコードでは、インスタンスの左側に衝突がないかどうかをチェックし、衝突がなければインスタンスを移動させます。
