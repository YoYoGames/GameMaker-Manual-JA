ポイント_イン_サークル

この関数は、円形の領域を定義すると、与えられた点がその範囲内にあるかどうかをGameMakerが計算します。点が定義された円内にある場合、この関数は true を返し、そうでない場合は false を返します。

構文です。

point_in_circle(px, py, x1, y1, rad);

論証タイプ 商品説明
px The x coordinate of the point to check.
py The y coordinate of the point to check.
x1 The x coordinate of the circle centre.
y1 The y coordinate of the circle centre.
rad The radius of the circle.

リターンです。

例

if point_in_circle(mouse_x, mouse_y, x, y, 16)
{
    over = true;
}
else
{
    over = false;
}

上記のコードでは、 point_in_circle 関数を使って、マウスの位置が定義された円形領域内にあるかどうかをチェックし、ある場合は変数 "over" に true を、そうでない場合は false をセットしています。
