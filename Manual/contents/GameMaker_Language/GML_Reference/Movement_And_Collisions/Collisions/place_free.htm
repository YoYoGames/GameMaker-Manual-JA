プレースフリー

この関数を使用すると、呼び出したインスタンスがゲーム内で ソリッドと判定されたインスタンスと衝突するかどうかを確認することができます。この関数が動作するためには、コードを実行するインスタンスが有効なコリジョンマスクを持っている必要があることに注意してください（sprite 自身か、あるいは mask_indexそして、有効なマスクを持っている固体フラグを持つインスタンスとのみ衝突を登録します。

この関数自体は、基本的にインスタンスを取得し、x/y 引数で指定された位置に配置されたときに、ソリッドフラグ付きのインスタンスのみと衝突するかどうかをテストすることで動作します。衝突チェックは、どのような衝突マスクが選択されたかに応じて、正確またはインスタンスのバウンディングボックスに基づくことができます。

なお、与えられたx/y座標は、チェックが行われる前に最も近い整数にフロアーされる。

構文です。

place_free(x, y);

論証タイプ 商品説明
x The x position to check.
y The y position to check.

リターンです。

例

if place_free(mouse_x, mouse_y)
{
    x = mouse_x;
    y = mouse_y;
}

上記のコードは、呼び出したインスタンスがマウスと同じ位置に配置された場合、任意のソリッドフラグ付きインスタンスとの衝突をチェックします。衝突が検出されなかった場合、そのインスタンスの x/y 座標はマウスの座標に設定されます。
