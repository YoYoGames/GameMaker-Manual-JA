collision_line

Collision_line は、点 x1,y1 から点 x2,y2 までの直線に沿って、引数 "obj" で指定された のobject 任意のインスタンスとの衝突をチェックします。このチェックは正確でもそうでなくてもかまいませんが、正確な衝突を有効にするには、チェックする またはインスタンスがその に対して正確な衝突も有効になってobject spriteいなければなりません。そうでない場合は、デフォルトのチェックがバウンディングボックスに基づきます。次の画像は、この動作を示しています。

精密衝突を考慮するためには、object sprite と collision 関数の両方に、on としてマークされている必要があることに注意してください。また、関数の戻り値には、線上で衝突したとみなされるインスタンスのうち、任意の1つのIDを指定することができるため、3つのインスタンスが定義された線に重なった場合、それらのIDのうちどれかが関数の戻り値となる可能性があることにも注意が必要です。

object インデックスの代わりにインスタンスを指定することができます。 idで特定のインスタンスをチェックするか、インスタンスキーワード all, または other (イベントと現在のコードスコープによる) をチェックします。

構文です。

collision_line(x1, y1, x2, y2, obj, prec, notme);

論証 タイプ 商品説明
x1 リアル The x coordinate of the start of the line.
y1 リアル The y coordinate of the start of the line.
x2 リアル The x coordinate of the end of the line.
y2 リアル The y coordinate of the end of the line.
obj オブジェクトのアセットまたはインスタンスID The object to check for instance collisions.
prec ブーリアン Whether the check is based on precise collisions (trueより遅い）、またはそのバウンディングボックス全般（false 、より速い）。
notme ブーリアン Whether the calling instance, if relevant, should be excluded (true) か、そうでないか (false) です。

リターンです。

Instance ID or noone

例

with (obj_Enemy)
{
    if collision_line(100, 400, 100, 600, id, false, false) instance_destroy();
}

このコードは、obj_Enemyのすべてのインスタンスを取得し、100x, 400y から 100x, 600y までのラインに沿って自分自身との衝突をチェックし、衝突があった場合は破壊するものです。
