ポジション・デストロイ

この関数は、指定された位置と衝突することが判明したすべてのインスタンスを単純に破壊する。衝突はインスタンスのマスクに基づいて行われ、もしマスクの一部が目標点と重なった場合、この関数はそのインスタンスを破壊します。このようにして破壊されたインスタンスは、DestroyおよびClean Upイベントをトリガします。

構文です。

position_destroy(x, y);

論証タイプ 商品説明
x The x coordinate of where to destroy colliding instances.
y The y coordinate of where to destroy colliding instances.

リターンです。

例

if mouse_check_button_pressed(mb_left)
{
    position_destroy(mouse_x, mouse_y);
}

これは、マウスの左ボタンを押したときに、マウスカーソルの位置と衝突するすべてのインスタンスを破壊します。
