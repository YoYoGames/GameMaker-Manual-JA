プレーススナップ

この関数により、インスタンスの原点（x、y位置）が、指定したhsnap、vsnap値のグリッドにアラインされているかどうかを確認することができます。

構文です。

place_snapped(hsnap, vsnap);

論証タイプ 商品説明
hsnap The horizontal snapping to check.
vsnap The vertical snapping to check.

リターンです。

例

with (obj_Pieces)
{
    if (!place_snapped(32, 32))
    {
        move_snap(32, 32);
    }
}

上記のコードは、"obj_Pieces "のすべてのインスタンスが32x32ピクセルのグリッドにスナップされているかどうかをチェックし、そうでない場合は、そのグリッドの最も近い位置にスナップされます。
