移動_連絡先すべて

この関数は、コードを実行しているインスタンスを、有効なマスクを持つ他のインスタンスに出会うまで、指定された方向に設定されたピクセル数だけ移動させます。最大距離には-1または0を指定でき、デフォルトでは1000ピクセルとなっています。

構文です。

move_contact_all(dir, maxdist);

論証タイプ 商品説明
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

リターンです。

例

if !place_meeting(x, y + 1, all)
{
    move_contact_all(270, -1);
}

上記のコードは、インスタンスの下に衝突がないかどうかをチェックし、衝突がない場合は、衝突があるまで、または1000ピクセルがカバーされるまでインスタンスを下に移動させるものです。
