モーションセット

この関数は、コードを実行しているインスタンスに新しい移動方向と速度を設定します。これはインスタンスの現在のスピードと方向には追加されないことに注意してください（そのためには motion_add()を強制的に設定する。

構文です。

motion_set(dir, speed);

論証タイプ 商品説明
dir The new direction.
speed The new speed.`

リターンです。

例

if irandom(9) = 1
{
    motion_set(random(360), 1 + random(3));
}

このコードによって、インスタンスはランダムな間隔で速度と方向を変更するようになります。
