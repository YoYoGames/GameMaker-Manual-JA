move_random

この関数を使うと、インスタンスをroom の任意の位置に設定できますが、「見えない」グリッドに整列させることができます。つまり、hsnap と vsnap の値を 32 にすると、32x32 の正方形のグリッドに整列したランダムな位置にインスタンスが設定されます。これらの値を 1 にすると、room の任意の位置に設定することができます。

注：この関数は、ゲームが新しく実行されるたびに同じ位置にスナップします。これは、GameMakerが毎回同じ初期ランダムシードを生成して、debugging のコードをはるかに簡単に作成できるようにするためです。この挙動を避けるには randomise()をゲーム開始時に指定します。これは、ゲームのテストとdebugging 、最終的な実行可能パッケージはこの動作を示さず、毎回のプレイがランダムになるためです。

構文です。

move_random(hsnap, vsnap);

論証タイプ 商品説明
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

リターンです。

例

if keyboard_check(vk_space)
{
    move_random(1, 1);
}

これは、スペースキーを押したときに、room の任意の位置にインスタンスを移動させます。
