ポイントへ移動

この関数は、コードを実行しているインスタンスに、設定された速度で設定されたポイントに向かって移動するように指示するだけです。ただ勝手に止まることはないので、そこに到達したら何をするかはあなた次第であることを心に留めておいてください。つまり、例えば3ピクセル前に移動し、x/y目標位置を2オーバーシュートし、また3ピクセル後ろに移動し、目標点を1オーバーシュートし、・・・というように「振動」するインスタンスを得ることができるのです。これを回避する方法はいくつかあります。 distance_to_point()を使用して、インスタンスが移動する前に指定位置からどれくらい離れているかを確認します。

注意：この関数はインスタンスの速度を設定するので、このコードの使用を停止しても、インスタンスは以前の方向に移動し続けるので、インスタンスを設定する必要があります。 speedを0に設定します。以下のコード例をご覧ください。

構文です。

move_towards_point( x, y, sp );

論証 タイプ 商品説明
x リアル The x position of the point to move towards.
y リアル The y position of the point to move towards.
sp リアル The speed to move at in pixels per step.

リターンです。

N/A

例

if point_distance(x, y, target.x, target.y) > 5
{
    move_towards_point(target.x, target.y, 5);
}
else speed = 0;

上記のコードは、変数 "target "でインデックスされたインスタンスの位置に向かって、1ステップあたり5ピクセルの速度でインスタンスを移動させます。その位置に到達すると、速度は0に設定される。
