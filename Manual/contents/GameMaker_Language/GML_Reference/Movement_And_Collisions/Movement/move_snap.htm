move_snap

この関数は、指定されたサイズのグリッドにインスタンスを「スナップ」させるために使用される。hsnapとvsnapの値で定義される「見えない」グリッド上の最も近い対応する位置にスナップされる。

構文です。

move_snap(hsnap, vsnap);

論証タイプ 商品説明
hsnap The horizontal snapping (the size in pixels between 'cells').
vsnap The vertical snapping (the size in pixels between 'cells').

リターンです。

例

with (obj_Pieces)
{
    if !place_snapped(32, 32)
    {
        move_snap(32, 32);
    }
}

上記のコードは、"obj_Pieces "のすべてのインスタンスが32x32ピクセルのグリッドにスナップされているかどうかをチェックし、そうでない場合は、そのグリッドの最も近い位置にスナップされます。
