move_bounce_all

この機能を使うと、room 内のすべてのインスタンスを跳ね返すように命令できます。ただし、sprite またはマスクインデックスが割り当てられていないインスタンスだけは例外です。ただし、この場合、すべてのインスタンスで精密なマスクが有効になっている必要があり、多くのインスタンスが関与している場合は、処理量が増えるためゲームの速度が大幅に低下することに注意してください。これは通常インスタンスの step イベントで使用されますが、以下のコード例のように collision イベントで選択的に使用することもできます。

構文です。

move_bounce_all( adv );

論証タイプ 商品説明
adv Whether to enable advanced bouncing (true) or not (false).

リターンです。

例

if other.visible
{
    move_bounce_all(false);
}

上記のコードは、別のobject との衝突イベントの中に置かれます。まずobject が表示されているかどうかを確認し、表示されていれば move_bounce_all() のアクションを実行します。この場合、バウンスは選択的で、すべてのインスタンス、すべてのステップに対してではなく、この衝突に対してのみ計算されることに注意してください。
