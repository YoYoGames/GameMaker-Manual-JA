motion_add

この関数は、コードを実行しているインスタンスの現在の direction と speed を変更し、新しい値を現在の値に追加します。これらの値を単純に変更したい場合は、関数 motion_set() を使用する必要があります。

構文です。

motion_add(dir, speed);

論証タイプ 商品説明
dir The added direction.
speed The added speed.

リターンです。

N/A

例

var pdir;
pdir = point_direction(other.x, other.y, x, y);
motion_add(pdir, other.speed);
if speed > 8 speed = 8;

上記のコードは、他のobject との衝突イベントで呼び出されます。これは、インスタンスの移動方向と速度に、衝突に関与した他のインスタンスの位置と速度に基づくベクトルを追加します。そして、その速度が 8 (pixels per step) を超えると制限されます。
