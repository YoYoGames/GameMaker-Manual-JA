移動接点ソリッド

この関数は、コードを実行しているインスタンスを、指定された方向に指定されたピクセル数だけ移動させ、そのインスタンスが ソリッドのフラグを立てる。最大距離には-1か0が使え、デフォルトでは1000pxである。つまり、GameMakerはインスタンスを衝突しなくなるまで1000ピクセル上に動かし続ける。

構文です。

move_contact_solid( dir, maxdist );

論証タイプ 商品説明
dir The direction to move in.
maxdist The maximum distance the object can travel (-1 or 0 a default value of 1000 pixels).

リターンです。

例

if !place_meeting(x, y + 1, obj_Floor)
{
    move_contact_solid(270, -1);
}

上記のコードは、インスタンスの下にある "obj_Floor" との衝突をチェックし、衝突がない場合は、衝突が発生するか、1000ピクセルがカバーされるまで下に移動させるものです。
