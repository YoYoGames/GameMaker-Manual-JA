mp_potential_step

この関数は、xgoal/ygoal で定義された特定の位置に向かって、障害物を避けながら一歩を踏み出すインスタンスを生成する。インスタンスが固体のインスタンス（または checkall が true のときの任意のインスタンス）にぶつかると、そのインスタンスを避けて移動するように進行方向を変更します。この方法は必ずうまくいくわけではないが、たいていの場合、インスタンスをゴールに向かって効果的に移動させることができる。この関数は、ゴールに到達したかどうかを返します。

構文です。

mp_potential_step(xgoal, ygoal, stepsize, checkall)

論証タイプ 商品説明
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

リターンです。

例

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step(inst.x, inst.y, 5, false);
}

上記のコードは、まずroom の中に「obj_Enemy」のインスタンスがあるかどうかを調べます。 もしあれば、最も近いものを見つけてその ID を変数に格納します。この変数を使って、 mp_potential_step に、見つかったobject の x/y 位置に向かって、固体と判定されたインスタンスだけを避けながら、1 ステップあたり 5 ピクセルの速度でインスタンスを移動させるように指示します。
