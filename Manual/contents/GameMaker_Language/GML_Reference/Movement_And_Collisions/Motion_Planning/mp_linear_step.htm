mp_linear_step

この関数では、xgoalとygoalの値で指定された特定の点に向かって「一歩」を踏み出すようインスタンスに指示する。ステップのサイズ（インスタンスが一歩ごとに何ピクセル移動するか）は stepize で示され、インスタンスがすでにその位置にある場合は、それ以上移動せずに停止します。 move_towards_point().ステップサイズは、object が各ステップで衝突をチェックする「前方」の距離でもあります。インスタンスがすべてのインスタンスと衝突したときに停止するか、固体と判定されたインスタンスのみ停止するかを選択することができます。この関数は、ゴール位置に到達したかどうか（true）、あるいは失敗したかどうか（false）を返します。

注意この機能は、障害物に遭遇しても迂回しようとせず、単に失敗して動きを止めます。

構文です。

mp_linear_step(xgoal, ygoal, stepsize, checkall);

論証タイプ 商品説明
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check all instances (true) or just solid ones (false).

リターンです。

例

if mp_linear_step(mouse_x, mouse_y, 5, 0)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上記のコードでは、object が1ステップあたり5ピクセルの速度でマウスに向かって移動します。マウスの位置に到達すると、object "obj_Explosion" を作成し、自滅する。
