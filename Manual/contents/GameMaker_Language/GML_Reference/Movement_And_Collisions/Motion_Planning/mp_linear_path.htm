mp_linear_path

この関数は、現在のインスタンス位置から "xgoal "と "ygoal "の値で指定された点までの直線path を計算する。この関数では、指定されたステップサイズ、つまり、関数 mp_linear_step().指定されたpath はすでに存在しなければならず、新しいpath で上書きされます。この関数は、完全なpath が見つかった場合 (true) か、そうでない場合 (false) を返します。false が返された場合，path が生成されますが，path がブロックされた位置までしか実行されません．

注：この関数はインスタンスを移動させません。これはpath を設定するだけで、移動にはPath関数を使用する必要があります。

構文です。

mp_linear_path(path, xgoal, ygoal, stepsize, checkall);

論証タイプ 商品説明
path The index of the path to be used.
xgoal The target x position.
ygoal The target y position.
stepsize The speed the instance moves in pixels per step.
checkall Whether to check for collisions with all instances (true) or just solid ones (false).

リターンです。

例

if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0)
{
    path_start(path, 2, 0, 0);
}

これは、object 、現在の位置からobj_Playerの位置まで直線的なpath があるかどうかをチェックするようになります。もし存在すれば、path を開始します。
