mp_grid_get_cell

この関数では、mp_grid の任意のセルが占有されているかどうかを、(以前に作成した) mp_grid のインデックスとチェックするセルの x および y 座標を与えて確認することができます。

占有されている場合、またはチェックする位置がグリッドの境界から外れている場合は-1、それ以外の場合は0を返す。

構文です。

mp_grid_get_cell(id, x , y);

論証 タイプ 商品説明
id MP Grid ID Index of the mp_grid that is to be used
x1 リアル The x coordinate of the grid to check.
y1 リアル The y coordinate of the grid to check.

リターンです。

Real

例

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

上記のコードでは、マウス位置に対応する mp_grid セルをチェックし、占有されていれば image_blend 変数を赤に、占有されていなければ緑に設定します。
