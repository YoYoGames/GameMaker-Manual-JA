mp_potential_step_object。

この関数は、xgoal/ygoal で定義された特定の位置に向かって、障害物を避けながら一歩を踏み出すインスタンスである。インスタンスは、引数 "obj" で指定されたobject のインスタンスにぶつかると、そのインスタンスを避けて移動するように移動方向を変更します。この方法は必ずうまくいくわけではないが、たいていの場合、インスタンスをゴールに向かって効果的に移動させることができる。この関数は、ゴールに到達したかどうかを返す。

構文です。

mp_potential_step_object(xgoal, ygoal, stepsize, obj)

論証 タイプ 商品説明
xgoal リアル The target x position.
ygoal リアル The target y position.
stepsize リアル The speed the instance moves in pixels per step.
obj オブジェクトのアセットまたはインスタンスID The object that is to block the path of the instance running the function. Can be an object index, an instance id or the special keyword ,all.

リターンです。

Boolean

例

if instance_exists(obj_Enemy)
{
    var inst;
    inst = instance_nearest(x, y, obj_Enemy);
    mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
}

上記のコードは、まずroom に「obj_Enemy」のインスタンスがあるかどうかをチェックします。 もしあれば、最も近いものを見つけてその ID を変数に格納します。この変数を使って、 mp_potential_step_object に、見つかったobject の x/y 位置に向かって、object の "obj_Wall" インスタンスだけを避けながら、1ステップあたり5ピクセルの速度でインスタンスを移動させるように指示します。
