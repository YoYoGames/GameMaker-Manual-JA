mp_grid_to_ds_grid。

この関数を使うと、与えられたMPグリッドをDSグリッドにコピーすることができます。DS グリッドは MP グリッドと同じサイズである必要があるが、必ずしもそうである必要はない (小さいとデータが失われ、大きいと余分なグリッドセルはすべて 0 になる)。DS グリッドのセルは、MP グリッドで占有フラグが立っていた場合は -1、立っていなかった場合は 0 を格納する。

構文です。

mp_grid_to_ds_grid(source, destination);

論証 タイプ 商品説明
source MP Grid ID Index of the mp_grid that is to be used
destination DS Grid ID Index of the ds_grid that is to be used to copy the grid data to.

リターンです。

Boolean

例

motion_grid = ds_grid_create(room_width / 32, room_height / 32);
mp_grid_to_ds_grid(mp_grid, motion_grid);

上記のコードでは、新しい DS グリッドを作成し、変数 "mp_grid" に含まれる MP グリッドのデータを新しい DS グリッドにコピーしています。
