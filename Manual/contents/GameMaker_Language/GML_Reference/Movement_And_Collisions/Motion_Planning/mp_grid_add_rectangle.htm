mp_grid_add_rectangle

この関数は、room 内に矩形を定義するよう求めます。そして、その矩形に「触れる」すべての MP グリッドセルを禁則としてマークし、パス検索関数がそれらを横切ることができないようにします。下の画像は、この機能を説明するためのものです。

見ての通り、(50, 90) から (200, 180) で定義される矩形は、それに接する等価な MP グリッドセルをすべて禁則としてマークしています。

構文です。

mp_grid_add_rectangle(id, x1, y1, x2, y2);

論証 タイプ 商品説明
id MP Grid ID Index of the mp_grid that is to be used
x1 リアル The x coordinate of the left side of the rectangle to check.
y1 リアル The y coordinate of the top side of the rectangle to check.
x2 リアル The x coordinate of the right side of the rectangle to check.
y2 リアル The x coordinate of the bottom side of the rectangle to check.

リターンです。

N/A

例

mp_grid_add_rectangle(grid, 0, 0, 100, 200);

上記のコードは、変数 "grid" でインデックスされた mp_grid のセルで、領域 0,0 から 100,200 に入るものをすべて禁じ手としてマークすることになります。
