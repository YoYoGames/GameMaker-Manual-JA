mp_grid_clear_cell

この関数は、MPグリッドの特定の "セル "をクリアするために使用できる。セルはroom の座標としてではなく、グリッド座標として計算されます。(0,0) はグリッドのトップレットのコーナーです。つまり、room の特定の位置のセルをクリアするには、x と y の座標を MP グリッドの分解能で割ったセル座標に変更しなければならないのです。以下のコード例は、この動作を示しています。

構文です。

mp_grid_clear_cell(id, h, v);

論証 タイプ 商品説明
id MP Grid ID Index of the mp_grid that is to be used
h リアル Horizontal position of the cell to clear
v リアル vertical position of the cell to clear

リターンです。

Boolean

例

with (obj_Box)
{
    mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
    instance_destroy();
}

上記のコードは、すべての "obj_Box" を自壊させ、変数 "grid" でインデックスされた mp_grid の中で占有していたセルを空きセルとしてマークさせます。この例では、object の x/y 座標をグリッドの解像度で割って、適切なセルを探します ( floor を使って、値を整数にしています)。
