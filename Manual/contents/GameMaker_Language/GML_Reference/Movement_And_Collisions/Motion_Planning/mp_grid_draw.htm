mp_grid_draw

この関数は、指定された MP グリッド（で定義される）を描画します。 mp_grid_create()) があり、空きセルは緑、禁止セルは赤で表示されます。この関数はデバッグツールとして不可欠ですが、非常に遅く、インスタンスのdrawイベントで使用したときのみ動作することに注意する必要があります。また、draw alpha を設定してグリッドの不透明度を変更できるため、オーバーレイとして描きながらroom に実際に何が表示されているかを同時に確認することができます。

構文です。

mp_grid_draw(id);

論証 タイプ 商品説明
id MP Grid ID Index of the mp_grid that is to be drawn

リターンです。

Boolean

例

draw_set_alpha(0.3);
mp_grid_draw(grid);
draw_set_alpha(1);

上記のコードは、変数 "grid" でインデックスされた mp_grid を半透明のオーバーレイとして描画します (ただし、このコードを実行しているインスタンスの深さが残りのすべてよりも低い場合に限ります)。
