mp_grid_destroy

この関数は、指示された mp_grid を破棄するために使用します(関数 mp_grid_create()に割り当てられたメモリを解放します。MPグリッドが終了するたびにこれを呼び出すことが重要です。さもないと、メモリリークが発生する可能性があり、時間とともにゲームの動作が遅くなり、最終的にはクラッシュすることになります。

注：グリッドが破壊された後に mp_grid_* 関数を使用すると、エラーになります。

構文です。

mp_grid_destroy(id);

論証 タイプ 商品説明
id MP Grid ID Index of the mp_grid that is to be destroyed

リターンです。

N/A

例

if timer = 0
{
    mp_grid_destroy(grid);
    room_goto(rm_Menu);
}

上記のコードは、変数 "timer" が 0 の場合、変数 "grid" でインデックスされた mp_grid を破壊し、別のroom に移動します。
