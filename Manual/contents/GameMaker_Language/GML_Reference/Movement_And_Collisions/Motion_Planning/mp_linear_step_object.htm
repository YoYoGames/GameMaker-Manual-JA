mp_linear_step_object

この関数では、xgoal と ygoal の値で指定された特定の点に向かって「一歩」を踏み出すようにインスタンスに指示する。ステップのサイズ（インスタンスが一歩ごとに何ピクセル移動するか）は stepize で示され、インスタンスがすでにその位置にある場合は、それ以上移動せずに停止します。 move_towards_point().ステップサイズは、object が各ステップで衝突をチェックする「前方」の距離でもあります。また、object または任意のインスタンスと衝突したときにインスタンスが停止するかどうかを選択することができます。

構文です。

mp_linear_step_object(xgoal, ygoal, stepsize, obj);

論証 タイプ 商品説明
xgoal リアル The target x position.
ygoal リアル The target y position.
stepsize リアル The speed the instance moves in pixels per step.
obj オブジェクトのアセットまたはインスタンスID The object that is to block the path. Can be an object index, an instance id or the special keyword, all

リターンです。

Boolean

例

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

上記のコードでは、object がマウスに向かって1ステップあたり5ピクセルの速度で移動し、object "obj_Wall" との衝突だけをチェックします。マウスの位置に到達すると、object "obj_Explosion" を生成して自滅します。
