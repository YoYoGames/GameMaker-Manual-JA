mp_linear_path_object

この関数は、現在のインスタンス位置から "xgoal "と "ygoal "の値で指定された点までの直線path を計算する。この関数では、指定されたステップサイズ、つまり、関数 mp_linear_step().指定されたpath はすでに存在しなければならず、新しいpath で上書きされます。この関数は、完全なpath が見つかった場合 (true) か、見つからなかった場合 (false) に返されます。完全なpath が見つかるのは，指定されたobject またはインスタンスと衝突しなかった場合のみです．false が返された場合もpath が生成されますが，それはpath がブロックされた位置までしか実行されません．

注：この関数はインスタンスを移動させません。これはpath を設定するだけで、移動にはPath関数を使用する必要があります。

構文です。

mp_linear_path_object(path, xgoal, ygoal, stepsize, obj);

論証 タイプ 商品説明
path パスアセット The index of the path to be used
xgoal リアル The target x position.
ygoal リアル The target y position.
stepsize リアル The speed the instance moves in pixels per step.
obj オブジェクトのアセットまたはインスタンスID The object that is to block the path. Can be an object index, an instance id or the special keyword ,all

リターンです。

Boolean

例

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
    path_start(path, 4, 0, 0);
}

上記のコードは、コードを実行しているpath とマウスの x/y 位置の間のobject に沿って "obj_Wall" と衝突していないかどうかをチェックします。衝突がなく、完全なpath が生成された場合、生成されたobject に沿ってpath を開始します。
