throw runtime 文では、次のような構文で、独自のランタイム例外を生成し、ゲームを終了し、エラーメッセージを表示することができます。

throw (<expression>);

使用する式は、値でもstring でも、その他のデータ型でも構いません。この式は例外エラーを生成し、デフォルトで画面に表示され、エラーメッセージを閉じるとゲームは終了します。たとえば、次のように呼び出します。

throw ("Hello World!");

を実行すると、次のような未処理の例外エラーが表示されます。

しかし、このエラーメッセージを引き継いで、独自のハンドラコードを使用することができます。 関数 exception_unhandled_handler().このランタイム関数では、カスタムメイドの method この例外は、ゲーム内で処理されない例外が発生するたびに呼び出されます。
