新

new 演算子は、与えられた引数で呼び出される関数でstruct を作成するために使用されます。この関数はstruct に引数の変数を代入し、 new 演算子はstruct を返します。この演算子を使用する前に、与えられた関数が constructor 関数としてフラグされていなければならないことに注意することが重要です。そうでなければ new 演算子はstruct を作成しません (以下のコード例でこれを示します。より詳しい情報はStruct のページを参照してください)。

注 struct 関数を使用すると、runtime をチェックして、どの関数で作成されたかを知ることができます。 instanceof().

この演算子の構文は次のとおりである。

<variable> = new <function> (<argument>, <argument>, ...);

関数で new 演算子を使用すると、指定した変数に格納されているstruct の参照が返され、関数によってその中に作成されたstruct と任意の変数にアクセスできるようになります。

次の例は、コンストラクターとして使用するために定義された関数を示しています。

function init_struct(_a, _b, _c) constructor
{
    a = _a;
    b = _b;
    c = _c;
}

この関数を new 演算子とともに使用して、次のようにstruct を作成し、その中に関数で使用した引数の値を設定した変数を入れることができます。

mystruct = new init_struct(10, 100, "Hello World");

注意すべきは、コンストラクタとしてフラグが立っていない関数でnewを呼び出すと、実行時例外が発生することです。
