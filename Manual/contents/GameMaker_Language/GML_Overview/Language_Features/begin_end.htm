開始/終了

GameMaker Language を使用すると、コードブロックを作成する際に、通常の中括弧 {} の代わりに、キーワード begin と end を使用することができます。以下のコード例では、この機能がどのように働くかを示しています。

if (!visible)
    begin
    exit;
    end
while (place_meeting(x, y))
    begin
    x -= lengthdir_x(1, direction - 180);
    y -= lengthdir_y(1, direction - 180);
    end

これらのキーワードを使用することは一般的ではなく，むしろレガシーサポートのために言語の一部として提供されており，将来いつでも非推奨となる可能性があることに注意してください。
