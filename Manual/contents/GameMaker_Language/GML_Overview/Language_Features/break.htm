break 文は、早期に終了させるためのものです。 for, repeat, whileまたは do / untilloop を指定する、あるいは switchステートメントをその時点で終了させるか、あるいは早期に終了させる。 withを呼び出すことができます。それぞれの状況下での使用方法についてのより詳細な説明は、これらの異なる関数の個々のページを参照してください。 break をこれらの状況以外で使用すると、エラーが発生することに注意してください。

以下では、 break の基本的な使用例をいくつか紹介します。その構文は簡単です。

break;

break を でループさせる。 for

for (var i = 0; i < 10; i += 1)
{
    if (array[i] == 234)
    {
        break;
    }
}

num = i;

break を でループさせる。 repeat

var i = 0;
var temp = 0;
repeat (10)
{
    temp += array[i];
    if (temp > max_total)
    {
        break;
    }
    else
    {
        i += 1;
    }
}

break を でループさせる。 while

var i = 0;
while (!place_free(x, y))
{
    x = random(room_width);
    y = random(room_height);
    if (i > 50)
    {
        break;
    }
    else
    {
        i+=1;
    }
}

break を でループさせる。 do / until

var _id = noone;
do
{
    _id = list[| 0];
    if instance_exists(_id)
    {
        break;
    }
    ds_list_delete(list, 0);
}
until (ds_list_empty(list));
target = _id;

break を使用する場合 : with

var count = 0;
with (obj_Enemy)
{
    count++;
    if (count > 10)
    {
        break;
    }
    hp = 100;
}

switch (keyboard_key)
{
