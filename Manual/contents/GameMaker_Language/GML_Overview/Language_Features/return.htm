return 文は次のような構文です。

return (<expression>)

return ステートメントは スクリプトの関数と メソッドでのみ使用し、関数から値を返して、次のコードや関数の呼び出しで使用するために使用します。関数の実行は return ステートメントで終了します。つまり、 return が呼び出された後に来るコードは実行されないということに注意してください。script 以下は sqr_calc という短い例です。この関数は渡された値の2乗を計算し、渡されたパラメータが実数でない場合のエラー処理も含まれています。

/// @function            sqr_calc(val);
/// @param {real}  val   The value to calculate the square of
/// @description         Calculate the square of the given value

function sqr_calc(value)
{
    if (!is_real(value))
    {
        return 0;
    }
    else
    {
        return (value * value);
    }
}

コードの一部から関数を呼び出すには、runtime 関数を呼び出すときと同じように使用します。つまり、関数名を書き、パラメータの値を括弧で囲んでください。つまり、上記の関数は次のように呼び出すことになります。

if keyboard_check_pressed(vk_enter)
{
    val = scr_sqr(amount);
}

ここで、変数 val は0（変数 amount が実数でないため）か、 value * value の合計のどちらかになります。
