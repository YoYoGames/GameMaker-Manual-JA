delete 演算子は、構造体の参照を解除するために使用され、次のような構文になります。

delete <variable>;

つまり、struct を保持する変数と一緒に delete 演算子を呼び出すと、与えられた変数に格納されているstruct への特定の参照が 削除されます（参照とは、単にメモリ領域を指す値です）。 struct とその内容を保存している)。この参照がゲーム内のstruct への最後の参照であった場合、ガベージコレクタは struct をメモリから削除します。 次の反復処理（通常は次のステップの開始時）。

注： delete は変数と共にのみ使用可能で、式は使用できません。

デフォルトでは、コード内でそれ以上参照されないstructs は、その使用に続くステップで自動的にガベージコレクションされますが、この演算子を使用してflag を明示的にガベージコレクタにメモリから削除させるのは良い習慣です。 次のステップの開始時にメモリを解放することで、より速く、より効率的にメモリを解放することができます。

次の例は、struct が作成される様子を示しています - たとえば、インスタンスの Create Event で。

mystruct =
{
    a : 0,
    b : 100,
    c : "Hello World"
}

このstruct は、通常通りインスタンスイベントで使用され、最終的に以下のようにクリーンアップイベントでガベージコレクションのフラグが立てられる。

delete mystruct;
