continue 文の基本的な構文は次のとおりです。

continue;

loop を形成する文の内部で使用された場合 (for, repeat, whileまたは do / until) を使用すると、現在の反復処理を直ちに終了し、loop の先頭にジャンプして新しい反復処理を開始し、loop 内の continue の後に来るコードはすべて省略されます。 with文では、次のインスタンスにスキップして、再度実行されます。 continue をこれらのコンテキスト以外で使用すると、エラーになることに注意してください。

以下は、 for ループでの使用例です。

var _val = 0;

for (var i = 0; i < 10; i += 1)
{
    if (val_array[i] <= 0)
    {
        continue;
    }
    _val += val_array[i];
}

draw_text(32, 32, "Positive Values Total = " + string(_val));

var file = file_text_open_read("Game_Data.txt");
