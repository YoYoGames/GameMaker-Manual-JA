言語機能

GameMaker Language (GML) には、この言語を使って書かれたすべてのプログラムの本質的な構造の一部となる、いくつかの機能があります。これらの各機能は、あなたが書いたコードで何度も使用されるので、プロジェクトのプログラミングを始める前に、それらが何であり、どのように動作するかを知っておく必要があります。

  • repeat
  • while
  • do / until
  • for
  • switch
  • break
  • continue
  • exit
  • with
  • return
  • begin / end
  • try / catch / finally
  • throw
  • new
  • delete
