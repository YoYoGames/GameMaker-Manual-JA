せいかつ

この変数はグローバルなスコープで、通常プレイヤーのライフに使用される数値を保持するために使用されます。この変数はGameMakerの旧バージョンからのレガシープロジェクトをサポートするためだけに設計されています。 新規プロジェクトでは使用しない将来的に非推奨となる可能性があるため。

構文です。

lives;

リターンです。

Real (single precision floating point value)

例

if (lives < 5)
{
    lives += 1;
}

上記のコードでは、変数 lives をチェックし、5より小さければ1が加算されます。
