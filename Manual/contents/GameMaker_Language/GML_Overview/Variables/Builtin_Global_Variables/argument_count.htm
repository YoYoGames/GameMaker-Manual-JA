引数カウント

この読み取り専用変数は、スクリプト関数に渡される「引数」の数を返すか、または method.通常、引数配列(argument[0...15]) を使って、与えられた関数の入力引数を変化させることができます。

注：これはもう厳密には必要ではなく、何よりもレガシーサポートのために提供されています。すべてのユーザ定義関数は変数の引数を受け取ります。関数に渡される引数のうち、名前付き入力の一部でないものは、デフォルトでundefinedに初期化されます。

構文です。

argument_count;

リターンです。

例

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

上記のコードでは、変数 argument_count と配列 argument を使って変数を初期化していますが、これはユーザー定義関数が可変数の引数を受け取ることを許可する方法の一例です。
