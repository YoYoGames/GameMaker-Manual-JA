健康

この変数はグローバルなもので、通常プレイヤーの体力に使用される数値を保持するために使用されます。この変数はGameMakerの旧バージョンからのレガシープロジェクトをサポートするために設計されており、また、GameMakerの旧バージョンからのレガシープロジェクトをサポートする必要があります。 新規プロジェクトでは使用しない将来的に非推奨となる可能性があるため。

構文です。

health;

リターンです。

Real (single precision floating point value)

例

if (health <= 0)
{
    global.state = "Game Over";
    instance_destroy();
}

上記のコードでは、変数 health をチェックし、0 以下であれば、グローバル変数を設定し、インスタンスを破棄しています。
