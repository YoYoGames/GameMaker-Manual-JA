議論

この変数は配列を保持し、読み取り専用の変数である argument_countスクリプトの関数や メソッドで使用されます。配列の各位置には関数の入力値が格納され、特に変数 引数関数で使用されます。

また、ユーザー定義関数の中で、異なる入力引数を参照するために使用できる、一連の独立したグローバルスコープの変数があることに注意してください。 argument0 argument1, argument2, etc...argument15 までは。

注：これはもう厳密には必要ではなく、何よりもレガシーサポートのために提供されています。すべてのユーザ定義関数は変数の引数を受け取ります。関数に渡される引数のうち、名前付き入力の一部でないものは、デフォルトでundefinedに初期化されます。

構文です。

argument[n]
argument1
argument2
...
argument15

リターンです。

Variable (can be of any data type supplied to the function)

例

function my_array()
{
    var i, arg;
    for (i = 0; i < 5; i += 1;)
    {
        if argument_count > i
        {
            arg[i] = argument[i]
        }
        else
        {
            arg[i] = -1;
        }
    }
}

上記のコードでは、変数 argument_count と配列 argument を使って変数を初期化していますが、これはユーザー定義関数が可変数の引数を受け取ることを許可する方法の一例です。
