点数

この変数はグローバルなもので、通常プレイヤーのスコアに使用される数値を保持するために使用されます。この変数はGameMakerの以前のバージョンからのレガシープロジェクトをサポートするためだけに設計されています。 新規プロジェクトでは使用しない将来的に非推奨となる可能性があるため。

構文です。

score;

リターンです。

Real (single precision floating point value)

例

switch (object_index)
{
    case obj_Enemy_Fighter:
        score += 10;
    break;

    case obj_Enemy_Mage:
        score += 25;
    break;

    case obj_Enemy_Boss:
        score += 100;
    break;
}
instance_destroy();

上記のコードでは、コードを実行しているインスタンスのobject のインデックスを switchで、それが何であるかに応じて、 score 変数に異なる量を追加しますobject.
